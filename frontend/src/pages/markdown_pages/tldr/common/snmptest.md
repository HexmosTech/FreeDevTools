# snmptest

> Start an interactive SNMP prompt.
> More information: <https://manned.org/snmptest>.

- Start the prompt:

`snmptest -v {{version}} -c {{community}} {{ip}}`

- Display help:

`snmptest {{[-h|--help]}}`
