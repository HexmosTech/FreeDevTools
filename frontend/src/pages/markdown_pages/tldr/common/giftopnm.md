# giftopnm

> Convert a GIF file into a PNM image.
> More information: <https://netpbm.sourceforge.net/doc/giftopnm.html>.

- Convert a GIF image to a Netpbm image pixel-for-pixel:

`giftopnm {{path/to/input.gif}} > {{path/to/output.pnm}}`

- Display version:

`giftopnm {{[-v|-version]}}`
