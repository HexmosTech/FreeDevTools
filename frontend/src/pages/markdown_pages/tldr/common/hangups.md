# hangups

> Third party client for Google Hangouts.
> More information: <https://github.com/tdryer/hangups>.

- Start `hangups`:

`hangups`

- Display troubleshooting information and help:

`hangups -h`

- Set a refresh token for hangups:

`hangups --token-path {{path/to/token}}`
