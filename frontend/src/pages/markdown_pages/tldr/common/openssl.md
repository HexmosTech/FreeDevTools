# openssl

> OpenSSL cryptographic toolkit.
> Some subcommands such as `req` have their own usage documentation.
> More information: <https://www.openssl.org>.

- Display help:

`openssl help`

- Display help for a specific subcommand:

`openssl help {{x509}}`

- Display version:

`openssl version`
