# snmpstatus

> Fetch fixed MIB status values.
> More information: <https://manned.org/snmpstatus>.

- Fetch the values:

`snmpstatus -v {{version}} -c {{community}} {{ip}}`

- Display help:

`snmpstatus {{[-h|--help]}}`
