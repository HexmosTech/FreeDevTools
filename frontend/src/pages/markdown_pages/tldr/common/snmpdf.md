# snmpdf

> Fetch disk space usage information.
> More information: <https://manned.org/snmpdf>.

- Fetch the disk space usage:

`snmpdf -v {{version}} -c {{community}} {{ip}}`

- Display help:

`snmpdf {{[-h|--help]}}`
