# srftopam

> Convert an SRF file to Netpbm format.
> See also: `pamtosrf`.
> More information: <https://netpbm.sourceforge.net/doc/srftopam.html>.

- Convert an SRF file to Netpbm format:

`srftopam {{path/to/file.srf}} > {{path/to/output.pam}}`
