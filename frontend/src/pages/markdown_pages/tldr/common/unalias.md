# unalias

> Remove aliases.
> See also: `alias`.
> More information: <https://www.gnu.org/software/bash/manual/bash.html#index-unalias>.

- Remove an alias:

`unalias {{alias_name}}`

- Remove all aliases:

`unalias -a`
