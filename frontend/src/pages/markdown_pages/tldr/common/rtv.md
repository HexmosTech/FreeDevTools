# rtv

> Reddit Terminal Viewer.
> Use `<ArrowKeys>` to navigate. `<ArrowRight>` and `<ArrowLeft>` to view and return from a submission, respectively.
> More information: <https://github.com/michael-lazar/rtv>.

- Open the front page:

`/front`

- Open a subreddit:

`/r/{{subreddit_name}}`

- Expand/collapse comments:

`<Space>`

- Open link:

`<o>`

- Log in:

`<u>`

- Display help:

`<?>`
