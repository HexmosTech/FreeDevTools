# gvim

> A Graphical User Interface version of Vim (Vi IMproved), a command-line text editor.
> See also: `vimdiff`, `vimtutor`, `nvim`, `vim`.
> More information: <https://www.vim.org>.

- Open `gvim`:

`gvim`

- Open a specific file:

`gvim {{path/to/file}}`
