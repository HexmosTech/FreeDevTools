# tpp

> Command-Line based presentation tool.
> More information: <https://github.com/cbbrowne/tpp>.

- View a presentation:

`tpp {{path/to/file}}`

- Output a presentation:

`tpp -t {{type}} -o {{path/to/output}} {{path/to/file}}`
