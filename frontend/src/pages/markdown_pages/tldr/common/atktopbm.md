# atktopbm

> Convert a Andrew Toolkit raster object to a PBM image.
> See also: `pbmtoatk`.
> More information: <https://netpbm.sourceforge.net/doc/atktopbm.html>.

- Convert a Andrew Toolkit raster object to a PBM image:

`atktopbm {{path/to/image.atk}} > {{path/to/output.pbm}}`
