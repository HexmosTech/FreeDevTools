# pnminvert

> Invert a PNM image.
> More information: <https://netpbm.sourceforge.net/doc/pnminvert.html>.

- Invert the colors or greyscale values in a PNM image:

`pnminvert {{path/to/input.pnm}} > {{path/to/output.pnm}}`
