# snmpnetstat

> Fetch networking status using SNMP.
> More information: <https://manned.org/snmpnetstat>.

- Fetch the networking status:

`snmpnetstat -v {{version}} -c {{community}} {{ip}}`

- Display help:

`snmpnetstat {{[-h|--help]}}`
