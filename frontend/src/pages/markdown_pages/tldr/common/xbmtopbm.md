# xbmtopbm

> Convert an X11 or X10 bitmap to a PBM image.
> More information: <https://netpbm.sourceforge.net/doc/xbmtopbm.html>.

- Convert an XBM image to a PPM image:

`xbmtopbm {{path/to/input_file.xbm}} > {{path/to/output_file.pbm}}`
