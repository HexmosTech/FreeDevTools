# bastet

> Clone of the game Tetris in the terminal.
> More information: <https://fph.altervista.org/prog/bastet.html>.

- Start a Tetris game:

`bastet`

- Navigate the piece horizontally:

`{{<ArrowLeft>|<ArrowRight>}}`

- Rotate the piece clockwise or counterclockwise:

`{{<Space>|<ArrowUp>}}`

- Soft drop the piece:

`<ArrowDown>`

- Hard drop the piece:

`<Enter>`

- Pause the game:

`<p>`

- Quit the game:

`<Ctrl c>`
