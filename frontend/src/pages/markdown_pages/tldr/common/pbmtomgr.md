# pbmtomgr

> Convert a PBM image into a MGR bitmap.
> See also: `mgrtopbm`.
> More information: <https://netpbm.sourceforge.net/doc/pbmtomgr.html>.

- Convert a PBM image into a MGR bitmap:

`pbmtomgr {{path/to/image.pbm}} > {{path/to/output.mgr}}`
