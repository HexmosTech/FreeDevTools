# aa-teardown

> Tear down all AppArmor profiles and disable AppArmor enforcement.
> More information: <https://gitlab.com/apparmor/apparmor/-/wikis/manpage_aa-teardown.8>.

- Disable all AppArmor profiles and stop enforcement:

`sudo aa-teardown`

- Display help:

`aa-teardown {{[-h|--help]}}`
