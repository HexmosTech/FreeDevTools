# informant

> Read Arch Linux package news.
> More information: <https://github.com/bradford-smith94/informant>.

- Read all unread news:

`sudo informant read`

- Check for news:

`informant check`

- List latest news:

`informant list`

- Display help:

`informant {{[-h|--help]}}`
