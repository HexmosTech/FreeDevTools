# nstat

> View network statistics.
> More information: <https://manned.org/nstat>.

- View network statistics since last time `nstat` was run:

`nstat`

- View all-time network statistics:

`nstat {{[-a|--ignore]}}`

- Display help:

`nstat {{[-h|--help]}}`
