# qm pending

> Get the virtual machine configuration with both current and pending values.
> More information: <https://pve.proxmox.com/pve-docs/qm.1.html>.

- Get the virtual machine configuration of a specific virtual machine:

`qm {{[p|pending]}} {{vm_id}}`
