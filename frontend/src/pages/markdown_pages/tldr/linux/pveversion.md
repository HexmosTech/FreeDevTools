# pveversion

> Proxmox VE version info.
> More information: <https://pve.proxmox.com/wiki/Command_Line_Tools#Software_version_check>.

- Print system version:

`pveversion`

- Print Proxmox subpackage versions:

`pveversion {{[-v|--verbose]}}`
