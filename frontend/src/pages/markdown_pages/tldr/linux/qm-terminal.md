# qm terminal

> Attach to the serial console of a VM.
> More information: <https://pve.proxmox.com/pve-docs/qm.1.html>.

- Attach to a terminal:

`qm {{[ter|terminal]}} {{100}}`

- Detach from a terminal:

`<Ctrl o>`
