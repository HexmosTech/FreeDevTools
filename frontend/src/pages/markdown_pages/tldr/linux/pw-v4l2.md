# pw-v4l2

> Use v4l2 programs within PipeWire.
> More information: <https://docs.pipewire.org/page_man_pw-v4l2_1.html>.

- Run a program:

`pw-v4l2 {{program}}`

- Display help:

`pw-v4l2 -h`
