# lxc-attach

> Attach to a container.
> More information: <https://linuxcontainers.org/lxc/getting-started/>.

- Attach to a container:

`sudo lxc-attach {{container_name}}`

- Display help:

`lxc-attach {{[-?|--help]}}`
