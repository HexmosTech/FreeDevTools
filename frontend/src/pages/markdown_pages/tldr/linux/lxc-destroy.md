# lxc-destroy

> Delete a container.
> More information: <https://linuxcontainers.org/lxc/getting-started/>.

- Delete a container:

`sudo lxc-destroy {{container_name}}`

- Display help:

`lxc-destroy {{[-?|--help]}}`
