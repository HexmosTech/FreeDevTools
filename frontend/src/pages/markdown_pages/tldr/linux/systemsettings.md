# systemsettings

> KDE central settings program.
> More information: <https://manned.org/systemsettings>.

- Open the settings GUI:

`systemsettings`

- List all possible KCM modules for `systemsettings`:

`systemsettings --list`

- Display help:

`systemsettings {{[-h|--help]}}`
