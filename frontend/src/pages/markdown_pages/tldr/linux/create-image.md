# create-image

> Create CPIO images.
> More information: <https://git.kernel.org/pub/scm/bluetooth/bluez.git/tree/tools/create-image.c>.

- Create a CPIO archive from the current directory:

`create-image {{[-o|--output]}} {{path/to/output.cpio}}`

- Display help:

`create-image {{[-h|--help]}}`
