# dconf reset

> Reset key values in dconf databases.
> See also: `dconf`.
> More information: <https://manned.org/dconf>.

- Reset a specific key value:

`dconf reset /{{path/to/key}}`

- Reset a specific directory:

`dconf reset -f /{{path/to/directory}}/`
