# eopkg

> Package manager for Solus.
> More information: <https://help.getsol.us/docs/user/package-management/basics>.

- Install a specific package:

`sudo eopkg install {{package}}`

- Update all packages:

`sudo eopkg upgrade`

- Search for packages:

`sudo eopkg search {{search_term}}`
