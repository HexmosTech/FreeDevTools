# go run

> Compile and run Go code without saving a binary.
> More information: <https://pkg.go.dev/cmd/go#hdr-Compile_and_run_Go_program>.

- Run a Go file:

`go run {{path/to/file.go}}`

- Run a main Go package:

`go run {{path/to/package}}`
