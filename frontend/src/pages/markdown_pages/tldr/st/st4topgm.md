# st4topgm

> Convert an SBIG ST-4 file to PGM.
> More information: <https://netpbm.sourceforge.net/doc/st4topgm.html>.

- Convert an SBIG ST-4 file to a PGM file:

`st4topgm {{path/to/input_file.st4}} > {{path/to/output.pgm}}`
