# gdm-stop

> Immediately stop running the GNOME Display Manager (GDM).
> See also: `gdm`, `gdm-binary`, `gdmsetup`, `gdm-restart`, `gdm-safe-restart`.
> More information: <https://manned.org/gdm>.

- Stop the GNOME Display Manager application:

`gdm-stop`
