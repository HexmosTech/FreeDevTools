# dolt add

> Add the contents of a table to the list of Dolt staged tables.
> More information: <https://docs.dolthub.com/cli-reference/cli#dolt-add>.

- Add a table to the list of staged tables (stage a table):

`dolt add {{table}}`

- Stage all tables:

`dolt add {{[-A|--all]}}`
