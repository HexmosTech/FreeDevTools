# sfdk make

> Executes make build step.
> More information: <https://github.com/sailfishos/sailfish-qtcreator/blob/master/share/qtcreator/sfdk/modules/20-building-mb2/doc/command.make.adoc>.

- Run make build:

`sfdk make`

- Run make with extra arguments:

`sfdk make {{arguments}}`
