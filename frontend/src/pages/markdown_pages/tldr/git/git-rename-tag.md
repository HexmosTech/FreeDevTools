# git rename-tag

> Rename a Git tag.
> Part of `git-extras`.
> More information: <https://github.com/tj/git-extras/blob/master/Commands.md#git-rename-tag>.

- Rename an existing Git tag locally and remotely:

`git rename-tag {{old_tag_name}} {{new_tag_name}}`
