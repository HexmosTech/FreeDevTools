# git extras

> Git extension pack.
> More information: <https://manned.org/git-extras>.

- Install or upgrade `git-extras` commands:

`git extras update`

- Display help:

`git extras {{[-h|--help]}}`

- Display version:

`git extras {{[-v|--version]}}`
