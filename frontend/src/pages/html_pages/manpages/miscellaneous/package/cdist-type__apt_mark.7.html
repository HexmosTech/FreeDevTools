NAME
       cdist-type__apt_mark - set package state as 'hold' or 'unhold'


DESCRIPTION
       See apt-mark(8) for details.


REQUIRED PARAMETERS
       state  Either "hold" or "unhold".


OPTIONAL PARAMETERS
       name   If supplied, use the name and not the object id as the package name.


EXAMPLES
          # hold package
          __apt_mark quagga --state hold
          # unhold package
          __apt_mark quagga --state unhold
