NAME
       cdist-type__package_akp - Manage packages with akp


DESCRIPTION
       apk is usually used on Alpine to manage packages.


REQUIRED PARAMETERS
       None


OPTIONAL PARAMETERS
       name   If supplied, use the name and not the object id as the package name.

       state  Either "present" or "absent", defaults to "present"


EXAMPLES
          # Ensure zsh in installed
          __package_apk zsh --state present

          # Remove package
          __package_apk apache2 --state absent
