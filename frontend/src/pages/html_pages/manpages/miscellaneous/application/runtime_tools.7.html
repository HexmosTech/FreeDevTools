NAME
       runtime_tools - The Runtime tools Application


DESCRIPTION
       This   chapter   describes   the   Runtime_Tools   application  in  OTP,  which  provides  low  footprint
       tracing/debugging tools suitable for inclusion in a production system.


CONFIGURATION
       There are currently no configuration parameters available for this application.
