NAME
       flames_utl_unpack - Unpack flames-uves packed frames


SYNOPSIS
       esorex flames_utl_unpack [OPTIONS] FILE.sof


DESCRIPTION
       This recipe unpack flames-uves packed products.

       Information on relevant parameters can be found with esorex --params flames_utl_unpack esorex --help
       flames_utl_unpack


OPTIONS
       Note that it is possible to create a configuration file containing these options, along with suitable
       default values. Please refer to the details provided by the 'esorex --help' command.
