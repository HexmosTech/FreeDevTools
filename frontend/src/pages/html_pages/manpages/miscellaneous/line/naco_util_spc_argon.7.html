NAME
       naco_util_spc_argon - Generate FITS table with argon lines


SYNOPSIS
       esorex naco_util_spc_argon [OPTIONS] FILE.sof


DESCRIPTION
       The input frame(s) must be tagged SPC_ARGON_ASCII.

       The  file  must  comprise  two columns:the 1st with the line wavelength [Angstrom], the 2nd with the line
       intensity.

       The default input ASCII-file is in the catalogs/ directory of the NACO source-code distribution.


OPTIONS
       Note that it is possible to create a configuration file containing these  options,  along  with  suitable
       default values. Please refer to the details provided by the 'esorex --help' command.
