NAME
       ECHO - Generic output tool


SYNOPSIS
       echo [ 'string']


DESCRIPTION
       Prints the string literal on standard output.


EXAMPLE
            ECHO 'Node 1 initialized successfully';


VERSION INFORMATION
       This command was introduced in Slony-I 1.0
