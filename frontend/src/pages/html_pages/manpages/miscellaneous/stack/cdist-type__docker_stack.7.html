NAME
       cdist-type__docker_stack - Manage Docker stacks


DESCRIPTION
       This type manages service stacks.

       NOTE:
          Since  there  is no easy way to tell whether a stack needs to be updated, docker stack deploy is being
          run every time this type is invoked.  However, it does not mean this type is not idempotent. If Docker
          does not detect changes, the existing stack will not be updated.


OPTIONAL PARAMETERS
       compose-file
              Path to the compose file. If it is '-' (dash), read standard input.

       state  'present' or 'absent', defaults to 'present' where:

              present
                     the stack is deployed

              absent the stack is removed


EXAMPLES
          # Deploys 'foo' stack defined in 'docker-compose.yml' compose file
          __docker_stack foo --compose-file docker-compose.yml
