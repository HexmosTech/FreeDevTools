NAME
       SELFHTML - A German HTML reference and tutorial


SYNOPSIS
       http://localhost/doc/selfhtml/html/index.htm


DESCRIPTION
       This  manual page was written for the Debian GNU/Linux distribution because the original program does not
       have a manual page.

       The German SELFHTML manual describes the HTML standard  and  related  standards  and  languages  such  as
       JavaScript, CSS etc.
