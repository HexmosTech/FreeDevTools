NAME
       cdist-type__docker_secret - Manage Docker secrets


DESCRIPTION
       This type manages Docker secrets.


OPTIONAL PARAMETERS
       source Path to the source file. If it is '-' (dash), read standard input.

       state  'present' or 'absent', defaults to 'present' where:

              present
                     if the secret does not exist, it is created

              absent the secret is removed


CAVEATS
       Since  Docker  secrets  cannot be updated once created, this type takes no action if the specified secret
       already exists.


EXAMPLES
          # Creates "foo" secret from "bar" source file
          __docker_secret foo --source bar
