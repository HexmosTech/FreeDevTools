NAME
       DROP_USER - remove a database role


SYNOPSIS
       DROP USER [ IF EXISTS ] name [, ...]


DESCRIPTION
       DROP USER is simply an alternate spelling of DROP ROLE.


COMPATIBILITY
       The DROP USER statement is a PostgreSQL extension. The SQL standard leaves the definition of users to the
       implementation.


SEE ALSO
       DROP ROLE (DROP_ROLE(7))
