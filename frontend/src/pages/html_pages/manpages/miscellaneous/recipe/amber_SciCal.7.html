NAME
       amber_SciCal - AMBER Science and Calibration Recipe


SYNOPSIS
       esorex amber_SciCal [OPTIONS] FILE.sof


DESCRIPTION
       This recipe calculates Visibilities from a Pixel-To-Visibility-Matrix and Raw Files in 2- and 3-Telescope
       Mode (HL-version)


OPTIONS
       --binning <int>
              Number of Frames to be averaged per Visibility (int; default: 1). The full name of this option for
              the EsoRex configuration file is amber.amber_SciCal.int_binning [default = 1].

       --selectPlusTrf <bool>
              select  good  visibilities  and  try to calculate the transfer function (bool; default: True). The
              full name of this option for the EsoRex  configuration  file  is  amber.amber_SciCal.selectPlusTrf
              [default = True].

       --activate_BEAUTIFY_PISTON <bool>
              TBD  (bool;  default:  True).  The  full  name of this option for the EsoRex configuration file is
              amber.amber_SciCal.activate_BEAUTIFY_PISTON [default = True].

       --activate_MIN_PHOTOMETRY <bool>
              TBD (bool; default: True). The full name of this option  for  the  EsoRex  configuration  file  is
              amber.amber_SciCal.activate_MIN_PHOTOMETRY [default = True].

       --activate_DROP <bool>
              TBD  (bool;  default:  False).  The  full name of this option for the EsoRex configuration file is
              amber.amber_SciCal.activate_DROP [default = False].

       --value_BEAUTIFY_PISTON <float>
              TBD (float; default: 1.0). The full name of this option  for  the  EsoRex  configuration  file  is
              amber.amber_SciCal.value_BEAUTIFY_PISTON [default = 1.0].

       --value_MIN_PHOTOMETRY <float>
              TBD  (float;  default:  0.0).  The  full  name of this option for the EsoRex configuration file is
              amber.amber_SciCal.value_MIN_PHOTOMETRY [default = 0.0].

       --value_DROP <float>
              TBD (float; default: 0.0). The full name of this option  for  the  EsoRex  configuration  file  is
              amber.amber_SciCal.value_DROP [default = 0.0].

       Note  that  it  is  possible to create a configuration file containing these options, along with suitable
       default values. Please refer to the details provided by the 'esorex --help' command.
