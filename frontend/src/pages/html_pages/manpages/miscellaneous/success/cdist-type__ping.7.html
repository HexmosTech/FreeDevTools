NAME
       cdist-type__ping - Try to connect to host and return 'pong' on success


DESCRIPTION
       A  simple  type which tries to connect to a remote host and runs a simple command to ensure everything is
       working.


REQUIRED PARAMETERS
       None.


OPTIONAL PARAMETERS
       None.


EXAMPLES
          __ping
