NAME
       DATE - Display current date


SYNOPSIS
       date [ (format)]


DESCRIPTION
       Prints the current date. Accepts an optional strftime()-conformant format string.


EXAMPLE
            DATE;
            DATE(format='%Y-%m-%d %H:%M:%S %Z');


VERSION INFORMATION
       This command was introduced in Slony-I 2.1
