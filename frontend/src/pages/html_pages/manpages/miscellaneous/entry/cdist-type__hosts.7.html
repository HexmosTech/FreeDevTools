NAME
       cdist-type__hosts - manage entries in /etc/hosts


DESCRIPTION
       Add or remove entries from /etc/hosts file.


OPTIONAL PARAMETERS
       state  If  state  is  present, make object_id resolve to ip. If state is absent, object_id will no longer
              resolve via /etc/hosts, if it was previously configured with this type.  Manually inserted entries
              are unaffected.

       ip     IP address, to which hostname (=object_id) must resolve. If state is present,  this  parameter  is
              mandatory, if state is absent, this parameter is silently ignored.

       alias  An alias for the hostname.  This parameter can be specified multiple times (once per alias).


EXAMPLES
          # Now `funny' resolves to 192.168.1.76,
          __hosts funny --ip 192.168.1.76
          # and `happy' no longer resolve via /etc/hosts if it was
          # previously configured via __hosts.
          __hosts happy --state absent

          __hosts srv1.example.com --ip 192.168.0.42 --alias srv1
