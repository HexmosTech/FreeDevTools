NAME
       votca-tools - The basic tools library of the VOTCA package


DESCRIPTION
       votca-tools provides the basic functions used by all parts of the VOTCA package.

       Please visit the program site at __http://www.votca.org__.


SYNOPSIS
       If  VOTCA  was  installed  in  a  non-default  location one can use the VOTCARC file to set the important
       environment variables (PATH, MANPATH, CSGSHARE).

       In bash shell: source VOTCARC.bash

       In tcsh shell: source VOTCARC.csh

       In sh shell: . VOTCARC.bash
