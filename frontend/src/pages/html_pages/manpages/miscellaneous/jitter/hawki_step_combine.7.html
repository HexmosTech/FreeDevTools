NAME
       hawki_step_combine - (OBSOLETE) Jitter image combination recipe


SYNOPSIS
       esorex hawki_step_combine [OPTIONS] FILE.sof


DESCRIPTION
       (OBSOLETE) hawki_step_combine -- hawki combine jitter images.

       The  files  listed  in  the  Set Of Frames (sof-file) must be tagged: science-file.fits DIST_CORRECTED or
       science-file.fits BKG_SUBTRACTED or bpm-file.fits BPM  (optional)  bkg_bpm-file.fits  BKG_BPM  (optional)
       offsets-file.fits  OFFSETS_REFINED  (optional)  The  recipe creates as an output: hawki_step_combine.fits
       (COMBINED): The recipe does the following steps: -Allocate an image with the proper combined size
          (depends  on  parameters  --comb_meth  and  --borders)  -Retrieve  the   offsets   either   from   the
       offsets-file.fits or from the header -For each combined pixel, the contribution of each individual frame
          is added using a resampling kernel. If any of the pixels involved in
          the resampling is a bad pixel (defined in bpm-file.fits), it is not
          taken into account.

          With the remaining pixels a minmax rejection is performed Return code: esorex exits with an error code
       of 0 if the recipe completes successfully or 1 otherwise


OPTIONS
       --offset_max <int>
              Maximum  offset  allowed  (int;  default:  1500).  The  full  name  of  this option for the EsoRex
              configuration file is hawki.hawki_step_combine.offset_max [default = 1500].

       --comb_meth <str>
              Final size of combination (union / inter / first) (str; default: ´union´). The full name  of  this
              option for the EsoRex configuration file is hawki.hawki_step_combine.comb_meth [default = union].

       --rej <str>
              Low and high number of rejected values (str; default: ´1,1´). The full name of this option for the
              EsoRex configuration file is hawki.hawki_step_combine.rej [default = 1,1].

       --borders <int>
              Border pixels trimmed (int; default: 4). The full name of this option for the EsoRex configuration
              file is hawki.hawki_step_combine.borders [default = 4].

       --resamp_kernel <str>
              Resampling  kernel  (default/tanh/sinc/sinc2/lanczos/hamming/hann)  (str; default: ´default´). The
              full    name    of    this     option     for     the     EsoRex     configuration     file     is
              hawki.hawki_step_combine.resamp_kernel [default = default].

       Note  that  it  is  possible to create a configuration file containing these options, along with suitable
       default values. Please refer to the details provided by the 'esorex --help' command.
