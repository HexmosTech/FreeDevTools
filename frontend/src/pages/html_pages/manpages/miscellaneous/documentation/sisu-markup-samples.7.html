NAME
       SiSU - Structured information, Serialized Units - a document publishing system


DESCRIPTION
       SiSU  is  a  lightweight  markup based document structuring, publishing and search framework for document
       collections.

       The sisu-markup-samples package contains SiSU examples that can be generated in multiple output  document
       formats  and  made  searchable  using  the package sisu. For content run sisu against the markup from the
       content  directory,  e.g.:  sisu  --html  -v  the_wealth_of_networks.yochai_benkler.sst  sisu  --epub  -v
       viral_spiral.david_bollier.sst sisu --pdf -v the_wealth_of_networks.yochai_benkler.sst sisu --html --epub
       --pdf  -v  democratizing_innovation.eric_von_hippel.sst  Markup  samples  located in /usr/share/doc/sisu-
       markup-samples include the following texts, see the COPYRIGHT file for copyright and license details:
