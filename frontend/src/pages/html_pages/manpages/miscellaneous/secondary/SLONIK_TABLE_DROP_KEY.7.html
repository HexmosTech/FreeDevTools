NAME
       TABLE DROP KEY - Removes a primary key added by TABLE ADD KEY


VERSION INFORMATION
       This command was introduced in Slony-I 1.0

       In Slony-I version 2.0, this command is removed as obsolete because triggers are no longer ‘messed around
       with’ in the system catalogue.
