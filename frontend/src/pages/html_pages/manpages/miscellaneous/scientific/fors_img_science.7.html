NAME
       fors_img_science - Reduce imaging scientific exposure


SYNOPSIS
       esorex fors_img_science [OPTIONS] FILE.sof


DESCRIPTION
       Input files:
         DO category:               Type:       Explanation:                        Number:
         SCIENCE_IMG                 Raw         Science image                            1
         MASTER_BIAS                 FITS image  Master bias                              1
         MASTER_SKY_FLAT_IMG         FITS image  Master sky flat field                    1
         PHOT_COEFF_TABLE            FITS table  Observed extinction coefficients       0+
         EXTINCTION_PER_NIGHT        FITS table  Extinction per night                     0+
         STATIC_PHOT_COEFF_TABLE     FITS table  Static filters photometry coefficients   0+
         DETECTOR_ILLUMINATED_REGION FITS table  Table with detector illuminated regions  0+

       Output files:
         DO category:               Data type:  Explanation:
         SCIENCE_REDUCED_IMG        FITS image  Reduced science image
         PHOT_BACKGROUND_SCI_IMG    FITS image  Reduced science image background
         SOURCES_SCI_IMG            FITS image  Unfiltered SExtractor output
         OBJECT_TABLE_SCI_IMG       FITS table  Extracted sources properties
         SCIENCE_REDUCED_IMG_IDP    FITS image  Reduced science image in IDP-compliant format
         SCIENCE_REDUCED_WEIGHT_IDP FITS image  Ancillary file with the weights for IDP the image
         SCIENCE_REDUCED_ERR_IDP    FITS image  Ancillary file with the errors for IDP the image


OPTIONS
       --magsyserr <float>
              Systematic  error in magnitude (float; default: 0.01). The full name of this option for the EsoRex
              configuration file is fors.fors_img_science.magsyserr [default = 0.01].

       --sex_exe <str>
              SExtractor executable (str; default: ´/usr/bin/source-extractor´). The full name  of  this  option
              for    the    EsoRex    configuration    file    is   fors.fors_img_science.sex_exe   [default   =
              /usr/bin/source-extractor].

       --sex_config <str>
              SExtractor        configuration        file        (str;         default:         ´/usr/share/cpl-
              plugins/fors-5.5.7/config/fors.sex´).  The  full  name of this option for the EsoRex configuration
              file            is             fors.fors_img_science.sex_config             [default             =
              /usr/share/cpl-plugins/fors-5.5.7/config/fors.sex].

       --sex_mag <str>
              SExtractor  magnitude  (str;  default:  ´MAG_APER´).  The  full name of this option for the EsoRex
              configuration file is fors.fors_img_science.sex_mag [default = MAG_APER].

       --sex_magerr <str>
              SExtractor magnitude error (str; default: ´MAGERR_APER´). The full name of  this  option  for  the
              EsoRex configuration file is fors.fors_img_science.sex_magerr [default = MAGERR_APER].

       --sex_radius <int>
              Background  error  map median filter radius (unbinned pixels) (int; default: 64). The full name of
              this option for the EsoRex configuration file is fors.fors_img_science.sex_radius [default = 64].

       --idp_generate <bool>
              If TRUE compliant IDP science products are generated (bool; default: False). The full name of this
              option for the EsoRex configuration file is fors.fors_img_science.idp_generate [default = False].

       --idp_weights_threshold <float>
              Every pixel in the MASTER_SKY_FLAT_IMG below the threshold will be put to 0 in the  weight-map,  1
              otherwise. Ignored if IDP generation is not requested (float; default: 0.5). The full name of this
              option for the EsoRex configuration file is fors.fors_img_science.idp_weights_threshold [default =
              0.5].

       Note  that  it  is  possible to create a configuration file containing these options, along with suitable
       default values. Please refer to the details provided by the 'esorex --help' command.
