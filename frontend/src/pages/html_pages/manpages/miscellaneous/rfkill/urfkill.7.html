NAME
       urfkill - RFKill Switches Management


DESCRIPTION
       urfkill provides an interface to enumerate rfkill switches on the system and control system-wide rfkill
       switches management. Any application can access the org.freedesktop.URfkill service on the system message
       bus. Some operations (such as setting soft block on the device) is restricted using PolicyKit.
