NAME
       cdist-type__timezone - Allows one to configure the desired localtime timezone.


DESCRIPTION
       This  type  creates  a  symlink  (/etc/localtime)  to the selected timezone (which should be available in
       /usr/share/zoneinfo).


REQUIRED PARAMETERS
       tz     The name of timezone to set.


OPTIONAL PARAMETERS
       None.


EXAMPLES
          # Set up Europe/Andorra as our timezone.
          __timezone --tz Europe/Andorra

          # Set up US/Central as our timezone.
          __timezone --tz US/Central
