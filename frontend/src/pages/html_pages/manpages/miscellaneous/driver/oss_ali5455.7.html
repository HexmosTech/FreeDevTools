NAME
       oss_ali5455 - ALIM5455 audio driver.


DESCRIPTION
       Open  Sound System driver for onboard audio on ALI M5455 motherboards. This driver is very similar to the
       Intel ICH driver.

       ALI 5455 device characteristics:

              •  8/16 bit playback/record

              •  mono/stereo/4/5.1 channel playback/recording

              •  8KHz to 48Khz sample rate.


OPTIONS
       None


FILES
       /etc/oss4/conf/oss_ali5455.conf
              Device configuration file.
