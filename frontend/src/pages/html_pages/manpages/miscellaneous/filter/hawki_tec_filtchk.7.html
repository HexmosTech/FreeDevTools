NAME
       hawki_tec_filtchk - (OBSOLETE) Filter checking recipe


SYNOPSIS
       esorex hawki_tec_filtchk [OPTIONS] FILE.sof


DESCRIPTION
       (OBSOLETE) hawki_tec_filtchk -- Check pairs of flats taken with different filters.

       The files listed in the Set Of Frames (sof-file) must be tagged: raw-file.fits TEC_FLAT


OPTIONS
       Note  that  it  is  possible to create a configuration file containing these options, along with suitable
       default values. Please refer to the details provided by the 'esorex --help' command.
