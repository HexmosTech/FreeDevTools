NAME
       muse_scipost_raman - Remove of 4GLSF Raman emission.


SYNOPSIS
       esorex muse_scipost_raman [OPTIONS] FILE.sof


DESCRIPTION
       This  recipe  removes  the Raman scattered light of the 4GLSF system from the exposure. It must be before
       the normal sky subtraction.


OPTIONS
       --lambdamin <float>
              Cut off the data below this wavelength after loading the pixel table(s). (float; default: 4000.0).
              The    full    name    of    this    option    for    the    EsoRex    configuration    file    is
              muse.muse_scipost_raman.lambdamin [default = 4000.0].

       --lambdamax <float>
              Cut  off  the  data  above  this  wavelength  after  loading  the pixel table(s). (float; default:
              10000.0).  The   full   name   of   this   option   for   the   EsoRex   configuration   file   is
              muse.muse_scipost_raman.lambdamax [default = 10000.0].

       --lambdaref <float>
              Reference  wavelength used for correction of differential atmospheric refraction. The R-band (peak
              wavelength ~7000 Angstrom) that is usually used for guiding, is close to the central wavelength of
              MUSE, so a value of 7000.0 Angstrom should be used if nothing else is known.  A  value  less  than
              zero  switches  DAR correction off. (float; default: 7000.0). The full name of this option for the
              EsoRex configuration file is muse.muse_scipost_raman.lambdaref [default = 7000.0].

       --width <float>
              Wavelength range around Raman lines [Angstrom]. (float; default: 20.0).  The  full  name  of  this
              option for the EsoRex configuration file is muse.muse_scipost_raman.width [default = 20.0].

       --crsigma <float>
              Sigma  level  clipping  for  cube-based CR rejection (using "median", see muse_scipost). It can be
              switched off, by passing zero or a negative value. (float; default: 15.0). The full name  of  this
              option for the EsoRex configuration file is muse.muse_scipost_raman.crsigma [default = 15.0].

       --fraction <float>
              Fraction  of the image (without the ignored part) to be considered as sky. If an input sky mask is
              provided, the fraction is applied to the regions within the mask. If the whole sky mask should  be
              used, set this parameter to 1. (float; default: 0.75). The full name of this option for the EsoRex
              configuration file is muse.muse_scipost_raman.fraction [default = 0.75].

       --ignore <float>
              Lowest  fraction  of  the  image  to be ignored. If an input sky mask is provided, the fraction is
              applied to the regions within the mask. If the whole sky mask should be used, set  this  parameter
              to  0.  (float;  default: 0.05). The full name of this option for the EsoRex configuration file is
              muse.muse_scipost_raman.ignore [default = 0.05].

       Note that it is possible to create a configuration file containing these  options,  along  with  suitable
       default values. Please refer to the details provided by the 'esorex --help' command.
