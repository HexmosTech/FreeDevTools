NAME
       sgepasswd - Grid Engine password file format


DESCRIPTION
       sgepasswd contains a list of user names and their corresponding encrypted Microsoft Windows passwords. If
       available, the password file will be used by sge_shepherd(8) and sge_execd(8).  To change the contents of
       this file please use the sgepasswd(1) command. It is not advised to change the file manually.
