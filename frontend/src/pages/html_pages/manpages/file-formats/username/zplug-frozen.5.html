NAME
       zplug-frozen - Do not update unless explicitly specified


SYNOPSIS
       zplug "username/reponame", frozen:yes

       The word ´package´ refers to the string in the format: "username/reponame".


DESCRIPTION
       By using frozen tag, you cant skip update of the repo when executing zplug command. In particular, this
       tag is particularly useful if you can describe the package with from:gh-r tag since it’s so heavy.

       Table 1. zplug frozen tag
       ┌─────────────────┬───────────────┐
       │ Possive Values  │ Default value │
       ├─────────────────┼───────────────┤
       │                 │               │
       │ (truthy, falsy) │ no            │
       └─────────────────┴───────────────┘

       Values truthy and falsy mean any of "true, yes, on, 1" and "false, no, off, 0", respectively.

       Besides, by using zstyle command, you can change default value:

           $ zstyle ":zplug:tag" frozen true


EXAMPLES
       Skip the update:

           $ zplug "stedolan/jq" \
               as:command, \
               from:gh-r, \
               use:"*darwin*", \
               frozen:1, \
