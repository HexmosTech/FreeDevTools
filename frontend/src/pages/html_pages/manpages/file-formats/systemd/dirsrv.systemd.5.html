NAME
       /etc/sysconfig/dirsrv.systemd - Systemd EnvironmentFile for Directory Server.


SYNOPSIS
       /etc/sysconfig/dirsrv.systemd


DESCRIPTION
       dirsrv.systemd

       This  controls  the  resources to the direct child of systemd, in this case ns-slapd. Because we are type
       notify we receive these limits correctly.

       For more information see man systemd.exec and man systemd.resource-control
