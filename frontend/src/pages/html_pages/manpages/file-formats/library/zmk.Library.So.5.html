NAME
       Library.So — template for compiling C/C++/ObjC shared ELF libraries


SYNOPSIS
       include z.mk

       # libName is any valid identifier.
       libName.so.Sources = hello.c
       $(eval $(call ZMK.Expand,Library.So,libName.so))


DESCRIPTION
       This manual page is a stub.

       When configure script is invoked with --disable-shared then this template behaves as if it was empty.
