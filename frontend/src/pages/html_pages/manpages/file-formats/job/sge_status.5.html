NAME
       sge_status - Grid Engine job status values


DESCRIPTION
   Job state
       The  following  table  lists  the  job states shown by qstat(1) and returned by drmaa_jobcontrol(3).  The
       DRMAA state corresponds to the DRMAA_PS_state value that may be returned by drmaa_job_ps(3).

       ┌────────────────────────┬────────────────────────┬───────────────────────┬──────────────────────┐
       │        Category        │         State          │          SGE          │     DRMAA state      │
       ├────────────────────────┼────────────────────────┼───────────────────────┼──────────────────────┤
       │ Pending                │ pending                │ qw, Rq                │ QUEUED_ACTIVE        │
       │                        ├────────────────────────┼───────────────────────┼──────────────────────┤
       │                        │ pending, user hold     │ hqw                   │ USER_ON_HOLD         │
       │                        ├────────────────────────┼───────────────────────┼──────────────────────┤
       │                        │ pending, system hold   │ hqw                   │ SYSTEM_ON_HOLD       │
       │                        ├────────────────────────┼───────────────────────┼──────────────────────┤
       │                        │ pending, user and      │ hqw                   │ USER_SYSTEM_ON_HOLD  │
       │                        │ system hold            │                       │                      │
       │                        ├────────────────────────┼───────────────────────┼──────────────────────┤
       │                        │ pending, user hold,    │ hRwq                  │ USER_ON_HOLD         │

       │                        │ re-queue               │                       │                      │
       │                        ├────────────────────────┼───────────────────────┼──────────────────────┤
       │                        │ pending, user and      │ hRwq                  │ USER_SYSTEM_ON_HOLD  │
       │                        │ system hold, re-queue  │                       │                      │
       ├────────────────────────┼────────────────────────┼───────────────────────┼──────────────────────┤
       │ Running / transferring │ running, transferring  │ r, hr, t              │ RUNNING              │
       │                        ├────────────────────────┼───────────────────────┼──────────────────────┤
       │                        │ running, re-run /      │ Rr, Rt                │ RUNNING              │
       │                        │ transferring           │                       │                      │
       ├────────────────────────┼────────────────────────┼───────────────────────┼──────────────────────┤
       │ Suspended              │ job suspended          │ s, ts                 │ USER_SUSPENDED       │

       │                        │ alarm                  │                       │                      │

       ├────────────────────────┼────────────────────────┼───────────────────────┼──────────────────────┤
       │ Error                  │ all pending states     │ Eqw, Ehqw, EhRqw      │ FAILED               │
       │                        │ with error             │                       │                      │
       ├────────────────────────┼────────────────────────┼───────────────────────┼──────────────────────┤
       │ Deleting               │ all running and        │ dr, dt, dRr, dRt, ds, │ same as equivalent   │

       ├────────────────────────┼────────────────────────┼───────────────────────┼──────────────────────┤
       │ Unkown                 │ status cannot be       │                       │ UNDETERMINED         │
       │                        │ determined             │                       │                      │
