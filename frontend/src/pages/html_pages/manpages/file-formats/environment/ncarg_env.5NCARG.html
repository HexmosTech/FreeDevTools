NAME
       NCAR Graphics environment variables


SYNOPSIS
       What follows is an alphabetical list of all the NCAR Graphics environment variables and what they mean.

       NCARG_BIN,  if  set, indicates where the NCAR Graphics binaries are installed.  This environment variable
       takes precedence over NCARG_ROOT and doesn't need to be set unless  you  want  to  override  the  default
       ($NCARG_ROOT/bin).

       NCARG_CONFIG,  if  set,  indicates  where  the NCAR Graphics configuration files are installed (these are
       needed by site installers only).  This environment variable takes precedence over NCARG_ROOT and  doesn't
       need to be set unless you want to override the default ($NCARG_ROOT/lib/ncarg/config).

       NCARG_DATA, if set, indicates where the NCAR Graphics example data files are installed.  This environment
       variable  takes  precedence  over  NCARG_ROOT  and doesn't need to be set unless you want to override the
       default ($NCARG_ROOT/lib/ncarg/data).

       NCARG_DATABASE, if set, indicates where the NCAR Graphics database files are installed.  This environment
       variable takes precedence over NCARG_ROOT and doesn't need to be set unless  you  want  to  override  the
       default ($NCARG_ROOT/lib/ncarg/database).

       NCARG_DOC,  if  set,  indicates  where  the  NCAR  Graphics  PostScript documentation is installed.  This
       environment variable takes precedence over NCARG_ROOT and doesn't need to  be  set  unless  you  want  to
       override the default ($NCARG_ROOT/lib/ncarg/doc).

       NCARG_EXAMPLES,  if  set,  indicates  where  some  of  the  NCAR  Graphics  examples are installed.  This
       environment variable takes precedence over NCARG_ROOT and doesn't need to  be  set  unless  you  want  to
       override the default ($NCARG_ROOT/lib/ncarg/examples).

       NCARG_FONTCAP, if set, indicates the default NCAR Graphics fontcap to be used.

       NCARG_FONTCAPS,  if  set,  indicates  where  the  NCAR Graphics fontcaps are installed.  This environment
       variable takes precedence over NCARG_ROOT and doesn't need to be set unless  you  want  to  override  the
       default ($NCARG_ROOT/lib/ncarg/fontcaps).

       NCARG_GKS_BUFSIZE provides you with control over how CGM output is buffered.

       NCARG_GKS_DEBUG, if set, will result in debugging information about the GKS I/O subsystem being output to
       stdout.

       NCARG_GKS_GENCGM  allows you to request that the NCAR GKS package generate binary encoded CGM rather than
       NCGM (the NCAR private encoding of binary encoded CGM).

       NCARG_GKS_OUTPUT allows you to specify what file or process CGM output is directed to.

       NCARG_GKS_PSOUTPUT allows you to specify what file any PostScript output from NCAR  GKS  is  directed  to
       (stdout is allowed).

       NCARG_GKS_PDFOUTPUT  allows  you to specify what file any PDF output from NCAR GKS is directed to (stdout
       is allowed).

       NCARG_GRAPHCAP, if set, indicates the default NCAR Graphics graphcap to be used.

       NCARG_GRAPHCAPS, if set, indicates where the NCAR Graphics graphcaps  are  installed.   This  environment
       variable  takes  precedence  over  NCARG_ROOT  and doesn't need to be set unless you want to override the
       default ($NCARG_ROOT/lib/ncarg/graphcaps).

       NCARG_HLUEX, if set, indicates where the NCAR Graphics HLU  examples  are  installed.   This  environment
       variable  takes  precedence  over  NCARG_ROOT  and doesn't need to be set unless you want to override the
       default ($NCARG_ROOT/lib/ncarg/hluex).

       NCARG_INCLUDE, if set, indicates where the NCAR Graphics include files are installed.   This  environment
       variable  takes  precedence  over  NCARG_ROOT  and doesn't need to be set unless you want to override the
       default ($NCARG_ROOT/include).

       NCARG_LIB, if set, indicates where the NCAR Graphics libraries are installed.  This environment  variable
       takes  precedence  over  NCARG_ROOT  and  doesn't  need to be set unless you want to override the default
       ($NCARG_ROOT/lib).

       NCARG_MAN, if set, indicates where the NCAR Graphics man pages are installed.  This environment  variable
       takes  precedence  over  NCARG_ROOT  and  doesn't  need to be set unless you want to override the default
       ($NCARG_ROOT/man).

       NCARG_NCARG, if set, indicates where the NCAR Graphics examples,  databases,  resource  files,  etc.  are
       installed.   This environment variable takes precedence over NCARG_ROOT and doesn't need to be set unless
       you want to override the default ($NCARG_ROOT/lib/ncarg).

       NCARG_NCLEX, if set, indicates where the NCAR Graphics NCL  examples  are  installed.   This  environment
       variable  takes  precedence  over  NCARG_ROOT  and doesn't need to be set unless you want to override the
       default ($NCARG_ROOT/lib/ncarg/nclex).

       NCARG_NGURL, if set, indicates where the file that contains the URL to the NCAR Graphics documentation is
       installed.  This environment variable takes precedence over NCARG_ROOT and doesn't need to be set  unless
       you want to override the default ($NCARG_ROOT/lib/ncarg/ngwww/ngurl).

       NCARG_NGWWW, if set, indicates where the NCAR Graphics HTML documentation is installed.  This environment
       variable  takes  precedence  over  NCARG_ROOT  and doesn't need to be set unless you want to override the
       default ($NCARG_ROOT/lib/ncarg/ngwww).

       NCARG_RESFILES, if set, indicates where the NCAR Graphics resource files are installed.  This environment
       variable takes precedence over NCARG_ROOT and doesn't need to be set unless  you  want  to  override  the
       default ($NCARG_ROOT/lib/ncarg/resfiles).

       NCARG_ROOT,  indicates  the  parent directory where all of the NCAR Graphics binaries, libraries, include
       files, man pages, examples, etc. are installed.  This environment variable must be  set  by  anybody  who
       wants to run NCAR Graphics (or you can optionally set NCARG_LIB, NCARG_BIN, and NCARG_INCLUDE).

       NCARG_SYSRESFILE,  if  set,  indicates  where  the NCAR Graphics system resource file is installed.  This
       environment variable takes precedence over NCARG_ROOT and doesn't need to  be  set  unless  you  want  to
       override the default ($NCARG_ROOT/lib/ncarg/sysappres).

       NCARG_TESTS,  if  set,  indicates  where the NCAR Graphics test examples are installed.  This environment
       variable takes precedence over NCARG_ROOT and doesn't need to be set unless  you  want  to  override  the
       default ($NCARG_ROOT/lib/ncarg/tests).

       NCARG_TMP, if set, indicates where NCAR Graphics temporary files should be written.

       NCARG_TUTORIAL,  if  set,  indicates  where  the  NCAR  Graphics  tutorial  examples are installed.  This
       environment variable takes precedence over NCARG_ROOT and doesn't need to  be  set  unless  you  want  to
       override the default ($NCARG_ROOT/lib/ncarg/tutorial).

       NCARG_USRRESFILE,  if  set,  indicates  where  the  NCAR  Graphics user resource file is installed.  This
       environment variable doesn't need to be set unless you want to override the default (~/.hluresfile).

       NCARG_XAPP, if set, indicates where the NCAR Graphics X application default files  are  installed.   This
       environment  variable  takes  precedence  over  NCARG_ROOT  and doesn't need to be set unless you want to
       override the default ($NCARG_ROOT/lib/ncarg/xapp).
