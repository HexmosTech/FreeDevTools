NAME
       update-initramfs.conf - configuration file for update-initramfs


DESCRIPTION
       The configuration file allows one to disable the update action from update-initramfs.


GENERAL VARIABLES
        update_initramfs
              Default is yes for running the latest initramfs-tools hooks in the newest Linux image.  Setting it
              to  all  updates  any known initramfs.  It is possible to set it to no for remote servers or boxes
              where conservative manners needs to be applied. This disables the update_initramfs -u call.

        backup_initramfs
              If set update_initramfs keeps an .bak  file  of  the  previous  initramfs.  If  unset  the  backup
              initramfs will not be kept.


FILES
       /etc/initramfs-tools/update-initramfs.conf
