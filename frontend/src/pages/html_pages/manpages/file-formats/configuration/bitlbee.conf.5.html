NAME
       bitlbee.conf - configuration file for bitlbee(8)


DESCRIPTION
       This  file  contains system-wide settings for the bitlbee(8) program. For more information about the file
       syntax, please read the example configuration file  which  comes  with  the  program.  The  default  file
       contains lots of comments which explain all the available options.
