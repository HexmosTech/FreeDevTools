NAME
       /proc/pid/ns/ - namespaces


DESCRIPTION
       /proc/pid/ns/ (since Linux 3.0)
              This  is a subdirectory containing one entry for each namespace that supports being manipulated by
              setns(2).  For more information, see namespaces(7).
