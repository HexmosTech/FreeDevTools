NAME
       pn53x-diagnose - PN53x diagnose tool


SYNOPSIS
       pn53x-diagnose


DESCRIPTION
       pn53x-diagnose is a utility to diagnose PN531, PN532 and PN533 chips.  It runs communication, RAM and ROM
       tests.
