NAME
       lcm_tester - Program for testing the functionality of the LC Communications library


SYNOPSIS
       lcm_tester


DESCRIPTION
       lcm_tester  is  a  program  for testing the functionality of the Lightweight Communications (LC) library.
       There are no command line options.
