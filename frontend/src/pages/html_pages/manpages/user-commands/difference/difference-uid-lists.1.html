NAME
       difference-uid-lists - compute symmetric difference of two numerical lists


SYNOPSIS
       difference-uid-lists filename1 filename2


DESCRIPTION
       difference-uid-lists computes the symmetric difference of the specified number lists (expected to contain
       one number per line) and sorts the result numerically.


SEE ALSO
       combine-uid-lists(1), efetch(1), efilter(1), esearch(1), exclude-uid-lists(1), intersect-uid-lists(1).
