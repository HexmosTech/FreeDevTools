NAME
       botch-packages-difference - calculate the asymmetric set difference


SYNOPSIS
       botch-packages-difference [options] inPackages1 inPackages2 outPackages


DESCRIPTION
       calculate  the  asymmetric  difference  of  two packages files, removing from the first packages file all
       packages that are in the second file


OPTIONS
       -h, --help
           Show help.

       --verbose
           Be verbose.
