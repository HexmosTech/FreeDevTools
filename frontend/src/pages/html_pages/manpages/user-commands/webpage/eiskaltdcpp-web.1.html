NAME
       eiskaltdcpp-web - launcher of simple Web UI for eiskaltdcpp-daemon


SYNOPSIS
       eiskaltdcpp-web <Key>


DESCRIPTION
       eiskaltdcpp-web  is  a  script  which  copies all necessary files to "~/.local/share/eiskaltdc++/web" and
       opens Web UI in default web browser.

       EiskaltDC++ is a cross-platform program that uses the Direct Connect and ADC protocols. It is  compatible
       with  DC++,  AirDC++, FlylinkDC++ and other DC clients. EiskaltDC++ also interoperates with all common DC
       hub software.


COMMAND LINE OPTIONS
       A summary of options is included below.

       -h,  --help
              Display help and exit


FILES
       ~/.local/share/eiskaltdc++/web/config.js
              Configuration file for current user
