NAME
       podofocrop - crop all pages


SYNOPSIS
       podofocrop input.pdf output.pdf


DESCRIPTION
       podofocrop  is  one  of  the  command  line  tools  from  the  PoDoFo library that provide several useful
       operations to work with PDF files. It can crop all pages in a PDF file.  It requires ghostscript to be in
       your PATH.
