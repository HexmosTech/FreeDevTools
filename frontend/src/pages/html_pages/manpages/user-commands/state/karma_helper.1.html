NAME
       karma_helper - adjusts the state of the Rio Karma


SYNOPSIS
       karma_helper [options] files...


DESCRIPTION
       This manual page documents briefly the karma_helper command.

       karma_helper is a program that sends USB commands to the Rio Karma to modify its state.


OPTIONS
       A summary of options is included below.

       -b     Bind driver to device.

       -e     Enter storage mode.

       -h     Show summary of options.

       -i     Retrieve device information string.

       -l     Leave storage mode.

       -r     Reset device (leave "Finished" screen).

       -t     Update time.

       -u     Unbind driver from device.


NOTES
       karma_helper  requires  root  priveliges  to directly access the USB layer and send most of the available
       commands.
