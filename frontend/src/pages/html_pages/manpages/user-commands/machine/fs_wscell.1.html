NAME
       fs_wscell - Returns the name of the cell to which a machine belongs


SYNOPSIS
       fs wscell [-help]

       fs ws [-h]


DESCRIPTION
       The fs wscell command returns the name of the local machine's home cell.


OPTIONS
       -help
           Prints the online help for this command. All other valid options are ignored.


OUTPUT
       The  output displays the client's current home cell, thus the contents of the local /etc/openafs/ThisCell
       file at the time when the client was started, in the format

          This workstation belongs to cell '<cellname>'


EXAMPLES
       The following example results when the fs wscell is issued on a machine in the Example Organization cell:

          % fs wscell
          This workstation belongs to cell 'example.org'


PRIVILEGE REQUIRED
       None

       fs_whereis(1), fs_whichcell(1)
