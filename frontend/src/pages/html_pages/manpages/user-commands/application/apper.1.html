NAME
       apper - An application and package manager using PackageKit


SYNOPSIS

       apper [--updates] [--settings] [--backend-details] [--type mime-type] [--name name] [--file file]
             [--resource font] [--catalog file]


DESCRIPTION
       Apper is an interface for PackageKit

       Apper is able to manage packages, applications and updates in a cross-distribution way thanks to the
       PackageKit project.

       It is providing LinuxÂ® users a single application that they can learn and use in whatever distribution
       they are.


OPTIONS
       --updates
           Show available updates

       --settings
           Show Apper configuration page.

       --backend-details
           Display information about the backend PackageKit uses at time.

       --type mime-type
           Call the mime-type installer.

       --name name
           Install package with the name name.

       --file file
           Install package providing file file.

       --resource font
           Install a font resource.

       --catalog file
           Install the package catalog in file.
