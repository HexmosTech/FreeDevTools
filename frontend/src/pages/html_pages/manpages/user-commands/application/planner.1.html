NAME
       planner - Project Management application for GNOME


SYNOPSIS
       planner [options] files...


DESCRIPTION
       planner  is  a  Project  Management  application  that  supports  Gantt  charts,  resource allocation and
       integration with other GNOME applications.


OPTIONS
       planner follows the standard GNU command line syntax, with long options starting with two dashes (`-').

       -?, --help
              Show summary of options.


SEE ALSO
       planner offers a complete User Guide via GNOME's Help Browser.
