NAME
       gcloseall - This tool terminates all applications


SYNOPSIS
       gcloseall


DESCRIPTION
       gcloseall  terminates  all applications. It does this by not asking any questions. It just terminates all
       applications. You have been warned!


OPTIONS
BUGS
       None known
