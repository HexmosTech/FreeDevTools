NAME
       swami  -  (Sampled  Waveforms  And  Musical  Instruments) is an application for editing and managing MIDI
       instruments, such as SoundFont files.  An programming API is also provided  for  integration  with  other
       applications.

       More information can be found on the Project Swami website http://swami.sourceforge.net.
