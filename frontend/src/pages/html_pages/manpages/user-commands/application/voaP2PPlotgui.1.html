NAME
       voaP2PPlotgui - Graphical frontend to the voaP2PPlot application


SYNOPSIS
       voaP2PPlotgui file


DESCRIPTION
       voaP2PPlotgui  is  a  graphical  user interface to the voaP2PPlot application. It plots the file contents
       according to the parameters set on the user interface.
