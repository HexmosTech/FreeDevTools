NAME
       osptest —  OSP test application


SYNOPSIS
       osptest


DESCRIPTION
       This manual page documents briefly the osptest command.

       This  manual  page  was  written for the Debian distribution because the original program does not have a
       manual page.

       osptest is a program that is use to test the OSP packages installed on Debian.


OPTIONS
       None
