NAME
       asc2xml - convert &#nnn; entities to UTF-8


SYNOPSIS
       asc2xml


DESCRIPTION
       Reads  an  ASCII  or  ISO-8859-1  (Latin-1) text from standard input and writes encoded UTF-8 to standard
       output.


SEE ALSO
       xml2asc(1), UTF-8 (RFC 2279)
