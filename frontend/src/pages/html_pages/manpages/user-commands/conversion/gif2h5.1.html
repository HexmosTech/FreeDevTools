NAME
       gif2h5 - Converts a GIF file to an HDF5 file.


SYNOPSIS
       gif2h5 gif_file h5_file


DESCRIPTION
       gif2h5 accepts as input the GIF file gif_file and produces the HDF5 file h5_file as output.
