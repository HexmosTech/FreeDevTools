NAME
       pxs2fa - convert seqfiles from nexus, phylip, fastq to fasta


SYNOPSIS
       pxs2fa [OPTIONS]...


DESCRIPTION
       Convert seqfiles from nexus, phylip, fastq to fasta.  Data can be read from a file or STDIN.


OPTIONS
       -s, --seqf=FILE
              input sequence file, STDIN otherwise

       -u, --uppercase
              export characters in uppercase

       -o, --outf=FILE
              output sequence file, STOUT otherwise

       -h, --help
              display this help and exit

       -V, --version
              display version and exit

       -C, --citation
              display phyx citation and exit
