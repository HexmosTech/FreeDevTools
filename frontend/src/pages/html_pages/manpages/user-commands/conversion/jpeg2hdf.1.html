NAME
       jpeg2hdf - convert JPEG images to HDF


SYNOPSIS
       jpeg2hdf jpeg-file output-hdf-file


DESCRIPTION
       jpeg2hdf reads an image from jpeg-file and converts it to HDF format in output-hdf-file.


SEE ALSO
       hdf2jpeg(1), hdf(5)
