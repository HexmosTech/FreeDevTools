NAME
       nuweb2noweb - convert nuweb files to noweb form


SYNOPSIS
       nuweb2noweb file ...


DESCRIPTION
       Nuweb2noweb  converts  a  file  or files in nuweb format to noweb format, writing the results on standard
       output.  Input is from the file(s) named on the command line.


SEE ALSO
       noweb(1), nuweb(1)
