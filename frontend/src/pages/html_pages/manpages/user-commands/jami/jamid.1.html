NAME
       jamid - core daemon for Jami: secure and distributed voice, video and chat communication platform.


SYNOPSIS
       jamid [OPTION]...


DESCRIPTION
       Jami is meant to be a robust enterprise-class desktop phone. It provides functions like call transfer,
       call hold, multiple lines, multiple accounts support.  Jami audio layer is build upon a native ALSA
       interface and a native PulseAudio interface.  jamid is the core of Jami; it provides the DBus service
       "cx.ring.Ring". You need to install a client to use the daemon, for instance the Qt-based client, jami-
       client-qt (now jami).


OPTIONS
       -c, --console
               Output the log to the console instead of syslog.

       -d, --debug
               Debug mode (more verbose output).

       -a, --auto-answer
               Force automatic answer to incoming calls.

       -p, --persistent
               Stay alive after all clients quit.

       -h, --help
               Print short list of command-line options.
