NAME
       gnome-dvb-control - control gui for gnome-dvb-daemon


SYNOPSIS
       gnome-dvb-control


DESCRIPTION
       This manual page documents briefly the gnome-dvb-control command.

       gnome-dvb-control  is a client for gnome-dvb-daemon written in Python. You can view EPG and setup records
       with it. You will have to start gnome-dvb-daemon before starting it.

       Please use gnome-dvb-setup to initialize your cards before starting the program the first time.
