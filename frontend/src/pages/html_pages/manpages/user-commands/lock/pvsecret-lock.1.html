NAME
       pvsecret-lock - Lock the secret-store (s390x only)


SYNOPSIS
       pvsecret lock


DESCRIPTION
       Lock the secret store (s390x only). After this command executed successfully, all subsequent add-secret
       requests will fail. Only available on s390x.

       pvsecret(1)
