NAME
       tigr-glimmer — Apply the suite of programs within glimmer3 to a a prokaryotic or archean genome.


SYNOPSIS
       tigr-run-glimmer3


DESCRIPTION
       A shell script that wraps a set of tigr-* utilities of the glimmer package to retrieve coding regions.
