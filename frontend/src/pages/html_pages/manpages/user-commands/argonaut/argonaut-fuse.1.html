NAME
       argonaut-fuse - FUSE/TFTP supplicant targeted to work with LDAP entries written by FusionDirectory


SYNOPSIS
       argonaut-fuse


DESCRIPTION
       argonaut-fuse is a modular fuse-tftp-supplicant written in perl which allows one to create pxelinux
       configurations for different types of clients using external modules.


BUGS

       users or to <https://gitlab.fusiondirectory.org/argonaut/argonaut/issues/new>
