NAME
       name-search - check if a certain name is available for use on a handful of open source related sites


SYNOPSIS
       name-search $@


DESCRIPTION
       name-search  is  a script that will check if a given namespace is available in the Ubuntu apt archive, on
       Github.com, on Launchpad.net, and if domain names might be available.

       This may be helpful when selecting the name of an open source project, for instance.


SEE ALSO
       wget(1), nslookup(1), apt-cache(8)
