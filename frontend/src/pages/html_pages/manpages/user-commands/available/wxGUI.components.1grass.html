wxGUI Components
KEYWORDS
       general, GUI


DESCRIPTION
       List of available wxGUI components:

           •   Module dialogs

           •   3D Viewer (nviz)

           •   Animation tool, available as a command line tool g.gui.animation

           •   Attribute Table Manager, available also as a command line tool g.gui.dbmgr

           •   Cartographic Composer, available also as a command line tool g.gui.psmap

           •   Data Catalog, available also as a command line tool g.gui.datacatalog

           •   Graphical Modeler, available also as a command line tool g.gui.gmodeler

           •   Ground Control Points Manager, available also as a command line tool g.gui.gcp

           •   Interactive Scatter Plot Tool

           •   Map Swipe, available also as a command line tool g.gui.mapswipe

           •   Supervised Classification Tool, available also as a command line tool g.gui.iclass

           •   Configuration tool for r.li modules, g.gui.rlisetup

           •   Timeline Tool, available also as a command line tool g.gui.timeline

           •   Temporal Plot Tool, available also as a command line tool g.gui.tplot

           •   Vector Digitizer, available also as a command line tool g.gui.vdigit

           •   Raster Digitizer available also as a command line tool g.gui.rdigit

           •   Vector Network Analysis Tool
