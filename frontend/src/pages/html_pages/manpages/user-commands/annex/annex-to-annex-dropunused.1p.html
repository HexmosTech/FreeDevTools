NAME
       annex-to-annex-dropunused - drop old hardlinks migrated by annex-to-annex


VERSION
       version 0.008


SYNOPSIS
       annex-to-annex-dropunused


DESCRIPTION
       This program drops files which have been migrated by annex-to-annex(1).  You can run it in the source
       annex(es) to get rid of the old unused files which are known to have been migrated.

       Only files which have been hardlinked into the destination annex will be removed, i.e., no data will be
       irrevocably removed by this command.


OPTIONS
       None.


SEE ALSO
       git-annex-dropunused(1), annex-to-annex(1), annex-to-annex-reinject(1)
