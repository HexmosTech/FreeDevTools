NAME
       git-annex-list - show which remotes contain files


SYNOPSIS
       git annex list [path ...]


DESCRIPTION
       Displays  a  table  of  remotes  that contain the contents of the specified files. This is similar to git
       annex whereis but a more compact display.


OPTIONS
       --allrepos

              Only configured remotes are shown by default; this option adds all known repositories to the list.

       file matching options
              The git-annex-matching-options(1) can be used to specify files to list.

       Also the git-annex-common-options(1) can be used.
