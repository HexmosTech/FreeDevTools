NAME
       qlcplus-fixtureeditor - Fixture definition editor for QLC+


SYNOPSIS
       qlcplus-fixtureeditor


DESCRIPTION
       Fixture  Definition  Editor is a separate application for creating and modifying fixture definitions used
       by QLC+ The definitions tell QLC+ (and users) important details about fixtures, such as which channel  is
       used for pan movement, what value in which channel changes the beam color green, how the fixture is reset
       etc...


OPTIONS
       None.
