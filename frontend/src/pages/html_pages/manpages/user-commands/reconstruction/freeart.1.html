NAME
       freeart - tomographic image reconstruction library using Algebraic Reconstruction Technique


DESCRIPTION
       usage: freeart [--version|--help] <command> [<args>]

   The freeart commands are:
              createfluosino  Build  a sinogram from a phantom, anabsorption matrix and a self absorption matrix
              help       Show help of the following command interpreter Run  sinograms  reconstructions  from  a
              configuration  file  and  for  a  given  number  of iteration reconsfluosino Reconstruction from a
              fluorescence sinogram and absorption and  self  attenuation  matrices  test        Launch  freeart
              unittest

       See 'freeart help <command>' to read about a specific subcommand


SEE ALSO
       The  full  documentation for freeart is maintained as a Texinfo manual.  If the info and freeart programs
       are properly installed at your site, the command

              info freeart

       should give you access to the complete manual.
