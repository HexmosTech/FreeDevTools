NAME
       bppml - Ancestral sequence reconstruction with Bio++


SYNOPSIS
       bppancestor [options]


AVAILABILITY
       All UNIX flavors


DESCRIPTION
       bppancestor  performs  ancestral sequence reconstructions under a large variety of models, including non-
       homogeneous ones.


OPTIONS
       You should refer to 'info bppsuite' or to the online manual of bppsuite for a complete list of  available
       options.

       --noninteractive

            generates output for redirection in a file.

       param=file

            reads a file for loading options


AUTHOR
       The Bio++ Development Team.
