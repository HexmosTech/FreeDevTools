NAME
       simul-landmarks - Robot dataset simulator


SYNOPSIS
       simul-landmarks config_file.ini


DESCRIPTION
       simul-landmarks is a command-line application which generates a synthetic rawlog of a simulated robot as
       it follows a predefined path and observes a set of landmarks (through range, and 2 direction angles).
