NAME
       policygetdigest - Runs TPM2_PolicyGetDigest


DESCRIPTION
       --version-string=v1045 is not a valid option

       policygetdigest

       Runs TPM2_PolicyGetDigest

       -ha policy session handle

              [-of binary digest file name (default do not save)]

       --version is not a valid option
