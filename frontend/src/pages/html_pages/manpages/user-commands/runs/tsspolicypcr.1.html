NAME
       policypcr - Runs TPM2_PolicyPCR


DESCRIPTION
       --version-string=v1045 is not a valid option

       policypcr

       Runs TPM2_PolicyPCR

       -ha policy session handle

       -halg (sha1, sha256) (default sha256)

       -bm pcr mask in hex

              e.g., -bm 10000 is PCR 16, 000001 is PCR 0

       --version is not a valid option
