NAME
       nvextend - Runs TPM2_NV_Extend


DESCRIPTION
       --version-string=v1045 is not a valid option

       nvextend

       Runs TPM2_NV_Extend

       -ha NV index handle

       -pwdn password for NV index (default empty)

       -ic data string

       -if data file

       -se[0-2] session handle / attributes (default PWAP)

              01 continue
