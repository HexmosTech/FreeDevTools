NAME
       lei-init - initialize storage


SYNOPSIS
       lei init [OPTIONS] [DIRNAME]


DESCRIPTION
       Initialize local writable storage for lei(1).  If "DIRNAME" is unspecified, the storage is created at
       "$XDG_DATA_HOME/lei/store".  "leistore.dir" in "$XDG_CONFIG_HOME/lei/config" records this location.


OPTIONS
       -q
       --quiet
           Suppress feedback messages.
