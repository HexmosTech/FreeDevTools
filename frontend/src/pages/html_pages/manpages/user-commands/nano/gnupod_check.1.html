NAME
       gnupod_check.pl - manual page for gnupod_check.pl 1.00


SYNOPSIS
       gnupod_check.pl [OPTION]...


DESCRIPTION
       Check  for  lost/zombie files on the iPod.  Use this tool if your OS crashed while uploading songs to the
       iPod or if you have too much time ;)

       --help display this help and exit

       --version
              output version information and exit

       -m, --mount=DIRECTORY
              iPod mountpoint. Default is $IPOD_MOUNTPOINT
