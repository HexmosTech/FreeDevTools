NAME
       quickcal - It is a Fast and Easy to use Calculator with support for filing.


SYNOPSIS
       quickcal


DESCRIPTION
       Quickcal is a Fast and Easy to use Calculator, for performing Arithmetic and Statistical Calculations, at
       just the click of a button. It allows quick pasting of a large set of numbers, separated with newlines or
       spaces or tabs and performs calculations based on the button pressed. As in other calculators, basic math
       can  be  performed  too, with the +,-,*,/,%,^ operators, in between numbers. Setting the Scale allows for
       more decimals to be displayed. Also, about 65 Statistical Calculations can be performed  by  choosing  an
       option from the More Stats List Box.


OPTIONS
       quickcal does not take any options.
