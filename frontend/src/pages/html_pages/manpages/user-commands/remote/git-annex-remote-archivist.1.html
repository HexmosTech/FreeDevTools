NAME
       git-annex-remote-archivist - git-annex custom special remote to git-annex custom special remote to access
       to annex keys stored within other archive-type annex keys


SYNOPSIS
       git-annex-remote-archivist [-h] [-l LEVEL] [--version] [--dbg]


OPTIONS
       -h, --help
              show a help message and exit

       -l LEVEL, --log-level LEVEL
              set  logging  verbosity  level.  Choose  among critical, error, warning, info, debug. Also you can
              specify an integer <10 to provide even more debugging information

       --version
              show version information

       --dbg  catch exceptions and fall into a debugger
