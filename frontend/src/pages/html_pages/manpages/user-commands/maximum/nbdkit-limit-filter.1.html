NAME
       nbdkit-limit-filter - limit number of clients that can connect concurrently


SYNOPSIS
        nbdkit --filter=limit PLUGIN [limit=N]


DESCRIPTION
       "nbdkit-limit-filter" is an nbdkit filter that limits the number of clients which can connect
       concurrently.  If more than "limit=N" (default: 1) clients try to connect at the same time then later
       clients are rejected.


PARAMETERS
       limit=N
           Limit  the  number  of concurrent clients to "N".  This parameter is optional.  If not specified then
           the limit defaults to 1.  You can also set this to 0 to make the number  of  clients  unlimited  (ie.
           disable the filter).


FILES
       $filterdir/nbdkit-limit-filter.so
           The filter.

           Use "nbdkit --dump-config" to find the location of $filterdir.


VERSION
       "nbdkit-limit-filter" first appeared in nbdkit 1.20.
