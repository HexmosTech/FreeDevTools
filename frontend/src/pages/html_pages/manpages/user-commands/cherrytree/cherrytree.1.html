NAME
       cherrytree - a hierarchical note taking application


SYNOPSIS
       cherrytree   [-V]   [-N]   [filepath   [-n   nodename]   [-a   anchorname]  [-x  export_to_html_dir]  [-t
       export_to_txt_dir] [-p export_to_pdf_path] [-P password] [-w] [-s]]


DESCRIPTION
       cherrytree is a hierarchical note taking application, featuring rich text,  syntax  highlighting,  images
       handling,  hyperlinks,  import/export  with support for multiple formats, support for multiple languages,
       and more.
