NAME
       fzputtygen - SFTP private key converter of FileZilla


DESCRIPTION
       fzputtygen is part of FileZilla. It is used to convert private keys from OpenSSH or ssh.com format into a
       format understood by fzsftp. It usually gets called by FileZilla and is not intended to be used directly.

       fzputtygen is based on the puttygen component of PuTTY.
