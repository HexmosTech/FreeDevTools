NAME
       pictor-rename - rename pictures sensibly, prepending the timestamp in YYYY-MM-DD_HH:MM:SS__* form


SYNOPSIS
       pictor-unload FILE1 [FILE2 FILE3 FILE4 ...]


DESCRIPTION
       The pictor-rename utility is useful for renaming pictures, in the order of their creation timestamps.

       If  you perfectly synchronize the embedded clock on each of several cameras, you can use pictor-rename to
       conveniently combine and interleave pictures from these cameras.

       This utility will attempt to use exiftool(1) to read the file  creation  time  from  the  picture's  exif
       headers.  If that fails, then it will use the file modification time as specified by stat(1).


SEE ALSO
       stat(1), exiftool(1)
