NAME
       char2sp - AME


SYNOPSIS
       [-v][-s char] string [string2][...][string n]


DESCRIPTION
       Replaces a specified character in a string with spaces (def=_)

       -s [char]: Replace this character with spaces in output

       -v: Verbose
