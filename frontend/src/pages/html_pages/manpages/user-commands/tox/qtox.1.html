NAME
       qtox - Powerful Tox client that follows the Tox design guidelines


SYNOPSIS
       qtox [OPTION...] URI


DESCRIPTION
       qTox  is  a  powerful  Tox  client  that  follows  the  Tox  design guidelines while running on all major
       platforms.


OPTIONS
       -h, --help
              Displays this help.

       -v, --version
              Displays version information.

       -p, --profile <profile>
              Starts new instance and loads specified profile.

       -l, --login
              Starts new instance and opens the login screen.

       URI    Tox URI to parse
