NAME
       hcxmactool - hcx tools set


DESCRIPTION
       hcxmactool 6.2.7 (C) 2022 ZeroBeat usage: hcxmactool <options>

       options:  -o <oui>    : filter access point by OUI -n <nic>    : filter access point by NIC -m <mac>    :
       filter access point by MAC -a <vendor> : filter access point by VENDOR name -O <oui>    :  filter  client
       by  OUI -N <nic>    : filter client by NIC -M <mac>    : filter client by MAC -A <vendor> : filter client
       by VENDOR name -h          : show this help -v          : show version

       --pmkideapolout=<file> : output PMKID/EAPOL hash line (22000 format) --pmkidin=<file>       : input PMKID
       file  --pmkidout=<file>       :  output  PMKID  file   --hccapxin=<file>        :   input   HCCAPX   file
       --hccapxout=<file>       :   output  HCCAPX  file  --help                  :  show  this  help  --version
       : show version
