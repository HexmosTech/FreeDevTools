NAME
       cif_CODify  -  extract  essential  data  from  the  input  CIF  file and format it using the rules of the
       Crystallography Open Database (COD).


SYNOPSIS
       cif_CODify 1000000.cif

       cif_CODify 4000042.cif --options


DESCRIPTION
       Extract essential data from the input CIF file and format it using the rules of the Crystallography  Open
       Database (COD).
