NAME
       XML2Modelica - Scilab's translator from XML model to flat Modelica


SYNOPSIS
       XML2Modelica -o <outputfile> <inputfile>


OPTIONS
       -o <outputfile>
              Set output file name

       -init  Generate flat Modelica code for initialization
