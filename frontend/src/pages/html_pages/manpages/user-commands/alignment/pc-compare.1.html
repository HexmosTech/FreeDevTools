NAME
       pc-compare - Program for scoring alignments according to a reference alignment with respect to sum-of-
       pairs and column scores.


SYNOPSIS

       pc-compare [TEST_ALIGNMENT] [REFERENCE_ALIGNMENT] [BALIBASE_ANNOT_FILE] [-col] [-core] [-caps]
                  [-annot FILENAME]


NOTE
       pc-compare is named compare in the original sources, but has been remamed to avoid collision with other
       program names.


SEE ALSO
       pc-makegnufile(1), pc-project(1),from the probcons-extra package, and probcons(1) and probcons-RNA(1)
       from the probcons package.


REFERENCE

       Consistency-based Multiple Sequence Alignment. Genome Research 15: 330-340.
