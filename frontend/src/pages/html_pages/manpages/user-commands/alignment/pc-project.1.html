NAME
       pc-project - Program to project multiple alignment to pairwise alignments.


SYNOPSIS

       pc-project [ALIGNMENT] [-nocompressgaps]


NOTE
       pc-project is named project in the original sources, but has been remamed to avoid collision with other
       program names.


SEE ALSO
       pc-makegnufile(1), pc-compare(1),from the probcons-extra package, and probcons(1) and probcons-RNA(1)
       from the probcons package.


REFERENCE
       Please cite Do, C.B., Mahabhashyam, M.S.P., Brudno, M., and Batzoglou, S. 2005. PROBCONS: Probabilistic
       Consistency-based Multiple Sequence Alignment. Genome Research 15: 330-340.
