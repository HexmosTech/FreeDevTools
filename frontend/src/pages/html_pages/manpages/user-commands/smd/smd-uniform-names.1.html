NAME
       smd-uniform-names - renames local mails as on the remote host


SYNOPSIS
       smd-uniform-names [-v|--verbose] [endpoint]


DESCRIPTION
       smd-uniform-names  is  useful  to  migrate  from offlineimap to smd.  offlineimap may name the same email
       differently on the local and remote host. This utility generates a script to be run on the local host  to
       uniform the names to the remote ones.

       The utility generates the script '~/smd-rename.sh' that has to be run by the user.


OPTIONS
       -v --verbose
              Verbose output

       endpoint
              Is  the  suffix for the name of the configuration file to use. If it is omitted, the configuration
              file ~/.smd/config.default is used.


FILES
       ~/.smd/config.*  ~/smd-rename.sh


SEE ALSO
       smd-pull(1), smd-push(1), smd-config(5)
