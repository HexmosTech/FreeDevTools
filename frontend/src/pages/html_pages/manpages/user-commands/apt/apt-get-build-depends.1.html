NAME
       apt-get-build-depends - apt-get install build depends


SYNOPSIS
       apt-get-build-depends


DESCRIPTION
       apt-get-build-depends  parse  the  Build-Depends{-Indep}  lines  from  a  debian/control file and apt-get
       installs the required packages.
