NAME
       pcp_proc_count - displays the list of Pgpool-II children process IDs


SYNOPSIS

       pcp_proc_count [options...]


DESCRIPTION
       pcp_proc_count displays the list of Pgpool-II children process IDs. If there is more than one process,
       IDs will be delimited by a white space.


OPTIONS
       See pcp_common_options(1).
