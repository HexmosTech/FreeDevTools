NAME
       km-kvk2ldml - Convert a Keyman on-screen keyboard file to LDML


SYNOPSIS
       km-kvk2ldml
        [-h] [-p] [-k] [-o LDMLFILE] [--version] [-v] [-vv] kvkfile


DESCRIPTION
       Convert  a  Keyman  kvk  on-screen keyboard file to an LDML file. Optionally print the details of the kvk
       file.

   positional arguments:
       kvkfile
              kvk file

   options:
       -h, --help
              show this help message and exit

       -p, --print
              print kvk details

       -k, --keys
              if printing also print all keys

       -o, --output LDMLFILE
              output LDML file location

       --version
              show program's version number and exit

       -v, --verbose
              verbose logging

       -vv, --veryverbose
              very verbose logging
