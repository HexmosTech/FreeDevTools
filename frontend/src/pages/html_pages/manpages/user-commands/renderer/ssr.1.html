NAME
       ssr - SoundScape Renderer


DESCRIPTION
       USAGE: ssr-<renderer> [OPTIONS] <scene-file>

       The  SoundScape  Renderer (SSR) is a tool for real-time spatial audio reproduction providing a variety of
       rendering algorithms.

       See the various ssr-<renderer>s for options
