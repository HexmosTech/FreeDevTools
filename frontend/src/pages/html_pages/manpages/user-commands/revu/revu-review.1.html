NAME
       revu-review - Debian packages reviewing tool


SYNOPSIS
       revu-review .if (0>(2688/2)) .nr x (2688/5) [ --help | -h ] [ package ] [ URL ]


DESCRIPTION
       Debian packages reviewing tool.

       revu-review  is  a reviewing tool from the revu-tools package, aimed to helping reviewers of Debian pack‐
       ages in their task. revu-review gets the latest package version on REVU and runs revu-report on it.


OPTIONS
       --help | -h
              Show informations about the program and the usage.

       package
              Indicate package name to use.

       URL    Indicate url to use. This is optional.

       revu-report(1)
