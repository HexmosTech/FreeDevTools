NAME
       topcom-lattice - compute the permutations of a lattice


SYNOPSIS
       topcom-lattice M N


DESCRIPTION
       Compute the permutations of a lattice.  The permutations are written to stdout.


EXIT STATUS
       0 if the correct number of command line arguments are given.

       1 otherwise.
