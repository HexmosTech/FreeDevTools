NAME
       obdistgen — generate rough 3D coordinates for SMILES (or other 0D files)


SYNOPSIS
       obdistgen filename


DESCRIPTION
       Generates SDFs with distance geometry.


EXAMPLES
             obdistgen molecule.smi > molecule.sdf

       Generate rough 3D coordinates for molecule.smi and write it to molecule.sdf


SEE ALSO
       obgen(1).
