NAME
       cg-sort - CG-3 v1.3.9


SYNOPSIS
       cg-sort [OPTIONS]


DESCRIPTION
       Pipe a CG stream through this to sort and unique the readings of each cohort.


OPTIONS
       -?, --help
              outputs this help

       -w, --weight
              sorts by a numeric tag; defaults to W

       -r, --reverse
              reverses the sort order
