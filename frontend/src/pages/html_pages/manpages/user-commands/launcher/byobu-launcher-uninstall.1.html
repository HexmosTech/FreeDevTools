NAME
       byobu-launcher-uninstall - Byobu Launcher uninstallation utility


SYNOPSIS
       byobu-launcher-uninstall


DESCRIPTION
       byobu-launcher-uninstall(1)  is  a  simple  utilty  that removes the line added to your $HOME/.profile by
       byobu-launcher-install(1) which launches byobu(1) any time you login through ssh(1) or on a console(4).
