NAME
       getinstrs - extracts TrueType instructions for one or more glyphs from a font.


SYNOPSIS
       getinstrs [options] fontfile glyphs...


DESCRIPTION
       getinstrs  calls the FontForge Python extension to extract instructions from a TrueType font (either .ttf
       or .sfd) and print them to stdout (redirect to capture them in a file). By default the  instructions  are
       printed  as  an  Xgridfit  glyph  program.  Use  the -r, --raw command-line option to print them as "raw"
       instructions instead, or use -x, --xgridfit to enclose the glyph program(s)  in  an  <xgridfit>  element,
       making a complete, valid Xgridfit program.


OPTIONS
       -h, --help
              Display a help message and exit.

       -r, --raw
              Print "raw" instructions instead of Xgridfit glyph programs.

       -x, --xgridfit
              Print XML declaration and <xgridfit> element, making a complete, valid Xgridfit program.
