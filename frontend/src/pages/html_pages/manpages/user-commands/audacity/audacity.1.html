NAME
       audacity - audio editor


SYNOPSIS
       audacity [ AUDIO-FILE ]


DESCRIPTION
       Audacity  is  a  graphical  audio  editor. It generally does not support command line options, outside of
       opening files.

       Documentation and support can be found on https://support.audacityteam.org/

       The manual for Audacity 3.x can be found on https://manual.audacityteam.org/
