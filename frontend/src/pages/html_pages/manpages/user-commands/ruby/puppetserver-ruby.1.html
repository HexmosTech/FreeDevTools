NAME
       puppetserver-ruby - Puppetserver ruby


SYNOPSIS
       puppetserver ruby (--help)
       puppetserver ruby <options>


DESCRIPTION
       Run code in Puppet Server's JRuby interpreter.


MORE INFORMATION
       This  command  behaves  exactly  like  the  regular  JRuby  ruby  command, but uses Puppet Server's JRuby
       environment.

       For more information on how to use this command, see the jruby(1) man page.
