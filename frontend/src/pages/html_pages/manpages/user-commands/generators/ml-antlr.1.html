NAME
       ml-antlr - an LL(k) parser generator


SYNOPSIS
       ml-antlr [OPTIONS] file


DESCRIPTION
       ML-Antlr is an LL(k) parser generator for Standard ML that is loosely modeled on Terence Parr’s ANTLR
       parser generator.


OPTIONS
       --dot
           generate DOT output (http://www.graphviz.org). The generated file will be named file.dot, where file
           is the input file.

       --latex
           generate a simple LaTeX version of the grammar, named file.tex.

       --unit-actions
           ignore the action code in the grammar, and instead return unit for every production.
