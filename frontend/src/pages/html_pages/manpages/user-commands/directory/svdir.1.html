NAME
       svdir - Find daemontools service directory


SYNOPSIS
       svdir [OPTION]...


DESCRIPTION
       Find daemontools service directory

       -h, --help
              Display help and exit

       -V, --version
              Display version and exit

       This  utility tries to find the daemontools service directory and print it to stdout. It first checks the
       current directory  of  a  running  svscan  process  and  then  it  checks  some  default  locations  like
       /var/lib/svscan and /service. It returns 0 if it have found the location and 1 if not.
