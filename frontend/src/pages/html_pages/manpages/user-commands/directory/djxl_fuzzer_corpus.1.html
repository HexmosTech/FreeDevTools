NAME
       djxl_fuzzer_corpus - djxl_fuzzer_corpus


DESCRIPTION
       Output path "--help" is not a directory.  Use: fuzzer_corpus [-r] [-q] [-j THREADS] [output_dir]

       -r Regenerate files if already exist.

       -q Be quiet.

       -j THREADS Number of parallel jobs to run.
