NAME
       fldiff - compare files and directories graphically


SYNOPSIS
       fldiff [ first [ second ] ]


DESCRIPTION
       fldiff  is  a  graphical  diff  program  that  shows  the differences between two files/directories, or a
       file/directory and a CVS or Subversion repository.
