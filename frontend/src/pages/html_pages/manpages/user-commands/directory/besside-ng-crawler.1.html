NAME
       besside-ng-crawler - filter EAPOL frames from a directory of capture files.


SYNOPSIS
       besside-ng-crawler <Input Directory> <Output File>


DESCRIPTION
       besside-ng-crawler

       Scans  recursively on input directory looking for pcap dumpfiles and filters out one beacon and all EAPOL
       frames for the WPA networks if finds in them.
