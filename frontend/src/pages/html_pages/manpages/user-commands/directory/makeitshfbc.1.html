NAME
       makeitshfbc - creates the itshfbc directory structure in current user's home directory


SYNOPSIS
       makeitshfbc


DESCRIPTION
       makeitshfbc   creates  a  local  copy  of  the  itshfbc/  directory  structure in the current user's home
       directory.
