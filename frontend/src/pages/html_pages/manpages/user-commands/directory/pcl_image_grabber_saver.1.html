NAME
       pcl_image_grabber_saver - pcl_image_grabber_saver


DESCRIPTION
       Syntax is: pcl_image_grabber_saver <options>

       No correct directory was given with the -out_dir flag. Setting to current dir

       No directory was given with the -<rgb/depth>_dir flag.

        where options are:

          -rgb_dir   <directory_path>   = directory path to RGB images to be read from

          -depth_dir <directory_path>   = directory path to Depth images to be read from

          -out_dir   <directory_path>   = directory path to put the pcd files
