NAME
       prr-apply - Apply a pull request to the working directory


SYNOPSIS
       prr apply [-h|--help] <PR>


DESCRIPTION
       Apply a pull request to the working directory

       This can be useful for building/testing PRs


OPTIONS
       -h, --help
              Print help (see a summary with '-h')

       <PR>
