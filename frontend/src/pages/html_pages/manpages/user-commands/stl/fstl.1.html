NAME
       fstl - A viewer for .stl files


SYNOPSIS
       fstl [file.stl]


DESCRIPTION
       This manual page documents briefly the fstl command.

       fstl is a viewer for .stl files.

       It is optimized to quickly load and render very high-polygon models.


OPTIONS
       fstl does not take any options. You can, however, supply the name of an .stl file to be viewed.
