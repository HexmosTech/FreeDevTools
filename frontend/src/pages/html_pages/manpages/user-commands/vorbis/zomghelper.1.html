NAME
       zomghelper - Ogg Vorbis/Opus metadata reader


SYNOPSIS

       zomghelper ogg_vorbis_or_opus_file


DESCRIPTION
       For the file given on the command line, zomghelper will read the Ogg Vorbis or Ogg Opus metadata and
       print it to standard output.
