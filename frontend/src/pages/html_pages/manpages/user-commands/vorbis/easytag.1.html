NAME
       easytag - tag editor for MP3, Ogg Vorbis files and more


SYNOPSIS

       easytag [OPTION]

       easytag [PATH]


DESCRIPTION
       easytag is a utility for viewing and editing tags for MP3, MP2, MP4/AAC, FLAC, Ogg Speex, Ogg Opus, Ogg
       Vorbis, MusePack, Monkey's Audio and WavPack files. Its simple and nice GTK+ interface makes tagging
       easier under Linux or Windows.


INVOCATION
       easytag can take an option or a path as arguments.

   Options
       --help, -h
           Print brief help and exit.

       --version, -v
           Print the version and exit.

   Paths
       Either an absolute or a relative path can be supplied, which will open the path in the browser on
       startup.
