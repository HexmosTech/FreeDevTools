NAME
       stdpods - print out the paths to the modules that came with Perl


DESCRIPTION
       This program shows the paths to module pods that are not in the site_perl directories.  This is the
       documentation that came with the standard system.


EXAMPLES
           $ podgrep closure `stdpods`


SEE ALSO
       podgrep(1), faqpods(1), modpods(1), pods(1), podpath(1), and sitepod(1).
