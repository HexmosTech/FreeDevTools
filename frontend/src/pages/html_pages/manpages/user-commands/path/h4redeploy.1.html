NAME
       h4redeploy - updates paths in h4cc/h4fc


SYNOPSIS
       h4redeploy [OPTIONS]


DESCRIPTION
       h4redeploy  updates  paths in h4cc/h4fc after the HDF4 pre-compiled binaries have been installed in a new
       location.


OPTIONS
       -help  Display usage information

       -echo  Show all the shell commands executed

       -force No prompt, just do it

       -prefix=DIR
              Prefix directory to find HDF4 lib/ and include/ subdirectories [default: current directory]

       -tool=TOOL
              Tool to update.  TOOL must be in the current directory and writable.  [default: h4cc h4fc]

       -show  Show the commands without executing them
