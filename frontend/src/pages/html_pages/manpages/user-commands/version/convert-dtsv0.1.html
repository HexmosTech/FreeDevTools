NAME
       convert-dtsv0 - Device Tree Compiler -- Conversion to Version 1


SYNOPSIS
       /usr/bin/convert-dtsv0 [<input_filename ... >]


DESCRIPTION
       convert-dtsv0 is a small utility program which converts (DTS)

       Device  Tree  Source  from  the  obsolete version 0 to version 1.  Version 1 DTS files are marked by line
       "/dts-v1/;" at the top of the file.

       Each file passed will be converted to the new /dts-v1/ version  by  creating  a  new  file  with  a  "v1"
       appended the filename.

       Comments, empty lines, etc. are preserved.
