NAME
       ssh-version - shows version of the SSH server you are connecting to


SYNOPSIS
       ssh-version [OPTIONS] hostname


DESCRIPTION
       shows version of the SSH server you are connecting to


EXAMPLES
       ssh-version 127.0.0.1

       ssh-version -p 35007 127.0.0.1
