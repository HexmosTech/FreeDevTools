NAME
       alr-version - Shows detailed version, configuration, and environment information


SYNOPSIS
       alr version [options]


GLOBAL OPTIONS
       -c, --config=ARG
            Override configuration folder location

       -f, --force
            Keep going after a recoverable troublesome situation

       -h, --help
            Display general or command-specific help

       -n, --non-interactive
            Assume default answers for all user prompts

       --no-color
            Disables colors in output

       --no-tty
            Disables control characters in output

       --prefer-oldest
            Prefer oldest versions instead of newest when resolving dependencies

       --version
            Displays version and exits

       -q   Limit output to errors

       -v   Be more verbose (use twice for extra detail)

       -d, --debug[]
            Enable debug-specific log messages


DESCRIPTION
       Shows  assorted  metadata  about  the alr executable, and about the crate or sandbox found in the current
       directory, if any.
