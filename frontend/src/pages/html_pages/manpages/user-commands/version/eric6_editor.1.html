NAME
       eric6_editpr - Simplified version of the eric6 editor


SYNOPSIS
       eric6_editor [options] files...


DESCRIPTION
       This  manual page documents briefly the eric6_editor program. This manual page was written for the Debian
       distribution because the original program does not have a manual page.   Instead,  it  has  documentation
       provided in HTML format.


OPTIONS
       These  programs  follow  the  usual  GNU  command line syntax, with long options starting with two dashes
       (`-').  A summary of options is included below.

       -h, --help
              Show summary of options.

       --version
              Show version of program.

       --nokde
              Don't use KDE widgets
