NAME
       buildah-version - Display the Buildah Version Information.


SYNOPSIS
       buildah version [options]


DESCRIPTION
       Shows the following information: Version, Go Version, Image Spec, Runtime Spec, CNI Spec, libcni Version,
       Git Commit, Build Time, OS, and Architecture.


OPTIONS
       --help, -h

       Print usage statement

       --json

       Output in JSON format.


EXAMPLE
       buildah version

       buildah version --help

       buildah version -h
