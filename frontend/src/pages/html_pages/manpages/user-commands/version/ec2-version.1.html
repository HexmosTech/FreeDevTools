NAME
       ec2-version - Output build and API version


SYNOPSIS
        ec2ver ([ec2-version])
        ec2ver [GENERAL OPTIONS]


GENERAL NOTES
        Any command option/parameter may be passed a value of '-' to indicate
        that values for that option should be read from stdin.


DESCRIPTION
        Dump the build and API versions the EC2 API tools.


GENERAL OPTIONS
        -v, --verbose
            Verbose output.

        -?, --help
            Display this help.

        -H, --headers
            Display column headers.

        --debug
            Display additional debugging information.

        --show-empty-fields
            Indicate empty fields.

        --hide-tags
            Do not display tags for tagged resources.
