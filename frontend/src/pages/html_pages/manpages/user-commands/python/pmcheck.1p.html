NAME
       pmcheck - check that Perl is set up correctly for Perl modules


DESCRIPTION
       pmcheck checks that Perl is correctly set up for Perl modules.  For now, pmcheck just verifies that the
       entries in @INC are existing readable directories.


SEE ALSO
       pmdirs(1), pmpath(1)
