NAME
       stl_spreadsheet - stl_spreadsheet outputs an STL file in a tab delimited text format.


DESCRIPTION
       stl_spreadsheet outputs an STL file in a tab delimited text format.

       usage: stl_spreadsheet <input file>

              Prints vertex and normal information for every triangle in STL file in a tab delimited format that
              can be opened as a spreadsheet.
