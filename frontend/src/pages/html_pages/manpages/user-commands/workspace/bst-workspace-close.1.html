NAME
       bst-workspace-close - Close workspaces


SYNOPSIS
       bst workspace close [OPTIONS] [ELEMENTS]...


DESCRIPTION
       Close a workspace


OPTIONS
       --remove-dir
              Remove the path that contains the closed workspace

       -a, --all
              Close all open workspaces
