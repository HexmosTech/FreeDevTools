NAME
       es2_info - list OpenGL ES extensions


SYNOPSIS
       es2_info [options]


DESCRIPTION
       The es2_info program shows information about the supported OpenGL ES extensions.


OPTIONS
       -display display
               Specify the X display to run on.

       -h      Print usage information.


AUTHOR
