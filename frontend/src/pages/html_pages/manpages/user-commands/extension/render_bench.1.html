NAME
       render_bench - benchmark the XRender extension


SYNOPSIS
       render_bench


DESCRIPTION
       render_bench is a program that performs a series of tests to benchmark the XRender X11 extension. It also
       runs the same tests using the Imlib2 library to provide a base for comparing performance.
