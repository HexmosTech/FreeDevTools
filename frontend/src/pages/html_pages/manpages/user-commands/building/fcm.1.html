NAME
       fcm - FCM, tools for managing and building source code.


SYNOPSIS
       fcm command [ options ] [ args ]


OVERVIEW
       fcm is a set of tools for managing and building source code.  For full detail of the system, please refer
       to the FCM user guide, which you should receive with this distribution.

       Run "fcm help" to access the built-in tool documentation.
