NAME
       dnetstat - lists DECnet connections


SYNOPSIS
       dnetstat [-n] [-h|--help]


DESCRIPTION
       dnetstat is a netstat(8) like tool that displays current DECnet network connections.


OPTIONS
   -n
       Show nodenames and numbered objects in as numerical addresses and don't resolve names.

   --help, -h
       Prints a small help.


SEE ALSO
       netstat(8).
