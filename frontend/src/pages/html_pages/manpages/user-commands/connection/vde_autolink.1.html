NAME
       vde_autolink - automatically create and maintain vde connections


DESCRIPTION
       -h, --help
              Display this help

       -f, --rcfile
              Configuration file (overrides /etc/vde_autolink.rc and ~/.vde_autolinkrc)

       -d, --daemon
              Daemonize vde_autolink once run

       -p, --pidfile PIDFILE
              Write pid of daemon to PIDFILE

       -M, --mgmt SOCK
              Path of the management UNIX socket

       --mgmtmode MODE
              Management UNIX socket access mode (octal)

       -s, --sock
              [*] Attach to this vde_switch socket

       -S, --switchmgmt
              [*] Attach to this vde_switch management socket

              [*]   ==   Required   option!    -h,   --help                   Display  this  help  -f,  --rcfile
              Configuration  file  (overrides   /etc/vde_autolink.rc   and   ~/.vde_autolinkrc)   -d,   --daemon
              Daemonize  vde_autolink  once  run  -p,  --pidfile PIDFILE      Write pid of daemon to PIDFILE -M,
              --mgmt SOCK            Path of the management UNIX socket

       --mgmtmode MODE
              Management UNIX socket access mode (octal)

       -s, --sock
              [*] Attach to this vde_switch socket

       -S, --switchmgmt
              [*] Attach to this vde_switch management socket

              [*] == Required option!
