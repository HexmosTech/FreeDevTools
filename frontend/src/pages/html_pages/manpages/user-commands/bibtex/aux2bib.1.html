NAME
       AUX2BIB - Extracts a BibTeX database according to an aux file.


SYNOPSIS
       aux2bib <auxfile>


DESCRIPTION
       Aux2bib  reads  a  .aux  file as produced by LaTeX and writes to standard output a bibtex file containing
       exactly the bibtex entries refereed in the aux file.
