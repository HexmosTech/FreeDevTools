NAME
       qemu-user-static - QEMU User Emulator (compatibility symlink)


DESCRIPTION
       qemu-user-static  is  a  compatibility  symlink  for  old  qemu-user-static package pointing to qemu-user
       binary.  Please do not use these compatibility links, instead, switch to  using  corresponding  qemu-user
       binaries (without the -static suffix).
