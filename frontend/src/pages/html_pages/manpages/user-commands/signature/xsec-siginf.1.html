NAME
       xmlsec-siginf - Output information about a signature in an XML file


SYNOPSIS
       xmlsec-siginf [-s] input


DESCRIPTION
       xmlsec-siginf prints to standard output various information about a signature found in an XML file.  The
       signature must comply with the XML Digital Signature and Encryption specifications.


OPTIONS
       --skiprefs, -s
           Skip information on references and print out only information about the main signature.
