NAME
       rdx - load and execute an RDOFF object


SYNOPSIS
       rdx object


DESCRIPTION
       rdx  loads  an RDOFF object, and then calls `_main', which it expects to be a C-style function, accepting
       two parameters, argc and argv in normal C style.
