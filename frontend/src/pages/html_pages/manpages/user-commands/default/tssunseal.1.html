NAME
       unseal - Runs TPM2_Unseal


DESCRIPTION
       --version-string=v1045 is not a valid option

       unseal

       Runs TPM2_Unseal

       -ha sealed data item handle

              [-pwd password sealed data item (default empty)] [-of output data (default do not save)]

       -se[0-2] session handle / attributes (default PWAP)

              01 continue

       --version is not a valid option
