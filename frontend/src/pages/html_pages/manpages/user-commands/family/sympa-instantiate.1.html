NAME
       sympa-instantiate - Instantiate the lists in a family


SYNOPSIS
       "sympa instantiate" "--input-file="/path/to/file.xml [ "--close-unknown" ] [ "--noout" ] family"@@"domain


DESCRIPTION
       Instantiate the lists described in the file.xml in specified family.  The family directory must exist;
       automatically close undefined lists in a new instantiation if "--close_unknown" is specified; do not
       print report if "--noout" is specified.
