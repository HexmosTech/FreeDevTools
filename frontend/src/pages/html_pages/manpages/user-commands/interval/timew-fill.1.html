NAME
       timew-fill - adjust intervals to fill in surrounding gaps


SYNOPSIS
       timew fill <id>...


DESCRIPTION
       The 'fill' command is used to adjust any interval to fill in surrounding gaps. Using the 'summary'
       command, and specifying the ':ids' hint shows interval IDs. Using the right ID, you can identify an
       interval to fill.


EXAMPLES
       For example, show the IDs:

           $ timew summary :week :ids

       Then having selected '@2' as the interval you wish to fill:

           $ timew fill @2

       Note that you can fill multiple intervals:

           $ timew fill @2 @10 @23
