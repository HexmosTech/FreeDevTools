NAME
       mlmmj-list - list people / subscribers associated with a list


SYNOPSIS
       mlmmj-list -L /path/to/listdir [-c] [-d] [-h] [-m] [-n] [-o] [-s] [-V]

       -c: Count the emailaddresses

       -d: Print for digesters list

       -L: Full path to list directory

       -m: Print moderators for list

       -n: Print for nomail version of list

       -o: Print the owner(s) of the list

       -s: Print normail subscribers (default)

       -V: Print version
