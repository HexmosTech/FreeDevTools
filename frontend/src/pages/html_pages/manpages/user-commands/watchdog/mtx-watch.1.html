NAME
        mtx-watch - ConTeXt Request Watchdog


SYNOPSIS
       mtxrun --script watch [ OPTIONS ...  ] [ FILENAMES ]


DESCRIPTION
       ConTeXt Request Watchdog


OPTIONS
       --logpath
              optional path for log files

       --watch
              watch given path [--delay]

       --pipe use pipe instead of execute

       --delay
              delay between sweeps

       --automachine
              replace /machine/ in path /servername/

       --collect
              condense log files

       --cleanup=delay
              remove files in given path [--force]

       --showlog
              show log data
