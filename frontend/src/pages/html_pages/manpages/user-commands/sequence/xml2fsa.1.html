NAME
       xml2fsa - print FASTA sequence data for INSDSeq XML


SYNOPSIS
       xml2fsa


DESCRIPTION
       xml2fsa  reads  INSDSeq  XML from standard input and writes its sequence data to standard output in FASTA
       format.


SEE ALSO
       efetch(1), gbf2xml(1), transmute(1), xml2tbl(1), xtract(1).
