NAME
       vdr-newplugin - create a new VDR plugin source tree


SYNOPSIS
       vdr-newplugin <pluginname>


DESCRIPTION
       vdr-newplugin creates a new plugin source directory from which to start implementing a plugin for VDR.

       The source directory will be created in the active working-directory.
