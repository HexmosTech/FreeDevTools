NAME
        mtx-scite - Scite Helper Script


SYNOPSIS
       mtxrun --script scite [ OPTIONS ...  ] [ FILENAMES ]


DESCRIPTION
       Scite Helper Script


OPTIONS
       --words
              convert spell-*.txt into spell-*.lua

       --tree converts a tree into an html tree (--source --target --numbers)

       --file converts a file into an html file (--source --target --numbers --lexer)
