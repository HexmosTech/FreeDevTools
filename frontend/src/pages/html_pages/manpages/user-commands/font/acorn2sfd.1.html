NAME
       acorn2sfd - creates FontForge sfd files from Acorn RISCOS fonts


SYNOPSIS
       acorn2sfd [--help] [--includestrokes] [--version] [acorndirs]


DESCRIPTION
       acorn2sfd  takes a list of directory names and attempts to convert the font data within each directory to
       one of FontForge's sfd files.

       Acorn RISCOS fonts consist of 2 files Outlines and Intmetrics in a directory.


OPTIONS
       --help          Provide a description with a list of available options.

       --includestrokes
                       Set a flag to include strokes.

       --version       Display the current version.
