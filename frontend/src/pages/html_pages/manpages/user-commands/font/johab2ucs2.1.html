NAME
       johab2ucs2 - Convert a Johab BDF font into GNU Unifont Hangul Syllables


SYNOPSIS
       johab2ucs2 < input-font.bdf > output-font.hex


DESCRIPTION
       johab2ucs2  reads  a Johab encoded BDF font (as used in Hanterm) from STDIN and writes a GNU Unifont .hex
       file containing the Unicode Hangul Syllables to STDOUT.


FILES
       *.bdf font files
