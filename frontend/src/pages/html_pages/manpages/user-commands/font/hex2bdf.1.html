NAME
       hex2bdf - Convert a GNU Unifont .hex file into a BDF font


SYNOPSIS
       hex2bdf [options] < input-font.hex > output-font.bdf


DESCRIPTION
       hex2bdf  reads a sorted GNU Unifont .hex file (sorted with the Unix sort utility) from STDIN and writes a
       BDF version of the font to STDOUT.


OPTIONS
       -f "font-name"
                   Specify the target font name.  If omitted, the default font name "Unifont" is assigned.

       -v "font-version"
                   Specify the target font version.  If omitted, the default version "1.0" is assigned.

       -c "font-copyright"
                   Specify the target font copyright information.  The default is the null string.

       -r pixel-rows
                   Specify how many pixel rows tall a glyph is.  The default is the traditional Unifont 16  rows
                   of  pixels.   This  is  an  addition  to support unihex2png(1) and unipng2hex(1), which allow
                   designing glyphs that are 16, 24, or 32 pixel rows tall.


EXAMPLE
       Sample usage:

              hex2bdf -f "Unifont" -c "(C) 2013..." unifont.hex > unifont.bdf


FILES
       *.hex GNU Unifont font files
