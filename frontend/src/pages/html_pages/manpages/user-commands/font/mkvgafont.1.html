NAME
       mkvgafont - produces fonts for X servers from a RAW-console font.


SYNOPSIS
       mkvgafont raw-font


DESCRIPTION
       mkvgafont  Creates  fonts  vga.bdf and vga.pcf.Z equivalent to the raw-font given as parameter.  raw-font
       must be raw 8x16 font.  You can create such a font by consolechars(8).
