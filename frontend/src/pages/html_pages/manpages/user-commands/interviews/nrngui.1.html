NAME
       nrngui - InterViews GUI for NEURON


SYNOPSIS
       nrniv [FILE] [ARGS]


DESCRIPTION
       This manual page documents the nrngui command.

       nrngui  is a GUI frontend for the NEURON simulation environment for computational models of physiological
       neurons.  It executes nrniv(1) with the GUI frontend already loaded via nrngui.hoc.  It  takes  the  same
       arguments as nrniv(1).


MORE INFORMATION
       Check out the NEURON website at www.neuron.yale.edu/neuron/
