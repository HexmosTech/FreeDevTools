NAME
       caja-actions-print - print a menu or an action to stdout


SYNOPSIS
       caja-actions-print [OPTION]


DESCRIPTION
       caja-actions-print print a caja-actions menu or action to stdout.  This program is intended to be used on
       the command-line for scripting caja actions.


OPTIONS
       Help options

       -?, -h, --help Show help options

       --help-all Show all help options

       --help-misc Show miscellaneous options

       Miscellaneous options

       -v, --version Output the version number

       Application options

       -i, --id=STRING The internal identifier of the action to be launched.

       -f, --format=<STRING> An export format [Desktop1]
