NAME
       mkafmmap - creates font map for AFM files


SYNOPSIS
       mkafmmap [options] files...


DESCRIPTION
       The  mkafmmap  command will generate a font map file reading all the AFM files given in the command line.
       By default it will write the map to a file named font.map, a different filename can be specified with the
       --output-file options. Using '-' will make the program write the map to stdout.


OPTIONS
       -h, --help
              Show summary of options.

       -v, --version
              Show version of program.

       -p, --output-file=NAME
              Print output to file NAME.
