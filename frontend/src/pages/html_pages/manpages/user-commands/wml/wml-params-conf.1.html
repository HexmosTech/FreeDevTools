NAME
       wml-params-conf - displays configuration parameters for the wml installation.


SYNOPSIS
           use lib do { my $s = `wml-params-conf --show-privlib`; chomp $s; $s;}


OPTIONS
   --show-privlib
       Outputs the private library path followed by a newline.
