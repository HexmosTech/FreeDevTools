NAME
       yaml2ini - Converts YAML files into INI-like configuration files.


SYNOPSIS
       yaml2ini in.yml


DESCRIPTION
       yaml2ini is a command-line application which processes a YAML file and prints to the console the INI-like
       configuration file.
