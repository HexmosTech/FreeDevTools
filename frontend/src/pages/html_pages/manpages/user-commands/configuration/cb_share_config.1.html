NAME
       cb_share_config - Import/export parts of Code::Blocks configuration


SYNOPSIS
       cb_share_config


DESCRIPTION
       cb_share_config  allows  you  to share (import/export) parts of Code::Blocks configuration. It is useful,
       for example, when you want to partially transfer settings between computers  or  even  between  different
       configurations.
