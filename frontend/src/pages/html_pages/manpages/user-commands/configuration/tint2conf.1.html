NAME
       tint2conf - tint configuration manager


SYNOPSIS
       tint2conf


DESCRIPTION
       tint2conf is a GTK based configuration previewer for tint2.


OPTIONS
       tint2conf takes no parameters.


AUTHOR
       tint2conf was written by the tint2 team.
