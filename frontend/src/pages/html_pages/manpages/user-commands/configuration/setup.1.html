NAME
       setup - graphical setup program for the esp package manager


SYNOPSIS
       setup [ directory ]


DESCRIPTION
       setup  provides  a  graphical installation interface for EPM-generated portable installation packages. It
       presents a step-by-step dialog for collecting a list of packages to install  and  accepting  any  license
       agreements for those packages.

       setup searches for products in the current directory or the directory specified on the command-line.


INSTALLATION TYPES
       The  default  type  of  installation  is "custom". That is, users will be able to select from the list of
       products and install them.

       setup also supports other types of installations. The setup.types file, if  present,  defines  the  other
       installation types.
