NAME
       lxqt-config-input - Application of LXQt: The Lightweight Qt Desktop Environment


SYNOPSIS
       lxqt-config-input


DESCRIPTION
       This  is a simple GUI application for keyboard, mouse, and touchpad settings.  (Currently it can only run
       under X11.)


OPTIONS
       -s, --show-page <name>  Choose the page to be shown.
       --load-touchpad         Load last touchpad settings.
       -v, --version           Displays version information.
       -h, --help              Displays help on command line options.
       --help-all              Displays help, including generic Qt options
