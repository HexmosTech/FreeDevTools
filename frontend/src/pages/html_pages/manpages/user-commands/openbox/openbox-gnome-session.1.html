NAME
       openbox-gnome-session — Run a GNOME session with Openbox as the window manager


DESCRIPTION
       openbox-gnome-session runs a GNOME session with Openbox as the window manager.

       openbox-gnome-session does not take any command line arguments.


SEE ALSO
       openbox(1), openbox-kde-session(1), openbox-session(1).
