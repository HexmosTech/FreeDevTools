NAME
       nbdkit-time-limit-filter - set an overall time limit for each connection


SYNOPSIS
        nbdkit --filter=time-limit PLUGIN [time-limit=TIME]


DESCRIPTION
       "nbdkit-time-limit-filter" is an nbdkit(1) filter that sets an overall limit on the length of time that a
       client may be connected for.  After this time limit is exceeded, client commands are rejected and we
       attempt to disconnect the client.

       Note: because of current limitations in the filter API, a client can hold an idle connection open for
       longer than the time limit.  The time limit is only enforced when the client sends a command to the
       server.


PARAMETERS
       time-limit=TIME
           Limit  the  length of a connection to "TIME" seconds.  You can also use other formats as specified in
           nbdkit_parse_delay(3).

           The default is 60 seconds.


FILES
       $filterdir/nbdkit-time-limit-filter.so
           The filter.

           Use "nbdkit --dump-config" to find the location of $filterdir.


VERSION
       "nbdkit-time-limit-filter" first appeared in nbdkit 1.42.
