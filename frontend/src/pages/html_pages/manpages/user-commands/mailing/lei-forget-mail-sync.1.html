NAME
       lei-forget-mail-sync - forget sync information for a mail folder


SYNOPSIS
       lei forget-mail-sync [OPTIONS] LOCATION [LOCATION...]


DESCRIPTION
       Forget synchronization information for "LOCATION", an IMAP or Maildir folder.  Note that this won't
       delete any messages on the filesystem.  Users using lei-index(1) without lei-import(1) will be left with
       dangling references in search results.
