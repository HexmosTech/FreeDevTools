NAME
            edfhed-gtk -- a simple EDF header editor


SYNOPSIS
            edfhed-gtk file.edf


DESCRIPTION
       edfhed-gtk is GTK+ utility to edit the header data of EDF files.

       For the particulars of how Aghermann treats the various fields, see edfhed(1).


SEE ALSO
       edfhed(1), aghermann(1).
