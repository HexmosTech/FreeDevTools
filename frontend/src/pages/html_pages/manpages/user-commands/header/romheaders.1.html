NAME
       romheaders - PCI rom header utility


SYNOPSIS
       romheaders romimage.img


DESCRIPTION
       romheaders dumps pci option rom headers according to PCI specs 2.2 in human readable form.


SEE ALSO
       toke(1), detok(1).
