NAME
       stl_header - stl_header inspects and modifies the header section of an STL file.


DESCRIPTION
       stl_header inspects and modifies the header section of an STL file.

       usage: stl_header [-s <header>] [-o <output file>] <input file>

              If  both  -s  and -o flags are specified <input file> is copied to <output file> and its header is
              set to <header>. If -o is not specified and -s is then the header is set on  the  input  file.  If
              neither -o nor -s is specified, the header of the model is output to the terminal.
