NAME
       buddy-ng - a tool to work with easside-ng


SYNOPSIS
       buddy-ng <options>


DESCRIPTION
       buddy-ng server echoes back the decrypted packets to the system running easside-ng in order to access the
       wireless  network  without  knowing  the WEP key. It is done by having the AP itself decrypt the packets.
       When ran, it automatically starts and listen to port 6969.


OPTIONS
       -h     Shows the help screen.

       -p     Don't drop privileges
