NAME
       apertium-filter-ambiguity — ambiguity class filter for Apertium


SYNOPSIS
       apertium-filter-ambiguity [input_file [output_file]]


DESCRIPTION
       apertium-filter-ambiguity  takes  input  from  stdin  or  input_file, gets tagger data, filters ambiguity
       classes and outputs on STDOUT or output_file, in each case.
