NAME
       how-many-binary - count the number of binary packages in Ubuntu


SYNOPSIS
       how-many-binary [-a|--arch ARCHITECTURE] [-r|--release] [-v|--verbose]


DESCRIPTION
       how-many-binary(1) is a simple program that counts the number of binary packages in Ubuntu


EXAMPLE
        $ how-many-binary -a amd64 -r trusty 44880
