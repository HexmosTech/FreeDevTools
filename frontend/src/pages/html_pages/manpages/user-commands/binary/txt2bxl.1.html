NAME
       txt2bxl - encode a BXL file provided in plain text and save binary


SYNPOSIS
       txt2bxl outfile.bxl &lt; infile.txt


DECSRIPTION
       txt2bxl  reads  and encodes a text bxl file in its standard input and saves the resulting binary bxl file
       on the file name specified in the command line argument.
