NAME
       fftest - tests force-feedback devices.


SYNOPSIS
       fftest <device>


DESCRIPTION
       fftest provides a variety of tests which can be applied to force-feedback devices.  Beware, the tests may
       damage your device!


OPTIONS
       <device>
              The device to test.


SEE ALSO
       ffcfstress(1), ffmvforce(1), jstest(1).
