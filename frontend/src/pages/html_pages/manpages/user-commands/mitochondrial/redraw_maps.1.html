NAME
       redraw_maps - visually comparing bacterial, plasmid, chloroplast, or mitochondrial sequences


DESCRIPTION
   USAGE:
              redraw_maps -p DIR [Options]

   DESCRIPTION:
              Used  to  redraw  the  maps.  This  can be used after editing the CGView XML file or to change the
              output image formats.

   REQUIRED ARGUMENTS:

       -p, --project DIR

              Path to a completed CCT project.

   OPTIONAL ARGUMENTS:

       -f, --format STRING

              Image format for output map. Options are png, jpg, svg, svgz.  (Default: png)

       -m, --memory STRING

              Memory value for Java's -Xmx option (Default: 1500m).

       -h, --help

              Show this message

   EXAMPLE:
              redraw_maps -p my_project -f svg
