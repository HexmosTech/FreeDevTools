NAME
       fst-compare - checks whether two transducers are equivalent


SYNOPSIS
       fst-compare file1 file2


DESCRIPTION
       fst-compare checks whether the two transducers stored in file1 and file2 are equivalent.


BUGS
       No bugs are known so far.
