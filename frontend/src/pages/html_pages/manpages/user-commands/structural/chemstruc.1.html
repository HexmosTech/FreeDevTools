NAME
       chemstruc - calls a browser to view molecular structures


SYNOPSIS

       chemstruc


DESCRIPTION
       chemstruc is a program to browse the chemical structures coming with the package chemical-structures.

       There are currently 253 molecular structures, in the following classes: alcohols, aldehydes, alkanes,
       alkenes, amides, amines, amino_acids, aromatics, carboxylic_acids, esters, ethers, fatty_acids,
       haloalkanes, ketones, nitriles, nucleobases, water.
