NAME
       gsalliere — Duplicate Bridge Scorer


SYNOPSIS
       gsalliere [boards.csv]            [names.csv]


DESCRIPTION
       This manual page documents briefly the gsalliere command.

       gsalliere  scores,  matchpoints  and  nicely  formats  the  boards for a duplicate pairs evening. It is a
       graphical frontend to salliere(1).

       Board and names files may be specified on the command line or loaded once the program has started.

       The boards file must contain lines with at least board number, NS pair number, EW pair number,  Contract,
       Declarer  and  either tricks or the contract must include whether it made (=) went down (-tricks) or made
       over tricks (+tricks). The names file must list at least pair number  followed  by  two  comma  separated
       names.
