NAME
       congruity - A simple GUI to program Logitech(R) Harmony(R) remotes


SYNOPSIS
       congruity FILE


DESCRIPTION
       congruity  is  intended  to  process  files provided by the Logitech(R) Harmony(R) configuration website,
       currently members.harmonyremote.com.

       You should configure your web browser to execute congruity whenever a file of type EZHex, EZUp, or  EZTut
       is downloaded.
