NAME
       nfc-barcode - NFC Barcode (Tag-Talks-First) reader


SYNOPSIS
       nfc-barcode


DESCRIPTION
       nfc-barcode is a utility to read and decode NFC Barcodes


OPTIONS
       -d Decode content, if possible.

       -v Verbose.
