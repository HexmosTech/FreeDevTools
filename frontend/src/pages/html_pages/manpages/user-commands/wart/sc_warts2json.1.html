NAME
       sc_warts2json — JSON dump of information contained in a warts file.


SYNOPSIS
       sc_warts2json [file ...]


DESCRIPTION
       The  sc_warts2json  utility  provides  a JSON dump of information contained in a sequence of warts files.
       The output is the same as that which would have been provided by scamper if the JSON  output  option  had
       been chosen instead of the warts output option when the data was collected.


EXAMPLES
       The command:

          sc_warts2json file1.warts file2.warts

       will decode and print the contents of file1.warts, followed by the contents of file2.warts.

       The command:

          gzcat file1.warts.gz | sc_warts2json

       will print the contents of the uncompressed file supplied on stdin.
