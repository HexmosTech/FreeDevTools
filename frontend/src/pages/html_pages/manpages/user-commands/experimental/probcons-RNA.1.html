NAME
       probcons-RNA - An experimental version of ProbCons compiled with parameters estimated via unsupervised
       training on BRAliBASE.


DESCRIPTION
       See the manpage of probcons(1) for the command-line arguments, and /usr/share/doc/README.Debian for more
       information about how probcons-RNA has been built.


SEE ALSO
       •   You can find more information in the manual of ProbCons, which is located in
           /usr/share/doc/probcons/manual.pdf in Debian systems.

       •    probcons(1), pc-makegnufile(1), pc-compare(1), pc-project(1), which are distributed separately in
           the probcons-extra package.
