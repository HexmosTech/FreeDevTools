NAME
       anc-describe-image  -  describes  the  description  of  cloud  images  available  for deployment from the
       Atlantic.Net Cloud.


DESCRIPTION
       anc-describe-image This method enables the client to retrieve the  description  of  all  available  cloud
       images or the description of a specific cloud image by providing the image id (e.g. ubuntu-14.04_64bit)


SYNOPSIS
       anc-describe-image [Format=[json|xml]] imageid=[Image ID]

       INPUT

       Format=[json|xml]
       The  Format  parameter  will  determine  how the output of the action is formatted. You can format output
       results in JSON or XML. Default is JSON.

       imageid=[Image ID]
       An imageid taken from the anc-describe-image function or 'all' to utilize all available images.
