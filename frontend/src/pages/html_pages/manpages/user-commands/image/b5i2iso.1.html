NAME
       b5i2iso - convert a BlindWrite image file into standard ISO image file


SYNOPSIS
       b5i2iso [OPTION] [BASENAME.b*i]


DESCRIPTION
       B5I2ISO  is  a  very  simple  utility  to convert a BlindWrite BIN image to the standard ISO-9660 format.
       BlindWrite image can have different extensions like .b5i or .bwi


OPTIONS
       --cue Generate cue file
