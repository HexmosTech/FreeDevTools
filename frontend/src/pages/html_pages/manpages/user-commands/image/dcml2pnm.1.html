NAME
       dcml2pnm - Convert DICOM images to PGM/PPM, PNG, TIFF or BMP


SYNOPSIS
       dcml2pnm [options] dcmfile-in [bitmap-out]


DESCRIPTION
       The  dcml2pnm  tool  is deprecated. Use dcm2img instead, which supports the same command line parameters,
       and more.
