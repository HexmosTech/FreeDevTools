NAME
       simple-image-filter -  Image Filter


SYNOPSIS
       simple-image-filter


DESCRIPTION
       Image  Filter  is  an  image  processing software, can handle image rotation, clip, basic beautification,
       filter and other graph functions.


OPTIONS
       This program does not accept any command line options.
