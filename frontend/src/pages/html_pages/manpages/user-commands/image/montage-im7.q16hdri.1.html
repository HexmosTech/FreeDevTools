NAME
       montage  -  create  a  composite  image by combining several separate images. The images are tiled on the
       composite image optionally adorned with a border, frame, image name, and more.


SYNOPSIS
       magick-im7.q16hdri montage input-file[s] [options] output-file


OVERVIEW
       The montage-im7.q16hdri program is a member of the ImageMagick-ims7.q16hdri(1) suite of tools.  Use it to
       create a composite image by combining several separate images. The images  are  tiled  on  the  composite
       image optionally adorned with a border, frame, image name, and more.

       For     more     information     about     the     montage     command,    point    your    browser    to
       file:///usr/share/doc/imagemagick-7-common/html/www/montage.html (on debian system you  may  install  the
       imagemagick-7 package) or https://www.imagemagick.org/script/montage.php.
