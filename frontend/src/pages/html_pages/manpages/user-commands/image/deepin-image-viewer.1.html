NAME
       deepin-image-viewer - the picture manager from UOS


SYNOPSIS
       deepin-image-viewer [PATH]


DESCRIPTION
       Deepin Image Viewer is a fashion photo manager for viewing and organizing pictures.

       This program does not accept any command line options.

       If the PATH is specified, open that picture with Image Viewer.  If the PATH is omitted, open Image Viewer
       and  show  all  pictures  in  it.  If Image Viewer has been launched, open the picture with Image Viewer.
       Otherwise, launch Image Viewer and open the picture with it.
