NAME
       pbmtopi3 - convert a PBM image into an Atari Degas .pi3 file


SYNOPSIS
       pbmtopi3 [pbmfile]


DESCRIPTION
       This program is part of Netpbm(1).

       pbmtopi3 reads a PBM image as input and produces an Atari Degas .pi3 file as output.


OPTIONS
       There are no command line options defined specifically for pbmtopi3, but it recognizes the options common
       to all programs based on libnetpbm (See  Common Options .)
