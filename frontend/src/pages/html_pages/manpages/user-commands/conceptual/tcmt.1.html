NAME
       tcmd - Toolkit for Conceptual Modeling Daemon


SYNOPSIS
       tcmt


DESCRIPTION
       tcmt  is  part  of  the  Toolkit  for Conceptual Modeling (TCM).  tcmt is not intended as a user program.
       Please use tcm in order to display and start up all available tools.
