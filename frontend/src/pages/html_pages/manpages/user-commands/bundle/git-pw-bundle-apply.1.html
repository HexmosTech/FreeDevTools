NAME
       git-pw-bundle-apply - Apply bundle.


SYNOPSIS
       git-pw bundle apply [OPTIONS] BUNDLE_ID [ARGS]...


DESCRIPTION
       Apply bundle.

       Apply  a  bundle  locally  using the 'git-am' command. Any additional ARGS provided will be passed to the
       'git-am' command.
