NAME
       bplist - Bundle Protocol (BP) utility for listing queued bundles


SYNOPSIS
       bplist


DESCRIPTION
       bplist is a utility program that lists all bundles currently in the local bundle agent's "timeline" list,
       in expiration-time sequence.  Identifying primary block information is printed, together with hex and
       ASCII dumps of the payload and all extension blocks.


EXIT STATUS
       "0" bplist terminated, for reasons noted in the ion.log file.

       "1" bplist  was unable to attach to Bundle Protocol operations, probably because bpadmin has not yet been
           run.


FILES
       No configuration files are needed.


ENVIRONMENT
       No environment variables apply.


DIAGNOSTICS
       The following diagnostics may be issued to the ion.log log file:

       Can't attach to BP.
           bpadmin has not yet initialized BP operations.
