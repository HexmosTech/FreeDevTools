NAME

DESCRIPTION
       Tools of the Chewing (酷音) intelligent Zhuyin input method.


OPTIONS
       -h, --help
              Print help

       -V, --version
              Print version


SUBCOMMANDS

              Create a new dictionary file

       chewing-cli-info(1)
              Display information about the dictionary

       chewing-cli-dump(1)
              Dump the dictionary entries into tsi.src formatted stream
