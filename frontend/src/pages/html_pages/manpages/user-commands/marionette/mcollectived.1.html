NAME
       mcollectived - Marionette Collective clustering framework


SYNOPSIS
       mcollectived [options]


DESCRIPTION
       The  Marionette  Collective  is  a framework for building server orchestration and parallel job execution
       systems.


OPTIONS
       -h, --help
              Show this help message and exit

       --config CONF
              Path to configuration file

       --pidfile PID
              Path to pid file

       --daemonize, --no-daemonize
              Daemonize or not mcollectived
