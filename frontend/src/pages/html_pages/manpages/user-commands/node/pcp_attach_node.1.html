NAME
       pcp_attach_node - attaches the given node to Pgpool-II.


SYNOPSIS

       pcp_attach_node [options...] [node_id]


DESCRIPTION
       pcp_attach_node attaches the given node to Pgpool-II.


OPTIONS
       -n node_id
       --node_id=node_id
           The index of backend node to attach.

       Other options
           See pcp_common_options(1).
