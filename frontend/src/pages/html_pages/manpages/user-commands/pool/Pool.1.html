NAME
       Pool - Sardana Pool Tango device server


SYNOPSIS
       Pool [instance_name]


DESCRIPTION
       Sardana is a software suite for Supervision, Control and Data Acquisition in scientific installations.

       The Pool command launches the Sardana Pool Tango device server.

       The best source for information about the available options is to run `Pool --help`.

       For more information about the Sardana project, visit http://sardana-controls.org.
