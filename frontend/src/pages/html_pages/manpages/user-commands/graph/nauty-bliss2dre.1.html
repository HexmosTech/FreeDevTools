NAME
       nauty-bliss2dre - convert files of graphs in Bliss/DIMACS format to stdout in sparse6 format


SYNOPSIS
       bliss2dre < bliss.graph > dreadnaut.graph


DESCRIPTION
              Reads a graph in Bliss/DIMACS format from stdin.  Writes the graph in dreadnaut format to stdout.
