NAME
       oodoc_test - OpenOffice::OODoc test document generation


DESCRIPTION
       This utility checks the current OpenOffice::OODoc installation and generates a test document from
       scratch. The generated document can then be found in the current directory, and its name is
       "odftest.odt".
