NAME
       SiSU - Structured information, Serialized Units - a document publishing system


SYNOPSIS
       sisu-odf sisu_document.sst | sisu_document.ssm


DESCRIPTION
       SiSU  is  a  lightweight  markup based document structuring, publishing and search framework for document
       collections.

       The sisu-odf command is an alias for sisu --odt (or sisu -o) which produces OpenDocument output of a SiSU
       markup file.
