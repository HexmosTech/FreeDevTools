NAME
       num - number utilities for mathematics


SYNOPSIS
       num sum median variance stddev data.txt


DESCRIPTION
       Num  is  a  command  line  tool  for  mathematics and statistics data processing.  Num can calculate sum,
       median, quartiles, standard deviation, and much more.  Num can sort, map, filter, round, and use  various
       input/output formats.


OPTIONS
       num sum median variance stddev data.txt
