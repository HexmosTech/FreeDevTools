NAME
       reform-help - help with MNT Reform


SYNOPSIS
       reform-help [--desktop,--root]


DESCRIPTION
       Prints help text for how to use the MNT Reform.


OPTIONS

       --desktop Print the MNT Reform desktop help text.

       --root
              Print the MNT Reform root shell help text.
