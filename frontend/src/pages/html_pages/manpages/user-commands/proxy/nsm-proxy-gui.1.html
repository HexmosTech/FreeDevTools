NAME
       nsm-proxy-gui - manual page for nsm-proxy-gui Version 1.6.1


DESCRIPTION
       nsm-proxy-gui - GUI for nsm-proxy, a wrapper for executables without direct NSM-Support.

   Usage:
              nsm-proxy-gui --help nsm-proxy-gui --connect-to


OPTIONS
       --help Show this screen

       --connect-to
              Connect to running nsm-proxy

       nsmd-proxy-gui  is  usually not called by the user directly, but autostarted when nsm-proxy is added to a
       session (through a GUI).
