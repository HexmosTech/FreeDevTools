NAME
       veyon-worker - Veyon Worker


SYNOPSIS
       veyon-worker


DESCRIPTION
       VEYON-WORKER  is   helper  program  started  by  the  server to provide specific functions in an isolated
       environment or in the context of the user that is currently logged in. Those specific  functions  include
       the demo server for the teacher computer and the demo client on the student computers.
