NAME
       sympa-restore - Restore users of the lists


SYNOPSIS
       "sympa restore" "--roles="role[","role...] list"@"domain|"*"


DESCRIPTION
       Restore users from files dumped by "--dump_users".


HISTORY
       This option was added on Sympa 6.2.34.
