NAME
       openshot-audio-demo - Display system info and play test sound


SYNOPSIS
       openshot-audio-demo


DESCRIPTION
       Display  information about detected audio interfaces, then play a test sound.  Uses the libopenshot-audio
       library.

       More information
               Project Website: http://www.openshot.org/
