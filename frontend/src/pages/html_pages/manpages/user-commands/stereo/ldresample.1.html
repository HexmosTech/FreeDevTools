NAME
       ldresample - MAME laserdisc audio manipulation tool


SYNOPSIS
       ldresample source.chd
       ldresample source.chd output.chd offset [slope]


DESCRIPTION
       Assists in resynchronizing audio tracks in a CHD (Compressed Hunks of Data) file with frames.

       Offset  parameter and slope option make a linear equation f(x) which describes the sample offset from the
       source as a function of field number.
