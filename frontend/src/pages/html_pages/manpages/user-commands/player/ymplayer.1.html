NAME
       ymplayer - a curses based player for the YM chiptune format.


DESCRIPTION
       STYMulator is an Open Source (GPL License) player which plays music files in the YM chiptune format.
       In  particular  the  YM  files  are Yamaha YM2149 soundchip music datas (registers) dumped from the ATARI
       16-bit computers.

       ymplayer (part of STYMulator) is a console based player for this format. It use ALSA for sound  rendering
       and ncurses library for its user interface.


SYNOPSIS
       ymplayer <input_ym_music_file>


OPTIONS
       ymplayer has no options.

       However it requires a <input_ym_music_file> argument which is a relative or absolute path to one YM music
       file.
