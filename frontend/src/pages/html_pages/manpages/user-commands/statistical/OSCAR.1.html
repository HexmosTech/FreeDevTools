NAME
       OSCAR - Open Source CPAP Analysis Reporter


DESCRIPTION
       OSCAR  is  a  Free Software, cross platform research tool for exploring data produced by CPAP (Continuous
       Positive Airway Pressure) machines, and related equipment, which are used in the treatment of sleep apnea
       and other sleep disorders.


OPTIONS
       This program does not accept command-line options.


BUGS
