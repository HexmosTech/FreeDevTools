NAME
       fid - List identifiers that occur in FILENAME


SYNOPSIS
       fid [OPTION] FILENAME [FILENAME2]


DESCRIPTION
       List identifiers that occur in FILENAME, or if FILENAME2 is also given list the identifiers that occur in
       both files.

       -f, --file=FILE
              file name of ID database

       --help display this help and exit

       --version
              output version information and exit
