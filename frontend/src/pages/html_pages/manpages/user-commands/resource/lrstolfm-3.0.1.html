NAME
       lrstolfm - The Lazarus resource Translator.


SYNOPSIS
       lrstolfm resourcefilename [resourcename]


DESCRIPTION
       lrstolfm Creates an LFM file from resources for a Lazarus project or package (LRS).


OPTIONS
       No option. If called without any argument it gives a short help message.
