NAME
       bt2vrml - convert from voxels to VRML2.0


SYNOPSIS
       bt2vrml [ <input.bt> ]


DESCRIPTION
       bt2vrml  This  tool  will convert the occupied voxels of a binary OctoMap file input.bt to a VRML2.0 file
       input.bt.wrl.

       WARNING: The output files will be quite large!
