NAME
       adwatchd-service - Manages the adwatchd service


SYNOPSIS
       adwatchd service COMMAND [flags]


DESCRIPTION
       The  service  command  allows the user to interact with the adwatchd service. It can manage and query the
       service status, and also install and uninstall the service.


OPTIONS
       -h, --help[=false]      help for service


OPTIONS INHERITED FROM PARENT COMMANDS
       -v, --verbose[=0]      issue INFO (-v), DEBUG (-vv) or DEBUG with caller (-vvv) output
