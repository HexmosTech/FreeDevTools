NAME
       xx - CLI client for the F*EX service (send, list, delete)


SYNOPSIS
       xx args


DESCRIPTION
       F*EX (Frams' Fast File EXchange) is a service to send big (large, huge, giant, ...) files from a user A
       to a user B.  This manual page documents briefly the xx command.

       This manual page was written for the Debian distribution because the original program does not have a
       manual page.


USAGE
       xx [:slot] file...
              send file(s)

       xx [:slot] -
              send STDIN

        ... | xx [:slot]
              send pipe

       xx [:slot]
              get file(s) or STDIN

       xx [:slot] --
              get file(s), no questions


EXAMPLES
       dmesg | xx
       xx project
       xx --
       xx :conf /etc /boot
