NAME
       reapr-seqrename - renames all sequences in a BAM file


SYNOPSIS
       reapr seqrename <renamefile> <in.bam> <out.bam>


DESCRIPTION
       <renamefile> is the file .info made by facheck, which contains the mapping of old name to new name.
