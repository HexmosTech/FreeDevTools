NAME
       upscale_relperm_benchmark - Benchmark version of upscale_relperm.


SYNOPSIS
       upscale_relperm_benchmark [option value] [grid file]


DESCRIPTION
       This  is a benchmark version of upscale_relperm. Input data (grid file, rock file and reference solution)
       is not provided from command line, but is built in at compiler time by  embedding  hexadecimal  (1  byte)
       input data files. No command line options are supported, and no output of data is given.

       Example usage:

       upscale_relperm_benchmark
