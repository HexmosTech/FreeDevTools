NAME
       organize-check - Checks whether a given config file is valid.


SYNOPSIS
       organize check [OPTIONS] [CONFIG]


DESCRIPTION
       Checks whether a given config file is valid.

           If called without arguments it will check the default config file.


OPTIONS
       --debug
              Verbose output
