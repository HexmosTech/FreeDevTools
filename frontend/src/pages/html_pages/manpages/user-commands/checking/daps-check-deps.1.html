NAME
       daps-check-deps - A program in the DAPS software package, a complete environment to build HTML, PDF, ePUB
       and other formats from DocBook XML.


DESCRIPTION
       This manual page documents briefly the daps-check-deps program.

       The daps-check-deps program checks the local system for the presence or absence of a number of
       dependencies of the DAPS software package. Simply run the program and it will show the results of the
       discovery in the form of
