NAME
       guider - Comprehensive Performance Analyzer


SYNOPSIS
       guider [--help] COMMAND [OPTIONS]


FEATURES
       Guider  provides  you  with everything you need to measure, analyze, and verify your system's performance
       with features like monitoring, profiling, tracing, visualization, control, logging, networking,  testing,
       and utilities,


EXAMPLES
       guider help
              Print guider command list

       guider COMMAND -h
              Print options and examples of COMMAND

       guider top -a
              Print resource usage on system

       guider top -a -o guider.out
              Collect resource usage on system and report it to guider.out file
