NAME
       gdm-screenshot - take a picture of the screen


SYNOPSIS
       gdm-screenshot [OPTION...]


DESCRIPTION
       gdm-screenshot is a utility for taking screenshots of the entire screen and saving them as PNG image
       files.


OPTIONS
       Help Options:

       -h, --help
           Show help options

       --help-all
           Show all help options

       --help-gtk
           Show GTK+ Options

       Application Options:

       -d, --debug
           Debugging output

       --display=DISPLAY
           X display to use


SEE ALSO
       gdm3(8).
