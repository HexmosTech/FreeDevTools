NAME
       mi2svg  -  a  tool  for  creating  svg (scalable vector graphics) presentations of maps stored in mapinfo
       interchange format.


SYNOPSIS
       mi2svg [OPTION] [FILEPREFIX[.mif]]...


DESCRIPTION
       Converts FILEPREFIX.mif map to FILEPREFIX.svg map.  Includes data fields from FILEPREFIX.mid  to  SVG  as
       class names.  Accepts multiple input files at once.

       -f     force owerwriting of existing targetfiles
