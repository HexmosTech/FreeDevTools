NAME
       synthv1 - an old-school polyphonic synthesizer


SYNOPSIS
       synthv1_jack [options] [preset-file]


DESCRIPTION
       This manual page documents briefly the synthv1_jack command.

       synthv1 is an old-school 4-oscillator subtractive polyphonic synthesizer with stereo fx.

       features:

              a  pure  stand-alone  JACK  client  with  JACK-session, NSM and both JACK MIDI and ALSA MIDI input
              support;

              a LV2 instrument plugin.  URI: http://synthv1.sourceforge.net/lv2


OPTIONS

       -g, --no-gui

              Disable the graphical user interface (GUI)

       -n, --client-name=[label]

              Set the JACK client name (default: synthv1)

       -h, --help

              Show help about command line options

       -v, --version

              Show version information


FILES
       Configuration settings are stored in ~/.config/rncbc.org/synthv1.conf
