NAME
       fmtave - Averages raw measurement data


SYNOPSIS
       fmtave [FILE]


DESCRIPTION
       fmtave is one of suite of command line tools designed to aid frequency measurement. These tools are aimed
       at those taking part in the ARRL FMT, but are also useful for general frequency measurement and
       calibration. They are included within the WSJT-X package for use with the FreqCal mode of WSJT-X.

       The input FILE is normally the output of a WSJT-X FreqCal run or the fmt tool (not bundled with WSJT-X).
       Results are saved in fmtave.out.and are suitable for entry to the ARRL Frequency Measuring Test (FMT).

       Refer to https://physics.princeton.edu/pulsar/k1jt/FMT_User.pdf for detailed instructions.
