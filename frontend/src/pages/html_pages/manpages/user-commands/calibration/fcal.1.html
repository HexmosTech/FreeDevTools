NAME
       fcal - Calculates a best-fit straight line for calibration data


SYNOPSIS
       fcal [FILE]


DESCRIPTION
       fcal is one of suite of command line tools designed to aid frequency measurement. These tools are aimed
       at those taking part in the ARRL FMT, but are also useful for general frequency measurement and
       calibration. They are included within the WSJT-X package for use with the FreqCal mode of WSJT-X.

       The input FILE is normally the output of the fmtavg(1) tool, after adjusting to remove bad sources.

       Refer to https://physics.princeton.edu/pulsar/k1jt/FMT_User.pdf for detailed instructions.
