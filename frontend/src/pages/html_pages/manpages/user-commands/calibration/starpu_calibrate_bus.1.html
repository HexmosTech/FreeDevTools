NAME
       starpu_calibrate_bus - Force StarPU bus calibration


SYNOPSIS
       starpu_calibrate_bus [OPTION]


DESCRIPTION
       Force a bus calibration.


OPTIONS
       -h, --help
              display this help and exit

       -v, --version
              output version information and exit
