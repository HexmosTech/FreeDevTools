NAME
       pvsecret-create-meta - Create a meta secret


SYNOPSIS
       pvsecret create meta


DESCRIPTION
       Use a meta secret to carry flags to the ultravisor without having to provide an actual secret value. Meta
       secrets do not appear in the list of secrets.


SEE ALSO
       pvsecret(1) pvsecret-create(1)
