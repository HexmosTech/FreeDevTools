NAME
       prr-status - Print a status summary of all known reviews


SYNOPSIS
       prr status [-n|--no-titles] [-h|--help]


DESCRIPTION
       Print a status summary of all known reviews


OPTIONS
       -n, --no-titles
              Hide column titles from output

       -h, --help
              Print help
