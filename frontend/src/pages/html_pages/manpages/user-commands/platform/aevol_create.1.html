NAME
       Aevol - an in silico experimental evolution platform


SYNOPSIS
       aevol_create -h or --help
       aevol_create -V or --version
       aevol_create [-f PARAM_FILE] [-C CHROM_FILE] [-p PLASMID_FILE]


DESCRIPTION
       Aevol  is  a  simulation  platform  that  allows  one  to  let populations of digital organisms evolve in
       different conditions and study experimentally the mechanisms responsible for  the  structuration  of  the
       genome and the transcriptome.

       aevol_create
              create an experiment with setup as specified in PARAM_FILE


OPTIONS
       -h, --help
              print help, then exit

       -V, --version
              print version number, then exit

       -f, --file PARAM_FILE
              specify parameter file (default: param.in)

       -C, --chromosome CHROM_FILE
              load chromosome from given text file instead of generating it

       -P, --plasmid PLASMID_FILE
              load plasmid from given text file instead of generating it
