NAME
       cutest - run tests in separate processes


SYNOPSIS
       cutest [OPTION]... [FILE]...


DESCRIPTION
       cutest run tests in separate processes to avoid shared state.

       -r   required libraries or helpers

       -v   version


USAGE
       EXAMPLES:

       cutest -r test/helper.rb test/*_test.rb
