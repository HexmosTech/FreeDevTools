NAME
       tradis_gene_insert_sites - Generate insertion site details from TraDIS pipeline plots


VERSION
       version 1.4.5


SYNOPSIS
DESCRIPTION
       This script is for TRADIS analysis. It takes in a plot file created by the TRADIS pathogen informatics
       pipeline, and an embl file of annotation.  It then outputs a tab delimited file with the insert site
       details. This is then used as input to an R script to calculate essentialit y.
