NAME
       shepherd - manual page for shepherd 1.0.2


DESCRIPTION
       shepherd [OPTIONS...]  This is a service manager for Unix and GNU.

       -s, --socket=FILE
              get commands from socket FILE or from stdin (-)

       -c, --config=FILE
              read configuration from FILE

       --pid[=FILE]
              when ready, write PID to FILE or stdout

       -l, --logfile=FILE
              log actions in FILE

       -I, --insecure
              don't ensure that the setup is secure

       -S, --silent
              don't do output to stdout

       --quiet
              synonym for --silent

       --help display this help and exit

       --usage
              display short usage message and exit

       --version
              display version information and exit

       Mandatory or optional arguments to long options are also mandatory or optional to the corresponding short
       options.
