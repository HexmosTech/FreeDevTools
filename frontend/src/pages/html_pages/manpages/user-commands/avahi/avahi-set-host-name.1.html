NAME
       avahi-set-host-name - Change mDNS host name


SYNOPSIS
       avahi-set-host-name host-name


DESCRIPTION
       Set  the  mDNS  host  name  of  a  currently  running  Avahi  daemon. The effect of this operation is not
       persistent across daemon restarts. This operation is usually privileged.


OPTIONS
       -v | --verbose
              Enable verbose mode.

       -h | --help
              Show help.

       -V | --version
              Show version information.
