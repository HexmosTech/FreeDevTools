NAME
       thesaurus2htmls - generates a HTML site (one file for each term)


SYNOPSIS
        thesaurus2htmls [-dir=D] thesaurus [lang*]


DESCRIPTION
       For each term in each language passed as argument, creates a linked HTML page.
