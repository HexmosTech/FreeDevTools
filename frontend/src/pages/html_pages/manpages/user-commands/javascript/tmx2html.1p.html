NAME
       tmx2html - Converts a TMX to an HTML formatted page


VERSION
       version 0.39


SYNOPSIS
          tmx2html [-icons] file.tmx > file.html
             -o=out-html      -- def: __.html
             -v               -- show with firefox
             -v=chromium      -- show with chromium


DESCRIPTION
       Pretty prints a TMX as an HTML file. Icons for languages can be produced, but still buggy.
