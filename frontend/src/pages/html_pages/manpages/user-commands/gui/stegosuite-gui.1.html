NAME
       stegosuite-gui - Starts the GUI


SYNOPSIS
       stegosuite gui [-d] [<image>]


DESCRIPTION
       This command starts the GUI, which is written using SWT.


OPTIONS
       -d, --debug
           Shows debug information.


ARGUMENTS
       [<image>]
           The image file to process.
