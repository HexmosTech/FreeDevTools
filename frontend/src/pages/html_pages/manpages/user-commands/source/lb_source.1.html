NAME
       lb source - Complete the source stage


SYNOPSIS
       lb source [live-build options]


DESCRIPTION
       lb source is a high-level command (porcelain) of live-build(7), the Debian Live tool suite.

       lb source calls all necessary live-build programs in the correct order to complete the source stage.


OPTIONS
       lb source has no specific options but understands all generic live-build options. See live-build(7) for a
       complete list of all generic live-build options.


FILES
       none


SEE ALSO
       live-build(7)

       This program is a part of live-build.
