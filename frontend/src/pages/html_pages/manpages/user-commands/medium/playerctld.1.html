NAME
       playerctld - maintain an ordered list of controllable media players for playerctl


SYNOPSIS
       playerctld [OPTION] COMMAND

       playerctld --help


DESCRIPTION
       playerctld  runs  as a user-controlled daemon, monitoring controllable media players for playerctl in the
       order that they are started.  playerctld allows a user to navigate forwards  and  backwards  through  the
       list of available media players using the shift and unshift commands.


COMMANDS
       daemon start playerctld as a daemon

       shift  shift to next controllable player

       unshift
              unshift to previous controllable player


OPTIONS
       -h, --help
              show usage details and exit
