NAME
       marisa-predictive-search - A tool to test predictive search


SYNOPSIS
       marisa-predictive-search [options] file


DESCRIPTION
       marisa-predictive-search  is  a  tool  to test predictive search. This tool searches keys starting with a
       query string and then prints the first m keys, where m is one of the parameters.

       See marisa-predictive-search -h for the list of options.
