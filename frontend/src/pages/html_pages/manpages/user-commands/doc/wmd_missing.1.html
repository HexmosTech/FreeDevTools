NAME
       WML Missing Doc - Requested Program Not Installed


DESCRIPTION
       WML is a bunch of programs, some are essential (like the 9 passes) but other are optional.  If you are
       reading this manpage, this means that you asked WMd to display the documentation of an optional program
       which is not installed.
