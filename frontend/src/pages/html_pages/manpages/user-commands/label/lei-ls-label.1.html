NAME
       lei-ls-label - list labels


SYNOPSIS
       lei ls-label [OPTIONS]


DESCRIPTION
       List all known message labels ("mailboxes" in JMAP terminology).  This is handy for writing lei-import(1)
       invocations.


OPTIONS
       -z
       -0  Use "\0" (NUL) instead of newline (CR) to delimit lines.

       -q
       --quiet
           Suppress feedback messages.
