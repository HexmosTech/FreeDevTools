NAME
       fa2dna - format fasta database for use with ANFO


SYNOPSIS
       file-info [ file ... ]


DESCRIPTION
       file-info  reads  one or more files in various formats that anfo(1) understands, and for each file prints
       to stdout

       •   the type of file if it was identified,

       •   a textual form of the configuration, if a configuration is contained in it,

       •   the substitution matrix and other alignment parameters, if an aligner configuration was contained.
