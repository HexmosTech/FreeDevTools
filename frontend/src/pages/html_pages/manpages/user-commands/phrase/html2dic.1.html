NAME
       html2dic - Dump word dictionary from html input file


SYNOPSIS
       html2dic <file>


DESCRIPTION
       html2dic  extract  all  words  from an HTML page, generating a dictionary of all word found, one word per
       line.  Output is printed on stdout.


SEE ALSO
       dirb(1),dirb-gendict(1)
