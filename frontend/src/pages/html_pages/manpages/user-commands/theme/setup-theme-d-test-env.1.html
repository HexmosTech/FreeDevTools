NAME
       setup-theme-d-test-env - install the Theme-D testing environment


SYNOPSIS
       setup-theme-d-test-env


DESCRIPTION
       Command setup-theme-d-test-env Installs the testing environment into the subdirectory theme-d-test-env of
       the current directory. If the subdirectory theme-d-test-env exists already nothing is done.


ENVIRONMENT
       THEME_D_CONFIG_FILE
              If  this  variable  is  defined its value is used as the Theme-D configuration file instead of the
              default configuration file.


FILES
       theme-d-test-env/*
              The installed Theme-D testing environment.

       /etc/theme-d-config
              The Theme-D configuration file.

       ~/.theme-d-config
              The Theme-D configuration file. This file should be normally present only if you  use  Theme-D  in
              local mode.
