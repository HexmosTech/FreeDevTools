NAME
       msd-datetime-mechanism - set system time via D-Bus


SYNOPSIS
       msd-datetime-mechanism


DESCRIPTION
       The msd-datetime-mechanism is used for setting the system time via D-Bus.  msd-datetime-mechanism is a D-
       Bus  wrapper  that gets executed via D-Bus with root privileges.  The execution of msd-datetime-mechanism
       gets triggered via the MATE clock applet.
