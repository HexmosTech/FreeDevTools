NAME
       hmtMapViewer - Robotic Hybrid Metric Topological (HMT) map viewer


SYNOPSIS
       hmtMapViewer


DESCRIPTION
       hmtMapViewer is a GUI application that allows loading and visualizing complex HMT maps for mobile
       robotics applications.


BUGS
