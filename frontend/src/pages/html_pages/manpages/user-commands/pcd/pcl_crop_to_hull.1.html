NAME
       pcl_crop_to_hull - pcl_crop_to_hull


DESCRIPTION
       Syntax is: pcl_crop_to_hull hull_cloud.pcd input.pcd output.pcd <options>

       Filter  a  point  cloud  using  the  convex  hull  of  another  point  cloud.  For more information, use:
       pcl_crop_to_hull -h

         where options are:

        -alpha X = the hull alpha value (0+) (default: 1000.000000)
