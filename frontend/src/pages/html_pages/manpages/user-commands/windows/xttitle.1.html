NAME
       xttitle - change window and icon title for Xterm windows


SYNOPSIS
       xttitle <window title> [ <icon title> ]


DESCRIPTION
       The  xttitle  program  is  simple utility that allows you to change the window title and icon name for an
       xterm window.
