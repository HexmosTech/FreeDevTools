NAME
       ukui-window-switch - Front of the window switcher in UKUI desktop environment


SYNOPSIS
       ukui-window-switch


DESCRIPTION
       Front  of  the window switcher in UKUI desktop environment.  Provides the display function(Display window
       thumbnails and application icons) when the window is switching (press Alt+Tab key).
