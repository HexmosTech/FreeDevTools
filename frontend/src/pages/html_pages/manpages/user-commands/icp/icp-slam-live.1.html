NAME
       icp-slam-live - ICP-based SLAM using scan matching with a LIDAR in real-time


SYNOPSIS
       icp-slam-live config_file.ini


DESCRIPTION
       icp-slam-live is a command-line application which connects to any MRPT-supported LIDAR and creates a map
       using simple scan matching.
