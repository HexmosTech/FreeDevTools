NAME
       merge_swath - merge TOPSAR swath


DESCRIPTION
       This program is part of GMTSAR.

       USAGE: merge_swath inputlist output [stem] [n1 n2]

       inputlist example:
         F1/intf/2015036_2015060/S1A_20150609.PRMF1/intf/2015036_2015060/phasefilt.grd
         F2/intf/2015036_2015060/S1A_20150609.PRMF2/intf/2015036_2015060/phasefilt.grd
         F3/intf/2015036_2015060/S1A_20150609.PRMF3/intf/2015036_2015060/phasefilt.grd

       note: use the aligned PRM which contains the shift information

       output: output.grd [stem.PRM]

       note: please put the files to stem.in in the order of swath numbers.

              make sure all images have same num_rng_bin

              the n1 and n2 will determine where to stitch, they have to be

              pairs when assigned. In case you have only two, use 0 for n2
