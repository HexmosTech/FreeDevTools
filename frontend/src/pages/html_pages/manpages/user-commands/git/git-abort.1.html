NAME
       git-abort - Abort current git operation


SYNOPSIS
       git-abort


DESCRIPTION
       Abort current git revert, rebase, merge or cherry-pick process.


OPTIONS
       There are no options, it just aborts current operation.


EXAMPLES
       git-abort
