NAME
       kvm - kvm-enabling link for qemu-system-x86_64


DESCRIPTION
       When executed as kvm, qemu-system is run with preference to native hardware-based virtualization, instead
       of   relying  solely  on  emulation.   Essentially  kvm  is  equivalent  to  qemu-system-x86_64  -machine
       accel=kvm:tcg so when the host CPU support the kvm mode, native hardware virtualization  is  enabled,  or
       qemu-system-x86_64 falls back to the emulation (TCG) mode.
