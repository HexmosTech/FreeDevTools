NAME
       pcl_pcd_change_viewpoint - pcl_pcd_change_viewpoint


DESCRIPTION
       Syntax is: pcl_pcd_change_viewpoint input.pcd output.pcd <options>

       Change viewpoint information in a PCD file. For more information, use: pcl_pcd_change_viewpoint -h

         where options are:

        -viewpoint Tx,Ty,Tz,Qw,Qx,Qy,Qz
