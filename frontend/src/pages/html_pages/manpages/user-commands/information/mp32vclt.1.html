NAME
       mp32vclt - Extract time and signal information from ID3/MP3 files


SYNOPSIS
       mp32vclt [--no-del] files...


DESCRIPTION
       This  tool  extracts  time  and  signal  information  from  ID3/MP3  files.   This is for internal use of
       dir2vclt(1) only.


OPTIONS
       --no-del
              Do not print entry separator.
