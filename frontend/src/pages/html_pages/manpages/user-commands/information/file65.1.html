NAME
       file65 - print information for o65 object files


SYNOPSIS
       file65 [OPTION]... FILE...


DESCRIPTION
       file65 prints file information for files in the o65 object format.


OPTIONS
       -v     Print undefined and global labels.

       -vv    Print undefined and global labels, and relocation tables.

       -P     Print the segment end addresses (suitable for the xa(1) command line parameter -b).

       -a offset
              Print  xa(1) "ROMmable" parameter for another file behind this one in the same ROM, located at the
              specified offset.

       -A offset
              Does the same thing as -a but only prints the starting address of the next file in the ROM.

       --help Show summary of options.

       --version
              Show version of program.
