NAME
       pcl_obj2vtk - pcl_obj2vtk


DESCRIPTION
       Syntax is: pcl_obj2vtk input.obj output.vtk

       Convert a OBJ file to VTK format. For more information, use: pcl_obj2vtk -h


AUTHOR
       pcl_obj2vtk is part of Point Cloud Library (PCL) - www.pointclouds.org

       The  Point Cloud Library (PCL) is a standalone, large scale, open project for 2D/3D image and point cloud
       processing.
