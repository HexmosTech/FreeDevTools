NAME
       pcl_vtk2ply - pcl_vtk2ply


DESCRIPTION
       Syntax is: pcl_vtk2ply input.vtk output.ply

       Convert a VTK file to PLY format. For more information, use: pcl_vtk2ply -h


AUTHOR
