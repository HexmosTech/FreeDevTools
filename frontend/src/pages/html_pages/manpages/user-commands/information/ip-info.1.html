NAME
       ip-info - query whois, AS, DNSBL and other network data in one go


SYNOPSIS
       ip-info <IP address>


DESCRIPTION
       ip-info  is  a script to query whois, AS, peering and DNSBL entries for a given IP (IPv4 or IPv6) address
       and especially helpful if you're tracking down external network abusers.

       It's the example script for the Perl  module  Net::Abuse::Utils,  i.e.  all  its  functionality  is  also
       available as a Perl library.


SEE ALSO
        Net::Abuse::Utils(3pm), Net::Whois::IP(3pm)
