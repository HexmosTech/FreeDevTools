NAME
       obchiral — print molecular chirality information


SYNOPSIS
       obchiral filename


DESCRIPTION
       The  obchiral  program  is a tool to print the chirality information for all molecules in a file. It also
       serves as example code for using the Open Babel library (libopenbabel).


EXAMPLES
             obchiral pyridines.sdf


SEE ALSO
       obabel(1), obfit(1), obgrep(1), obprop(1), obrotate(1).
       The web pages for Open Babel can be found at: <http://openbabel.org/>
