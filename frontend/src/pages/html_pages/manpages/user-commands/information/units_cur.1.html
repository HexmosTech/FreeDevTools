NAME
       units_cur - update currency information for units(1)


SYNOPSIS
       units_cur [<file>]


DESCRIPTION
       units_cur updates the currency information and precious metal prices for units.


OPTIONS
       If  a  file  is  specified  as  argument, the updated information will be written there instead of in the
       default file (see below).  Specifying - causes the updated information to be written to standard output.


FILES
       /usr/share/units/currency.units
              Default currency information.


SEE ALSO
       units(1)
