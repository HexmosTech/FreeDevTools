NAME
       nvddsinfo -Outputs information about the given DDS file.


SYNOPSIS
       nvddsinfo FILE


DESCRIPTION
       nvddsinfo is part of the NVIDIA texture toolset. The tool shows available information about the DDS image
       file.


INPUT OPTIONS
