NAME
       metaflac2time - Extract time information from metaflac output


SYNOPSIS
       metaflac2time


DESCRIPTION
       This tool extracts time information from metaflac output.  This is for internal use of dir2vclt(1) only.

       No options are supported. Reading is always done on stdin, writing is always done on stdout.


OPTIONS
       This program does not take any options.
