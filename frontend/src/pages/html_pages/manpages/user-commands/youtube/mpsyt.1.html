NAME
       mps-youtube - Terminal based YouTube jukebox with playlist management


DESCRIPTION
       This  manual  page documents briefly the mps-yotube software which is based on mps, and uses YouTube as a
       source of content and can play and download video as well as audio.

       mps-youtube has an excellent built-in help. Once you are inside the application just type  help  and  hit
       Enter.
