NAME
       gcm-import - GNOME Color Manager Import Tool


SYNOPSIS
       gcm-import [ --verbose ]


DESCRIPTION
       This manual page documents briefly the gcm-import command.

       gcm-import allows you to import ICC files supplied by vendors.
