NAME
       fiximports - fixes import declarations to use the canonical import path


SYNOPSIS
       fiximports


DESCRIPTION
       The  fiximports command fixes import declarations to use the canonical import path for packages that have
       an "import comment" as defined by https://golang.org/s/go14customimport.

       Run fiximports -help for more details.
