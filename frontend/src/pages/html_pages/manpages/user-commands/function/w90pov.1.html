NAME
       w90pov - create PovRay input files to visualise Wannier functions


SYNOPSIS
       w90pov w90pov.inp


OPTIONS
       w90pov.inp
               Input file. The format of input file is described in
               /usr/share/doc/wannier90-data/w90pov.pdf.gz of
               'wannier90-data' package.

                                                                                                       w90pov(1)
