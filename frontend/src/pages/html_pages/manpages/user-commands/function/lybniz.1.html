NAME
       lybniz - mathematical function graph plotter


DESCRIPTION
       Lybniz  is  a  simple  desktop  graph  plotter.  It  can currently plot three functions and allows you to
       navigate the plot. Functions are entered in Python notation, eg sin(x)/x ; 2*x**2 + 3*x - 4.
