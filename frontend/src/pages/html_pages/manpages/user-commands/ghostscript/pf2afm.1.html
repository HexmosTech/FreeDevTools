NAME
       pf2afm - Make an AFM file from Postscript (PFB/PFA/PFM) font files using ghostscript


SYNOPSIS
       pf2afm  fontfilename


DESCRIPTION
       This  script invokes gs(1) to make an AFM file from PFB / PFA and (optionally) PFM files.  Output goes to
       fontfilename.afm, which must not already exist.


SEE ALSO
       gs(1)
       pf2afm.ps in the Ghostscript lib directory.
