NAME
       change_paired_read_orientation - Reverses the orientation of each paired-end FASTA sequence


DESCRIPTION
   Usage:
              change_paired_read_orientation   <in_fasta>   <out_fasta>   change_paired_read_orientation  --help
              change_paired_read_orientation        --man         change_paired_read_orientation         --usage
              change_paired_read_orientation --version

   Required arguments:
              <in_fasta>

              FASTA file containing the reads to re-orient.

              <out_fasta>

              Output FASTA file where to write the reads.
