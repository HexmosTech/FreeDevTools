NAME
       nacl-sha512 — program to calculate the sha512 hash


SYNOPSIS
       nacl-sha512


DESCRIPTION
       This manual page documents briefly the nacl-sha512 command.

       This  manual  page  was  written for the Debian distribution because the original program does not have a
       manual page.  Instead, it has documentation in the GNU.

       nacl-sha512 is a program that uses NaCl to calculate the sha512 hash.


OPTIONS
       These program expects input on standard input and prints the sha512 hash on standard output.
