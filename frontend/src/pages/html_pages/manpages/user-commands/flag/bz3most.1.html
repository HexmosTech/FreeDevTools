NAME
       bz3most - view bzip3-compressed files


SYNOPSIS
       bz3most [flags] [file ...]


DESCRIPTION
       bz3most  runs  most(1)  on  files  or stdin, if no files argument is given, after decompressing them with
       bz3cat(1).


SEE ALSO
       bzip3(1), most(1)
