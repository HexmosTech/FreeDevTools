NAME
       lei-forget-search - forget saved search


SYNOPSIS
       lei forget-search [OPTIONS] OUTPUT


DESCRIPTION
       Forget a saved search at "OUTPUT".


OPTIONS
       --prune[=<local|remote>]
           "--prune"  will  forget saved searches if the "OUTPUT" no longer exists.  "--prune=local" only prunes
           local mailboxes, "--prune=remote" only prunes remote mailboxes (currently "imap://" and "imaps://").
