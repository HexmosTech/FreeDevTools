NAME
       search-ccsb


SYNOPSIS
       search-ccsb <expression>


DESCRIPTION
       This Perl script helps searching the "The Collection of Computer Science Bibliographies" provided by Alf-
       Christian  Achilles <achilles@ira.uka.de>. With it you can perform a query without browser, directly from
       the command line.


EXAMPLE
       search-ccsb "operating systems"
