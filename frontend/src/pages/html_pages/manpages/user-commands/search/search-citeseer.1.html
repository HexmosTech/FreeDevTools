NAME
       search-citeseer - search BibTex entries


SYNOPSIS
       search-citeseer <expression>


DESCRIPTION
       This Perl script helps searching Citeseer (http://citeseer.nj.nec.com/cs) for BibTeX entries. With it you
       can perform a query without browser, directly from the command line.


EXAMPLE
       search-citeseer "operating systems"
