NAME
       3d-rotation-converter - A GUI tool to visualize SO(3) rotations


SYNOPSIS
       3d-rotation-converter


DESCRIPTION
       3d-rotation-converter is a GUI application to help visualizing relative 3D poses (the mathematical group
       SO(3)) and to convert between different common parameterizations of rotations.
