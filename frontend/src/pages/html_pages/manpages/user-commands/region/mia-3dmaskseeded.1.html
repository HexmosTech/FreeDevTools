NAME
       mia-3dmaskseeded - Mask an area by seeded region growing


SYNOPSIS
       mia-3dmaskseeded -i <in-file> -o <out-file> [options]


DESCRIPTION
       mia-3dmaskseeded  3D image segmentation based on region growing from a seed point. Neighboring points are
       added, either when their intensity is equal or higher than that of the seed point, or of the intensity is
       lower or equal then that of the neighoring point. After region growing is finished, this mask is used  to
       zero out the region in the original image yielding the resulting image.


OPTIONS
              -i --in-file=(input, required); io
                     input image(s) to be filtered
                      For supported file types see PLUGINS:3dimage/io

              -o --out-file=(required, output); io
                     output image(s) that have been filtered
                      For supported file types see PLUGINS:3dimage/io

              -s --seed=[0,0,0]
                     seed point

              -n --neighborhood=6n
                     neighborhood shape
                      For supported plugins see PLUGINS:3dimage/shape

EXAMPLE
       Run a region growing on input.v starting at point <10,23,21> and use the 18n neighbourhood.

       mia-3dmaskseeded -i image.v -o masked.v -s '<10,23,21>' -n 18n
