NAME
       gammaReadfiles.csh - import form Gamma format


DESCRIPTION
              USAGE:gammaReadfiles.csh processParameterFile sensorParameterFile slcFile

              where  processParameterFile  is  the gamma SLC processing  par file (pXXX.slc.par, or XXX.pslcpar)
              sensorParameterFile is the gamma sensor par file (ERS1_ESA.par,  or  System.par)  slcfile  is  the
              complex SAR image file in 2 bytes integer format (CI2).

              EXAMPLE:

              gammaReadfiles.csh p20070105.slc.par System.par 20070105.slc

              where  processParameterFile  is  the gamma SLC processing  par file (pXXX.slc.par, or XXX.pslcpar)
              sensorParameterFile is the gamma sensor par file (ERS1_ESA.par,  or  System.par)  slcfile  is  the
              complex SAR image file in 2 bytes integer format (CI2).

              EXAMPLE:

              gammaReadfiles.csh p20070105.slc.par System.par 20070105.slc
