NAME
       darknet - program to run darknet function


SYNOPSIS

       darknet function


DESCRIPTION
       This manual page documents briefly the darknet command.

       This manual page was written for the Debian distribution because the original program does not have a
       manual page.

       darknet is a program that perform darknet functions


OPTIONS
       The program has several functions can be performed. A summary of funcitons is included below.

       detector action file_config network_config weights
           Run detector with assigned network
