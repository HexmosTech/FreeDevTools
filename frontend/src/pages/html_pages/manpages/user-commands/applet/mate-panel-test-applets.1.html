NAME
       mate-panel-test-applets - display and test installed applets


SYNOPSIS
       mate-panel-test-applets [OPTIONS]


DESCRIPTION
       mate-panel-test-applets display installed applets in a dialog box and allow you to test them.


OPTIONS
       --iid  Specify an applet IID to load

       --prefs-path
              Specify a gsettings path in which the applet preferences should be stored

       --size Specify the initial size of the applet (xx-small, medium, large etc.)

       --orient
              Specify the initial orientation of the applet (top, bottom, left or right)

       --display=DISPLAY
              X display to use.

       -?, -h, --help
              Print standard command line options.

       --help-all
              Print all command line options.

       This program also accepts the standard GTK options.
