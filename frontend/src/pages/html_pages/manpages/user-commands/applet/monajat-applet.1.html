NAME
       monajat-applet - Islamic supplications tray applet


SYNOPSIS
       monajat-applet


DESCRIPTION
       monajat-applet  is  an  Islamic  supplications  tray applet for GNOME (also works with KDE) that displays
       Islamic supplications (azkar) at predetermined times.
