NAME
       yuzu-room — manage yuzu network rooms


SYNOPSIS
       yuzu-room  -n  |  --room-name name -d | --room-description description -p | --port number -w | --password
                 password -g | --preferred-game name -i | --preferred-game-id id [-u |  --username  name]  -t  |
                 --token  token  -a  |  --web-api-url  url  -b  | --ban-list-file file -l | --log-file file -e |
                 --enable-yuzu-mods -h | --help -v | --version


DESCRIPTION
       yuzu-room allows you to set up networked game rooms to play with your friends.


EXIT STATUS
       The yuzu-room utility exits 0 on success, and >0 if an error occurs.
