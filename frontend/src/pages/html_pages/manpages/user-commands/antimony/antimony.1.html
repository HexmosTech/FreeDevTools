NAME
       antimony - CAD from a parallel universe


SYNOPSIS
       antimony [options]file


DESCRIPTION
       This manual page documents briefly the antimony command.


OPTIONS
       -h, --help
              Show summary of options.

       --heightmap
              Open 3D windows in heightmap mode

   Arguments:
       file   File to open
