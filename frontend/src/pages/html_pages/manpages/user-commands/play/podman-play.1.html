NAME
       podman-play - Play containers, pods or volumes based on a structured input file


SYNOPSIS
       podman play subcommand


DESCRIPTION
       The  play  command  will  recreate containers, pods or volumes based on the input from a structured (like
       YAML) file input.  Containers will be automatically started.
