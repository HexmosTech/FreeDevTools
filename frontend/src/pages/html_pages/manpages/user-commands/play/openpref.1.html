NAME
       openpref - card game against two virtual players


DESCRIPTION
       OpenPref  is  an  open  source implementation of European trick-taking game Preferans against two virtual
       players. Currently it allows one to play Sochi  version  of  Preferans,  allowing  to  change  some  game
       conventions.


SEE ALSO
       Website: http://openpref.sourceforge.net/
