NAME
       ftp-cp - download files from an FTP server (DEPRECATED)


SYNOPSIS
       ftp-cp (edirect -ftpcp) server dir [filename...]


DESCRIPTION
       Note: ftp-cp will not appear in future releases of NCBI Entrez Direct.  Instead, please use nquire -ftp.

       ftp-cp  downloads  one or more files from a given directory on a specified file server.  If no file names
       appear on the command line, it will read the list of names to use from standard input.
