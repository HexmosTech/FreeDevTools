NAME
       minc — incorporate new messages


SYNOPSIS
       minc [-q] dirs ...


DESCRIPTION
       minc  incorporates new messages in the maildir folders dirs by moving them from new to cur, and adjusting
       the filenames.

       By default, the new filenames are printed, separated by newlines.

       The options are as follows:

       -q      Quiet: don't print new filenames.


EXIT STATUS
       The minc utility exits 0 on success, and >0 if an error occurs.


SEE ALSO
       Using maildir format: https://cr.yp.to/proto/maildir.html
