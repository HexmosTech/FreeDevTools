NAME
       visual-regexp — program to visualise and debug regular expressions


SYNOPSIS
       visual-regexp


DESCRIPTION
       This manual page documents briefly the visual-regexp command.

       This  manual page was written for the Debian GNU/Linux distribution because the original program does not
       have a manual page.  It was considered too self explanatory for documentation.

       visual-regexp is a program that interactivly creates and shows the output from regular  expressions.   It
       is ideal for debugging compliated perl expressions.


OPTIONS
       These programs does not accept any options.
