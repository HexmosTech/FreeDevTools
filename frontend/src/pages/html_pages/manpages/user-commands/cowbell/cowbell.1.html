NAME
       cowbell - An easy-to-use tag editor for your music files


SYNOPSIS

       cowbell [OPTIONS] [FILE]


DESCRIPTION
       An easy-to-use tag editor for your music files

       Cowbell is an elegant music organizer intended to make keeping your collection tidy both fun and easy. It
       allows viewing and editing of the tags, guessing of tag information with the help of Amazon Web Services
       and has an easy-to-use interface.


OPTIONS
       -h, --help
           Shows help about options

       -v, --version
           Shows the current version and exits

       --debug
           Enables debugging output
