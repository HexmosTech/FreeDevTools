NAME
       pinot - A metasearch tool for the Free Desktop


SYNOPSIS
       pinot [OPTIONS]


DESCRIPTION
       pinot - Desktop search and metasearch


OPTIONS
       -h, --help
              display this help and exit

       -p, --preferences
              show preferences

       -q, --query
              open with query terms

       -v, --version
              output version information and exit
