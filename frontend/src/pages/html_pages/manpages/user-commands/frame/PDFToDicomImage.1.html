NAME
       PDFToDicomImage  -  Read  PDF file, and create a single or multi frame DICOM Secondary Capture image with
       one frame per page.


SYNOPSIS
       PDFToDicomImage  inputFile  outputFile  patientName   patientID   studyID   seriesNumber   instanceNumber
       requestedDpi


DESCRIPTION
       Error: Incorrect number of arguments


OPTIONS
       arg  -  six  parameters,  the  inputFile,  outputFile,  patientName,  patientID,  studyID,  seriesNumber,
       instanceNumber, dots per inch
