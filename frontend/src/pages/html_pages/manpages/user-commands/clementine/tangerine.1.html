NAME
       Tangerine - DAAP Server


SYNOPSIS
       tangerine [config]


DESCRIPTION
       Tangerine  is a DAAP server. You can use it to listen to music remotely through a number of applications,
       including iTunes, Banshee, and Rhythmbox.

       config is the configuration file for tangerine, and can be configured using the graphical tool tangerine-
       properties(1). It defaults to ~/.tangerine if not provided.
