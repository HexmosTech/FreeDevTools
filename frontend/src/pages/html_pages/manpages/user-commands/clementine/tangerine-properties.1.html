NAME
       Tangerine-Properties - DAAP Server configuration tool


SYNOPSIS
       tangerine-properties


DESCRIPTION
       tangerine-properties  provides  a  graphical  user  interface  for  configuring ~/.tangerine which is the
       configuration file for tangerien(1), and automatically launching or  terminating  tangerine(1)  based  on
       whether it is enabled in the configuration.
