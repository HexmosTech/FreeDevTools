NAME
       lp2 - print duplex on non duplex printers


SYNOPSIS
       lp2 [OPTIONS] FILE...


DESCRIPTION
       Print  a  file  duplex  on a simplex printer.n First print the odd pages, then prompt the user to put the
       printed pages back in, then print the even pages in reverse order.


OPTIONS
       -h, --help
              display this help and exit

       -V, --version
              display version information and exit

       -f, --front
              print only the front pages (recto)

       -b, --back
              print only the back pages (verso)
