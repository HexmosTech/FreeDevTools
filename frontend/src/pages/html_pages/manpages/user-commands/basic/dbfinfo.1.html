NAME
       dbfinfo - Displays basic information for a given xBase file

       ##SYNOPSIS dbfinfo xbase_file

       ##DESCRIPTION  Displays  basic  information  for  a  given  xBase file, like number of columns, number of
       records and type of each column.

       ##OPTIONS

       xbase_file
              The name of an existing xBase file.

       ##EXAMPLE dbfinfo testbase

           Info for testbase.dbf
           3 Columns,  1 Records in file
                      NAME          string  (20,0)
                      AREA           float  (9,3)
                     VALUE           float  (9,2)
