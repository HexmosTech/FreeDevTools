NAME
       surankco-prediction - prediction of scores and ranking of contigs using trained random forests and contig
       features


SYNOPSIS
       surankco-prediction [options]


DESCRIPTION
       Prediction  of scores and ranking of contigs using trained random forests created by surankco-training(1)
       and contig features from surankco-feature(1).


OPTIONS

       -f FEATURES, --features=FEATURES

              Indicate one surankco feature file, a correct suffixe is mandatory ("*.features.txt")

       -r RANDOM.FORESTS, --random.forests=RANDOM.FORESTS

              Indicate a surankco random forests file ("*.RData")

       -o OUTPUT.FILENAME, --output.filename=OUTPUT.FILENAME

              Indicate a name for the final results file

       -h, --help

              Show this help message and exit
