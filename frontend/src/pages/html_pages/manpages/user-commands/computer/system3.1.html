NAME
       system3 - simulate a s3 computer


SYNOPSIS
       system3


DESCRIPTION
       This  manual  page  documents  briefly  the system3 command.  This manual page was written for the Debian
       distribution because the original program does not have a manual page.  Instead, it has documentation  in
       various text files;  see below.

       This emulator was renamed in 2010 due to an executable name collision.  It was formerly known as s3


OPTIONS
       For a complete description, see the text files.
