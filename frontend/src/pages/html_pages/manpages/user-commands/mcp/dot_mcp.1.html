NAME
       dot_mcp - mcp for s3d


SYNOPSIS

       dot_mcp [--s3d-url url]


OPTIONS
       This program follows the usual GNU command line syntax. A summary of the options supported by dot_mcp is
       below.

       --s3d-url url
           skip S3D environment and connect to this url


DESCRIPTION
       dot_mcp(1) is a mcp (which is the window manager in s3d-terms) for s3d.
