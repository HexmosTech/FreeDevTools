NAME
       nbd-trplay - replay all or parts of an nbd transaction log


SYNOPSIS
       nbd-trplay


DESCRIPTION
       nbd-trplay  replays  all  or  parts  of  a  transaction  log  produced by nbd-server (specifically by the
       transactionlog configuration directive with the option datalog ).

       See nbd-trplay --help for the command line parameters.


OUTPUT
       The file updates the image provided via -i.


SEE ALSO
       nbd-server (1).
