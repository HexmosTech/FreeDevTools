NAME
       display_qt - a Gnu Radio Example gr-qtgui


DESCRIPTION
       Display a GUI using QT of a sine wave in noise.

       Example  program  instantiates  a  GNU Radio flow graph using a sine wave source, a noise source, and gr-
       qtgui blocks. This new (in version 3.7.10) example shows how to build a C++ only QT based application.
