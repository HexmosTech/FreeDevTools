NAME
       srvarv - serve an instance of the ataqv web results viewer


DESCRIPTION
       usage: srvarv [-h] [-p PORT] [instance]

       srvarv: serve an instance of the ataqv web results viewer

   positional arguments:
       instance
              The directory containing the viewer instance (default: current directory).

   options:
       -h, --help
              show this help message and exit

       -p PORT, --port PORT
              The TCP port on which to serve the viewer (default: 8000).
