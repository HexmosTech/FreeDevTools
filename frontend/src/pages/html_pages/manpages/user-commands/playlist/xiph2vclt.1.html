NAME
       xiph2vclt - Generate VCLT playlist from xiph/icecast stream directory


SYNOPSIS
       xiph2vclt [infile.xml]


DESCRIPTION
       This tool generates a VCLT playlist from Xiph foundation's icecast stream directory.

       If no infile is given 'yp.xml' is used.

       The playlist is print to stdout.


OPTIONS
       This program does not take any options.
