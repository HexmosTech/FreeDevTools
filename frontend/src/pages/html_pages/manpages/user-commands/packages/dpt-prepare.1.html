NAME
       dpt-prepare - prepare yourself and a source directory before working on a package


SYNOPSIS
       dpt prepare


DESCRIPTION
       dpt prepare runs a couple of commands in the checked out directory of a source package which are useful
       -- either for the person or the code -- as a preparation before working on the package.


CONFIGURATION, OPTIONS, & PARAMETERS
       None (yet).
