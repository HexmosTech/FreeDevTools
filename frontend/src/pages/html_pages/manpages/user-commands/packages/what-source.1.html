NAME
       what-source - determine what source package provides some binary package


SYNOPSIS
       what-source SOME_BINARY_PACKAGE


DESCRIPTION
       what-source(1) is a simple program that determines what source package provides some binary package

       It's a simple wrapper for 'apt-cache(1) show | grep'.


EXAMPLE
        $ what-source sl
        universe


SEE ALSO
       apt-cache(1)
