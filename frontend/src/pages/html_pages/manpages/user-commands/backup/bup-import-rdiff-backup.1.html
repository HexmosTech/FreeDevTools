NAME
       bup-import-rdiff-backup - import a rdiff-backup archive


SYNOPSIS
       bup import-rdiff-backup [-n]


DESCRIPTION
       bup import-rdiff-backup imports a rdiff-backup archive.  The timestamps for the backups are preserved and
       the path to the rdiff-backup archive is stripped from the paths.

       -n, --dry-run
