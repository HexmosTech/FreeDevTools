NAME
       restic-rebuild-index - Build a new index


SYNOPSIS
       restic rebuild-index [flags]


DESCRIPTION
       The "rebuild-index" command creates a new index based on the pack files in the repository.


EXIT STATUS
       Exit status is 0 if the command was successful, and non-zero if there was any error.


OPTIONS
       -h, --help[=false]      help for rebuild-index

       --read-all-packs[=false]      read all pack files to generate new index from scratch
