NAME
       localsearch-index - Index content using the Tracker filesystem miner


SYNOPSIS
       localsearch index
       localsearch index --add [--recursive] <dir> [[dir] ...]
       localsearch index --remove <path> [[dir] ...]


DESCRIPTION
       This command causes the Tracker filesystem miner to process the given directories. These changes are
       persistent across future restarts.

       If invoked without arguments, the currently indexed locations will be listed.
