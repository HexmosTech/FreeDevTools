NAME
       dbxedam - Index the EDAM ontology using b+tree indices


SYNOPSIS

       dbxedam -dbname string -standardname string -dbresource string -directory directory -filenames string
               -fields list -compressed boolean -release string -date string -exclude string -indexoutdir outdir
               -outfile outfile

       dbxedam -help


DESCRIPTION
       dbxedam is a command line program from EMBOSS (“the European Molecular Biology Open Software Suite”). It
       is part of the "Utils:Database indexing" command group(s).


OPTIONS
   Input section
       -dbname string
           Default value: edam

       -standardname string
           Default value: $(dbname)

       -dbresource string
           Default value: edamresource

       -directory directory
           Default value: .

       -filenames string
           Default value: EDAM*.obo

   Required section
       -fields list
           Default value: *

       -compressed boolean
           Default value: Y

   Advanced section
       -release string
           Default value: 0.0

       -date string
           Default value: 00/00/00

       -exclude string

   Output section
       -indexoutdir outdir
           Default value: .

       -outfile outfile
