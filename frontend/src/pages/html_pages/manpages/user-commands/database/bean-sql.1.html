NAME
       bean-sql - convert a Beancount ledger to an SQL database


DESCRIPTION
       usage: bean-sql [-h] [--version] filename database

       Convert a Beancount ledger into an SQL database.

   positional arguments:
       filename
              Beancount input filename

       database
              Filename of database file to create

   options:
       -h, --help
              show this help message and exit

       --version, -V
              show program's version number and exit
