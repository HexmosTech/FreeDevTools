NAME
       cdbdump - dump the contents of a constant database


SYNOPSIS
       cdbdump


DESCRIPTION
       cdbdump reads a constant database from its standard input and prints the database contents, in cdbmake(1)
       format, on standard output.


SEE ALSO
       cdbmake(1), cdbget(1), cdbstats(1)
