NAME
       pmdasmart - S.M.A.R.T Data PMDA


DESCRIPTION
       pmdasmart  is  a Performance Metrics Domain Agent (PMDA) which exports metric values for Device Mapper on
       the local system.

       This PMDA collects its data through the smartctl(8) utility and requires that the program is installed in
       order to function.

       Further details on smartctl and smartmontools can be found at https://smartmontools.org.


INSTALLATION
       Install the SMART PMDA by using the Install script as root:

           # cd $PCP_PMDAS_DIR/smart
           # ./Install

       To uninstall, do the following as root:

           # cd $PCP_PMDAS_DIR/smart
           # ./Remove

       pmdasmart is launched by pmcd(1) and should never be executed directly.  The Install and  Remove  scripts
       notify pmcd when the agent is installed or removed.


FILES
       $PCP_PMDAS_DIR/smart/help
           default help text file for the smart metrics

       $PCP_PMDAS_DIR/smart/Install
           installation script for the pmdasmart agent

       $PCP_PMDAS_DIR/smart/Remove
           undo installation script for the pmdasmart agent

       $PCP_LOG_DIR/pmcd/smart.log
           default log file for error messages from pmdasmart


PCP ENVIRONMENT
       Environment  variables with the prefix PCP_ are used to parameterize the file and directory names used by
       PCP. On each installation, the file /etc/pcp.conf contains the local values  for  these  variables.   The
       $PCP_CONF variable may be used to specify an alternative configuration file, as described in pcp.conf(5).
