NAME
       kma_shm - sets up a shared database (sysV) for mapping with KMA


DESCRIPTION
              kma_shm  sets  up  a  shared  database  (sysV) for mapping with KMA.  Options are:           Desc:
              Default:        Requirements:

       -t_db  Template DB                     None            REQUIRED

       -destroy
              Destroy shared DB               False

       -shmLvl
              Level of shared memory          1

       -shm-h Explain shm levels

       -v     Version

       -h     Shows this help message
