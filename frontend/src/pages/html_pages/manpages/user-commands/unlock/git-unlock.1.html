NAME
       git-unlock - Unlock a file excluded from version control


SYNOPSIS
       git-unlock <filename>


DESCRIPTION
       Unlock local files from version control.


OPTIONS
       <filename>

       The name of the filename.


EXAMPLES
       $ git unlock config/database.yml
