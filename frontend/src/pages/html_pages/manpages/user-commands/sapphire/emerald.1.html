NAME
       emerald - The Emerald Windows Decorator


SYNOPSIS
       emerald [--replace] [--help] [--version]


DESCRIPTION
       emerald emerald is the windows decorator of the compiz project.


OPTIONS
       --replace
            Replaces a current running decorator

       --version
            outputs the current version of emerald

       --help
            short usage statement about emerald


SIGNALS
       USR1 Reloads its settings.


FILES
       $HOME/.emerald/settings.ini
            Here are the settings stored.
