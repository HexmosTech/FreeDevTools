NAME
       gWakeOnLAN - Wake up your machines using Wake on LAN


SYNOPSIS
       gwakeonlan [options]


DESCRIPTION
       gWakeOnLAN is a GTK+ utility to awaken turned off computers through the Wake on LAN feature.

       The machines to turn on need to be shut off with the Wake on LAN magic packet enabled. Linux environments
       can enable this using:

       ethtool -s <nic device> wol g


OPTIONS
       This  program follow the usual GNU command line syntax, with long options starting with two dashes (`-').
       A summary of options is included below.

       -h, --help
              Show summary of options

       -v, --verbose
              Show all error and information messages

       -q, --quiet
              Hide all error and information messages


FILES
       Settings will be kept under ~/.config/gwakeonlan/
