NAME
       jekyll_help - show the help message for jekyll or a given subcommand


USAGE
       jekyll help [ COMMAND ]


OPTIONS
       The command accepts no special options but optionally a COMMAND to show its usage.

       For a list of general options please read the jekyll(1) manual page.

       jekyll(1)
