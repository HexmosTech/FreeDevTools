NAME
       psbl - A program for automatic booklet formatting of postscript files


SYNOPSIS

       psbl [-help] [-lastbbox] [-impose_option io] [-lpi ] [-max_pages mp ] [-evenodd ] postscript.file


DESCRIPTION
       This manual page documents briefly the psbl. Run it with '-help' to see description of options

       This manual page was written for the Debian(TM) distribution because the original program does not have a
       manual page.


SEE ALSO
       impose (1), fixtd (1), bboxx (1).
