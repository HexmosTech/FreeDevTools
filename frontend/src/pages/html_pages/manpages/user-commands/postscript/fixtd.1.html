NAME
       fixtd - insert tumble and duplex directives in a PostScript file


SYNOPSIS

       fixtd [-help] [-simplex] [-tumble] postscript.file


DESCRIPTION
       This manual page documents briefly the fixtd. Run it with '-help' to see description of options

       This manual page was written for the Debian(TM) distribution because the original program does not have a
       manual page.
