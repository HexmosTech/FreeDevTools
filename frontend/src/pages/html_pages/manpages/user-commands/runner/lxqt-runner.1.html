NAME
       lxqt-runner - Application launcher module of LXQt, the Lightweight Desktop Environment


SYNOPSIS
       lxqt-runner


DESCRIPTION
       This  module  allows  to launch any application by typing the configured shortcut, offers quick access to
       logout, shutdown and other functions. A calculator is also implemented: e.g 2*4+10/2=.


BEHAVIOR
       As you type, suggestions will appear. Once the application you wish to run appears you can navigate to it
       using the mouse or arrow keys on your keyboard. Pressing Enter or clicking it launches  the  application.
       The activation shortcut can be changed by the configuration menu, lxqt-globalkeyshortcuts (X11) or in the
       compositor (Wayland).


AUTOSTART
       This  module  is only shown in the LXQt desktop environment but can be used in other desktop environments
       by adding it to autostart.
