NAME
       vcfgeno2alleles


SYNOPSIS
       vcfgeno2alleles <[vcf file]


DESCRIPTION
       modifies the genotypes field to provide the literal alleles rather than indexes


OPTIONS
              Type: transformation


EXIT VALUES
       0      Success

       not 0  Failure

       vcflib(1)
