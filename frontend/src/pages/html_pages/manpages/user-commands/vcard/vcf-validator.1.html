NAME
       vcf-validator - validate VCF file


SYNOPSIS
       vcf-validator [OPTIONS] file.vcf.gz


OPTIONS
       -d, --duplicates
              Warn about duplicate positions.

       -u, --unique-messages
              Output all messages only once.

       -h, -?, --help
              This help message.
