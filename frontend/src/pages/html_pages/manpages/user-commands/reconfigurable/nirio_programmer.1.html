NAME
       nirio_programmer - National Instruments reconfigurable I/O tool


DESCRIPTION
       USRP-NIRIO-Programmer

              Allowed options:

       --help help message

       --interface arg (=0)
              The interface number to communicate with.

       --port arg (=5444)
              Port to communicate with RPC server.

       --fpga arg
              The absolute path to the LVBITX file to download to the FPGA.

       --flash arg
              The path to the image to download to the flash OR 'erase' to erase the FPGA image from flash.

       --peek arg
              Peek32.

       --poke arg
              Poke32.

       --stats
              Dump interface and DMA stats.
