NAME
       esedbinfo — determines information about an Extensible Storage Engine (ESE) Database File (EDB)


SYNOPSIS
       esedbinfo [-hvV] source


DESCRIPTION
       esedbinfo  is  a  utility to determine information about an Extensible Storage Engine (ESE) Database File
       (EDB)

       esedbinfo is part of the libesedb package.  libesedb is a library to access the Extensible Storage Engine
       (ESE) Database File (EDB) format

       source is the source file.

       The options are as follows:

       -h      shows this help

       -v      verbose output to stderr

       -V      print version


ENVIRONMENT
       None


FILES
       None
