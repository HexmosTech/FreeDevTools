NAME
       granite-7-demo - small demo application to show Granite Widgets

       granite-7-demo  is  a  small  demo application to show Granite Widgets.  Granite is an extension of GTK+.
       Among other things, it provides complex widgets and convenience functions designed for use in apps  built
       for elementary OS.
