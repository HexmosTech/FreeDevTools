NAME
       cfba2fba - copy a compressed FBA DASD file to a plain FBA DASD file


SYNOPSIS
       cfba2fba [-options] ifile [sf=sfile] ofile


DESCRIPTION
       This manual page documents the cfba2fba program.

       cfba2fba copies a compressed FBA file to a FBA file.


PARAMETERS
       ifile
                 input compressed FBA DASD file

       sfile
                 input compressed FBA shadow file (optional)

       ofile
                 output fba dasd file


OPTIONS
       -v        display program version and quit

       -h        display help and quit

       -q        quiet mode, don't display status

       -r        replace the output file if it exists

       -lfs      create single large output file

       -blks n   size of output file
