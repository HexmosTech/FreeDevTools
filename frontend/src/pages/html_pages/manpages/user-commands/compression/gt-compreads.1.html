NAME
       gt-compreads - Call a fastq file compression tool.


SYNOPSIS
       gt compreads [option ...] tool [argument ...]


DESCRIPTION
       -help
           display help and exit

       -version
           display version information and exit

       Tools:

       •   compress

       •   decompress

       •   refcompress

       •   refdecompress


REPORTING BUGS
       Report bugs to https://github.com/genometools/genometools/issues.
