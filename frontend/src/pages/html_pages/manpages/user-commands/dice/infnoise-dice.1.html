NAME
       dice - roll dice


SYNOPSIS
       dice <file> <dice> <sides>


DESCRIPTION
       dice simulates a dice roll, using data from file to roll dice dice with sides sides each.

       A common value for file is /dev/random.


OPTIONS
       This program has no options.


SEE ALSO
       infnoise(8).
