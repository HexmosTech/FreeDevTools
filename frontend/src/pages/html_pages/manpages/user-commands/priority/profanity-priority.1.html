NAME
       /priority - the Profanity /priority command.


DESCRIPTION
       Set  priority  for  the  current  account.  See  the  /account command for specific priority settings per
       presence status.


SYNOPSIS
       /priority <priority>


ARGUMENTS
       <priority>
           Number between -128 and 127, default: 0.
