NAME
       argonaut-clean-audit - delete old audit entries from the LDAP


SYNOPSIS
       argonaut-clean-audit [--verbose]


DESCRIPTION
       argonaut-clean-audit is a program used to delete old audit entries from the LDAP.  It reads the delay
       before deletion from LDAP in fdAuditRotationDelay.


OPTIONS
       --verbose
          be verbose
