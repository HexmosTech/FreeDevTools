NAME
       vcfqual2info


SYNOPSIS
       vcfqual2info [key] [vcf_file]


DESCRIPTION
       Puts QUAL into an info field tag keyed by [key].


OPTIONS
              Type: transformation


EXIT VALUES
       0      Success

       not 0  Failure
