NAME
       ssh-conv - script to convert OpenSSH/SSH2 keys to lsh format


SYNOPSIS
       ssh-conv [OPTION...]


DESCRIPTION
       CAUTION!  The information in this manpage may be invalid or outdated. For authorative information on lsh,
       please see it's Texinfo manual (see the SEE ALSO section).

       Converts an OpenSSH style public key to spki format used by lsh. Reads the key from stdin and outputs the
       converted key on stdout.

       Example usage:

       $ ssh-conv < /home/pont/.ssh/id_dsa.pub
