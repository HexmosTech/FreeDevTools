NAME
       sigsum-key-generate - create new key pair in OpenSSH format


SYNOPSIS
       sigsum-key generate [--help] [-o key-file]


DESCRIPTION
       Create  a  new  key  pair. The private key is stored in the given file in OpenSSH private-key format. The
       corresponding public-key file gets a ".pub" suffix and is written in OpenSSH public-key format.


OPTIONS
       --help Show usage message and exit

       -o, --output=key-file

              File to store the private key in (required)


RETURN CODES
       A non-zero return code is used to indicate failure.
