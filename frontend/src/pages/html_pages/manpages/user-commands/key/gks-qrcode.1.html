NAME
       gks-qrcode - show QR code for GPG key


SYNOPSIS
       gks-qrcode <keyid>


DESCRIPTION
       The  gks-qrcode tools displays a X window with the QR code for the key associated with a GPG key ID given
       as a parameter.
