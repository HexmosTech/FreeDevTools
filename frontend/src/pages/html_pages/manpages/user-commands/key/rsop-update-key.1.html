NAME
       rsop-update-key - Keep a Secret Key Up-To-Date


SYNOPSIS
       rsop    update-key    [--no-armor]   [--signing-only]   [--no-added-capabilities]   [--with-key-password]
       [--merge-certs] [--debug] [-h|--help]


DESCRIPTION
       Keep a Secret Key Up-To-Date


OPTIONS
       --no-armor
              Don't ASCII-armor output

       --signing-only
              Don't make the updated key encryption-capable if it isn't already

       --no-added-capabilities
              Don't advertise support for capabilities that aren't already advertised by the key

       --with-key-password=WITH_KEY_PASSWORD
              Unlock the keys with these passwords

       --merge-certs=MERGE_CERTS
              Merge updates into the key

       --debug
              Emit verbose output for debugging

       -h, --help
              Print help
