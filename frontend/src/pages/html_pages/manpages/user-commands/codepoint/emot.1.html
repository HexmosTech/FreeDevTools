NAME
       emot - show emoji and codepoint


USAGE
       emot   [ COMMAND ]


COMMAND
       help [COMMAND]
              Describe available commands or one specific command

       icons  show all emoji icons

       names  show all available names for emoji

       show [NAME]
              show emoji icon and unicode for NAME

       version
              Show Emot version
