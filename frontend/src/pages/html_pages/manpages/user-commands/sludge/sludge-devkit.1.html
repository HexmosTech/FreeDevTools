NAME
       sludge-devkit - development tools for SLUDGE games


SYNOPSIS

       sludge-projectmanager [projectfile.slp]

       sludge-floormaker [floorplan.flo]

       sludge-zbuffermaker [zbufferfile.zbu]

       sludge-spritebankeditor [spritebank.duc]

       sludge-translationeditor [translationfile.tra]


DESCRIPTION
       This manual page documents briefly the sludge-devkit package.

       This manual page was written for the Debian distribution because the original program does not have a
       manual page. Instead, it has documentation in the html format; see below.


FILES
       ${HOME}/.config/sludge-devkit/
           This directory is used to store configuration files.


BUGS
       Please report bugs in the SLUDGE forum at http://opensludge.sourceforge.net.


SEE ALSO
       sludge-engine(1)

       The full SLUDGE documentation is included in the package sludge-doc. It is installed to
       /usr/share/doc/sludge/SLUDGE_Help.html.
