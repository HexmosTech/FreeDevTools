NAME
       download-pubmed - download an NCBI PubMed archive dump


SYNOPSIS
       download-pubmed [-ftp|-http[s]] [section...]


DESCRIPTION
       download-pubmed  downloads  one or more sections of NCBI's PubMed archive to the current directory.  When
       run without arguments, it downloads the baseline and updatefiles sections.


OPTIONS
       -ftp   Download via FTP (default).

       -http[s]
              Download via HTTPS.

       section...
              Archive section(s) to download.
