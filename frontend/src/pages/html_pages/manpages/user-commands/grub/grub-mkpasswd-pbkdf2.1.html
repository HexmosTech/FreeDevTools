NAME
       grub-mkpasswd-pbkdf2 - generate hashed password for GRUB


SYNOPSIS
       grub-mkpasswd-pbkdf2 [OPTION...] [OPTIONS]


DESCRIPTION
       Generate PBKDF2 password hash.

       -c, --iteration-count=NUM
              Number of PBKDF2 iterations

       -l, --buflen=NUM
              Length of generated hash

       -s, --salt=NUM
              Length of salt

       -?, --help
              give this help list

       --usage
              give a short usage message

       -V, --version
              print program version

       Mandatory  or  optional  arguments  to  long options are also mandatory or optional for any corresponding
       short options.
