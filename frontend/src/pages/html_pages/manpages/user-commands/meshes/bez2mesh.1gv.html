NAME
       bez2mesh - dices a BEZ file to a list of MESHes


SYNOPSIS
       bez2mesh [[-]N] [file.bez]


DESCRIPTION
       This manual page documents briefly the bez2mesh command.

       bez2mesh Dices a BEZ file to a list of MESHes, each diced NxN.  If N negative, everts each patch.
