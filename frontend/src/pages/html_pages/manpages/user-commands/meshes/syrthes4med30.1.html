NAME
       syrthes4med30 - Transform results from Syrthes 4.x to MED format 3.0


SYNOPSIS
       syrthes4med30 -m mesh.syr -o mesh.med
       syrthes4med30 -m mesh.syr -r resu -o resu.med
       syrthes4med30 -h


DESCRIPTION
       This manual page documents briefly the syrthes4med30 command.

       syrthes4med30 transforms SYRTHES geometry and results into MED 3.0 files.

       A detailed documentation about SYRTHES and its tools is available at <http://rd.edf.com/syrthes>
