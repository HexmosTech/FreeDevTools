NAME
       yaskkserv_make_dictionary - dictionary converter for yaskkserv


SYNOPSIS
       yaskkserv_make_dictionary [OPTION] skk-dictionary output-dictionary


DESCRIPTION
       This  manual page was written for the Debian GNU/Linux distribution because the original program does not
       have a manual page.

       yaskkserv is a dictionary server for the SKK Japanese input method system. yaskkserv is  written  in  C++
       from  scratch  in  consideration  of embedded environment, but is compatible with skkserv on the protocol
       behavior.

       yaskkserv_make_dictionary converts dictionary file of original format into that of yaskkserv format.


OPTIONS
       -a, --alignment
              enable alignment (default disable)

       -b, --block-size=SIZE
              set block size (default 8192)

       -d, --debug
              print debug information

       -h, --help
              print this help and exit

       -s, --short-block
              enable short block (must set --alignment) (default disable)

       -v, --version
              print version
