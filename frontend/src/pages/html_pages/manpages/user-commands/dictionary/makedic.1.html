NAME
       makedic - Utility program for generating your own "kanjidic" style dictionaries


USAGE
       makedic OUTPUTFILE < inputfile


input format
         makedict expects:
           one line JIS encoding of a SINGLE character, in hex, with spaces
           one line of english translation

         It outputs in kanjidic format.
