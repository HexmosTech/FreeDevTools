NAME
       cif_mpod_v1_to_v3 - convert MPOD structures from MPOD dictionary V1 to MPOD dict. V3 based data names.


SYNOPSIS
       cif_mpod_v1_to_v3 --options input1.cif input*.cif


DESCRIPTION
       Convert MPOD structures from MPOD dictionary V1 to MPOD dict. V3 based data names.


OPTIONS
       --use-perl-parser
                         Use development CIF parser written in Perl.

       --use-c-parser
                         Use faster C/Yacc CIF parser (default).

       --help, --usage
                         Output a short usage message (this message) and exit.

       --version
                         Output version information and exit.
