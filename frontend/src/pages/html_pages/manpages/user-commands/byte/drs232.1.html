NAME
       drs232 - Horus SSTV reception using the codec2 library.


SYNOPSIS
       drs232 usage: InputOneBitPerChar OutputPacket [-v[v]]


DESCRIPTION
       Looks  for  a  unique word in series of bits.  When found, deframes a RS232 encoded frame of bytes.  Used
       for high bit rate Horus SSTV reception.

         Frame format:

           16 bytes 0x55 - 0xabcdef01 UW - 256 bytes of payload - 2 bytes CRC

         Each byte is encoded as a 10 bit RS232 serial word:

           0 LSB .... MSB 1


OPTIONS
       The first argument, InputOneBitPerChar is the file name of a file with Horus  encoded  bits.  If  "-"  is
       given the input is taken from stdin.

       The  second  argument,  OutputPackets is the file name of a file to be created with modem sample data. If
       "-" is given the output is sent to stdout.

       The optional third argument, [-v] [-vv] set the verbosity of the output.
