NAME
       hdel - delete both forks of an HFS file


SYNOPSIS
       hdel hfs-path [...]


DESCRIPTION
       hdel  deletes  files  from  the current HFS volume. Both forks (resource and data) of each named file are
       removed, freeing space for other files.


SEE ALSO
       hfsutils(1), hrmdir(1)


FILES
       $HOME/.hcwd
