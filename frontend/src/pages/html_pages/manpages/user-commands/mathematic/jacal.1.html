NAME
       JACAL - symbolic mathematics system


SYNOPSIS
       jacal [ scheme | scm | gsi | gosh | guile | scheme48 | larceny | scmlit | elk | sisc | kawa | mzscheme ]


DESCRIPTION
       JACAL  is  a  symbolic mathematics system for the simplification and manipulation of equations and single
       and multiple valued algebraic expressions constructed of  numbers,  variables,  radicals,  and  algebraic
       functions, differential, and holonomic functions.  In addition, vectors and matrices of the above objects
       are included.


OPTIONS
       The  optional  argument to the jacal script is the Scheme implementation to run.  Absent the argument, it
       searches for implementations in the above order.


ENVIRONMENT VARIABLES
       SCHEME_LIBRARY_PATH
              is the SLIB Scheme library directory.


FILES
       jacal.info
              Texinfo documentation of jacal.

       demo
              batch("demo"); to demonstrate commands in jacal

       rw.math
              batch("rw.math"); to demonstrate tensors in "The Robertson-Walker Cosmology Model".

       test.math
              batch("test.math"); for regression suite.
