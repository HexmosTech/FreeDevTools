NAME
       organize-edit - Edit the rules.


SYNOPSIS
       organize edit [OPTIONS] [CONFIG]


DESCRIPTION
       Edit the rules.

           If called without arguments it will open the default config file in $EDITOR.


OPTIONS
       --editor TEXT
              The editor to use. (Default: $EDITOR)
