NAME
       oarmonitor_graph_gen - Generate graphics from a job resource usage.


SYNOPSIS
       oarmonitor_graph_gen [-h] | [-j job_id]


DESCRIPTION
       This command is an example of graphics that can be generated when the OAR monitoring feture is activated
       on jobs.


OPTIONS
       -j, --job_id jobid
           Give the job id to get data.
