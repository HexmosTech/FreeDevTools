NAME
       octo_pusher - Octopussy Pusher program


SYNOPSIS
       octo_pusher


DESCRIPTION
       octo_pusher is the program used by the Octopussy Project to parse 'no-syslog' Logs for 'no-syslog'
       Devices


FUNCTIONS
   Init()
       Inits Pusher

   Handle_Dir($device, $day, $month, $year)
       Handles Directory ($device/$year/$month/$day)

   Convert_To_Syslog($line, $device)
       Converts line $line to a syslog line

   Write_To_Incoming($dev, $year, $month, $mday, $hour, $min, @lines)
       Writes to Incoming Directory
