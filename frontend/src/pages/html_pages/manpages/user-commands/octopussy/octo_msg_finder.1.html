NAME
       octo_msg_finder - Octopussy message finder program


SYNOPSIS
       octo_msg_finder "<line>" | <filename>


DESCRIPTION
       octo_msg_finder is the program used by the Octopussy Project to help users to determine to wich
       Message/Service a log belongs to


FUNCTIONS
   Search_Line($line, $n)
       Prints Service, MsgID, Taxonomy & Table of line $line
