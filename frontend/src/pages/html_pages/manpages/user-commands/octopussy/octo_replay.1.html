NAME
       octo_replay - Octopussy Replay program


SYNOPSIS
       octo_replay --device <device> --service <service> --begin YYYYMMDDHHMM --end YYYYMMDDHHMM


DESCRIPTION
       octo_replay is the program used by the Octopussy Project to replay Logs for some Device / Service (it
       gets 'recognized' logs and put it back in Incoming directory)
