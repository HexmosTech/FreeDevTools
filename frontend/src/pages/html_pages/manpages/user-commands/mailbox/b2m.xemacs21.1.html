NAME
       b2m - This program converts a babyl mailbox to a standard unix mailbox.


SYNOPSIS
       b2m [options] < babylmailbox > unixmailbox


DESCRIPTION
       This  manual  page  documents  briefly  the  b2m,  command.   This manual page was written for the Debian
       GNU/Linux distribution because the original program does not have a manual page.


OPTIONS
       None


AUTHOR

       may be used by others).
