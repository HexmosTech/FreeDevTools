NAME
       notify - desktop notification on cross platforms


SYNOPSIS
           notify  [OPTION...] <SUMMARY> [BODY]


DESCRIPTION
       The  notify  provides  desktop  notification  on  cross platforms.  In Debian, notify execute notify-send
       command.
