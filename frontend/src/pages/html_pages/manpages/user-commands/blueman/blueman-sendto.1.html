NAME
       blueman-sendto - application for sending files to bluetooth devices


SYNOPSIS
       blueman-sendto [options]


DESCRIPTION
       blueman-sendto sends files to specified device, if there is no device specified, it will display a dialog
       to choose from all known devices.


OPTIONS
       --device=ADDRESS
              Address of the device to send to
