NAME
       /room - the Profanity /room command.


DESCRIPTION
       Chat room configuration.


SYNOPSIS
       /room accept|destroy|config


ARGUMENTS
       accept
           Accept default room configuration.

       destroy
           Reject default room configuration, and destroy the room.

       config
           Edit room configuration.
