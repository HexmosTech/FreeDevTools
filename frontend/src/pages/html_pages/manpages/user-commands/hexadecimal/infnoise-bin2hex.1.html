NAME
       bin2hex - display binary content as hexadecimal


SYNOPSIS
       bin2hex


DESCRIPTION
       bin2hex reads from its standard input, and outputs the value of each byte it reads as a hexadecimal pair.


OPTIONS
       This program has no options.

       infnoise(8).
