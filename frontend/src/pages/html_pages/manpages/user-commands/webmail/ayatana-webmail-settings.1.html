NAME
       ayatana-webmail-settings - Webmail Integration into your Desktop


SYNOPSIS
       ayatana-webmail-settings [ options ]


DESCRIPTION
       Ayatana  Webmail  is an application that integrates your webmail into MATE, Xfce, LXDE and other environ‐
       ments. For further details see the ayatana-webmail(1) man page.

       The ayatana-webmail-settings executable does not have any command line options. It launches  the  Ayatana
       Webmail Configuration dialog window.


SEE ALSO
