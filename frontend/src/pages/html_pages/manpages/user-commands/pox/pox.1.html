NAME
       pox - Utilities for filesystem exploration and automated builds


SYNOPSIS
       pox ["func(arguments)"]


DESCRIPTION
       Run  a  pox  command, as specified in the argument.  The available pox commands are listed in the OPTIONS
       section.  Run
         pox "help(command)"
       to get more details about how to use command.
