NAME
       timew-tags - display a list of tags


SYNOPSIS
       timew tags [<range>]


DESCRIPTION
       Displays all the tags that have been used by default. When a filter is specified, shows only the tags
       that were used during that time.


CONFIGURATION
       tags.<tag>.color
           Assigns a specific foreground and background color to a tag. Examples of valid colors include
           'white', 'gray8', 'black on yellow', and 'rgb345'.
