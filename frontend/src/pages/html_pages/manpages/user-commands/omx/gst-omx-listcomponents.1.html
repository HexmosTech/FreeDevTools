NAME
       gst-omx-listcomponents - program to list the name of components in openmax IL


SYNOPSIS

       gst-omx-listcomponents [libopenmax.so]


DESCRIPTION
       This manual page documents briefly the gst-omx-listcomponents command.

       This manual page was written for the Debian distribution because the original program does not have a
       manual page.

       gst-omx-listcomponents is a program that can list the name of components in libopenmax.so. It helps you
       to write the /etc/xdg/gstomx.conf


OPTIONS
       The program doesn't have any options yet.
