NAME
       mergelog - a fast tool to merge http log files by date


SYNOPSIS
       mergelog logfile1 logfile2 ...


DESCRIPTION
       The  mergelog program is a tool which merges by date http log files in ´Common Log Format´. The result is
       sent to the standard output. It is useful if you want to create a single  log  file  for  multiple  hosts
       using round-robin DNS.
