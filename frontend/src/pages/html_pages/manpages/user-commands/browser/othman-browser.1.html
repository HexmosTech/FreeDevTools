NAME
       othman-browser - A Qur'an browser


SYNOPSIS
       othman-browser


DESCRIPTION
       othman-browser  is  a Qur'an browser which displays Quranic text in Othmani script style as written under
       authority of Othman ibn Affan the companion of prophet Muhammad (Peace Be Upon Him).
