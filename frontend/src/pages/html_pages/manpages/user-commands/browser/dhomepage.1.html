NAME
       dhomepage - show the homepage of a package in a browser


SYNOPSIS
       dhomepage PACKAGE


DESCRIPTION
       Shows the homepage of a Debian package using a sensible browser.  Outputs the URL if no browser is found.


OPTIONS
       --help Print help message

       --version
              Print version message


EXIT STATUS
       Returns 0 if homepage was found, and 1 otherwise.
