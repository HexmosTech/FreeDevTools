NAME
       eric7_browser - eric7 web browser (QtWebEngine)


SYNOPSIS
       eric7_browser [options] files...


DESCRIPTION
       This  manual  page  documents  briefly  the  eric7_browser command.  This manual page was written for the
       Debian distribution because the original program does not have a manual page.

       eric7_browser is a web browser to read the HTML pages for the Qt API and the eric7 documentation. It  can
       be used from inside the eric7 IDE or as a standalone application.


OPTIONS
       These  programs  follow  the  usual  GNU  command line syntax, with long options starting with two dashes
       (`-').  A summary of options is included below.  For a complete description, see the Info files.

       -h, --help
              Show summary of options.

       -v, --version
              Show version of program  and  exit.   --config=configDir  Use  the  given  directory  as  the  one
              containing  the  config  files.   --private  Start the browser in private browsing mode.  --qthelp
              Start  the  browser  with  support  for  QtHelp.   --search=word  Search  for   the   given   word
              --settings=settingsDir Use the given directory to store the settings files.
