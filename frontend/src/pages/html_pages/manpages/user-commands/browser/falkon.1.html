NAME
       falkon - lightweight web browser


SYNOPSIS

       falkon


DESCRIPTION
       falkon is a new and very fast QtWebKit browser. It aims to be a lightweight web browser available through
       all major platforms. This project has been originally started only for educational purposes. But from its
       start, falkon has grown into a feature-rich browser.
