NAME
       pycode-browser - environment to teach with Python code snippets


SYNOPSIS

       pycode-browser


DESCRIPTION
       pycode-browser is part of the project PHOENIX, i. e. Phyiscs with Home-made Equipment & Innovative
       Experiments, developed at Inter University Accelerator Centre, New Delhi. It allows students to browse
       Python source code easily, and to understand what this code is meaning.

       see also other programs from the project PHOENIX.
