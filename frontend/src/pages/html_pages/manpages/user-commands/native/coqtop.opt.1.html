NAME
       coqtop.opt - The native-code Coq toplevel


SYNOPSIS
       coqopt.opt [ options ] [ file ]


DESCRIPTION
       coqopt.opt  is  the  native-code version of Coq. It should not be called directly, but only by coqtop and
       coqc
