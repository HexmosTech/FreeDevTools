NAME
       base58 - Base58 encode or decode


SYNOPSIS
       base58 [-h] [-d] [-c] [FILE]


DESCRIPTION
       Base58 encode or decode FILE, or standard input, to standard output.


OPTIONS
       -h, --help
              show this help message and exit

       -d, --decode
              decode data

       -c, --check
              append a checksum before encoding
