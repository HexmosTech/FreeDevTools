NAME
       runjade — Wraps call to openjade to catch errors


SYNOPSIS
       runjade jade_binary [jade_options ...]


Description
       runjade  allows  to  run openjade or jade and return a conventional non-zero exit status on error.  It is
       only a workaround of those programs not setting correctly the exit status on error, and works by  looking
       for :E: error tags in the program's standard error stream.
