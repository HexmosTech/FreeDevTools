NAME
       ptg-configurator - GUI app to evaluate and test PTGs for navigation


SYNOPSIS
       ptg-configurator


DESCRIPTION
       A GUI application for visualizing, debugging and selecting the optimal parameters for Parameterized
       Trajectory Generators (PTGs), entities employed in a variety of navigation algorithms in MRPT.
