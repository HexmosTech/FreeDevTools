NAME
       sigsum-key-to-hash - output public-key hash in hex format


SYNOPSIS
       sigsum-key to-hash [--help] [-k key-file] [-o output-file]


DESCRIPTION
       Read  a  public key in OpenSSH format and output its hash in hex format.  The public key is read on stdin
       and output is written on stdout.  Override the default behavior using the -k and -o options.


OPTIONS
       --help Show usage message and exit

       -k, --key=key-file

              Public key in OpenSSH format

       -o, --output=output-file

              Hashed public key in hex format


RETURN CODES
       A non-zero return code is used to indicate failure.
