NAME
       sffview — X11 viewer for structured fax files (SFF)


SYNOPSIS
       sffview


DESCRIPTION
       This manual page documents briefly the sffview command.

       This  manual  page  was  written for the Debian distribution because the original program does not have a
       manual page.

       sffview is a tool to view "structured fax files" in X11.  "Structured fax file" (SFF) is the file  format
       that ISDN cards with CAPI interface expect and create.


OPTIONS
       NONE


BUGS
       Only German localisation
