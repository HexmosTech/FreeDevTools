NAME
       dpt-debian-upstream - convert resources from META.{json,yml} to debian/upstream/metadata


SYNOPSIS
       dpt debian-upstream


DESCRIPTION
       dpt debian-upstream reads information from META.json or META.yml and writes it to
       debian/upstream/metadata, according to the UpstreamMetaData specification.

       Please, note that upstream links will be switched to https URLs.


SEE ALSO
       •   CPAN::Meta::Spec

       •   UpstreamMetaData <https://wiki.debian.org/UpstreamMetadata>
