NAME
       checkXML6 - An XML lint tool for KDE DocBook XML documents


SYNOPSIS

       checkXML6 [filename]


DESCRIPTION
       checkXML6 is a tool to check for syntax errors in KDE DocBook XML files. It can also be used for other
       DocBook based XML files, but you should use the less specific xmllint(1) tool if you are not writing or
       otherwise working on KDE documentation.


SEE ALSO
       meinproc6(1) xmllint(1)
