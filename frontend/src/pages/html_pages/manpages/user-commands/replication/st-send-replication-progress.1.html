NAME
       st-send-replication-progress - a Swift-tools command


SYNOPSIS
       st-status [OPTION]...


DESCRIPTION
       st-send-replication-progress Send replication progress to graphite serve

       [-s|--server]

              Graphite server to send data

       [-p|--port]

              Graphite server port to send data 92003 by default)

       [-c|--cluster]

              Cluster Name
