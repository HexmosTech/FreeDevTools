NAME
       nntp-get - print out a single article


SYNOPSIS

       nntp-get [options...] message-id


DESCRIPTION
       Print a single article to the standard output. The requested article is specified by the value of
       Message-Id header.

       A single message-id argument is a value of the requested article's Message-Id header. It can be provided
       with or without angle brackets.

       Global options are described in sinntp(1).
