NAME
       update-indymedia-cities - create cities.inc from http://www.indymedia.org/cities.xml


SYNOPSIS
       update-indymedia-cities > cities.inc


DESCRIPTION
       update-indymedia-cities  Retrieves  cities.xml file holding the list of all Independent Media Centers and
       produces an HTML rendering of this list suitable for inclusion in a static footer of Samizdat front page.


OPTIONS
       Accepts no options. The URL to fetch cities.xml is hardcoded into  the  script,  the  resulting  HTML  is
       printed to standard output.
