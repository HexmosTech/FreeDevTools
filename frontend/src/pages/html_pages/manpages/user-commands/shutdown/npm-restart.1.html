NAME
       npm-restart


Synopsis
       <!-- AUTOGENERATED USAGE DESCRIPTIONS -->


Description
       This restarts a project.  It is equivalent to running npm run-script restart.

       If the current project has a &quot;restart&quot; script specified in
       package.json, then the following scripts will be run:

        • prerestart

        • restart

        • postrestart

       If it does not have a &quot;restart&quot; script specified, but it does have
       stop and/or start scripts, then the following scripts will be run:

        • prerestart

        • prestop

        • stop

        • poststop

        • prestart

        • start

        • poststart

        • postrestart
