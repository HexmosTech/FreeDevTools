NAME
       thunar-settings - display thunar(1) preferences dialog


SYNOPSIS
       thunar-settings


DESCRIPTION
       This manual page documents briefly the thunar-settings command.

       thunar-settings shows the thunar(1) preferences dialog, and accept no options.
