NAME
       ropy - calculate entropy measurement of byte streams


SYNOPSIS
       ropy FILE


DESCRIPTION
       The  library  libdisorder  provides  a function to calculate the Shannon index (H) of byte streams.  This
       program is provided as an example and does this for a given file.


AUTHOR

       of the program.
