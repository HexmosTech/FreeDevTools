NAME
       vmware-xferlogs - dump vm-support output to vmx logfile


SYNOPSIS
       vmware-xferlogs OPTIONS FILE


DESCRIPTION
       vmware-xferlogs  is  a  simple  console application and can be run as any user. It is based on rpctool to
       dump the vm-support output to the vmx log file base64 encoded. It can  dump  any  file  supplied  on  the
       commandline  to the vmx log. It also does the decoding part of it. It can read the vmware.log file decode
       and write the encoded files in the directory it was invoked.
