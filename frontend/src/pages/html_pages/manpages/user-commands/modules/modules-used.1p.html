NAME
       modules-used - List modules used by a set of Perl source files without running them.


VERSION
       This document describes modules-used version 1.3.0.


USAGE
           modules-used source-file [...]


DESCRIPTION
       Dumps a list of modules used by some Perl code.  Modules are found statically based upon "use" and
       "require" statements.  If use of the base or parent is found, both that module and the referenced ones
       will be emitted.

       Dynamically loaded modules will not be found.


REQUIRED ARGUMENTS
       X source-file [...]
           A list of files to search.


OPTIONS
       None.


DIAGNOSTICS
       Could not find module "%s" in @INC.
           Cannot  find  the location of the named module.  Note that this program will not find any dynamically
           loaded modules.

       "%s" does not exist.
           Cannot find the file.

       "%s" is not readable.
           Cannot read the file.

       "%s" is a directory.
           The "file" was actually a directory.

       Could not parse "%s".
           PPI could not interpret the file as a Perl document.
