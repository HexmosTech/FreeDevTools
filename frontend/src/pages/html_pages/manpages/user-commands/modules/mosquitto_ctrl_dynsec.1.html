NAME
       mosquitto_ctrl_dynsec - mosquitto_ctrl module for controlling the Mosquitto Dynamic Security plugin.


SYNOPSIS

       mosquitto_ctrl [connection-options] dynsec dynsec-command [command-options]


DESCRIPTION
       This page describes the dynsec module for mosquitto_ctrl(1). See the mosquitto_ctrl man page for details
       of the options for connecting to remote brokers, in particular since this module works with
       authentication and access control, it is crucial that secure encrypted connections are used.
