NAME
       unlink - Unlink the file at `FILE`.


SYNOPSIS
       unlink [-h|--help] [-V|--version] <FILE>


DESCRIPTION
       Unlink the file at `FILE`.


OPTIONS
       -h, --help
              Print help

       -V, --version
              Print version
