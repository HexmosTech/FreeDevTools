NAME
       usd-locate-pointer - create ripples around mouse pointer


SYNOPSIS
       usd-locate-pointer


DESCRIPTION
       The  usd-locate-pointer  utility  is  a  program  that creates ripples (like throwing a stone into water)
       around the mouse pointer. This helps to locate the position of the mousepointer.

       With ukui-settings-daemon i is possible to assign a hotkey for evoking the usd-locate-pointer command.
