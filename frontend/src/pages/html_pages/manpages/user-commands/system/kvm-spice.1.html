NAME
       kvm-spice, qemu-system-x86_64-spice - compatibility names for qemu-system-x86_64


DESCRIPTION
       The  two  names  are  aliases for qemu-system-x86_64, where kvm-spice enables kvm native hardware mode by
       default.  These names are kept for backward compatibility wih old package, when spice-enabled  qemu  were
       packaged separately. Now main qemu-system has spice functionality built in. Please use qemu-system-x86_64
       instead of the old compat names.
