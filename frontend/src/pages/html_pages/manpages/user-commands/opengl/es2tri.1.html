NAME
       es2tri - a draw demonstration using X/EGL and OpenGL ES 2.x


SYNOPSIS
       es2tri


DESCRIPTION
       The  es2tri program demonstrate the usage of X/EGL and OpenGL ES 2.x by drawing a triangle at the screen.
       This program is most useful when studied in its source code form.


AUTHOR
