NAME
       celery-amqp - AMQP Administration Shell.


SYNOPSIS
       celery amqp [OPTIONS] COMMAND [ARGS]...


DESCRIPTION
       AMQP Administration Shell.

       Also works for non-AMQP transports (but not ones that store declarations in memory).
