NAME
       ucimf_keyboard - ucimf jfbterm keyboard shortcut mapping tool


DESCRIPTION
       ucimf_keyboard is a jfbterm keyboard shortcut mapping tool for ucimf, the input method framework designed
       for Linux unicode console.

       ucimf_keyboard  maps Ctrl+Space to F12, and Ctrl+ShiftLeft to F11, which action makes the shortcut can be
       kept the same as users who are running FbTerm.
