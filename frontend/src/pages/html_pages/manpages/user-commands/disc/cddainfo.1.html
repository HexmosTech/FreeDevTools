NAME
       cddainfo - prints information about a compact disc


SYNOPSIS
       cddainfo [OPTIONS]


DESCRIPTION
       cddainfo  takes  a  compact disc device and writes information about the current disc to standard output.
       This includes the total number of tracks, the total length of the disc, the FreeDB and  MusicBrainz  disc
       IDs, and the length/offset information of each track.


OPTIONS
       -h, --help
              show a list of options and exit

       -c, --cdrom=CDROM
              the CD-ROM device to read CD information from
