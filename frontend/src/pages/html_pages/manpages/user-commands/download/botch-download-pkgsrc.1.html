NAME
       botch-download-pkgsrc - download Packages and Sources


SYNOPSIS
       botch-download-pkgsrc [arch suite]


DESCRIPTION
       Download  Packages and Sources files from a mirror. If arch and suite are not specified, then the default
       is to download a snapshot of Debian Sid of January 1. 2014 from snapshot.debian.org.

       The value arch can be set to a valid Debian architecture and the value  suite  can  either  be  a  Debian
       release  in  which case, the Packages and Sources files are retrieved from http.debian.net or it can be a
       Ubuntu release in which case the files are retrieved from archive.ubuntu.com or it can be  a  year  after
       2005  in  which  case  a  snapshot  of  Debian  sid  of  January  1.  of  that  year  is  retrieved  from
       snapshot.debian.org
