NAME
       lomiri-download-manager - The Lomiri Download Manager


SYNOPSIS
       lomiri-download-manager


DESCRIPTION
       lomiri-download-manager  is  a  user  space  DBus  daemon  of  the Lomiri Operating Environment centrally
       managing downloads.
