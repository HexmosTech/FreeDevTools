NAME
       guilt-add - Add a file to git and guilt


SYNOPSIS
       guilt add <file>...


DESCRIPTION
       Adds the files specified to git using git-add(1) making it available to guilt.


OPTIONS
       <files>
           Files to add


EXAMPLES
       Create and add a new file example.c

           $ touch example.c
           $ guilt add example.c
