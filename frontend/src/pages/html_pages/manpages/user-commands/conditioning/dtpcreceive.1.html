NAME
       dtpcreceive - Delay-Tolerant Payload Conditioning reception test program


SYNOPSIS
       dtpcreceive topic_ID


DESCRIPTION
       dtpcreceive uses DTPC to acquire application data items on topic topic_ID sent by dtpcsend.  Upon
       termination it prints the total number of application data items received and the mean rate of
       application data transmission.

       Use CTRL-C to terminate the program.


EXIT STATUS
       0   dtpcreceive has terminated.


FILES
       No configuration files are needed.


ENVIRONMENT
       No environment variables apply.
