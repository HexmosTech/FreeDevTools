NAME
       showq - MIDI controlled theatre cue player


SYNOPSIS
       showq


DESCRIPTION
       This  manual  page  documents  briefly  the  showq  program.  This manual page was written for the Debian
       distribution because the original program does not have a manual page.

       showq is a MIDI controlled theatre cue player that can play audio or trigger MIDI events.  Applicaton can
       be controlled by midi.  It can play through alsa or through Jack Audio Connection Kit.


OPTIONS
       This program takes no arguments


SEE ALSO
       jackd(1).
