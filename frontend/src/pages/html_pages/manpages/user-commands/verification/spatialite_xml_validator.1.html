NAME
       spatialite_xml_validator - perform XML schema validation


SYNOPSIS
       spatialite_xml_validator [-f xml-path | -l list-of-paths-file]


DESCRIPTION
       spatialite_xml_validator is a tool for performing XML schema validation.


OPTIONS
       -f xml-path
              the XML file path

       -l list-of-paths-file
              path to file listing multiple XML files to validate
