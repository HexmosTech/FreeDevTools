NAME
       fusecram - utility to mount cramfs filesystem images with FUSE


SYNOPSIS
       fusecram [ cramfs_image_file ] [ mount_point ] [ FUSE library options ]


DESCRIPTION
       utility to mount cramfs filesystem images with FUSE.

       Fusecram is a utility to mount cramfs filesystem images using the FUSE module. It is required by the klik
       system.
