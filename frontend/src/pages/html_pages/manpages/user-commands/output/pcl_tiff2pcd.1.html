NAME
       pcl_tiff2pcd - pcl_tiff2pcd


DESCRIPTION
       usage: convert -rgb <rgb_path> -depth <depth_path> -out <output_path> options

       This program converts rgb and depth tiff files to pcd files Options:

           -v Set verbose output

           -c Create color pcd output

           -d Create depth output

           -b Set to save pcd binary, otherwise ascii

           -f Focal length, default 525

           -h This help
