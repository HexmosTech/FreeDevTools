NAME
       gts2stl - converts a GTS file to STL format


SYNOPSIS
       gts2stl [OPTIONS] < input.gts > output.stl


DESCRIPTION
       This manual page documents briefly the gts2stl command.


OPTIONS
       This  program follow the usual GNU command line syntax, with long options starting with two dashes (`-').
       A summary of options is included below.

       -r, --revert
              Revert face normals.

       -v, --verbose
              Display surface statistics.

       -h, --help
              Display the help and exit.
