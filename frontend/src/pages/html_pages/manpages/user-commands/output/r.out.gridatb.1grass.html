NAME
       r.out.gridatb  - Exports GRASS raster map to GRIDATB.FOR map file (TOPMODEL).


KEYWORDS
       raster, export, output


SYNOPSIS
       r.out.gridatb
       r.out.gridatb --help
       r.out.gridatb input=name output=name  [--overwrite]  [--help]  [--verbose]  [--quiet]  [--ui]

   Flags:
       --overwrite
           Allow output files to overwrite existing files

       --help
           Print usage summary

       --verbose
           Verbose module output

       --quiet
           Quiet module output

       --ui
           Force launching GUI dialog

   Parameters:
       input=name [required]
           Name of input raster map

       output=name [required]
           Name for output file


DESCRIPTION
       r.out.gridatb exports a GRASS raster map to GRIDATB.FOR map file (TOPMODEL)

        r.topmodel, r.in.gridatb
