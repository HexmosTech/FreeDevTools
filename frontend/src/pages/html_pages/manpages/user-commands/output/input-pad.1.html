NAME
       input-pad - On-screen Input Pad to Send Characters with Mouse


SYNOPSIS
       input-pad


DESCRIPTION
       input-pad  is  a tool to send a character on button to text applications.  It provides the GTK+ based GUI
       and can send characteres on the GTK+ buttons.

       Homepage: http://code.google.com/p/input-pad/
