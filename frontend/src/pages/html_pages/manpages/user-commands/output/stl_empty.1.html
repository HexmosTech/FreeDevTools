NAME
       stl_empty - stl_empty outputs an empty STL file.


DESCRIPTION
       stl_empty outputs an empty STL file.

       usage: stl_empty <output file>

              Outputs  a  properly  formatted,  but empty stl file (just an 80 byte header and a 0 indicating no
              triangles).
