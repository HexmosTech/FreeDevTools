NAME
       jack_simple_client - The JACK Audio Connection Kit example client


SYNOPSIS
       jack_simple_client client-name

       The client-name must be a yet unused client name.


DESCRIPTION
       jack_simple_client  is  an  example client for the JACK Audio Connection Kit. It creates two output ports
       (client-name:output1 and client-name:output2) and plays a sine wave to these ports.


EXAMPLE
       jack_simple_client in_process_test
