NAME
       gh-org - Manage organizations


SYNOPSIS
       gh org <command> [flags]


DESCRIPTION
       Work with Github organizations.


GENERAL COMMANDS
       gh-org-list(1)
              List organizations for the authenticated user.


EXAMPLE
       $ gh org list
