NAME
       /cmd - the Profanity /cmd command.


DESCRIPTION
       Execute ad hoc commands.


SYNOPSIS
       /cmd list [<jid>]

       /cmd exec <command> [<jid>]


ARGUMENTS
       list
           List supported ad hoc commands.

       exec <command>
           Execute a command.


EXAMPLES
       /cmd list

       /cmd exec ping
