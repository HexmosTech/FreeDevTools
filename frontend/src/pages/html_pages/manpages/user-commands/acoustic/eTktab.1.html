NAME
       eTktab - Tk based ascii guitar tab editor


SYNTAX
       eTktab <filename>


DESCRIPTION
       This  program is used to write out guitar tablature in the typical style of ASCII tab, often found around
       the internet. The code is based on TkTab by Giovanni Chierico. Many of  the  ideas  for  the  alterations
       found here came from EMACS tablature mode by Mark R. Rubin.
