NAME
       snek-lilypad-instal - Install Snek to LilyPad


SYNOPSIS
       snek-lilypad-install [OPTION]... [COMMAND]


DESCRIPTION
       snek-lilypad-install  installs  the  Snek  programming  language on an ATmega328P LilyPad board using the
       Optiboot boot loader.


OPTIONS
       -hex <snek.hex>
              Specifies the hex file to load to the board. The default is the  currently  installed  version  of
              Snek.

       -port <port>
              Specifies the serial port to use. The default is /dev/ttyUSB0.

       -mcu <MCU name>
              Specifies  which  micro  controller  to  expect  to find on the board. Some boards have m328 while
              others have m328p. The default is atmega328p, but  if  you  get  an  error  message  indicating  a
              mismatch, you can try the other.


OPTIBOOT
       Snek  for  the  LilyPad  requires Optiboot. If your board does not already have Optiboot, you can use the
       Arduino IDE to install Optiboot on your LilyPad. Follow  the  instructions  on  the  Optiboot  web  site:
       https://github.com/Optiboot/optiboot to learn how to do that.
