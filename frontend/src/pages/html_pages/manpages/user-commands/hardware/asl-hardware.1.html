NAME
       asl-hardware - display certain capabilities of the available OpenCL-enalbed devices


SYNOPSIS
       asl-hardware


DESCRIPTION
       Display certain essential capabilities of all available OpenCL-enalbed devices.  This information is use‐
       ful for configuration of kernels automatically generated by the computational engine of the Advanced Sim‐
       ulation Library.
