NAME
       smbios-passwd - src/bin/smbios-passwd


SYNOPSIS
       smbios-passwd


OPTIONS
       --version
              show program's version number and exit

       -h, --help
              show this help message and exit

       --info Show password information

       -v, --verbose
              Display more verbose output.

       -q, --quiet
              Minimize program output. Only errors and warnings are displayed.

       --trace
              Enable verbose function tracing.

       --logconfig=LOGCONFIG
              Specify alternate config log.

       --password=PASSWORD
              BIOS Setup password for set/activate operations.

       -r, --rawpassword
              Do not auto-convert password to scancodes.

       --memory-dat=MEMORY_DAT
              Path to a memory dump file to use instead of real RAM

       --cmos-dat=CMOS_DAT
              Path to a CMOS dump file to use instead of real CMOS


SEE ALSO
       The  full  documentation  for  smbios-passwd  is maintained as a Texinfo manual.  If the info and smbios-
       passwd programs are properly installed at your site, the command

              info smbios-passwd

       should give you access to the complete manual.
