NAME
       hime-ts-edit - tsin vocabulary database editor for hime


SYNOPSIS
       hime-ts-edit


DESCRIPTION
       hime-ts-edit  is  the  tsin  vocabulary  database  editor,  which  you can search and delete the database
       entries.
