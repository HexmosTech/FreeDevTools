NAME
       dump-stl - Dump STL into Perl syntax for writing Slic3r tests


SYNOPSIS
       dump-stl file.stl
       dump-stl modelname file.stl


DESCRIPTION
       dump-stl  is  a script which dumps an STL file into Perl syntax for writing Slic3r tests, or dumps a test
       model into an STL file.
