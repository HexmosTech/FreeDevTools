NAME
       ghwdump - Dumps info contained in GHW files


SYNTAX
       ghwdump [option]... GHWFILE [[GHWFILE]...]


DESCRIPTION
       Dumps information contained in GHW files for debugging purposes.


OPTIONS
       -t     Display types

       -h     Display hierarchy

       -H     Display hierarchy with full pathnames

       -T     Display time

       -s     Display signals (and time)

       -S     Display strings

       -f LIST
              List of signals to display (default: all, example: -f 1,3,5-7,21-33)

       -l     Display list of sections

       -v     Verbose
