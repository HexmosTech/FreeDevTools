NAME
       gfaim - a recipes database and search engine


DESCRIPTION
       This manual page explains the gfaim program.

       gfaim  allows  you to search into a database of about 10'000 recipes. Juste enter the keyword you want in
       the search box and gfaim finds the recipes matching your request.
