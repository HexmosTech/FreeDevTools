NAME
       aiocoap-keygen - A tool for creating key pairs for use in credentials Note that this tool


DESCRIPTION
       usage: aiocoap-keygen [-h] [--version] {generate} ...

       A  tool  for  creating key pairs for use in credentials Note that this tool operates on secrets stored in
       unencrypted files, protectd by restrictively set file system permissions. While this is  common  practice
       with many tools in the UNIX world, it might be surprising to users coming from multi-factor environments.

   positional arguments:
              {generate}

       generate
              This generates an EDHOC key, stores it in a key file that is only readable by the user, and prints
              the corresponding public key information in a way suitable for inclusion in credentials maps.

   options:
       -h, --help
              show this help message and exit

       --version
              show program's version number and exit
