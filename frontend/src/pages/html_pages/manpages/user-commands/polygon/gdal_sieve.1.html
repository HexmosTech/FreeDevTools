NAME
       gdal_sieve - Removes small raster polygons.


SYNOPSIS
          gdal_sieve [--help] [--help-general]
                        [-q] [-st threshold] [-4] [-8] [-o name=value]
                        <srcfile> [-nomask] [-mask filename] [-of format] [<dstfile>]


DESCRIPTION
       gdal_sieve  removes  raster polygons smaller than a provided threshold size (in pixels) and replaces them
       with the pixel value of the largest neighbour polygon. The result can be written  back  to  the  existing
       raster band, or copied into a new file.

       The input dataset is read as integer data which means that floating point values are rounded to integers.
       Re-scaling  source  data  may  be necessary in some cases (e.g. 32-bit floating point data with min=0 and
       max=1).

       Additional details on the algorithm are available in the GDALSieveFilter() docs.

       NOTE:
          gdal_sieve is a Python utility, and is only available if GDAL Python bindings are available.
