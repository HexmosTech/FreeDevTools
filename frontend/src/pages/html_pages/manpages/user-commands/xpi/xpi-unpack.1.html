NAME
       xpi-unpack - unpack a XPI file to a directory


SYNOPSIS
       xpi-unpack <input_XPI_file> <output_directory>


DESCRIPTION
       xpi-pack unpacks a XPI file to the specified output directory. It takes in consideration JAR files, which
       are  usually  shipped  inside XPI files, and unpacks them, in order to be able to distribute them through
       the repositories. The unpacked JAR files are packed during build process on call of xpi-pack script.

       It is distributed as a part of mozilla-devscripts package. It should be called by developer, when the XPI
       file needs to be unpacked during packaging.


OPTIONS
       None.


ENVIRONMENT VARIABLES
       None.


EXAMPLE
       xpi-unpack ubufox.xpi ubufox
              Unpacks the contents of ubufox.xpi to directory ubufox.
