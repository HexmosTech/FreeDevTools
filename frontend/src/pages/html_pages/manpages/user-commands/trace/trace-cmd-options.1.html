NAME
       trace-cmd-options - list available options from trace-cmd plugins


SYNOPSIS
       trace-cmd options


DESCRIPTION
       The trace-cmd(1) options command will examine all the trace-cmd plugins that are used by trace-cmd
       report(1) and list them.

       trace-cmd(1), trace-cmd-record(1), trace-cmd-start(1), trace-cmd-stop(1), trace-cmd-extract(1),
