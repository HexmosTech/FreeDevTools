NAME
       arch - Display machine architecture


SYNOPSIS
       arch [-h|--help] [-V|--version]


DESCRIPTION
       Display machine architecture


OPTIONS
       -h, --help
              Print help

       -V, --version
              Print version


EXTRA
       Determine architecture name for current machine.
