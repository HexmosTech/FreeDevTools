NAME
       unixcat - prints data from a UNIX domain socket.


SYNOPSIS
       unixcat socket


DESCRIPTION
       unixcat connects to the UNIX domain socket socket.  It prints any data it receives.


SEE ALSO
       unixserver(1), unixclient(1)
