NAME
       hostbyname - Expand domain or urls list with IP addresses


SYNOPSIS
       hostbyname < urls > urls+ip

       hostbyname  < domains > domains+ip


DESCRIPTION
       Hostbyname takes a squidGuard domain or urls list and does some half harted effort to expand it with the
       corresponding IP-addresses.
