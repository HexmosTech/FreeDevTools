NAME
       sympa-create - Create a list


SYNOPSIS
       "sympa create" "--input-file="/path/to/file.xml [ domain ]

       "sympa create" "--input-file="/path/to/file.xml family"@@"domain


DESCRIPTION
       Create a list with the XML file under specified domain or family.  If the context was not specified,
       primary domain will be used.
