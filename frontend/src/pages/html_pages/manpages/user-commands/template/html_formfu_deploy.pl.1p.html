NAME
       html_formfu_deploy.pl - deploy local copy of HTML::FormFu template files


VERSION
       version 2.07


SYNOPSIS
       html_formfu_deploy.pl target-directory


DESCRIPTION
       The "html_formfu_deploy.pl" script creates a local copy of the HTML::FormFu template files for
       customization in the directory target-directory.

       If no customization is needed, HTML::FormFu should use the system-wide installation of the template
       files.
