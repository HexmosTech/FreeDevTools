NAME
       jigdump - Dump the contents of a template file


SYNOPSIS
       jigdump <template file>


DESCRIPTION
       jigdump  will  parse  a  template  file and list its contents. Mainly useful just for debugging jigdo and
       friends...


SEE ALSO
       jigdo-file(1), jigit(1), jigsum(1) and mkimage(1).
