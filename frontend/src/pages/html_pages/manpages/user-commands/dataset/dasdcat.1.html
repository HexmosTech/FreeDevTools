NAME
       DASDCAT - display pds datasets and members from a DASD image file


SYNOPSIS
       dasdcat [-i dasd_image [sf=shadowfile] pdsname/spec:flags]


DESCRIPTION
       This manual page documents the dasdcat program.

       The dasdcat utility displays pds datasets and members from a DASD image file.


PARAMETERS
       dasd_image
                 input DASD filename

       shadowfile
                 associated shadow DASD filename (optional)

       pdsname   name of the partitioned dataset from which information will be extracted

       spec      spec can be one of the following :

                 • name    list PDS member name

                 • *       list all PDS members

                 • ?       list the membernames of the PDS

       flags     flags can be one of the following :

                 • c       list members as cards

                 • a       list members in ASCII
