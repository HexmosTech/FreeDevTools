NAME
       mir-shell - Mir Display Server shell launch script


SYNOPSIS
       mir-shell [options] [shell options]


DESCRIPTION
       mir-shell launches a Mir Display Server shell.


OPTIONS
       -terminal <terminal>
              Use <terminal> instead of 'x-terminal-emulator'.

       -bindir <bindir>
              Path to the miral-shell executable [/usr/bin].


SHELL OPTIONS
       See miral-shell(1) for further Mir shell options.
