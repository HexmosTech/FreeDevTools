NAME
       ldnsd - simple daemon example code


SYNOPSIS
       ldnsd port zone zonefile


DESCRIPTION
       ldnsd  is  a  simple  daemon  that  answers queries for a zone.  This is NOT a full-fledged authoritative
       nameserver!


OPTIONS
       ldnsd takes a port, zone and zonefile as arguments.


AUTHOR
       Written by the ldns team as an example for ldns usage.
