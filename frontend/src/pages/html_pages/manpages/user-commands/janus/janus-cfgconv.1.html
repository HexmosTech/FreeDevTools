NAME
       janus-cfgconv - Janus config file conversion utility.


SYNOPSIS
       janus-cfgconv source.[cfg|jcfg] destination.[cfg|jcfg]


DESCRIPTION
       janus-cfgconv  is  a simple utility that allows you to convert Janus INI configuration files to libconfig
       format and viceversa.


OPTIONS
       -h, --help
              Print help and exit


EXAMPLES
       janus-cfgconv janus.cfg janus.jcfg - Convert provided INI file to libconfig format
