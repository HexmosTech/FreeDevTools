Name
       kitten  -  Fast,  statically compiled implementations of various kittens (command line tools for use with
       kitty)


Usage
       kitten command [command options] [command args]


Description
       kitten serves as a launcher for running individual kittens. Each kitten can be run as kitten command. The
       list of available kittens is given below.


Commands
       @ Control kitty remotely. See: kitten-@(1)

       ask
         Ask the user for input. See: kitten-ask(1)

       choose-fonts
         Choose the fonts used in kitty. See: kitten-choose-fonts(1)

       clipboard
         Copy/paste with the system clipboard, even over SSH. See: kitten-clipboard(1)

       diff
         Pretty, side-by-side diffing of files and images. See: kitten-diff(1)

       edit-in-kitty
         Edit a file in a kitty overlay window. See: kitten-edit-in-kitty(1)

       hints
         Select text from screen with keyboard. See: kitten-hints(1)

       hyperlinked-grep
         Add hyperlinks to the output of ripgrep. See: kitten-hyperlinked-grep(1)

       icat
         Display images in the terminal. See: kitten-icat(1)

       mouse-demo
         Demo the mouse handling kitty implements for terminal programs. See: kitten-mouse-demo(1)

       notify
         Send notifications to the user. See: kitten-notify(1)

       query-terminal
         Query the terminal for various capabilities. See: kitten-query-terminal(1)

       run-shell
         Run the user's shell with shell integration enabled. See: kitten-run-shell(1)

       show-key
         Show the codes generated by the terminal for key presses in various keyboard modes.  See:  kitten-show-
         key(1)

       ssh
         Truly convenient SSH. See: kitten-ssh(1)

       themes
         Manage kitty color schemes easily. See: kitten-themes(1)

       transfer
         Transfer files easily over the TTY device. See: kitten-transfer(1)

       unicode-input
         Browse and select unicode characters by name. See: kitten-unicode-input(1)

       update-self
         Update this kitten binary. See: kitten-update-self(1)

       Get help for an individual command by running:

       kitten command -h


Options
       --version
              The current kitten version.

       --help, -h
              Show help for this command
