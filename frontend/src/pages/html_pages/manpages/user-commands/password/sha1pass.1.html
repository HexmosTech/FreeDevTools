NAME
       sha1pass - Create a SHA1 password hash


SYNOPSIS
       sha1pass [PASSWORD] [SALT]


DESCRIPTION
       sha1pass  creates a SHA1 password hash. In the absence of a salt value on the command line, a random salt
       vector will be generated.

       syslinux(1)
