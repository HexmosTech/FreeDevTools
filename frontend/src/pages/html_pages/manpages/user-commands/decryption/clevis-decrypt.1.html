NAME
       clevis-decrypt - Decrypts using the policy defined at encryption time


SYNOPSIS
       clevis decrypt CONFIG < JWE > PT


OVERVIEW
       The clevis decrypt command decrypts data using the policy defined at encryption time. The specific
       decryption pin is inferred during decryption. There are no parameters.
