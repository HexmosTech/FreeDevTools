NAME
       mrpt-performance - Complete performance test for MRPT


SYNOPSIS
       mrpt-performance


DESCRIPTION
       mrpt-performance is a command-line tool for performing a complete test of performance (in execution time)
       of many different modules of MRPT. The results are dumped as an HTML document.
