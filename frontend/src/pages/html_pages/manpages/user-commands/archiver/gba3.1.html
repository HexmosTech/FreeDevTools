NAME
       gba3 - GAMBAS Archiver


SYNOPSIS
       gba3 [options] [<project directory>]
       gba3 -x <archive path> <file>


DESCRIPTION
       Gambas  is  a  free development environment based on a Basic interpreter with object extensions, a mix of
       Java(tm) and Visual Basic(tm).  With Gambas, you can quickly design your program GUI  with  Qt  or  GTK+,
       access  database  systems like MySQL, PostgreSQL, SQlite or ODBC databases, pilot applications with DBus,
       translate your program into many languages, create GStreamer, OpenGL or SDL applications, and so on...

       gba3 is the archiver that allows you to create a standalone one-file executable from a Gambas project  or
       extract a specific file from a Gambas executable (-x option).


OPTIONS
       -h, --help
              display help and exit

       -i, --ignore=PATH
              exclude a specific file or directory

       -l, --list=ARCHIVE
              list archive files

       -L, --license
              display license

       -o, --output=ARCHIVE
              archive path [<project directory>/<project name>.gambas]

       -p, --ignore-public
              exclude the '.public' directory

       -s, --swap
              swap endianness

       -x, --extract=ARCHIVE
              archive path

       -v, --verbose
              verbose output

       -V, --version
              display version
