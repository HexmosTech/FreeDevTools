NAME
       Camera - downloads files from your digital camera


SYNOPSIS
       Camera


DESCRIPTION
       This  manual  page  documents  briefly  the  Camera command.  This manual page was written for the Debian
       distribution because the original program does not have a manual page.

       Camera downloads files from your digital camera. It is based on GNUstep.
