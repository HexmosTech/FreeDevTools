NAME
       ctdbd_wrapper - Wrapper for ctdbd


SYNOPSIS

       ctdbd_wrapper {start | stop}


DESCRIPTION
       ctdbd_wrapper is used to start or stop the main CTDB daemon.

       See ctdb(7) for an overview of CTDB.


SEE ALSO
       ctdbd(1), ctdb.sysconfig(5), ctdb(7), http://ctdb.samba.org/
