NAME
       tabble-wrapper - restart tabble for $LOGNAME


SYNOPSIS
         tabble-wrapper


DESCRIPTION
       After you have added new entries, this program kills existing tabble(1) and restarts it.

       head1 OPTIONS

       See table(1) manual page.


ENVIRONMENT
       The tabble owned by $LOGNAME is killed.


FILES
       None.


SEE ALSO
       tabble(1)
