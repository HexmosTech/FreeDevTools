NAME
       bsdl2jtag - UrJTAG declaration file conversion


SYNOPSIS
       bsdl2jtag bsdlfile jtagfile


DESCRIPTION
       bsdl2jtag  converts BSDL files to JTAG files which can be used by jtag(1).  bsdl2jtag reads from bsdlfile
       and produces its output in the file jtagfile.


OPTIONS
       None.


BUGS
       Bugs are tracked at the project homepage, http://www.urjtag.org


SEE ALSO

       an independent project named UrJTAG in late 2007.
