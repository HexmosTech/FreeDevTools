NAME
       foreman-runner - Run a command with exec.


SYNOPSIS
       foreman-runner [options]... <command> [args]...

       Run supplied command and args with exec, optionally changing directory first


DESCRIPTION
       -d dir    Change directory to dir

       -p        Read $HOME/.profile
