NAME
       mongrel2 - mongrel2 web server


SYNOPSIS
       mongrel2 config.sqlite server_uuid [config_module.so]


DESCRIPTION
       This  starts  th  mongrel2  web  server, using the configuration given in config.sqlite and the specified
       server uuid.

       Optionally, a configuration module can be specified.

       For a complete description, see the mongrel2 documentation ⟨http://mongrel2.org/manual/book-final.html⟩.
