NAME
       thawab-gtk  -  Arabic/Islamic  encyclopedia  system  thawab-server  -  Arabic/Islamic encyclopedia system
       (server only)


SYNOPSIS
       thawab-gtk
       thawab-server


DESCRIPTION
       thawab is an Arabic/Islamic encyclopedia system.

       thawab-gtk is the GUI application.

       thawab-server is a standalone server that runs at port 18080, which can  be  connected  to  using  a  web
       browser.
