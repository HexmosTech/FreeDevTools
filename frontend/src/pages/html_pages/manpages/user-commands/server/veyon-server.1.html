NAME
       veyon-server - Veyon Server


SYNOPSIS
       veyon-server


DESCRIPTION
       VEYON-SERVER  is  a  server  application  which  provides  access  to  a  computer as well as control and
       application functions.

       Under normal conditions this program is started by the Veyon  Service  automatically  and  with  elevated
       privileges so it canâ€™t be terminated by users.
