NAME
       mate-maximus - force all windows to be full-screen


SYNOPSIS
       mate-maximus


DESCRIPTION
       This manual page documents briefly maximus.

       mate-maximus  is an X program for the MATE Desktop Environment that forces all windows to be full-screen.
       This tool is shipped with the MATE netbook utilities.


OPTIONS
       mate-maximus takes no command line arguments.
