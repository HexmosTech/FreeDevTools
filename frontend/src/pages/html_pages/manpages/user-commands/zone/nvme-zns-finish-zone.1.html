NAME
       nvme-zns-finish-zone - Finishes one or all zones


SYNOPSIS
       nvme zns finish-zone <device> [--namespace-id=<NUM> | -n <NUM>]
                                                       [--start-lba=<LBA> | -s <LBA>]
                                                       [--select-all | -a]
                                                       [--timeout=<timeout> | -t <timeout>]


DESCRIPTION
       For the NVMe device given, issues the Zone Management Send command with the "Finish Zone" action. This
       will transition the zone to the full state on success.

       The <device> parameter is mandatory and may be either the NVMe character device (ex: /dev/nvme0), or a
       namespace block device (ex: /dev/nvme0n1).


OPTIONS
       -n <NUM>, --namespace-id=<NUM>
           Use the provided namespace id for the command. If not provided, the namespace id of the block device
           will be used. If the command is issued to a non-block device, the parameter is required.

       -s <lba>, --start-lba=<lba>
           The starting LBA of the zone to finish.

       -a, --select-all
           Select all zones for this action.

       -t <timeout>, --timeout=<timeout>
           Override default timeout value. In milliseconds.
