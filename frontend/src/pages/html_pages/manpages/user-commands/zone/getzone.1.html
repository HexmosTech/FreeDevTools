NAME
       getzone - get dns zone from server


DESCRIPTION
       getzone transfers a user-specified dns zone from a zone server and displays it in csv1 format on the
       standard output. This program is here for compatibility with older setups that use getzone to get zone
       files; newer setups may wish to consider using the fetchzone tool to obtain csv2-compatible zone files.


USAGE
       getzone zone_name zone_server_IP [query_class]


OPTIONS
       zone_name
           Name of the dns zone to be transferred.

       zone_server_IP
           IP address of dns server

       query_class
           Optional  argument  which  can change the query class from 1 (the default) to 255. This may be needed
           for some versions of Bind.


EXAMPLES
       To obtain the zone example.com from the server 192.168.9.8:

           getzone example.com 192.168.9.8

       To obtain the zone example.org from the server 10.9.8.78 using a query class of 255:

           getzone example.com 10.9.8.78 255
