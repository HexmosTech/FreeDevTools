NAME
       ddv - DeuxD-Viewer for multiple sequence alignment


SYNOPSIS
       ddv [-] [-e F] [-g F]


DESCRIPTION
       ddv is a standalone graphical viewer for multiple sequence alignment.


OPTIONS
       A summary of options is included below.

       -      Print usage message

       -e F   Do not allow edits

       -g F   Do not allow gaps
