SYNOPSIS
          shell$ prun ...options...


DESCRIPTION
       prun  submits a job to the PMIx Reference Runtime Environment (PRRTE).  The user has control over various
       distributed virtual machine (DVM) options.

       Much of this same help documentation for this command is also provided through prun --help [topic].
