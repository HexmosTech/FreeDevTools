SYNOPSIS
          shell$ prterun ...options...


DESCRIPTION
       prterun  submits  a  job to the PMIx Reference Runtime Environment (PRRTE).  A default set of distributed
       virtual machine (DVM) options are used; use prun(1) if you wish to utilize specific DVM options.

       Much of this same help documentation for this command is also provided through prun --help [topic].
