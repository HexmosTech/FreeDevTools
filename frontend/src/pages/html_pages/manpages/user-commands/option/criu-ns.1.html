NAME criu-ns - run criu in different namespaces


SYNOPSIS

       criu-ns dump -t PID [<options>]

       criu-ns pre-dump -t PID [<options>]

       criu-ns restore [<options>]

       criu-ns check [<options>]


DESCRIPTION

       The criu-ns command executes criu in a new PID and mount namespace. The purpose of this wrapper script is
       to  enable restoring a process tree that might require a specific PID that is already used on the system;
       so called "PID mismatch" problem.
