NAME
       Shape_PointInPoly - calculate a point in a polygon

       ##SYNOPSIS Shape_PointInPoly shpfile_path quality

       ##DESCRIPTION Commandline program to generate points-in-polygons from a shapefile as a shapefile.

       ##OPTIONS

       shpfile_path
              shapefile

       quality
              quality of points to create
