NAME
       maf-sort - Sort MAF-format alignments by sequence name


SYNOPSIS
       maf-sort [options] my-alignments.maf


OPTIONS
       -h     show this help message and exit

       -d     only print duplicate alignments

       -n     sort by the n-th sequence (default: 1)
