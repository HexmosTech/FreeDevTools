NAME
       tulip — program to visualize very large graphs


SYNOPSIS
       tulip


DESCRIPTION
       tulip is an information visualization framework dedicated to the analysis and visualization of relational
       data.   Invoking  tulip  opens  the Tulip launcher which can be used to browse projects, perspectives and
       plugins.

       This manual page was written for the Debian GNU/Linux distribution because the original program does  not
       have a manual page.  Instead, it has documentation in HTML format; see below.


SEE ALSO
       tulip_perspective(1), tulip-config(1)

       The  programs  are  documented fully by the Tulip User Manual available on the Tulip website (link to URL
       https://tulip.labri.fr/Documentation/current/tulip-user/html/index.html)
