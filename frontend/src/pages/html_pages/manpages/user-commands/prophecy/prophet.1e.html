NAME
       prophet - Scan one or more sequences with a Gribskov or Henikoff profile


SYNOPSIS

       prophet -sequence seqall -infile infile -gapopen float -gapextend float -outfile align

       prophet -help


DESCRIPTION
       prophet is a command line program from EMBOSS (“the European Molecular Biology Open Software Suite”). It
       is part of the "Nucleic:Profiles,Protein:Profiles" command group(s).


OPTIONS
   Input section
       -sequence seqall

       -infile infile

   Required section
       -gapopen float
           Default value: 1.0

       -gapextend float
           Default value: 1.0

   Output section
       -outfile align
