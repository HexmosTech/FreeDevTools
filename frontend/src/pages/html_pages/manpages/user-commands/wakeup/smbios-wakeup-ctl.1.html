NAME
       smbios-wakeup-ctl - src/bin/smbios-wakeup-ctl


SYNOPSIS
       smbios-wakeup-ctl


OPTIONS
       --version
              show program's version number and exit

       -h, --help
              show this help message and exit

       --hour=HOUR
              Set wakeup hour. Must be 0-23

       --minute=MINUTE
              Set wakeup minute. Must be 0-59

       --everyday
              Set wakeup for every day

       --weekday
              Set wakeup for weekdays only

       --disable
              Disable wakeups

       -v, --verbose
              Display more verbose output.

       -q, --quiet
              Minimize program output. Only errors and warnings are displayed.

       --trace
              Enable verbose function tracing.

       --logconfig=LOGCONFIG
              Specify alternate config log.

       --password=PASSWORD
              BIOS Setup password for set/activate operations.

       -r, --rawpassword
              Do not auto-convert password to scancodes.

       --memory-dat=MEMORY_DAT
              Path to a memory dump file to use instead of real RAM

       --cmos-dat=CMOS_DAT
              Path to a CMOS dump file to use instead of real CMOS

       wakeup-ctl programs are properly installed at your site, the command
