NAME
       fvwm-crystal.videomodeswitch- - xrandr wrapper for FVWM-Crystal


SYNOPSIS


       fvwm-crystal.videomodeswitch-

DESCRIPTION
       fvwm-crystal.videomodeswitch- will use xrandr to change the current mode to the next one or the previous
       one.

       This manual page was written for the Debian distribution because the original program does not have a
       manual page.
