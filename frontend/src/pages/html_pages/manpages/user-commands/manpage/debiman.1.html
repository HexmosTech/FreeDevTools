NAME
       debiman - generate a static manpage HTML repository out of a Debian archive


SYNOPSIS
       debiman [flag]


DESCRIPTION
       debiman makes (Debian) manpages accessible in a web browser. Its goals are, in order:

       completeness: all manpages in Debian should be available.

       visually appealing and convenient: reading manpages should be fun, convenience features (e.g. permalinks,
       URL redirects, easy navigation) should be available

       speed:  manpages should be quick to load, new manpages should be quickly ingested, the program should run
       quickly for pleasant development


OPTIONS
       Run debiman --help for more details.
