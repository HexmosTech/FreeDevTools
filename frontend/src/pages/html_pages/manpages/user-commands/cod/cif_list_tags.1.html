NAME
       cif_list_tags - parse CIF file(s) and output a list of CIF data items.


SYNOPSIS
       /build/cod-tools-hi23w0/cod-tools-3.10.0+dfsg/src/components/codcif/cif_list_tags --options < input.cif

       /build/cod-tools-hi23w0/cod-tools-3.10.0+dfsg/src/components/codcif/cif_list_tags --options input.cif

       /build/cod-tools-hi23w0/cod-tools-3.10.0+dfsg/src/components/codcif/cif_list_tags   --options  input1.cif
       input2.cif inputs*.cif


DESCRIPTION
       Parse CIF file(s) and output a list of CIF data items.


OPTIONS
       -c, --compile-only
                         Only compile the CIF (check syntax). Prints out
                         the filename and 'OK' or 'FAILED' to STDOUT, along
                         with error messages to STDERR.

       -q, --quiet
                         Be quiet, only output error messages and data.

       -q-, --no-quiet, --verbose
                         Produce verbose output of the parsing process.

       --version
                         Print program version (SVN Id) and exit.

       --help
                         Print short usage message (this message) and exit.
