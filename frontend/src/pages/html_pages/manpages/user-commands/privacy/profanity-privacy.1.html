NAME
       /privacy - the Profanity /privacy command.


DESCRIPTION
       Configure  privacy settings.Also check the the following settings in /account: clientid to set the client
       identification namesession_alarm to configure an alarm when more clients log in.


SYNOPSIS
       /privacy logging on|redact|off

       /privacy os on|off


ARGUMENTS
       logging on|redact|off
           Switch chat logging. This will also disable  logging  in  the  internally  used  SQL  database.  Your
           messages  will  not  be  saved anywhere locally. This might have unintended consequences, such as not
           being able to decrypt OMEMO encrypted messages received later  via  MAM,  and  should  be  used  with
           caution.

       os on|off
           Choose whether to include the OS name if a user asks for software information (XEP-0092).


EXAMPLES
       /privacy logging off

       /privacy os off
