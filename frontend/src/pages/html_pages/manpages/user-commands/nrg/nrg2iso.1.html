NAME
       nrg2iso - extract ISO9660 data from Nero ".nrg" files


SYNOPSIS
       nrg2iso [nrg-file] [iso-file]

       nrg2iso [--help|--version]


DESCRIPTION
       This manual page documents briefly the nrg2iso commands.

       nrg2iso is a program that extracts ISO9660 data from Nero ".nrg" CD-ROM image files.


OPTIONS
       --help

          Display the help screen.

       --version

          Display the version.
