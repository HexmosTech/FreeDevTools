NAME
       legend - create a legend image from a mapfile


SYNOPSIS
       legend [mapfile outputimage | -v]


DESCRIPTION
       legend  creates  a  legend  image from a mapfile.  Output format depends on the graphics library used for
       rendering.


OPTIONS
       mapfile
              Path to the mapfile.

       outputimage
              Path to the output image file.

       -v     Display mapserver version and build options.
