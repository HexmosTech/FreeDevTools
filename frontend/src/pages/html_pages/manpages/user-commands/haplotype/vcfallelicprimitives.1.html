NAME
       vcfallelicprimitives - Converts stdin or given VCF file and reduces alleles.


SYNOPSIS
       vcfallelicprimitives


DESCRIPTION
       vcfallelicprimitives converts stdin or given VCF file to tab-delimited format,

       Realign  reference  and  alternate  alleles with WFA, parsing out the primitive alleles into multiple VCF
       records.  New records have IDs that reference the source record ID.  Genotypes are handled.  Deletion al‐
       leles will result in haploid (missing allele) genotypes.

       Note that this tool is considered legacy and will emit a warning!  Use vcfwave instead.

   Options
       -h, –help
              shows help message and exits.

       See more below.


EXIT VALUES
       0      Success

       not 0  Failure
