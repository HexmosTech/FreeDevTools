NAME
       dc1394_reset_bus - resets the IEEE1394 bus


SYNOPSIS
       dc1394_reset_bus


DESCRIPTION
       Finds the 1394 bus of the first camera found and resets it


OPTIONS
       There are no command-line options for this program.
