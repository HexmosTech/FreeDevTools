NAME
       gbmave - Produce moving average from data


SYNOPSIS
       gbmave [options]


DESCRIPTION
       Compute moving average along each column. Data are read from standard input.


OPTIONS
       -s     number of steps to average (default 10)

       -n     print a progressive tailing integer

       -F     specify the input fields separators (default " \t")

       -h     this help
