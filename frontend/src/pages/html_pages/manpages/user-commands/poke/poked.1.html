NAME
       poked - The poke daemon


SYNOPSIS
       poked [OPTION]...


DESCRIPTION
       Daemonized poke interactive editor for binary files.

       -h, --help
              print a help message and exit

       -v, --version
              show version and exit

       -d, --debug
              be more verbose during the execution

       -S, --socket-path=PATH
              path of unix domain socket to listen on
