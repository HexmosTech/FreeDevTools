NAME
       pan-config  -  PAN  common  metapackage  helper pan-<metapackage_name> - package information and auto-apt
       helper


SYNOPSIS
       pan-config pan-<metapackage_name>


DESCRIPTION
       Print just a simple information page about every pan-* package of the  Photon  And  Neutron  Blend.  Each
       metapackage  contains  a  /usr/bin/pan-<metapackage_name> file which should print some useful information
       and could be used as auto-apt helper.
