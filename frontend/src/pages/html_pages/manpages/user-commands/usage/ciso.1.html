NAME
       Usage: - manual page for Usage: ciso level infile outfile


SYNOPSIS
       ciso level infile outfile


DESCRIPTION
              level: 1-9 compress ISO to CSO (1=fast/large - 9=small/slow

       0      decompress CSO to ISO

              level: 1-9 compress ISO to CSO (1=fast/large - 9=small/slow

       0      decompress CSO to ISO
