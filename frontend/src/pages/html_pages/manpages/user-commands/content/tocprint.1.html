NAME
     tocprint  −  prints  the table of contents for a PSI striped
     direct‐access file.


DESCRIPTION
     The program tocprint prints the  table  of  contents  for  a
     given PSI unit number.


FILES REQUIRED
          input.dat


INPUT OPTIONS
     The following command‐line options are available:

     ‐unit,‐u
          The unit number of the PSI direct‐access file of inter‐
          est.

     ‐cc  The  string  identifier  of  the  file  in  as given in
          $PSI3/include/ccfiles.h.   For  example,  for  the   CC
          amplitude  file, one may specify: tocprint ‐cc TAMPS or
          tocprint ‐cc CC_TAMPS.

          Note that the  FILES:  section  of  input.dat  provides
          tocprint  with  the  necessary volume and name informa‐
          tion.
