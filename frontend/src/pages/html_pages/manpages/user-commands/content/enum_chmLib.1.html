NAME
       enum_chmLib - Lists the contents of a chm file.


SYNOPSIS
       enum_chmLib [FILE]...


DESCRIPTION
       enum_chmLib lists the contents of the specified chm file to stdout.


USAGE
       enum_chmLib <chmfile>


OPTIONS
       enum_chmLib has no options.
