NAME
       bxl2txt - decode a binary BXL file and dump the plain text content


SYNPOSIS
       bxl2txt < infile.bxl > outfile.txt


DECSRIPTION
       bxl2txt  reads  and  decodes a binary bxl file in its standard input and prints the plain text content on
       its standard output.
