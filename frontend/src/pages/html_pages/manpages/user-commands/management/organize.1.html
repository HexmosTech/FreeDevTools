NAME
       organize - organize


SYNOPSIS
       organize [OPTIONS] COMMAND [ARGS]...


DESCRIPTION
       organize

       The file management automation tool.


OPTIONS
       --version
              Show the version and exit.


COMMANDS
       run
         Organizes your files according to your rules.
         See organize-run(1) for full documentation on the run command.

       sim
         Simulates a run (does not touch your files).
         See organize-sim(1) for full documentation on the sim command.

       edit
         Edit the rules.
         See organize-edit(1) for full documentation on the edit command.

       check
         Checks whether a given config file is valid.
         See organize-check(1) for full documentation on the check command.

       reveal
         Reveals the default config file.
         See organize-reveal(1) for full documentation on the reveal command.

       schema
         Prints the json schema for config files.
         See organize-schema(1) for full documentation on the schema command.

       docs
         Opens the documentation.
         See organize-docs(1) for full documentation on the docs command.
