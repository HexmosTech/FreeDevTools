NAME

       galera_new_cluster - starting a new Galera cluster

DESCRIPTION

       Used to bootstrap a new Galera Cluster when all nodes are down.  Run galera_new_cluster on the first node
       only.  On the remaining nodes simply run 'service @DAEMON_NAME@ start'.
