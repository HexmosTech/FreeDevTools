NAME
       rifiuti - tool to extract information from MS recycle bins info2 files.


SYNOPSIS
       rifiuti [-t] FILE


DESCRIPTION
       rifiuti  is  a tool to extract valuable information (from a forensics investigator point of view) from MS
       recycle bins info2 files.

       It will extract the deleted time, original drive number, original path ans  size  of  the  deleted  files
       found in the recycle bin.


OPTIONS
       -t FD
           Change the default field delimiter (TAB) to FD.

       <file>
           INFO2 file to parse.
