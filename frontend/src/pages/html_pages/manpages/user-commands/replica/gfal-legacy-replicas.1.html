NAME
       gfal-legacy-replicas - list replicas


SYNOPSIS
       gfal-legacy-replicas [ -h ] [ -V ] [ -v ] [ -D DEFINITION ] [ -C CLIENT-INFO ] [ -t TIMEOUT ]  lfc


DESCRIPTION
       gfal-legacy-replicas List the replicas of an entry in a catalog.


OPTIONS
       -h, --help
              show help about the command.

       -V, --version
              output version information and exit.

       -v, --verbose
              verbose mode. Repeat up to 3 times for the most verbose mode.

       -D, --definition
              define properties for gfal2.

       -C, --client-info
              provide custom client-side information

       -t, --timeout
              global  timeout for the execution of the command. Command is interrupted if time expires before it
              finishes.

       lfc    LFC entry (lfc:// or guid:)


EXAMPLES
       gfal-legacy-replicas guid:12345-678-abcdef

       srm://endpoint.cern.ch/path/group


FILES
       /etc/gfal2.d/*
              All configuration files in this directory affect the behavior of the underlying gfal2.
