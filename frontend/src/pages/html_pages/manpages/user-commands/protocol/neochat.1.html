NAME
       neochat - Client for interacting with the matrix messaging protocol


SYNOPSIS

       neochat [URI]


DESCRIPTION
       neochat is a chat application for the matrix protocol that work on both desktop and mobile.


OPTIONS
       URI
           The matrix uri for an user or a room. e.g matrix:u/user:example.org and matrix:r/root:example.org.
           This will makes NeoChat try to open the given room or conversation.
