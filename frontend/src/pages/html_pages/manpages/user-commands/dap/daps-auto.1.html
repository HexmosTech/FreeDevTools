NAME
       daps-auto - A program in the DAPS software package, a complete environment to build HTML, PDF, ePUB and
       other formats from DocBook XML.


DESCRIPTION
       This manual page documents briefly the daps-auto program.

       The daps-auto program creates documentation output with DAPS and rsync it to given servers. It needs a
       config file in ini style. A sample such file is located at
       /usr/share/daps/init_templates/daps-auto_sample.ini.
