NAME
       fit_affine_dfield - Fit Affine Transformation to Nonrigid Transformation


SYNOPSIS
       fit_affine_dfield InputDField OutputXform


DESCRIPTION
       Fit  a linear affine transformation to a nonrigid transformation, either a B-spline free-form deformation
       or a non-parametric deformation field.
