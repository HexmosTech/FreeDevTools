NAME
       nfs-cat - Utility to read a file off NFS


SYNOPSIS

       nfs-cat <NFS-URL>


DESCRIPTION
       nfs-cat is a utility to read a file off an NFS server.

       Example: Print the content of a file:

           $ nfs-cat nfs://127.0.0.1/data/tmp/foo.c
