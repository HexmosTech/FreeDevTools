NAME
       chiaki — PlayStation remote play client


SYNOPSIS
       chiaki [-e this]  [--example that]


DESCRIPTION
       This manual page documents briefly the chiaki command.

       chiaki is a remote play client for PlayStation consoles.


OPTIONS
       -h           --help
                 Displays help on commandline options.

       --help-all
                 Displays help including Qt specific options.

       --registkey <registkey>
                 Uses specified registkey.

       --morning <morning>
                 Uses specified morning.

       --fullscreen
                 Start window in fullscreen (only for use with stream command)


ARGUMENTS
       command   stream, list

       nickname  Needed for stream command to get credentials for onnecting. Use 'list' to get the nickname.

       host      Address to connect to (when using the stream command)
