NAME
       vimix - Video live mixer

       vimix [OPTIONS] [FILE]


DESCRIPTION
       vimix performs graphical mixing and blending of several movie clips and computer generated graphics, with
       image processing effects in real-time.

       Its  intuitive  and hands-on user interface gives direct control on image opacity and shape for producing
       live graphics during concerts or VJ-ing sessions.

       The output image is typically projected full-screen on an external monitor or a  projector,  but  can  be
       streamed live (SRT) or recorded (no audio).


OPTIONS
       -V, --version
              Output the version number of vimix and exit.

       -C, --clean
              Clean the user stored preferences of vimix and exit.

       -T, --test
              Perform a minimal rendering test and exit.


FILE
       Launches vimix and tries to open the provided session file (.mix extension).
