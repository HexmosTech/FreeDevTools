NAME
       gst-stats-1.0 - print info gathered from a GStreamer log file


SYNOPSIS
       gst-stats-1.0 [OPTION...] FILE


DESCRIPTION
       gst-stats-1.0  is  a  tool  that analyses information collected from a log file based on GStreamer tracer
       messages.


OPTIONS
       gst-stats-1.0 accepts the following arguments and options:

       FILE    Name of a file

       -h, --help
               Print help synopsis and available FLAGS

       --gst-help-all
               Show all help options

       --gst-help-gst
               Show Gstreamer options
