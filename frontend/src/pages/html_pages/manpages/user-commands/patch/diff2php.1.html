Name
       diff2php — transform a patch into a self-serving PHP file


Syntax
       diff2php file


Description
       Create  a  PHP  file  from  a  patch.  The PHP file will server the patch as colored HTML when viewed, or
       alternatively as raw text/plain for downloading.


See also
       hxtools(7)
