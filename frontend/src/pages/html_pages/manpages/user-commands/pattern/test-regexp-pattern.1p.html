NAME
       test-regexp-pattern - Test Regexp::Pattern patterns in all modules of your distro


VERSION
       This document describes version 0.010 of test-regexp-pattern (from Perl distribution Test-Regexp-
       Pattern), released on 2022-07-24.


SYNOPSIS
       In your distro directory:

        % test-regexp-pattern [options]


DESCRIPTION
       This script is a command-line interface for Test::Regexp::Pattern's "regexp_patterns_in_all_modules_ok()"
       function.
