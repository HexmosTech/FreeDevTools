NAME
       ng_stl  — Netgen (nglib) STL Testing


SYNOPSIS
       ng_stl [OPTIONS...]  [STL_filename]


DESCRIPTION
       This manual page documents briefly the ng_stl and bar       commands.

       This  manual  page  was  written for the Debian distribution because the original program does not have a
       manual page.

       ng_stl is an STL testing utility for nglib.

       Homepage: https://ngsolve.org/
