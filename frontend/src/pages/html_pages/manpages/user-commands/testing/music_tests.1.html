NAME
       music_tests - run MUSIC tests


SYNOPSIS
       music_tests [options]


DESCRIPTION
       Run a suite of tests validating the correct operation of MUSIC.


OPTIONS
       --help Show program options and exit.

       --showme-tests
              Print test names and exit.

       --output-dir=<PATH>
              Output directory (default: ./reports).

       music(1).
