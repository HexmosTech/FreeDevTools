NAME
       lou_checkyaml - A tool to check Braille tests defined in a YAML file


SYNOPSIS
       lou_checkyaml YAML_TEST_FILE


DESCRIPTION
       Run  the  tests  defined in the YAML_TEST_FILE. Return 0 if all tests pass or 1 if any of the tests fail.
       The details of failing tests are printed to stderr.

       -h, --help
              display this help and exit

       -v, --version
              display version information and exit

       --verbose
              report expected failures
