NAME
       rmidish — front-end to midish(1), using the readline(3) library


SYNOPSIS
       rmidish


DESCRIPTION
       The  rmidish  utility is a very simple front-end to midish(1) that uses the readline(3) library.  It runs
       midish(1) in the background, issues an interactive prompt and passes user  commands  to  midish(1).   The
       initial purpose of rmidish was to illustrate how to use the -v option of midish(1) to create a front-end,
       but it can be run instead of midish(1) to make the later more user-friendly when used interactively.


SEE ALSO
       midish(1), readline(3)
       Midish user's manual and tutorial
       http://caoua.org/midish/


BUGS
       The midish(1) executable has to be in the same directoy as rmidish
