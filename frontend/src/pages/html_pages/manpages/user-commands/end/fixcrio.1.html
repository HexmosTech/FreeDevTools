NAME
       fixcrio - inserts missing CRs at the ends of lines.


SYNOPSIS
       fixcrio prog


DESCRIPTION
       fixcrio runs prog.  It inserts CR at the end of each line of input or output to prog that does not have a
       CR.

       fixcrio runs as a child process of prog.  It exits when it sees the end of prog's output.
