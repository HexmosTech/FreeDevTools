NAME
       prr-edit - Open an existing review in $EDITOR


SYNOPSIS
       prr edit [-h|--help] <PR>


DESCRIPTION
       Open an existing review in $EDITOR


OPTIONS
       -h, --help
              Print help

       <PR>   Pull request to edit (eg. `danobi/prr/24`)
