NAME
       psi2log - PSI metrics monitor and logger


SYNOPSIS
       psi2log [OPTION]...


DESCRIPTION
       psi2log  is  a CLI tool that can check and log PSI metrics from specified target.  psi2log is part of no‚Äê
       hang package.


OPTIONS
   -h, --help
       show this help message and exit

   -t TARGET, --target TARGET
       target (cgroup_v2 or SYTSTEM_WIDE)

   -i INTERVAL, --interval INTERVAL
       interval in sec

   -l LOG, --log LOG
       path to log file

   -m MODE, --mode MODE
       mode (1 or 2)

   -s SUPPRESS_OUTPUT, --suppress-output SUPPRESS_OUTPUT
       suppress output


EXAMPLES
       $ psi2log

       $ psi2log --mode 2

       $ psi2log --target /user.slice --interval 1.5 --log psi.log


SIGNALS
       Sending SIGTERM, SIGINT, SIGQUIT or SIGHUP signals to the psi2log process causes it displays peak  values
       and exits..
