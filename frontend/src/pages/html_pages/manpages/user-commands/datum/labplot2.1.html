NAME
       labplot2 - an application by KDE for interactive graphing and analysis of scientific data.


SYNOPSIS

       labplot2 [LabPlot options] [file]


DESCRIPTION
       LabPlot is a data plotting and analysis tool.

       Its features include 2D and 3D data and function plotting, easy editing of plots, analysis of data and
       functions, support for different worksheets using MDI and LabPlot project files.

       Note that LabPlot is called using labplot2.


LABPLOT OPTIONS
       For a full summary of generic options, run labplot2 --help.

       --no-splash
           Do not show the splash screen.

       --presenter
           Start in the presenter mode.


ARGUMENTS
       file
           The location of the project to be opened, usually ending with .lml.

           The project file can be also compressed, with its filename ending with .lml.gz, .lml.bz2 or .lml.xz.
