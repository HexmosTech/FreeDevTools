NAME
       removeSbeFromMipe.pl - Removes SBE data from MIPE file
         based on MIPE version v0.9
         arguments: * mipe_file
                    * STDIN: tab-delimited list of data, in the following order:                 PCR ID, SNP ID,
       SBE ID


SYNOPSIS
       removeSbeFromMipe.pl my_file.mipe < my_data.txt
