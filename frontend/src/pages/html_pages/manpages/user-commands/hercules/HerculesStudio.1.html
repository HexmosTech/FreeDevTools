NAME
       HerculesStudio - Hercules GUI Interface


SYNOPSIS
       HerculesStudio [-f <hercules-config-file>] [-r <hercules-run-commands-file>] [-h]


DESCRIPTION
       Hercules Studio is a GUI front-end to the Hercules mainframe Emulator


OPTIONS
       -f <hercules-config-file>
              Use <hercules-config-file> as hercules virtual machine configuration file.

       -r <hercules-run-commands-file>
              Run <hercules-run-commands-file> after hercules is started.

       -h     Print help information.
