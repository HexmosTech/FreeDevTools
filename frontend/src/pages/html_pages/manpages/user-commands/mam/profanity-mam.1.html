NAME
       /mam - the Profanity /mam command.


DESCRIPTION
       Enable/Disable  Message  Archive  Management  (XEP-0313)  Currently  MAM  in  groupchats  (MUCs)  is  not
       supported.Use the PG UP key to load more history.


SYNOPSIS
       /mam <on>|<off>


ARGUMENTS
       on|off
           Enable or disable MAM
