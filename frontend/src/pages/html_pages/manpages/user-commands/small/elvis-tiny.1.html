NAME
       elvis-tiny - Tiny vi-compatible editor


DESCRIPTION
       Elvis-tiny is a very small vi-compatible editor with limited capabilities. If you want more features (and
       documentation), install its big brother elvis or any of the other vi clones such as nvi or vim.
