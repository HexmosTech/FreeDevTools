NAME
       quadrature - show a quadrature formula (rheolef-7.2)


SYNOPSIS
       quadrature element quadname


EXAMPLES
         quadrature e 'gauss(3)'
         quadrature T 'gauss(3)'


DESCRIPTION
       Shows a quadrature formula: lists the nodes on the specified reference elements.
