NAME
       mrtg-ip-acct - Returns the number of IP packets that have traversed a network interface


SYNOPSIS
       mrtg-ip-acct [<interface name>]


DESCRIPTION
       mrtg-ip-acct returns the number of bytes that have been received and sent on a given interface. The
       output is written in a form useful as input for the mrtg utility.

       If no interface name is given, the default is eth0.
