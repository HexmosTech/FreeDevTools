NAME
       rtf2brl - Translate an RTF file into an embosser-ready Braille file.


DESCRIPTION
       rtf2brl [OPTION] infile outfile


OPTIONS
       --help Print this message

       --version
              Print version information

       Infile  must  be  a  rich-text (rtf) file. The script first calls the `rtf2xml' program, so you must have
       this installed on your machine.  You can find it on the downloads page of www.abilitiessoft.com.  rtf2brl
       produces an xml file with the root element `doc'. This is  piped  to  `xml2brl'.  The  output  file  from
       `xml2brl' contains much of the formatting, including emphasis, of the rtf file.
