NAME
       stl_merge - stl_merge concatenates multiple STL files.


DESCRIPTION
       stl_merge concatenates multiple STL files.

       usage: stl_merge [ -o <out file> ] [ <in file1> ... ]

              Merges binary stl files into a single file. If no out file is provided, data is output to stdout.
