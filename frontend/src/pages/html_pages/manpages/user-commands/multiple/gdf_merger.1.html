NAME
       gdf_merger - Merge multiple GDF files


SYNOPSIS
       gdf_merger [options]


DESCRIPTION
   Allowed options:
       -h [ --help ]
              produce help message

       -i [ --input-files ] arg input files

       -o [ --output-file ] arg output file
