NAME
       balboa-help - Help about any command


SYNOPSIS
       balboa help [command] [flags]


DESCRIPTION
       Help  provides  help  for  any command in the application.  Simply type balboa help [path to command] for
       full details.


OPTIONS
       -h, --help[=false]      help for help


SEE ALSO
       balboa(1)
