NAME
       grab-merge - grabs a merge's files from merges.ubuntu.com.


SYNOPSIS
       grab-merge <package name>


DESCRIPTION
       grab-merge  is  a  script  that  downloads  a  merge's packaging files and report from merges.ubuntu.com.
       Placing them in a new directory for working from.
