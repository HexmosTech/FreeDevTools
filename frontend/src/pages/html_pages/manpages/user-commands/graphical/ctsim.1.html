NAME
       ctsim — A Graphical Computed Tomography Simulator


SYNOPSIS
       ctsim files-to-open... [OPTIONS]


DESCRIPTION
       ctsim is a graphical user interface to the various components of the CTSim package. The home for CTSim is
       http://www.ctsim.org.


OPTIONS
       --version
               Display version

       --help  Print a terse help message


AUTHORS
       Kevin Rosenberg, M.D. <kevin@ctsim.org>


HISTORY
       CTSim  was  begun in 1983 using MS-DOS and an EGA display adapter. In 1999 it was ported to GNU/Linux and
       later ported to Microsoft Windows.
