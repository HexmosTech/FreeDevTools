NAME
       seek_script - script using OutGuess to find an image that yields the best embedding


SYNOPSIS
        seek_script [param for outguess]


DESCRIPTION
       The  seek_script  scans  the  current directory to find the best JPEG file in which hide the message that
       resides in /tmp/fortune. At the end, the script will print the best and the worst JPEG file in  which  to
       put the message.


LIMITATIONS
       seek_script works only for JPEG images.


FILES
       /tmp/fortune
              File containing the message to be used by OutGuess after the test.
