NAME
       ubuntu-kylin-wizard - setup wizard for Ubuntu/Ubuntu Kylin


SYNOPSIS
       ubuntu-kylin-wizard [options]


DESCRIPTION
       The  ubuntu-kylin-wizard  provides  a  friendly  guide  when user first logs in, and show them some basic
       information about the launcher, dash, indicators, control center, and so on.


OPTIONS
       --debug
              Runs ubuntu-kylin-wizard under gdb and prints a stack trace on crash.

       -h, --help
              Prints a usage message and exits.
