NAME
       ubuntu-iso - tool to examine Ubuntu CD (ISO) installation media


SYNOPSIS
       ubuntu-iso <path to ISO>


DESCRIPTION
       When  given  a path to an ISO, ubuntu-iso will determine whether it is an Ubuntu ISO or not. If it is, it
       will extract and display the version information.
