NAME
       IMGMERGE - script to concatenate several images into a PDF file.


DESCRIPTION
       This manual page documents briefly the imgmerge command.

       imgmerge  is  batch  script  'imgmerge'  which  uses xpaint to convert and concatenate image files into a
       single PDF file. This also makes use of Peter Szabo's 'pdfconcat'.


SEE ALSO
       pdfconcat(1), xpaint(1).
