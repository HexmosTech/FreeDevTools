NAME
       gt-condenseq - Call one of the CONDENSER tools to prepare or manipulate redundancy compressed genomic
       data.


SYNOPSIS
       gt condenseq tool [option ...]


DESCRIPTION
       -help
           display help and exit

       -version
           display version information and exit

       Tools:

       •   compress

       •   extract

       •   info

       •   search
