NAME
       download_atm_correction_luts - download the atm correction LUT files


DESCRIPTION
       usage: download_atm_correction_luts [-h]

       [--aerosol_types [AEROSOL_TYPES [AEROSOL_TYPES ...]]]
              [-d] [-v]

       Download the atm correction LUT files

   optional arguments:
       -h, --help
              show this help message and exit

       --aerosol_types [AEROSOL_TYPES [AEROSOL_TYPES ...]], -a [AEROSOL_TYPES [AEROSOL_TYPES ...]]
              Aerosol types

       -d, --dry_run
              Dry run - no action

       -v, --verbose
              Turn logging on
