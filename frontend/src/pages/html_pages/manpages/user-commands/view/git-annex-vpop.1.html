NAME
       git-annex-vpop - switch back to previous view


SYNOPSIS
       git annex vpop [N]


DESCRIPTION
       Switches  from  the  currently  active  view  back to the previous view.  Or, from the first view back to
       original branch.

       The optional number tells how many views to pop.


OPTIONS
       The git-annex-common-options(1) can be used.
