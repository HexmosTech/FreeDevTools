NAME
       migrate-repository - Migration of pre 0.4.5 SQLAlchemy migrate repositories


SYNOPSIS
       migrate-repository repository_directory


DESCRIPTION
       SQLAlchemy migrate uses a new repository format since version 0.4.5.

       All documentation for migrate is available in HTML format under /usr/share/doc/python-migrate/html.


SEE ALSO
       migrate's homepage at http://code.google.com/p/sqlalchemy-migrate/
