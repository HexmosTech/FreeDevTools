NAME
       vcf2birthday - Convert VCARD files to birthday database


DESCRIPTION
       The  vcf2birthday  command  reads a VCARD file from the standard input, and outputs a birthday-compatible
       database.


EXAMPLE
       cat ~/.gnome/GnomeCard.gcrd | vcf2birthday.pl | birthday -f -
