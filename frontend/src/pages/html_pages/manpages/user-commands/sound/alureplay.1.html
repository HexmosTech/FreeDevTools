NAME
       alureplay - example program for playing sounds with alure


SYNOPSIS

       alureplay soundfile


DESCRIPTION
       This manual page documents briefly the alureplay command.

       This manual page was written for the Debian distribution because the original program does not have a
       manual page. Instead, it has documentation in the html format; see below.

       alureplay plays a given sound file.


FILES
       /usr/share/doc/libalure-dev/examples/alureplay.c
           The source code for of this example program.


SEE ALSO
       alurecdplay(1), alurestream(1)
