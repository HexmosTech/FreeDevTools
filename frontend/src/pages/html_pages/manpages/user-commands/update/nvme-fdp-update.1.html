NAME
       nvme-fdp-update - Reclaim Unit Handle Update


SYNOPSIS
       nvme fdp update <device> [--namespace-id=<NUM> | -n <NUM>]
                               [--pids=<NUM,> | -p <NUM,>]


DESCRIPTION
       For the NVMe device given, update the given Placement Identifiers to reference a different Reclaim Unit
       accessible by the specified namespace.


OPTIONS
       -n <NUM>, --namespace-id=<NUM>
           Namespace identifier.

       -p <NUM,>, --pids=<NUM,>
           Comma-separated list of placement identifiers to update.
