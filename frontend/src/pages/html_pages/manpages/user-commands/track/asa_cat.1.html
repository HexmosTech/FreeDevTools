NAME
       asa_cat - asa_cat


SYNOPSIS
       asa_cat <NumFiles2Cat> <asa_file1> <asa_file2> <...>  <out_file> [catAbort]


DESCRIPTION
       This program is part of GMTSAR.

       *** asa_cat v1.3 by Dochul Yang and Vikas Gudipati***

       Concatenates Envisat ASAR Image Mode / Wide Swath Mode Level 0 data.

              NumFiles2Cat  Number  of  files to be concatenated (positive integer, Max: 100) asa_file1    First
              file to be merged  (along  track)  asa_file2     Second  file  to  be  merged  (along  track)  ...
              Rest  of  files  to  be  merged  (along  track) out_file     output data file [catAbort]   Flag to
              discontinue concatenation if non-overlaping  frames  are  found.  (0:  Discontinue,  1:  Continue,
              Default: 0)

       Notes:
