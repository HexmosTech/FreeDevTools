NAME
       disco-suggest - suggest DiscoDOS tracks based on various criteria


SYNOPSIS
       disco suggest [--help] [--bpm] [--key] [search_terms]


DESCRIPTION
       the  suggest  command  suggests tracks based on what you've played before or views tracks based on BPM or
       musical KEY.


ARGUMENTS
       [search_terms]
               track or release name you want to show a "track-combination report" for.  [search_terms] must  be
               left out when -b or -k is used.


OPTIONS
       -h, --help
               show this help message and exit

       -b BPM, --bpm BPM
               suggests tracks based on BPM value, within a pitch-range of +/-6 percent.

       -k KEY, --key KEY
               suggests tracks based on musical key.
