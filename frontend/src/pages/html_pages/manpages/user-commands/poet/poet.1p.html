NAME
       poet - main Poet command-line interface


SYNOPSIS
          poet new [-d dir] [-q] <AppName>

          poet script <script_name>


DESCRIPTION
       For general help, run "poet help".

       For help with a specific command, run "poet help _subcommand_".
