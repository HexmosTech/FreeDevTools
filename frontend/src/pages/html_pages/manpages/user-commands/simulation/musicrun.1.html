NAME
       musicrun - launch a MUSIC co-simulation


SYNOPSIS
       music_tests number_of_tasks configuration_file


DESCRIPTION
       Launch a MUSIC co-simulation.

       This is a convenience utility which provides a similar interface over multiple MPI implementations.

       music(1).
