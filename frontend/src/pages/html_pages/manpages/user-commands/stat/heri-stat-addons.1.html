NAME
       heri-stat-addons - calculates mean, maximum and standard deviations


SYNOPSIS
       heri-stat-addons [OPTIONS] [file...]


DESCRIPTION
       heri-stat-addons takes files produced by heri-stat -R on input and calculates mean, maximum and standard
       deviations.


OPTIONS
       -h, --help
             Display help information.

       -R, --raw
             Raw tab-separated output.
