NAME
       marc2dc - convert a MARC record to Dublin Core


SYNOPSIS
           % marc2dc [-q] marc.dat


DESCRIPTION
       This script takes a MARC record as input and converts it to an ad-hoc text format created from the
       conversion of the record into Dublin Core.
