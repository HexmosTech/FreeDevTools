NAME
       sigsum-token-record - format rate-limit key as a TXT record


SYNOPSIS
       sigsum-token record [--help] [-k key-file] [-o output-file]


DESCRIPTION
       Format the rate-limit key as a TXT record in zone-file format.


OPTIONS
       --help Display help

       -k, --key=key-file

              Rate-limit public-key in OpenSSH format (required)

       -o, --output=output-file

              Store output in a file


RETURN CODES
       A non-zero return code is used to indicate failure.
