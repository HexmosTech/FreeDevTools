NAME
        mtx-epub - ConTeXt EPUB Helpers


SYNOPSIS
       mtxrun --script epub [ OPTIONS ...  ] [ FILENAMES ]


DESCRIPTION
       ConTeXt EPUB Helpers


OPTIONS
       --make create epub zip file

       --purge
              remove obsolete files

       --rename
              rename images to sane names

       --svgmath
              convert mathml to svg

       --svgstyle
              use given tex style for svg generation (overloads style in specification)

       --all  assume: --purge --rename --svgmath (for fast testing)
