NAME
       dom-safe-pull - Safely pull changes from Git remote


DESCRIPTION
       dom-safe-pull looks for tracked branches and fetches changes from remote. For each branch, if the changes
       are fast-forward then it will merge the local branch with the corresponding remote branch.
