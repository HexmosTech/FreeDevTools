NAME
       pm-prepare - prepare directories to hold a local NCBI PubMed archive


SYNOPSIS
       pm-prepare master working


DESCRIPTION
       pm-prepare  prepares master and working directories to hold a local NCBI PubMed archive.  It also advises
       the user on steps to ensure that access to this tree is quick and that the system knows to treat it as  a
       cache.


SEE ALSO
       archive-pubmed(1), index-pubmed(1).
