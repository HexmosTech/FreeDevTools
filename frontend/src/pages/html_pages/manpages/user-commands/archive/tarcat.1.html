NAME
       tarcat - concatenates the pieces of a GNU tar multi-volume archive


SYNOPSIS
       tarcat files ...


DESCRIPTION
       The  tarcat  command  simply concatenates the files from a GNU tar multi-volume archive into a single tar
       archive.


SEE ALSO
       tar(1).
