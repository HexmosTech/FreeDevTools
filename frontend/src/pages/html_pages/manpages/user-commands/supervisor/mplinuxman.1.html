NAME
       mplinuxman — an mp3 player manager


SYNOPSIS
       mplinuxman


DESCRIPTION
       This manual page documents briefly the mplinuxman command.

       This  manual  page  was  written for the Debian distribution because the original program does not have a
       manual page.

       mplinuxman MPlinuxman is an mp3 player manager for the MPman F50 & F60


OPTIONS
       This command doesn't understand any commandline parameters.
