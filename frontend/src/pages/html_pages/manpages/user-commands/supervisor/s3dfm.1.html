NAME
       s3dfm - s3d file manager


SYNOPSIS

       s3dfm [-h|--help] [--s3d-url url]


DESCRIPTION
       s3dfm(1) is a 3D file manager.


OPTIONS
       This program follows the usual GNU command line syntax. A summary of the options supported by s3dfm is
       below.

       -h, --help
           Show summary of options and exit.

       --s3d-url url
           skip S3D environment and connect to this url
