NAME
       printcbm - list a Commodore BASIC file (DEPRECATED)


NOTICE
       As  of  xa 2.4, printcbm is deprecated and will be removed in a future version. Please consider migrating
       your usage to VICE petcat(1), which has many more options.


SYNOPSIS
       printcbm FILE


DESCRIPTION
       printcbm lists all lines of the specified Commodore BASIC program.


OPTIONS
       --help Show summary of options.

       --version
              Show version of program.
