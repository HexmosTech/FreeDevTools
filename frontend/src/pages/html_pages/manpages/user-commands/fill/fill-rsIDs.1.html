NAME
       fill-rsIDs - fill missing rsIDs


SYNOPSIS
       fill-rsIDs [OPTIONS]


DESCRIPTION
              zcat file.vcf.gz | fill-rsIDs -r dbSNP_ids_129.txt.bgz | bgzip -c > out.vcf.gz


OPTIONS
       -r <file.bgz>
              file containing ids
