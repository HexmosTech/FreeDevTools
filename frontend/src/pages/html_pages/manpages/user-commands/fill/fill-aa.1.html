NAME
       fill-aa - fill in ancestral alleles


SYNOPSIS
       fill-aa [OPTIONS]


DESCRIPTION
              zcat file.vcf.gz | fill-aa -a ancestral-alleles.fa.gz | bgzip -c > out.vcf.gz


OPTIONS
       -a <file.gz>
              file containing acestral alleles
