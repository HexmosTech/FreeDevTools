       zoe-loop - SNAP module zoe-loop


SYNOPSIS
       zoe-loop <test> [iterations - default 1000]


OPTIONS

       -all

       -Vec, -IVec, -FVec, -TVec, -Hash -Xtree

       -Feature, -FeatureVec

       -FastaFile, -DNA, -Protein, -Alignment -CDS -Palign

       -FeatureTable <file>

       -Model, -Scanner, -Counter

       -HMM <file>
