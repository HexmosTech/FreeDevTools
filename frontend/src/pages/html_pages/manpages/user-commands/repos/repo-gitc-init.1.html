NAME
       repo - repo gitc-init - manual page for repo gitc-init


SYNOPSIS
       repo gitc-init [options] [client name]


DESCRIPTION
       Summary

       Initialize a GITC Client.

       Run `repo help gitc-init` to view the detailed manual.


DETAILS
       The 'repo gitc-init' command is ran to initialize a new GITC client for use with the GITC file system.

       This command will setup the client directory, initialize  repo,  just  like  repo  init  does,  and  then
       downloads the manifest collection and installs it in the .repo/directory of the GITC client.

       Once  this  is  done, a GITC manifest is generated by pulling the HEAD SHA for each project and generates
       the properly formatted XML file and installs it as .manifest in the GITC client directory.

       The -c argument is required to specify the GITC client name.

       The optional -f argument can be used to specify the manifest file to use for this GITC client.
