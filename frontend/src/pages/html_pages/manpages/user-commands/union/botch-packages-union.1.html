NAME
       botch-packages-union - calculate the set union


SYNOPSIS
       botch-packages-union [options] inPackages.. outPackages


DESCRIPTION
       calculate the union of two packages files, effectively merging the two.


OPTIONS
       -h, --help
           Show help.

       --verbose
           Be verbose.
