NAME
       xbindkeys-config - GTK+ configuration tool for xbindkeys


SYNOPSIS
       xbindkeys-config [--file|-f file] [--help|-h] [--show|-s]

       -h, --help
              This Help

       -f, --file
              Use an alternative rc file

       -s, --show
              show only the rc file
