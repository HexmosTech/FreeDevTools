NAME
       calceph_queryposition  -  computes  the  position  and  velocity  of the target from the center using the
       ephemeris files for a Julian day TDB


SYNOPSIS
       calceph_queryposition [juliandate] [target] [center] [file ...]


DESCRIPTION
       calceph_queryposition displays to the standard output the position and velocity of the target  body  from
       the center body for a Julian day TDB using the ephemeris files.

       The  NAIF  identification numbering system is used for the target body and the center body.  The position
       is expressed in kilometers and the velocity is expressed in kilometers per second.  These quantities  are
       expressed in the ICRF frame.


EXAMPLES
       calceph_queryposition 2450457.0 5 10 examples/example1.bsp
              Display the position and velocity of the Jupiter barycenter relative to the Sun at the julian date
              2450457.0 TDB using the ephemeris file examples/example1.bsp
