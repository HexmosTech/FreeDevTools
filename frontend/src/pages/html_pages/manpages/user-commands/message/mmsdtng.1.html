NAME
       mmsd-tng - Send and receive MMS messages


SYNOPSIS
       mmsdtng [ -d ] [ -v ]


DESCRIPTION
       mmsdtng This is a daemon that sends and receives MMSes. It works with modem manager.


OPTIONS
       d      Enable debug mode

       v      Show version information and exit
