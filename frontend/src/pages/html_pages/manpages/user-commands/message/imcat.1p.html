NAME
       imcat - display mail/news message


SYNOPSIS
       imcat [OPTIONS] [FOLDER] MSG


DESCRIPTION
       The imcat command shows the contents of the mail/news message stored in a folder.

       This command is provided by IM (Internet Message).


OPTIONS
       -s, --src=FOLDER
            Folder name.  Default value is "+inbox".  "--src=+xxx" is equivalent to "+xxx".

       -j, --join={on,off}
            Join partial messages.

       -b, --boundary={on,off}
            Print start boundary for Mew version 1.x.

       -S, --sshserver=SERVER
            SSH port relay server.

       -v, --verbose={on,off}
            Print verbose messages when running.

       --debug=DEBUG_OPTION
            Print debug messages when running.

       -h, --help
            Display help message and exit.

       --version
            Output version information and exit.
