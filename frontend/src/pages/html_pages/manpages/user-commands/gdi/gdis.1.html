NAME
       gdis - molecules and periodic systems manipulator


SYNOPSIS
       gdis


DESCRIPTION
       GDIS  is a GTK based program for the display and manipulation of isolated molecules and periodic systems.
       It is in development, but is nonetheless fairly functional.
