NAME
       ezmsim - simulate enzyme cut assembly sequences


SYNOPSIS
       ezmsim <LR|EF> [options]


DESCRIPTION
       Program:   ezmsim  (simulate  enzyme  cut  assembly  sequences)  Version:  0.1.1  Contact:  Zechen  Chong
       <chongzechen@gmail.com>

       Options: LR          simulate LR reads

       RAD    simulate RAD reads
