NAME
       xfce4-screensaver - The Xfce Desktop Screensaver and Locker


SYNOPSIS
       xfce4-screensaver [OPTIONS]


DESCRIPTION
       Xfce  Screensaver  is  the  default  screen  saver  and  locker for the Xfce Desktop. It is designed with
       simplicity and security in mind. xfce4-screensaver integrates well with the Xfce desktop, and it supports
       User Switching and the ability to lock down configuration settings.


OPTIONS
       --display=DISPLAY
              X display to use.

       --debug
              Enable debugging messages for xfce4-screensaver

       --version
              Output version information and exit.

       -h, --help
              Print standard command line options.

       --help-all
              Print all command line options.

       This program also accepts the standard GTK options.
