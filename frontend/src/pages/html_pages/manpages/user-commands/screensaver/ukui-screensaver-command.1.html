NAME
       ukui-screensaver - The UKUI Desktop Screensaver and Locker


SYNOPSIS
       ukui-screensaver [OPTIONS]


DESCRIPTION
       UKUI  Screensaver  is  the  default  screen  saver  and  locker for the UKUI Desktop. It is designed with
       simplicity and security in mind. ukui-screensaver integrates well with the UKUI desktop, and it  supports
       User Switching and the ability to lock down configuration settings.


OPTIONS
       --display=DISPLAY
              X display to use.

       --no-daemon
              Don't start as a daemon

        --debug
              Enable debugging messages for ukui-screensaver

       --version
              Output version information and exit.

       -h, --help
              Print standard command line options.

       --help-all
              Print all command line options.

       This program also accepts the standard GTK options.
