NAME
       mate-screensaver - The MATE Desktop Screensaver and Locker


SYNOPSIS
       mate-screensaver [OPTIONS]


DESCRIPTION
       MATE  Screensaver  is  the  default  screen  saver  and  locker for the MATE Desktop. It is designed with
       simplicity and security in mind. mate-screensaver integrates well with the MATE desktop, and it  supports
       User Switching and the ability to lock down configuration settings.


OPTIONS
       --display=DISPLAY
              X display to use.

       --no-daemon
              Don't start as a daemon

       --debug
              Enable debugging messages for mate-screensaver

       --version
              Output version information and exit.

       -h, --help
              Print standard command line options.

       --help-all
              Print all command line options.

       This program also accepts the standard GTK options.
