NAME
       hmac - Runs TPM2_Hmac


DESCRIPTION
       --version-string=v1045 is not a valid option

       hmac

       Runs TPM2_HMAC

       -hk key handle

       -pwdk password for key (default empty)

              [-halg  (sha1, sha256, sha384) (default sha256)] -if input file to be HMACed -ic data string to be
              HMACed [-os hmac file name (default do not save)]

       -se[0-2] session handle / attributes (default PWAP)

              01 continue

       --version is not a valid option
