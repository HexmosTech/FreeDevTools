NAME
       pmap_clear_modify — set information about physical pages


SYNOPSIS
       #include <sys/param.h>
       #include <vm/vm.h>
       #include <vm/pmap.h>

       void
       pmap_clear_modify(vm_page_t m);


DESCRIPTION
       The pmap_clear_modify() function clears the “modified” bit on the physical page m.


SEE ALSO
       pmap(9), pmap_is_modified(9)
