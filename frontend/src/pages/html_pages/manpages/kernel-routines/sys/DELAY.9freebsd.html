NAME
       DELAY — busy loop for an interval


SYNOPSIS
       #include <sys/types.h>
       #include <sys/systm.h>

       void
       DELAY(int delay);


DESCRIPTION
       Delay for delay microseconds (1/1000000th of a second).


SEE ALSO
       pause(9)
