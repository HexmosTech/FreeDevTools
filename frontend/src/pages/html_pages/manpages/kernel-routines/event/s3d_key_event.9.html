NAME
       s3d_key_event - key event information


SYNOPSIS
       #include <s3d.h>


STRUCTURE MEMBERS
           struct s3d_key_event {
                uint16_t keysym;
                uint16_t unicode;
                uint16_t modifier;
                uint16_t state;
           }


DESCRIPTION
       Can be used on the buffer of an event of type S3D_EVENT_KEY*.

       Simon Wunderlich
