NAME
       Sympa::Request::Handler::invite - invite request handler


DESCRIPTION
       Invites someone to subscribe a list by sending him template 'invite'.

       Subscription request of target user is stored into held request spool.


SEE ALSO
       Sympa::Request::Handler, Sympa::Spool::Auth.
