NAME
       MPIO_Request_c2f -  Translates a C I/O-request handle to a  Fortran I/O-request handle


SYNOPSIS
       #include "mpi.h"
       MPI_Fint MPIO_Request_c2f(MPIO_Request request)


INPUT PARAMETERS
       request
              - C I/O-request handle (handle)


RETURN VALUE
       Fortran I/O-request handle (integer)
