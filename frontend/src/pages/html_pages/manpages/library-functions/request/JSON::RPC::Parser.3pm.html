NAME
       JSON::RPC::Parser - Parse JSON RPC Requests from Plack::Request


SYNOPSIS
           use JSON::RPC::Parser;

           my $parser = JSON::RPC::Parser->new(
               coder => JSON->new
           );
           my $procedure = $parser->construct_from_req( $request );


DESCRIPTION
       Constructs a JSON::RPC::Procedure object from a Plack::Request object
