NAME
       gsasl_mechanism_name_p - API function


SYNOPSIS
       #include <gsasl.h>

       int gsasl_mechanism_name_p(const char * mech);


ARGUMENTS
       const char * mech
                   input variable with mechanism name string.


DESCRIPTION
       Check  if  the mechanism name string  mech follows syntactical rules.  It does not check that the name is
       registered with IANA.  It does not check that the mechanism name is actually implemented and supported.

       SASL mechanisms are named by strings, from 1  to  20  characters  in  length,  consisting  of  upper-case
       letters, digits, hyphens, and/or underscores.


RETURNS
       non-zero when mechanism name string  mech conforms to rules, zero when it does not meet the requirements.


SINCE
       2.0.0
