NAME
       SASL::NTLM - Implementation of SASL NTLM mechanism for Tcl


SYNOPSIS
       package require Tcl 8.5 9

       package require SASL::NTLM ?1.1.4?

________________________________________________________________________________________________________________


DESCRIPTION
       This  package provides the NTLM authentication mechanism for the Simple Authentication and Security Layer
       (SASL).

       Please read the documentation for package sasl for details.


REFERENCES
       [1]    No  official  specification  is  available.  However,   http://davenport.sourceforge.net/ntlm.html
              provides a good description.
