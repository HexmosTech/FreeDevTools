NAME
       MARC::Charset::Constants - constants for MARC::Charset


SYNOPSIS
           use MARC::Charset::Constants qw(:all);


DESCRIPTION
       MARC::Charset needs to recognize various codes which are given descriptive names and assigned to
       constants in this package.
