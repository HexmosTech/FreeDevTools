NAME
       Net::Proxy::Connector::dummy - Dummy Net::Proxy connector


DESCRIPTION
       "Net::Proxy::Connecter::dummy" is a "Net::Proxy::Connector" that does nothing. It doesn't listen for
       incoming connections and does connect to other hosts.

       Future connectors may have their accept_from() method also handle the connection to a remote host. In
       this case, "dummy" may be used as an 'out' connector.

       You could also use the source code of this module as a template for writing new "Net::Proxy::Connector"
       classes.


CONNECTOR OPTIONS
       None.
