Name
       SWISS::OSs


Description
       SWISS::OSs represents the OS lines within an SWISS-PROT + TrEMBL entry as specified in the user manual
        http://www.expasy.org/sprot/userman.html . The OSs object is a container object which holds a list of
       SWISS::OS objects.

       n.b. entries from distinct species are not merged anymore, OSs will therefore only contain one OS (OS is
       still divided into a list of OS elements to keep compatibility with old code)!


Inherits from
       SWISS::ListBase.pm


Attributes
       "list"
             Each list element is a SWISS::OS object.
