NAME
       Memoize::SDBM_File - DEPRECATED compability shim


DESCRIPTION
       This class used to provide EXISTS support for SDBM_File before support for "EXISTS" was added to
       SDBM_File itself in Perl 5.6.0.

       Any code still using this class should be rewritten to use SBDM_File directly.
