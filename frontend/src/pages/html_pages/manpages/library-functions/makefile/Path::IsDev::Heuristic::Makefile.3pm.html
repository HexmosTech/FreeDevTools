NAME
       Path::IsDev::Heuristic::Makefile - Determine if a path contains a Makefile


VERSION
       version 1.001003


METHODS
   "files"
       Files relevant to this heuristic:

           GNUmakefile
           makefile
           Makefile

   "matches"
       Matches if any of the files in "files" exist as children of the "path"
