NAME
       Forest::Tree::Roles::JSONable - An abstract role for providing JSON serialization


DESCRIPTION
       This is just an abstract role for trees capable of JSON serialization.


REQUIRED METHODS
       as_json (?%options)
           Return  a  JSON string of the invocant. Takes %options parameter to specify the way the tree is to be
           dumped.
