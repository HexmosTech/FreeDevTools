NAME
       Date::Manip::TZ_Base - Methods common to the TZ and Base classes


SYNOPSIS
       The Date::Manip::TZ_Base class is a second base class used for the following Date::Manip classes:

       Date::Manip::Base
       Date::Manip::TZ

       This module is not intended to be called directly and performs no useful function by itself. Instead, use
       the various derived classes which inherit from it.


DESCRIPTION
       This  module  contains  no  public  methods  and  is  used internally by the Base and TZ classes. In some
       instances, basic date operations depend on timezone information, and all timezone  operations  depend  on
       the  base class.  In order to avoid a circular dependency, I created this class which allowed me to avoid
       that.
