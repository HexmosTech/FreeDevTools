Name
       App::Sqitch::Command::checkout - Revert, change checkout a VCS branch, and redeploy


Synopsis
         my $cmd = App::Sqitch::Command::checkout->new(%params);
         $cmd->execute;


Description
       If you want to know how to use the "checkout" command, you probably want to be reading "sqitch-checkout".
       But if you really want to know how the "checkout" command works, read on.


Interface
   Class Methods
       "options"

         my @opts = App::Sqitch::Command::checkout->options;

       Returns a list of Getopt::Long option specifications for the command-line options for the "checkout"
       command.

   Instance Methods
       "execute"

         $checkout->execute;

       Executes the checkout command.


See Also
       sqitch-checkout
           Documentation for the "checkout" command to the Sqitch command-line client.

       sqitch
           The Sqitch command-line client.
