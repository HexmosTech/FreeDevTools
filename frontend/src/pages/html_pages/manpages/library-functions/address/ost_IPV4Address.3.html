Detailed Description
       The network name and address objects are all derived from a common IPV4Address base class.

       Specific classes, such as IPV4Host, IPV4Mask, etc, are defined from IPV4Address entirely so that the
       manner a network address is being used can easily be documented and understood from the code and to avoid
       common errors and accidental misuse of the wrong address object. For example, a 'connection' to something
       that is declared as a 'IPV4Host' can be kept type-safe from a 'connection' accidently being made to
       something that was declared a 'IPV4Broadcast'.
