NAME
       Web::ID::SAN::Email - represents subjectAltNames that are e-mail addresses


DESCRIPTION
       This module uses WWW::Finger (if installed) to attempt to locate some RDF data about the holder of the
       given e-mail address. It is probably not especially interoperable with other WebID implementations.
