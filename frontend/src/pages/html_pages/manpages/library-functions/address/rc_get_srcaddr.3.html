NAME
       rc_get_srcaddr -


SYNOPSIS
       #include <radcli/radcli.h>

       int rc_get_srcaddr(
           struct sockaddr        *lia,
           const struct sockaddr  *ria
       );


DESCRIPTION
       Find outbound interface address for a given destination

       Given  remote  address  find  local  address  which  the  system will use as a source address for sending
       datagrams to that remote address.


PARAMETERS
       lia    local address.

       ria    the remove address.


RETURN VALUE
       0 in success, -1 on failure, address is filled into the first argument.
