NAME
       Net::Hotline - Perl libraries for the Hotline internet client


SYNOPSIS
           use Net::Hotline;


DESCRIPTION
       The Net::Hotline module simply loads Net::Hotline::Client.  For more information on the
       Net::Hotline::Client module, please see its documentation.  For more information on Hotline, see
       http://www.hotlinesw.com/
