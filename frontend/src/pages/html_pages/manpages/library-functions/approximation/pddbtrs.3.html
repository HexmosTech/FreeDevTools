NAME
       PDDBTRS - solve a system of linear equations   A(1:N, JA:JA+N-1) * X = B(IB:IB+N-1, 1:NRHS)


SYNOPSIS
       SUBROUTINE PDDBTRS( TRANS, N, BWL, BWU, NRHS, A, JA, DESCA, B, IB, DESCB, AF, LAF, WORK, LWORK, INFO )

           CHARACTER       TRANS

           INTEGER         BWL, BWU, IB, INFO, JA, LAF, LWORK, N, NRHS

           INTEGER         DESCA( * ), DESCB( * )

           DOUBLE          PRECISION A( * ), AF( * ), B( * ), WORK( * )


PURPOSE
       PDDBTRS solves a system of linear equations
                                         or
                 A(1:N, JA:JA+N-1)' * X = B(IB:IB+N-1, 1:NRHS)

       where  A(1:N,  JA:JA+N-1)  is the matrix used to produce the factors stored in A(1:N,JA:JA+N-1) and AF by
       PDDBTRF.
       A(1:N, JA:JA+N-1) is an N-by-N real
       banded diagonally dominant-like distributed
       matrix with bandwidth BWL, BWU.

       Routine PDDBTRF MUST be called first.
