NAME
       SDL_AUDIO_DEVICE_DEFAULT_PLAYBACK - A value used to request a default playback audio device.


HEADER FILE
       Defined in SDL3/SDL_audio.h


SYNOPSIS
       #include "SDL3/SDL.h"

       #define SDL_AUDIO_DEVICE_DEFAULT_PLAYBACK ((SDL_AudioDeviceID) 0xFFFFFFFFu)


DESCRIPTION
       Several functions that require an SDL_AudioDeviceID

       will accept this value to signify the app just wants the system to choose a default device instead of the
       app providing a specific one.


AVAILABILITY
       This macro is available since SDL 3.2.0.
