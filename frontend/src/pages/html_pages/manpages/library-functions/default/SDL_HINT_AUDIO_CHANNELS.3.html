NAME
       SDL_HINT_AUDIO_CHANNELS - A variable controlling the default audio channel count.


SYNOPSIS
       #include <SDL3/SDL_hints.h>

       #define SDL_HINT_AUDIO_CHANNELS "SDL_AUDIO_CHANNELS"


DESCRIPTION
       If the application doesn't specify the audio channel count when opening the device, this hint can be used
       to  specify  a  default  channel  count that will be used. This defaults to "1" for recording and "2" for
       playback devices.

       This hint should be set before an audio device is opened.


AVAILABILITY
       This hint is available since SDL 3.2.0.
