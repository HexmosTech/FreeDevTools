NAME
       SDL_SSE3_INTRINSICS - Defined if (and only if) the compiler supports Intel SSE3 intrinsics.


HEADER FILE
       Defined in SDL3/SDL_intrin.h


SYNOPSIS
       #include "SDL3/SDL.h"

       #define SDL_SSE3_INTRINSICS 1


DESCRIPTION
       If this macro is defined, SDL will have already included <pmmintrin.h>


AVAILABILITY
       This macro is available since SDL 3.2.0.
