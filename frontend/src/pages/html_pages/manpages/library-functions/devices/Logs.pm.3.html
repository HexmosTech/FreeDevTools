NAME
       Octopussy::Logs - Octopussy Logs module

   Device_List($devices)
       Returns Device List

   Service_List($services)
       Returns Service List

   Get_Directories($dir)
       Gets Logs Directories

   Init_Directories($device)
       Inits Logs Directories for Device '$device'

   Remove_Directories($device)
       Removes Logs Directories for Device '$device'

   Files_Year_Month_Day($start, $finish, $dir_service, $y)
   Files_Year_Month_Day_Hour_Min($dir_service, $start_num, $finish_num, $files)
   Files($devices, $services, $start, $finish)
       Get logs files from '$services' on devices '$devices' between '$start' & '$finish' (don't get Incoming
       logs files)

   Availability($device, $start, $finish)
       Returns Logs Availability for specified $device and period ($start-$finish)

   Minutes_Hash($ref_devices, $ref_services, $start, $finish)
   Get_TimePeriod_Files($devices, $services, $begin, $end)
       Returns list of Files for Devices $devices, Services $services and Period $begin-$end

   Get($devices, $services, $start, $finish, $re_incl, $re_excl, $limit)
       Get logs lines from '$services' on devices '$devices' between '$start' & '$finish'

   Incoming_Files($device)
       Returns list of 'Incoming Files'

   Unknown_Files($device, $first_file)
       Returns list of 'Unknown Files' (lines of these files doesn't match any defined message)

   Unknown_Number($device)
       Returns number of unknown log messages for device '$device'

   Remove($device, $log)
       Removes Log '$log' from Unknown Logs for Device '$device'

   Remove_Minute($device, $year, $month, $day, $hour, $min)
       Removes 1 minute of Log File for Device '$device'

   Extract_Cmd_Line($conf)
       Generate Command Line for octo_extractor
