Detailed Description

       RTE_FLOW_ACTION_TYPE_JUMP

       Redirects packets to a group on the current device.

       In a hierarchy of groups, which can be used to represent physical or logical flow tables on the device,
       this action allows the action to be a redirect to a group on that device.
