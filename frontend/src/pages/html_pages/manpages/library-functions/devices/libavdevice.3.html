NAME
       libavdevice - multimedia device handling library


DESCRIPTION
       The libavdevice library provides a generic framework for grabbing from and rendering to many common
       multimedia input/output devices, and supports several input and output devices, including Video4Linux2,
       VfW, DShow, and ALSA.
