NAME
       XML::LibXML::ErrNo - Structured Errors


DESCRIPTION
       This module is based on xmlerror.h libxml2 C header file. It defines symbolic constants for all libxml2
       error codes. Currently libxml2 uses over 480 different error codes. See also XML::LibXML::Error.
