Detailed Description
       The SemaphoreLock class is used to protect a section of code through a semaphore so that only x instances
       of the member function may execute concurrently.

       A common use is

       void func_to_protect() { SemaphoreLock lock(semaphore); ... operation ... }

       NOTE: do not declare variable as 'SemaohoreLock (semaphore)', the mutex will be released at statement
       end.
