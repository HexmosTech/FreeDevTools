NAME
       MCE::Mutex::Flock - Mutex locking via Fcntl


VERSION
       This document describes MCE::Mutex::Flock version 1.901


DESCRIPTION
       A Fcntl implementation for "MCE::Mutex".

       The API is described in MCE::Mutex.

       new
       lock
       lock_exclusive
       lock_shared
       guard_lock
       unlock
       synchronize
       enter
       timedwait
