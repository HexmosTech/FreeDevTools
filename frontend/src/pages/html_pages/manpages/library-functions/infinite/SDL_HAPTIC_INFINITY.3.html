NAME
       SDL_HAPTIC_INFINITY - Used to play a device an infinite number of times.


SYNOPSIS
       #include <SDL3/SDL_haptic.h>

       #define SDL_HAPTIC_INFINITY   4294967295U


AVAILABILITY
       This macro is available since SDL 3.2.0.
