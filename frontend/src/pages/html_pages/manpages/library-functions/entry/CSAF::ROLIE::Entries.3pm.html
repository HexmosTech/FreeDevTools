NAME
       CSAF::ROLIE::Entries - ROLIE entry collection


SYNOPSIS
           use CSAF::ROLIE::Entries;
           my $entries = CSAF::ROLIE::Entries->new( );


DESCRIPTION
       CSAF::ROLIE::Entries is a collection of CSAF::ROLIE::Entry elements.

   METHODS
       CSAF::ROLIE::Entries inherits all methods from CSAF::Util::List.
