NAME
       ck_ht_entry_key — return pointer to key as specified in hash table entry


LIBRARY
       Concurrency Kit (libck, -lck)


SYNOPSIS
       #include <ck_ht.h>

       void *
       ck_ht_entry_key(ck_ht_entry_t *entry);


DESCRIPTION
       The  ck_ht_entry_key()  function will return the key pointer as specified in the object pointed to by the
       entry argument.

       It is expected that the entry is associated with a hash table initialized with CK_HT_MODE_BYTESTRING (see
       ck_ht_init(3) for more information).


RETURN VALUES
       ck_ht_entry_key() returns NULL if the entry is empty.


ERRORS
       Behavior is undefined if entry has not been initialized.
