NAME
       Git::Raw::Tree::Entry - Git tree entry class


VERSION
       version 0.90


DESCRIPTION
       A Git::Raw::Tree::Entry represents an entry in a Git::Raw::Tree.

       WARNING: The API of this module is unstable and may change without warning (any change will be
       appropriately documented in the changelog).


METHODS
   id( )
       Get the id of the object pointed by the entry, as a string.

   name( )
       Retrieve the filename of the tree entry.

   file_mode( )
       Retrieve the file mode of the tree entry, as an integer. For example, a normal file has mode 0100644 =
       33188.

   type( )
       Retrieve the type of the object pointed by the tree entry. Returns a constant from the Git::Raw::Object
       namespace.

   object( )
       Retrieve the object pointed by the tree entry.
