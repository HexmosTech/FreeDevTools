NAME
       CSS::DOM::Value - CSSValue class for CSS::DOM


VERSION
       Version 0.17


SYNOPSIS
         # ...


DESCRIPTION
       This module implements objects that represent CSS property values. It implements the DOM CSSValue
       interface.

       This class is used only for custom values (neither primitive values nor lists) and the special 'inherit'
       value.


METHODS
   Object Methods
       cssText
           Returns a string representation of the attribute. Pass an argument to set it.

       cssValueType
           Returns one of the constants below.

   Constructor
       You probably don't need to call this, but here it is anyway:

         $val = new CSS::DOM::Value %arguments;

       The hash-style %arguments are as follows:

       type
           "CSS_INHERIT" or "CSS_CUSTOM"

       css A string of CSS code. This is only used when "TYPE" is "CSS_CUSTOM".

       owner
           The  style  object  that  owns this value; if this is omitted, then the value is read-only. The value
           object holds a weak reference to the owner.

       property
           The name of the CSS property to which this value belongs. "cssText" uses this  to  determine  how  to
           parse text passed to it.


CONSTANTS
       The  following  constants  can be imported with "use CSS::DOM::Value ':all'".  They represent the type of
       CSS value.

       CSS_INHERIT (0)
       CSS_PRIMITIVE_VALUE (1)
       CSS_VALUE_LIST (2)
       CSS_CUSTOM (3)
