NAME
       cgiGetValue - Return the value of a CGI variable


SYNOPSYS
       #include <cgi.h>

       char *cgiGetValue (s_cgi *parms, const char *name);


DESCRIPTION
       This  routine  returns  a  pointer  to the value of a cgi variable.  Encoded characters (%nn) are already
       decoded.  One must not free the pointer.

       If multiple fields are used (i.e. a variable that may contain several values) the value returned contains
       all these values concatenated together with a newline character as separator.

       Memory allocated by the returned value is automatically freed by the final call to cgiFree(3).


RETURN VALUE
       On success a pointer to a string is returned.  If the variable wasn't  transmitted  through  CGI  or  was
       empty NULL is returned.
