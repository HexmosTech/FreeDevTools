NAME
       Test2::Harness::Util::File::Value - Utility class for a file that contains exactly 1 value.


DESCRIPTION
       This is a subclass of Test2::Harness::Util::File for files expected to have exactly 1 value stored in
       them.


SYNOPSIS
           use Test2::Harness::Util::File::Value;

           my $vf = Test2::Harness::Util::File::Value->new(name => 'path/to/file');
           my $val = $vf->read;


METHODS
       $val = $vf->read()
           Read all contents from the file, chomp() it, and return it.

       $val = $vf->read_line()
           Read  the  first line from the file, chomp() it, and return it. Note, this may not return anything if
           the value in the file does not terminate with a newline.
