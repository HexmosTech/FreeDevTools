NAME
       ginq_char_up_vec - retrieves the current value of the character up vector


SYNOPSIS
       #include <ncarg/gks.h>

       void ginq_char_up_vec(Gint *err_ind, Gvec *char_up_vec);


DESCRIPTION
       err_ind     (Output) - If the inquired values cannot be returned correctly, a non-zero error indicator is
                   returned  in  err_ind,  otherwise  a zero is returned.  Consult "User's Guide for NCAR GKS-0A
                   Graphics" for a description of the meaning of the error indicators.

       char_up_vec.delta_x
                   (Gfloat, Output) -  Gives the X world coordinate of a the up vector.

       char_up_vec.delta_y
                   (Gfloat, Output) - Gives the Y world coordinate of a the up vector.


USAGE
       The   up   vector   is   that   vector   formed   by   the   two   coordinate    points    (0.,0.)    and
       (char_up_vec.delta_x,char_up_vec.delta_y).   See  gset_char_up_vec for more details on the meaning of the
       character up vector.


ACCESS
       To use the GKS C-binding routines, load the ncarg_gks and ncarg_c libraries.
