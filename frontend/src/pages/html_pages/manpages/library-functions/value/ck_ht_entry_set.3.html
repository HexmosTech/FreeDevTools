NAME
       ck_ht_entry_set — initialize a key-value pair


LIBRARY
       Concurrency Kit (libck, -lck)


SYNOPSIS
       #include <ck_ht.h>

       void
       ck_ht_entry_set(ck_ht_entry_t   *entry,   ck_ht_hash_t   h,   const   void   *key,  uint16_t  key_length,
           const void *value);


DESCRIPTION
       The ck_ht_entry_set() function will initialize the object pointed to by entry with a key  pointed  to  by
       the  key  argument  and  a  value pointed to by the value argument. The length of the key is specified by
       key_length. The maximum value of key_length is defined by the CK_HT_KEY_LENGTH macro.  This  function  is
       typically  used  to  initialize  an  entry  for ck_ht_set_spmc(3) and ck_ht_put_spmc(3) operations. It is
       expected that the entry will be associated with a hash table initialized with CK_HT_MODE_BYTESTRING  (see
       ck_ht_init(3) for more information).


RETURN VALUES
       ck_ht_entry_set() has no return value.


ERRORS
       This function will never fail.
