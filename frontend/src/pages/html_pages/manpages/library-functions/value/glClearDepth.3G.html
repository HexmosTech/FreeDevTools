NAME
       glClearDepth - specify the clear value for the depth buffer


C SPECIFICATION

       void glClearDepth(GLdouble depth);

       void glClearDepthf(GLfloat depth);


PARAMETERS
       depth
           Specifies the depth value used when the depth buffer is cleared. The initial value is 1.


DESCRIPTION
       glClearDepth specifies the depth value used by glClear() to clear the depth buffer. Values specified by
       glClearDepth are clamped to the range 0 1.


NOTES
       The type of the depth parameter was changed from GLclampf to GLfloat for glClearDepthf and from GLclampd
       to GLdouble for glClearDepth. This change is transparent to user code and is described in detail on the
       removedTypes() page.


ASSOCIATED GETS
       glGet() with argument GL_DEPTH_CLEAR_VALUE
