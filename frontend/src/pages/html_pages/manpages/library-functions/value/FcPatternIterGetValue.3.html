NAME
       FcPatternIterGetValue - Returns a value which the iterator point to


SYNOPSIS
       #include <fontconfig/fontconfig.h>

       FcResult   FcPatternIterGetValue   (const   FcPattern   *p,   FcPatternIter  *iter,  intid,  FcValue  *v,
       FcValueBinding *b);


DESCRIPTION
       Returns in v the id'th value which iter point to. also binding to b if given.  The value returned is  not
       a  copy,  but  rather  refers to the data stored within the pattern directly.  Applications must not free
       this value.
