NAME
       UR::BoolExpr::Template::PropertyComparison::Like - perform an SQL-ish like test


DESCRIPTION
       The input test value is assumed to be an SQL 'like' value, where '_' represents a one character wildcard,
       and '%' means a 0 or more character wildcard.  It gets converted to a perl regular expression and used to
       match against an object's properties.

       If the property returns multiple values, this comparison returns true if any of the values match.
