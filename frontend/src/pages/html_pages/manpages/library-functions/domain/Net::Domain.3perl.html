NAME
       Net::Domain - Attempt to evaluate the current host's internet name and domain


SYNOPSIS
           use Net::Domain qw(hostname hostfqdn hostdomain domainname);


DESCRIPTION
       Using various methods attempt to find the Fully Qualified Domain Name (FQDN) of the current host. From
       this determine the host-name and the host-domain.

       Each of the functions will return undef if the FQDN cannot be determined.

   Functions
       hostfqdn()
           Identify and return the FQDN of the current host.

       domainname()
           An alias for hostfqdn().

       hostname()
           Returns the smallest part of the FQDN which can be used to identify the host.

       hostdomain()
           Returns the remainder of the FQDN after the hostname has been removed.
