NAME
       libssh2_session_hostkey - get the remote key


SYNOPSIS
       #include <libssh2.h>

       const char *
       libssh2_session_hostkey(LIBSSH2_SESSION *session,
                               size_t *len, int *type);


DESCRIPTION
       Returns a pointer to the current host key, the value len points to will get the length of the key.

       The   value   type   points   to   the  type  of  hostkey  which  is  one  of:  LIBSSH2_HOSTKEY_TYPE_RSA,
       LIBSSH2_HOSTKEY_TYPE_DSS (deprecated), or LIBSSH2_HOSTKEY_TYPE_UNKNOWN.


RETURN VALUE
       A pointer, or NULL if something went wrong.
