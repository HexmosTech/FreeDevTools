NAME
       qb_map_put - Inserts a new key and value into a qb_map_t.


SYNOPSIS
       #include <qb/qbmap.h>

       void qb_map_put(
           qb_map_t     *map,
           const char   *key,
           const void   *value
       );


DESCRIPTION
       If the key already exists in the qb_map_t, it gets replaced by the new key.
