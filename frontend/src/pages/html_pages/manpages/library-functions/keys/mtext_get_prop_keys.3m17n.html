NAME
       mtext_get_prop_keys -


SYNOPSIS
       int mtext_get_prop_keys (MText * mt, int pos, MSymbol ** keys)


DESCRIPTION
       Get a list of text property keys at a position of an M-text. The mtext_get_prop_keys() function creates
       an array whose elements are the keys of text properties found at position pos in M-text mt, and sets
       *keys to the address of the created array. The user is responsible to free the memory allocated for the
       array.


RETURN VALUE
           If the operation was successful, mtext_get_prop_keys() returns the length of the key list. Otherwise
           it returns -1 and assigns an error code to the external variable merror_code.


ERRORS
           MERROR_RANGE
