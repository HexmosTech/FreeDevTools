NAME
       Web::ID::RSAKey - an RSA key


DESCRIPTION
   Constructor
       "new"
           Standard Moose-style constructor.

   Attributes
       "exponent"
           The exponent as a Math::BigInt object.

       "modulus"
           The modulus as a Math::BigInt object.

   Methods
       "rsa_equal($that)"
           Returns true iff this key is the same as that key.
