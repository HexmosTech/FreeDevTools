NAME
       Mail::ListDetector::Detector::RFC2919 - RFC2919 message detector


SYNOPSIS
         use Mail::ListDetector::Detector::RFC2919;


DESCRIPTION
       An implementation of a mailing list detector, for RFC2919 compliant mailing lists, i.e., those with List-
       ID lines in the header.


METHODS
   new()
       Inherited from Mail::ListDetector::Detector::Base.

   match()
       Accepts a Mail::Internet object and returns either a Mail::ListDetector::List object if it is a post to a
       RFC2919 compliant mailing list, or "undef".

       Since RFC2919 only specifies a (mostly) unique ID for a mailing list,
       Mail::ListDetector::Detector::RFC2369 is used to attempt to extract further information about the list.


BUGS
