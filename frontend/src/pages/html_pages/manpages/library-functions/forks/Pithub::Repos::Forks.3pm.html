NAME
       Pithub::Repos::Forks - Github v3 Repo Forks API


VERSION
       version 0.01043


METHODS
   create
       •   Create a fork for the authenticated user.

               POST /repos/:user/:repo/forks

           Examples:

               my $f = Pithub::Repos::Forks->new;
               my $result = $f->create(
                   user => 'plu',
                   repo => 'Pithub',
               );

               # or fork to an org
               my $result = $f->create(
                   user => 'plu',
                   repo => 'Pithub',
                   org  => 'CPAN-API',
               );

   list
       •   List forks

               GET /repos/:user/:repo/forks

           Examples:

               my $f = Pithub::Repos::Forks->new;
               my $result = $f->list(
                   user => 'plu',
                   repo => 'Pithub',
               );
