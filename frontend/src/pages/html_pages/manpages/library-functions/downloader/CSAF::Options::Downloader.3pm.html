NAME
       CSAF::Options::Downloader - CSAF::Downloader configurator


SYNOPSIS
           use CSAF::Options::Downloader;
           my $options = CSAF::Options::Downloader->new( );

           $options->configure(
               url             => 'https://security.acme.tld/advisories/csaf/index.txt',
               insecure        => 1,
               directory       => './csaf-acme-advisories',
               include_pattern => qr{acme-sa-2024-\d+\.json}
           );


DESCRIPTION
       CSAF::Options::Downloader is a configurator of CSAF::Downloader.

   METHODS
       CSAF::Options::Downloader inherits all methods from CSAF::Util::Options.

   ATTRIBUTES
       directory
       exclude_pattern
       include_pattern
       insecure
       integrity_check
       parallel_downloads
       signature_check
       url
       validate
       after_date
       before_date
