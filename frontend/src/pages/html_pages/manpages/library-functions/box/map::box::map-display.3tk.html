NAME
       map::box::map-display - Map Action Engine - Layer to display box definitions


SYNOPSIS

       package require Tcl 8.6-

       package require Tk 8.6-
       package require canvas::edit::rectangle

       package require debug

       package require debug::caller

       package require map::slippy 0.8

       package require snit

       package require map::box::map-display ?0.1?

________________________________________________________________________________________________________________


DESCRIPTION
       This  package  provides  a map::display attachment extending the map's behaviour. This attachment shows a
       set of geo/box definitions. It ensures that only the visible boxes use canvas resources (items).
