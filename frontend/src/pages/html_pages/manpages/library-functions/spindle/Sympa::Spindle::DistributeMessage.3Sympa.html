NAME
       Sympa::Spindle::DistributeMessage - Workflow to distribute messages to list members


DESCRIPTION
       Sympa::Spindle::DistributeMessage distributes incoming messages to list members.

       This class represents the series of following processes:

       Sympa::Spindle::TransformIncoming
           Process to transform messages - first stage

       Sympa::Spindle::ToArchive
           Process to store messages into archiving spool

       Sympa::Spindle::TransformOutgoing
           Process to transform messages - second stage

       Sympa::Spindle::ToDigest
           Process to store messages into digest spool

       Sympa::Spindle::ToList
           Process to distribute messages to list members

   Public methods
       See also "Public methods" in Sympa::Spindle.

       new ( key => value, ... )
           In  most cases, Sympa::Spindle::DoMessage splices messages to this class.  This method is not used in
           ordinal case.

       spin ( )
           Not implemented.
