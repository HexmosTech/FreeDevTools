NAME
       glIsQuery - determine if a name corresponds to a query object


C SPECIFICATION

       GLboolean glIsQuery(GLuint id);


PARAMETERS
       id
           Specifies a value that may be the name of a query object.


DESCRIPTION
       glIsQuery returns GL_TRUE if id is currently the name of a query object. If id is zero, or is a non-zero
       value that is not currently the name of a query object, or if an error occurs, glIsQuery returns
       GL_FALSE.

       A name returned by glGenQueries(), but not yet associated with a query object by calling glBeginQuery(),
       is not the name of a query object.
