NAME
       probe::stap.cache_get - Found item in stap cache


SYNOPSIS
       stap.cache_get


VALUES
       source_path
           the path of the .c source file

       module_path
           the path of the .ko kernel module file


DESCRIPTION
       Fires just before the return of get_from_cache, when the cache grab is successful.
