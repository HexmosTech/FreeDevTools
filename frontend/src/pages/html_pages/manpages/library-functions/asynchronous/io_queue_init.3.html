NAME
       io_queue_init - Initialize asynchronous io state machine


SYNOPSIS
       #include <errno.h>

       #include <libaio.h>

       int io_queue_init(int maxevents, io_context_t *ctx);


DESCRIPTION
       io_queue_init()  attempts  to  create an aio context capable of receiving at least maxevents events.  ctx
       must point to an aio context that already exists and must be initialized to 0 before the  call.   If  the
       operation is successful, *cxtp is filled with the resulting handle.


RETURN VALUES
       On  success,  io_queue_init()  returns  0.   Otherwise,  -error is return, where error is one of the Exxx
       values defined in the ERRORS section.


ERRORS
       EFAULT iocbs referenced data outside of the program's accessible address space.

       EINVAL maxevents is <= 0 or ctx is an invalid memory location.

       ENOSYS Not implemented.

       EAGAIN maxevents > max_aio_reqs where max_aio_reqs is a tunable value.
