NAME
       XML::DOM::Comment - An XML comment in XML::DOM


DESCRIPTION
       XML::DOM::Comment extends XML::DOM::CharacterData which extends XML::DOM::Node.

       This node represents the content of a comment, i.e., all the characters between the starting '<!--' and
       ending '-->'. Note that this is the definition of a comment in XML, and, in practice, HTML, although some
       HTML tools may implement the full SGML comment structure.
