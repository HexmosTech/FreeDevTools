NAME
       Mail::SpamAssassin::Bayes - support for learning classifiers


DESCRIPTION
       This is the general class used to train a learning classifier with new samples of spam and ham mail, and
       classify based on prior training.

       Prior to version 3.3.0, the default Bayes implementation was here; if you're looking for information on
       that, it has moved to "Mail::SpamAssassin::Plugin::Bayes".
