NAME
       Sisimai::Rhost::Spectrum - Detect the bounce reason returned from Spectrum.


SYNOPSIS
           use Sisimai::Rhost::Spectrum;


DESCRIPTION
       "Sisimai::Rhost::Spectrum" detects the bounce reason from the content of "Sisimai::Fact" object as an
       argument of find() method when the value of "rhost" or "destination" of the object is "charter.net". This
       class is called only "Sisimai::Fact" class.


CLASS METHODS
   "find(Sisimai::Fact Object)"
       find() method detects the bounce reason.
