NAME
       Mail::ListDetector::Detector::Lyris - Lyris message detector


SYNOPSIS
         use Mail::ListDetector::Detector::Lyris;


DESCRIPTION
       An implementation of a mailing list detector, for Lyris mailing lists, Lyris is a ???? by MCF Software,
       see http://www.liststar.com/ for details about Lyris.


METHODS
   new()
       Inherited from Mail::ListDetector::Detector::Base.

   match()
       Accepts a Mail::Internet object and returns either a Mail::ListDetector::List object if it is a post to a
       Lyris mailing list, or "undef".
