NAME
       Bio::TreeIO::TreeEventBuilder - Build Bio::Tree::Tree's and
         Bio::Tree::Node's from Events


SYNOPSIS
       # internal use only


DESCRIPTION
       This object will take events and build a Bio::Tree::TreeI compliant object makde up of Bio::Tree::NodeI
       objects.
