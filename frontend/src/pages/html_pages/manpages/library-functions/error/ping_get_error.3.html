NAME
       ping_get_error - Return the last error message


SYNOPSIS
         #include <oping.h>

         const char *ping_get_error (pingobj_t *obj);


DESCRIPTION
       The ping_get_error method returns an error message indicating the last error encountered. This method is
       not thread safe whatsoever.


RETURN VALUE
       A "const" string representing the last error or an empty string if no error was encountered yet.
