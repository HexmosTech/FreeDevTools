NAME
       nerr_ignore - free the clearsilver error chain


SYNOPSIS
       #include <util/neo_err.h>

       void nerr_ignore (NEOERR **err);


DESCRIPTION
       you  should  only  call  this  if  you actually handle the error (should I rename it?).  Free's the error
       chain.
