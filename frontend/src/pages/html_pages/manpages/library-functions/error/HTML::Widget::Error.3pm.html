NAME
       HTML::Widget::Error - Error


SYNOPSIS
           my @errors = $form->errors('foo');
           for my $error (@errors) {
               print $error->type;
               print $error->message;
           }


DESCRIPTION
       Error.


METHODS
   name
       Arguments: $name

       Return Value: $name

   message
       Arguments: $message

       Return Value: $message

   no_render
       Arguments: $bool

       If true, this error will not be displayed by "$result->as_xml".

       This is used internally used by "render_errors" in HTML::Widget::Constraint.

   type
       Arguments: $type

       Return Value: $type
