DESCRIPTION
       The libbfio_get_version() function is used to retrieve the library version.


RETURN VALUES
       Most  of  the  functions return NULL or -1 on error, dependent on the return type.  For the actual return
       values see "libbfio.h".

NOTES
       libbfio allows to be compiled with wide character support (wchar_t).

       To compile libbfio with wide character support use: ./configure --enable-wide-character-type=yes
        or define: _UNICODE
        or UNICODE
        during compilation.

       LIBBFIO_WIDE_CHARACTER_TYPE
        in libbfio/features.h can be used to determine if libbfio was compiled with wide character support.
