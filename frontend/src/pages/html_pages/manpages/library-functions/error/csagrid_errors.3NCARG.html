NAME
       csagrid_errors - This document briefly describes all the errors reported from Csagrid.


DESCRIPTION
       Each entry below includes the error number and its meaning.

        33    Internal error in matrix computation, contact a consultant.

        34    Failure in matrix computation, usually indicates insufficient number of input data.

        35    Internal error in matrix computation, contact a consultant.

       102    Number of knots is less than 4.

       103    Input data range is zero.

       104    Partial derivative order out of range zero to two.

       105    Insufficient number of input data.

       106    Insufficient work space (Fortran entry only).

       107    Failure in matrix computation, usually indicates insufficient number of input data.

       202    Number of knots is less than 4 in some coordinate direction.

       203    Insufficient workspace.
