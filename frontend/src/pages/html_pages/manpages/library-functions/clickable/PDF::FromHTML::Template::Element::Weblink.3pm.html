NAME
       PDF::FromHTML::Template::Element::WebLink - To provide a clickable web-link


NODE NAME
       WEBLINK


INHERITANCE
       PDF::FromHTML::Template::Element


ATTRIBUTES
       •   URL The URL to go to, when clicked

       •   X1 / X2 / Y1 / Y2

           The dimensions of the clickable area


CHILDREN
       None


AFFECTS
       Nothing


DEPENDENCIES
       None


USAGE
       This  node  is  currently  under  review as to whether it should be removed and a URL attribute should be
       added to various nodes, such as IMAGE, TEXTBOX, and ROW.
