NAME
       Net::DNS::RR::CAA - DNS CAA resource record


SYNOPSIS
               use Net::DNS;
               $rr = Net::DNS::RR->new('name IN CAA flags tag value');


DESCRIPTION
       Class for Certification Authority Authorization (CAA) DNS resource records.


METHODS
       The available methods are those inherited from the base class augmented by the type-specific methods
       defined in this package.

       Use of undocumented package features or direct access to internal data structures is discouraged and
       could result in program termination or other unpredictable behaviour.

   flags
               $flags = $rr->flags;
               $rr->flags( $flags );

       Unsigned 8-bit number representing Boolean flags.

       critical
                   $rr->critical(1);

                   if ( $rr->critical ) {
                           ...
                   }

           Issuer critical flag.

   tag
               $tag = $rr->tag;
               $rr->tag( $tag );

       Property  identifier  which may contain the characters a-z, A-Z, and 0-9.  The tag field must not contain
       any other characters.  Matching of tags is not case sensitive.

   value
               $value = $rr->value;
               $rr->value( $value );

       A sequence of octets representing the property value.  Property values are encoded as binary  values  and
       may employ sub-formats.
