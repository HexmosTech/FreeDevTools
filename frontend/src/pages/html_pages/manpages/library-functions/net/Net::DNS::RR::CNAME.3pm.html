NAME
       Net::DNS::RR::CNAME - DNS CNAME resource record


SYNOPSIS
               use Net::DNS;
               $rr = Net::DNS::RR->new('name CNAME cname');

               $rr = Net::DNS::RR->new(
                               name  => 'alias.example.com',
                               type  => 'CNAME',
                               cname => 'example.com',
                               );


DESCRIPTION
       Class for DNS Canonical Name (CNAME) resource records.


METHODS
       The available methods are those inherited from the base class augmented by the type-specific methods
       defined in this package.

       Use of undocumented package features or direct access to internal data structures is discouraged and
       could result in program termination or other unpredictable behaviour.

   cname
               $cname = $rr->cname;
               $rr->cname( $cname );

       A domain name which specifies the canonical or primary name for the owner.  The owner name is an alias.
