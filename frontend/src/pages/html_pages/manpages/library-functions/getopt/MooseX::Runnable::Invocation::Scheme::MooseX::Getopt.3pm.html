NAME
       MooseX::Runnable::Invocation::Scheme::MooseX::Getopt - run MX::Getopt classes


DESCRIPTION
       This role will be used by "MooseX::Runnable::Invocation" to create an instance of the class to be run
       with "MooseX::Getopt".  Any args not consumed by MX::Getopt will be passed to the class's run method.

       (See the test "t/basic-mx-getopt.t" for an example.)
