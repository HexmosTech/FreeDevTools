Detailed Description
       RTE_FLOW_ITEM_TYPE_RAW

       Matches a byte string of a given length at a given offset.

       Offset is either absolute (using the start of the packet) or relative to the end of the previous matched
       item in the stack, in which case negative values are allowed.

       If search is enabled, offset is used as the starting point. The search area can be delimited by setting
       limit to a nonzero value, which is the maximum number of bytes after offset where the pattern may start.

       Matching a zero-length pattern is allowed, doing so resets the relative offset for subsequent items.

       This type does not support ranges (struct rte_flow_item.last).
