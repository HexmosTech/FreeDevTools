NAME
       FcPatternDuplicate - Copy a pattern


SYNOPSIS
       #include <fontconfig/fontconfig.h>

       FcPattern * FcPatternDuplicate (const FcPattern *p);


DESCRIPTION
       Copy  a  pattern,  returning a new pattern that matches p. Each pattern may be modified without affecting
       the other.
