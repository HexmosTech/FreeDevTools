NAME
       c_nnpnts - interpolate at a single point


FUNCTION PROTOTYPE
       void c_nnpnts(float, float, float *);


SYNOPSIS
       void c_nnpntinits (x, y, z);


DESCRIPTION
       x           The X coordinate of the point where interpolation is desired.

       y           The Y coordinate of the point where interpolation is desired.

       z           *z is the interpolated functional value at the point (X,Y).


USAGE
       This function is invoked when you want to interpolate at an individal point.  c_nnpntinits must be called
       prior to calling c_nnpnts.  c_nnpntend must be called to terminate single point mode.


ACCESS
       To use c_nnpnts, load the NCAR Graphics library ngmath.
