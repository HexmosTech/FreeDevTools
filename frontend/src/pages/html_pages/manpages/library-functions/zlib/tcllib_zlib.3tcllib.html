NAME
       tcl::transform::zlib - zlib (de)compression


SYNOPSIS
       package require Tcl 8.6 9

       package require tcl::transform::core ?1.1?

       package require tcl::transform::zlib ?1.1?

       ::tcl::transform::zlib chan ?level?

________________________________________________________________________________________________________________


DESCRIPTION
       The  tcl::transform::zlib  package  provides  a  command  creating  a  channel  transformation which zlib
       compresses the written data, and decompresses on reading.

       The internal TclOO class implementing the transform handler is a sub-class  of  the  tcl::transform::core
       framework.


API
       ::tcl::transform::zlib chan ?level?
              This  command  creates a zlib compressor transformation on top of the channel chan and returns its
              handle.

              The level specifies how much effort is put into the compression, from 0 to 9, and defaults to 4.
