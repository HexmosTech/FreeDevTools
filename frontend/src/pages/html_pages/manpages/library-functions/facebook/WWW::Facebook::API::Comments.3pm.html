NAME
       WWW::Facebook::API::Comments - Facebook Comments


SYNOPSIS
           use WWW::Facebook::API;


DESCRIPTION
       Methods for accessing comments with WWW::Facebook::API


SUBROUTINES/METHODS
       base
           Returns the WWW::Facebook::API base object.

       new Constructor.

       add( %params )
           The comments.add method of the Facebook API:

               $response = $client->comments->add(
                   xid  => $xid,
                   text => $text,
                   uid  => $uid,
               );

       get( %params )
           The comments.get method of the Facebook API:

               $response = $client->comments->get(
                   xid => $xid,
               );

       remove( %params )
           The comments.remove method of the Facebook API:

               $response = $client->comments->remove( xid => $xid, comment_id => $id );
