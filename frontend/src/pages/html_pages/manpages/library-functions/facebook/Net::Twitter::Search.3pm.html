NAME
       Net::Twitter::Search - A perl interface to the Twitter Search API


VERSION
       version 4.01043


SYNOPSIS
         use Net::Twitter;

         $nt = Net::Twitter::Search->new;

         $tweets = $nt->search('perl_api');


DESCRIPTION
       This module simply creates an instance of "Net::Twitter" with the "Legacy" trait for backwards
       compatibility with prior versions.  Consider Net::Twitter::Lite if you need a lighter, non-Moose
       alternative.

       See Net::Twitter for full documentation.


DEPRECATION NOTICE
       This module is deprecated.  Use Net::Twitter instead.

           use Net::Twitter;

           # Just the Search API; exceptions thrown on error
           $nt = Net::Twitter->new(traits => [qw/API::Search/]);

           # Just the Search API; errors wrapped - use $nt->get_error
           $nt = Net::Twitter->new(traits => [qw/API::Search WrapError/]);

           # Or, for code that uses legacy Net::Twitter idioms
           $nt = Net::Twitter->new(traits => [qw/Legacy/]);

           $tweets = $nt->search('pot of gold');


METHODS
       new Creates  a  "Net::Twitter"  object  with  the  "Legacy"  trait.   See "new" in Net::Twitter for "new"
           options.
