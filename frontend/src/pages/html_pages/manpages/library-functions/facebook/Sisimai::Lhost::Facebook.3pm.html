NAME
       Sisimai::Lhost::Facebook - bounce mail parser class for "Facebook".


SYNOPSIS
           use Sisimai::Lhost::Facebook;


DESCRIPTION
       Sisimai::Lhost::Facebook parses a bounce email which created by "Facebook".  Methods in the module are
       called from only Sisimai::Message.


CLASS METHODS
   "description()"
       "description()" returns description string of this module.

           print Sisimai::Lhost::Facebook->description;

   "make(header data, reference to body string)"
       "make()" method parses a bounced email and return results as a array reference.  See Sisimai::Message for
       more details.
