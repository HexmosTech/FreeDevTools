NAME
       Net::Twitter::Role::API::REST - A definition of the Twitter REST API as a Moose role


VERSION
       version 4.01043


SYNOPSIS
         package My::Twitter;
         use Moose;
         with 'Net::Twitter::API::REST';


DESCRIPTION
       Net::Twitter::Role::API::REST provides definitions for all the Twitter REST API methods.   Applying  this
       role to any class provides methods for all of the Twitter REST API methods.
