NAME
       Font::TTF::Glat - Hold glyph attributes


DESCRIPTION
       Holds glyph attributes associated with each glyph.

       Version
           Table format version

       attribs
           An  array  of  hashes.  On array entry for each glyph id. Since the glyph attributes are usually in a
           sparse array, they are stored in a hash keyed by the attribute id and with  the  value  as  attribute
           value.

   $t->minsize()
       Returns  the  minimum  size this table can be. If it is smaller than this, then the table must be bad and
       should be deleted or whatever.
