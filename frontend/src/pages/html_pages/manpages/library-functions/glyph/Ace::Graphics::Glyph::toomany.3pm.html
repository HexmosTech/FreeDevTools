NAME
       Ace::Graphics::Glyph::toomany - The "too many to show" glyph


SYNOPSIS
         See L<Ace::Graphics::Panel> and L<Ace::Graphics::Glyph>.


DESCRIPTION
       This glyph is intended for features that are too dense to show properly.  Mostly a placeholder, it
       currently shows a filled oval.  If you choose a bump of 0, the ovals will overlap, to give a cloud
       effect.

   OPTIONS
       There are no glyph-specific options.
