NAME
       Image::ExifTool::MPEG - Read MPEG-1 and MPEG-2 meta information


SYNOPSIS
       This module is used by Image::ExifTool


DESCRIPTION
       This module contains definitions required by Image::ExifTool to read MPEG-1 and MPEG-2 audio/video files.


NOTES
       Since ISO charges money for the official MPEG specification, this module is based on unofficial sources
       which may be incomplete, inaccurate or outdated.
