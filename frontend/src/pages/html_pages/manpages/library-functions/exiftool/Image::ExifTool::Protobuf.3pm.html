NAME
       Image::ExifTool::Protobuf - Decode protocol buffer information


SYNOPSIS
       This module is loaded automatically by Image::ExifTool when required.


DESCRIPTION
       This module contains definitions required by Image::ExifTool to decode information in protocol buffer
       (protobuf) format.
