NAME
       Image::ExifTool::GM - Read GM PDR Data from automobile videos


SYNOPSIS
       This module is loaded automatically by Image::ExifTool when required.


DESCRIPTION
       This module contains definitions required by Image::ExifTool to read PDR metadata from videos written by
       some GM models such as Corvette and Camero.
