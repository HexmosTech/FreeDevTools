NAME
       Image::ExifTool::MISB - Read Motion Industry Standards Board metadata


SYNOPSIS
       This module is used by Image::ExifTool


DESCRIPTION
       This module contains code to extract STANAG-4609 Motion Industry Standards Board (MISB) KLV-format
       metadata from M2TS videos.
