NAME
       Image::ExifTool::MXF - Read MXF meta information


SYNOPSIS
       This module is used by Image::ExifTool


DESCRIPTION
       This module contains definitions required by Image::ExifTool to read meta information from MXF (Material
       Exchange Format) files.
