NAME
       Image::ExifTool::ICC_Profile - Read ICC Profile meta information


SYNOPSIS
       This module is loaded automatically by Image::ExifTool when required.


DESCRIPTION
       This module contains the definitions to read information from ICC profiles.  ICC (International Color
       Consortium) profiles are used to translate color data created on one device into another device's native
       color space.
