NAME
       Image::ExifTool::QuickTime - Extract embedded information from media data


SYNOPSIS
       These routines are autoloaded by Image::ExifTool::QuickTime.


DESCRIPTION
       This file contains routines used by Image::ExifTool to extract embedded information like GPS tracks from
       MOV, MP4 and INSV media data.
