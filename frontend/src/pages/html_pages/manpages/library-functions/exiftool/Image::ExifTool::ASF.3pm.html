NAME
       Image::ExifTool::ASF - Read ASF/WMA/WMV meta information


SYNOPSIS
       This module is used by Image::ExifTool


DESCRIPTION
       This module contains routines required by Image::ExifTool to extract information from Microsoft Advanced
       Systems Format (ASF) files, including Windows Media Audio (WMA) and Windows Media Video (WMV) files.
