NAME
       Image::ExifTool::M2TS - Read M2TS (AVCHD) meta information


SYNOPSIS
       This module is used by Image::ExifTool


DESCRIPTION
       This module contains routines required by Image::ExifTool to extract information from MPEG-2 transport
       streams, such as those used by AVCHD video.
