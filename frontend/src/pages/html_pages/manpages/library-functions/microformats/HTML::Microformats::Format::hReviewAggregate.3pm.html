NAME
       HTML::Microformats::Format::hReviewAggregate - the hReview-aggregate microformat


SYNOPSIS
        use HTML::Microformats::DocumentContext;
        use HTML::Microformats::Format::hReviewAggregate;

        my $context = HTML::Microformats::DocumentContext->new($dom, $uri);
        my @reviews = HTML::Microformats::Format::hReviewAggregate->extract_all(
                          $dom->documentElement, $context);
        foreach my $review (@reviews)
        {
          print Dumper($review->data) . "\n";
        }


DESCRIPTION
       HTML::Microformats::Format::hReviewAggregate inherits from HTML::Microformats::Format. See the base class
       definition for a description of property getter/setter methods, constructors, etc.


MICROFORMAT
       HTML::Microformats::Format::hReviewAggregate supports hReview-aggregate 0.2 as described at
       <http://microformats.org/wiki/hreview-aggregate> with the following differences:

       •   hAudio

           hAudio microformats can be used as the reviewed item.

       •   hReview properties

           A  few properties are supported from (non-aggregate) hReview - e.g.  'bookmark', 'tag', 'description'
           and 'type'.
