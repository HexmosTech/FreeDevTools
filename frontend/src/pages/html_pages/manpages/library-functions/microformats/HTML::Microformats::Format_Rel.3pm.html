NAME
       HTML::Microformats::Format_Rel - base rel-* microformat class


SYNOPSIS
        my @tags = HTML::Microformats::RelTag->extract_all(
                          $doc->documentElement, $context);
        foreach my $tag (@tags)
        {
          print $tag->get_href . "\n";
        }


DESCRIPTION
       HTML::Microformats::Format_Rel inherits from HTML::Microformats::Format. See the base class definition
       for a description of property getter/setter methods, constructors, etc.

   Additional Methods
       "$relfoo->get_href()"
           Returns the absolute URL of the resource being linked to.

       "$relfoo->get_label()"
           Returns the linked text of the <a> element. Microformats patterns like value excerpting are used.

       "$relfoo->get_title()"
           Returns the contents of the title attribute of the <a> element, or the same as "$relfoo->get_label()"
           if the attribute is not set.
