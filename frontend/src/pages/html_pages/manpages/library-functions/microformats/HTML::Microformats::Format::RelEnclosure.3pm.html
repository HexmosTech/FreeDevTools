NAME
       HTML::Microformats::Format::RelEnclosure - the rel-enclosure microformat


SYNOPSIS
        my @enclosures = HTML::Microformats::Format::RelEnclosure->extract_all(
                          $doc->documentElement, $context);
        foreach my $e (@enclosures)
        {
          my $type = $l->get_type || 'unknown';
          printf("%s (%s)\n"), $l->get_href, $type);
        }


DESCRIPTION
       HTML::Microformats::Format::RelEnclosure inherits from HTML::Microformats::Format_Rel. See the base class
       definition for a description of property getter/setter methods, constructors, etc.

   Additional Method
       "$relenc->get_type()"
           Returns  the  media  type (Content-Type) of the resource being linked to. This is taken from the HTML
           'type' attribute, so if that's not present, returns undef.


MICROFORMAT
       HTML::Microformats::Format::RelEnclosure      supports      rel-enclosure      as      described       at
       <http://microformats.org/wiki/rel-enclosure>.

       The "title" attribute on the link, and the linked text are taken to be significant.


RDF OUTPUT
       Data  is  returned  using  the  RSS  Enclosures  vocabulary (<http://purl.oclc.org/net/rss_2.0/enc#>) and
       occasional other terms.
