NAME
       Type::Coercion::Union - a set of coercions to a union type constraint


STATUS
       This module is covered by the Type-Tiny stability policy.


DESCRIPTION
       This package inherits from Type::Coercion; see that for most documentation.  The major differences are
       that "add_type_coercions" always throws an exception, and the "type_coercion_map" is automatically
       populated from the child constraints of the union type constraint.
