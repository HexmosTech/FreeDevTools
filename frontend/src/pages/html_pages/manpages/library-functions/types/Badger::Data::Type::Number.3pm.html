NAME
       Badger::Data::Type::Number - base class data type for numbers


DESCRIPTION
       This module implements a base class for numerical data types.


METHODS
       This module inherits all methodsfrom the Badger::Data::Type::Simple, Badger::Data::Type and Badger::Base
       base classes.
