NAME
       GSMK (Set marker type) - sets the type of polymarker to be used in subsequent GPM calls.


SYNOPSIS
       CALL GSMK (MTYPE)


C-BINDING SYNOPSIS
       #include <ncarg/gks.h>

       void gset_marker_type(Gint marker_type);


DESCRIPTION
       MTYPE       (Integer, Input) - Selects the type of marker to be drawn.  Options are:

                   < 0    implementation dependent (not used in NCAR GKS-0A)

                     1
                           . (dot)

                     2
                           + (plus)

                     3
                           * (asterisk) This is the default

                     4
                           o (circle)

                     5
                           X (cross)

                   â‰¥ 6
                           reserved for registration or future standardization


ACCESS
       To use GKS routines, load the NCAR GKS-0A library ncarg_gks.
