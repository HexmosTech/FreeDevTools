NAME
       CSS::Parse::Heavy - A CSS::Parse module using Parse::RecDescent


SYNOPSIS
         use CSS;

         # Create a css stylesheet
         my $CSS = CSS->new({'parser' => 'CSS::Parse::Heavy'});


DESCRIPTION
       This module is a parser for CSS.pm. Read the CSS.pm pod for more details
