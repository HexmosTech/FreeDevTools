SYNOPSIS
          bool
          mongoc_uri_has_option (const mongoc_uri_t *uri, const char *option);


PARAMETERS
       • uri: A mongoc_uri_t.

       • option: The name of an option, case insensitive.


DESCRIPTION
       Returns true if the option was present in the initial MongoDB URI.
