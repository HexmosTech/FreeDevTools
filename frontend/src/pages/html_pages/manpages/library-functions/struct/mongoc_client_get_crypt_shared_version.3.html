SYNOPSIS
          const char *
          mongoc_client_get_crypt_shared_version (const mongoc_client_t *client);

       Obtain  the version string of the crypt_shared that is loaded for auto-encryption on the given client. If
       no crypt_shared library is loaded, the returned pointer will be NULL.


PARAMETERS
       • client: A live mongoc_client_t


RETURNS
       A pointer to a null-terminated character array string designating the version of  crypt_shared  that  was
       loaded  for  auto-encryption with client. If no crypt_shared library is loaded, or auto-encryption is not
       loaded for the given client, the returned pointer will be NULL. The pointed-to array must not be modified
       or freed. The returned pointer is only valid for the lifetime of client.
