DESCRIPTION
       Fetches a linked list of hosts that were defined in the URI (the comma-separated host section).


RETURNS
       A linked list of mongoc_host_list_t structures that should not be modified or freed. Returns NULL if this
       URI's scheme is "mongodb+srv://".
