NAME
       auparse_get_node - get the event's machine node name


SYNOPSIS
       #include <auparse.h>

       const char *auparse_get_node(const auparse_state_t *au);


DESCRIPTION
       auparse_get_node  gets  the  machine's node name if it exists in the audit event from the current event's
       timestamp data structure. Not all records have node names since its an admin configurable option.


RETURN VALUE
       Returns a copy of the node name or NULL if it does not exist or there was an error. The caller must  free
       the string.
