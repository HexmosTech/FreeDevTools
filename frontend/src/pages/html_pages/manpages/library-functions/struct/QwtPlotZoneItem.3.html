Detailed Description
       A plot item, which displays a zone.

       A horizontal zone highlights an interval of the y axis - a vertical zone an interval of the x axis - and
       is unbounded in the opposite direction. It is filled with a brush and its border lines are optionally
       displayed with a pen.

       Note:
           For displaying an area that is bounded for x and y coordinates use QwtPlotShapeItem
