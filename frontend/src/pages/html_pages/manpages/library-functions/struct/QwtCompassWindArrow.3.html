Detailed Description
       An indicator for the wind direction.

       QwtCompassWindArrow shows the direction where the wind comes from.

       • QPalette::Light
          Used for Style1, or the light half of Style2

       • QPalette::Dark
          Used for the dark half of Style2

       See also:
           QwtDial, QwtCompass
