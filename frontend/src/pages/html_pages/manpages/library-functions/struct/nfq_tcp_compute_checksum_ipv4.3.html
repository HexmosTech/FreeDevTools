       nfq_tcp_compute_checksum_ipv4 - computes IPv4/TCP packet checksum

       Parameters
           tcph pointer to the TCP header
           iph pointer to the IPv4 header

       Note
           nfq_tcp_mangle_ipv4() invokes this function. As long as developers always use nfq_tcp_mangle_ipv4
