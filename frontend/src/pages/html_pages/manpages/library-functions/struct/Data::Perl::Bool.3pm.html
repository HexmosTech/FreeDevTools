NAME
       Data::Perl::Bool - Wrapping class for boolean values.


VERSION
       version 0.002011


SYNOPSIS
         use Data::Perl qw/bool/;

         my $bool = bool(0);

         $bool->toggle; # 1

         $bool->unset; # 0


DESCRIPTION
       This class is a simple consumer of the Data::Perl::Role::Bool role, which provides all functionality. You
       probably want to look there instead.
