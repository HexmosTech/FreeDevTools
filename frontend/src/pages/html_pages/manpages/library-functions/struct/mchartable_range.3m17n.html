NAME
       mchartable_range -


SYNOPSIS
       void mchartable_range (MCharTable * table, int * from, int * to)


DESCRIPTION
       Search for characters that have non-default value. The mchartable_range() function searches chartable
       table for the first and the last character codes that do not have the default value of table, and set
       from and to to them, respectively. If all characters have the default value, both from and to are set to
       -1.
