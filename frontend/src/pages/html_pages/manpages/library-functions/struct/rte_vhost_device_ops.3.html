   void(* guest_notified) (int vid)
       This callback gets called each time a guest gets notified about waiting packets. This is the interrupt
       handling through the eventfd_write(callfd), which can be used for counting these 'slow' syscalls.

       Definition at line 300 of file rte_vhost.h.

   bool(* guest_notify) (int vid, uint16_t queue_id)
       If this callback is registered, notification to the guest can be handled by the front-end calling
       rte_vhost_notify_guest(). If it's not handled, 'false' should be returned. This can be used to remove the
       'slow' eventfd_write() syscall from the datapath.
