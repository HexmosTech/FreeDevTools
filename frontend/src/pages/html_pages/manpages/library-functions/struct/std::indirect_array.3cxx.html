Detailed Description
   template<class _Tp>
       class std::indirect_array< _Tp >"Reference to arbitrary subset of an array.

       An indirect_array is a reference to the actual elements of an array specified by an ordered array of
       indices. The way to get an indirect_array is to call operator[](valarray<size_t>) on a valarray. The
       returned indirect_array then permits carrying operations out on the referenced subset of elements in the
       original valarray.

       For example, if an indirect_array is obtained using the array (4,2,0) as an argument, and then assigned
       to an array containing (1,2,3), then the underlying array will have array[0]==3, array[2]==2, and
       array[4]==1.
