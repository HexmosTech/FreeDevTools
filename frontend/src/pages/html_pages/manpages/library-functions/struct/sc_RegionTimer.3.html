Detailed Description
       The RegionTimer class is used to record the time spent in a section of code.

       During the run of a code, enter and exit members are called to begin and end timed sections. The print
       member is used to display the obtained times. Multiple enter calls for a region with the same name
       aggregate the timings. Nested regions are supported.
