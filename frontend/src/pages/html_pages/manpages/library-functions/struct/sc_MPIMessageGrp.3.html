Constructor & Destructor Documentation
   sc::MPIMessageGrp::MPIMessageGrp (MPI_Comm comm)
       Use an MPI communicator to create a MessageGrp. The comm argument could be a subset of MPI_COMM_WORLD,
       for example.

   sc::MPIMessageGrp::MPIMessageGrp (int * argc, char *** argv)
       Use argc and argv to create a MPIMessageGrp. This would have to be used for implementations of MPI that
       have MPI_Init fill in argc and argv.
