NAME
       ginq_norm_tran - Inquire normalization transformation


DESCRIPTION
       ginq_norm_tran (Inquire normalization transformation) - retrieves the window and viewport associated with
       a  given  normalization  transformation.   Users of NCAR Graphics are encouraged to use the SPPS routines
       c_set and c_getset to handle normalization transformations instead of the GKS entries gset_win,  gset_vp,
       gsel_norm_tran, and ginq_norm_tran.


SYNOPSIS
       #include <ncarg/gks.h>

       void ginq_norm_tran(Gint num, Gint *err_ind, Gtran *norm_tran);


DESCRIPTION
       num         (Input) - Gives the number of the normalization transformation.

       err_ind     (Output)  - If the inquired value cannot be returned correctly, a non-zero error indicator is
                   returned in err_ind, otherwise a zero is returned.  Consult "User's  Guide  for  NCAR  GKS-0A
                   Graphics" for a description of the meaning of the error indicators.

       norm_tran.win
                   (Glimit, Output) - The world coordinate window limits.

       norm_tran.vp
                   (Glimit, Output) - The normalized device coordinate viewport limits.


ACCESS
       To use the GKS C-binding routines, load the ncarg_gks and ncarg_c libraries.
