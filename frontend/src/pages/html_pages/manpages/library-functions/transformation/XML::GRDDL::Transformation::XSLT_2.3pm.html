NAME
       XML::GRDDL::Transformation::XSLT_2 - represents an XSLT 2.0 transformation


DESCRIPTION
       Implements XSLT transformations. Uses XML::Saxon::XSLT2.

       XML::GRDDL::Transformation loads this module in an eval block, so if you don't have XML::Saxon::XSLT2
       then things shouldn't break too badly - however you'll only be able to use XSLT 1.0.


SEE ALSO
       XML::GRDDL, XML::GRDDL::Transformation.
