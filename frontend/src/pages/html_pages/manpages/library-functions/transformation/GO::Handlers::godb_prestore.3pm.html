NAME
         GO::Handlers::godb_prestore - transforms OBO XML to GODB XML


SYNOPSIS
DESCRIPTION
       This is a transform for turning OBO XML events into XML events that are isomorphic to the GO Database (ie
       XML element names match GO DB table and column names).

       This transformation is suitable for direct loading into a db using the generic DBIx::DBStag loader (see
       http://stag.sourceforge.net)

       This perl transform may later be replaced by an XSL transform (for speed)
