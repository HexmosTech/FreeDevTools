NAME
       Math::Prime::Util::PPFE - PP front end for Math::Prime::Util


SYNOPSIS
       This loads the PP code and adds input validation front ends.  It is only meant to be used when XS is not
       used.


DESCRIPTION
       Loads PP module and implements PP front-end functions for all XS code.  This is used only if the XS code
       is not loaded.
