NAME
       XML::Grove::Builder - PerlSAX handler for building an XML::Grove


SYNOPSIS
        use PerlSAXParser;
        use XML::Grove::Builder;

        $builder = XML::Grove::Builder->new();
        $parser = PerlSAXParser->new( Handler => $builder );

        $grove = $parser->parse( Source => [SOURCE] );


DESCRIPTION
       "XML::Grove::Builder" is a PerlSAX handler for building an XML::Grove.

       "XML::Grove::Builder" is used by creating a new instance of "XML::Grove::Builder" and providing it as the
       Handler for a PerlSAX parser.  Calling `"parse()"' on the PerlSAX parser will return the grove built from
       that parse.
