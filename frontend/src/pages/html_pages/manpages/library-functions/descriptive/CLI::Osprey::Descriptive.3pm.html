NAME
       CLI::Osprey::Descriptive - Getopt::Long::Descriptive subclass for CLI::Osprey use


VERSION
       version 0.08


DESCRIPTION
       This class overrides Getopt::Long::Descriptive's "usage_class" method to
       Getopt::Long::Descriptive::Usage, which provides customized help text. You probably don't need to use it
       yourself.
