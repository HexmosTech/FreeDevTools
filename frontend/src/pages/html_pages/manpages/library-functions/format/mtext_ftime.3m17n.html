NAME
       mtext_ftime -


SYNOPSIS
       int mtext_ftime (MText * mt, const char * format, const struct tm * tm, MLocale * locale)


DESCRIPTION
       Format date and time. The mtext_ftime() function formats the broken-down time tm according to the format
       specification format and append the result to the M-text mt. The formating is done according to the
       locale locale (if not NULL) or the current locale (LC_TIME).

       The meaning of the arguments tm and format are the same as those of strftime().
