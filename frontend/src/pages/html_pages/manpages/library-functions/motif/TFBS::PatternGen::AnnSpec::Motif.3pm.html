NAME
       TFBS::PatternGen::AnnSpec::Motif - class for unprocessed motifs and associated numerical scores created
       by the Gibbs program


SYNOPSIS
DESCRIPTION
       TFBS::PatternGen::AnnSpec::Motif is used to store and manipulate unprocessed motifs and associated
       numerical scores created by the AnnSpec program. You do not normally want to create a
       TFBS::PatternGen::AnnSpec::Motif yourself. They are created by running TFBS::PatternGen::AnnSpec
