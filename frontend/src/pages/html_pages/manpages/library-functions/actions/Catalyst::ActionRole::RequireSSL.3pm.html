NAME
       Catalyst::ActionRole::RequireSSL - Force an action to be secure only.


VERSION
       version 1.00


SYNOPSIS
         package MyApp::Controller::Foo;

         use parent qw/Catalyst::Controller::ActionRole/;

         sub bar : Local Does('RequireSSL') { ... }
         sub bar : Local Does('NoSSL') { ... }

   HIERARCHY
       You can chain the SSL Roles to allow for enforced combinations such as all secure apart from a certain
       action and vice versa. See the tests to see this in action.
