   uint64_t encap_mask
       Bit mask defining the set of packet encapsulations enabled for the current table action profile. If bit
       (1 << N) is set in encap_mask, then packet encapsulation N is enabled, otherwise it is disabled.

       See also
           enum rte_table_action_encap_type

   uint32_t data_offset
       Input packet to be encapsulated: offset within the input packet buffer to the start of the Ethernet frame
       to be encapsulated. Offset 0 points to the first byte of the MBUF structure.
