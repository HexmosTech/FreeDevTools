NAME
       Mojolicious::Plugin::AssetPack::Asset::Null - Skipped asset


DESCRIPTION
       Mojolicious::Plugin::AssetPack::Asset::Null is a subclass of Mojolicious::Plugin::AssetPack::Asset with
       no new functionality.

       The special thing about this class is that it will be ignored when generating output assets.
