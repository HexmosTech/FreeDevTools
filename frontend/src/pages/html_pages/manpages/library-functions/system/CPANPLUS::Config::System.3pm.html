NAME
       CPANPLUS::Config::System - CPANPLUS configuration file for Debian systems


DESCRIPTION
       This is a CPANPLUS configuration file that sets appropriate default settings on Debian systems.

       The only preconfigured settings are "makemakerflags" (set to "INSTALLDIRS=site") and "buildflags" (set to
       "--installdirs site").

       These settings will not have any effect if "/etc/perl/CPANPLUS/Config/System.pm" is present.
