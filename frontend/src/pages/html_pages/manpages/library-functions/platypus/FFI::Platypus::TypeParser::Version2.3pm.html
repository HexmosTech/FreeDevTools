NAME
       FFI::Platypus::TypeParser::Version2 - FFI Type Parser Version Two


VERSION
       version 2.10


SYNOPSIS
        use FFI::Platypus 2.00;
        my $ffi = FFI::Platypus->new( api => 2 );
        $ffi->type('string(10)');


DESCRIPTION
       This documents the third (version 2) type parser for FFI::Platypus.  This type parser was included with
       FFI::Platypus starting with version 1.58 in an experimental capability, and 2.00 as a stable interface.
       Starting with version 1.00 the main FFI::Platypus documentation describes the version 2 API and you can
       refer to FFI::Platypus::TypeParser::Version1 for details on the version1 API.
