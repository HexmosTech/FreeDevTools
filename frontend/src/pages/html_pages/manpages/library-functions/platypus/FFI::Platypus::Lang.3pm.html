NAME
       FFI::Platypus::Lang - Language specific customizations


VERSION
       version 2.10


SYNOPSIS
        perldoc FFI::Platypus::Lang;


DESCRIPTION
       This namespace is reserved for language specific customizations of FFI::Platypus.  This usually involves
       providing native type maps.  It can also involve computing mangled names.  The default language is C, and
       is defined in FFI::Platypus::Lang::C.

       This package itself doesn't do anything, it serves only as documentation.


SEE ALSO
       FFI::Platypus
           Platypus itself.

       FFI::Platypus::Lang::ASM
           This  language plugin provides no type aliases, and is intended for use with assembly language or for
           when no other language plugin is appropriate.

           Language plugin for the C programming language.

           Non-core language plugin for the C++ programming language.

       FFI::Platypus::Lang::Go
           Non-core language plugin for the Go programming language.

       FFI::Platypus::Lang::Pascal
           Non-core language plugin for the Pascal programming language.

       FFI::Platypus::Lang::Rust

       FFI::Platypus::Lang::Win32
           Language plugin for use with the Win32 API.
