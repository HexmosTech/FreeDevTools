Detailed Description
       A peer associated RTP socket pair for physically connected peer hosts.

       This has no RTCP and assumes the receiver is connected to a known transmitter, hence no 'foreign' packets
       will arrive.

       Author
           David Sugar

       RTP peer host over UDP.

       Examples
           rtpduphello.cpp.
