NAME
       inversesqrt - return the inverse of the square root of the parameter


DECLARATION

       genType inversesqrt(genType x);

       genDType inversesqrt(genDType x);


PARAMETERS
       x
           Specify the value of which to take the inverse of the square root.


DESCRIPTION
       inversesqrt returns the inverse of the square root of $x$; i.e. the value $1 \over { \sqrt x }$. The
       result is undefined if $x \leq 0$.
