NAME
       Geometry::Primitive::Equal - Equality Role


DESCRIPTION
       Geometry::Primitive::Equal is a Moose role for equality.


SYNOPSIS
         with 'Geometry::Primitive::Equal';

         sub equal_to {
             my ($self, $other) = @_;

             # compare and return!
         }


METHODS
   equal_to
       Implement this.

   not_equal_to
       Provided you implement "equal_to", this will be implemented for you!
