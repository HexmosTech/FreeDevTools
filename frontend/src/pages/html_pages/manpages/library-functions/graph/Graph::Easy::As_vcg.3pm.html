NAME
       Graph::Easy::As_vcg - Generate VCG/GDL text from Graph::Easy object


SYNOPSIS
               use Graph::Easy;

               my $graph = Graph::Easy->new();

               my $bonn = Graph::Easy::Node->new(
                       name => 'Bonn',
               );
               my $berlin = Graph::Easy::Node->new(
                       name => 'Berlin',
               );

               $graph->add_edge ($bonn, $berlin);

               print $graph->as_vcg();

       This prints something like this:

               graph: {
                       node: { title: "Bonn" }
                       node: { title: "Berlin" }
                       edge: { sourcename: "Bonn" targetname: "Berlin" }
               }


DESCRIPTION
       "Graph::Easy::As_vcg" contains just the code for converting a Graph::Easy object to either a VCG or GDL
       textual description.

       Note that the generated format is compatible to "GDL" aka Graph Description Language.


EXPORT
       Exports nothing.
