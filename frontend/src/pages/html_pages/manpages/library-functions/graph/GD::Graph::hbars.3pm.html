NAME
       GD::Graph::hbars - make bar graphs with horizontal bars


SYNOPSIS
       use GD::Graph::hbars;


DESCRIPTION
       This is a wrapper module which is completely identical to creating a GD::Graph::bars object with the
       "rotate_chart" attribute set to a true value.


SEE ALSO
       GD::Graph
