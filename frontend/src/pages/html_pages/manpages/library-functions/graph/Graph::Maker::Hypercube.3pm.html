NAME
       Graph::Maker::Hypercube - Create the N-dimensional hypercube graph


VERSION
       Version 0.01


SYNOPSIS
       Creates the N-dimensional hypercube graph.  If the graph is directed then edges are added in both
       directions to create an undirected graph.

               use strict;
               use warnings;
               use Graph;
               use Graph::Maker;
               use Graph::Maker::Hypercube;

               my $g = new Graph::Maker('hypercube', N => 2, undirected => 1);
               # work with the graph


FUNCTIONS
   new %params
       Creates the N-dimensional hypercube graph.  The recognized parameters are N, and graph_maker any others
       are passed onto Graph's constructor.  If N is note given it defaults to 0.  If graph_maker is specified
       and is it will be called to create the Graph class as desired (for example if you have a subclass of
       Graph).
