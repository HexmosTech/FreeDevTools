NAME
       PPIx::QuoteLike::Token::Structure - Represent the structure of the string.


SYNOPSIS
       This class should not be instantiated by the user. See below for public methods.


INHERITANCE
       "PPIx::QuoteLike::Token::Structure" is a PPIx::QuoteLike::Token.

       "PPIx::QuoteLike::Token::Structure" is the parent of PPIx::QuoteLike::Token::Delimiter.


DESCRIPTION
       This Perl class represents the initial token in the string; that is, the 'q', 'qq', 'qx', or (for here
       documents)  '<<' , together with any trailing space. For strings that have no initial token (that is,
       those simply enclosed in quotes) this will be the empty string.
