NAME
       SDL_Cursor - The structure used to identify an SDL cursor.


SYNOPSIS
       #include <SDL3/SDL_mouse.h>

       typedef struct SDL_Cursor SDL_Cursor;


DESCRIPTION
       This is opaque data.


AVAILABILITY
       This struct is available since SDL 3.2.0.
