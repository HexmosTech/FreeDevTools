NAME
       Mat_GetHeader — Returns the header of a MATLAB MAT file given a MAT file structure.


SYNOPSIS
       #include <matio.h>

       const char *
       Mat_GetHeader(const mat_t *matfp);


DESCRIPTION
       The Mat_GetHeader() function returns the header of the MATLAB MAT file given by the MAT file structure.
