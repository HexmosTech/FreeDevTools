NAME
       CGI::Simple::Util - Internal utilities used by CGI::Simple module


SYNOPSIS
           $escaped     = escapeHTML('In HTML you need to escape < > " and & chars');
           $unescaped   = unescapeHTML('&lt;&gt;&quot;&amp;');
           $url_encoded = escape($string);
           $decoded     = unescape($url_encoded);


DESCRIPTION
       CGI::Simple::Util contains essentially non public subroutines used by CGI::Simple. There are HTML and URL
       escape and unescape routines that may be of some use.

       An internal object is used to store a number of system specific details to enable the escape routines to
       be accurate.
