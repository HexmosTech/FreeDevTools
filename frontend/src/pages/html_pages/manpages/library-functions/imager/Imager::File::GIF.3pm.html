NAME
       Imager::File::GIF - read and write GIF files


SYNOPSIS
         use Imager;

         my $img = Imager->new;
         $img->read(file=>"foo.gif")
           or die $img->errstr;

         $img->write(file => "foo.gif")
           or die $img->errstr;


DESCRIPTION
       Imager's GIF support is documented in Imager::Files.
