SYNOPSIS
          bool
          mongoc_read_concern_is_default (mongoc_read_concern_t *read_concern);


PARAMETERS
       â€¢ read_concern: A pointer to a mongoc_read_concern_t.


DESCRIPTION
       Returns  true  if  read_concern  has not been modified from the default. For example, if no "readConcern"
       option is set in  the  MongoDB  URI  and  you  have  not  called  mongoc_client_set_read_concern(),  then
       mongoc_read_concern_is_default()     is     true     for     the     read     concern     returned     by
       mongoc_client_get_read_concern().
