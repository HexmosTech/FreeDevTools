NAME
       DBIx::Class::Helper::ResultSet::Errors - add exceptions to help when calling Result methods on an
       ResultSets


SYNOPSIS
        package MyApp::Schema::ResultSet::Foo;

        __PACKAGE__->load_components(qw{Helper::ResultSet::Errors});

        ...

        1;

       And then in a script or something:

        my $col = $rs->id

        # dies with a helpful error!


DESCRIPTION
       Users new to "DBIx::Class" often make the mistake of treating ResultSets like Results.  This helper
       ameliorates the situation by giving a helpful error when the user calls methods for the result on the
       ResultSet.  See "NOTE" in DBIx::Class::Helper::ResultSet for a nice way to apply it to your entire
       schema.
