NAME
       FcPatternCreate - Create a pattern


SYNOPSIS
       #include <fontconfig/fontconfig.h>

       FcPattern * FcPatternCreate (void);


DESCRIPTION
       Creates a pattern with no properties; used to build patterns from scratch.
