NAME
       FcCharSetEqual - Compare two charsets


SYNOPSIS
       #include <fontconfig/fontconfig.h>

       FcBool FcCharSetEqual (const FcCharSet *a, const FcCharSet *b);


DESCRIPTION
       Returns whether a and b contain the same set of Unicode chars.
