NAME
       FcUtf8ToUcs4 - convert UTF-8 to UCS4


SYNOPSIS
       #include <fontconfig/fontconfig.h>

       int FcUtf8ToUcs4 (FcChar8 *src, FcChar32 *dst, int len);


DESCRIPTION
       Converts  the  next  Unicode  char from src into dst and returns the number of bytes containing the char.
       src must be at least len bytes long.
