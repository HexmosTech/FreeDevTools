NAME
       FcFileIsDir - check whether a file is a directory


SYNOPSIS
       #include <fontconfig/fontconfig.h>

       FcBool FcFileIsDir (const FcChar8 *file);


DESCRIPTION
       Returns FcTrue if file is a directory, otherwise returns FcFalse.
