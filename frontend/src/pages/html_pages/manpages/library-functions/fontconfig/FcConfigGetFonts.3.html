NAME
       FcConfigGetFonts - Get config font set


SYNOPSIS
       #include <fontconfig/fontconfig.h>

       FcFontSet * FcConfigGetFonts (FcConfig *config, FcSetName set);


DESCRIPTION
       Returns  one  of the two sets of fonts from the configuration as specified by set. This font set is owned
       by the library and must not be modified or freed.  If config is NULL, the current configuration is used.

       This function isn't MT-safe. FcConfigReference must be called before using this and then  FcConfigDestroy
       when the return value is no longer referenced.
