NAME
       FcLangSetCopy - copy a langset object


SYNOPSIS
       #include <fontconfig/fontconfig.h>

       FcLangSet * FcLangSetCopy (const FcLangSet *ls);


DESCRIPTION
       FcLangSetCopy creates a new FcLangSet object and populates it with the contents of ls.
