NAME
       FcCharSetSubtract - Subtract charsets


SYNOPSIS
       #include <fontconfig/fontconfig.h>

       FcCharSet * FcCharSetSubtract (const FcCharSet *a, const FcCharSet *b);


DESCRIPTION
       Returns a set including only those chars found in a but not b.
