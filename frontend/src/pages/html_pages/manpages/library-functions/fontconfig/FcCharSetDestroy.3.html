NAME
       FcCharSetDestroy - Destroy a character set


SYNOPSIS
       #include <fontconfig/fontconfig.h>

       void FcCharSetDestroy (FcCharSet *fcs);


DESCRIPTION
       FcCharSetDestroy  decrements  the  reference  count fcs.  If the reference count becomes zero, all memory
       referenced is freed.
