NAME
       FcAtomicNewFile - return new temporary file name


SYNOPSIS
       #include <fontconfig/fontconfig.h>

       FcChar8 * FcAtomicNewFile (FcAtomic *atomic);


DESCRIPTION
       Returns the filename for writing a new version of the file referenced by atomic.
