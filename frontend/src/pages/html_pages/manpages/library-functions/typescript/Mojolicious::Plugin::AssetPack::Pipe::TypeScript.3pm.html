NAME
       Mojolicious::Plugin::AssetPack::Pipe::TypeScript - Process TypeScript .ts files


SYNOPSIS
         $app->plugin(pipes => [qw(TypeScript JavaScript Combine)]);
         $app->asset->process("app.js" => qw(foo.ts));


DESCRIPTION
       Mojolicious::Plugin::AssetPack::Pipe::TypeScript will process <https://www.typescriptlang.org/> files
       into JavaScript.

       This module require the "typescript-simple" nodejs library to be installed.  "typescript-simple" will be
       automatically installed using <https://www.npmjs.com/> unless already installed.


METHODS
   process
       See "process" in Mojolicious::Plugin::AssetPack::Pipe.
