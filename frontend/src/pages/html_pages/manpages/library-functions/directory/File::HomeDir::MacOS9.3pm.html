NAME
       File::HomeDir::MacOS9 - Find your home and other directories on legacy Macintosh systems


SYNOPSIS
         use File::HomeDir;

         # Find directories for the current user
         $home    = File::HomeDir->my_home;
         $desktop = File::HomeDir->my_desktop;


DESCRIPTION
       This module provides implementations for determining common user directories on legacy Mac hosts. In
       normal usage this module will always be used via File::HomeDir.

       This module is no longer actively maintained, and is included only for extreme back-compatibility.

       Only the "my_home" and "my_desktop" methods are supported.
