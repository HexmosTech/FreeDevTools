NAME
       FcConfigGetFontDirs - Get font directories


SYNOPSIS
       #include <fontconfig/fontconfig.h>

       FcStrList * FcConfigGetFontDirs (FcConfig *config);


DESCRIPTION
       Returns  the list of font directories in config. This includes the configured font directories along with
       any directories below those in the filesystem.  If config is NULL, the current configuration is used.
