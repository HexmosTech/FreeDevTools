NAME
       dnsjit.core - Core modules for dnsjit


DESCRIPTION
       Core modules for handling things like logging, DNS messages and receiver/receive functionality.

   Global Variables
       The following global variables exists in dnsjit.

       arg    A  table  with  the  arguments given on the command line, the first will be the path to the dnsjit
              binary, second will be the path to the script.
