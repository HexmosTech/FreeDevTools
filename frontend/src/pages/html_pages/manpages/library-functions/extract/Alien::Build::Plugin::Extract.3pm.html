NAME
       Alien::Build::Plugin::Extract - Extract Alien::Build plugins


VERSION
       version 2.84


SYNOPSIS
        use alienfile
        share {
          plugin 'Extract' => 'tar.gz';
        };


DESCRIPTION
       Extract plugins extract packages that have been downloaded from the internet.  Unless you are doing
       something unusual you will likely want to use the Alien::Build::Plugin::Extract::Negotiate plugin to
       select the best Extract plugin available.

       Alien::Build::Plugin::Extract::ArchiveTar
           Extract using "tar".  Typically also works with compressed tarballs like "tar.gz".

       Alien::Build::Plugin::Extract::ArchiveZip
           Extract using Archive::Zip.

       Alien::Build::Plugin::Extract::CommandLine
           Extract using command line tools like "tar" or "unxip".

       Alien::Build::Plugin::Extract::Directory
           Extract a local directory.

       Alien::Build::Plugin::Extract::File
           "Extract" a single file.

       Alien::Build::Plugin::Extract::Negotiate
           Pick the best extract plugin based on the extension of the package archive.
