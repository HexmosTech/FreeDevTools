NAME
       Locale::Maketext::Extract::Plugin::Haml - HAML format parser


VERSION
       version 1.00


SYNOPSIS
           $plugin = Locale::Maketext::Extract::Plugin::Haml->new(
               $lexicon            # A Locale::Maketext::Extract object
               @file_types         # Optionally specify a list of recognised file types
           )

           $plugin->extract($filename,$filecontents);


DESCRIPTION
       Extracts strings to localise from HAML files.


SHORT PLUGIN NAME
           haml


VALID FORMATS
       Extracts strings in the same way as Locale::Maketext::Extract::Plugin::Perl, but only ones within "text"
       components of HAML files.


KNOWN FILE TYPES
       .haml


SEE ALSO
       xgettext.pl
           for extracting translatable strings from common template systems and perl source files.
