NAME
       Verilog::Netlist::PinSelection - Nets attached to a Verilog Cell's Pins


DESCRIPTION
       Verilog::Netlist::PinSelection objects are used by Verilog::Netlist::Pin to define ranges of nets
       attached to the respective pin of a cell.


ACCESSORS
       $self->netname
           Name of the respective net, or, if use_pinselects is disabled, the string representation of the whole
           pin value. In the case of a sized constant only the part following the ' is stored while the width is
           encoded in the msb and lsb fields.

       $self->lsb
           Least significant bit of the underlying net within the selection.

       $self->msb
           Most significant bit of the underlying net within the selection.


MEMBER FUNCTIONS
       $self->bracketed_msb_lsb
           Returns the common string representation of a vectored net, e.g. netA[15:8].
