NAME
       RT::Interface::Email::Action::CommandByMail - Change metadata of ticket via email


DESCRIPTION
       This action provides compatibility with the new mail plugin system introduced in RT 4.4. It provides an
       alternate to the default comment and correspond handlers provided by RT.

   CBMHandleEither
       This method is the CommandByMail version of RT's _HandleEither.
