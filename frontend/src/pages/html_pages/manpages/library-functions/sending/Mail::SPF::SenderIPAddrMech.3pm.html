NAME
       Mail::SPF::SenderIPAddrMech - Abstract base class for SPF record mechanisms that operate on the SMTP
       sender's IP address


VERSION
       version 3.20250505


DESCRIPTION
       Mail::SPF::SenderIPAddrMech is an abstract base class for SPF record mechanisms that operate on the SMTP
       sender's IP address.  It cannot be instantiated directly.  Create an instance of a concrete sub-class
       instead.

       See "Constructors" in Mail::SPF::Mech.
