Detailed Description
       All PStreams classes are declared in namespace redi.

       Like the standard iostreams, PStreams is a set of class templates, taking a character type and traits
       type. As with the standard streams they are most likely to be used with char and the default traits type,
       so typedefs for this most common case are provided.

       The pstream_common class template is not intended to be used directly, it is used internally to provide
       the common functionality for the other stream classes.
