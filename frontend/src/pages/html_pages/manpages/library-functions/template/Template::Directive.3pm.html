NAME
       Template::Directive - Perl code generator for template directives


SYNOPSIS
           # no user serviceable parts inside


DESCRIPTION
       The "Template::Directive" module defines a number of methods that generate Perl code for the runtime
       representation of the various Template Toolkit directives.

       It is used internally by the Template::Parser module.
