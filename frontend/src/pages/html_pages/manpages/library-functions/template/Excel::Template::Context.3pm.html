NAME
       Excel::Template::Context - Excel::Template::Context


PURPOSE
       This is a helper node that provides the global context for the nodes do their processing within. It
       provides attribute scoping, parameter resolution, and other very nice things.

       Documentation is provided for if you wish to subclass another node.
