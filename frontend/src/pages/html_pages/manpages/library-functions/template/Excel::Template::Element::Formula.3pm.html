NAME
       Excel::Template::Element::Formula - Excel::Template::Element::Formula


PURPOSE
       To write formulas to the worksheet


NODE NAME
       FORMULA


INHERITANCE
       Excel::Template::Element::Cell


ATTRIBUTES
       All attributes a CELL can have, a FORMULA can have, including the ability to be referenced using the
       'ref' attribute.


CHILDREN
       None


EFFECTS
       This will consume one column on the current row.


DEPENDENCIES
       None


USAGE
         <formula text="=(1 + 2)"/>
         <formula>=SUM(A1:A5)</formula>

         <formula text="$Param2"/>
         <formula>=(A1 + <var name="Param">)</formula>

       In the above example, four formulas are written out. The first two have the formula hard-coded. The
       second two have variables. The third and fourth items have another thing that should be noted. If you
       have a formula where you want a variable in the middle, you have to use the latter form. Variables within
       parameters are the entire parameter's value.
