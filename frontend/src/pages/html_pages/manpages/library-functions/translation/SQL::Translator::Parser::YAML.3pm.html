NAME
       SQL::Translator::Parser::YAML - Parse a YAML representation of a schema


SYNOPSIS
           use SQL::Translator;

           my $translator = SQL::Translator->new(parser => "YAML");


DESCRIPTION
       "SQL::Translator::Parser::YAML" parses a schema serialized with YAML.
