NAME
       SQL::Translator::Parser::JSON - Parse a JSON representation of a schema


SYNOPSIS
           use SQL::Translator;

           my $translator = SQL::Translator->new(parser => "JSON");


DESCRIPTION
       "SQL::Translator::Parser::JSON" parses a schema serialized with JSON.
