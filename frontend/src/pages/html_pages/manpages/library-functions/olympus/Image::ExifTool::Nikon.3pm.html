NAME
       Image::ExifTool::Nikon - Nikon EXIF maker notes tags


SYNOPSIS
       This module is loaded automatically by Image::ExifTool when required.


DESCRIPTION
       This module contains definitions required by Image::ExifTool to interpret Nikon maker notes in EXIF
       information.
