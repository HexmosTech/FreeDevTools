NAME
       iob_free - free an I/O batch


SYNTAX
       #include <libowfat/iob.h>

       void iob_free(io_batch* b);


DESCRIPTION
       iob_free  frees all resources associated with b.  Calling iob_free is equivalent to calling iob_reset and
       free.

       iob_new(3), iob_reset(3)
