NAME
       Net::Server::INET - Net::Server personality


SYNOPSIS
           use base qw(Net::Server::INET);

           sub process_request {
               #...code...
           }

           Net::Server::INET->run();


DESCRIPTION
       Please read the pod on Net::Server first.  This module is a personality, or extension, or sub class, of
       the Net::Server module.

       This personality is intended for use with inetd.  It offers no methods beyond the Net::Server base class.
       This module operates by overriding the pre_bind, bind, accept, and post_accept methods to let all socket
       processing to be done by inetd.
