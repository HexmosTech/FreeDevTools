NAME
       Boulder::Blast::NCBI - Parse and read NCBI BLAST files


SYNOPSIS
       Not for direct use.  Use Boulder::Blast instead.


DESCRIPTION
       Specialized parser for NCBI format BLAST output.  Loaded automatically by Boulder::Blast.
