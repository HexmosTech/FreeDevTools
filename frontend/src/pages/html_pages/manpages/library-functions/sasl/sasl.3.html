NAME
       sasl - Cyrus SASL documentation


DESCRIPTION
       The CMU Cyrus SASL library is a general purpose authentication library for sever and client applications.

       System Administrators

       For  information  on  setting  up/configuring  the  SASL  library  see the System Administrators Guide: ‚Äê
       https://www.cyrusimap.org/sasl/sasl/sysadmin.html

       Programmers

       See man pages for individual sasl functions or the Programmers Guide in the doc/ directory  of  the  SASL
       distribution.

       Conforming to RFC 2222
