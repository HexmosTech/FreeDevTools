NAME
       sasl_auxprop_getctx - Cyrus SASL documentation


SYNOPSIS
       #include <sasl/sasl.h>

       int sasl_auxprop_getctx(sasl_conn_t *conn)


DESCRIPTION
       int sasl_auxprop_getctx(sasl_conn_t *conn)
              Fetches  an  auxiliary  property  context  for  the connection on which the functions described in
              sasl_auxprop(3) can operate.

              Parameters

                     • conn – pointer to the sasl_conn_t for which the request is being made.

              Returns
                     A pointer to the context on success. Returns NULL on failure.

       type sasl_conn_t
              Context for a SASL connection negotiation


CONFORMING TO
       RFC 4422


SEE ALSO
       sasl(3), sasl_auxprop(3), sasl_auxprop_request(3)
