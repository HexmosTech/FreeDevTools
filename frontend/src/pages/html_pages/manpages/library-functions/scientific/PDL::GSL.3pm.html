NAME
       PDL::GSL - PDL interface to the GNU Scientific Library

       PDL::GSL::CDF
       PDL::GSL::DIFF
       PDL::GSL::INTEG
       PDL::GSL::INTERP
       PDL::GSL::LINALG
       PDL::GSL::MROOT
       PDL::GSL::RNG
       PDL::GSL::SF
       PDL::Stats::Distr
