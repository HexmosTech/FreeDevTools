NAME
       PS_setmiterlimit — Sets the miter limit


SYNOPSIS
       #include <pslib.h>

       void PS_setmiterlimit(PSDoc *psdoc, float value)


DESCRIPTION
       If  two  lines  join in a small angle and the line join is set to miter, then the resulting spike will be
       very long. The miter limit is the maximum ratio of the miter length (the length of  the  spike)  and  the
       line width.
