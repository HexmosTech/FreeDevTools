NAME
       igraph - a library for creating and manipulating graphs


DESCRIPTION
       igraph  is  a  C  library  for  complex  network  analysis and graph theory, with emphasis on efficiency,
       portability and ease of use.


DOCUMENTATION
       The full documentation can be downloaded from the homepage of the library: <https://igraph.org/c/doc>


BUGS
       If you think you have found a bug in igraph, feel free to file a bug report  in  the  issue  tracker  at:
       <https://github.com/igraph/igraph/issues>
