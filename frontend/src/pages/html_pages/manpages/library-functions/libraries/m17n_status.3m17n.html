NAME
       m17n_status -


SYNOPSIS
       enum M17NStatus m17n_status (void)


DESCRIPTION
       Report which part of the m17n library is initialized. The m17n_status() function returns one of these
       values depending on which part of the m17n library is initialized:

       M17N_NOT_INITIALIZED, M17N_CORE_INITIALIZED, M17N_SHELL_INITIALIZED, M17N_GUI_INITIALIZED
