Name
       Text::Markup::Trac - Trac wiki syntax parser for Text::Markup


Synopsis
         my $html = Text::Markup->new->parse(file => 'README.trac');
         my $raw  = Text::Markup->new->parse(
             file => 'README.trac',
             options => [ raw => 1 ],
         );


Description
       This is the Trac wiki syntax <https://trac.edgewall.org/wiki/WikiFormatting> parser for Text::Markup. It
       reads in the file (relying on a BOM <https://www.unicode.org/unicode/faq/utf_bom.html#BOM>), hands it off
       to Text::Trac for parsing, and then returns the generated HTML as an encoded UTF-8 string with an
       "http-equiv="Content-Type"" element identifying the encoding as UTF-8.

       It recognizes files with the following extensions as Trac:

       .trac
       .trc

       To  change  it  the files it recognizes, load this module directly and pass a regular expression matching
       the desired extension(s), like so:

         use Text::Markup::Trac qr{tr[au]ck?};

       Normally this module returns the output wrapped in a minimal HTML document skeleton. If  you  would  like
       the raw output without the skeleton, you can pass the "raw" option to "parse".

       In addition, Text::Markup::Mediawiki supports all of the Text::Trac options, including:

       "trac_url"
       "disable_links"
       "enable_links"
