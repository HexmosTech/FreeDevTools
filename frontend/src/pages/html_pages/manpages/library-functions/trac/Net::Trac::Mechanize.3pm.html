NAME
       Net::Trac::Mechanize - Provides persistent credentials for the Trac instance


DESCRIPTION
       This class subclasses WWW::Mechanize to provide persistent HTTP credentials when accessing a Trac
       instance.


ACCESSORS / MUTATORS
   trac_user
   trac_password

METHODS
   get_basic_credentials
       Returns the credentials that WWW::Mechanize expects.
