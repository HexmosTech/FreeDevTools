NAME
       reflect - calculate the reflection direction for an incident vector


DECLARATION

       genType reflect(genType I, genType N);

       genDType reflect(genDType I, genDType N);


PARAMETERS
       I
           Specifies the incident vector.

       N
           Specifies the normal vector.


DESCRIPTION
       For a given incident vector I and surface normal N reflect returns the reflection direction calculated as
       I - 2.0 * dot(N, I) * N.

       N should be normalized in order to achieve the desired result.
