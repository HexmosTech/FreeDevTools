NAME
       "Future::IO::Impl::IOAsync" - implement "Future::IO" using "IO::Async"


DESCRIPTION
       This module provides an implementation for Future::IO which uses IO::Async.

       There are no additional methods to use in this module; it simply has to be loaded, and will provide the
       "Future::IO" implementation methods.

          use Future::IO;
          use Future::IO::Impl::IOAsync;

          my $f = Future::IO->sleep(5);
          ...
