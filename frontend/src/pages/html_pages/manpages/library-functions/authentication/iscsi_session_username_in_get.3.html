NAME
       iscsi_session_username_in_get - Retrieve authentication username of specified session.


SYNOPSIS
       const char * iscsi_session_username_in_get (struct iscsi_session *se);


ARGUMENTS
       se          Pointer  of 'struct iscsi_session'.  If this pointer is NULL, your program will be terminated
                   by assert.


DESCRIPTION
       Retrieve the inbound authentication username of  specified  iSCSI  session.   Currently  open-iscsi  only
       support  CHAP  authentication  method.   The  inbound  authentication  here  means  the  iSCSI  initiator
       authenticates the iSCSI target using CHAP.  It's controlled this setting in iscsid.conf:


RETURN
       const char *. Empty string if not using inbound CHAP authentication  or  failed  to  read  authentication
       information.   No  need  to  free  this  memory,  the  resources  will get freed by iscsi_session_free or
       iscsi_sessions_free.
