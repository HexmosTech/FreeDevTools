NAME
       gset_marker_size  (Set  marker size scale factor) - sets the polymarker size scale factor -- the relative
       size of the marker.


SYNOPSIS
       #include <ncarg/gks.h>

       void gset_marker_size(Gdouble marker_size);


DESCRIPTION
       marker_size (Input) A scale factor to control the size of the marker.  It must be  greater than or  equal
                   to 0.  Since marker_size = 1.0 by default, setting marker_size = 2.0 doubles polymarker size.


ACCESS
       To use the GKS C-binding routines, load the ncarg_gks and ncarg_c libraries.
