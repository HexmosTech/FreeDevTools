NAME
       Net::Time - time and daytime network client interface


SYNOPSIS
           use Net::Time qw(inet_time inet_daytime);

           print inet_time();          # use default host from Net::Config
           print inet_time('localhost');
           print inet_time('localhost', 'tcp');

           print inet_daytime();       # use default host from Net::Config
           print inet_daytime('localhost');
           print inet_daytime('localhost', 'tcp');


DESCRIPTION
       "Net::Time" provides subroutines that obtain the time on a remote machine.

   Functions
       "inet_time([$host[, $protocol[, $timeout]]])"
           Obtain  the  time  on  $host,  or  some  default host if $host is not given or not defined, using the
           protocol as defined in RFC868. The optional argument $protocol should define  the  protocol  to  use,
           either  "tcp"  or  "udp".  The result will be a time value in the same units as returned by time() or
           undef upon failure.

       "inet_daytime([$host[, $protocol[, $timeout]]])"
           Obtain the time on $host, or some default host if $host is  not  given  or  not  defined,  using  the
           protocol  as  defined  in  RFC867. The optional argument $protocol should define the protocol to use,
           either "tcp" or "udp". The result will be an ASCII string or undef upon failure.
