NAME
       SDL_HAPTIC_FRICTION - Friction effect supported - uses axes movement.


HEADER FILE
       Defined in SDL3/SDL_haptic.h


SYNOPSIS
       #include "SDL3/SDL.h"

       #define SDL_HAPTIC_FRICTION     (1u<<10)


DESCRIPTION
       Condition haptic effect that simulates friction. Effect is based on the axes movement.


AVAILABILITY
       This macro is available since SDL 3.2.0.
