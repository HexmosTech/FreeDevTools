NAME
       Image::ExifTool::Trailer - Read JPEG trailer written by various phone makes


SYNOPSIS
       This module is used by Image::ExifTool


DESCRIPTION
       This module contains definitions required by Image::ExifTool to read metadata the trailer written by some
       Vivo, OnePlus and Google phones.
