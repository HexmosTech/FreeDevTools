NAME
       ck_hs_stat — get hash set status


LIBRARY
       Concurrency Kit (libck, -lck)


SYNOPSIS
       #include <ck_hs.h>

       void
       ck_hs_stat(ck_hs_t *hs, struct ck_hs_stat *st);


DESCRIPTION
       The  ck_hs_stat(3)  function  will store various hash set statistics in the object pointed to by st.  The
       ck_hs_stat structure is defined as follows:

             struct ck_hs_stat {
                     unsigned long tombstones;   /* Current number of tombstones in hash set. */
                     unsigned long n_entries;    /* Current number of keys in hash set. */
                     unsigned int probe_maximum; /* Longest read-side probe sequence. */
             };


RETURN VALUES
       ck_hs_stat(3) has no return value.


ERRORS
       Behavior is undefined if hs is uninitialized. Behavior is undefined if this function is called by a  non-
       writer thread.
