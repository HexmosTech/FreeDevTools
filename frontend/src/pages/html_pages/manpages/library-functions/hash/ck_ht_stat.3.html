NAME
       ck_ht_stat — get hash table status


LIBRARY
       Concurrency Kit (libck, -lck)


SYNOPSIS
       #include <ck_ht.h>

       void
       ck_ht_stat(ck_ht_t *ht, struct ck_ht_stat *st);


DESCRIPTION
       The  ck_ht_stat(3)  function  will store various hash set statistics in the object pointed to by st.  The
       ck_ht_stat structure is defined as follows:

             struct ck_ht_stat {
                     uint64_t probe_maximum; /* Longest read-side probe sequence. */
                     uint64_t n_entries;     /* Current number of keys in hash set. */
             };


RETURN VALUES
       ck_ht_stat(3) has no return value.


ERRORS
       Behavior is undefined if ht has not been initialized.
