NAME
       HTML::FormFu::Element::Radiogroup - Group of radiobutton form fields


VERSION
       version 2.07


SYNOPSIS
       YAML config:

           ---
           elements:
             - type: Radiogroup
               name: sex
               options:
                 - [ 'm', 'Male' ]
                 - [ 'f', 'Female' ]


DESCRIPTION
       Convenient to use group of radio button fields.

       Use the same syntax as you would to create a Select element optgroup to create RadioGroup sub-groups, see
       "options" in HTML::FormFu::Role::Element::Group for details.


METHODS
   options
       See "options" in HTML::FormFu::Role::Element::Group.

   values
       See "values" in HTML::FormFu::Role::Element::Group.

   value_range
       See "value_range" in HTML::FormFu::Role::Element::Group.

   empty_first
       See "empty_first" in HTML::FormFu::Role::Element::Group.

   auto_id
       In addition to the substitutions documented by "auto_id" in HTML::FormFu, %c will be replaced by an
       incremented integer, to ensure there are no duplicated ID's.

           ---
           elements:
             type: Radiogroup
             name: foo
             auto_id: "%n_%c"

   reverse_group
       See "reverse_group" in HTML::FormFu::Element::Checkboxgroup.


DEPRECATED METHODS
   radiogroup_filename
