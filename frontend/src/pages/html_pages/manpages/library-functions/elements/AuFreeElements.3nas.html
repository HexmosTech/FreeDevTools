Name
       AuFreeElements - free the memory used by lists of flow elements


Synopsis
       #include <audio/audiolib.h>

       void AuFreeElements(server, num_elements, elements)
           AuServer *server;
           int num_elements;
           AuElement *elements;


Arguments
       server    Specifies the connection to the audio server.

       num_elements
                 Specifies the number of elements to free.

       elements  Specifies a list of elements returned by AuGetElements.


Description
       AuFreeElements frees the memory allocated by AuGetElements.

       audiolib - Network Audio System C Language Interface
