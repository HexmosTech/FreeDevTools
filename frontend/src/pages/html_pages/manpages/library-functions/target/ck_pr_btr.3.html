NAME
       ck_pr_btr_ptr, ck_pr_btr_uint, ck_pr_btr_int, ck_pr_btr_64, ck_pr_btr_32, ck_pr_btr_16 — atomic bit test-
       and-reset operations


LIBRARY
       Concurrency Kit (libck, -lck)


SYNOPSIS
       #include <ck_pr.h>

       bool
       ck_pr_btr_ptr(void *target, unsigned int bit_index);

       bool
       ck_pr_btr_uint(uint *target, unsigned int bit_index);

       bool
       ck_pr_btr_int(int *target, unsigned int bit_index);

       bool
       ck_pr_btr_64(uint64_t *target, unsigned int bit_index);

       bool
       ck_pr_btr_32(uint32_t *target, unsigned int bit_index);

       bool
       ck_pr_btr_16(uint16_t *target, unsigned int bit_index);


DESCRIPTION
       The  ck_pr_btr(3)  family of functions atomically fetch the value of the bit in target at index bit_index
       and set that bit to 0.


RETURN VALUES
       This family of functions returns the original value of the bit at offset bit_index that is in  the  value
       pointed to by target.
