NAME
       Bio::Annotation::Target - Provides an object which represents a target (ie, a similarity hit) from one
       object to something in another database


SYNOPSIS
          $target1 = Bio::Annotation::Target->new(-target_id  => 'F321966.1',
                                                 -start      => 1,
                                                 -end        => 200,
                                                 -strand     => 1,   # or -1
                                                );

          # or

          $target2 = Bio::Annotation::Target->new();
          $target2->target_id('Q75IM5');
          $target2->start(7);
          # ... etc ...

          # Target is-a Bio::AnnotationI object, can be added to annotation
          # collections, e.g. the one on features or seqs
          $feat->annotation->add_Annotation('Target', $target2);


DESCRIPTION
       Provides an object which represents a target (ie, a similarity hit) from one object to something in
       another database without prescribing what is in the other database

   tagname
        Title   : tagname
        Usage   : $obj->tagname($newval)
        Function: Get/set the tagname for this annotation value.

                  Setting this is optional. If set, it obviates the need to
                  provide a tag to Bio::AnnotationCollectionI when adding
                  this object. When obtaining an AnnotationI object from the
                  collection, the collection will set the value to the tag
                  under which it was stored unless the object has a tag
                  stored already.

        Example :
        Returns : value of tagname (a scalar)
        Args    : new value (a scalar, optional)
