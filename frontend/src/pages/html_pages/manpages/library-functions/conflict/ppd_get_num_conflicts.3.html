NAME
       ppd_get_num_conflicts - Check a PPD file for conflicts


SYNOPSIS
       #include <ppd.h>

       int ppd_get_num_conflicts (PpdFile *ppd);


DESCRIPTION
       The  ppd_get_num_conflicts()  function is used to determine the number of option conflicts within a given
       PPD file.  The input to ppd_get_num_conflicts is a pointer to a PpdFile structure which  is  obtained  by
       one of the ppd_file_new(3) functions.


RETURN VALUE
       The  integer returned from the functions is the number of option conflicts located in the PPD file.  Note
       that if the input parameter is a NULL pointer, the function will return 0.
