NAME
       Pod::Perldoc::BaseTo - Base for Pod::Perldoc formatters


SYNOPSIS
           package Pod::Perldoc::ToMyFormat;

           use parent qw( Pod::Perldoc::BaseTo );
           ...


DESCRIPTION
       This package is meant as a base of Pod::Perldoc formatters, like Pod::Perldoc::ToText,
       Pod::Perldoc::ToMan, etc.

       It provides default implementations for the methods

           is_pageable
           write_with_binmode
           output_extension
           _perldoc_elem

       The concrete formatter must implement

           new
           parse_from_file
