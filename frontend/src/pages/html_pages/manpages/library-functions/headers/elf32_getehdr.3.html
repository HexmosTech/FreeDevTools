NAME
       elf32_getehdr - retrieve the ELF header for a 32-bit ELF descriptor


SYNOPSIS
       #include <libelf.h>

       Elf32_Ehdr *elf32_getehdr(Elf *elf);


DESCRIPTION
       elf32_getehdr  retrieves the ELF header for the given 32-bit ELF descriptor elf.  The ELF header contains
       crucial metadata about the ELF file, such as the type, machine,  version,  entry  point,  program  header
       table offset, and section header table offset. See elf(5) for more information regarding the ELF header.


PARAMETERS
       elf    Pointer to the ELF descriptor from which to retrieve the ELF header.


RETURN VALUE
       On success, elf32_getehdr returns a pointer to the Elf32_Ehdr structure. If the ELF descriptor is invalid
       or not a 32-bit ELF, it returns NULL and sets an error code retrievable by elf_errno(3).


SEE ALSO
       elf64_getehdr(3), elf_begin(3), elf_errno(3), elf(5)
