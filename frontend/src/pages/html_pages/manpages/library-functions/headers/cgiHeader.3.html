NAME
       cgiHeader - Print the HTTP header


SYNOPSYS
       #include <cgi.h>

       void cgiHeader ();


DESCRIPTION
       This  routine  just  prints out the Contents-type: line to make the web server happy.  By default the CGI
       library assumes that you return regular HTML code and uses "text/html" as  type.   If  your  CGI  program
       returns  something  else,  you  may set the type with cgiSetType(3).  You may also  add additional header
       lines by using cgiSetHeader(3).


RETURN VALUE
       This routine does not return a value.
