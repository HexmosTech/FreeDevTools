NAME
       probe::nfs.aop.readpages - NFS client reading multiple pages


SYNOPSIS
       nfs.aop.readpages


VALUES
       dev
           device identifier

       nr_pages
           number of pages attempted to read in this execution

       file
           filp argument

       rsize
           read size (in bytes)

       ino
           inode number

       size
           number of pages attempted to read in this execution

       rpages
           read size (in pages)


DESCRIPTION
       Fires when in readahead way, read several pages once
