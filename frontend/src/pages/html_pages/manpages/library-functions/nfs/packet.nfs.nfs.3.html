NAME
       packet.nfs.nfs - NFS module


DESCRIPTION
       Process  the  NFS layer and return the correct NFS object. The function returns either a NULL(), CB_NULL,
       COMPOUND or CB_COMPOUND object.


FUNCTIONS
       NFS(rpc, callback)
       Process the NFS layer and return the correct NFS object
