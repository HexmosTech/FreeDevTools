NAME
       function::tcpmib_local_port - Get the local port


SYNOPSIS
           tcpmib_local_port:long(sk:long)


ARGUMENTS
       sk
           pointer to a struct inet_sock


DESCRIPTION
       Returns the sport from a struct inet_sock in host order.
