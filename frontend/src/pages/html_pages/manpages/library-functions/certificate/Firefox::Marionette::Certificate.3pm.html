NAME
       Firefox::Marionette::Certificate - Represents a x509 Certificate from Firefox


VERSION
       Version 1.22


SYNOPSIS
           use Firefox::Marionette();
           use v5.10;

           my $firefox = Firefox::Marionette->new();
           foreach my $certificate (sort { $a->display_name() cmp $b->display_name() } $firefox->certificates()) {
              ...
           }


DESCRIPTION
       This module handles the implementation of a x509 Certificate from Firefox


SUBROUTINES/METHODS
   common_name
       returns the common name from the certificate.  This can contain the domain name (or wildcard) attached to
       the certificate or a Certificate Authority name, such as 'VeriSign Class 3 Public Primary Certification
       Authority - G4'

   db_key
       returns a unique value for the certificate, such as
       'AAAAAAAAAAAAAAAQAAAAzS+A/iOMDiIPSGcSKJGHrLMwgcoxCzAJBgNVBAYTAlVTMRcwFQYDVQQKEw5WZXJpU2lnbiwgSW5jLjEfMB0GA1UECxMWVmVyaVNpZ24gVHJ1c3QgTmV0d29yazE6MDgGA1UECxMxKGMpIDIwMDcgVmVyaVNpZ24sIEluYy4gLSBGb3IgYXV0aG9yaXplZCB1c2Ugb25seTFFMEMGA1UEAxM8VmVyaVNpZ24gQ2xhc3MgMyBQdWJsaWMgUHJpbWFyeSBDZXJ0aWZpY2F0aW9uIEF1dGhvcml0eSAtIEc0'

   display_name
       returns the display name field, such as 'VeriSign Class 3 Public Primary Certification Authority - G4'

   email_address
       returns the emailAddress field if supplied, otherwise it will return undef.

   is_any_cert
       returns a boolean value to determine if the certificate is a certificate.  I would regard it as quite
       surprising to get a certificate that returned false.

   is_built_in_root
       returns a boolean value to determine if the certificate is a built in root certificate.

   is_ca_cert
       returns a boolean value to determine if the certificate is a certificate authority certificate

   is_email_cert
       returns a boolean value to determine if the certificate is an email certificate.

   is_server_cert
       returns a boolean value to determine if the certificate is a server certificate.

   is_unknown_cert
       returns a boolean value to determine if the certificate type is unknown.

   is_user_cert
       returns a boolean value to determine if the certificate is a user certificate.

   issuer_common_name
       returns the issuer common name <https://datatracker.ietf.org/doc/html/rfc5280#section-5.1.2.3> from the
       certificate, such as 'VeriSign Class 3 Public Primary Certification Authority - G4'

   issuer_name
       returns the issuer name <https://datatracker.ietf.org/doc/html/rfc5280#section-5.1.2.3> from the
       certificate, such as 'CN=VeriSign Class 3 Public Primary Certification Authority - G4,OU="(c) 2007
       VeriSign, Inc. - For authorized use only",OU=VeriSign Trust Network,O="VeriSign, Inc.",C=US'

   issuer_organization
       returns the issuer organisation <https://datatracker.ietf.org/doc/html/rfc5280#section-5.1.2.3> from the
       certificate, such as 'VeriSign, Inc.'

   issuer_organization_unit
       returns the issuer organization unit <https://datatracker.ietf.org/doc/html/rfc5280#section-5.1.2.3> from
       the certificate, such as 'VeriSign Trust Network'

   key_usages
       returns a string describing the intended usages of the certificate, such as 'Certificate Signer'

   new
       This method is intended for use exclusively by the Firefox::Marionette module.  You should not need to
       call this method from your code.

   nickname
       returns the nickname field, such as 'Builtin Object Token:VeriSign Class 3 Public Primary Certification
       Authority - G4'

   not_valid_after
       returns the not valid after <https://datatracker.ietf.org/doc/html/rfc5280#section-4.1.2.5> time in
       seconds since the UNIX epoch.

   not_valid_before
       returns the not valid before <https://datatracker.ietf.org/doc/html/rfc5280#section-4.1.2.5> time in
       seconds since the UNIX epoch.

   organization
       returns the organization field, such as 'VeriSign, Inc.'

   organizational_unit
       returns the organization unit field, such as 'VeriSign Trust Network'

   serial_number
       returns the serial number <https://datatracker.ietf.org/doc/html/rfc5280#section-4.1.2.2> of the
       certificate, such as '2F:80:FE:23:8C:0E:22:0F:48:67:12:28:91:87:AC:B3'

   sha1_fingerprint
       returns the sha1Fingerprint field, such as '22:D5:D8:DF:8F:02:31:D1:8D:F7:9D:B7:CF:8A:2D:64:C9:3F:6C:3A'

   sha256_fingerprint
       returns the sha256Fingerprint field, such as
       '69:DD:D7:EA:90:BB:57:C9:3E:13:5D:C8:5E:A6:FC:D5:48:0B:60:32:39:BD:C4:54:FC:75:8B:2A:26:CF:7F:79'

   sha256_subject_public_key_info_digest
       returns the base64 encoded sha256 digest of the subject public key info
       <https://datatracker.ietf.org/doc/html/rfc5280#section-4.1.2.7> field, such as
       'UZJDjsNp1+4M5x9cbbdflB779y5YRBcV6Z6rBMLIrO4='

   subject_name
       returns the name from the subject <https://datatracker.ietf.org/doc/html/rfc5280#section-4.1.2.6> field,
       such as 'CN=VeriSign Class 3 Public Primary Certification Authority - G4,OU="(c) 2007 VeriSign, Inc. -
       For authorized use only",OU=VeriSign Trust Network,O="VeriSign, Inc.",C=US'

   token_name
       returns a string describing the type of certificate, such as 'Builtin Object Token'
