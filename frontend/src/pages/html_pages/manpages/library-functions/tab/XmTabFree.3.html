NAME
       XmTabFree — A convenience function that frees a tab "XmTabFree"


SYNOPSIS
       #include <Xm/Xm.h>
       void XmTabFree(
       XmTab tab);


DESCRIPTION
       XmTabFree frees the memory associated with the specified tab.

       tab       Specifies the tab to be freed.


RELATED
       XmTab(3).
