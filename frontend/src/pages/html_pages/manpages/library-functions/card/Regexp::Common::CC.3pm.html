NAME
       Regexp::Common::CC -- provide patterns for credit card numbers.


SYNOPSIS
           use Regexp::Common qw /CC/;

           while (<>) {
               /^$RE{CC}{Mastercard}$/   and  print "Mastercard card number\n";
           }


DESCRIPTION
       Please consult the manual of Regexp::Common for a general description of the works of this interface.

       Do not use this module directly, but load it via Regexp::Common.

       This module offers patterns for credit card numbers of several major credit card types. Currently, the
       supported cards are: Mastercard, Amex, Diners Club, and Discover.
