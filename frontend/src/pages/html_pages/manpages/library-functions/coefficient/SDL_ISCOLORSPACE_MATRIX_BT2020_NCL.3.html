NAME
       SDL_ISCOLORSPACE_MATRIX_BT2020_NCL - A macro to determine if an SDL_Colorspace
        uses BT2020_NCL matrix coefficients.


HEADER FILE
       Defined in SDL3/SDL_pixels.h


SYNOPSIS
       #include "SDL3/SDL.h"

       #define SDL_ISCOLORSPACE_MATRIX_BT2020_NCL(cspace)   (SDL_COLORSPACEMATRIX(cspace) == SDL_MATRIX_COEFFICIENTS_BT2020_NCL)


RETURN VALUE
       Returns true if BT2020_NCL, false otherwise.


THREAD SAFETY
       It is safe to call this macro from any thread.


AVAILABILITY
       This macro is available since SDL 3.2.0.
