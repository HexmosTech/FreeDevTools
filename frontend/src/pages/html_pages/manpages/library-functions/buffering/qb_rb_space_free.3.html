NAME
       qb_rb_space_free - The amount of free space in the ring buffer.


SYNOPSIS
       #include <qb/qbrb.h>

       ssize_t qb_rb_space_free(
           qb_ringbuffer_t   *rb   /* ringbuffer instance */
       );


PARAMS
       rb ringbuffer instance


DESCRIPTION
NOTE
       Some of this space will be consumed by the chunk headers.
