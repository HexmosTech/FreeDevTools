NAME
       buffer_getc - read one char from buffer


SYNTAX
       #include <libowfat/buffer.h>

       int buffer_getc(buffer* b,char* x);


DESCRIPTION
       buffer_getc(b,x) is similar to buffer_get(b,x,1).
