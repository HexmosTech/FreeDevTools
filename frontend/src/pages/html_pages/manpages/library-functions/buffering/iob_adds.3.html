NAME
       iob_adds - add buffer to I/O batch


SYNTAX
       #include <libowfat/iob.h>

       void iob_adds(io_batch* b,const char* s);


DESCRIPTION
       iob_adds schedules the ASCIIZ C string s to be written with the I/O batch b.

       Use iob_adds if you want the string to be deallocated automatically by iob_reset.
