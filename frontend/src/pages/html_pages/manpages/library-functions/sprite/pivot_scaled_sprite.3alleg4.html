NAME
       pivot_scaled_sprite  -  Rotates and stretches a sprite around a specified point. Allegro game programming
       library.


SYNOPSIS
       #include <allegro.h>

       void pivot_scaled_sprite(BITMAP *bmp, BITMAP *sprite, int x, int y, int cx, int cy,  fixed  angle,  fixed
       scale);


DESCRIPTION
       Like  rotate_scaled_sprite(),  but  aligns  the  point  in  the sprite given by (cx, cy) to (x, y) in the
       bitmap, then rotates and scales around this point.
