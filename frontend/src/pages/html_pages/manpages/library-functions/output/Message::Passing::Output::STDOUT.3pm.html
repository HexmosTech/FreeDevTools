NAME
       Message::Passing::Output::STDOUT - STDOUT output


SYNOPSIS
           message-pass --input STDIN --output STDOUT
           {"foo": "bar"}
           {"foo":"bar"}


DESCRIPTION
       Output messages to STDOUT


METHODS
   consume
       Consumes a message by printing it, followed by \n
