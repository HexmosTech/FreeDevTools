NAME
       Message::Passing::Input::Socket::UDP - UDP input


DESCRIPTION
       An input which gets messages from a UDP network socket using AnyEvent::Handle::UDP.


ATTRIBUTES
   hostname
       The hostname "new" in AnyEvent::Handle::UDP will bind to.

   port
       The port "new" in AnyEvent::Handle::UDP will bind to.
