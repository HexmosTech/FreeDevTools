NAME
       Bio::Roary::Output::GroupsMultifastasNucleotide - Take in a set of GFF files and a groups file and output
       one multifasta file per group with nucleotide sequences.


VERSION
       version 3.13.0


SYNOPSIS
       Take in a set of GFF files and a groups file and output one multifasta file per group with nucleotide
       sequences.
          use Bio::Roary::Output::GroupsMultifastasNucleotide;

          my $obj = Bio::Roary::Output::GroupsMultifastasNucleotide->new(
              group_names      => ['aaa','bbb'],
              analyse_groups  => $analyse_groups
            );
          $obj->create_files();
