NAME
       Parse::DMIDecode::Constants - SMBIOS Constants


SYNOPSIS
        use strict;
        use Parse::DMIDecode::Constants qw(@TYPES %GROUPS);


DESCRIPTION
       This module provides and can export constants relating to the SMBIOS specification and dmidecode
       interface command.


EXPORTS
   @TYPES
   %GROUPS

       Parse::DMIDecode
