NAME
       ibv_query_ece - query ECE options.


SYNOPSIS
              #include <infiniband/verbs.h>

              int ibv_query_ece(struct ibv_qp *qp, struct ibv_ece *ece);


DESCRIPTION
       ibv_query_ece() query ECE options.

       Return to the user current ECE state for the QP.


ARGUMENTS
       qp     The queue pair (QP) associated with the ECE options.

       ## ece Argument
              The ECE values.

              struct ibv_ece {
                  uint32_t vendor_id;
                  uint32_t options;
                  uint32_t comp_mask;
              };

       vendor_id
              Unique  identifier of the provider vendor on the network.  The providers will set IEEE OUI here to
              distinguish itself in non-homogenius network.

       options
              Provider specific attributes which are supported.

       comp_mask
              Bitmask specifying what fields in the structure are valid.


RETURN VALUE
       ibv_query_ece() returns 0 when the call was successful, or the errno value which  indicates  the  failure
       reason.

       EINVAL In one of the following: o The QP is invalid.  o The ECE options are invalid.
