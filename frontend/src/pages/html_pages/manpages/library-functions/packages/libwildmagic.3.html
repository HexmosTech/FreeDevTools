NAME
       libwildmagic — The libwildmagic package ships a number of libraries useful for performing computations in
       the following fields: computer graphics, mathematics, physics, numerical methods, image analysis.


SYNOPSIS
       libwildmagic


DESCRIPTION
       This manual page documents briefly the libwildmagic source package.

       This  manual  page  was  written for the Debian distribution because the original program does not have a
       manual page.

       The libwildmagic source package is used to create the following packages:

          •  libwildmagic5 runtime binary package

          •  libwildmagic5-dbg; runtime binary package (debugging symbols)

          •  libwildmagic-dev development binary package

          •  libwildmagic-common common data binary package

          •  libwildmagic-examples example binary package

       The libwildmagic5 runtime binary package ships the following libraries

          •  libWm5Core.so

          •  libWm5Applications.so

          •  libWm5Graphics.so

          •  libWm5Imagics.so

          •  libWm5Mathematics.so

          •  libWm5Physics.so

       The libwildmagic-dev development binary package ships the following files:

          •  Include files, shipped in /usr/include/libwildmagic

       The libwildmagic-common common data files binary package ships the following files:

          •  Files contained in the Data directory, shipped in /usr/share/libwildmagic/;

       The libwildmagic-examples binary package ships a large set  of  example  programs  that  are  shipped  in
       /usr/lib/libwildmagic.
