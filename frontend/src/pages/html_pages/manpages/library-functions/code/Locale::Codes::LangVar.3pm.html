NAME
       Locale::Codes::LangVar - module for dealing with langvar code sets


SYNOPSIS
          use Locale::Codes::LangVar;

          $name = code2langvar(CODE);
          $code = langvar2code(NAME);

          @codes   = all_langvar_codes();
          @names   = all_langvar_names();


DESCRIPTION
       This module provides access to langvar code sets.

       Please refer to the Locale::Codes::Types document for a description of the code sets available.

       Most of the routines take an optional additional argument which specifies the code set to use. The code
       set can be specified using the name of a code set, or the perl constant specified in the above document.
       If not specified, the default code set will be used.


ROUTINES
       All routines in this module call the appropriate method in the Locale::Codes module, using an object of
       type: langvar Please refer to the documentation of the Locale::Codes module for details about each
       function.

       The following functions are exported automatically:

       code2langvar(CODE [,CODESET] [,'retired'])
           See code2name in Locale::Codes

       code2langvars(CODE [,CODESET])
           See code2names in Locale::Codes

       langvar2code(NAME [,CODESET] [,'retired'])
           See name2code in Locale::Codes

       langvar_code2code(CODE ,CODESET ,CODESET2)
           See code2code in Locale::Codes

       all_langvar_codes([CODESET] [,'retired'])
           See all_codes in Locale::Codes

       all_langvar_names([CODESET] [,'retired'])
           See all_names in Locale::Codes

       The following functions are not exported and must be called fully qualified with the package name:

       Locale::Codes::Langvar::show_errors(FLAG)
           By  default, invalid input will produce empty results, but no errors.  By passing in a non-zero value
           of FLAG, errors will be produced.

           See show_errors in Locale::Codes but note that the default for the non-OO modules are to NOT  produce
           errors.

       Locale::Codes::Langvar::rename_langvar(CODE ,NEW_NAME [,CODESET])
           See rename_code in Locale::Codes

       Locale::Codes::Langvar::add_langvar(CODE ,NAME [,CODESET])
           See add_code in Locale::Codes

       Locale::Codes::Langvar::delete_langvar(CODE [,CODESET])
           See delete_code in Locale::Codes

       Locale::Codes::Langvar::add_langvar_alias(NAME ,NEW_NAME)
           See add_alias in Locale::Codes

       Locale::Codes::Langvar::delete_langvar_alias(NAME)
           See delete_alias in Locale::Codes

       Locale::Codes::Langvar::rename_langvar_code(CODE ,NEW_CODE [,CODESET])
           See replace_code in Locale::Codes

       Locale::Codes::Langvar::add_langvar_code_alias(CODE ,NEW_CODE [,CODESET])
           See add_code_alias in Locale::Codes

       Locale::Codes::Langvar::delete_langvar_code_alias(CODE [,CODESET])
           See delete_code_alias in Locale::Codes
