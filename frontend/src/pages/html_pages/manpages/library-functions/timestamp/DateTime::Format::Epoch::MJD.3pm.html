NAME
       DateTime::Format::Epoch::MJD - Convert DateTimes to/from Modified Julian Days


SYNOPSIS
         use DateTime::Format::Epoch::MJD;

         my $dt = DateTime::Format::Epoch::MJD->parse_datetime( 53244 );
          # 2004-08-27T00:00:00

         DateTime::Format::Epoch::MJD->format_datetime($dt);
          # 53244

         my $formatter = DateTime::Format::Epoch::MJD->new();
         my $dt2 = $formatter->parse_datetime( 53244 );
          # 2004-08-27T00:00:00
         $formatter->format_datetime($dt2);
          # 53244


DESCRIPTION
       This module can convert a DateTime object (or any object that can be converted to a DateTime object) to
       the Modified Julian Day number. See DateTime::Format::Epoch::JD for a description.


METHODS
       Most of the methods are the same as those in DateTime::Format::Epoch.  The only difference is the
       constructor.
