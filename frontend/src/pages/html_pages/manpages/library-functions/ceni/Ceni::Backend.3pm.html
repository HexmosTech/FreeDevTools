NAME
       Ceni::Backend - Perl extension for Ceni


SYNOPSIS
         use Ceni::Backend;

         my $ceni = Ceni::Backend->new( \%options );


DESCRIPTION
       This is the backend for Ceni(8), it is not provided for use by scripts not provided by the ceni package,
       therefore remains undocuemnted.
