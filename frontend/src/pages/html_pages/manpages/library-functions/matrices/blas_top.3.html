Detailed Description
       BLAS are defined by three papers: Basic linear algebra subprograms for {FORTRAN} usage, Lawson et al,
       1979. An extended set of {FORTRAN} basic linear algebra subprograms, Dongarra et al, 1988. A set of level
       3 basic linear algebra subprograms, Dongarra et al, 1990. Some BLAS-like routines (e.g., csymv, crot,
       csum1, icmax1) exist in LAPACK rather than the classic BLAS. These were extended by the Extra Precision
       BLAS (XBLAS, not documented here) https://www.netlib.org/xblas/
