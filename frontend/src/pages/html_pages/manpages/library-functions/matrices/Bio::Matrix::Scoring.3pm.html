NAME
       Bio::Matrix::Scoring - Object which can hold scoring matrix information


SYNOPSIS
         use Bio::Matrix::Scoring;


DESCRIPTION
       An object which can handle AA or NT scoring matrix information.  Some transformation properties are
       available too.
