Detailed Description
       The SymRep class provides an n dimensional matrix representation of a symmetry operation, such as a
       rotation or reflection.

       The trace of a SymRep can be used as the character for that symmetry operation. d is hardwired to 5x5
       since the H irrep in Ih is 5 dimensional.
