NAME
       Tk::DragDrop::SunConst - Constants for Sun's Drag&Drop protocol


DESCRIPTION
       This module defines symbolic name subs for the numeric constants that make up Sun's Drag&Drop protocol.
       They are in this module with Exporter as they are shared between the two halves (Dropper and Receiver) of
       the protocol.
