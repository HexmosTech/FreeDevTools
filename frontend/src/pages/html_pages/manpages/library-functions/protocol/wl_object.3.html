NAME
       wl_object - A protocol object.


SYNOPSIS
       #include <wayland-server.h>

   Data Fields
       const struct wl_interface * interface

       uint32_t id


Detailed Description
       A protocol object.

       A wl_object is an opaque struct identifying the protocol object underlying a wl_proxy or wl_resource.

       Note
           Functions accessing a wl_object are not normally used by client code. Clients should normally use the
           higher level interface generated by the scanner to interact with compositor objects.
