SYNOPSIS
          const mongoc_ssl_opt_t *
          mongoc_ssl_opt_get_default (void);

       NOTE:
          Though some API names include the term "ssl", the C driver only support TLS protocols, which supersede
          SSL.


RETURNS
       Returns the default TLS options for the process. This should not be modified or freed.
