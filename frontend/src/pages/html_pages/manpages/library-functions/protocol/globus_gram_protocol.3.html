Detailed Description
       GRAM Protocol.

       The Globus GRAM Protocol Library implements the GRAM protocol. It is used by the GRAM Client and GRAM Job
       Manager. It provides the constants used by in the sending and receiving of GRAM messages. It also
       provides functions to encode GRAM requests and replies, and to send and receive the GRAM queries.
