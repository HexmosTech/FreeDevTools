       val varify_constructors : Ast_helper.str list -> Parsetree.core_type -> Parsetree.core_type

       varify_constructors  newtypes  te  is type expression te , of which any of nullary type constructor tc is
       replaced  by  type  variable  of  the  same  name,  if  tc  's  name  appears  in   newtypes   .    Raise
       Syntaxerr.Variable_in_scope if any type variable inside te appears in newtypes .
