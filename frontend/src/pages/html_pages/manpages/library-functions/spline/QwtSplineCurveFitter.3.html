Member Enumeration Documentation
   enum QwtSplineCurveFitter::FitMode
       Spline type The default setting is Auto

       See also:
           setFitMode(), FitMode()

       Enumerator

       Auto   Use  the  default  spline  algorithm  for  polygons  with  increasing  x values ( p[i-1] < p[i] ),
              otherwise use a parametric spline algorithm.

       Spline Use a default spline algorithm.

       ParametricSpline
              Use a parametric spline algorithm.
