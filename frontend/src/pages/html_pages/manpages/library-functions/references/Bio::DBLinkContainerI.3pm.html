NAME
       Bio::DBLinkContainerI - Abstract interface for any object wanting to use
                               database cross references


SYNOPSIS
           # get an objects containing database cross reference

               foreach $obj ( @objs ) {
                       if( $obj->isa('Bio::DBLinkContainerI') ) {
                               foreach $dblink ( $obj->each_DBLink() ) {
                                       # do stuff
                               }
                       }
               }


DESCRIPTION
       This interface defines the functions one can expect for any object wanting to use database cross-
       references. This class does not actually provide any implementation, it just provides the definitions of
       what methods one can call.

       The database cross-references are implemented as Bio::Annotation::DBLink objects.
