Detailed Description

       Base class that supports tracking of iterators that reference a sequence.

       The _Safe_sequence_base class provides basic support for tracking iterators into a sequence. Sequences
       that track iterators must derived from _Safe_sequence_base publicly, so that safe iterators (which
       inherit _Safe_iterator_base) can attach to them. This class contains two linked lists of iterators, one
       for constant iterators and one for mutable iterators, and a version number that allows very fast
       invalidation of all iterators that reference the container.

       This class must ensure that no operation on it may throw an exception, otherwise safe sequences may fail
       to provide the exception-safety guarantees required by the C++ standard.
