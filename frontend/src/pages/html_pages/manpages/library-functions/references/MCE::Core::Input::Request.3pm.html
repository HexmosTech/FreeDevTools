NAME
       MCE::Core::Input::Request - Array reference and Glob reference input reader


VERSION
       This document describes MCE::Core::Input::Request version 1.901


DESCRIPTION
       This package provides the request chunk method used internally by the worker process. Distribution
       follows a bank-queuing model.

       There is no public API.


SEE ALSO
       The syntax for the "input_data" option is described in MCE::Core.
