NAME
       Net::LDAP::Reference - search reference


SYNOPSIS
         use Net::LDAP;

         $ldap->search( @search_args, callback => \&process);

         sub process {
           my $mesg = shift;
           my $obj = shift;
           if (!$obj) {
             # Search complete
           }
           elsif ($obj->isa('Net::LDAP::Reference')) {
             my $ref;

             foreach $ref ($obj->references) {
               # process ref
             }
           }
           else {
             # Process Net::LDAP::Entry
           }
         }


DESCRIPTION
       The Net::LDAP::Reference object represents a reference (sometimes called a "referral") returned by the
       directory from a search.


METHODS
       references
           Returns a list of references from the server.

       Net::LDAP, Net::LDAP::Search
