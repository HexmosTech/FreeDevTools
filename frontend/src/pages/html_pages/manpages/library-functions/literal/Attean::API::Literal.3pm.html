NAME
       Attean::API::Literal - Role representing RDF Literal terms


VERSION
       This document describes Attean::API::Literal version 0.035


SYNOPSIS
         use v5.14;
         use Attean;


DESCRIPTION
       This is a Moo role that Literal RDF terms consume.


ROLES
       This role consumes the Attean::API::Term and Attean::API::BlankOrIRI roles.


ATTRIBUTES
       The following attributes exist:

       "language"
           A language tag string.

       "datatype"
           An Attean::API::IRI datatype.


METHODS
       This role provides default implementations of the following methods:

       "ebv"
           Returns the boolean value of this term.

       compare( $literal )
           Returns  -1, 0, or 1 if $literal is less than, equal to, or greater than the referent based on SPARQL
           sorting order.
