NAME
       mtext_character -


SYNOPSIS
       int mtext_character (MText * mt, int from, int to, int c)


DESCRIPTION
       Search a character in an M-text. The mtext_character() function searches M-text mt for character c. If
       from is less than to, the search begins at position from and goes forward but does not exceed (to - 1).
       Otherwise, the search begins at position (from - 1) and goes backward but does not exceed to. An invalid
       position specification is regarded as both from and to being 0.


RETURN VALUE
           If c is found, mtext_character() returns the position of its first occurrence. Otherwise it returns
           -1 without changing the external variable merror_code. If an error is detected, it returns -1 and
           assigns an error code to the external variable merror_code.
