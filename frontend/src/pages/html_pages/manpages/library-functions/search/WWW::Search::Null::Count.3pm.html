NAME
       WWW::Search::Null::Count - class for testing WWW::Search clients


SYNOPSIS
         use WWW::Search;
         my $iCount = 4;
         my $oSearch = new WWW::Search('Null::Count',
                                       '_null_count' => $iCount,
                                      );
         $oSearch->native_query('Makes no difference what you search for...');
         my @aoResults = $oSearch->results;
         # ...You get $iCount results.


DESCRIPTION
       This class is a specialization of WWW::Search that returns some hits, but no error message.  The number
       of hits returned can be controlled by adding a '_null_count' hash entry onto the call to
       WWW::Search::new().  The default is 5.

       This module might be useful for testing a client program without actually being connected to any
       particular search engine.
