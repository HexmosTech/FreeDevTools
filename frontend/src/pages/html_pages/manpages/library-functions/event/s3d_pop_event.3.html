NAME
       s3d_pop_event - pop event from stack


SYNOPSIS
       #include <s3d.h>

       struct s3d_evt *s3d_pop_event(void);


DESCRIPTION
       Pops the latest event from the stack. Don't forget to free() both the event and its buffer! Returns a
       pointer to struct s3d_evt.


AUTHOR
       Simon Wunderlich
           Author of s3d
