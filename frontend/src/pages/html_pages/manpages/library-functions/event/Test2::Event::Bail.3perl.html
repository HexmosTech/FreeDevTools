NAME
       Test2::Event::Bail - Bailout!


DESCRIPTION
       The bailout event is generated when things go horribly wrong and you need to halt all testing in the
       current file.


SYNOPSIS
           use Test2::API qw/context/;
           use Test2::Event::Bail;

           my $ctx = context();
           my $event = $ctx->bail('Stuff is broken');


METHODS
       Inherits from Test2::Event. Also defines:

       $reason = $e->reason
           The reason for the bailout.
