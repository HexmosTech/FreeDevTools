SYNOPSIS
          const mongoc_host_list_t *
          mongoc_apm_server_heartbeat_failed_get_host (
             const mongoc_apm_server_heartbeat_failed_t *event);

       Returns  this event's host. This mongoc_host_list_t is not part of a linked list, it is solely the server
       for this event. The data is only valid in the scope of the callback that receives this event; copy it  if
       it will be accessed after the callback returns.
