NAME
       Markdent::Event::CodeBlock - An event for a code block


VERSION
       version 0.40


DESCRIPTION
       This class represents a block of code


ATTRIBUTES
       This class has the following attributes:

   code
       The code in the block, including newlines and additional leading space, etc.

   language
       An optional language associated with the block, if one was specified. You can use the has_language()
       method to see if one is set.


ROLES
       This class does the Markdent::Role::Event role.
