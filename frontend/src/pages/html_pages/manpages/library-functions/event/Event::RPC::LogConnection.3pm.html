NAME
       Event::RPC::LogConnection - Represents a logging connection


SYNOPSIS
         # Internal module. No documented public interface.


DESCRIPTION
       Objects of this class are created by Event::RPC server if a client connects to the logging port of the
       server. It's an internal module and has no public interface.
