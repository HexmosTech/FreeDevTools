NAME
       Sisimai::Reason::MailerError - Bounce reason is "mailererror" or not.


SYNOPSIS
           use Sisimai::Reason::MailerError;
           print Sisimai::Reason::MailerError->match('X-Unix; 255');   # 1


DESCRIPTION
       "Sisimai::Reason::MailerError" checks the bounce reason is "mailererror" or not. This class is called
       only "Sisimai::Reason" class.

       This is the error that the mailer program has not exited successfully or exited unexpectedly on the
       destination mail server.

           X-Actual-Recipient: X-Unix; |/home/kijitora/mail/catch.php
           Diagnostic-Code: X-Unix; 255


CLASS METHODS
   "text()"
       text() method returns the fixed string "mailererror".

           print Sisimai::Reason::MailerError->text;  # mailererror

   "match(string)"
       match() method returns 1 if the argument matched with patterns defined in this class.

           print Sisimai::Reason::MailerError->match('X-Unix; 255');   # 1

   "true(Sisimai::Fact)"
       true() method returns 1 if the bounce reason is "mailererror". The argument must be "Sisimai::Fact"
       object and this method is called only from "Sisimai::Reason" class.
