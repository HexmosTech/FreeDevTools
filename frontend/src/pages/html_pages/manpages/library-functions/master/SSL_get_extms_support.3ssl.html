NAME
       SSL_get_extms_support - extended master secret support


SYNOPSIS
        #include <openssl/ssl.h>

        int SSL_get_extms_support(SSL *ssl);


DESCRIPTION
       SSL_get_extms_support() indicates whether the current session used extended master secret.

       This function is implemented as a macro.


RETURN VALUES
       SSL_get_extms_support() returns 1 if the current session used extended master secret, 0 if it did not and
       -1 if a handshake is currently in progress i.e. it is not possible to determine if extended master secret
       was used.
