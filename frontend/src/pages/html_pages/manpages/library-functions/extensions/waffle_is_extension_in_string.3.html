NAME
       waffle_is_extension_in_string - Check if a name appears in an OpenGL-style extension string


SYNOPSIS
       #include <waffle.h>

       bool waffle_is_extension_in_string(const char * extension_string, const char * extension_name);


DESCRIPTION
       waffle_is_extension_in_string()
           Check if an extension name appears in an OpenGL-style extension string.

           The extension_string must be in the format returned by glGetString(GL_EXTENSIONS). Note that this
           function is not restricted to the OpenGL extension string; it can also be used on the WGL, GLX and
           EGL extension strings.

           This function can be called before waffle has been successfully initialized with waffle_init(3).


ERRORS
       This function sets the error code to WAFFLE_NO_ERROR.
