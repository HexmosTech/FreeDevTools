NAME
       revtwoway - Reverse strings sample two-way processor extension


SYNOPSIS
       @load "revtwoway"

       BEGIN {
           cmd = "/magic/mirror"
           print "hello, world" |& cmd
           cmd |& getline result
           print result
           close(cmd)
       }


DESCRIPTION
       The revtwoway extension adds a simple two-way processor that reverses the characters in each line sent to
       it  for  reading back by the AWK program.  It's main purpose is to show how to write a two-way extension,
       although it may also be mildly amusing.
