NAME
       ExtUtils::Constant::XS - generate C code for XS modules' constants.


SYNOPSIS
           require ExtUtils::Constant::XS;


DESCRIPTION
       ExtUtils::Constant::XS overrides ExtUtils::Constant::Base to generate C code for XS modules' constants.


BUGS
