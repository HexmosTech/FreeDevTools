NAME
       Test2::AsyncSubtest::Hub - Hub used by async subtests.


DESCRIPTION
       This is a subclass of Test2::Hub::Subtest used for async subtests.


SYNOPSIS
       You should not use this directly.


METHODS
       $ast = $hub->ast
           Get the Test2::AsyncSubtest object to which this hub is bound.
