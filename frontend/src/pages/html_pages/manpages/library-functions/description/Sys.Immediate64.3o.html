       Module Immediate64
        : sig end

       This  module allows to define a type t with the immediate64 attribute. This attribute means that the type
       is immediate on 64 bit architectures. On other architectures, it might or might not be immediate.

       module type Non_immediate = sig end

       module type Immediate = sig end
