NAME
       Mail::Box-Index - list of all extended documentation


DESCRIPTION
       All methods defined by MailBox are documented, but you still need to get a grip on the large picture: the
       relations between the modules and the methods.  These larger explanations are included in the DETAILS of
       the manual pages, and listed here below.  Of course, each page has a DESCRIPTION section as well, which
       tells some bare necessities about the module.

       Only the HTML version of this manual page will list the sections.
