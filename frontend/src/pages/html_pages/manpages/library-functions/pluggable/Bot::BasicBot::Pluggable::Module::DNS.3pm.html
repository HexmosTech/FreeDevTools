NAME
       Bot::BasicBot::Pluggable::Module::DNS - DNS lookups for hostnames or IP addresses


VERSION
       version 1.30


IRC USAGE
       dns <ip address>
           Returns the hostname of that IP address

       nslookup <hostname>
           Returns the IP address of the hostname.
