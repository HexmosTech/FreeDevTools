NAME
       Test::BDD::Cucumber::Loader - Simplify loading of Step Definition and feature files


VERSION
       version 0.87


DESCRIPTION
       Makes loading Step Definition files and Feature files a breeze...


METHODS
   load
       Accepts a path, and returns a Test::BDD::Cucumber::Executor object with the Step Definition files loaded,
       and a list of Test::BDD::Cucumber::Model::Feature objects.

   load_steps
       Accepts an Test::BDD::Cucumber::Executor object and a string representing either a step file, or a
       directory containing zero or more "*_steps.pl" files, and loads the steps in to the executor; if you've
       used "load" we'll have already scanned the feature directory for "*_steps.pl" files.
