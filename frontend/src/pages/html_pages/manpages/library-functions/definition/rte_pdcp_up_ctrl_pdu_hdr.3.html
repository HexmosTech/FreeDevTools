   uint8_t bitmap[]
       6.3.10 Bitmap

       Length: Variable. The length of the bitmap field can be 0.

       This field indicates which SDUs are missing and which SDUs are correctly received in the receiving PDCP
       entity. The bit position of Nth bit in the Bitmap is N, i.e., the bit position of the first bit in the
       Bitmap is 1.
