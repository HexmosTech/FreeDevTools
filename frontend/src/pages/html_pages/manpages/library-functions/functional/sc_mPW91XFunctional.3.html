Detailed Description
       Implements a modified 1991 Perdew-Wang exchange functional.

       C. Adamo and V. Barone, J. Chem. Phys., 108(2), pp. 664-674, 1998.


Constructor & Destructor Documentation
   sc::mPW91XFunctional::mPW91XFunctional (Func variant)
       Construct an mPW form exchange functional using the given functional variant. The variant can be B88,
       PW91, or mPW91.

   sc::mPW91XFunctional::mPW91XFunctional (const Ref< KeyVal > &)
       Construct an mPW form exchange functional. The following keywords are recognized:

       constants
           This  can  be  B88 to give the Becke88 exchange functional; PW91, to give results similar to the PW91
           exchange functional; or mPW91, to give the new functional developed by Adamo and Barone.

       b

       beta

       c

       d

       x_d_coef
           The coefficient of $x^d$, where $x$ is the reduced gradient.
