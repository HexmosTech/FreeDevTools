NAME
       SDL_GetWindowPixelDensity - Get the pixel density of a window.


SYNOPSIS
       #include <SDL3/SDL_video.h>

       float SDL_GetWindowPixelDensity(SDL_Window *window);


DESCRIPTION
       This  is  a ratio of pixel size to window size. For example, if the window is 1920x1080 and it has a high
       density back buffer of 3840x2160 pixels, it would have a pixel density of 2.0.


FUNCTION PARAMETERS
       window the window to query.


RETURN VALUE
       for more information.


THREAD SAFETY
       This function should only be called on the main thread.


AVAILABILITY
       This function is available since SDL 3.2.0.
