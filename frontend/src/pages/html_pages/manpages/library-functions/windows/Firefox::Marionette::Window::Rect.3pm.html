NAME
       Firefox::Marionette::Window::Rect - Represents the browser window's shape and size


VERSION
       Version 1.22


SYNOPSIS
           use Firefox::Marionette();
           use v5.10;

           my $firefox = Firefox::Marionette->new()->go('https://metacpan.org/');
           my $rect = $firefox->rect();
           say "Current height of the window " . $window->height();


DESCRIPTION
       This module handles the representation of the browser window


SUBROUTINES/METHODS
   new
       accepts a hash as a parameter.  Allowed keys are below;

       •   pos_x - the X position of the window.  This function will return undef for older Firefoxen.

       •   pos_y - the Y position of the window.  This function will return undef for older Firefoxen.

       •   height - the height of the window

       •   width - the width of the window

       •   wstate  -  the  state  of the window as a scalar string.  This function will return undef for Firefox
           versions less than 57

       This method returns a new rect object.

   pos_x
       returns the X position of the window

   pos_y
       returns the Y position of the window

   height
       returns the height of the window

   width
       returns the width of the window

   wstate
       returns a scalar representing the state of the window. For example 'maximized'.
