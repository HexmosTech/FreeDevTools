Detailed Description
       Produces a dump of a buffer in a hexdump way with its code Ascii translation and relative buffer address.

       For instance: 0000000 - 77 98 21 49 0e 00 05 00 40 1c 01 1c 2f 00 00 00 w.!I......./...
