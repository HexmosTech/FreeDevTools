NAME
       RDF::Query::Parser::RDQL - An RDQL parser for RDF::Query


VERSION
       This document describes RDF::Query::Parser::RDQL version 2.919.


METHODS
       "new ( $query_object ) "
           Returns a new RDF::Query object.

       "parse ( $query ) "
           Parses the supplied RDQL query string, returning a parse tree.
