NAME
       Bio::Matrix::PSM::PsmHeader - PSM mast parser implementation


SYNOPSIS
         # See Bio::Matrix::PSM::IO for detailed documentation on how to use
         # PSM parsers


DESCRIPTION
       Parser for mast. This driver unlike meme or transfac for example is dedicated more to PSM sequence
       matches
