NAME
       Syntax::Highlight::Engine::Kate::Common_Lisp - a Plugin for Common Lisp syntax highlighting


SYNOPSIS
        require Syntax::Highlight::Engine::Kate::Common_Lisp;
        my $sh = new Syntax::Highlight::Engine::Kate::Common_Lisp([
        ]);


DESCRIPTION
       Syntax::Highlight::Engine::Kate::Common_Lisp is a  plugin module that provides syntax highlighting for
       Common Lisp to the Syntax::Haghlight::Engine::Kate highlighting engine.

       This code is generated from the syntax definition files used by the Kate project.  It works quite fine,
       but can use refinement and optimization.

       It inherits Syntax::Higlight::Engine::Kate::Template. See also there.
