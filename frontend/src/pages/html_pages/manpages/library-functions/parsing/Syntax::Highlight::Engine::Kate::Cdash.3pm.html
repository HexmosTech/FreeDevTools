NAME
       Syntax::Highlight::Engine::Kate::Cdash - a Plugin for C# syntax highlighting


SYNOPSIS
        require Syntax::Highlight::Engine::Kate::Cdash;
        my $sh = new Syntax::Highlight::Engine::Kate::Cdash([
        ]);


DESCRIPTION
       Syntax::Highlight::Engine::Kate::Cdash is a  plugin module that provides syntax highlighting for C# to
       the Syntax::Haghlight::Engine::Kate highlighting engine.

       This code is generated from the syntax definition files used by the Kate project.  It works quite fine,
       but can use refinement and optimization.

       It inherits Syntax::Higlight::Engine::Kate::Template. See also there.
