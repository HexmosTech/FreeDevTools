NAME
       AtteanX::Parser::SPARQLXML - SPARQL XML Parser


VERSION
       This document describes AtteanX::Parser::SPARQLXML version 0.035


SYNOPSIS
        use Attean;
        my $parser = Attean->get_parser('SPARQLXML')->new();
        $parser->parse_cb_from_io( $fh );


DESCRIPTION
       ...


METHODS
       "canonical_media_type"
           Returns the canonical media type for SPARQL XML: application/sparql-results+xml.

       "media_types"
           Returns   a   list   of   media   types   that   may   be   parsed   with   the  SPARQL  XML  parser:
           application/sparql-results+xml.

       "file_extensions"
           Returns a list of file extensions that may be parsed with the parser.

       parse_cb_from_io( $fh )
           Calls the "$parser->handler" function once for each  Attean::API::Binding  object  that  result  from
           parsing the data read from the IO::Handle object $fh.

       parse_cb_from_bytes( $data )
           Calls  the  "$parser->handler"  function  once  for each Attean::API::Binding object that result from
           parsing the data read from the UTF-8 encoded byte string $data.
