NAME
       Embperl::Syntax::Perl - define Perl syntax for Embperl


SYNOPSIS
       Execute ({inputfile => 'code.pl', syntax => 'Perl'}) ;


DESCRIPTION
       This syntax cause Embperl to interpret the whole file as Perl script without any markup.
