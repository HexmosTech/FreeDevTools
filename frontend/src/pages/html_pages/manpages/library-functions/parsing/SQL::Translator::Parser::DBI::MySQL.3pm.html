NAME
       SQL::Translator::Parser::DBI::MySQL - parser for DBD::mysql


SYNOPSIS
       This module will be invoked automatically by SQL::Translator::Parser::DBI, so there is no need to use it
       directly.


DESCRIPTION
       Uses SQL calls to query database directly for schema rather than parsing a create file.  Should be much
       faster for larger schemas.
