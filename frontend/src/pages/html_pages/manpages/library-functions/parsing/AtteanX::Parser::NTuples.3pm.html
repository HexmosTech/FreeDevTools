NAME
       AtteanX::Parser::NTuples - Shared functionality for N-Triples and N-Quads parsers


VERSION
       This document describes AtteanX::Parser::NTuples version 0.035


SYNOPSIS
        use Attean;


DESCRIPTION
       This module provides a base class for RDF formats N-Triples and N-Quads.


ATTRIBUTES
       "blank_nodes"
       "has_blank_nodes_map"


METHODS
       parse_term_from_bytes( $bytes )
           Parses the given $bytes and returns a corresponding Attean::API::Term object.

       parse_iter_from_bytes( $data )
           Returns  an  iterator of Attean::API::Binding objects that result from parsing the data read from the
           UTF-8 encoded byte string $data.

       parse_iter_from_io( $fh )
           Returns an iterator of Attean::API::Binding objects that result from parsing the data read  from  the
           IO::Handle object $fh.
