NAME
       Syntax::Highlight::Engine::Kate::REXX - a Plugin for REXX syntax highlighting


SYNOPSIS
        require Syntax::Highlight::Engine::Kate::REXX;
        my $sh = new Syntax::Highlight::Engine::Kate::REXX([
        ]);


DESCRIPTION
       Syntax::Highlight::Engine::Kate::REXX is a  plugin module that provides syntax highlighting for REXX to
       the Syntax::Haghlight::Engine::Kate highlighting engine.

       This code is generated from the syntax definition files used by the Kate project.  It works quite fine,
       but can use refinement and optimization.

       It inherits Syntax::Higlight::Engine::Kate::Template. See also there.
