NAME
       SQL::Translator::Parser::DB2 - parser for DB2


SYNOPSIS
         use SQL::Translator;
         use SQL::Translator::Parser::DB2;

         my $translator = SQL::Translator->new;
         $translator->parser("SQL::Translator::Parser::DB2");


DESCRIPTION
       This is a grammar for parsing CREATE statements for DB2
