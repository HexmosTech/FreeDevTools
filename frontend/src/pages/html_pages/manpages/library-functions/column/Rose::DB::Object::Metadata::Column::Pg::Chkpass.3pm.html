NAME
       Rose::DB::Object::Metadata::Column::Pg::Chkpass - PostgreSQL CHKPASS column metadata.


SYNOPSIS
         use Rose::DB::Object::Metadata::Column::Pg::Chkpass;

         $col = Rose::DB::Object::Metadata::Column::Pg::Chkpass->new(...);
         $col->make_methods(...);
         ...


DESCRIPTION
       Objects of this class store and manipulate metadata for CHKPASS columns in a PostgreSQL database.  Column
       metadata objects store information about columns (data type, size, etc.) and are responsible for creating
       object methods that manipulate column values.  See the Rose::DB::Object::MakeMethods::Pg for more
       information on PostgreSQL's CHKPASS data type.

       This class inherits from Rose::DB::Object::Metadata::Column. Inherited methods that are not overridden
       will not be documented a second time here.  See the Rose::DB::Object::Metadata::Column documentation for
       more information.


METHOD MAP
       "get_set"
           Rose::DB::Object::MakeMethods::Pg, chkpass, ...

       "get"
           Rose::DB::Object::MakeMethods::Pg, chkpass, ...

       "get_set"
           Rose::DB::Object::MakeMethods::Pg, chkpass, ...

       See the Rose::DB::Object::Metadata::Column documentation for an explanation of this method map.


OBJECT METHODS
       cmp_suffix [STRING]
           Get or set the suffix used to form the name of the comparison method.   See the documentation for the
           "chkpass" method type in Rose::DB::Object::MakeMethods::Pg for more information.

       encrypted_suffix [STRING]
           Get  or  set the suffix used to form the name of the accessor method for the encrypted version of the
           column    value.      See    the    documentation    for    the    "chkpass"    method    type     in
           Rose::DB::Object::MakeMethods::Pg for more information.

       type
           Returns "chkpass".
