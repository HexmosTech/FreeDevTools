NAME
       File::KDBX::KDF::Argon2 - The Argon2 family of key derivation functions


VERSION
       version 0.906


DESCRIPTION
       An Argon2 key derivation function. This is a File::KDBX::KDF subclass.

       This KDF allows for excellent resistance to ASIC password cracking. It's a solid choice but doesn't have
       the track record of File::KDBX::KDF::AES and requires using the KDBX4+ file format.


ATTRIBUTES
   salt
   parallelism
   memory
   iterations
   version
   secret
   assocdata
       Get various KDF parameters.

       "version", "secret" and "assocdata" are currently unused.
