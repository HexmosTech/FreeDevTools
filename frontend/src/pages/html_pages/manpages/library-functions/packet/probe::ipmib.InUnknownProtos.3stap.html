NAME
       probe::ipmib.InUnknownProtos - Count arriving packets with an unbound proto


SYNOPSIS
       ipmib.InUnknownProtos


VALUES
       op
           value to be added to the counter (default value of 1)

       skb
           pointer to the struct sk_buff being acted on


DESCRIPTION
       The packet pointed to by skb is filtered by the function ipmib_filter_key. If the packet passes the
       filter is is counted in the global InUnknownProtos (equivalent to SNMP's MIB IPSTATS_MIB_INUNKNOWNPROTOS)
