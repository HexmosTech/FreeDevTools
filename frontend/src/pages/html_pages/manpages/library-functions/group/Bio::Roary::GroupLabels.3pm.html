NAME
       Bio::Roary::GroupLabels - Add labels to the groups


VERSION
       version 3.13.0


SYNOPSIS
       Add labels to the groups
          use Bio::Roary::GroupLabels;

          my $obj = Bio::Roary::GroupLabels->new(
            groups_filename   => 'abc.groups',
            output_filename => 'output.groups'
          );
          $obj->add_labels;
