NAME
       Autodia::Handler::Mason - Allows Autodia to parse HTML::Mason files


SYNOPSIS
       See Autodia and HTML::Mason.  Use -p to specify the comp_root and -i fetch one or more components, f.e.
       ./autodia -l Mason -p 'examples/mason' -i 'index.html login.html' If you need to allow globals, f.e. $c
       and $l, add -G '$c $l' to the command line


DESCRIPTION
       Autodia::Handler using introspection provided by HTML::Mason to visualize all components used by a
       request.


API
    _initialise creates the HTML::Mason::Interp instance used for introspection.
    _parse_file walks through the request and initiates the recursion.
    _process_component adds a component to the diagram. This is done recursively for the parent and each called
       component.
    helper method to convert the declared_args of components to attributes
    helper method to convert the declared_args of methods and subcomponents to Params
