OBJECT DOCUMENTATION
       Mason::Interp
           Mason::Interp is the central Mason object, returned from "Mason->new". It is responsible for creating
           new requests, compiling components, and maintaining the cache of loaded components.

       Mason::Request
           Mason::Request  represents  a  single  request  for  a  page,  and is the access point for most Mason
           features not provided by syntactic tags.

       Mason::Component
           Mason::Component is the base class from which all generated component classes  inherit,  directly  or
           indirectly.
