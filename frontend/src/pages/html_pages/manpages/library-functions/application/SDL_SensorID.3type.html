NAME
       SDL_SensorID  - This is a unique ID for a sensor for the time it is connected to the system, and is never
       reused for the lifetime of the application.


HEADER FILE
       Defined in SDL3/SDL_sensor.h


SYNOPSIS
       #include "SDL3/SDL.h"

       typedef Uint32 SDL_SensorID;


DESCRIPTION
       The value 0 is an invalid ID.


AVAILABILITY
       This datatype is available since SDL 3.2.0.
