Detailed Description
       SpatialMOPairIter_eq gives the ordering of same-spin and different-spin orbital pairs if both orbitals of
       the pairs are from the same space.

       It iterates over all i >= j combinations (total of (ni_+1)*(ni_+2)/2 pairs).
