NAME
         GO::Handlers::owl     - writes OWL


SYNOPSIS
         use GO::Parser;
         my $p = GO::Parser->new({handler=>'owl'});
         $p->parse("function.ontology");


DESCRIPTION
       Consumes an OBO-XML event stream and generates OWL XML

       See the file

         go-dev/doc/mapping-obo-to-owl.txt

       For more details


COMMAND LINE
       go2fmt.pl -w owl function.obo
