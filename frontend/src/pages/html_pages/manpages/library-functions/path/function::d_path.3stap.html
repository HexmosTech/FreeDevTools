NAME
       function::d_path - get the full nameidata path


SYNOPSIS
           d_path:string(nd:long)


ARGUMENTS
       nd
           Pointer to nameidata.


DESCRIPTION
       Returns the full dirent name (full path to the root), like the kernel d_path function.
