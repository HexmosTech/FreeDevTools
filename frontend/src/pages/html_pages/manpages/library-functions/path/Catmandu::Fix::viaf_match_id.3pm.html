NAME
       Catmandu::Fix::viaf_match_id - Fetch the RDF representation of a person based on his/her VIAF ID.


SYNOPSIS
         viaf_match_id(path)


DESCRIPTION
       Perform a direct match between a VIAF ID and a mainHeadingEl and the local.personalNames of a Person in
       VIAF.

   PARAMETERS
       Required parameters

       "path"
           Path to the VIAF ID.
