NAME
       modf - separate a value into its integer and fractional components


DECLARATION

       genType modf(genType x, out genType i);

       genDType modf(genDType x, out genDType i);


PARAMETERS
       x
           Specify the value to separate.

       out i
           A variable that receives the integer part of the argument.


DESCRIPTION
       modf separates a floating point value x into its integer and fractional parts. The fractional part of the
       number is returned from the function and the integer part (as a floating point quantity) is returned in
       the output parameter i.
