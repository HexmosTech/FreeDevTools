NAME
       SDL_VERSIONNUM_MINOR - This macro extracts the minor version from a version number


SYNOPSIS
       #include <SDL3/SDL_version.h>

       #define SDL_VERSIONNUM_MINOR(version) (((version) / 1000) % 1000)


DESCRIPTION
       1002003 becomes 2.


MACRO PARAMETERS
       version
              the version number.


AVAILABILITY
       This macro is available since SDL 3.2.0.
