NAME
       SQL::Translator::Parser::Excel - parser for Excel


SYNOPSIS
         use SQL::Translator;

         my $translator = SQL::Translator->new;
         $translator->parser('Excel');


DESCRIPTION
       Parses an Excel spreadsheet file using Spreadsheet::ParseExcel.
