NAME
       Class::Accessor::Chained::Fast - Faster, but less expandable, chained accessors


SYNOPSIS
        package Foo;
        use base qw(Class::Accessor::Chained::Fast);

         # The rest as Class::Accessor::Chained except no set() or get().


DESCRIPTION
       By analogue to Class::Accessor and Class::Accessor::Fast this module provides a faster less-flexible
       chained accessor maker.
