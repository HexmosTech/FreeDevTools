NAME
       ck_bitmap_union — generates union of two bitmaps


LIBRARY
       Concurrency Kit (libck, -lck)


SYNOPSIS
       #include <ck_bitmap.h>

       void
       ck_bitmap_union(ck_bitmap_t *dst, ck_bitmap_t *src);


DESCRIPTION
       The  ck_bitmap_union() function sets all bits in the bitmap pointed to by src in the bitmap pointed to by
       dst.


RETURN VALUES
       This function has no return value.
