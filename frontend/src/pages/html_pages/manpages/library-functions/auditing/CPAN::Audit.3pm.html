NAME
       CPAN::Audit - Audit CPAN distributions for known vulnerabilities


SYNOPSIS
               use CPAN::Audit;


DESCRIPTION
       CPAN::Audit is a module and a database at the same time. It is used by cpan-audit command line
       application to query for vulnerabilities.
