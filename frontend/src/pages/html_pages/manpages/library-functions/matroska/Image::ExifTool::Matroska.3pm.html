NAME
       Image::ExifTool::Matroska - Read meta information from Matroska files


SYNOPSIS
       This module is used by Image::ExifTool


DESCRIPTION
       This module contains definitions required by Image::ExifTool to read meta information from Matroska
       multimedia files (MKA, MKV, MKS and WEBM).
