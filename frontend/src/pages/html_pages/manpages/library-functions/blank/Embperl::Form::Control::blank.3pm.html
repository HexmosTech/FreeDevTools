NAME
       Embperl::Form::Control::blank - A blank area inside an Embperl Form


SYNOPSIS
         {
         type => 'blank',
         text => 'blabla'
         }


DESCRIPTION
       Used to create a blank area with optional text inside an Embperl Form.  See Embperl::Form on how to
       specify parameters.

   PARAMETER
       type

       Needs to be 'blank'

       text (optional)

       Could be used to give a text that should be displayed inside the blank area
