NAME
       Code::TidyAll::Plugin::JSLint - Use jslint with tidyall


VERSION
       version 0.84


SYNOPSIS
          In configuration:

          [JSLint]
          select = static/**/*.js
          argv = --white --vars --regex


DESCRIPTION
       Runs jslint <http://www.jslint.com/>, a JavaScript validator, and dies if any problems were found.


INSTALLATION
       Install npm <https://npmjs.org/>, then run

           npm install jslint


CONFIGURATION
       This plugin accepts the following configuration options:

   argv
       Arguments to pass to "jslint".

   cmd
       The path for the "jslint" command. By default this is just "jslint", meaning that the user's "PATH" will
       be searched for the command.
