NAME
       Config::Any::JSON - Load JSON config files


DESCRIPTION
       Loads JSON files. Example:

           {
               "name": "TestApp",
               "Controller::Foo": {
                   "foo": "bar"
               },
               "Model::Baz": {
                   "qux": "xyzzy"
               }
           }


METHODS
   extensions( )
       return an array of valid extensions ("json", "jsn").

   load( $file )
       Attempts to load $file as a JSON file.

   requires_any_of( )
       Specifies that this modules requires one of, Cpanel::JSON::XS, JSON::MaybeXS, JSON::DWIW, JSON::XS,
       JSON::Syck, JSON::PP or JSON in order to work.
