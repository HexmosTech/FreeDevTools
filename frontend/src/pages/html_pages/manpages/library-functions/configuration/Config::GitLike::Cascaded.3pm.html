NAME
       Config::GitLike::Cascaded - git-like config file parsing with cascaded inheritance


SYNOPSIS
       This module exists purely for backwards compatibility; its use is deprecated, and will be removed in a
       future release.


METHODS

       This module simply defaults "cascade" in Config::GitLike to a true value.
