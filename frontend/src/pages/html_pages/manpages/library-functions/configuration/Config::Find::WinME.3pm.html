NAME
       Config::Find::WinME - WinME idiosyncrasies for Config::Find


SYNOPSIS
         # don't use Config::Find::WinME directly
         use Config::Find;


ABSTRACT
       Implements WinME specific features for Config::Find


DESCRIPTION
       Contains any idiosyncrasies found within WinME, that do not apply to the standard Win32 base.


SEE ALSO
       Config::Find, Config::Find::WinAny, Config::Find::Any
