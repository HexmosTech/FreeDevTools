NAME
       Perl::Critic::Exception::Configuration::Option::Global::ExtraParameter - The configuration referred to a
       non-existent global option.


DESCRIPTION
       A representation of the configuration attempting to specify a value for an option that Perl::Critic
       doesn't have, whether from a .perlcriticrc, another profile file, or command line.


INTERFACE SUPPORT
       This is considered to be a public class.  Any changes to its interface will go through a deprecation
       cycle.


CLASS METHODS
       "throw( option_name => $option_name, source => $source )"
           See "throw" in Exception::Class.

       "new( option_name => $option_name, source => $source )"
           See "new" in Exception::Class.


METHODS
       full_message()
           Provide  a  standard message for values for non-existent parameters for policies.  See "full_message"
           in Exception::Class.
