NAME
       OAuth::Lite2::Agent::Strict - Preset User Agent class for strict SSL


SYNOPSIS
           my $client = OAuth::Lite2::Client::WebApp->new(
               ..., # other params
               agent => OAuth::Lite2::Client::Agent::Strict->new(
                   https_version => $https_version,
                   ..., # https parameters
               ),
           );


DESCRIPTION
       This module is one of preset user-agent class.  This is useful when you want check the SSL strictly.


METHODS
   request ($req)
       Append to the behavior of parent class, this method verify the SSL, and if it fails, it throws the
       exception.
