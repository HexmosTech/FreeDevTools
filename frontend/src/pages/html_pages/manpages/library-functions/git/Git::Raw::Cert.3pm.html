NAME
       Git::Raw::Cert - Git certificate class


VERSION
       version 0.90


DESCRIPTION
       A Git::Raw::Cert object represents a certificate.

       WARNING: The API of this module is unstable and may change without warning (any change will be
       appropriately documented in the changelog).


METHODS
   type( )
       Type of certificate, "x509", "hostkey", "strarray" or "none".
