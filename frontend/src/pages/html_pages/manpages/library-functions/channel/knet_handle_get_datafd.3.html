NAME
       knet_handle_get_datafd - Get the file descriptor associated with a channel.


SYNOPSIS
       #include <libknet.h>

       int knet_handle_get_datafd(
           knet_handle_t  knet_h,
           const int8_t   channel,
           int           *datafd
       );


DESCRIPTION
       knet_handle_get_datafd

       knet_h - pointer to knet_handle_t

       channel - get the datafd associated to this channel

       *datafd - will contain the result


RETURN VALUE
       knet_handle_get_datafd returns

       0          on success and *datafd will contain the results

       -1         on error and errno is set. and *datafd content is meaningless
