NAME
       s3d_pep_material - rewrite material


SYNOPSIS
       #include <s3d.h>

       int s3d_pep_material(int object, float amb_r, float amb_g, float amb_b, float spec_r, float spec_g,
                            float spec_b, float diff_r, float diff_g, float diff_b);


DESCRIPTION
       Overwriting the latest pushed material, overwriting the current value with the specified one. See
       s3d_pep_materials_a if you want to pep more materials.


AUTHOR
       Simon Wunderlich
           Author of s3d
