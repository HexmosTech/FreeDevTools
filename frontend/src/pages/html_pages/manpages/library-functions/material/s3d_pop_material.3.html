NAME
       s3d_pop_material - remove materials


SYNOPSIS
       #include <s3d.h>

       int s3d_pop_material(int object, uint32_t n);


DESCRIPTION
       Deletes the latest n material from the material stack of the object.


AUTHOR
       Simon Wunderlich
           Author of s3d
