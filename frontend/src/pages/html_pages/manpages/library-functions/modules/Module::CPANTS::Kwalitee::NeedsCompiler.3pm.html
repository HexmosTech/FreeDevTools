NAME
       Module::CPANTS::Kwalitee::NeedsCompiler - Checks if the module needs a (probably C) compiler


SYNOPSIS
       Checks if there is some indication in the module that it needs a C compiler to build and install


DESCRIPTION
   Methods
       order

       Defines the order in which Kwalitee tests should be run.

       Returns 200.

       analyse

       Checks for file with .c, .h or .xs extensions.  Check is the module depends on any of the Inline::
       modules or on ExtUtils::CBuilder or ExtUtils::ParseXS.
