NAME
       Badger::Apps - factory module for application modules


DESCRIPTION
       This module implements a subclass of Badger::Factory for loading and instantiating Badger::App
       application modules.


METHODS
       The following methods are defined in addition to those inherited from the Badger::Factory and
       Badger::Base base classes.

   found_module($type, $module, @args)
   not_found($type, @args)
