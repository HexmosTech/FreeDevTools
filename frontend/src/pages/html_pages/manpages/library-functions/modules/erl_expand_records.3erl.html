NAME
       erl_expand_records - Expands records in a module.


DESCRIPTION
       This module expands records in a module.


EXPORTS
       module(AbsForms, CompileOptions) -> AbsForms2

              Types:

                 AbsForms = AbsForms2 = [erl_parse:abstract_form()]
                 CompileOptions = [compile:option()]

              Expands all records in a module to use explicit tuple operations and adds explicit module names to
              calls  to  BIFs  and  imported  functions.  The  returned  module  has  no  references to records,
              attributes, or code.
