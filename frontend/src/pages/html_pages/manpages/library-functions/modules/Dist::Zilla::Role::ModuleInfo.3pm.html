NAME
       Dist::Zilla::Role::ModuleInfo - Create Module::Metadata object from Dist::Zilla::File


VERSION
       This document describes version 6.000 of Dist::Zilla::Role::ModuleInfo, released December 17, 2017 as
       part of Dist-Zilla-Plugins-CJM version 6.000.


DESCRIPTION
       Plugins implementing ModuleInfo may call their own "get_module_info" method to construct a
       Module::Metadata object.  (Module::Metadata is the new name for Module::Build::ModuleInfo, now that it's
       been split from the Module-Build distribution.)


METHODS
   get_module_info
         my $info = $plugin->get_module_info($file);

       This constructs a Module::Metadata object from the contents of a $file object that does
       Dist::Zilla::Role::File.  Any additional arguments are passed along to "Module::Metadata->new_from_file".


CONFIGURATION AND ENVIRONMENT
       Dist::Zilla::Role::ModuleInfo requires no configuration files or environment variables.


DEPENDENCIES
       Module::Metadata.


INCOMPATIBILITIES
       None reported.
