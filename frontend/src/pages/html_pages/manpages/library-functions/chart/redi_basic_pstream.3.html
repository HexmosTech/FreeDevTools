Detailed Description
   template<typename CharT, typename Traits = std::char_traits<CharT>>
       class redi::basic_pstream< CharT, Traits >" Class template for Bidirectional PStreams.

       Writing to a pstream opened with pmode pstdin writes to the standard input of the command. Reading from a
       pstream opened with pmode pstdout and/or pstderr reads the command's standard output and/or standard
       error. Any of the process' stdin, stdout or stderr that is not connected to the pstream (as specified by
       the pmode) will be the same as the process that created the pstream object, unless altered by the command
       itself.
