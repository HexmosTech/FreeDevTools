Detailed Description
   template<typename _CharT>
       class std::__detail::_Scanner< _CharT >"Scans an input range for regex tokens.

       The _Scanner class interprets the regular expression pattern in the input range passed to its constructor
       as a sequence of parse tokens passed to the regular expression compiler. The sequence of tokens provided
       depends on the flag settings passed to the constructor: different regular expression grammars will
       interpret the same input pattern in syntactically different ways.
