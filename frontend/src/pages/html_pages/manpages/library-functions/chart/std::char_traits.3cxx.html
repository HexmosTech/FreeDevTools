Detailed Description
   template<typename _CharT>
       struct std::char_traits< _CharT >"Basis for explicit traits specializations.

       Note
           For any given actual character type, this definition is probably wrong. Since this is just a thin
           wrapper around __gnu_cxx::char_traits, it is possible to achieve a more appropriate definition by
           specializing __gnu_cxx::char_traits.

       See https://gcc.gnu.org/onlinedocs/libstdc++/manual/strings.html#strings.string.character_types for
       advice on how to make use of this class for unusual character types. Also, check out
       include/ext/pod_string.
