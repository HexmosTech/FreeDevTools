Detailed Description
   template<typename _CharT, typename _OutIter>
       class std::time_put< _CharT, _OutIter >"Primary class template time_put.

       This facet encapsulates the code to format and output dates and times according to formats used by
       strftime().

       The time_put template uses protected virtual functions to provide the actual results. The public
       accessors forward the call to the virtual functions. These virtual functions are hooks for developers to
       implement the behavior they require from the time_put facet.
