Detailed Description

   template<typename _Tp>
       struct __gnu_parallel::_LoserTreeTraits< _Tp >"Traits for determining whether the loser tree should use
       pointers or copies.

       The field '_M_use_pointer' is used to determine whether to use pointers in he loser trees or whether to
       copy the values into the loser tree.

       The default behavior is to use pointers if the data type is 4 times as big as the pointer to it.

       Specialize for your data type to customize the behavior.

       Example:

       template<> struct _LoserTreeTraits<int> { static const bool _M_use_pointer = false; };

       template<> struct _LoserTreeTraits<heavyweight_type> { static const bool _M_use_pointer = true; };
