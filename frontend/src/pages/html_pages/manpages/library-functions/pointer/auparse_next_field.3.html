NAME
       auparse_next_field - move field cursor


SYNOPSIS
       #include <auparse.h>

       int auparse_next_field(const auparse_state_t *au);


DESCRIPTION
       auparse_next_field  moves  the library's internal cursor to point to the next field in the current record
       of the current event.


RETURN VALUE
       Returns 0 if no more fields exist and 1 for success.


SEE ALSO
       auparse_next_record(3).
