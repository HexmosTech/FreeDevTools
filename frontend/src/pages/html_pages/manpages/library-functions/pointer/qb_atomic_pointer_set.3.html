NAME
       qb_atomic_pointer_set - Sets the value of the pointer pointed to by atomic.


SYNOPSIS
       #include <qb/qbatomic.h>

       void qb_atomic_pointer_set(
           volatile void *QB_GNUC_MAY_ALIAS   *atomic,  /* a pointer to a void* */
           void                               *newval   /* the new value */
       );


PARAMS
       atomic a pointer to a void*

       newval the new value


DESCRIPTION
       Also acts as a memory barrier.
