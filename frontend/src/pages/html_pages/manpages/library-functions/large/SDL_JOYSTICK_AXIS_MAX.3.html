NAME
       SDL_JOYSTICK_AXIS_MAX - The largest value an SDL_Joystick's axis can report.


SYNOPSIS
       #include <SDL3/SDL_joystick.h>

       #define SDL_JOYSTICK_AXIS_MAX   32767


AVAILABILITY
       This macro is available since SDL 3.2.0.
