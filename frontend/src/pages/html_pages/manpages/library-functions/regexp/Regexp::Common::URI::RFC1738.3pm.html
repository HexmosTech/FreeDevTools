NAME
       Regexp::Common::URI::RFC1738 -- Definitions from RFC1738;


SYNOPSIS
           use Regexp::Common::URI::RFC1738 qw /:ALL/;


DESCRIPTION
       This package exports definitions from RFC1738. It's intended usage is for Regexp::Common::URI submodules
       only. Its interface might change without notice.


REFERENCES
       [RFC 1738]
           Berners-Lee, Tim, Masinter, L., McCahill, M.: Uniform Resource Locators (URL). December 1994.
