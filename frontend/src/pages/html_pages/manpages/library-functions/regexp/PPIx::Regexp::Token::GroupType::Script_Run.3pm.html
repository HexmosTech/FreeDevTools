NAME
       PPIx::Regexp::Token::GroupType::Script_Run - Represent a script run specifier


SYNOPSIS
        use PPIx::Regexp::Dumper;
        PPIx::Regexp::Dumper->new( 'qr{(*script_run:\d)}' )
            ->print();


INHERITANCE
       "PPIx::Regexp::Token::GroupType::Script_Run" is a PPIx::Regexp::Token::GroupType.

       "PPIx::Regexp::Token::GroupType::Script_Run" has no descendants.


DESCRIPTION
       This token represents the specifier for a script run - namely the '+script_run:', '*script_run:', or
       '*sr:' that comes after the left parenthesis. The first form was added in Perl 5.27.8 but retracted in
       favor of the second ant third forms (which are equivalent) in Perl 5.27.9.

       If this construction does not make it into Perl 5.28, this class will be retracted.


METHODS
       This class provides no public methods beyond those provided by its superclass.
