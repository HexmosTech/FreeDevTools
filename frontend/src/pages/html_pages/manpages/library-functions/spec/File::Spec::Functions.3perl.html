NAME
       File::Spec::Functions - portably perform operations on file names


SYNOPSIS
               use File::Spec::Functions;
               my $x = catfile('a', 'b');


DESCRIPTION
       This module exports convenience functions for all of the class methods provided by File::Spec.

       For a reference of available functions, please consult File::Spec::Unix, which contains the entire set,
       and which is inherited by the modules for other platforms. For further information, please see
       File::Spec::Mac, File::Spec::OS2, File::Spec::Win32, or File::Spec::VMS.

   Exports
       The following functions are exported by default.

               canonpath
               catdir
               catfile
               curdir
               rootdir
               updir
               no_upwards
               file_name_is_absolute
               path

       The following functions are exported only by request.

               devnull
               tmpdir
               splitpath
               splitdir
               catpath
               abs2rel
               rel2abs
               case_tolerant

       All the functions may be imported using the ":ALL" tag.
