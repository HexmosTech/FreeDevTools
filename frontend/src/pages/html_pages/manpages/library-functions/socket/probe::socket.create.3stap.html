NAME
       probe::socket.create - Creation of a socket


SYNOPSIS
       socket.create


VALUES
       name
           Name of this probe

       protocol
           Protocol value

       family
           Protocol family value

       requester
           Requested by user process or the kernel (1 = kernel, 0 = user)

       type
           Socket type value


CONTEXT
       The requester (see requester variable)


DESCRIPTION
       Fires at the beginning of creating a socket.

       tapset::socket(3stap)
