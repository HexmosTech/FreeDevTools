NAME
       probe::socket.writev - Message sent via socket_writev


SYNOPSIS
       socket.writev


VALUES
       family
           Protocol family value

       type
           Socket type value

       state
           Socket state value

       size
           Message size in bytes

       flags
           Socket flags value

       protocol
           Protocol value

       name
           Name of this probe

DESCRIPTION
