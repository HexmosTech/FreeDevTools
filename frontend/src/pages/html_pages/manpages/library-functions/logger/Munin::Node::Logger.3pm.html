NAME
       Munin::Node::Logger - The logger for munin node.


SYNOPSIS
       Exports a logger() subroutine.

        use Munin::Node::Logger;

        logger("Nice log message");


SUBROUTINES
       logger()
             logger($message);

           Writes $message to STDERR together with the timestamp and process id.
