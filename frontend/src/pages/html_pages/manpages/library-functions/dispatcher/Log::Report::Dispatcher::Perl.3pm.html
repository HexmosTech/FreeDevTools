NAME
       Log::Report::Dispatcher::Perl - send messages to die and warn


INHERITANCE
        Log::Report::Dispatcher::Perl
          is a Log::Report::Dispatcher


SYNOPSIS
        dispatcher Log::Report::Dispatcher::Perl => 'default'
          , accept => 'NOTICE-';

        # close the default dispatcher
        dispatcher close => 'default';


DESCRIPTION
       Ventilate the problem reports via the standard Perl error mechanisms: die(), warn(), and print().  There
       can be only one such dispatcher (per thread), because once die() is called, we are not able to return.
       Therefore, this dispatcher will always be called last.

       In the early releases of Log::Report, it tried to simulate the behavior of warn and die using STDERR and
       exit; however: that is not possible.

       Extends "DESCRIPTION" in Log::Report::Dispatcher.
