NAME
       TAP::Parser::Result::Pragma - TAP pragma token.


VERSION
       Version 3.48


DESCRIPTION
       This is a subclass of TAP::Parser::Result.  A token of this class will be returned if a pragma is
       encountered.

        TAP version 13
        pragma +strict, -foo

       Pragmas are only supported from TAP version 13 onwards.
