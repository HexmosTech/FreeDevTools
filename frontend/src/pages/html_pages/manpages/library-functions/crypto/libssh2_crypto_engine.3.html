NAME
       libssh2_crypto_engine - retrieve used crypto engine


SYNOPSIS
       #include <libssh2.h>

       libssh2_crypto_engine_t
       libssh2_crypto_engine(void);


DESCRIPTION
       Returns currently used crypto engine, as en enum value.


AVAILABILITY
       Added in libssh2 1.11
