NAME
       Apache::Session::Serialize::Base64 - Use Storable and MIME::Base64 to zip up persistent data


SYNOPSIS
        use Apache::Session::Serialize::Base64;

        $zipped = Apache::Session::Serialize::Base64::serialize($ref);
        $ref = Apache::Session::Serialize::Base64::unserialize($zipped);


DESCRIPTION
       This module fulfills the serialization interface of Apache::Session.  It serializes the data in the
       session object by use of Storable's "nfreeze()" and "thaw()" functions, and MIME::Base64's "encode_bas64"
       and "decode_base64".  The serialized data is ASCII text, suitable for storage in backing stores that
       don't handle binary data gracefully, such as Postgres.
