NAME
       Dancer::Session::Simple - in-memory session backend for Dancer


VERSION
       version 1.3521


DESCRIPTION
       This module implements a very simple session backend, holding all session data in memory.  This means
       that sessions are volatile, and no longer exist when the process exits.  This module is likely to be most
       useful for testing purposes.


CONFIGURATION
       The setting session should be set to "Simple" in order to use this session engine in a Dancer
       application.
