NAME
       plend1 - End plotting session for current stream


SYNOPSIS
       plend1()


DESCRIPTION
       Ends a plotting session for the current output stream only.  See plsstrm(3plplot) for more info.

       Redacted form: plend1()

       This function is used in examples 1 and 20.
