NAME
       Apache::Session::Generate::ModUniqueId - mod_unique_id for session ID generation


SYNOPSIS
         use Apache::Session::Flex;

         tie %session, 'Apache::Session::Flex', $id, {
              Store     => 'MySQL',
              Lock      => 'Null',
              Generate  => 'ModUniqueId',
              Serialize => 'Storable',
         };


DESCRIPTION
       Apache::Session::Generate::ModUniqueId enables you to use unique id generated by mod_unique_id as session
       id for Apache::Session framework. Using mod_unique_id would ensure higher level uniquess of id.
