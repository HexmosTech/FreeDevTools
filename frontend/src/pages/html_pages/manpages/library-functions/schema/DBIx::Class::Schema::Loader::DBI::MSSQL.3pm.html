NAME
       DBIx::Class::Schema::Loader::DBI::MSSQL - DBIx::Class::Schema::Loader::DBI MSSQL Implementation.


DESCRIPTION
       Base driver for Microsoft SQL Server, used by
       DBIx::Class::Schema::Loader::DBI::Sybase::Microsoft_SQL_Server for support via DBD::Sybase and
       DBIx::Class::Schema::Loader::DBI::ODBC::Microsoft_SQL_Server for support via DBD::ODBC.

       See DBIx::Class::Schema::Loader and DBIx::Class::Schema::Loader::Base for usage information.


CASE SENSITIVITY
       Most MSSQL databases use "CI" (case-insensitive) collation, for this reason generated column names are
       lower-cased as this makes them easier to work with in DBIx::Class.

       We attempt to detect the database collation at startup for any database included in db_schema, and set
       the column lowercasing behavior accordingly, as lower-cased column names do not work on case-sensitive
       databases.

       To manually control case-sensitive mode, put:

           preserve_case => 1|0

       in your Loader options.

       See preserve_case.

       NOTE: this option used to be called "case_sensitive_collation", but has been renamed to a more generic
       option.
