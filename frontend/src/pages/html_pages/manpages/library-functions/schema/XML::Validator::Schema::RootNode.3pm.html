NAME
       XML::Validator::Schema::RootNode - the root node in a schema document


DESCRIPTION
       This is an internal module used by XML::Validator::Schema to represent the root node in an XML Schema
       document.  Holds references to the libraries for the schema document and is responsible for hooking up
       named types to their uses in the node tree at the end of parsing.
