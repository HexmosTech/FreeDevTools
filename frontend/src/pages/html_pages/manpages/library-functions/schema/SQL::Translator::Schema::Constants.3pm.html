NAME
       SQL::Translator::Schema::Constants - constants module


SYNOPSIS
         use SQL::Translator::Schema::Constants;

         $table->add_constraint(
             name => 'foo',
             type => PRIMARY_KEY,
         );


DESCRIPTION
       This module exports the following constants for Schema features;

       CHECK_C
       FOREIGN_KEY
       FULL_TEXT
       NOT_NULL
       NORMAL
       NULL
       PRIMARY_KEY
       UNIQUE
       EXCLUDE
