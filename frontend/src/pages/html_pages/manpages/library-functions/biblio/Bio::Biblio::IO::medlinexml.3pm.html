NAME
       Bio::Biblio::IO::medlinexml - a converter of XML files with MEDLINE citations


VERSION
       version 1.70


SYNOPSIS
       Do not use this object directly, it is recommended to access it and use it through the Bio::Biblio::IO
       module:

         use Bio::Biblio::IO;
         my $io = Bio::Biblio::IO->new(-format => 'medlinexml');


DESCRIPTION
       This object reads bibliographic citations in XML/MEDLINE format and converts them into Bio::Biblio::RefI
       objects. It is an implementation of methods defined in Bio::Biblio::IO.

       The main documentation details are to be found in Bio::Biblio::IO.

INTERNAL METHODS
   _initialize
   _parse
   _process_citation
       What to do when we have the whole $citation ready

   _add_element
       Add $element into an array named $key to the top object at @ObjectStack; if $element is empty, take it
       from @PCDataStack

   _data2obj
       Remove top of @PCDataStack and put it into top object at @ObjectStack under name $key

   _obj2obj
       Remove top of @ObjectStack and put it into now-top at @ObjectStack under name $key

   _eq_hash_elem
       Check if a $key exists in a ref-hash $rh and if it is equal to $value

   _debug_object_stack
       --- only for debugging
