NAME
       Biblio::SICI::Role::RecursiveLink - Role to provide a "link" to the parent Biblio::SICI


VERSION
       version 0.04


DESCRIPTION
       A role that provides an attribute used to provide internal access to the parent "Biblio::SICI" object
       from the three segment classes.

       For internal use only!


ATTRIBUTES
       _sici
           Weak ref to the parent object.
