NAME
       Bio::Cluster::UniGeneI - abstract interface of UniGene object


SYNOPSIS
         #


DESCRIPTION
       This is the general interface for a UniGene cluster representation in Bioperl. You cannot use this module
       directly, use an implementation instead.

       You can create UniGene cluster objects yourself by instantiating Bio::Cluster::UniGene. If you read
       UniGene clusters from a ClusterIO parser, you will get objects implementing this interface, most likely
       instances of said UniGene class.

       Bio::Cluster::UniGeneI inherits from Bio::ClusterI, so you can use it wherever a cluster object is
       expected.
