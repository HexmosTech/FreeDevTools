NAME
       HTML::FormFu::Filter::CopyValue - copy the value from another field


VERSION
       version 2.07


SYNOPSIS
          elements:
             - type: Text
               name: username
             - type: Text
               name: nickname
               filters:
                  - type: CopyValue
                    field: username


DESCRIPTION
       Filter copying the value of another field if the original value of this field is empty.


CAVEATS
       If the original field contains an invalid value (a value that will be constrained through a constraint)
       that invalid value will be copied to this field (the field with the CopyValue filter).  So, the user has
       to change two fields, or you can remove the invalid value in a custom constraint.
