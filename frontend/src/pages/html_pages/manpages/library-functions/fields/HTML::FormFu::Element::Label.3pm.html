NAME
       HTML::FormFu::Element::Label - field for displaying only


VERSION
       version 2.07


DESCRIPTION
       This element displays a value. This is useful if you use a model like HTML::FormFu::Model::DBIC and want
       to display a value from the database.  The value of this field cannot be set by the client.

       See "Set a field read only" in HTML::FormFu::Model::DBIC for more information on read only fields.


METHODS
   tag
       Set the tag for this element.


SEE ALSO
       Is a sub-class of, and inherits methods from HTML::FormFu::Role::Element::Field, HTML::FormFu::Element
