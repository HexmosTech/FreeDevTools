NAME
       HTML::FormHandler::Field::NoValue - base class for submit field


VERSION
       version 0.40068


SYNOPSIS
       This is the base class for the Submit & Reset fields. It can be used for fields that do not produce valid
       'values'. It should not be used for fields that produce a value or need validating.
