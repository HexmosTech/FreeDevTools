NAME
       TIFFFieldName - get TIFF field name from field information


SYNOPSIS
          #include <tiffio.h>

       const char *TIFFFieldName(const TIFFField *fip)


DESCRIPTION
       TIFFFieldName() returns the textual name for a TIFF field.  fip is a field information pointer previously
       returned by TIFFFindField(), TIFFFieldWithTag(), or TIFFFieldWithName().


RETURN VALUES
       TIFFFieldName() returns a constant C string.
