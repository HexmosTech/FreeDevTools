NAME
       Pango::AttrList - holds attributes that apply to a section of text


DESCRIPTION
       Pango::AttrList is a collection of Pango::Attributes.  These attributes annotate text with styles.


HIERARCHY
         Glib::Boxed
         +----Pango::AttrList

   ($attr_list, $text, $accel_char) = Pango->parse_markup ($markup_text, $accel_marker)
       •   $markup_text (string)

       •   $accel_marker (character)

       Parses marked-up text to create a plaintext string and an attribute list.

       If  $accel_marker is supplied and nonzero, the given character will mark the character following it as an
       accelerator.  For example, the accel marker might be an ampersand or underscore.  All  characters  marked
       as  an acclerator will receive a PANGO_UNDERLINE_LOW attribute, and the first character so marked will be
       returned in $accel_char.  Two $accel_marker characters following each other reduce to  a  single  literal
       $accel_marker character.

       May croak with a Glib::Error in $@ on failure.
