NAME
       Moose::Autobox::Indexed - the Indexed role


VERSION
       version 0.16


DESCRIPTION
       This is a role to describes an collection whose values can be accessed by a key of some kind.

       The role is entirely abstract, those which implement it must supply all it's methods. Currently both
       Moose::Autobox::Array and Moose::Autobox::Hash implement this role.


METHODS
       "meta"


REQUIRED METHODS
       "at"
       "put"
       "exists"
       "keys"
       "values"
       "kv"
       "slice"
       "each"
       "each_key"
       "each_value"
