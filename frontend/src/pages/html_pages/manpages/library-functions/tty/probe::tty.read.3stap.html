NAME
       probe::tty.read - called when a tty line will be read


SYNOPSIS
       tty.read


VALUES

           the file name lreated to the tty

       buffer
           the buffer that will receive the characters

       driver_name
           the driver name

       nr
           The amount of characters to be read
