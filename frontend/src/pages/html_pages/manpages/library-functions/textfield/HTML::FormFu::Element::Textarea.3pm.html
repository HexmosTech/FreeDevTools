NAME
       HTML::FormFu::Element::Textarea - Textarea form field


VERSION
       version 2.07


SYNOPSIS
           my $element = $form->element( Textarea => 'foo' );


DESCRIPTION
       Textarea form field.


ATTRIBUTE ACCESSORS
       Get / set input attributes directly with these methods.

       Arguments: [$string]

       Return Value: $string

   autocomplete
   cols
   maxlength
   rows
   placeholder

       Is a sub-class of, and inherits methods from HTML::FormFu::Role::Element::Field, HTML::FormFu::Element
