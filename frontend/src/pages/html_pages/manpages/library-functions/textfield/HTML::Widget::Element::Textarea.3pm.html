NAME
       HTML::Widget::Element::Textarea - Textarea Element


SYNOPSIS
           my $e = $widget->element( 'Textarea', 'foo' );
           $e->comment('(Required)');
           $e->label('Foo');
           $e->cols(30);
           $e->rows(40);
           $e->value('bar');
           $e->wrap('wrap');


DESCRIPTION
       Textarea Element.


METHODS
   new
       Create new textarea with default size of 20 rows and 40 columns

   containerize
   label
   value
   cols
   rows
   wrap
   retain_default
       If true, overrides the default behaviour, so that after a field is missing from the form submission, the
       xml output will contain the default value, rather than be empty.
