NAME
       CSC2S - convert from  Cartesian to  lat/lon coordinates


SYNOPSIS
       CALL CSC2S (N, X, Y, Z, RLAT, RLON)


DESCRIPTION
       N           (integer, input) The number of coordinates to convert.

       X,Y,Z       (real, input) The Cartesian coordinates of the input points.

       RLAT        (real, output) The latitudes of the output coordinates, in degrees.

       RLON        (real, output) The longitudes of the output coordinates, in degrees.


USAGE
       CSC2S  is called to find equivalent latitude and longitude coordinates to specified Cartesian coordinates
       on the unit sphere.  The coordinate (1.,0.,0.) is mapped to the  latitude/longitude  coordinate  (0.,0.).
       The latitude/longitude coordinates are returned in degrees.


ACCESS
       To use CSC2S, load the NCAR Graphics library ngmath.
