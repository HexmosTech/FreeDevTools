NAME
       NNSETI - Sets the value of an internal parameter of type INTEGER.


SYNOPSIS
       CALL NNSETI (PNAM,IVAL)


DESCRIPTION
       PNAM        A character string that specifies the name of the parameter to be set.

       IVAL        An  INTEGER  value  that  is  the value to be assigned to the internal parameter specified by
                   PNAM.


USAGE
       This routine allows you to set the  current  value  of  Natgrid  parameters.   For  a  complete  list  of
       parameters available in this utility, see the natgrid_params man page.


ACCESS
       To use NNSETI, load the NCAR Graphics library ngmath.


SEE ALSO
       natgrid, natgrid_params, natgrids, nngeti.

       http://ngwww.ucar.edu/ngdoc/ng/ngmath/natgrid/nnhome.html
