NAME
       c_csgetr - Retrieves the value of an internal parameter of type float.


FUNCTION PROTOTYPE
       void c_csgetr(char *, float *);


SYNOPSIS
       c_csgetr (pnam,rval)


DESCRIPTION
       pnam        A  string  that specifies the name of the parameter to get. The name must appear as the first
                   three characters of the string.

       rval        *rval will be the value currently assigned to the control parameter whose name is pointed  to
                   by pnam.


USAGE
       This  routine  allows  you  to  retrieve the current value of Cssgrid parameters.  For a complete list of
       parameters available in this utility, see the csgrid_params man page.


ACCESS
       To use c_csgetr, load the NCAR Graphics library ngmath.


SEE ALSO
       c_csgrid, csgrid_params, c_cssetr.

       Complete documentation for Cssgrid is available at URL
       http://ngwww.ucar.edu/ngdoc/ng/ngmath/csgrid/cshome.html
