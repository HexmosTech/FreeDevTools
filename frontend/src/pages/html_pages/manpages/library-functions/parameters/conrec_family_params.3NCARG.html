NAME
       Conrec_family_params - Includes a brief description of all Conrec_family internal parameters.


DESCRIPTION
       No parameter setting functions are available for the Conrec_family utility.  Parameters were set through
       common blocks CONRE1 and CONRE4.  There are three versions of the CONREC code.  The normal version is in
       the standard NCAR Graphics library.  The other two versions are in optional libraries one of which is
       linked when the ncargf77 option -quick, or -super is declared on the command line.

       The CONREC and EZCNTR entries can be invoked in four different ways to create contour plots which vary
       considerably in appearance.  The four variations include quick, normal, smooth, and super contour lines.
       This progression represents a tradeoff between speed of computation and the appearance of the contour
       plots.  These variations are specified through selected command line options of the ncargf77 command.

       For normal or smooth options the parameter set is:

        COMMON /CONRE1/ IOFFP      ,SPVAL
        COMMON /CONRE4/ ISIZEL     ,ISIZEM     ,ISIZEP     ,NREP       ,
       1                NCRT       ,ILAB       ,NULBLL     ,IOFFD      ,
       2                EXT        ,IOFFM      ,ISOLID     ,NLA        ,
       3                NLM        ,XLT        ,YBT        ,SIDE

       For the quick option the parameter set is:

        COMMON /CONRE1/ IOFFP      ,SPVAL
        COMMON /CONRE4/ ISIZEM     ,ISIZEP     ,NLA        ,NLM        ,
       1                XLT        ,YBT        ,SIDE       ,ISOLID     ,
       2                EXT        ,IOFFD      ,IOFFM

       For the super option the parameter set is:

        COMMON /CONRE1/ IOFFP      ,SPVAL      ,IHILO
        DATA ISIZEL,ISIZEM,ISIZEP,NLA,NLM,XLT,YBT,SIDE,ISOLID,NREP,NCRT/
       1       1,  2,   0, 16, 40,.05,.05,  .9,  1023,   6,   4/
        DATA EXT,IOFFD,NULBLL,IOFFM,ILAB/.25,0,3,0,1/

       Definitions of the inclusive set follow:

       ISIZEL      Size of line labels in Plotter Address Units (PAUs).
                   0 = 8 PAUs,  1 = 12 PAUs, 2 = 16 PAUs, 3 = 24 PAUs,
                   > 3 means use this number of PAUs.  [Default = 1]

       ISIZEM      Size of labels for minimums and maximums, as per
                   the size definitions given for ISIZEL.
                   [Default = 2]

       ISIZEP      Size of labels for data point values as per the
                   size definitions given for ISIZEL.  [Default = 0]

       NLA         Approximate number of contour levels when
                   internally generated.  [Default = 16]

       NLM         Maximum number of contour levels.  If this is to be
                   increased, the dimensions of CL and RWORK in conrec.f
                   must be increased by the same amount.  [Default = 40]

       XLT         Left hand edge of the plot (0.0 is the left edge of
                   the frame and 1.0 is the right edge of the frame.)
                   [Default = .05]

       YBT         Bottom edge of the plot (0.0 is the bottom of the
                   frame and 1.0 is the top of the frame.)
                   [Default = .05]

       SIDE        Length of longer edge of plot (see also EXT).
                   [Default = .90]

       NREP        Number of repetitions of the dash pattern between
                   line labels.  [Default = 6]

       NCRT        Number of CRT units per element (bit) in the dash
                   pattern.  [Default = 4]

       ILAB        Flag to control the drawing of line labels.

                   - ILAB non-zero means label the lines.

                   - ILAB = 0 means do not label the lines.

                   [Default = 1]

       NULBUL      Number of unlabeled lines between labeled lines.
                   For example, when NULBLL = 3, every fourth level
                   is labeled.  [Default = 3]

       IOFFD       Flag to control normalization of label numbers.

                   - IOFFD = 0 means include decimal point when
                     possible (do not normalize unless required).

                   - IOFFD non-zero means normalize all label
                     numbers and output a scale factor in the
                     message below the graph.  [Default = 0]

       EXT         Lengths of the sides of the plot are proportional
                   to M and N (when CONREC sets the window and viewport).
                   In extreme cases, when MIN(M,N)/MAX(M,N) is less
                   than EXT, Conrec produces a square plot.
                   [Default = .25]

       IOFFP       Flag to control special value feature.

                   - IOFFP = 0 means the special value feature is
                     not in use.

                   - IOFFP non-zero means the special value feature
                     is in use.  (SPVAL is set to the special value.)
                     Contour lines will then be omitted from any cell
                     with any corner equal to the special value.

                   [Default = 0]

       SPVAL       Contains the special value when IOFFP is non-zero.
                   [Default = 0.]

       IOFFM       Flag to control the message below the plot.

                   - IOFFM = 0  if the message is to be plotted.

                   - IOFFM non-zero if the message is to be omitted.

                   [Default = 0]

       ISOLID      Dash pattern for non-negative contour lines.
                   [Default = 1023]
