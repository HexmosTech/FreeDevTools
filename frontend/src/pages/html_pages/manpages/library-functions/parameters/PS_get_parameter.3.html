NAME
       PS_get_parameter — Gets certain parameters


SYNOPSIS
       #include <pslib.h>

       char *PS_get_parameter(PSDoc *psdoc, char *name, float modifier)


DESCRIPTION
       Gets  several  parameters  which  were  set  by PS_set_parameter(3).  Parameters are by definition string
       values. This function cannot be used to retrieve resources which were also set by PS_set_parameter(3).


RETURN VALUE
       Returns the value of the parameter or NULL.


SEE ALSO
       PS_set_parameter(3)
