NAME
       cgiGetFiles - Returns a list of CGI file variables


SYNOPSYS
       #include <cgi.h>

       char **cgiGetFiles (s_cgi *parms);


DESCRIPTION
       This  routine returns a NULL terminated array of names of CGI file variables that are available.  Details
       are provided with  the  cgiGetFile(3)  function.   To  free  the  memory  allocated  for  the  array  use
       cgiFreeList(3).


RETURN VALUE
       On success a NULL terminated array of strings is returned.  The last element is set to NULL.  If an error
       occurred or if no files variables are available NULL is returned.
