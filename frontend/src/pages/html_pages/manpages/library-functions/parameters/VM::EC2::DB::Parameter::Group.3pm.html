NAME
       VM::EC2::DB::Parameter::Group - An RDS Database Parameter Group


SYNOPSIS
        use VM::EC2;

        $ec2 = VM::EC2->new(...);
        my @grps = $ec2->describe_db_parameter_groups();
        print $_,"\n" foreach @grps;


DESCRIPTION
       This object represents a DB Parameter Group.  It is the result of a VM::EC2->create_db_parameter_group
       and VM::EC2->describe_db_parameter_groups call and is an element returned by other calls.


STRING OVERLOADING
       In string context, the object returns the Parameter Group Name.
