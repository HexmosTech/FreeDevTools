NAME
       cgiGetFile - Return information of a CGI file variable


SYNOPSYS
       #include <cgi.h>

       s_file *cgiGetFile (s_cgi *parms, const char *name);


DESCRIPTION
       This  routine returns a pointer to a datastructure associated with the value of a CGI file variable.  The
       pointer must not be freed.

       The s_file structure is declared as follows:

              typedef struct file_s {
                  char   *name,
                         *type,
                         *filename,
                         *tmpfile;
              } s_file;

       Memory allocated by this data structure is automatically freed by the final call to cgiFree(3).


RETURN VALUE
       On success a pointer to the s_file datastructure is returned.  If the variable wasn't transmitted or  the
       through CGI or if the filename was empty NULL is returned.
