NAME
       DSSETRD - Sets the value of an internal parameter of type DOUBLE PRECISION.


SYNOPSIS
       CALL DSSETRD (PNAM, RVAL);


DESCRIPTION
       PNAM        A character string that specifies the name of the parameter to be set.

       RVAL        A DOUBLE PRECISION value that is the value to be assigned to the internal parameter specified
                   by pnam.


USAGE
       This routine allows you to set the current value of Dsgrid parameters.  For a complete list of parameters
       available in this utility, see the dsgrid_params man page.


ACCESS
       To use DSSETRD, load the NCAR Graphics library ngmath.
