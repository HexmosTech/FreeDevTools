NAME
       modbus_tcp_pi_accept - accept a new connection on a TCP PI Modbus socket (IPv6)


SYNOPSIS
       int modbus_tcp_pi_accept(modbus_t *ctx, int *'s);


DESCRIPTION
       The modbus_tcp_pi_accept() function shall extract the first connection on the queue of pending
       connections, create a new socket and store it in libmodbus context given in argument. If available,
       accept4() with SOCK_CLOEXEC will be called instead of accept().


RETURN VALUE
       The function shall return a new socket if successful. Otherwise it shall return -1 and set errno.


EXAMPLE
       For detailed example, see unit-test-server.c source file in tests directory.

           ...

           ctx = modbus_new_tcp_pi("::0", 502);
           s = modbus_tcp_pi_listen(ctx, 1);
           modbus_tcp_pi_accept(ctx, &s);

           ...

           close(s)
           modbus_free(ctx);
