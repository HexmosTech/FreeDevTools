NAME
       Embperl::Form::Control::price - A price input control with optional unit inside an Embperl Form


SYNOPSIS
         {
         type => 'price',
         text => 'blabla',
         name => 'foo',
         unit => 'sec',
         }


DESCRIPTION
       Used to create a price input control inside an Embperl Form.  Will format number as a money ammout.
       Optionally it can display an unit after the input field.  See Embperl::Form on how to specify parameters.

   PARAMETER
       type

       Needs to be 'price'

       name

       Specifies the name of the control

       text

       Will be used as label for the numeric input control

       size

       Gives the size in characters. (Default: 10)

       maxlength

       Gives the maximun length in characters

       unit

       Gives a string that should be displayed right of the input field.  (Default: Â€)

       use_comma

       If set the decimal character is comma instead of point (Default: on)
