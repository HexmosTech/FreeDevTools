NAME
       ONVIF::Device::Types::Profile


DESCRIPTION
       Perl data type class for the XML Schema defined complexType Profile from the namespace
       http://www.onvif.org/ver10/schema.

       A profile consists of a set of interconnected configuration entities. Configurations are provided by the
       NVT and can be either static or created dynamically by the NVT. For example, the dynamic configurations
       can be created by the NVT depending on current available encoding resources.
