NAME
       PS_set_gstate — Sets a previously created graphic state


SYNOPSIS
       #include <pslib.h>

       void PS_set_gstate(PSDoc *psdoc, int gstate)


DESCRIPTION
       Sets a graphic state which must have been created with PS_create_gstate(3).
