NAME
       Bio::Graphics::Glyph::group - The "group" glyph


SYNOPSIS
         See L<Bio::Graphics::Panel> and L<Bio::Graphics::Glyph>.


DESCRIPTION
       This glyph is used internally by Bio::Graphics::Panel for laying out groups of glyphs that move in
       concert.  It should not be used explicitly.
