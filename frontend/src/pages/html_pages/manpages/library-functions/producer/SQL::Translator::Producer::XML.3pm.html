NAME
       SQL::Translator::Producer::XML - Alias to XML::SQLFairy producer


DESCRIPTION
       Previous versions of SQL::Translator included an XML producer, but the namespace has since been further
       subdivided.  Therefore, this module is now just an alias to the XML::SQLFairy producer.
