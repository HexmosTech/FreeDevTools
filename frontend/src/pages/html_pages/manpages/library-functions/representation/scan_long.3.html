NAME
       scan_long - parse an long integer in decimal ASCII representation


SYNTAX
       #include <libowfat/scan.h>

       size_t scan_long(const char *src,long *dest);


DESCRIPTION
       scan_long parses a long integer in decimal ASCII representation from src and writes the result into dest.
       It returns the number of bytes read from src.
