NAME
       scan_short - parse an integer in decimal ASCII representation


SYNTAX
       #include <libowfat/scan.h>

       size_t scan_short(const char *src,short int *dest);


DESCRIPTION
       scan_short  parses  a  short  integer in decimal ASCII representation from src and writes the result into
       dest. It returns the number of bytes read from src.
