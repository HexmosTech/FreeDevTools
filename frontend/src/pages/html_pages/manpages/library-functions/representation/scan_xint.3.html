NAME
       scan_xint - parse an unsigned integer in hexadecimal ASCII representation


SYNTAX
       #include <libowfat/scan.h>

       size_t scan_xint(const char *src,int *dest);


DESCRIPTION
       scan_xint  parses  an unsigned integer in hexadecimal ASCII representation from src and writes the result
       into dest. It returns the number of bytes read from src.
