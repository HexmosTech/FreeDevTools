Detailed Description
       Support different endian modes when reading and writing the metafile.

       To support different endian modes, rather than just writing the structures directly to a file via fwrite(
       &emr, ...), we have to write each element of the structure separately, swapping bytes as necessary.
       datastream supports this. Remarkably similar to the QDataStream class from Qt. So, too, for reading.
