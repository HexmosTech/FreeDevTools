NAME
       Test2::Event::Warning - A Test2 event for warnings


VERSION
       version 0.10


DESCRIPTION
       An event representing an unwanted warning. This is treated as a failure.


ACCESSORS
       $warning = $event->warning
           Returns the warning that this event captured.
