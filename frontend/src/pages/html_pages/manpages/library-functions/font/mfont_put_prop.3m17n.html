NAME
       mfont_put_prop -


SYNOPSIS
       int mfont_put_prop (MFont * font, MSymbol key, void * val)


DESCRIPTION
       Put a property value to a font. The mfont_put_prop() function puts a font property whose key is key and
       value is val to font font.  key must be one of the following symbols:

       Mfoundry, Mfamily, Mweight, Mstyle, Mstretch, Madstyle, Mregistry, Msize, Mresolution.

       If key is Msize or Mresolution, val must be an integer. Otherwise, val must be a symbol of a property
       value name. But, if the name is 'nil', a symbol of name 'Nil' must be specified.
