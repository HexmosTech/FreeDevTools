NAME
       Image::ExifTool::Font - Read meta information from font files


SYNOPSIS
       This module is used by Image::ExifTool


DESCRIPTION
       This module contains the routines required by Image::ExifTool to read meta information from various
       format font files.  Currently recognized font file types are OTF, TTF, TTC, DFONT, PFA, PFB, PFM, AFM,
       ACFM and AMFM.  As well, WOFF and WOFF2 font files are identified, but metadata is not currently
       extracted from these formats.
