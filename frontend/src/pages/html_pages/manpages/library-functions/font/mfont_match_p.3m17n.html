NAME
       mfont_match_p -


SYNOPSIS
       int mfont_match_p (MFont * font, MFont * spec)


DESCRIPTION
       Check is a font matches with a font spec. The mfont_match_p() function checks if font matches with the
       font-spec spec.


RETURN VALUE
           If the font matches, 1 is returned. Otherwise 0 is returned.
