NAME
       mdebug_dump_font - Dump a font.


SYNOPSIS
       MFont* mdebug_dump_font (MFont * font)


DESCRIPTION
       Dump a font. The mdebug_dump_font() function prints font font in a human readable way to the stderr or to
       what specified by the environment variable MDEBUG_OUTPUT_FILE.


RETURN VALUE
           This function returns font.
