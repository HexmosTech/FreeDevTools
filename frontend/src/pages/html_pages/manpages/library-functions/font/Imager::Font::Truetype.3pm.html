NAME
         Imager::Font::Truetype - low-level functions for Truetype fonts


DESCRIPTION
       Imager::Font creates a Imager::Font::Truetype object when asked to create a font object based on a .ttf
       file.

       See Imager::Font to see how to use this type.

       This class provides low-level functions that require the caller to perform data validation.
