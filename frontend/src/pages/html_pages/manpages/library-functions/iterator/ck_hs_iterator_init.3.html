NAME
       ck_hs_iterator_init — initialize hash set iterator


LIBRARY
       Concurrency Kit (libck, -lck)


SYNOPSIS
       #include <ck_hs.h>

       ck_hs_iterator_t iterator = CK_HS_ITERATOR_INITIALIZER

       void
       ck_hs_iterator_init(ck_hs_iterator_t *iterator);


DESCRIPTION
       The  ck_hs_iterator_init(3)  function  will  initialize  the  object pointed to by the iterator argument.
       Alternatively,   an   iterator   may   be   statically   initialized   by    assigning    it    to    the
       CK_HS_ITERATOR_INITIALIZER value.

       An iterator is used to iterate through hash set entries with the ck_hs_next(3) function.


RETURN VALUES
       ck_hs_iterator_init(3) has no return value.


ERRORS
       This function will not fail.
