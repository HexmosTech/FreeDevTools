NAME
       Log::ger::Plugin - Use a plugin


VERSION
       version 0.042


SYNOPSIS
       To set globally:

        use Log::ger::Plugin;
        Log::ger::Plugin->set('OptAway');

       or:

        use Log::ger::Plugin 'OptAway';

       To set for current package only:

        use Log::ger::Plugin;
        Log::ger::Plugin->set_for_current_package('OptAway');
