NAME
       CGI::Test::Form::Widget::Menu::Popup - A popup menu


SYNOPSIS
        # Inherits from CGI::Test::Form::Widget::Menu
        # $form is a CGI::Test::Form

        my $action = $form->menu_by_name("action");
        $action->select("reboot");


DESCRIPTION
       This class models a popup menu, from which one item at most may be selected, and for which there is at
       least one item selected, i.e. where exactly one item is chosen.

       If no item was explicitely selected, "CGI::Test" arbitrarily chooses the first item in the popup (if not
       empty) and warns you via "warn".


INTERFACE
       The interface is the same as the one described in CGI::Test::Form::Widget::Menu.
