NAME
       CGI::Test::Form::Widget::Input::File - A file upload control


SYNOPSIS
        # Inherits from CGI::Test::Form::Widget::Input
        # $form is a CGI::Test::Form

        my $upload = $form->input_by_name("upload");
        $upload->replace("/tmp/file");


DESCRIPTION
       This class models a file upload control, which is a text field to enter a file name, with a little
       "browse" control button nearby that allows the user to select a file via a GUI...

       The interface is the same as the one described in CGI::Test::Form::Widget::Input::Text_Field.
