NAME
       CGI::Test::Form::Widget::Input::Text_Area - A text area


SYNOPSIS
        # Inherits from CGI::Test::Form::Widget::Input
        # $form is a CGI::Test::Form

        my $comments = $form->input_by_name("comments");
        $comments->append(<<EOM);
        --
        There's more than one way to do it.
            --Larry Wall
        EOM


DESCRIPTION
       This class models a text area, where users can type text.


INTERFACE
       The interface is the same as the one described in CGI::Test::Form::Widget::Input, with the following
       additional attributes:

       "columns"
           Amount of displayed columns.

       "rows"
           Amount of displayed text rows.

       "wrap_mode"
           The selected work wrapping mode.


BUGS
       Does  not handle "wrap_mode" and "columns" yet.  There is actually some work done by the browser when the
       wrapping mode is set to "hard", which alters the value transmitted back to the script upon submit.
