NAME
       array_cats0 - append C string to an array


SYNTAX
       #include <libowfat/array.h>

       void array_cats0(array* x,const char* y);

         array x;
         array_cats0(&x,"fnord");


DESCRIPTION
       array_cats0  appends  the contents of the 0-terminated string y, including the terminating 0 byte, to the
       array x.

       array_cats0 handles failure in the same way as array_cat.
