NAME
       buffer_feed - low-level component of buffer_get


SYNTAX
       #include <libowfat/buffer.h>

       int buffer_feed(buffer* b);


DESCRIPTION
       If  the  string  is  nonempty,  buffer_feed  returns  the  length of the string.  If the string is empty,
       buffer_feed uses the read operation to feed data into the string; it then returns the new length  of  the
       string, or 0 for end of input, or -1 for error.
