NAME
       MAPIQ - Terminates a string of calls to MAPIT.


SYNOPSIS
       CALL MAPIQ


C-BINDING SYNOPSIS
       #include <ncarg/ncargC.h>

       void c_mapiq()


DESCRIPTION
       MAPIQ has no arguments.


USAGE
       MAPIQ flushes MAPIT's buffers.  It is particularly important that this be done before a STOP or a CALL
       FRAME, and before changing the color index, dash pattern, etc.

       For more information, see the man page for MAPIT.


EXAMPLES
       Use the ncargex command to see the following relevant examples: cmpgci, cmptra, mpexfi.


ACCESS
       To use MAPIQ or c_mapiq, load the NCAR Graphics libraries ncarg, ncarg_gks, and ncarg_c, preferably in
       that order.
