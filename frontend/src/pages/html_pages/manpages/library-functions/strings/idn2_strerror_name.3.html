NAME
       idn2_strerror_name - API function


SYNOPSIS
       #include <idn2.h>

       const char * idn2_strerror_name(int rc);


ARGUMENTS
       int rc      return code from another libidn2 function.


DESCRIPTION
       Convert  internal  libidn2  error  code  to  a string corresponding to internal header file symbols.  For
       example, idn2_strerror_name(IDN2_MALLOC) will return the string "IDN2_MALLOC".

       The caller must not attempt to de-allocate the returned string.

       Return value: A string corresponding to error code symbol.
