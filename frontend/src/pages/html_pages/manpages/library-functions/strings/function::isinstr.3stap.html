NAME
       function::isinstr - Returns whether a string is a substring of another string


SYNOPSIS
           isinstr:long(s1:string,s2:string)


ARGUMENTS
       s1
           string to search in

       s2
           substring to find


DESCRIPTION
       This function returns 1 if string s1 contains s2, otherwise zero.
