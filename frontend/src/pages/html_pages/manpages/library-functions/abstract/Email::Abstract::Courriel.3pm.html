NAME
       Email::Abstract::Courriel - Email::Abstract wrapper for Courriel


VERSION
       version 0.49


DESCRIPTION
       This module wraps the Courriel mail handling library with an abstract interface, to be used with
       Email::Abstract.


SEE ALSO
       Email::Abstract, Courriel.
