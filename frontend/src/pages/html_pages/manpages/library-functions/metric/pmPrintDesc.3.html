NAME
       pmPrintDesc - print a metric descriptor


C SYNOPSIS
       #include <pcp/pmapi.h>

       void pmPrintDesc(FILE *f, const pmDesc *desc);

       cc ... -lpcp


DESCRIPTION
       Display all the fields of the Performance Metrics Descriptor desc on the output stream f.


SEE ALSO
       PMAPI(3) and pmLookupDesc(3).
