Detailed Description
   template<typename _Tp, typename _Alloc>
       class std::_Deque_base< _Tp, _Alloc >"Deque base class. This class provides the unified face for deque's
       allocation. This class's constructor and destructor allocate and deallocate (but do not initialize)
       storage. This makes exception safety easier.

       Nothing in this class ever constructs or destroys an actual Tp element. (Deque handles that itself.)
       Only/All memory management is performed here.
