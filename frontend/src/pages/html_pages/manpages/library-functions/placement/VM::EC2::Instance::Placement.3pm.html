NAME
       VM::EC2::Instance::Placement - Object describing an Amazon EC2 instance's placement


SYNOPSIS
         use VM::EC2;

         $ec2     = VM::EC2->new(...);


DESCRIPTION
       This object represents the placement of an instance in an Amazon EC2 availability region and cluster
       group. It is returned by VM::EC2->describe_instances


METHODS
       These object methods are supported:

         availabilityZone   -- The availability zone of the instance.
         groupName          -- The name of the placement group in a
                               cluster computing instance.
         tenancy            -- Tenancy of the instance, e.g. "dedicated".


STRING OVERLOADING
       When used in a string context, this object will interpolate the availabilityZone.
