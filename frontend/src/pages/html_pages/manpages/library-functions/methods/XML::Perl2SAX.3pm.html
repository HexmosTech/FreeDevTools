NAME
       XML::SAX2Perl -- translate Perl SAX methods to Java/CORBA style methods


SYNOPSIS
        use XML::Perl2SAX;

        $perl2sax = XML::Perl2SAX(handler => $java_style_handler);


DESCRIPTION
       "XML::Perl2SAX" is a SAX filter that translates Perl style SAX methods to Java/CORBA style method calls.
       This module performs the inverse operation from "XML::SAX2Perl".

       "Perl2SAX" is a Perl SAX document handler.  The `"new"' method takes a `"handler"' argument that is a
       Java/CORBA style handler that the new Perl2SAX instance will call.  The SAX interfaces are defined at
       <http://www.megginson.com/SAX/>.
