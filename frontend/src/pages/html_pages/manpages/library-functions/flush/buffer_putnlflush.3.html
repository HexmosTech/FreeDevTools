NAME
       buffer_putnlflush - write '\n' to buffer and flush


SYNTAX
       #include <libowfat/buffer.h>

       int buffer_putnlflush(buffer* b);


DESCRIPTION
       buffer_putnlflush writes a single ASCII new-line character ('\n') to b and calls buffer_flush(b).


RETURN VALUE
       0 if everything is fine, -1 on error (setting errno).
