NAME
       shishi_prompt_password - API function


SYNOPSIS
       #include <shishi.h>

       int shishi_prompt_password(Shishi * handle, char ** s, const char * format, ...);


ARGUMENTS
       Shishi * handle
                   shishi handle as allocated by shishi_init().

       char ** s   pointer to newly allocated output string with read password.

       const char * format
                   printf(3) style format string.

       ...         printf(3) style arguments.


DESCRIPTION
       Format  and  print  a  prompt,  and read a password from user.  The password is possibly converted (e.g.,
       converted from Latin-1 to UTF-8, or processed using Stringprep  profile)  following  any  "stringprocess"
       keywords in configuration files.


RETURN VALUE
       Returns SHISHI_OK iff successful.
