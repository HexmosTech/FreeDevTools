NAME
       shishi_keys_to_keytab_mem - API function


SYNOPSIS
       #include <shishi.h>

       int shishi_keys_to_keytab_mem(Shishi * handle, Shishi_keys * keys, char ** out, size_t * len);


ARGUMENTS
       Shishi * handle
                   shishi handle as allocated by shishi_init().

       Shishi_keys * keys
                   key set to convert to keytab format.

       char ** out constant memory buffer with keytab of len size.

       size_t * len
                   size of memory buffer with keytab data.


DESCRIPTION
       Write keys to a MIT keytab data structure.

       The  format  of keytab's is proprietary, and this function writes the 0x0502 format.  See the section The
       MIT Kerberos Keytab Binary File Format in the Shishi manual for a description of  the  reverse-engineered
       format.


RETURNS
       On success SHISHI_OK is returned, otherwise an error code.
