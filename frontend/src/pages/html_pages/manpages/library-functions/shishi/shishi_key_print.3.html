NAME
       shishi_key_print - API function


SYNOPSIS
       #include <shishi.h>

       int shishi_key_print(Shishi * handle, FILE * fh, const Shishi_key * key);


ARGUMENTS
       Shishi * handle
                   Shishi handle as allocated by shishi_init().

       FILE * fh   File handle open for writing.

       const Shishi_key * key
                   Key to print.


DESCRIPTION
       Prints an ASCII representation of a key structure key to the file descriptor fh.  Example output:


       -----BEGIN SHISHI KEY-----

KEYTYPE
       18 (aes256-cts-hmac-sha1-96)


PRINCIPAL
       host/latte.josefsson.org


REALM
       JOSEFSSON.ORG Key-Version-Number: 1


TIMESTAMP
       20130420150337Z


       P1QdeW/oSiag/bTyVEBAY2msiGSTmgLXlopuCKoppDs= -----END SHISHI KEY-----

RETURN VALUE
       Returns SHISHI_OK if successful.  The only failure is SHISHI_MALLOC_ERROR.
