NAME
       shishi_krberror_read - API function


SYNOPSIS
       #include <shishi.h>

       int shishi_krberror_read(Shishi * handle, FILE * fh, Shishi_asn1 * krberror);


ARGUMENTS
       Shishi * handle
                   shishi handle as allocated by shishi_init().

       FILE * fh   file handle open for reading.

       Shishi_asn1 * krberror
                   output variable with newly allocated KRB-ERROR.


DESCRIPTION
       Read DER encoded KRB-ERROR from file and populate given variable.


RETURN VALUE
       Returns SHISHI_OK iff successful.
