NAME
       shishi_kdcreq_renewable_p - API function


SYNOPSIS
       #include <shishi.h>

       int shishi_kdcreq_renewable_p(Shishi * handle, Shishi_asn1 kdcreq);


ARGUMENTS
       Shishi * handle
                   shishi handle as allocated by shishi_init().

       Shishi_asn1 kdcreq
                   KDC-REQ variable to get kdc-options field from.


DESCRIPTION
       Determine if KDC-Option renewable flag is set.

       The  RENEWABLE  option  indicates  that the ticket to be issued is to have its RENEWABLE flag set. It may
       only be set on the initial request, or when the ticket-granting ticket on which the request is  based  is
       also  renewable.  If  this  option is requested, then the rtime field in the request contains the desired
       absolute expiration time for the ticket.


RETURN VALUE
       Returns non-0 iff renewable flag is set in KDC-REQ.
