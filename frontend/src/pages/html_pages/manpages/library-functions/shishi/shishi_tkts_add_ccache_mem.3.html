NAME
       shishi_tkts_add_ccache_mem - API function


SYNOPSIS
       #include <shishi.h>

       int shishi_tkts_add_ccache_mem(Shishi * handle, const char * data, size_t len, Shishi_tkts * tkts);


ARGUMENTS
       Shishi * handle
                   shishi handle as allocated by shishi_init().

       const char * data
                   constant memory buffer with ccache of len size.

       size_t len  size of memory buffer with ccache data.

       Shishi_tkts * tkts
                   allocated key set to store tickets in.


DESCRIPTION
       Read tickets from a ccache data structure, and add them to the ticket set.

       The  ccache  format  is  proprietary,  and  this  function support (at least) the 0x0504 format.  See the
       section The Credential Cache Binary File Format in the Shishi  manual  for  a  description  of  the  file
       format.


RETURNS
       Returns  SHISHI_CCACHE_ERROR  if  the  data does not represent a valid ccache structure, and SHISHI_OK on
       success.
