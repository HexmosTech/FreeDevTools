NAME
       shishi_authenticator_print - API function


SYNOPSIS
       #include <shishi.h>

       int shishi_authenticator_print(Shishi * handle, FILE * fh, Shishi_asn1 authenticator);


ARGUMENTS
       Shishi * handle
                   shishi handle as allocated by shishi_init().

       FILE * fh   file handle open for writing.

       Shishi_asn1 authenticator
                   authenticator as allocated by shishi_authenticator().


DESCRIPTION
       Print ASCII armored DER encoding of authenticator to file.


RETURN VALUE
       Returns SHISHI_OK iff successful.
