NAME
       ck_bitmap_test — determine if the bit at the specified index is set


LIBRARY
       Concurrency Kit (libck, -lck)


SYNOPSIS
       #include <ck_bitmap.h>

       bool
       ck_bitmap_test(ck_bitmap_t *bitmap, unsigned int n);


DESCRIPTION
       The ck_bitmap_test() determines if the bit at the offset specified by the argument n is set to 1.


RETURN VALUES
       This function returns true if the bit at the specified offset is set to 1 and otherwise returns false.
