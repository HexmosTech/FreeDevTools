NAME
       Pod::Abstract::Filter::unoverlay - paf command to remove "overlay" blocks from a Pod document, as created
       by the paf overlay command.


METHODS
   new
   filter
       Strips any sections marked "=for overlay" from the listed overlay specification from the target document.
       This will expunge everything that has been previously overlaid or marked for overlay from the specified
       documents.
