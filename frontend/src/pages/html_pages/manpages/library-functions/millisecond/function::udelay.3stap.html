NAME
       function::udelay - microsecond delay


SYNOPSIS
           udelay(us:long)


ARGUMENTS
       us
           Number of microseconds to delay.


DESCRIPTION
       This function inserts a multi-microsecond busy-delay into a probe handler. It requires guru mode.
