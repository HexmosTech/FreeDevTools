NAME
       Supported_Image_Formats - Supported Image Formats


DESCRIPTION
       For read-write or write-only 2D and 3D images, the minimum list of supported image formats is given by
       the table below:

       │            image_num_channels │            image_channel_order │            image_channel_data_type │
       ├───────────────────────────────┼────────────────────────────────┼────────────────────────────────────┤
       │ 4                             │ CL_RGBA                        │            CL_UNORM_INT8           │
       │                               │                                │            CL_UNORM_INT16          │
       │                               │                                │                                    │
       │                               │                                │            CL_SIGNED_INT8          │
       │                               │                                │            CL_SIGNED_INT16         │
       │                               │                                │            CL_SIGNED_INT32         │
       │                               │                                │                                    │
       │                               │                                │            CL_UNSIGNED_INT8        │
       │                               │                                │            CL_UNSIGNED_INT16       │
       │                               │                                │            CL_UNSIGNED_INT32       │
       │                               │                                │                                    │
       │                               │                                │            CL_HALF_FLOAT           │
       │                               │                                │            CL_FLOAT                │
