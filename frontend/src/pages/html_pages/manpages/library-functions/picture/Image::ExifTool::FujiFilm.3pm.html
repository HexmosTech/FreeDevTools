NAME
       Image::ExifTool::FujiFilm - Read/write FujiFilm maker notes and RAF images


SYNOPSIS
       This module is loaded automatically by Image::ExifTool when required.


DESCRIPTION
       This module contains definitions required by Image::ExifTool to interpret FujiFilm maker notes in EXIF
       information, and to read/write FujiFilm RAW (RAF) images.
