NAME
       Image::ExifTool::MRC - Read MRC meta information


SYNOPSIS
       This module is used by Image::ExifTool


DESCRIPTION
       This module contains definitions required by Image::ExifTool to read metadata from Medical Research
       Council (MRC) images.
