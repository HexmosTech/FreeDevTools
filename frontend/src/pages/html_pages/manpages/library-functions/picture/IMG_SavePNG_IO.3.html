NAME
       IMG_SavePNG_IO - Save an SDL_Surface into PNG image data, via an SDL_IOStream.


HEADER FILE
       Defined in SDL3_image/SDL_image.h


SYNOPSIS
       #include "SDL3_image/SDL_image.h"

       bool IMG_SavePNG_IO(SDL_Surface *surface, SDL_IOStream *dst, bool closeio);


DESCRIPTION
       If you just want to save to a filename, you can use

       IMG_SavePNG () instead.

       If closeio is true, dst will be closed before returning, whether this function succeeds or not.


FUNCTION PARAMETERS
       surface
              the SDL surface to save.

       dst    the SDL_IOStream to save the image data to.

       closeio
              true to close/free the SDL_IOStream before returning, false to leave it open.


RETURN VALUE
       Returns true on success or false on failure; call SDL_GetError() for more information.


AVAILABILITY
       This function is available since SDL_image 3.0.0.
