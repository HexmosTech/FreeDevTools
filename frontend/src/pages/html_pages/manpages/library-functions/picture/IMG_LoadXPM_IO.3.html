NAME
       IMG_LoadXPM_IO - Load a XPM image directly.


HEADER FILE
       Defined in SDL3_image/SDL_image.h


SYNOPSIS
       #include "SDL3_image/SDL_image.h"

       SDL_Surface * IMG_LoadXPM_IO(SDL_IOStream *src);


DESCRIPTION
       If you know you definitely have a XPM image, you can call this function, which will skip SDL_image's file
       format  detection  routines. Generally it's better to use the abstract interfaces; also, there is only an
       SDL_IOStream interface available here.


FUNCTION PARAMETERS
       src    an SDL_IOStream to load image data from.


RETURN VALUE
       Returns SDL surface, or NULL on error.


AVAILABILITY
       This function is available since SDL_image 3.0.0.
