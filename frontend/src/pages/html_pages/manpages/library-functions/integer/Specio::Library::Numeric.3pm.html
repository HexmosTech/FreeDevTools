NAME
       Specio::Library::Numeric - Implements type constraint objects for some common numeric types


VERSION
       version 0.50


DESCRIPTION
       This library provides some additional string numeric for common cases.

   PositiveNum
   PositiveOrZeroNum
   PositiveInt
   PositiveOrZeroInt
   NegativeNum
   NegativeOrZeroNum
   NegativeInt
   NegativeOrZeroInt
   SingleDigit
       A single digit from -9 to 9.
