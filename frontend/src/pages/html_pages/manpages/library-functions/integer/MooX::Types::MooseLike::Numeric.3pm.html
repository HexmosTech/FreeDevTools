NAME
       MooX::Types::MooseLike::Numeric - Moo types for numbers


SYNOPSIS
         package MyPackage;
         use Moo;
         use MooX::Types::MooseLike::Numeric qw(PositiveInt);

         has "daily_breathes" => (
           is  => 'rw',
           isa => PositiveInt
         );


DESCRIPTION
       A set of numeric types to be used in Moo-based classes. Adapted from MooseX::Types::Common::Numeric


TYPES (subroutines)
       Available types are listed below.

       PositiveNum
       PositiveOrZeroNum
       PositiveInt
       PositiveOrZeroInt
       NegativeNum
       NegativeOrZeroNum
       NegativeInt
       NegativeOrZeroInt
       SingleDigit
