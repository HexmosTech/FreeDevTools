NAME
       SDL_SINT64_C - Append the 64 bit integer suffix to a signed integer literal.


HEADER FILE
       Defined in SDL3/SDL_stdinc.h


SYNOPSIS
       #include "SDL3/SDL.h"

       #define SDL_SINT64_C(c)  c ## LL  /* or whatever the current compiler uses. */


DESCRIPTION
       This  helps compilers that might believe a integer literal larger than 0xFFFFFFFF is overflowing a 32-bit
       value. Use SDL_SINT64_C(0xFFFFFFFF1) instead of 0xFFFFFFFF1 by itself.


AVAILABILITY
       This macro is available since SDL 3.2.0.
