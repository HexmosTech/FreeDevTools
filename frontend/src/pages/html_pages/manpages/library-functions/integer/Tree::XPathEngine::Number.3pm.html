NAME
       Tree::XPathEngine::Number - Simple numeric values.


DESCRIPTION
       This class holds simple numeric values. It doesn't support -0, +/- Infinity, or NaN, as the XPath spec
       says it should, but I'm not hurting anyone I don't think.


API
   new($num)
       Creates a new Tree::XPathEngine::Number object, with the value in $num. Does some rudimentary numeric
       checking on $num to ensure it actually is a number.

   value()
       Also as overloaded stringification. Returns the numeric value held.

   Other Methods
       Those are needed so the objects can be properly processed in various contexts

       as_string
       as_xml
       value
       xpath_cmp
       evaluate
       xpath_to_boolean
       xpath_to_literal
       xpath_to_number
       xpath_string_value
