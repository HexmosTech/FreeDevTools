NAME
       ESYS_TR - Esys Tpm Resource ESYS_TR


SYNOPSIS
   Modules
       Global ESYS_TR objects


Detailed Description
       Virtual objects with meta data for Tpm Resources. The Esys TPM Resources are the virtual objects inside
       the ESYSCONTEXT that hold the metadata for corresponding TPM Resources.

       There are a set of global ESYS_TR objects such as ESYS_TR_NON or for the various PCRs and hierarchies.
       ESYS_TR_MIN_OBJECT denotes the first non-global ESYS_TR object.
