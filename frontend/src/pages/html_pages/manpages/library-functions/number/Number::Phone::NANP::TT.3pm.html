NAME
       Number::Phone::NANP::TT - TT-specific methods for Number::Phone


DESCRIPTION
       This class implements TT-specific methods for Number::Phone.  It is a subclass of Number::Phone::NANP,
       which is in turn a subclass of Number::Phone.  Number::Phone::NANP sits in the middle because all NANP
       countries can share some significant chunks of code.  You should never need to "use" this module
       directly, as "Number::Phone::NANP" will load it automatically when needed.


SYNOPSIS
           use Number::Phone::NANP;

           my $phone_number = Number::Phone->new('+1 868 624 6982');
           # returns a Number::Phone::NANP::TT object


METHODS
       The following methods from Number::Phone are overridden:

       regulator
           Returns information about the national telecoms regulator.
