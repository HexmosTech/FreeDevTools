NAME

       __gnu_cxx::__mt_alloc< _Tp, _Poolp > - This is a fixed size (power of 2) allocator which - when compiled
       with thread support - will maintain one freelist per size per thread plus a global one. Steps are taken
       to limit the per thread freelist sizes (by returning excess back to the global list).
