NAME
       "LaTeXML::Core::Mouth::Binding" - a fake Mouth for processing a Binding file


DESCRIPTION
       This is a fake mouth, used for processing binding files (ie. "*.ltxml" and "*.latexml"). It exists
       primarily for the purposes of (1) getting a locator on anything defined in the file and (2) serving as a
       placekeeper in the chain of Mouth's in a Gullet, when the binding reads in a proper TeX file.
