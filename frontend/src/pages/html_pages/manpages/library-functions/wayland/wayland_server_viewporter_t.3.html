Detailed Description
       surface cropping and scaling

       The global interface exposing surface cropping and scaling capabilities is used to instantiate an
       interface extension for a wl_surface object. This extended interface will then allow cropping and scaling
       the surface contents, effectively disconnecting the direct relationship between the buffer and the
       surface size.
