NAME
       "Tickit::Widget::Fill" - fill an area with repeated text


DESCRIPTION
       This class provides a widget which displays a given piece of text repeatedly over its entire area. By
       default the text is a single space, meaning the area will be entirely drawn with spaces.


STYLE
       The default style pen is used as the widget pen.

       The following style keys are used:

       text => STRING
           The text to display in a repeating pattern on the window.

       skew => INT
           If  defined,  successive  lines will be advanced by this number of columns (which may be negative) to
           create a skewed repeating pattern.
