NAME
       iob_addfile_close - add file region to I/O batch


SYNTAX
       #include <libowfat/iob.h>

       int iob_addfile_close(io_batch* b,int64 fd,uint64 off,uint64 n);


DESCRIPTION
       iob_addfile_close schedules n bytes from the file descriptor fd starting at offset off to be written with
       the I/O batch b.

       You may not close fd before iob_send indicates it is done.

       iob_reset will call io_close(fd).


RETURN VALUE
       Returns 1 on success, 0 on failure.  In the failure case, the file descriptor has been closed.
