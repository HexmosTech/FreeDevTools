NAME
       Attean::API::Quad - Role representing quads


VERSION
       This document describes Attean::API::Quad version 0.035


SYNOPSIS
         use v5.14;
         use Attean;


DESCRIPTION
       This is a Moo role representing quads.


ROLES
       This role consumes Attean::API::QuadPattern, Attean::API::TripleOrQuad, Attean::API::QuadOrQuadPattern
       and Attean::API::Binding, which provide the following methods:

       "mapping"
       "variables"
       "values"
       "value"
       "tuples_string"
       "as_string"
       "apply_map"


METHODS
       "subject"
       "predicate"
       "object"
       "graph"
