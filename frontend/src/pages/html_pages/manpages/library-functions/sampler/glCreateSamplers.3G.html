NAME
       glCreateSamplers - create sampler objects


C SPECIFICATION

       void glCreateSamplers(GLsizei n, GLuint *samplers);


PARAMETERS
       n
           Number of sampler objects to create.

       samplers
           Specifies an array in which names of the new sampler objects are stored.


DESCRIPTION
       glCreateSamplers returns n previously unused sampler names in samplers, each representing a new sampler
       object initialized to the default state.


ERRORS
       GL_INVALID_VALUE is generated if n is negative.
