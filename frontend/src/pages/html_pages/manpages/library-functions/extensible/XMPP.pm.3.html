FUNCTIONS
   Configuration($appli)
       Returns the XMPP configuration

   Configured($appli)
       Returns '1' if XMPP is configured (server & port) else '0'

   Connection_Test($appli)
       Checks the XMPP Connection

   Send_Message($appli, $msg, @dests)
       Sends message '$msg' to '@dests' through XMPP
