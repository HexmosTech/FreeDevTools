NAME
       Dancer2::Core::DSL - Dancer2's Domain Specific Language (DSL)


VERSION
       version 1.1.2


FUNCTIONS
   setting
       Lets you define settings and access them:

           setting('foo' => 42);
           setting('foo' => 42, 'bar' => 43);
           my $foo=setting('foo');

       If settings were defined returns number of settings.

   set ()
       alias for setting:

           set('foo' => '42');
           my $port=set('port');
