NAME
       CHI::Driver::Memcached::Fast -- Distributed cache via Cache::Memcached::Fast


VERSION
       version 0.16


SYNOPSIS
           use CHI;

           my $cache = CHI->new(
               driver => 'Memcached::Fast',
               namespace => 'products',
               servers => [ "10.0.0.15:11211", "10.0.0.15:11212", "/var/sock/memcached",
               "10.0.0.17:11211", [ "10.0.0.17:11211", 3 ] ],
               debug => 0,
               compress_threshold => 10_000,
           );


DESCRIPTION
       A CHI driver that uses Cache::Memcached::Fast to store data in the specified memcached server(s). From
       the perspective of the CHI API, the feature set is nearly identical to CHI::Driver::Memcached.
