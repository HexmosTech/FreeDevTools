NAME
       Plack::Handler::Twiggy - Adapter for Twiggy


SYNOPSIS
         plackup -s Twiggy --port 9090

         # with start_server
         start_server --port=9090 plackup -s Twiggy


DESCRIPTION
       This is an adapter to run PSGI apps on Twiggy via plackup.
