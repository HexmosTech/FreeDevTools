NAME
       Data::UUID::LibUUID::DataUUIDCompat - Drop in Data::UUID replacement


SYNOPSIS

           my $uuid_gen = Data::UUID::LibUUID::DataUUIDCompat->new; # Data::UUID->new;

           my $bin_uuid = $uuid_gen->create;


DESCRIPTION
       See Data::UUID for the API.
