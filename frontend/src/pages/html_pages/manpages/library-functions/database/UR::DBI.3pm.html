NAME
       UR::DBI - methods for interacting with a database.


SYNOPSIS
           ##- use UR::DBI;
           UR::DBI->monitor_sql(1);
           my $dbh = UR::DBI->connect(...);


DESCRIPTION
       This module subclasses DBI, and provides a few extra methods useful when using a database.
