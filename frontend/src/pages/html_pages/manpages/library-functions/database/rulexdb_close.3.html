NAME
       rulexdb_close - close a rulex database


SYNOPSIS
       #include <rulexdb.h>

       void rulexdb_open(RULEXDB *rulexdb);


DESCRIPTION
       The  rulexdb_close()  function closes the database referenced by its argument, flushes all cached data if
       any, frees all associated resources and closes the underlying file.
