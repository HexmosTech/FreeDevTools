NAME
       MooseX::MultiInitArg::Trait - A composable role to add multiple init arguments to your attributes.


DESCRIPTION
       This is a composable trait which you can add to an attribute so that you can specify a list of aliases
       for your attribute to be recognized as constructor arguments.
