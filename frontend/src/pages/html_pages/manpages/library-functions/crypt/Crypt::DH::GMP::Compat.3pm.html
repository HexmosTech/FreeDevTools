NAME
       Crypt::DH::GMP::Compat - Compatibility Mode For Crypt::DH


SYNOPSIS
         use Crypt::DH;
         use Crypt::DH::GMP qw(-compat);


DESCRIPTION
       Crypt::DH::GMP::Compat is a very invasive module that rewrites Crypt::DH's @ISA and method names so that
       it uses Crypt::DH::GMP
