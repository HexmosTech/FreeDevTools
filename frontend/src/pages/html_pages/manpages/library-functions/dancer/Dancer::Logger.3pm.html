NAME
       Dancer::Logger - common interface for logging in Dancer


VERSION
       version 1.3521


DESCRIPTION
       This module is the wrapper that provides support for different logger engines.


USAGE
   Default engine
       The setting logger defines which logger engine to use.  If this setting is not set, logging will not be
       available in the application code.

       Dancer comes with the logger engines Dancer::Logger::File and Dancer::Logger::Console, but more are
       available on the CPAN.

   Configuration
       The logger configuration variable tells Dancer which engine to use.

       You can change it either in your config.yml file:

           # logging to console
           logger: "console"

       Or in the application code:

           # logging to file
           set logger => 'file';

       The log format can also be configured, please see "logger_format" in Dancer::Logger::Abstract for
       details.

   Auto-serializing
       The loggers allow auto-serializing of all inputs:

           debug( 'User credentials: ', \%creds );

       Will provide you with an output in a single log message of the string and the reference dump.
