NAME
       XML::SAX::ExpatXS::Encoding - Encoding support for XML::SAX::ExpatXS


DESCRIPTION
       This module is derived from XML::Parser::Expat. It provides XML::SAX::ExpatXS parser with support of not-
       built-in encodings.
