NAME
       Test2::Tools::Encoding - Tools for managing the encoding of Test2 based tests.


DESCRIPTION
       This module exports a function that lets you dynamically change the output encoding at will.


SYNOPSIS
           use Test2::Tools::Encoding;

           set_encoding('utf8');


EXPORTS
       All subs are exported by default.

       set_encoding($encoding)
           This  will  set the encoding to whatever you specify. This will only affect the output of the current
           formatter, which is usually your TAP output formatter.
