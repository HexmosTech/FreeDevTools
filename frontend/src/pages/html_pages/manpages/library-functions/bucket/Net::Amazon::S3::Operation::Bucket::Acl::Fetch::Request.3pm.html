NAME
       Net::Amazon::S3::Operation::Bucket::Acl::Fetch::Request - An internal class to get a bucket's access
       control


VERSION
       version 0.991


SYNOPSIS
               my $http_request = Net::Amazon::S3::Operation::Bucket::Acl::Fetch::Request->new (
                       s3     => $s3,
                       bucket => $bucket,
               );


DESCRIPTION
       Implements operation GetBucketAcl <https://docs.aws.amazon.com/AmazonS3/latest/API/API_GetBucketAcl.html>

       This module gets a bucket's access control.


METHODS
   http_request
       This method returns a HTTP::Request object.
