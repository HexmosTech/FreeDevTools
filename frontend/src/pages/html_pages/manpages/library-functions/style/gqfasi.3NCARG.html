NAME
       GQFASI (Inquire fill area style index) - retrieves the current value of the fill area style index.


SYNOPSIS
       CALL GQFASI (ERRIND, STYLI)


C-BINDING SYNOPSIS
       #include <ncarg/gks.h>

       void ginq_fill_style_ind(Gint *err_ind, Gint *fill_style_ind);


DESCRIPTION
       ERRIND      (Integer,  Output)  -  If  the  inquired value cannot be returned correctly, a non-zero error
                   indicator is returned in ERRIND, otherwise a zero is returned.   Consult  "User's  Guide  for
                   NCAR GKS-0A Graphics" for a description of the meaning of the error indicators.

       STYLI       (Integer,  Output)  -  Returns  the  fill  area style index as set by default or by a call to
                   GSFASI.

                   1      Horizontal lines (default)

                   2      Vertical lines

                   3      Lines of positive slope

                   4      Lines of negative slope

                   5      Horizontal and vertical lines

                   6      Lines of positive and negative slope


ACCESS
       To use GKS routines, load the NCAR GKS-0A library ncarg_gks.
