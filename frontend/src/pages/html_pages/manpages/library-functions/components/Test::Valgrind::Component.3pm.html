NAME
       Test::Valgrind::Component - Base class for Test::Valgrind components.


VERSION
       Version 1.19


DESCRIPTION
       This class is the base for all others that act as components that can be started and stopped.


METHODS
   "new"
           my $tvc = Test::Valgrind::Component->new;

       Basic constructor.

   "started"
           $tvc->started($bool);

       Specifies whether the component is running (1), stopped (0) or was never started ("undef").

   "start"
           $tvc->start;

       Marks the component as started, and throws an exception if it was already.  Returns its self object.

   "finish"
           $tvc->finish;

       Marks the component as stopped, and throws an exception if it wasn't started.  Returns its self object.
