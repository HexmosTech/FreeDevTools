NAME
       DBIx::Class::Schema::Loader::DBI::Component::QuotedDefault -- Loader::DBI Component to parse quoted
       default constants and functions


DESCRIPTION
       If "COLUMN_DEF" from "column_info" in DBI returns character constants quoted, then we need to remove the
       quotes. This also allows distinguishing between default functions without information schema
       introspection.
