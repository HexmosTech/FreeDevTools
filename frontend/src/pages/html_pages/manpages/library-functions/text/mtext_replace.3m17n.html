NAME
       mtext_replace -


SYNOPSIS
       int mtext_replace (MText * mt1, int from1, int to1, MText * mt2, int from2, int to2)


DESCRIPTION
       Replace sub-text of M-text with another. The mtext_replace() function replaces sub-text of M-text mt1
       between from1 (inclusive) and to1 (exclusive) with the sub-text of M-text mt2 between from2 (inclusive)
       and to2 (exclusive). The new sub-text inherits text properties of the old sub-text.


RETURN VALUE
           If the operation was successful, mtext_replace() returns 0. Otherwise, it returns -1 and assigns an
           error code to the external variable merror_code.


ERRORS
           MERROR_MTEXT, MERROR_RANGE
