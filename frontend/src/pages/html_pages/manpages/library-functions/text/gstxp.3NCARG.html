NAME
       GSTXP (Set text path) - sets the text paths or directions in which text is to be drawn.


SYNOPSIS
       CALL GSTXP (TXP)


C-BINDING SYNOPSIS
       #include <ncarg/gks.h>

       void gset_text_path(Gtext_path text_path);


DESCRIPTION
       TXP        (Integer,  Input)  -  Gives the direction in which a character string is to be drawn.  Options
                  are:

                  0  Draw successive characters in the string such that character n+l appears to  the  right  of
                     character n. This is text path "right" and is the default.

                  1  Draw character n+1 to the left of character n.  This is text path "left".

                  2  Draw character n+1 above character n.  This is text path "up".

                  3  Draw character n+1 below character n.  This is text path "down".

                  The  right,  left,  up, and down directions are relative to the character up vector. The right
                  text path direction is perpendicular to the up vector direction.  Thus, to draw a text  string
                  at a 45 degree angle the character up vector would be (-1,1), and the text path would be right
                  .


ACCESS
       To use GKS routines, load the NCAR GKS-0A library ncarg_gks.
