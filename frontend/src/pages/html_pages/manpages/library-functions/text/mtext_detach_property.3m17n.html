NAME
       mtext_detach_property -


SYNOPSIS
       int mtext_detach_property (MTextProperty * prop)


DESCRIPTION
       Detach a text property from an M-text. The mtext_detach_property() function makes text property prop
       detached.


RETURN VALUE
           This function always returns 0.
