NAME
       mtext_property_mtext -


SYNOPSIS
       MText* mtext_property_mtext (MTextProperty * prop)


DESCRIPTION
       Return the M-text of a text property. The mtext_property_mtext() function returns the M-text to which
       text property prop is attached. If prop is currently detached, NULL is returned.
