NAME
       TTF_SetTextScript - Set the script to be used for text shaping a text object.


HEADER FILE
       Defined in SDL3_ttf/SDL_ttf.h


SYNOPSIS
       #include "SDL3_ttf/SDL_ttf.h"

       bool TTF_SetTextScript(TTF_Text *text, Uint32 script);


DESCRIPTION
       This returns false if SDL_ttf isn't built with HarfBuzz support.


FUNCTION PARAMETERS
       text   the text to modify.

       script an ISO 15924 code ⟨https://unicode.www:lenstrwww:splitstr:nwww:splitstr:r[char46]org/iso15924/
              iso15924-codes.www:lenstrwww:splitstr:nwww:splitstr:r[char46]html⟩
               .


RETURN VALUE
       Returns true on success or false on failure; call SDL_GetError() for more information.


THREAD SAFETY
       This function should be called on the thread that created the text.


AVAILABILITY
       This function is available since SDL_ttf 3.0.0.
