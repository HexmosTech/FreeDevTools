NAME
       function::task_fd_lookup - get the file struct for a task's fd


SYNOPSIS
           task_fd_lookup:long(task:long,fd:long)


ARGUMENTS
       task
           task_struct pointer.

       fd
           file descriptor number.


DESCRIPTION
       Returns the file struct pointer for a task's file descriptor.
