NAME
       Biblio::Citation::Parser::Templates - citation templates collection


DESCRIPTION
       This package contains a list of templates that are used to parse citations. Updates will be made
       available at http://paracite.eprints.org/developers/ (hopefully with a merging tool to ensure you don't
       lose any changes). To add new fields, edit Standard.pm.
