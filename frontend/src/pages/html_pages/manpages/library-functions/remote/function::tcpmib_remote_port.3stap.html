NAME
       function::tcpmib_remote_port - Get the remote port


SYNOPSIS
           tcpmib_remote_port:long(sk:long)


ARGUMENTS
       sk
           pointer to a struct inet_sock


DESCRIPTION
       Returns the dport from a struct inet_sock in host order.
