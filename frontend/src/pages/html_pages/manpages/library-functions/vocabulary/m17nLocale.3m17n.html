Detailed Description
       Locale objects and API for them.

       The m17n library represents locale related information as objects of type MLocale.


Typedef Documentation
   typedef struct MLocale MLocale
       struct MLocale. The structure MLocale is used to hold information about name, language, territory,
       modifier, codeset, and the corresponding coding system of locales.

       The contents of this structure are implementation dependent. Its internal structure is concealed from
       application programs.
