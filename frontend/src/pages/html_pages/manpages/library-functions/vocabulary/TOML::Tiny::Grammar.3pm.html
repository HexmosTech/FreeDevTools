NAME
       TOML::Tiny::Grammar - exports regex definitions used to parse TOML source


VERSION
       version 0.20


SYNOPSIS
         use TOML::Tiny::Grammar;

         if ($src =~ /$MultiLineString/) {
           ...
         }


DESCRIPTION
       Exports various regexex for parsing TOML source.


PATTERNS
   White space and ignorables
       $WS

       $CRLF

       $EOL

       $Comment

   Keys
       $BareKey

       $QuotedKey

       $SimpleKey

       $DottedKey

       $Key

   Values
       $Boolean

       $Escape

       $StringLiteral

       $MultiLineStringLiteral

       $BasicString

       $MultiLineString

       $String

       $Date

       $Time

       $DateTime

       $Hex

       $Oct

       $Bin

       $Dec

       $Integer

       $Float

   $SpecialFloat
