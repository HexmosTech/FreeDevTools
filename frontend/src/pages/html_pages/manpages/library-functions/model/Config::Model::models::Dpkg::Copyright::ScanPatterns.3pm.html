NAME
       Config::Model::models::Dpkg::Copyright::ScanPatterns - Configuration class Dpkg::Copyright::ScanPatterns


DESCRIPTION
       Configuration classes used by Config::Model


Elements
   check
       Files matching any of these patterns will be scanned by licensecheck. See "electing or ignoring files to
       scan" in Dpkg::Copyright::Scanner  Optional. Type node of class Dpkg::Copyright::ScanPatterns::Lists .

   ignore
       Files matching any of these patterns will be ignored by licensecheck. See "electing or ignoring files to
       scan" in Dpkg::Copyright::Scanner  Optional. Type node of class Dpkg::Copyright::ScanPatterns::Lists .
