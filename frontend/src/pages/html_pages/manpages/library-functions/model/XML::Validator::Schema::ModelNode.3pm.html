NAME
       XML:Validator::Schema::ModelNode - content model


DESCRIPTION
       Objects of this class represent the content models encountered while parsing a schema.  After a model is
       completely parsed it is compiled into a regular expression and a human-readbale description and assigned
       to the element or complex type's 'model' attribute.
