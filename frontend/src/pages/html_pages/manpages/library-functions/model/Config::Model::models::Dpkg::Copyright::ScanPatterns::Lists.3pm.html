NAME
       Config::Model::models::Dpkg::Copyright::ScanPatterns::Lists - Configuration class
       Dpkg::Copyright::ScanPatterns::Lists


DESCRIPTION
       Configuration classes used by Config::Model


Elements
   suffixes
       Files matching any of these suffixes will be scanned by licensecheck. Do not specify the dot with the
       suffixes (e.g. enter "jpg" and not ".jpg"). Default suffixes of licensecheck will also be used. See
       "Selecting or ignoring files to scan" in Dpkg::Copyright::Scanner  Optional. Type list of uniline.

   pattern
       Files matching any of these patterns will be scanned by licensecheck. See "electing or ignoring files to
       scan" in Dpkg::Copyright::Scanner  Optional. Type list of uniline.
