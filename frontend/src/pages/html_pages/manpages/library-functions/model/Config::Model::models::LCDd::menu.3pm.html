NAME
       Config::Model::models::LCDd::menu - Configuration class LCDd::menu


DESCRIPTION
       Configuration classes used by Config::Model

       generated from LCDd.conf


Elements
   DownKey
        Optional. Type uniline.

       default value :
           Down

   EnterKey
        Optional. Type uniline.

       default value :
           Enter

   LeftKey
        Optional. Type uniline.

       upstream_default value :
           Left

   MenuKey
       You  can configure what keys the menu should use. Note that the MenuKey will be reserved exclusively, the
       others work in shared mode.  Up to six keys are supported. The MenuKey (to enter and exit the menu),  the
       EnterKey  (to  select  values)  and  at  least one movement keys are required.  These are the default key
       assignments:  Optional. Type uniline.

       default value :
           Escape

   PermissiveGoto
       If true the server allows transitions between different client's menus.   Optional.  Type  enum.  choice:
       'true', 'false'.

       upstream_default value :
           false

   RightKey
        Optional. Type uniline.

       upstream_default value :
           Right

   UpKey
        Optional. Type uniline.

       default value :
           Up
