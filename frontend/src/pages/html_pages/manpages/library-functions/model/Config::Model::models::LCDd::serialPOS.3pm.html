NAME
       Config::Model::models::LCDd::serialPOS - Configuration class LCDd::serialPOS


DESCRIPTION
       Configuration classes used by Config::Model

       generated from LCDd.conf


Elements
   Device
       Device to use in serial mode  Optional. Type uniline.

       upstream_default value :
           /dev/lcd

   Size
       Specifies the size of the display in characters.  Optional. Type uniline.

       upstream_default value :
           16x2

   Speed
       communication baud rate with the display  Optional. Type enum. choice: '1200', '2400', '19200', '115200'.

       upstream_default value :
           9600

   Type
       Set the communication protocol to use with the POS display.  Optional. Type enum. choice: 'IEE', 'Epson',
       'Emax', 'IBM', 'LogicControls', 'Ultimate'.

       upstream_default value :
           AEDEX
