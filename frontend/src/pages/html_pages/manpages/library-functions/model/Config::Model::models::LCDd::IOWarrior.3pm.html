NAME
       Config::Model::models::LCDd::IOWarrior - Configuration class LCDd::IOWarrior


DESCRIPTION
       Configuration classes used by Config::Model

       generated from LCDd.conf


Elements
   ExtendedMode
       If you have an HD66712, a KS0073 or another 'almost HD44780-compatible', set this flag to get into
       extended mode (4-line linear).  Optional. Type uniline.

       upstream_default value :
           yes

   Lastline
       Specifies if the last line is pixel addressable (yes) or it controls an underline effect (no).  Optional.
       Type boolean.

       upstream_default value :
           yes

   SerialNumber
       serial  number.  Must  be  exactly as listed by usbview (if not given, the 1st IOWarrior found gets used)
       Optional. Type uniline.

       upstream_default value :
           00000674

   Size
       display dimensions.  Optional. Type uniline.

       default value :
           20x4
