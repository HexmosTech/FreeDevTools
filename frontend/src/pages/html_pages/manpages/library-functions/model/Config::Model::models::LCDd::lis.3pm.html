NAME
       Config::Model::models::LCDd::lis - Configuration class LCDd::lis


DESCRIPTION
       Configuration classes used by Config::Model

       generated from LCDd.conf


Elements
   Brightness
       Set the initial brightness 0-250 = 25%, 251-500 = 50%, 501-750 = 75%, 751-1000 = 100%  Optional. Type
       integer.

       upstream_default value :
           1000

   Lastline
       Specifies  if  the  last  line  is  pixel addressable (yes) or it only controls an underline effect (no).
       Optional. Type boolean.

       upstream_default value :
           yes

   ProductID
       USB Product ID Change only if testing a compatible device.  Optional. Type uniline.

       upstream_default value :
           0x6001

   Size
       Columns by lines  Optional. Type uniline.

       upstream_default value :
           20x2

   VendorID
       USB Vendor ID Change only if testing a compatible device.  Optional. Type uniline.

       upstream_default value :
           0x0403
