NAME
       Config::Model::models::LCDd::xosd - Configuration class LCDd::xosd


DESCRIPTION
       Configuration classes used by Config::Model

       generated from LCDd.conf


Elements
   Font
       X font to use, in XLFD format, as given by "xfontsel"  Optional. Type uniline.

       default value :
           -*-terminus-*-r-*-*-*-320-*-*-*-*-*

   Offset
       Offset in pixels from the top-left corner of the monitor  Optional. Type uniline.

       upstream_default value :
           0x0

   Size
       set display size  Optional. Type uniline.

       upstream_default value :
           20x4
