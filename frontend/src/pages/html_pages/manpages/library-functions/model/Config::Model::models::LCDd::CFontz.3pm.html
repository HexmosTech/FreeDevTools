NAME
       Config::Model::models::LCDd::CFontz - Configuration class LCDd::CFontz


DESCRIPTION
       Configuration classes used by Config::Model

       generated from LCDd.conf


Elements
   Brightness
       Set the initial brightness  Optional. Type integer.

       upstream_default value :
           1000

   Contrast
       Set the initial contrast  Optional. Type integer.

       upstream_default value :
           560

   Device
       Select the output device to use  Optional. Type uniline.

       upstream_default value :
           /dev/lcd

   NewFirmware
       Set the firmware version (New means >= 2.0)  Optional. Type boolean.

       upstream_default value :
           no

   OffBrightness
       Set  the initial off-brightness This value is used when the display is normally switched off in case LCDd
       is inactive.  Optional. Type integer.

       upstream_default value :
           0

   Reboot
       Reinitialize the LCD's BIOS normally you shouldn't need this.  Optional. Type boolean.

       upstream_default value :
           no

   Size
       Select the LCD size  Optional. Type uniline.

       upstream_default value :
           20x4

   Speed
       Set the communication speed  Optional. Type enum. choice: '1200', '2400', '9600', '19200', '115200'.

       upstream_default value :
           9600
