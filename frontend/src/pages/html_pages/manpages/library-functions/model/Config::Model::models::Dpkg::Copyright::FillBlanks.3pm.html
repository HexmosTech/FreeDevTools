NAME
       Config::Model::models::Dpkg::Copyright::FillBlanks - Configuration class Dpkg::Copyright::FillBlanks


DESCRIPTION
       Configuration classes used by Config::Model


Elements
   path - Perl pattern to match file path
       Patterns are matched from the beginning a path. I.e. "share/websockify/" pattern will match
       "share/websockify/foo.rb" but will not match "web/share/websockify/foo.rb". See "Filling the blanks" in
       Dpkg::Copyright::Scanner for more details.  Optional. Type hash of node of class
       Dpkg::Copyright::FillBlanks::Pattern .
