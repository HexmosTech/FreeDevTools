NAME
       Config::Model::models::LCDd::ea65 - Configuration class LCDd::ea65


DESCRIPTION
       Configuration classes used by Config::Model

       generated from LCDd.conf


Elements
   Brightness
       Device is fixed /dev/ttyS1 Width and Height are fixed 9x1 As the VFD is self luminescent we don't have a
       backlight But we can use the backlight functions to control the front LEDs Brightness 0 to 299 -> LEDs
       off Brightness 300 to 699 -> LEDs half bright Brightness 700 to 1000 -> LEDs full bright.  Optional. Type
       uniline.

       default value :
           500

   OffBrightness
       OffBrightness is the the value used for the 'backlight off' state.  Optional. Type uniline.

       default value :
           0
