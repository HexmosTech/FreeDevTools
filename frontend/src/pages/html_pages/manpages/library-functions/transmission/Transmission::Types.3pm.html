NAME
       Transmission::Types - Moose types for Transmission


DESCRIPTION
       The types below is pretty much what you would expect them to be, execpt for some (maybe weird?) default
       values - that is for coercion from "Any".

       The type names correspond to types used in the Transmission RPC specification.


TYPES
   number
   double
   string
   boolean
   array
