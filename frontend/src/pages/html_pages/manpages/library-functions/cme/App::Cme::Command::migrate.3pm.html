NAME
       App::Cme::Command::migrate - Migrate the configuration of an application


VERSION
       version 1.041


SYNOPSIS
         # check dpkg files, update deprecated parameters and save
         cme migrate dpkg


DESCRIPTION
       Checks the content of the configuration file of an application (and show warnings if needed), update
       deprecated parameters (old value are saved to new parameters) and save the new configuration. See
       App::Cme::Command::migrate.

       For more details, see "Upgrade" in Config::Model::Value


Common options
       See "Global Options" in cme.
