NAME
       FcRangeDestroy - destroy a range object


SYNOPSIS
       #include <fontconfig/fontconfig.h>

       void FcRangeDestroy (FcRange *range);


DESCRIPTION
       FcRangeDestroy destroys a FcRange object, freeing all memory associated with it.
