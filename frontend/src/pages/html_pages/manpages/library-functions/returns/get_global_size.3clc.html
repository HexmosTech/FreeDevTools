NAME
       get_global_size - Returns the number of global work-items specified for dimension identified by dimindx.

       size_t get_global_size(uint dimindx);


DESCRIPTION
       This value is given by the global_work_size argument to clEnqueueNDRangeKernel(3clc). Valid values of
       dimindx are 0 to get_work_dim(3clc)() - 1. For other values of dimindx, get_global_size() returns 1.

       For clEnqueueTask(3clc), this always returns 1.
