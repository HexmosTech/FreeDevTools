NAME
       RDF::Query::Algebra::Update - Algebra class for UPDATE operations


VERSION
       This document describes RDF::Query::Algebra::Update version 2.919.


METHODS
       Beyond the methods documented below, this class inherits methods from the RDF::Query::Algebra class.

       "new ( $delete_template, $insert_template, $pattern, \%dataset, $data_only_flag )"
           Returns a new UPDATE structure.

       "construct_args"
           Returns  a  list  of  arguments that, passed to this class' constructor, will produce a clone of this
           algebra pattern.

       "sse"
           Returns the SSE string for this algebra expression.

       "as_sparql"
           Returns the SPARQL string for this algebra expression.

       "referenced_blanks"
           Returns a list of the blank node names used in this algebra expression.

       "referenced_variables"
       "delete_template"
       "insert_template"
       "pattern"
       "dataset"
       "data_only"
       "check_duplicate_blanks"
           Returns true if blank nodes respect the SPARQL rule of no blank-label re-use across  BGPs,  otherwise
           throws a RDF::Query::Error::QueryPatternError exception.
