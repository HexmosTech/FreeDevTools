NAME
       RDF::Query::Algebra::Move - Algebra class for MOVE operations


VERSION
       This document describes RDF::Query::Algebra::Move version 2.919.


METHODS
       Beyond the methods documented below, this class inherits methods from the RDF::Query::Algebra class.

       "new ( $from, $to, $silent )"
           Returns a new MOVE structure.

       "construct_args"
           Returns  a  list  of  arguments that, passed to this class' constructor, will produce a clone of this
           algebra pattern.

       "as_sparql"
           Returns the SPARQL string for this algebra expression.

       "sse"
           Returns the SSE string for this algebra expression.

       "referenced_blanks"
           Returns a list of the blank node names used in this algebra expression.

       "referenced_variables"
       "from"
       "to"
       "silent"
