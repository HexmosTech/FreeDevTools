NAME
       SDL_ALPHA_TRANSPARENT_FLOAT - A fully transparent floating point alpha value.


SYNOPSIS
       #include <SDL3/SDL_pixels.h>

       #define SDL_ALPHA_TRANSPARENT_FLOAT 0.0f


AVAILABILITY
       This macro is available since SDL 3.2.0.
