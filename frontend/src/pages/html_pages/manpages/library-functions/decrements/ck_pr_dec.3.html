DESCRIPTION
       The ck_pr_dec(3) family of functions atomically decrement the value pointed to by target.


RETURN VALUES
       The ck_pr_dec_zero family of functions set the value pointed to by  z  to  true  if  the  result  of  the
       decrement  operation was 0. They set the value pointed to by z to false otherwise.  The ck_pr_dec_is_zero
       family of function return true if the result of the decrement operation was 0 and false otherwise.
