NAME
       CGI::Test::Form::Widget::Button::Reset - A reset button


SYNOPSIS
        # Inherits from CGI::Test::Form::Widget::Button
        # $form is a CGI::Test::Form

        my @reset = $form->buttons_matching(sub { $_[0]->is_reset });
        $reset[0]->press if @reset;


DESCRIPTION
       This class models a reset button.  Pressing this buttom immediately resets the form to its original
       state.  The processing is done on the client-side, and no request is made to the HTTP server.

       The interface is the same as the one described in CGI::Test::Form::Widget::Button.
