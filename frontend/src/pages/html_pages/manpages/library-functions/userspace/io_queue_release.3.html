NAME
       io_queue_release - Release the context associated with the userspace handle


SYNOPSIS
       #include <errno.h>
       #include <libaio.h>

       int io_queue_release(io_context_t ctx);


DESCRIPTION
       io_queue_release  destroys  the  context associated with the userspace handle. May cancel any outstanding
       AIOs and block on completion.


RETURN VALUES
       On success, io_queue_release() returns 0.  Otherwise, -error is return, where error is one  of  the  Exxx
       values defined in the Errors section.


ERRORS
       EINVAL ctx  refers  to  an uninitialized aio context, the iocb pointed to by iocbs contains an improperly
              initialized iocb.

       ENOSYS Not implemented.
