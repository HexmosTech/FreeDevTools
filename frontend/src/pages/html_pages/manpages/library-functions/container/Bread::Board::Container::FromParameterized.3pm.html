NAME
       Bread::Board::Container::FromParameterized - container with weak parent reference


VERSION
       version 0.37


DESCRIPTION
       When creating an actual container from a parameterized container, the returned container is re-blessed
       into this class.

       The only difference between this class and Bread::Board::Container is that the "parent" attribute here is
       a weak reference.
