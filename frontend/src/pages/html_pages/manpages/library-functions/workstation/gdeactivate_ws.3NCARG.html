NAME
       gdeactivate_ws - deactivates a GKS workstation


SYNOPSIS
       #include <ncarg/gks.h>

       void gdeactivate_ws(Gint ws_id);


DESCRIPTION
       ws_id       (Input)  - A number identifying the workstation to be deactivated.  ws_id must be the same as
                   that used in some previous gactivate_ws call.


USAGE
       The specified workstation is removed from the set of active workstations.  GKS output primitives are  not
       sent to workstations that are not active.


ACCESS
       To use the GKS C-binding routines, load the ncarg_gks and ncarg_c libraries.
