SYNOPSIS
          mongoc_collection_t *
          mongoc_collection_copy (mongoc_collection_t *collection);


PARAMETERS
       â€¢ collection: A mongoc_collection_t.


DESCRIPTION
       Performs  a  deep  copy  of  the  collection  struct  and its configuration. Useful if you intend to call
       mongoc_collection_set_write_concern(),               mongoc_collection_set_read_prefs(),               or
       mongoc_collection_set_read_concern(), and want to preserve an unaltered copy of the struct.

       This function does not copy the contents of the collection on the MongoDB server.


RETURNS
       A  newly  allocated  mongoc_collection_t  that  should  be freed with mongoc_collection_destroy() when no
       longer in use.
