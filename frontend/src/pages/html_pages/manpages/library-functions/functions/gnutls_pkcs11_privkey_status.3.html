NAME
       gnutls_pkcs11_privkey_status - API function


SYNOPSIS
       #include <gnutls/pkcs11.h>

       unsigned gnutls_pkcs11_privkey_status(gnutls_pkcs11_privkey_t key);


ARGUMENTS
       gnutls_pkcs11_privkey_t key
                   Holds the key


DESCRIPTION
       Checks the status of the private key token.


RETURNS
       this  function  will  return non-zero if the token holding the private key is still available (inserted),
       and zero otherwise.


SINCE
       3.1.9
