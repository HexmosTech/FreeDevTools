NAME
       stringprep_ucs4_to_utf8 - API function


SYNOPSIS
       #include <stringprep.h>

       char  *  stringprep_ucs4_to_utf8(const  uint32_t  *  str,  ssize_t  len,  size_t  *  items_read, size_t *
       items_written);


ARGUMENTS
       const uint32_t * str
                   a UCS-4 encoded string

       ssize_t len the maximum length of  str to use. If  len < 0, then  the  string  is  terminated  with  a  0
                   character.

       size_t * items_read
                   location to store number of characters read read, or NULL.

       size_t * items_written
                   location  to  store  number of bytes written or NULL.  The value here stored does not include
                   the trailing 0 byte.


DESCRIPTION
       Convert a string from a 32-bit fixed width representation  as  UCS-4.   to  UTF-8.  The  result  will  be
       terminated with a 0 byte.

       Return value: a pointer to a newly allocated UTF-8 string.  This value must be deallocated by the caller.
       If an error occurs, NULL will be returned.
