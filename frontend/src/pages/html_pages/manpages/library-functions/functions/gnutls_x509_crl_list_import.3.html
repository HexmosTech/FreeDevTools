NAME
       gnutls_x509_crl_list_import - API function


SYNOPSIS
       #include <gnutls/x509.h>

       int  gnutls_x509_crl_list_import(gnutls_x509_crl_t * crls, unsigned int * crl_max, const gnutls_datum_t *
       data, gnutls_x509_crt_fmt_t format, unsigned int flags);


ARGUMENTS
       gnutls_x509_crl_t * crls
                   Indicates where the parsed CRLs will be copied to. Must not be initialized.

       unsigned int * crl_max
                   Initially must hold the maximum number of crls. It will be updated with the  number  of  crls
                   available.

       const gnutls_datum_t * data
                   The CRL data

       gnutls_x509_crt_fmt_t format
                   One of DER or PEM.

       unsigned int flags
                   must be (0) or an OR'd sequence of gnutls_certificate_import_flags.


DESCRIPTION
       This  function will convert the given CRL list to the native gnutls_x509_crl_t format. The output will be
       stored in  crls .  They will be automatically initialized.

       If the Certificate is PEM encoded it should have a header of "X509 CRL".


RETURNS
       the number of certificates read or a negative error value.


SINCE
       3.0
