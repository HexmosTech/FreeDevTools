FUNCTIONS
   bs_quote
   bs_unquote
       bs_quote() "makes safe" strings with backslashed characters in it, so other operations can be done on
       them. "bs_unquote" goes the other way.

        $string = "foo \<bar\> <baz>";
        $string = bs_quote( $string );
        $string =~ s/<([^>]+)>/"$1"/g;
        $string = bs_unquote( $string );
        $string eq 'foo <bar> "baz"';
