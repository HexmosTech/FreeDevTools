NAME
       gnutls_hash_squeeze - API function


SYNOPSIS
       #include <gnutls/crypto.h>

       int gnutls_hash_squeeze(gnutls_hash_hd_t handle, void * output, size_t length);


ARGUMENTS
       gnutls_hash_hd_t handle
                   a gnutls_hash_hd_t

       void * output
                   destination to store the output; must be equal to or larger than  length

       size_t length
                   length of  output


DESCRIPTION
       This  function  will  extract  digest  output  of   length  bytes.  The   handle must be initialized with
       gnutls_hash_init()  as  an  extended   output   function   (XOF),   such   as   GNUTLS_DIG_SHAKE_128   or
       GNUTLS_DIG_SHAKE_256.

       This  function  can  be  called multiple times. To reset the state of  handle , call gnutls_hash_deinit()
       with NULL as the digest argument.


RETURNS
       GNUTLS_E_SUCCESS (0) on success; negative error code otherwise.


SINCE
       3.8.6
