NAME
       gnutls_psk_allocate_server_credentials - API function


SYNOPSIS
       #include <gnutls/gnutls.h>

       int gnutls_psk_allocate_server_credentials(gnutls_psk_server_credentials_t * sc);


ARGUMENTS
       gnutls_psk_server_credentials_t * sc
                   is a pointer to a gnutls_psk_server_credentials_t type.


DESCRIPTION
       Allocate a gnutls_psk_server_credentials_t structure.


RETURNS
       On success, GNUTLS_E_SUCCESS (0) is returned, otherwise an error code is returned.
