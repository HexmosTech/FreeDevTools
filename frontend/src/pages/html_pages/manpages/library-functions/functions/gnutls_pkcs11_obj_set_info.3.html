NAME
       gnutls_pkcs11_obj_set_info - API function


SYNOPSIS
       #include <gnutls/pkcs11.h>

       int  gnutls_pkcs11_obj_set_info(gnutls_pkcs11_obj_t  obj,  gnutls_pkcs11_obj_info_t  itype,  const void *
       data, size_t data_size, unsigned flags);


ARGUMENTS
       gnutls_pkcs11_obj_t obj
                   should contain a gnutls_pkcs11_obj_t type

       gnutls_pkcs11_obj_info_t itype
                   Denotes the type of information to be set

       const void * data
                   the data to set

       size_t data_size
                   the size of data

       unsigned flags
                   Or sequence of GNUTLS_PKCS11_OBJ_* flags


DESCRIPTION
       This  function  will  set  attributes  on  the  provided  object.   Available  options  for   itype   are
       GNUTLS_PKCS11_OBJ_LABEL, GNUTLS_PKCS11_OBJ_ID_HEX, and GNUTLS_PKCS11_OBJ_ID.


RETURNS
       GNUTLS_E_SUCCESS (0) on success or a negative error code on error.


SINCE
       3.4.0
