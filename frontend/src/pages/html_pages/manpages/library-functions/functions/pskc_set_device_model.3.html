NAME
       pskc_set_device_model - API function


SYNOPSIS
       #include <pskc.h>

       void pskc_set_device_model(pskc_key_t * key, const char * model);


ARGUMENTS
       pskc_key_t * key
                   a pskc_key_t handle from, e.g., pskc_add_keypackage().

       const char * model
                   a string with model name to set.


DESCRIPTION
       Set  the  PSKC  KeyPackage DeviceInfo Model value.  This element describes the model of the device (e.g.,
       "one-button-HOTP-token-V1").

       The pointer is stored in container, not a copy of the data, so you must not deallocate  the  data  before
       another call to this function or the last call to any function using container.
