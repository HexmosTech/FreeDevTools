NAME
       shishi_tgs_sendrecv_hint - API function


SYNOPSIS
       #include <shishi.h>

       int shishi_tgs_sendrecv_hint(Shishi_tgs * tgs, Shishi_tkts_hint * hint);


ARGUMENTS
       Shishi_tgs * tgs
                   structure that holds information about TGS exchange

       Shishi_tkts_hint * hint
                   additional parameters that modify connection behaviour, or NULL.


DESCRIPTION
       Send  TGS-REQ  and  receive TGS-REP or KRB-ERROR.  This is the subsequent authentication, usually used to
       acquire server tickets.  The hint structure can be used to set, e.g., parameters for TLS authentication.


RETURN VALUE
       Returns SHISHI_OK iff successful.
