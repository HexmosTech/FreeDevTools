NAME
       oath_bin2hex - API function


SYNOPSIS
       #include <oath.h>

       void oath_bin2hex(const char * binstr, size_t binlen, char * hexstr);


ARGUMENTS
       const char * binstr
                   input binary data

       size_t binlen
                   length of input binary data binstr

       char * hexstr
                   output string with hex data, must have room for 2*binlen+1.


DESCRIPTION
       Convert binary data to NUL-terminated string with hex data.  The output hexstr is allocated by the caller
       and  must  have  room  for at least 2*binlen+1, to make room for the encoded data and the terminating NUL
       byte.


SINCE
       1.12.0
