NAME
       shishi_resolv - API function


SYNOPSIS
       #include <shishi.h>

       Shishi_dns shishi_resolv(const char * zone, uint16_t querytype);


ARGUMENTS
       const char * zone
                   Domain name of authentication zone, e.g. "EXAMPLE.ORG"

       uint16_t querytype
                   Type of domain data to query for.


DESCRIPTION
       Queries  the  DNS  resolver  for data of type querytype about the domain name zone.  Currently, the types
       SHISHI_DNS_TXT and SHISHI_DNS_SRV are the only supported kinds.

       After its use, the returned list should be deallocated by a call to shishi_resolv_free().


RETURN VALUE
       Returns a linked list of DNS resource records, or NULL if the query failed.
