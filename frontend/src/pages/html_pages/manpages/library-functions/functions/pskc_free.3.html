NAME
       pskc_free - API function


SYNOPSIS
       #include <pskc.h>

       void pskc_free(void * ptr);


ARGUMENTS
       void * ptr  memory region to deallocate, or NULL.


DESCRIPTION
       Deallocates memory region by calling free().  If ptr is NULL no operation is performed.

       This  function  is  necessary on Windows, where different parts of the same application may use different
       memory heaps.
