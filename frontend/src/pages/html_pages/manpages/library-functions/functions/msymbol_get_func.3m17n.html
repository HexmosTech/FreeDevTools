NAME
       msymbol_get_func -


SYNOPSIS
       M17NFunc msymbol_get_func (MSymbol symbol, MSymbol key)


DESCRIPTION
       Get the value (function pointer) of a symbol property. The msymbol_get_func() function is similar to
       msymbol_get() but for getting a function pointer form the property of symbol symbol.


SEE ALSO
           msymbol_get()
