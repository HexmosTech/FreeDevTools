NAME
       TSHttpTxnNextHopNameGet - TSHttpTxnNextHopNameGet API function


SYNOPSIS
          #include <ts/ts.h>

       const char *TSHttpTxnNextHopNameGet(TSHttpTxn txnp)


DESCRIPTION
       Get the next hop name for transaction txnp.

       NOTE:
          The  pointer  is  valid  only  for  the  current  callback. Clients that need to keep the value across
          callbacks must maintain their own storage.
