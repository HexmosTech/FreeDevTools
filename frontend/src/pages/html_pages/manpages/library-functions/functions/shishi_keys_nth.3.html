NAME
       shishi_keys_nth - API function


SYNOPSIS
       #include <shishi.h>

       const Shishi_key * shishi_keys_nth(Shishi_keys * keys, int keyno);


ARGUMENTS
       Shishi_keys * keys
                   key set handle as allocated by shishi_keys().

       int keyno   integer indicating requested key in key set.


GET THE N
       th ticket in key set.


RETURN VALUE
       Returns  a  key  handle to the keyno:th key in the key set, or NULL if keys is invalid or keyno is out of
       bounds.  The first key is keyno 0, the second key keyno 1, and so on.
