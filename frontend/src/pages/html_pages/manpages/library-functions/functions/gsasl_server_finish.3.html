NAME
       gsasl_server_finish - API function


SYNOPSIS
       #include <gsasl.h>

       void gsasl_server_finish(Gsasl_session * sctx);


ARGUMENTS
       Gsasl_session * sctx
                   libgsasl server handle.


DESCRIPTION
       Destroy  a  libgsasl server handle.  The handle must not be used with other libgsasl functions after this
       call.


DEPRECATED
       Use gsasl_finish() instead.
