NAME
       gsasl_mechanism_name - API function


SYNOPSIS
       #include <gsasl.h>

       const char * gsasl_mechanism_name(Gsasl_session * sctx);


ARGUMENTS
       Gsasl_session * sctx
                   libgsasl session handle.


DESCRIPTION
       This  function  returns  the  name  of  the  SASL mechanism used in the session.  The pointer must not be
       deallocated by the caller.

       Return value: Returns a zero terminated character array with the name of the SASL mechanism, or  NULL  if
       not known.

       0.2.28
