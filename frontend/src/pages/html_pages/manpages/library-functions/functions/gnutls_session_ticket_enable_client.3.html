NAME
       gnutls_session_ticket_enable_client - API function


SYNOPSIS
       #include <gnutls/gnutls.h>

       int gnutls_session_ticket_enable_client(gnutls_session_t session);


ARGUMENTS
       gnutls_session_t session
                   is a gnutls_session_t type.


DESCRIPTION
       Request  that  the  client  should attempt session resumption using SessionTicket. This call is typically
       unnecessary as session tickets are enabled by default.


RETURNS
       On success, GNUTLS_E_SUCCESS (0) is returned, or an error code.


SINCE
       2.10.0
