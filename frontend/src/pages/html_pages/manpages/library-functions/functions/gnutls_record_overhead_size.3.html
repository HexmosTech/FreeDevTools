NAME
       gnutls_record_overhead_size - API function


SYNOPSIS
       #include <gnutls/gnutls.h>

       size_t gnutls_record_overhead_size(gnutls_session_t session);


ARGUMENTS
       gnutls_session_t session
                   is gnutls_session_t


DESCRIPTION
       This  function  will return the size in bytes of the overhead due to TLS (or DTLS) per record. On certain
       occasions (e.g., CBC ciphers) the returned value is the maximum possible overhead.


SINCE
       3.2.2
