NAME
       gsasl_server_mechlist - API function


SYNOPSIS
       #include <gsasl.h>

       int gsasl_server_mechlist(Gsasl * ctx, char ** out);


ARGUMENTS
       Gsasl * ctx libgsasl handle.

       char ** out newly allocated output character array.


DESCRIPTION
       Return a newly allocated string containing SASL names, separated by space, of mechanisms supported by the
       libgsasl  server.    out  is  allocated  by  this  function,  and  it  is the responsibility of caller to
       deallocate it.

       Return value: Returns GSASL_OK if successful, or error code.
