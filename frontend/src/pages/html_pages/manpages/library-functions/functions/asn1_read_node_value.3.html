NAME
       asn1_read_node_value - API function


SYNOPSIS
       #include <libtasn1.h>

       int asn1_read_node_value(asn1_node_const node, asn1_data_node_st * data);


ARGUMENTS
       asn1_node_const node
                   pointer to a node.

       asn1_data_node_st * data
                   a point to a asn1_data_node_st


DESCRIPTION
       Returns  the  value  a  data  node  inside a asn1_node structure.  The data returned should be handled as
       constant values.


RETURNS
       ASN1_SUCCESS if the node exists.
