NAME
       shishi_ap_req_der_set - API function


SYNOPSIS
       #include <shishi.h>

       int shishi_ap_req_der_set(Shishi_ap * ap, char * der, size_t derlen);


ARGUMENTS
       Shishi_ap * ap
                   structure that holds information about AP exchange

       char * der  input array with DER encoded AP-REQ.

       size_t derlen
                   length of input array with DER encoded AP-REQ.


DESCRIPTION
       DER decode AP-REQ and set it AP exchange.  If decoding fails, the AP-REQ in the AP exchange is lost.


RETURN VALUE
       Returns SHISHI_OK.
