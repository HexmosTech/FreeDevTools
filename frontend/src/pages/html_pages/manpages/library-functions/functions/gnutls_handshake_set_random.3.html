NAME
       gnutls_handshake_set_random - API function


SYNOPSIS
       #include <gnutls/gnutls.h>

       int gnutls_handshake_set_random(gnutls_session_t session, const gnutls_datum_t * random);


ARGUMENTS
       gnutls_session_t session
                   is a gnutls_session_t type.

       const gnutls_datum_t * random
                   a random value of 32-bytes


DESCRIPTION
       This  function  will  explicitly  set  the  server  or  client  hello  random value in the subsequent TLS
       handshake. The random value should be a 32-byte value.

       Note that this function should not normally be used as gnutls will select automatically  a  random  value
       for the handshake.

       This function should not be used when resuming a session.


RETURNS
       GNUTLS_E_SUCCESS on success, or an error code.

       Since 3.1.9
