NAME
       function::set_s32_arg - Set function argument as signed 32-bit value


SYNOPSIS
           set_s32_arg(n:long,v:long)


ARGUMENTS
       n
           index of argument to return

       v
           value to set


DESCRIPTION
       Set the signed 32-bit value of argument n, same as int_arg.
