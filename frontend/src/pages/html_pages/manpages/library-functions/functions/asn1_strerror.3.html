NAME
       asn1_strerror - API function


SYNOPSIS
       #include <libtasn1.h>

       const char * asn1_strerror(int error);


ARGUMENTS
       int error   is an error returned by a libtasn1 function.


DESCRIPTION
       Returns  a  string  with  a  description  of  an  error.  This function is similar to strerror.  The only
       difference is that it accepts an error (number) returned by a libtasn1 function.


RETURNS
       Pointer to static zero-terminated string describing error code.


SINCE
       1.6
