NAME
       oath_done - API function


SYNOPSIS
       #include <oath.h>

       int oath_done( void);


ARGUMENTS
        void


DESCRIPTION
       This  function  deinitializes  the OATH library, which were initialized using oath_init().  After calling
       this function, no other OATH library function may be called except for to re-initialize the library using
       oath_init().


RETURNS
       On success, OATH_OK (zero) is returned, otherwise an error code is returned.
