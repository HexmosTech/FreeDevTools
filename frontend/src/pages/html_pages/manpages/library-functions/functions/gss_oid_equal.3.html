NAME
       gss_oid_equal - API function


SYNOPSIS
       #include <gss.h>

       int gss_oid_equal(gss_const_OID first_oid, gss_const_OID second_oid);


ARGUMENTS
       gss_const_OID first_oid
                   (Object ID, read) First Object identifier.

       gss_const_OID second_oid
                   (Object ID, read) First Object identifier.


DESCRIPTION
       Compare two OIDs for equality.  The comparison is "deep", i.e., the actual byte sequences of the OIDs are
       compared instead of just the pointer equality.  This function is standardized in RFC 6339.


RETURN VALUE
       Returns boolean value true when the two OIDs are
         equal, otherwise false.
