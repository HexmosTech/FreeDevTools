NAME
       gsasl_hash_length - API function


SYNOPSIS
       #include <gsasl.h>

       size_t gsasl_hash_length(Gsasl_hash hash);


ARGUMENTS
       Gsasl_hash hash
                   a Gsasl_hash element, e.g., GSASL_HASH_SHA256.


DESCRIPTION
       Return     the     digest     output    size    for    hash    function     hash    .     For    example,
       gsasl_hash_length(GSASL_HASH_SHA256) returns GSASL_HASH_SHA256_SIZE which is 32.


RETURNS
       size of supplied Gsasl_hash element.


SINCE
       1.10
