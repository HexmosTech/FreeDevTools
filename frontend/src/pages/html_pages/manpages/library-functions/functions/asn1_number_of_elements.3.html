NAME
       asn1_number_of_elements - API function


SYNOPSIS
       #include <libtasn1.h>

       int asn1_number_of_elements(asn1_node_const element, const char * name, int * num);


ARGUMENTS
       asn1_node_const element
                   pointer to the root of an ASN1 structure.

       const char * name
                   the name of a sub-structure of ROOT.

       int * num   pointer to an integer where the result will be stored


DESCRIPTION
       Counts the number of elements of a sub-structure called NAME with names equal to "?1","?2", ...


RETURNS
       ASN1_SUCCESS if successful, ASN1_ELEMENT_NOT_FOUND if
        name is not known, ASN1_GENERIC_ERROR if pointer  num is NULL.
