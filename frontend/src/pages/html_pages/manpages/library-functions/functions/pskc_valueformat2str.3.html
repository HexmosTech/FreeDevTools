NAME
       pskc_valueformat2str - API function


SYNOPSIS
       #include <pskc.h>

       const char * pskc_valueformat2str(pskc_valueformat valueformat);


ARGUMENTS
       pskc_valueformat valueformat
                   an pskc_valueformat enumeration type


DESCRIPTION
       Convert  pskc_valueformat  to a string.  For example, pskc_valueformat2str(PSKC_VALUEFORMAT_DECIMAL) will
       return "DECIMAL".  The returned string must not be deallocated.


RETURNS
       String corresponding to pskc_valueformat.
