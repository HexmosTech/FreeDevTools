   Strftime Patterns
       %c (%a %d %b %Y %T) - date time format

          2008-02-05T18:30:30 = mar. 05 févr. 2008 18:30:30
          1995-12-22T09:05:02 = ven. 22 déc. 1995 09:05:02
         -0010-09-15T04:44:23 = sam. 15 sept. -10 04:44:23

       %x (%Y-%m-%d) - date format

          2008-02-05T18:30:30 = 2008-02-05
          1995-12-22T09:05:02 = 1995-12-22
         -0010-09-15T04:44:23 = -10-09-15

       %X (%T) - time format

          2008-02-05T18:30:30 = 18:30:30
          1995-12-22T09:05:02 = 09:05:02
