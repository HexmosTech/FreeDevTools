   Strftime Patterns
       %c (%Y m. %B %d d. %T) - date time format

          2008-02-05T18:30:30 = 2008 m. vasario 05 d. 18:30:30
          1995-12-22T09:05:02 = 1995 m. gruodžio 22 d. 09:05:02
         -0010-09-15T04:44:23 = -10 m. rugsėjo 15 d. 04:44:23

       %x (%Y-%m-%d) - date format

          2008-02-05T18:30:30 = 2008-02-05
          1995-12-22T09:05:02 = 1995-12-22
         -0010-09-15T04:44:23 = -10-09-15

       %X (%T) - time format

          2008-02-05T18:30:30 = 18:30:30
          1995-12-22T09:05:02 = 09:05:02
         -0010-09-15T04:44:23 = 04:44:23
