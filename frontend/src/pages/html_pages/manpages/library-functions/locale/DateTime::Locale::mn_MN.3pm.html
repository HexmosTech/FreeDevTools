   Strftime Patterns
       %c (%Y %b %d, %a %T) - date time format

          2008-02-05T18:30:30 = 2008 2-р сар 05, Мя 18:30:30
          1995-12-22T09:05:02 = 1995 12-р сар 22, Ба 09:05:02
         -0010-09-15T04:44:23 = -10 9-р сар 15, Бя 04:44:23

       %x (%Y.%m.%d) - date format

          2008-02-05T18:30:30 = 2008.02.05
          1995-12-22T09:05:02 = 1995.12.22
         -0010-09-15T04:44:23 = -10.09.15

       %X (%T) - time format

          2008-02-05T18:30:30 = 18:30:30
          1995-12-22T09:05:02 = 09:05:02
         -0010-09-15T04:44:23 = 04:44:23
