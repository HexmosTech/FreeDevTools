         -0010-09-15T04:44:23 = ᏈᏕᎾ 15 ᏚᎵ -10 04:44:23 ᏌᎾᎴ

       %x (%m/%d/%Y) - date format

          2008-02-05T18:30:30 = 02/05/2008
          1995-12-22T09:05:02 = 12/22/1995
         -0010-09-15T04:44:23 = 09/15/-10

       %X (%r) - time format

          2008-02-05T18:30:30 = 06:30:30 ᏒᎯᏱᎢ
          1995-12-22T09:05:02 = 09:05:02 ᏌᎾᎴ
         -0010-09-15T04:44:23 = 04:44:23 ᏌᎾᎴ
