   Strftime Patterns
       %c (%a %d %b %Y %T) - date time format

          2008-02-05T18:30:30 = Bil 05 Feb 2008 18:30:30
          1995-12-22T09:05:02 = Hla 22 Dis 1995 09:05:02
         -0010-09-15T04:44:23 = Mgq 15 Sep -10 04:44:23

       %x (%d/%m/%Y) - date format

          2008-02-05T18:30:30 = 05/02/2008
          1995-12-22T09:05:02 = 22/12/1995
         -0010-09-15T04:44:23 = 15/09/-10

       %X (%T) - time format

          2008-02-05T18:30:30 = 18:30:30
          1995-12-22T09:05:02 = 09:05:02
