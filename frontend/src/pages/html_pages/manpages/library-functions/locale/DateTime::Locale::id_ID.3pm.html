   Strftime Patterns
       %c (%a %d %b %Y %r) - date time format

          2008-02-05T18:30:30 = Sel 05 Feb 2008 06:30:30 PM
          1995-12-22T09:05:02 = Jum 22 Des 1995 09:05:02 AM
         -0010-09-15T04:44:23 = Sab 15 Sep -10 04:44:23 AM

       %x (%d/%m/%y) - date format

          2008-02-05T18:30:30 = 05/02/08
          1995-12-22T09:05:02 = 22/12/95
         -0010-09-15T04:44:23 = 15/09/10

       %X (%T) - time format

          2008-02-05T18:30:30 = 18:30:30
          1995-12-22T09:05:02 = 09:05:02
