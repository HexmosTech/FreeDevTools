   Strftime Patterns
       %c (པསྱི་ལོ%yཟལ%mཚེས%dཆུ་ཚོད%Hཀསར་མ%Mཀསར་ཆ%S) - date time format

          2008-02-05T18:30:30 = པསྱི་ལོ08ཟལ02ཚེས05ཆུ་ཚོད18ཀསར་མ30ཀསར་ཆ30
          1995-12-22T09:05:02 = པསྱི་ལོ95ཟལ12ཚེས22ཆུ་ཚོད09ཀསར་མ05ཀསར་ཆ02
         -0010-09-15T04:44:23 = པསྱི་ལོ10ཟལ09ཚེས15ཆུ་ཚོད04ཀསར་མ44ཀསར་ཆ23

       %x (པསྱི་ལོ%yཟལ%mཚེས%d) - date format

          2008-02-05T18:30:30 = པསྱི་ལོ08ཟལ02ཚེས05
          1995-12-22T09:05:02 = པསྱི་ལོ95ཟལ12ཚེས22
         -0010-09-15T04:44:23 = པསྱི་ལོ10ཟལ09ཚེས15

       %X (ཆུ་ཚོད%Hཀསར་མ%Mཀསར་ཆ%S) - time format

          2008-02-05T18:30:30 = ཆུ་ཚོད18ཀསར་མ30ཀསར་ཆ30
          1995-12-22T09:05:02 = ཆུ་ཚོད09ཀསར་མ05ཀསར་ཆ02
         -0010-09-15T04:44:23 = ཆུ་ཚོད04ཀསར་མ44ཀསར་ཆ23
