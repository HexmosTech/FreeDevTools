   Strftime Patterns
       %c (%a %b %e %H:%M:%S %Y) - date time format

          2008-02-05T18:30:30 = Koo Ngat  5 18:30:30 2008
          1995-12-22T09:05:02 = Kom Kpa 22 09:05:02 1995
         -0010-09-15T04:44:23 = Kol Bur 15 04:44:23 -10

       %x (%m/%d/%y) - date format

          2008-02-05T18:30:30 = 02/05/08
          1995-12-22T09:05:02 = 12/22/95
         -0010-09-15T04:44:23 = 09/15/10

          2008-02-05T18:30:30 = 18:30:30
