   Strftime Patterns
       %c (%a %d %b %Y %r %Z) - date time format

          2008-02-05T18:30:30 = Tldo 05 Feb 2008 06:30:30 GD UTC
          1995-12-22T09:05:02 = Jmc 22 Dis 1995 09:05:02 GH UTC
         -0010-09-15T04:44:23 = Sbti 15 Seb -10 04:44:23 GH UTC

       %x (%d.%m.%Y) - date format

          2008-02-05T18:30:30 = 05.02.2008
          1995-12-22T09:05:02 = 22.12.1995
         -0010-09-15T04:44:23 = 15.09.-10

       %X (%l:%M:%S %p) - time format

          2008-02-05T18:30:30 =  6:30:30 GD
          1995-12-22T09:05:02 =  9:05:02 GH
         -0010-09-15T04:44:23 =  4:44:23 GH
