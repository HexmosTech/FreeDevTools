   Strftime Patterns
       %c (%a %Y %b %d %T (%Z)) - date time format

          2008-02-05T18:30:30 = оп 2008 Олун 05 18:30:30 (UTC)
          1995-12-22T09:05:02 = бэ 1995 Ахс 22 09:05:02 (UTC)
         -0010-09-15T04:44:23 = сб -10 Блҕ 15 04:44:23 (UTC)

       %x (%Y.%m.%d) - date format

          2008-02-05T18:30:30 = 2008.02.05
          1995-12-22T09:05:02 = 1995.12.22
         -0010-09-15T04:44:23 = -10.09.15

       %X (%T) - time format

          2008-02-05T18:30:30 = 18:30:30
          1995-12-22T09:05:02 = 09:05:02
         -0010-09-15T04:44:23 = 04:44:23
