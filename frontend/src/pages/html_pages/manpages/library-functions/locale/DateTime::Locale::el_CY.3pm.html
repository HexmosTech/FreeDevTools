   Strftime Patterns
       %c (%a %d %b %Y %r %Z) - date time format

          2008-02-05T18:30:30 = Τρί 05 Φεβ 2008 06:30:30 μ.μ. UTC
          1995-12-22T09:05:02 = Παρ 22 Δεκ 1995 09:05:02 π.μ. UTC
         -0010-09-15T04:44:23 = Σάβ 15 Σεπ -10 04:44:23 π.μ. UTC

       %x (%d/%m/%Y) - date format

          2008-02-05T18:30:30 = 05/02/2008
          1995-12-22T09:05:02 = 22/12/1995
         -0010-09-15T04:44:23 = 15/09/-10

       %X (%r) - time format

          2008-02-05T18:30:30 = 06:30:30 μ.μ.
          1995-12-22T09:05:02 = 09:05:02 π.μ.
         -0010-09-15T04:44:23 = 04:44:23 π.μ.
