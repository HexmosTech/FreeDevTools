   Strftime Patterns
       %c (%x (%a) %X) - date time format

          2008-02-05T18:30:30 = 5.02.2008 г. (вт) 18:30:30
          1995-12-22T09:05:02 = 22.12.1995 г. (пт) 9:05:02
         -0010-09-15T04:44:23 = 15.09.-10 г. (сб) 4:44:23

       %x (%e.%m.%Y) - date format

          2008-02-05T18:30:30 =  5.02.2008
          1995-12-22T09:05:02 = 22.12.1995
         -0010-09-15T04:44:23 = 15.09.-10

       %X (%k:%M:%S) - time format

          2008-02-05T18:30:30 = 18:30:30
          1995-12-22T09:05:02 =  9:05:02
         -0010-09-15T04:44:23 =  4:44:23
