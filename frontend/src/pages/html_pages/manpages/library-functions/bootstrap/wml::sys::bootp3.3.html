NAME
       wml::sys::bootp3 - WML Bootstrap Code for Pass 3


SYNOPSIS
       No synopsis, because this is automatically included by WML on startup.


DESCRIPTION
       This include file currently just bootstraps the (perhaps) privately installed Perl 5 modules of WML which
       stay below @libdir@@libsubdir@/perl/.
