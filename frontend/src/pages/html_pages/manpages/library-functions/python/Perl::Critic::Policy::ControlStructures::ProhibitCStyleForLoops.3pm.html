NAME
       Perl::Critic::Policy::ControlStructures::ProhibitCStyleForLoops - Write "for(0..20)" instead of
       "for($i=0; $i<=20; $i++)".


AFFILIATION
       This Policy is part of the core Perl::Critic distribution.


DESCRIPTION
       The 3-part "for" loop that Perl inherits from C is butt-ugly, and only really necessary if you need
       irregular counting.  The very Perlish ".." operator is much more elegant and readable.

           for($i=0; $i<=$max; $i++){      #ick!
               do_something($i);
           }

           for(0..$max){                   #very nice
               do_something($_);
           }


CONFIGURATION
       This Policy is not configurable except for the standard options.
