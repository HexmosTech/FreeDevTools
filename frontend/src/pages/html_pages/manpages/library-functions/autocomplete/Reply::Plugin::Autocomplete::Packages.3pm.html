NAME
       Reply::Plugin::Autocomplete::Packages - tab completion for package names


VERSION
       version 0.42


SYNOPSIS
         ; .replyrc
         [ReadLine]
         [Autocomplete::Packages]


DESCRIPTION
       This plugin registers a tab key handler to autocomplete package names in Perl code.
