NAME
       Sympa::ListDef - Definition of list configuration parameters


DESCRIPTION
       This module keeps definition of configuration parameters for each list.

   Global variable
       %alias
           Deprecated by Sympa 6.2.16.

       %pinfo
           This  hash  COMPLETELY  defines  ALL list parameters.  It is then used to load, save, view, edit list
           config files.  See Sympa::Config::Schema for details about the content.
