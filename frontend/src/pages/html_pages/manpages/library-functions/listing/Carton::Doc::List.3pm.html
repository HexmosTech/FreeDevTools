NAME
       Carton::Doc::List - List dependencies tracked in the cpanfile.snapshot file


SYNOPSIS
         carton list


DESCRIPTION
       List the dependencies and version information tracked in the cpanfile.snapshot file. This command by
       default displays the name of the distribution (e.g. Foo-Bar-0.01) in a flat list.


OPTIONS
       --distfile
           Displays the list of distributions in a distfile format (i.e. "AUTHOR/Dist-1.23.tar.gz")
