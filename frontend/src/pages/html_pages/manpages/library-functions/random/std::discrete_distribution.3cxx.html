Detailed Description
   template<typename _IntType = int>
       class std::discrete_distribution< _IntType >"A discrete_distribution random number distribution.

       This distribution produces random numbers $ i, 0  i < n $, distributed according to the probability mass
       function $ p(i | p_0, ..., p_{n-1}) = p_i $.

       Since
           C++11
