NAME
       ZoneMinder::Control::Vivotek_ePTZ - ZoneMinder Perl extension for Vivotek ePTZ camera control protocol


SYNOPSIS
         use ZoneMinder::Control::Vivotek_ePTZ;


DESCRIPTION
       This module implements the ePTZ protocol used in various Vivotek IP cameras, developed with a Vivotek
       IB8369 model.

       Currently, only simple pan, tilt and zoom function is implemented. Presets will follow later.

   EXPORT
       None.
