NAME
       ZoneMinder::ONVIF - perl module to access onvif functions for ZoneMinder


SYNOPSIS
       use ZoneMinder::ONVIF;


DESCRIPTION
       This is a module to contain useful functions and import all the other modules required for ONVIF to work.

   EXPORT
       None by default.
