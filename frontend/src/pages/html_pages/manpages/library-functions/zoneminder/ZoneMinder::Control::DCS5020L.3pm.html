NAME
       ZoneMinder::Control::DCS5020L - Perl extension for DCS-5020L


SYNOPSIS
         use ZoneMinder::Database;
         DLINK DCS-5020L


DESCRIPTION
       ZoneMinder driver for the D-Link consumer camera DCS-5020L.

   EXPORT
