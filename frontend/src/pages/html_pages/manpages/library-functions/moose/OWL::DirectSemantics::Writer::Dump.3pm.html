NAME
       OWL::DirectSemantics::Writer::Dump - Moose::Role providing reasonably readable output


DESCRIPTION
       This Moose::Role provides a "dump" method to output an element for debugging.

       It requires the object or class it is composed with to provide an "element_name" method.

       "element_name" is essentially the name of the "function" the object represents, e.g.  'ClassAssertion'.


SEE ALSO
       OWL::DirectSemantics, OWL::DirectSemantics::Element.
