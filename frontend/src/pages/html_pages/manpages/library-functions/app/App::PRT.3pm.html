NAME
       App::PRT - Command line Perl Refactoring Tool


SYNOPSIS
           use App::PRT::CLI;
           my $cli = App::PRT::CLI->new;
           $cli->parse(@ARGV);
           $cli->run;


DESCRIPTION
       App::PRT is command line tools for Refactoring Perl.


CONTRIBUTING
       App::PRT uses Minilla for development.  The tests assume "." is in the Perl library path.  On Perl 5.26+,
       before running "minil test", add "."  to the path.  For example, in "bash":

           export PERL5LIB="$PERL5LIB":.

       Each command in the prt tool is implemented by a corresponding class under "App::PRT::Command".  For
       example, "rename_class" is implemented by App::PRT::Command::RenameClass.
