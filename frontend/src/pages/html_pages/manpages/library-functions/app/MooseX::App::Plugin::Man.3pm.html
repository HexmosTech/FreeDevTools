NAME
       MooseX::App::Plugin::Man - Adds a command to display the full manual


SYNOPSIS
       In your base class:

        package MyApp;
        use MooseX::App qw(Man);

       In your shell

        bash$ myapp man somecommand


DESCRIPTION
       This plugin adds a command to display the full manpage/perldoc of your application.
