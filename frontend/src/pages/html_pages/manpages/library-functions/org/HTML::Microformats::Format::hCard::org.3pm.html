NAME
       HTML::Microformats::Format::hCard::org - helper for hCards; handles the org property


DESCRIPTION
       Technically, this inherits from HTML::Microformats::Format, so can be used in the same way as any of the
       other microformat module, though I don't know why you'd want to.
