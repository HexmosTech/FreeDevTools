NAME
       dnsjit.core.object.payload - Application data payload


DESCRIPTION
       Payload object contains the data carried by the underlying transport protocol.  Payload is usually at the
       top of the object chain after parsing with, for example, dnsjit.filter.layer.

   Attributes
       payload
              A pointer to the payload.

       len    The length of the payload.

       padding
              The length of padding in the underlying packet.

   Functions
       Payload:type()
              Return the textual type of the object.

       Payload:prev()
              Return the previous object.

       Payload:cast()
              Cast the object to the underlining object module and return it.

       Payload:uncast()
              Cast the object to the generic object module and return it.

       Payload:copy()
              Make a copy of the object and return it.

       Payload:free()
              Free the object, should only be used on copies or otherwise allocated.
