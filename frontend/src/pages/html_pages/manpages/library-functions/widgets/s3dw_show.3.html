NAME
       s3dw_show - make widget visible


SYNOPSIS
       #include <s3dw.h>

       void s3dw_show(s3dw_widget *widget);


DESCRIPTION
       Switches a widget visible. Should be casted with S3DWIDGET().


AUTHOR
       Simon Wunderlich
           Author of s3d
