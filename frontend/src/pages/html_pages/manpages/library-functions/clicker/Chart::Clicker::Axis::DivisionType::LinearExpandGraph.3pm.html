NAME
       Chart::Clicker::Axis::DivisionType::LinearExpandGraph - Role describing how to divide data for
       Chart::Clicker::Axis


VERSION
       version 2.90


SYNOPSIS
         use Chart::Clicker::Axis;

         my $axis = Chart::Clicker::Axis->new({
           tick_division_type  => 'LinearRounded'
         });


DESCRIPTION
       Role describing how to divide data for Chart::Clicker::Axis.


NAME
       Chart::Clicker::Axis::DivisionType::LinearRounded - Nicely rounded segments on a linear scale.


ATTRIBUTES
   tick_slop
       This setting determines whether to add a tick outside of the data. If the tick would be within the
       percentage of a ticks size specified here as a decimal (10% would be 0.1), then the tick will be added
       expanding the graph.


METHODS
   best_tick_size
       The tick division considered best for the approximate number of ticks requested and data within the
       range.

   divvy
       Divides the range up into nicely rounded chunks for Chart::Clicker::Axis.
