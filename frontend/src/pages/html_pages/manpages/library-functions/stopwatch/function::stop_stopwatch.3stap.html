NAME
       function::stop_stopwatch - Stop a stopwatch


SYNOPSIS
           stop_stopwatch(name:string)


ARGUMENTS
       name
           the stopwatch name


DESCRIPTION
       Stop stopwatch name. Creates stopwatch name if it does not currently exist.
