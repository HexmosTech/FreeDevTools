NAME
       Message::Passing::Decoder::Crypt::CBC - Use Crypt::CBC to decrypt messages


SYNOPSIS
           message-pass --input STDIN --decoder Crypt::CBC \
               --decoder_options '{}' \
               --output ZeroMQ --output_options '...'


DESCRIPTION
       Decrypts messages with Crypt::CBC.
