NAME
       qb_ipcs_response_send - Send a response to a incoming request.


SYNOPSIS
       #include <qb/qbipcs.h>

       ssize_t qb_ipcs_response_send(
           qb_ipcs_connection_t   *c,     /* connection instance */
           const void             *data,  /* the message to send */
           size_t                  size   /* the size of the message */
       );


PARAMS
       c    connection instance

       data the message to send

       size the size of the message


DESCRIPTION
RETURN VALUE
       size sent or -errno for errors


NOTE
       the  data  must include a qb_ipc_response_header at the top of the message. The client will read the size
       field to determine how much to recv.
