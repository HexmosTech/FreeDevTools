NAME
       GraphQL::Role::Named - GraphQL "named" object role


SYNOPSIS
         with qw(GraphQL::Role::Named);


DESCRIPTION
       Allows type constraints for named objects, providing also "name" and "description" attributes.


ATTRIBUTES
   name
   description
       Optional description.


METHODS
   to_string
       Part of serialisation.
