NAME
       Code::TidyAll::Role::GenericExecutable - A role for plugins which allow you to use any executable as a
       transformer or validator


VERSION
       version 0.84


SYNOPSIS
           package Whatever;
           use Moo;
           with 'Code::TidyAll::Role::GenericExecutable';


DESCRIPTION
       This role exists for the benefit of the Code::TidyAll::Plugin::GenericTransformer and
       Code::TidyAll::Plugin::GenericValidator plugin classes.


ATTRIBUTES
       cmd This attribute is require for any class which consumes this role.

       file_flag
           If  this  is  set  then  this  flag  is  used to indicate the file passed to the command, for example
           something like "-f", "--file", or "--input". By default, the  file  is  simply  passed  as  the  last
           argument to the command.
