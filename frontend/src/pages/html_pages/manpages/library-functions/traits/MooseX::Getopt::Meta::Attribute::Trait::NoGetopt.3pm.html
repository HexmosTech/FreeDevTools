NAME
       MooseX::Getopt::Meta::Attribute::Trait::NoGetopt - Optional meta attribute trait for ignoring parameters


VERSION
       version 0.78


SYNOPSIS
         package App;
         use Moose;

         with 'MooseX::Getopt';

         has 'data' => (
             traits  => [ 'NoGetopt' ],  # do not attempt to capture this param
             is      => 'ro',
             isa     => 'Str',
             default => 'file.dat',
         );


DESCRIPTION
       This is a custom attribute metaclass trait which can be used to specify that a specific attribute should
       not be processed by "MooseX::Getopt". All you need to do is specify the "NoGetopt" metaclass trait.

         has 'foo' => (traits => [ 'NoGetopt', ... ], ... );
