NAME
       File::ChangeNotify::Watcher::Default - Fallback default watcher subclass


VERSION
       version 0.31


DESCRIPTION
       This class implements watching by comparing two snapshots of the filesystem tree. It if inefficient and
       dumb, and so it is the subclass of last resort.

       Its "$watcher->wait_for_events" method sleeps between comparisons of the filesystem snapshot it takes.
