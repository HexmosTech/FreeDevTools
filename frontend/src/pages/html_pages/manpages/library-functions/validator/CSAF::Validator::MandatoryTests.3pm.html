NAME
       CSAF::Validator::MandatoryTests


SYNOPSIS
           use CSAF::Validator::MandatoryTests;

           my $v = CSAF::Validator::MandatoryTests->new( csaf => $csaf );

           $v->exec_test('6.1.5');
           $v->TEST_6_1_5;


DESCRIPTION
       Mandatory tests MUST NOT fail at a valid CSAF document.
