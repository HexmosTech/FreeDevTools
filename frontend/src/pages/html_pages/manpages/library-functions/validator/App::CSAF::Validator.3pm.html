NAME
       App::CSAF::Validator - Validator Command Line Interface


SYNOPSIS
           use App::CSAF::Validator qw(run);

           run(\@ARGV);


DESCRIPTION
       App::CSAF::Validator is a "Command Line Interface" helper module for "csaf-validator(1)" command.

   METHODS
       App::CSAF::Validator->run(@args)

       Execute the command
