NAME
       Audio::MPD::Common::Item - a generic collection item


VERSION
       version 2.003


SYNOPSIS
           my $item = Audio::MPD::Common::Item->new( %params );


DESCRIPTION
       Audio::MPD::Common::Item is a virtual class representing a generic item of mpd's collection. It can be
       either a song, a directory or a playlist.

       Depending on the params given to "new", it will create and return an Audio::MPD::Common::Item::Song, an
       Audio::MPD::Common::Item::Directory or an Audio::MPD::Common::Playlist object. Currently, the
       discrimination is done on the existence of the "file" key of %params.


METHODS
   my $item = Audio::MPD::Common::Item->new( %params );
       Create and return either an Audio::MPD::Common::Item::Song, an Audio::MPD::Common::Item::Directory or an
       Audio::MPD::Common::Playlist object, depending on the existence of a key "file", "directory" or
       "playlist" (respectively).
