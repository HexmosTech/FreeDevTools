Detailed Description

   template<typename _Tp>
       class __gnu_cxx::new_allocator< _Tp >"An allocator that uses global new, as per C++03 [20.4.1].

       This is precisely the allocator defined in the C++ Standard.

       • all allocation calls operator new

       • all deallocation calls operator delete

       This  is  a  non-standard extension that can be used to guarantee allocation from new even if the library
       has been configured to use a different implementation for std::allocator.
