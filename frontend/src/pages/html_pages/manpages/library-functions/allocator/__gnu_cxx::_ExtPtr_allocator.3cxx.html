Detailed Description

   template<typename _Tp>
       class __gnu_cxx::_ExtPtr_allocator< _Tp >"An example allocator which uses a non-standard pointer type.

       This allocator specifies that containers use a 'relative pointer' as it's pointer type. (See
       ext/pointer.h) Memory allocation in this example is still performed using std::allocator.
