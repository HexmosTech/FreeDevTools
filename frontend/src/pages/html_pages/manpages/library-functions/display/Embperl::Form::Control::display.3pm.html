NAME
       Embperl::Form::Control::display - A text display control inside an Embperl Form


SYNOPSIS
         {
         type   => 'display',
         text   => 'blabla',
         hidden => 1,
         name   => 'foo',
         split  => 1
         }


DESCRIPTION
       Used to create a display only control inside an Embperl Form.  See Embperl::Form on how to specify
       parameters.

   PARAMETER
       type

       Needs to be set to 'display'.

       text

       Will be used as label for the text display control.

       value

       value to display. If not given $fdat{<name>} will be used.  If the data given within value is an
       arrayref, every element will be displayed on a separate line.

       hidden

       If set, an appropriate hidden input field will be created automatically.

       name

       Will be used as name for the hidden input field.

       split

       Splits the value into an array at \t if set and displays every array element on a new line.

       splitlines

       Splits the value into an array at \n if set and displays every array element on a new line.

       value2text

       Will run the value prefixed with the given paramenter through convert_text, so it can be translated.
