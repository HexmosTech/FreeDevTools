NAME
       Perl::Critic::Policy::BuiltinFunctions::ProhibitSleepViaSelect - Use Time::HiRes instead of something
       like "select(undef, undef, undef, .05)".


AFFILIATION
       This Policy is part of the core Perl::Critic distribution.


DESCRIPTION
       Conway discourages the use of select() for performing non-integer sleeps.  Although documented in
       perlfunc, it's something that generally requires the reader to read "perldoc -f select" to figure out
       what it should be doing.  Instead, Conway recommends that you use the "Time::HiRes" module when you want
       to sleep.

           select undef, undef, undef, 0.25;         # not ok

           use Time::HiRes;
           sleep( 0.25 );                            # ok


CONFIGURATION
       This Policy is not configurable except for the standard options.


SEE ALSO
       Time::HiRes.
