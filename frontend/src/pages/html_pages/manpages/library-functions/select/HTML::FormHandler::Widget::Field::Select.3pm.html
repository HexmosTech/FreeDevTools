NAME
       HTML::FormHandler::Widget::Field::Select - select field rendering widget


VERSION
       version 0.40068


DESCRIPTION
       Renders single and multiple selects. Options hashrefs must have 'value' and 'label' keys, and may have an
       'attributes' key.

       See HTML::FormHandler::Field::Select for documentation on select fields and options.
