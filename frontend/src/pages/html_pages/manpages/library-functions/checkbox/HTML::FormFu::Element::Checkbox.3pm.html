NAME
       HTML::FormFu::Element::Checkbox - Checkbox form field


VERSION
       version 2.07


SYNOPSIS
           my $e = $form->element( Checkbox => 'foo' );


DESCRIPTION
       Checkbox form field.


METHODS
   value
       Default Value: 1

   default_empty_value
       Inherited. See "default_empty_value" in HTML::FormFu::Role::Element::Field for details.

   multi_layout
       Overrides the default value of multi_layout to swap the "field" and "label" around.

       Is a sub-class of, and inherits methods from HTML::FormFu::Role::Element::Input,
       HTML::FormFu::Role::Element::Field, HTML::FormFu::Element
