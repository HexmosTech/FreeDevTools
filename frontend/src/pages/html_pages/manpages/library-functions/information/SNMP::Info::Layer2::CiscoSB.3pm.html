NAME
       SNMP::Info::Layer2::CiscoSB - SNMP Interface to Cisco Small Business series


AUTHOR
       Nic Bernstein (shamelessly stolen from Max Baker's Aironet code)


SYNOPSIS
        # Let SNMP::Info determine the correct subclass for you.
        my $ciscosb = new SNMP::Info(
                                 AutoSpecify => 1,
                                 Debug       => 1,
                                 DestHost    => 'myswitch',
                                 Community   => 'public',
                                 Version     => 2
                               )
           or die "Can't connect to DestHost.\n";

        my $class      = $ciscosb->class();
        print "SNMP::Info determined this device to fall under subclass : $class\n";


DESCRIPTION
       Provides interface to SNMP Data available on Cisco Small Business (nee LinkSys) managed switches. [i.e.
       those matching enterprises(1).cisco(9).otherEnterprises(6).ciscosb(1)]

   Inherited Classes
       SNMP::Info::CDP
       SNMP::Info::CiscoConfig
       SNMP::Info::CiscoStats
       SNMP::Info::EtherLike
       SNMP::Info::Layer2

   Required MIBs
       CISCOSB-DEVICEPARAMS-MIB
       CISCOSB-POE-MIB
       Inherited Classes
           MIBs required by the inherited classes listed above.


GLOBALS
       These are methods that return scalar value from SNMP.

       $ciscosb->mac()
           Returns mac from ("rndBasePhysicalAddress")

       $ciscosb->os_ver()
           Returns software version ("entPhysicalSoftwareRev")

       $ciscosb->serial()
           Returns serial number of unit ("entPhysicalSerialNum")

       $ciscosb->model()
           Returns model and hardware revision of unit ("entPhysicalModelName+entPhysicalHardwareRev")

   Overrides
       $ciscosb->vendor()
           Returns 'cisco'.

       $ciscosb->os()
           Returns 'ros'.

   Globals imported from SNMP::Info::Layer2
       See documentation in "GLOBALS" in SNMP::Info::Layer2 for details.

   Globals imported from SNMP::Info::EtherLike
       See documentation in "GLOBALS" in SNMP::Info::EtherLike for details.


TABLE METHODS
       $ciscosb->peth_port_power()
           Power supplied by PoE ports, in milliwatts.  ("rlPethPsePortOutputPower")

       $ciscosb->i_duplex()
           Return duplex based upon the result of EtherLike->el_duplex().

   Overrides
       $ciscosb->interfaces()
           Uses the i_name() field.
