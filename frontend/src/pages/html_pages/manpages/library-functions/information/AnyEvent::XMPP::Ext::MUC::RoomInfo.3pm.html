NAME
       AnyEvent::XMPP::Ext::MUC::RoomInfo - Room information


SYNOPSIS
DESCRIPTION
       This module represents the room information for a MUC.


METHODS
       new (%args)
       disco_info
           This  method  returns  the info discovery object AnyEvent::XMPP::Ext::Disco::Info for the disco query
           that this roominfo was obtained from.

       as_debug_string
           Returns the MUC room information as string for debugging.
