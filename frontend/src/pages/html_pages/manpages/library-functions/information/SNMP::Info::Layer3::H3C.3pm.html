NAME
       SNMP::Info::Layer3::H3C - SNMP Interface to L3 Devices, H3C & HP A-series


AUTHORS
       Jeroen van Ingen


SYNOPSIS
        # Let SNMP::Info determine the correct subclass for you.
        my $h3c = new SNMP::Info(
                                 AutoSpecify => 1,
                                 Debug       => 1,
                                 DestHost    => 'myrouter',
                                 Community   => 'public',
                                 Version     => 2
                               )
           or die "Can't connect to DestHost.\n";

        my $class      = $h3c->class();
        print "SNMP::Info determined this device to fall under subclass : $class\n";


DESCRIPTION
       Subclass for H3C & HP A-series devices

   Inherited Classes
       SNMP::Info::Layer3
       SNMP::Info::IEEE802dot3ad

   Required MIBs
       HH3C-LswDEVM-MIB
       HH3C-LswINF-MIB
       HH3C-LSW-DEV-ADM-MIB
       HH3C-PRODUCT-ID-MIB
       HH3C-ENTITY-VENDORTYPE-OID-MIB
       Inherited Classes' MIBs
           See SNMP::Info::IEEE802dot3ad for its own MIB requirements.

           See SNMP::Info::Layer3 for its own MIB requirements.


GLOBALS
       These are methods that return scalar value from SNMP

       $h3c->model()
           Returns the model from "sysObjectID" or "entPhysicalClass".

       $h3c->vendor()
           Returns value for "entPhysicalMfgName.1".

       $h3c->os()
           Returns the OS extracted from "sysDescr".

       $h3c->os_ver()
           Returns the software version. Either "entPhysicalSoftwareRev.2" or extracted from "sysDescr".

       $h3c->qb_fdb_index()
           overwritten to always return empty hash as implementation on h3c is broken.

   Globals imported from SNMP::Info::Layer3
       See documentation in SNMP::Info::Layer3 for details.

   Globals imported from SNMP::Info::IEEE802dot3ad
       See documentation in SNMP::Info::IEEE802dot3ad for details.


TABLE ENTRIES
       These are methods that return tables of information in the form of a reference to a hash.

   Overrides
       $h3c->i_ignore()
           Returns reference to hash.  Increments value of IID if port is to be ignored.

           Ignores loopback

       "agg_ports"
           Returns  a  HASH  reference mapping from slave to master port for each member of a port bundle on the
           device. Keys are ifIndex of the slave ports, Values are ifIndex of the corresponding master ports.

   Table Methods imported from SNMP::Info::Layer3
       See documentation in SNMP::Info::Layer3 for details.

   Table Methods imported from SNMP::Info::IEEE802dot3ad
       See documentation in SNMP::Info::IEEE802dot3ad for details.
