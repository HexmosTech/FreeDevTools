NAME
       SNMP::Info::Layer2::Hirschmann - SNMP Interface to L2 Hirschmann Switches


AUTHOR
       Christophe COMTE


SYNOPSIS
        # Let SNMP::Info determine the correct subclass for you.
        my $router = new SNMP::Info(
                                 AutoSpecify => 1,
                                 Debug       => 1,
                                 DestHost    => 'myrouter',
                                 Community   => 'public',
                                 Version     => 2
                               )
           or die "Can't connect to DestHost.\n";

        my $class      = $router->class();
        print "SNMP::Info determined this device to fall under subclass : $class\n";


DESCRIPTION
       Subclass for Hirschmann L2 devices

   Inherited Classes
       SNMP::Info::Layer2
       SNMP::Info::CDP

   Required MIBs
       HMPRIV-MGMT-SNMP-MIB
       Inherited Classes
           MIBs required by the inherited classes listed above.


GLOBALS
       These are methods that return scalar value from SNMP

   Overrides
       $device->vendor()
           Returns 'hirschmann'

       $device->os()
           Returns 'hirschmann'

       $device->os_ver()
           Return os version

       $device->mac()
           Return Interface MAC addresse of the switch (on the CPU pseudo interface).

       $device->model()
           Returns device model extracted from description

       $device->serial()
           Returns serial number
