Detailed Description
       A structure used to get the information of flow director filter. It supports RTE_ETH_FILTER_FDIR with
       RTE_ETH_FILTER_INFO operation. It includes the mode, flexible payload configuration information,
       capabilities and supported flow types, flexible payload characters. It can be gotten to help taking
       specific configurations per device.
