NAME
       SNMP::Info::Layer3::ERX - SNMP Interface to Juniper ERX Layer 3 routers.


AUTHORS
       Rob Woodward


SYNOPSIS
        # Let SNMP::Info determine the correct subclass for you.
        my $ERX = new SNMP::Info(
                                 AutoSpecify => 1,
                                 Debug       => 1,
                                 DestHost    => 'myrouter',
                                 Community   => 'public',
                                 Version     => 2
                               )
           or die "Can't connect to DestHost.\n";

        my $class      = $ERX->class();
        print "SNMP::Info determined this device to fall under subclass : $class\n";


DESCRIPTION
       Subclass for Juniper ERX switches

   Inherited Classes
       SNMP::Info::Layer3

   Required MIBs
       Inherited Classes' MIBs
           See SNMP::Info::Layer3 for its own MIB requirements.


GLOBALS
       These are methods that return scalar value from SNMP

       $ERX->vendor()
           Returns 'juniper'.

       $ERX->os()
           Returns 'JunOSe'.

       $ERX->os_ver()
           Returns the software version extracted from "sysDescr".

       $ERX->model()
           Returns the hardware model extracted from "sysDescr".

   Globals imported from SNMP::Info::Layer3
       See documentation in SNMP::Info::Layer3 for details.


TABLE ENTRIES
       These are methods that return tables of information in the form of a reference to a hash.

   Overrides
       $ERX->i_ignore()
           Returns reference to hash. Increments value of IID if port is to be ignored.

           Ignores InLoopback and Console interfaces

   Table Methods imported from SNMP::Info::Layer3
       See documentation in SNMP::Info::Layer3 for details.
