NAME
       gfs_stat_free - free dynamic memory of file status


SYNOPSIS
       #include <gfarm/gfarm.h>

       void gfs_stat_free(struct gfs_stat * status);


DESCRIPTION
       gfs_stat_free() frees the memory space of file status status allocated by gfs_stat(3).
