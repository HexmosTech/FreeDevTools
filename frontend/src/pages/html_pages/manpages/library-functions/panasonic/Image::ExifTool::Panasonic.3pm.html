NAME
       Image::ExifTool::Panasonic - Panasonic/Leica maker notes tags


SYNOPSIS
       This module is loaded automatically by Image::ExifTool when required.


DESCRIPTION
       This module contains definitions required by Image::ExifTool to interpret Panasonic and Leica maker notes
       in EXIF information.
