NAME
       MAB2::Writer::Handle - Utility class for common MAB2::Writer arguments and methods.


Arguments
       "file"
           Path to file.

       "fh"
           Open filehandle.

       "encoding"
           Set encoding.


METHODS
   _set_fh()
       Open filehandle (with specified encoding) from file.

   close_fh()
       Close filehandle.

   write()
       Write record to filehandle.
