NAME
       HTML::Widgets::NavMenu::Iterator::JQTreeView - an iterator for JQuery TreeView's navigation menus.


VERSION
       version 1.1000


SYNOPSIS
       See <http://bassistance.de/jquery-plugins/jquery-plugin-treeview/> .

       For internal use only.


METHODS
   get_currently_active_text ( $node )
       Calculates the highlighted text for the node $node. Normally surrounds it with "<b> ... </b>" tags.
