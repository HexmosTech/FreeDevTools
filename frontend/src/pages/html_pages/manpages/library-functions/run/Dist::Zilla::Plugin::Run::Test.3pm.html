NAME
       Dist::Zilla::Plugin::Run::Test - execute a command of the distribution after build


VERSION
       version 0.050


SYNOPSIS
         [Run::Test]
         run = script/tester.pl --name %n --version %v some_file.ext


DESCRIPTION
       This plugin executes the specified command during the test phase.


CAVEAT
       Unlike the other [Run::*] plugins, when running the scripts, the current working directory will be the
       directory with newly built distribution. This is the way Dist::Zilla works.


POSITIONAL PARAMETERS
       See "CONVERSIONS" in Dist::Zilla::Plugin::Run for the list of common formatting variables available to
       all plugins.

       There are no positional parameters for this plugin.
