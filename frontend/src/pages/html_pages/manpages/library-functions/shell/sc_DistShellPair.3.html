Member Function Documentation
   int sc::DistShellPair::get_task (int & P, int & Q)
       Puts the current PQ shell pair into P and Q and returns 1. When there are no more shell pairs to be
       processed by this processor, 0 is returned. Once we start doing get_tasks, we have to go to the end if
       dynamic load balancing is used.

       P belongs to bs1, and Q belongs to bs2. If (bs1 == bs2) then P is greater or equal to Q.
