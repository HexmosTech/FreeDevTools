Function Documentation
   int daemon_exec (const char * dir, int * ret, const char * prog,  ...)
       Run the specified executable with the specified arguments in the specified directory and return the
       return value of the program in the specified pointer. The calling process is blocked until the child
       finishes and all child output (either STDOUT or STDIN) has been written to syslog. Running this function
       requires that daemon_signal() has been called with SIGCHLD as argument.

       Parameters
           dir Working directory for the process.
           ret A pointer to an integer to write the return value of the program to.
           prog The path to the executable
           ... The arguments to be passed to the program, followed by a (char *) NULL

       Returns
           Nonzero on failure, zero on success

       Since
           0.4

       See also
           DAEMON_EXEC_AVAILABLE

           testd.c.
