NAME
       s3d_push_texture - push texture


SYNOPSIS
       #include <s3d.h>

       int s3d_push_texture(int object, uint16_t w, uint16_t h);


DESCRIPTION
       Adds a new texture with height w and height h on the texture stack.

           Author of s3d
