NAME
       SDL_BindGPUFragmentSamplers - Binds texture-sampler pairs for use on the fragment shader.


HEADER FILE
       Defined in SDL3/SDL_gpu.h


SYNOPSIS
       #include "SDL3/SDL.h"

       void SDL_BindGPUFragmentSamplers(
           SDL_GPURenderPass *render_pass,
           Uint32 first_slot,
           const SDL_GPUTextureSamplerBinding *texture_sampler_bindings,
           Uint32 num_bindings);


DESCRIPTION
       The textures must have been created with

       SDL_GPU_TEXTUREUSAGE_SAMPLER .

       Be sure your shader is set up according to the requirements documented in

       SDL_CreateGPUShader ().


FUNCTION PARAMETERS
       render_pass
              a render pass handle.

       first_slot
              the fragment sampler slot to begin binding from.

       texture_sampler_bindings
              an array of texture-sampler binding structs.

       num_bindings
              the number of texture-sampler pairs to bind from the array.


AVAILABILITY
       This function is available since SDL 3.2.0.
