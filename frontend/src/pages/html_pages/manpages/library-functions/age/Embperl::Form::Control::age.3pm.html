NAME
       Embperl::Form::Control::age - A age input control with optional unit inside an Embperl Form


SYNOPSIS
         {
         type => 'age',
         text => 'blabla',
         name => 'foo',
         unit => 'sec',
         }


DESCRIPTION
       Used to create an age input control inside an Embperl Form.  Will format date as days:hours:minutes from
       current time.  Optionally it can display an unit after the input field.  See Embperl::Form on how to
       specify parameters.

   PARAMETER
       type

       Needs to be 'age'

       name

       Specifies the name of the control

       text

       Will be used as label for the numeric input control

       size

       Gives the size in characters. (Default: 10)

       maxlength

       Gives the maximun length in characters

       unit

       Gives a string that should be displayed right of the input field.  (
