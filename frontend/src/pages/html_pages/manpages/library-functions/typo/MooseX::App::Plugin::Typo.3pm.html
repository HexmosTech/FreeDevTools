NAME
       MooseX::App::Plugin::Typo - Handle typos in command names


SYNOPSIS
       In your base class:

        package MyApp;
        use MooseX::App qw(Typo);

       In your shell

        bash$ myapp pusl
        Ambiguous command 'pusl'
        Which command did you mean?
        * push
        * pull


DESCRIPTION
       This plugin tries to handle typos in command names
