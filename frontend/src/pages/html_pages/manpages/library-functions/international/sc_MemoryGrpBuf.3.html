Detailed Description
   template<class data_t>
       class sc::MemoryGrpBuf< data_t >" The MemoryGrpBuf class provides access to pieces of the global shared
       memory that have been obtained with MemoryGrp.

       MemoryGrpBuf is a template class that is parameterized on data_t. All lengths and offsets of given in
       terms of sizeof(data_t).

       Request read write access to global memory at the global address offset and with size length. This will
       lock the memory it uses until release is called unless locking has been turned off in the MemoryGrp
       object.

       Request write only access to global memory at the global address offset and with size length. Writing the
       same bit of memory twice without an intervening sync of the MemoryGrp will have undefined results.
