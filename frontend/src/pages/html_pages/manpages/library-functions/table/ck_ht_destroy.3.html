NAME
       ck_ht_destroy — immediately destroy a hash table


LIBRARY
       Concurrency Kit (libck, -lck)


SYNOPSIS
       #include <ck_ht.h>

       void
       ck_ht_destroy(ck_ht_t *ht);


DESCRIPTION
       The   ck_ht_destroy()  function  will  request  that  the  underlying  allocator,  as  specified  by  the
       ck_ht_init(3) function, immediately destroy the object pointed to by the ht argument.

       The user must guarantee that no threads are accessing the object pointed to by ht when ck_ht_destroy() is
       called.


RETURN VALUES
       ck_ht_destroy() has no return value.


ERRORS
       This function is guaranteed not to fail.
