   void __gnu_profile::__report () [inline]
       Final report method, registered with atexit. This can also be called directly by user code, including
       signal handlers. It is protected against deadlocks by the reentrance guard in profiler.h. However, when
       called from a signal handler that triggers while within __gnu_profile (under the guarded zone), no output
       will be produced.
