NAME
       Devel::REPL::Plugin::Peek - Devel::Peek plugin for Devel::REPL.


VERSION
       version 1.003029


SYNOPSIS
         repl> #peek "foo"
         SV = PV(0xb3dba0) at 0xb4abc0
           REFCNT = 1
           FLAGS = (POK,READONLY,pPOK)
           PV = 0x12bcf70 "foo"\0
           CUR = 3
           LEN = 4


DESCRIPTION
       This Devel::REPL::Plugin adds a "peek" command that calls "Dump" in Devel::Peek instead of the normal
       printing.


SEE ALSO
       Devel::REPL, Devel::Peek
