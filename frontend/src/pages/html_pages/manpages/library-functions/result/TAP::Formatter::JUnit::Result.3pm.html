NAME
       TAP::Formatter::JUnit::Result - Wrapper for a TAP result


DESCRIPTION
       "TAP::Formatter::JUnit::Result" is an internal class, used to wrap/augment "TAP::Parser::Result" objects
       with timing information.

       NOT recommended for public consumption; internal use only.
