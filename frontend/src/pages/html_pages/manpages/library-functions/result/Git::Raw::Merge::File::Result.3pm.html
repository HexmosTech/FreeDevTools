NAME
       Git::Raw::Merge::File::Result - Git merge result class


VERSION
       version 0.90


DESCRIPTION
       A Git::Raw::Merge::File::Results represents the result of a file merge.

       WARNING: The API of this module is unstable and may change without warning (any change will be
       appropriately documented in the changelog).


METHODS
   automergeable( )
       Return a truthy value if the file was automerged, or a false value if the content contains conflict
       markers.

   content( )
       The contents of the merge.

   path( )
       The path that the resultant merge file should use, or "undef" if a filename conflict would occur.
