NAME
       Mason::Result - Result returned from Mason request


SYNOPSIS
           my $interp = Mason->new(...);
           my $output = $result->output;
           my $data   = $result->data;


DESCRIPTION
       An object of this class is returned from "$interp->run". It contains the page output and any values set
       in "$m->result". Plugins may add additional accessors.


METHODS
       output
           The output of the entire page, unless out_method was defined in which case this will be empty.

       data
           A hashref of arbitrary data that can be set via

               $m->result->data->{'key'} = 'value';
