NAME
       Bio::DB::HTS::PileupWrapper -- Add high-level methods to Bio::DB::HTS::Pileup


SYNOPSIS
       See "The generic fetch() and pileup() methods" in Bio::DB::HTS for usage of the pileup() method.


DESCRIPTION
       See Bio::DB::HTS::Pileup for documentation of this object's methods. This class is used by the high-level
       API to return Bio::DB::HTS::AlignWrapper objects from the call to alignment() rather than
       Bio::DB::HTS::Alignment.
