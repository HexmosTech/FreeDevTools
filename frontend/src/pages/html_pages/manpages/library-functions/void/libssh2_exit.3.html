NAME
       libssh2_exit - global library deinitialization


SYNOPSIS
       #include <libssh2.h>

       void
       libssh2_exit(void);


DESCRIPTION
       Exit the libssh2 functions and frees all memory used internal.


AVAILABILITY
       Added in libssh2 1.2.5

       libssh2_init(3)
