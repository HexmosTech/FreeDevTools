NAME
       sasl_done - Cyrus SASL documentation


SYNOPSIS
       #include <sasl/sasl.h>

       void sasl_done( void );


DESCRIPTION
       void sasl_done(void);
              sasl_done() is  called  when  the application is completely done with the SASL library.


RETURN VALUE
       No return values.

       RFC 4422,:saslman:sasl(3), sasl_server_init(3), sasl_client_init(3)
