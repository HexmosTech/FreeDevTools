Detailed Description
       RTE Port

       This tool is part of the DPDK Packet Framework tool suite and provides a standard interface to implement
       different types of packet ports.

   typedef int(* rte_port_out_op_tx_bulk) (void *port, struct rte_mbuf **pkts, uint64_t pkts_mask)
       Output port packet burst TX

       Parameters
           port Handle to output port instance
           pkts Burst of input packets specified as array of up to 64 pointers to struct rte_mbuf
           pkts_mask 64-bit bitmask specifying which packets in the input burst are valid. When pkts_mask bit n
           is set, then element n of pkts array is pointing to a valid packet. Otherwise, element n of pkts
           array will not be accessed.

       Returns
           0 on success, error code otherwise
