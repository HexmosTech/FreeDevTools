NAME
       gclose_seg  - closes a GKS segment.


SYNOPSIS
       #include <ncarg/gks.h>

       void gclose_seg( void );


USAGE
       A  call  to  gclose_seg  closes  the currently open segment (graphics output primitives will no longer be
       stored in that segment).  gclose_seg can be called only if a segment is open.


ACCESS
       To use the GKS C-binding routines, load the ncarg_gks and ncarg_c libraries.
