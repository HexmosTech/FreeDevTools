NAME
       Autodia::Handler::umbrello - AutoDia handler for umbrello


DESCRIPTION
       This provides Autodia with the ability to read umbrello files, allowing you to convert them via the
       Diagram Export methods to images (using GraphViz and VCG) or html/xml using custom templates.

       The umbrello handler will parse umbrello xml/xmi files using XML::Simple and populating the diagram
       object with class, superclass and package objects.

       the umbrello handler is registered in the Autodia.pm module, which contains a hash of language names and
       the name of their respective language - in this case:


SYNOPSIS
       use Autodia::Handler::umbrello;

       my $handler = Autodia::Handler::umbrello->New(\%Config);

       $handler->Parse(filename); # where filename includes full or relative path.


METHODS
   CONSTRUCTION METHOD
       use Autodia::Handler::umbrello;

       my $handler = Autodia::Handler::umbrello->New(\%Config); This creates a new handler using the
       Configuration hash to provide rules selected at the command line.

   ACCESS METHODS
       $handler->Parse(filename); # where filename includes full or relative path.

       This parses the named file and returns 1 if successful or 0 if the file could not be opened.
