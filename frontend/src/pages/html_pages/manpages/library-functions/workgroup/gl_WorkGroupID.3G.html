NAME
       gl_WorkGroupID - contains the index of the workgroup currently being operated on by a compute shader


DECLARATION
       in uvec3 gl_WorkGroupID ;.SH "DESCRIPTION"

       In the compute language, gl_WorkGroupID contains the 3-dimensional index of the global work group that
       the current compute shader invocation is executing within. The possible values range across the
       parameters passed into glDispatchCompute, i.e., from (0, 0, 0) to (gl_NumWorkGroups.x - 1,
       gl_NumWorkGroups.y - 1, gl_NumWorkGroups.z - 1).
