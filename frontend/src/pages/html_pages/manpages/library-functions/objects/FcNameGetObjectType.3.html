NAME
       FcNameGetObjectType - Lookup an object type


SYNOPSIS
       #include <fontconfig/fontconfig.h>

       const FcObjectType * FcNameGetObjectType (const char *object);


DESCRIPTION
       Return the object type for the pattern element named object.
