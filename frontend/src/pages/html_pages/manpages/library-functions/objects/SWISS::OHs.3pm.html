Name
       SWISS::OHs


Description
       SWISS::OHs represents the OH lines within an SWISS-PROT + TrEMBL entry as specified in the user manual
        http://www.expasy.org/sprot/userman.html . The OHs object is a container
        object which holds a list of SWISS::OH objects.
