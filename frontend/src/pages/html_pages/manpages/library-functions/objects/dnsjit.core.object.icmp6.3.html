NAME
       dnsjit.core.object.icmp6 - An ICMPv6 packet


DESCRIPTION
       An  ICMPv6  packet which is usually at the top of the object chain after parsing with, for example, Layer
       filter.

   Attributes
       type   The type of ICMPv6 message.

       code   The (response/error) code for the ICMPv6 type message.

       cksum  The ICMPv6 checksum.

   Functions
       Icmp6:type()
              Return the textual type of the object.

       Icmp6:prev()
              Return the previous object.

       Icmp6:cast()
              Cast the object to the underlining object module and return it.

       Icmp6:uncast()
              Cast the object to the generic object module and return it.

       Icmp6:copy()
              Make a copy of the object and return it.

       Icmp6:free()
              Free the object, should only be used on copies or otherwise allocated.
