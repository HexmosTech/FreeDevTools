NAME
       dnsjit.core.object.udp - A UDP datagram header


DESCRIPTION
       A  UDP  datagram  header.  The data itself is in the dnsjit.core.object.payload object, which is the next
       object in the chain after parsing with, for example, dnsjit.filter.layer.

   Attributes
       sport  Source port.

       dport  Destination port.

       ulen   UDP length (as described in the UDP header).

       sum    Checksum.

   Functions
       Udp:type()
              Return the textual type of the object.

       Udp:prev()
              Return the previous object.

       Udp:cast()
              Cast the object to the underlining object module and return it.

       Udp:uncast()
              Cast the object to the generic object module and return it.

       Udp:copy()
              Make a copy of the object and return it.

       Udp:free()
              Free the object, should only be used on copies or otherwise allocated.
