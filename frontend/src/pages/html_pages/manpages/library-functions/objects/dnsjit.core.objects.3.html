NAME
       dnsjit.core.objects - Easy way to import all objects


SYNOPSIS
         require("dnsjit.core.objects")


DESCRIPTION
       Helper  module  to  require  all  available  objects,  returns  what  dnsjit.core.object returned so that
       constants like object types can be used.
