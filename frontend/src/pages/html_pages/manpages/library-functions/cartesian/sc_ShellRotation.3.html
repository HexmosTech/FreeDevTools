NAME
       sc::ShellRotation - Compute the transformation matrices that maps a set of Cartesian functions to another
       set of Cartesian functions in a rotated coordinate system.

Detailed Description
       Compute the transformation matrices that maps a set of Cartesian functions to another set of Cartesian
       functions in a rotated coordinate system.
