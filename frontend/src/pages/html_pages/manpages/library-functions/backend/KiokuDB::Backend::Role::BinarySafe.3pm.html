NAME
       KiokuDB::Backend::Role::BinarySafe - An informational role for binary data safe backends.


VERSION
       version 0.57


SYNOPSIS
           package KiokuDB::Backend::MySpecialBackend;
           use Moose;

           use namespace::clean -except => 'meta';

           with qw(KiokuDB::Backend::Role::BinarySafe);


DESCRIPTION
       This backend is an informational role for backends which can store arbitrary binary strings, especially
       utf8 data as bytes (without reinterpreting it as unicode strings when inflating).

       This mostly has to do with KiokuDB::Backend::Serialize variants (for example
       KiokuDB::Backend::Serialize::Storable is binary safe, while KiokuDB::Backend::Serialize::JSON is not).
