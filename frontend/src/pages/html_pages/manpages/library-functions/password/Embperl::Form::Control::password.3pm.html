NAME
       Embperl::Form::Control::password - A password input control inside an Embperl Form


SYNOPSIS
         {
         type => 'password',
         text => 'blabla',
         name => 'foo',
         size => 10,
         }


DESCRIPTION
       Used to create a password control inside an Embperl Form.  See Embperl::Form on how to specify
       parameters.

   PARAMETER
       type

       Needs to be 'password'

       name

       Specifies the name of the control

       text

       Will be used as label for the text input control

       size

       Gives the size in characters

       retype_name

       Name of control that is used to repeat the password.  An automatic validation rule will be generated, to
       make sure both inputs are equal.

       maxlength

       Gives the maximun length in characters
