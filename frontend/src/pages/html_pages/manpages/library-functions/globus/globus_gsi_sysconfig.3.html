Detailed Description
       Globus GSI System Config API.

       This API provides helper functions for detecting installation and environment specific settings
       applicable to GSI. It also servers as a abstraction layer for OS specific programming details. The public
       interface for this library consists of the GLOBUS_GSI_SYSCONFIG_* macros that point at the resolve to
       platform specific functions. You should never use the platform specific functions directly..

       Any program that uses Globus GSI System Config functions must include the globus_gsi_system_config.h
       header.
