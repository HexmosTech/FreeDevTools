NAME
       Arch - GNU Arch Perl library


SYNOPSIS
           use Arch 0.5.2;

           # perldoc Arch
           # axp man Arch

           # example: produce ChangeLog for the current project
           use Arch::Tree;

           foreach my $log (Arch::Tree->new->get_logs) {
               print "-" x 80, "\n";
               print $log->standard_date, "\n";
               print $log->summary, "\n\n";
               print $log->body;
           }


DESCRIPTION
       The Arch-Perl library allows Perl developers to create GNU Arch front-ends in an object oriented fashion.
       GNU Arch is a decentralized, changeset-oriented revision control system.

       Currently, a pragmatic high-level interface is built around tla or baz.  This functionality was initially
       developed for ArchZoom project, and was highly enhanced to serve AXP and ArchWay projects as well.
