NAME
       KiokuDB::TypeMap::Entry::Std::Intrinsic - Provides a compile_collapse implementation.


VERSION
       version 0.57


SYNOPSIS
       TODO


DESCRIPTION
       This role provides a compile_collapse implementation by breaking it down and requiring the implementation
       of a 'compile_collapse_body' method.


REQUIRED METHODS
       compile_collapse_body
           Must  return a code reference or a method name.  The method is called on the KiokuDB::Collapser, with
           a hash of parameters.  The parameters can include (but are not limited to) the following:

           object  The object to collapse.

           id      The id under which the object is to be stored.

           class   The type of "object".

           The method should return a KiokuDB::Reference object.
