NAME
       nozzle_set_mtu - set mtu on a given nozzle interface


SYNOPSIS
       #include <libnozzle.h>

       int nozzle_set_mtu(
           nozzle_t   nozzle,
           const int  mtu
       );


DESCRIPTION
       nozzle_set_mtu

       nozzle - pointer to the nozzle struct

       mtu - new MTU value


RETURN VALUE
       0 on success -1 on error and errno is set.
