NAME
       Slurm::Constant - Constants for use with Slurm


SYNOPSIS
        use Slurm qw(:constant);

        if ($rc != SLURM_SUCCESS {
                print STDERR "action failed!\n";
        }


DESCRIPTION
       This package export constants for use with Slurm. This includes enumerations and defined macros. The
       constants will be exported to package Slurm and the package which "use Slurm qw(:constant);".
