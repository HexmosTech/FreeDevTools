NAME
       observer - A GUI tool for observing an Erlang system.


DESCRIPTION
       Observer  is  a  graphical  tool  for  observing the characteristics of Erlang systems. The tool Observer
       displays system information, application  supervisor  trees,  process  information,  ETS  tables,  Mnesia
       tables, and contains a front end for Erlang tracing with module ttb.

       For details about how to get started, see the User's Guide.


EXPORTS
       start() -> ok

              Starts the Observer GUI. To stop the tool, close the window.
