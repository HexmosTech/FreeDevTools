NAME
       c_dsgetc - Retrieve values for char valued parameters


FUNCTION PROTOTYPE
       void c_dsgetc(char *, char *);


SYNOPSIS
       void c_dsgetc (char *pnam, char *cval);


DESCRIPTION
       pnam        A string that specifies the name of the parameter to be set.

       cval        A string specifying the value to be assigned to the internal parameter specified by pnam.


USAGE
       This routine allows you to set the current value of Dsgrid parameters.  For a complete list of parameters
       available in this utility, see the dsgrid_params man page.


ACCESS
       To use c_dsgetc, load the NCAR Graphics library ngmath.
