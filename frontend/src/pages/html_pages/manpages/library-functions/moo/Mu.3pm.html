NAME
       Mu - Moo but with less typing


VERSION
       version 1.191300


SYNOPSIS
           use Mu;

           ro "hro";
           lazy hlazy => sub { 2 };
           rwp "hrwp";
           rw "hrw";


DESCRIPTION
       Mu imports both Moo and MooX::ShortHas, making it even less work in typing and reading to set up an
       object.
