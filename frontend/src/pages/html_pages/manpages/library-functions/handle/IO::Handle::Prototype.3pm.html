NAME
       IO::Handle::Prototype - base class for callback based handles.


SYNOPSIS
           my $fh = IO::Handle::Prototype->new(
               getline => sub {
                   my $fh = shift;

                   ...
               },
           );


DESCRIPTION
       You probably want IO::Handle::Prototype::Fallback instead.
