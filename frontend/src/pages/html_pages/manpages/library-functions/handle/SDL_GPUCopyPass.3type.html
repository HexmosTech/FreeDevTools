NAME
       SDL_GPUCopyPass - An opaque handle representing a copy pass.


HEADER FILE
       Defined in SDL3/SDL_gpu.h


SYNOPSIS
       #include "SDL3/SDL.h"

       typedef struct SDL_GPUCopyPass SDL_GPUCopyPass;


DESCRIPTION
       This handle is transient and should not be held or referenced after

       SDL_EndGPUCopyPass
        is called.
