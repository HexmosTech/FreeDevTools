DESCRIPTION
       The ck_pr_neg(3) family of functions atomically negate the value pointed to by target.


RETURN VALUES
       The ck_pr_neg_zero functions set the value pointed to by z if the result of the negation operation was 0.
       They set the pointed to value to false otherwise.
