NAME
       nng_aio_result - return result of asynchronous operation


SYNOPSIS
           #include <nng/nng.h>

           int nng_aio_result(nng_aio *aio);


DESCRIPTION
       The nng_aio_result() returns the result of the operation associated with the handle aio. If the operation
       was successful, then 0 is returned. Otherwise a non-zero error code is returned.

           Note

           The return value from this function is undefined if the operation has not completed yet. Either call
           this from the handleâ€™s completion callback, or after waiting for the operation to complete with
           nng_aio_wait().


RETURN VALUES
       The result of the operation, either zero on success, or an error number on failure.


ERRORS
       NNG_ETIMEDOUT
           The operation timed out.

       NNG_ECANCELED
           The operation was canceled.

       Various other return values are possible depending on the operation.
