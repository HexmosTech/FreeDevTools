NAME
       MooseX::Has::Options::Handler::NoInit - Option shortcut for prohibiting init_arg


VERSION
       version 0.003


DESCRIPTION
       This module provides the following shortcut options for MooseX::Has::Options:

       :no_init
           Translates to "init_arg => undef"
