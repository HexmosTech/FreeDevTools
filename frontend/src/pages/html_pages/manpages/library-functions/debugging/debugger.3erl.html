NAME
       debugger - Erlang Debugger.


DESCRIPTION
       Erlang Debugger for debugging and testing of Erlang programs.


EXPORTS
       start()
       start(File)
       start(Mode)
       start(Mode, File)

              Types:

                 Mode = local | global
                 File = string()

              Starts Debugger.

              If  a  filename  is specified as argument, Debugger tries to load its settings from this file. For
              details about settings, see the User's Guide.

              If local is specified as argument, Debugger interprets code only at the current node. If global is
              specified as argument, Debugger interprets code at all known nodes, this is the default.

       quick(Module, Name, Args)

              Types:

                 Module = Name = atom()
                 Args = [term()]

              Debugs a single process. The module Module is interpreted and apply(Module,Name,Args)  is  called.
              This opens an Attach Process window. For details, see the User's Guide.
