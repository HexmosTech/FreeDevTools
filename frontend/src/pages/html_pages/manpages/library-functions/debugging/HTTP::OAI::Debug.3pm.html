NAME
       HTTP::OAI::Debug - debug the HTTP::OAI libraries


DESCRIPTION
       This package is a copy of LWP::Debug and exposes the same API. In addition to "trace", "debug" and
       "conns" this exposes a "sax" level for debugging SAX events.
