NAME
       Plack::Middleware::Debug::Base - Base class for Debug panels


SYNOPSIS
         package Plack::Middleware::Debug::YourPanel;
         use parent qw(Plack::Middleware::Debug::Base);

         sub run {
             my($self, $env, $panel) = @_;

             # Do something before the application runs

             return sub {
                 my $res = shift;

                 # Do something after the application returns

             };
         }


DESCRIPTION
       This is the base class for panels.


METHODS
       "run"
           This  method  is  called when a request has arrived, before the main application runs. The parameters
           are $env, the PSGI environment hash reference and $panel, a Plack::Middleware::Debug::Panel object.

           If your panel needs to do some response munging, you should return a callback  that  takes  $res  the
           response  object.  Because you can return a closure, the response filter can also use $env and $panel
           easily.
