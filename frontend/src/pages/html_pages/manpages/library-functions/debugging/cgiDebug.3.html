NAME
       cgiDebug - Set the debug level for CGI programming


SYNOPSYS
       #include <cgi.h>

       void cgiDebug (int level, int where);


DESCRIPTION
       This  routine controls debugging for the cgi library.  At the moment only level 0 (default, no debugging)
       and 1 (debugging enabled) are supported.  The second argument where specifies if debug output  should  be
       written to stdout using HTML (1), to stderr as plain text (0, default), or to syslog (2).

       This shoud be the first one called from the CGI library.


RETURN VALUE
       cgiDebug() does not return a value.
