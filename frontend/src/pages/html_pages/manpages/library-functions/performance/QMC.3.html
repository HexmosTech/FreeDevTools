NAME
       QMC - library for managing groups of Performance Co-Pilot metrics


C++ SYNOPSIS
       #include <qmc.h>

       c++ ... -lpcp_qmc -lpcp


DESCRIPTION
       The  Qt-based Performance Metrics Class (QMC) class library is a collection of C++ classes for managing a
       large set of performance metrics from a variety of sources.  This library is built over PMAPI(3).


DIAGNOSTICS
       Error messages are generated using pmprintf(3) but are not flushed. It is the responsibility of the  user
       to call pmflush(3) to output any messages.

       Additional  diagnostics  may be activated by adding the option pmc to the global debugging specification,
       as described in pmSetDebug(3).


SEE ALSO
       PMAPI(3), QmcContext(3), QmcDesc(3), QmcGroup(3), QmcIndom(3), QmcMetric(3), QmcSource(3), pmflush(3) and
       pmprintf(3).
