NAME
       Catmandu::Store::ElasticSearch::Bag - Catmandu::Bag implementation for Elasticsearch


DESCRIPTION
       See the main documentation at Catmandu::Store::ElasticSearch.


METHODS
       This class inherits all the methods of Catmandu::Bag, Catmandu::CQLSearchable and Catmandu::Droppable.
       It also provides the following methods:

   create_index()
       This method is called automatically when the bag is instantiated. You only need to call it manually it
       after deleting the index with "drop" or the Elasticsearch API.
