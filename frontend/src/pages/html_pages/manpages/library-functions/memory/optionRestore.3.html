NAME
       optionRestore - restore option state from memory copy


SYNOPSIS
       #include <your-opts.h>
       cc [...] -o outfile infile.c -lopts [...]

       void optionRestore(tOptions * pOpts);


DESCRIPTION
       Copy  back  the option state from saved memory.  The allocated memory is left intact, so this routine can
       be called repeatedly without having to call optionSaveState again.  If you are restoring a state that was
       saved before the first call to optionProcess(3AO), then you may change  the  contents  of  the  argc/argv
       parameters to optionProcess.

       pOpts  program options descriptor


ERRORS
       If you have not called optionSaveState before, a diagnostic is printed to stderr and exit is called.
