NAME
       function::set_user_char - Writes a char value to user memory


SYNOPSIS
           set_user_char(addr:long,val:long)


ARGUMENTS
       addr
           The user address to write the char to

       val
           The char which is to be written


DESCRIPTION
       Writes the char value to a given user memory address. Reports an error when writing to the given address
       fails. Requires the use of guru mode (-g).
