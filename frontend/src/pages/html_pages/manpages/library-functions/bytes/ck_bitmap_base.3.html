NAME
       ck_bitmap_base — determine the size of a bit array in bytes


LIBRARY
       Concurrency Kit (libck, -lck)


SYNOPSIS
       #include <ck_bitmap.h>

       unsigned int
       ck_bitmap_base(unsigned int n_bits);


DESCRIPTION
       The  ck_bitmap_base() function returns the number of bytes that would be used to store the number of bits
       specified by n_bits.


RETURN VALUES
       This function returns a non-zero value that is guaranteed to be a multiple of sizeof(CK_BITMAP_WORD).
