NAME
       CGI::FormBuilder::Template::Builtin - Builtin HTML rendering


SYNOPSIS
           my $form = CGI::FormBuilder->new;
           $form->render;


DESCRIPTION
       This module provides default rendering for FormBuilder. It is automatically called by FormBuilder's
       "render()" method if no external template is specified.  See the documentation in CGI::FormBuilder for
       more details.


SEE ALSO
       CGI::FormBuilder, CGI::FormBuilder::Template::HTML, CGI::FormBuilder::Template::Text,
       CGI::FormBuilder::Template::TT2, CGI::FormBuilder::Template::Fast, CGI::FormBuilder::Template::CGI_SSI
