NAME
       MPI_Wtick -  Returns the resolution of MPI_Wtime


SYNOPSIS
       #include <mpi.h>
       double MPI_Wtick(void)


RETURN VALUE
       Time in seconds of resolution of MPI_Wtime .


NOTES FOR FORTRAN
       This is function is declared as DOUBLE PRECISION MPI_WTICK() in Fortran.

       MPI_Wtime
