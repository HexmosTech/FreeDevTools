NAME
       mtext_putenv -


SYNOPSIS
       int mtext_putenv (MText * mt)


DESCRIPTION
       Change or add an environment variable. The mtext_putenv() function changes or adds the value of
       environment variables according to M-text mt. It calls the function putenv with an argument generated by
       encoding mt according to the current locale (LC_CTYPE).


RETURN VALUE
           This function returns zero on success, or -1 if an error occurs.
