Name
       App::Sqitch::Engine::sqlite - Sqitch SQLite Engine


Synopsis
         my $sqlite = App::Sqitch::Engine->load( engine => 'sqlite' );


Description
       App::Sqitch::Engine::sqlite provides the SQLite storage engine for Sqitch.


Interface
   Accessors
       "client"

       Returns the path to the SQLite client. If "--client" was passed to "sqitch", that's what will be
       returned. Otherwise, it uses the "engine.sqlite.client" configuration value, or else defaults to
       "sqlite3" (or "sqlite3.exe" on Windows), which should work if it's in your path.

   Instance Methods
       "sqlite3"

       Returns a list containing the "sqlite3" client and options to be passed to it.  Used internally when
       executing scripts.
