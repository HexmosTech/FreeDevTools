NAME
       Alzabo::RDBMSRules::PostgreSQL - PostgreSQL specific database rules


SYNOPSIS
         use Alzabo::RDBMSRules::PostgreSQL;


DESCRIPTION
       This module implements all the methods descibed in Alzabo::RDBMSRules for the PostgreSQL database.  The
       syntax rules follow those of the 7.0 releases.  Older versions may work but are not supported.
