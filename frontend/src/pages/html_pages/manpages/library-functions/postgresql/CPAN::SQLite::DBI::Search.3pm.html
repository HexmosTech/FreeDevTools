NAME
       CPAN::SQLite::DBI::Search - DBI information for searching the CPAN::SQLite database


VERSION
       version 0.220


DESCRIPTION
       This module provides methods for CPAN::SQLite::Search for searching the "CPAN::SQLite" database. There
       are two main methods.

       "fetch"
           This takes information from "CPAN::SQLite::Search" and sets up a query on the database, returning the
           results found.

       "sql_statement"
           This is used by the "fetch" method to construct the appropriate SQL statement.
