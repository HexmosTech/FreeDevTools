NAME
       DBIx::Connector::Driver::Pg - PostgreSQL-specific connection interface


DESCRIPTION
       This subclass of DBIx::Connector::Driver provides PostgreSQL-specific implementations of the following
       methods:

       "savepoint"
       "release"
       "rollback_to"
           NOTE:  Due  to  a  bug  <https://rt.cpan.org/Ticket/Display.html?id=100648>  in the implementation of
           DBD::Pg's "ping" method, DBD::Pg 3.5.0 or later is strongly recommended.
