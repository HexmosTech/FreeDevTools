NAME
       SDL_GL_GetSwapInterval - Get the swap interval for the current OpenGL context.


SYNOPSIS
       #include <SDL3/SDL_video.h>

       bool SDL_GL_GetSwapInterval(int *interval);


DESCRIPTION
       If  the  system  can't determine the swap interval, or there isn't a valid current context, this function
       will set *interval to 0 as a safe default.


FUNCTION PARAMETERS
       interval
              output interval value. 0 if there is no vertical retrace synchronization, 1 if the buffer swap  is
              synchronized with the vertical retrace, and -1 if late swaps happen immediately instead of waiting
              for the next retrace.


RETURN VALUE
       for more information.


THREAD SAFETY
       This function should only be called on the main thread.


AVAILABILITY
       This function is available since SDL 3.2.0.
