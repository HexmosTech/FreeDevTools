REGISTERING
       Before you can use OAuth 2 with Withings you need to register a developer account at
       <https://account.withings.com/connectionuser/account_create>

       You also need to create an application at <https://account.withings.com/partner/> which will provide you
       with a "clientId" and "client_secret".
