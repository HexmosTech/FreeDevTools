NAME
       Test::Valgrind::Parser::Text - Parse valgrind output as a text stream.


VERSION
       Version 1.19


DESCRIPTION
       This is a Test::Valgrind::Parser object that can extract suppressions from "valgrind"'s text output.


METHODS
   "args"
           my @args = $tvp->args($session, $fh);

       Returns the arguments needed to tell "valgrind" to print in text to the filehandle $fh.

       Test::Valgrind, Test::Valgrind::Parser.
