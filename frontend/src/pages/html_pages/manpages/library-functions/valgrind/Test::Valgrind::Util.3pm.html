NAME
       Test::Valgrind::Util - Utility routines for Test::Valgrind.


VERSION
       Version 1.19


DESCRIPTION
       This module contains some helpers used by Test::Valgrind.  It is not really designed to be used anywhere
       else.


FUNCTIONS
   "validate_subclass"
           my ($validated_type, $error_msg) = validate_subclass($type);

       Try to interpret $type as a subclass of the caller package, and load it if its @ISA is empty.  Returns
       the validated type, or "undef" and the relevant error message.


EXPORT
       This module does not export anything.
