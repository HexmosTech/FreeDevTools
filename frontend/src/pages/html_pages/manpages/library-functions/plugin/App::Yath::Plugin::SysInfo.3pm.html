NAME
       App::Yath::Plugin::SysInfo - Plugin to attach system information to a run.


DESCRIPTION
       This plugin attaches a lot of system information to the yath log. This is mainly useful if you intend to
       view the log in Test2::Harness::UI.

       The source code repository for Test2-Harness can be found at http://github.com/Test-More/Test2-Harness/.
