NAME
       Template::Plugin::Procedural - Base class for procedural plugins


SYNOPSIS
           package Template::Plugin::LWPSimple;
           use base qw(Template::Plugin::Procedural);
           use LWP::Simple;  # exports 'get'
           1;

           [% USE LWPSimple %]
           [% LWPSimple.get("http://www.tt2.org/") %]


DESCRIPTION
       "Template::Plugin::Procedural" is a base class for Template Toolkit plugins that causes defined
       subroutines to be called directly rather than as a method.  Essentially this means that subroutines will
       not receive the class name or object as its first argument.

       This is most useful when creating plugins for modules that normally work by exporting subroutines that do
       not expect such additional arguments.

       Despite the fact that subroutines will not be called in an OO manner, inheritance still function as
       normal.  A class that uses "Template::Plugin::Procedural" can be subclassed and both subroutines defined
       in the subclass and subroutines defined in the original class will be available to the Template Toolkit
       and will be called without the class/object argument.
