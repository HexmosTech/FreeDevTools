NAME
       Dist::Zilla::Plugin::PodSpellingTests - (DEPRECATED) The old name of the PodSpelling plugin


VERSION
       version 2.007006


SYNOPSIS
       This is a plugin that runs at the gather files stage, providing the file:

         xt/author/pod-spell.t - a standard Test::Spelling test

       THIS MODULE IS DEPRECATED. Please use Dist::Zilla::Plugin::Test::PodSpelling instead. it may be removed
       at a later time (but not before April 2016).
