NAME
       Alien::Build::Plugin::Core::FFI - Core FFI plugin


VERSION
       version 2.84


SYNOPSIS
        use alienfile;
        # already loaded


DESCRIPTION
       This plugin helps make the build_ffi work.  You should not need to interact with it directly.
