NAME
       Alien::Build::Plugin::Digest::Negotiate - Plugin negotiator for cryptographic signatures


VERSION
       version 2.84


SYNOPSIS
       for a single file:

        use alienfile;
        plugin 'Digest' => [ SHA256 => $digest ];

       or for multiple files:

        use alienfile;
        plugin 'Digest' => {
          file1 => [ SHA256 => $digest1 ],
          file2 => [ SHA256 => $digest2 ],
        };


DESCRIPTION
       This plugin is experimental.
