NAME
       Alien::Build::Plugin::Gather::IsolateDynamic - Plugin to gather dynamic libraries into a separate
       directory


VERSION
       version 2.84


SYNOPSIS
        use alienfile;
        plugin 'Gather::IsolateDynamic';


DESCRIPTION
       This plugin moves dynamic libraries from the "lib" and "bin" directories and puts them in their own
       "dynamic" directory.  This allows them to be used by FFI modules, but to be ignored by XS modules.

       This plugin provides the equivalent functionality of the "alien_isolate_dynamic" attribute from
       Alien::Base::ModuleBuild.
