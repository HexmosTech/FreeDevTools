NAME
       MooseX::Runnable::Invocation::Plugin::Debug - print debugging information


VERSION
       version 0.10


DESCRIPTION
       This is an example plugin, showing how you could write your own.  It prints a message for each stage of
       the "run" process.  It is also used by other plugins to determine whether or not to print debugging
       messages.
