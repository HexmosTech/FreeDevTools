NAME
       Log::Agent::Driver::Datum - interceptor driver to cooperate with Carp::Datum


SYNOPSIS
       NONE


DESCRIPTION
       The purpose of the interceptor is to cooperate with Carp::Datum by emitting traces to the debug channel
       via Carp::Datum's traces facilities.

       This driver is automatically installed by Log::Agent when Carp::Datum is in use and debug was activated
       through it.
