NAME
       SDL_LoadFile - Load all the data from a file path.


SYNOPSIS
       #include <SDL3/SDL_iostream.h>

       void * SDL_LoadFile(const char *file, size_t *datasize);


DESCRIPTION
       The  data  is allocated with a zero byte at the end (null terminated) for convenience. This extra byte is
       not included in the value reported via datasize.

       The data should be freed with SDL_free().


FUNCTION PARAMETERS
       file   the path to read all available data from.

       datasize
              if not NULL, will store the number of bytes read.


RETURN VALUE
       for more information.


THREAD SAFETY
       This function is not thread safe.


AVAILABILITY
       This function is available since SDL 3.2.0.
