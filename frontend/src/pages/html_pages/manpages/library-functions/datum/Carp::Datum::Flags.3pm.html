NAME
       Carp::Datum::Flags - Flag Constants


SYNOPSIS
        # Used internally to define debugging and tracing flag constants


DESCRIPTION
       This module is used internally by "Carp::Datum".  It defines the constants that are exported and made
       available automatically to all users of "Carp::Datum".
