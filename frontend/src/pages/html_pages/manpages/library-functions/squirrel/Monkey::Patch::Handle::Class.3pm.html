NAME
       Monkey::Patch::Handle::Class - Handle for monkey-patching classes


DESCRIPTION
       Monkey::Patch::Handle::Class provides the default Monkey::Patch::Handle for monkey-patching a whole
       class.

       You should never need to use this directly, so read L(Monkey::Patch) instead.
