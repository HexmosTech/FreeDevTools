NAME
       Algorithm::Dependency::Source::Invert - Logically invert a source


SYNOPSIS
         my $inverted = Algorithm::Dependency::Source::Invert->new( $source );


DESCRIPTION
       This class creates a source from another source, but with all dependencies reversed.
