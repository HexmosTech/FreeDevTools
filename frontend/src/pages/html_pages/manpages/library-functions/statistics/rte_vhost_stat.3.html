Detailed Description
       Vhost virtqueue statistics structure

       This structure is used by rte_vhost_vring_stats_get() to provide virtqueue statistics to the calling
       application. It maps a name ID, corresponding to an index in the array returned by
       rte_vhost_vring_stats_get_names(), to a statistic value.
