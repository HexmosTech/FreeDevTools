NAME
       nng_stat_find_dialer - find dialer statistics


SYNOPSIS
           #include <nng/nng.h>

           typedef struct nng_stat nng_stat;
           typedef struct nng_dialer nng_dialer;

           nng_stat *nng_stat_find_dialer(nng_stat *stat, nng_dialer dialer);


DESCRIPTION
       The nng_stat_find_dialer() function returns the statistics tree within stat associated with the dialer
       dialer, if such a tree exists. Otherwise NULL is returned.

       Generally, there will be child statistics of the returned value, each corresponding to a specific metric.
       These can be further scanned using either nng_stat_find(3) or by walking the tree with nng_stat_child(3).


RETURN VALUES
       The matching statistic, or NULL if no match is found.


ERRORS
       None.
