Detailed Description
       An Ethernet device extended statistic structure

       This structure is used by rte_eth_xstats_get() to provide statistics that are not provided in the generic
       rte_eth_stats structure. It maps a name ID, corresponding to an index in the array returned by
       rte_eth_xstats_get_names(), to a statistic value.
