SYNOPSIS

Detailed Description
       RTE_FLOW_ITEM_TYPE_VXLAN.

       Matches a VXLAN header (RFC 7348), including GPE (draft-ietf-nvo3-vxlan-gpe-13.txt) and GBP (draft-smith-
       vxlan-group-policy-05.txt).

       GPE is distinguished with its UDP port. UDP port may be specified with rte_eth_dev_udp_tunnel_port_add().
