NAME
       Metabase::User::Secret - Metabase fact for user shared authentication secret


VERSION
       version 0.025


SYNOPSIS
         my $secret = Metabase::User::Secret->new(
           resource => 'metabase:user:B66C7662-1D34-11DE-A668-0DF08D1878C0',
           content  => 'aixuZuo8',
         );


DESCRIPTION
       This fact is a simple string, storing the shared secret that will be used to authenticate user during
       fact submission.
