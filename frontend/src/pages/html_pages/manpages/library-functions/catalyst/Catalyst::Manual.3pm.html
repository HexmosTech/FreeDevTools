NAME
       Catalyst::Manual - The Catalyst developer's manual


SYNOPSIS
           perldoc Catalyst::Manual::Intro
           perldoc Catalyst::Manual::Tutorial


SEE ALSO
       Install Task::Catalyst::Tutorial to install all the dependencies you need to follow along with the
       Tutorial.  You can also refer to Catalyst::Manual::Tutorial::Intro for more information on installation
       options.
