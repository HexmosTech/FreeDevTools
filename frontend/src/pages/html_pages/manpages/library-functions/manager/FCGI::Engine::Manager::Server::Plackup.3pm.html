NAME
       FCGI::Engine::Manager::Server::Plackup - A subclass of FCGI::Engine::Manager::Server for Plack apps


DESCRIPTION
       This uses the "plackup" utility that comes with Plack to manage a Plack FCGI application.  See
       FCGI::Engine::Manager for details on how to configure things.
