NAME
       function::set_long_arg - Set argument as signed long


SYNOPSIS
           set_long_arg(n:long,v:long)


ARGUMENTS
       n
           index of argument to set

       v
           value to set


DESCRIPTION
       Set the value of argument n as a signed long. On architectures where a long is 32 bits, the value is
       sign-extended to 64 bits.
