NAME
       function::is_sig_blocked - Returns 1 if the signal is currently blocked, or 0 if it is not


SYNOPSIS
           is_sig_blocked:long(task:long,sig:long)


ARGUMENTS
       task
           address of the task_struct to query.

       sig
           the signal number to test.
