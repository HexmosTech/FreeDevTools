NAME
       function::fp_lt - fp comparison function less than


SYNOPSIS
           fp_lt:long(infp1:long,infp2:long)


ARGUMENTS
       infp1
           the 64 bit floating point input

       infp2
           second 64 bit floating point input


DESCRIPTION
       check if infp1 is strictly less than infp2
