NAME
       function::MKDEV - Creates a value that can be compared to a kernel device number (kdev_t)


SYNOPSIS
           MKDEV:long(major:long,minor:long)


ARGUMENTS
       major
           Intended major device number.

       minor
           Intended minor device number.
