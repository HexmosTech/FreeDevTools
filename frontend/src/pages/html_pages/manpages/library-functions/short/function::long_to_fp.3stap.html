NAME
       function::long_to_fp - Convert long int to 64 bit floating point


SYNOPSIS
           long_to_fp:long(input:long)


ARGUMENTS
       input
           a long integer


DESCRIPTION
       Convert from a long to a 64 bit softfloat floating point.
