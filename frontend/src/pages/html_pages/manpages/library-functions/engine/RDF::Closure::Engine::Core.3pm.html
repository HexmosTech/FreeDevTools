NAME
       RDF::Closure::Engine::Core - common code used by inference engines


ANALOGOUS PYTHON
       RDFClosure/Closure.py


DESCRIPTION
       This is a basic forward-chaining engine.

       Inference engines don't have to inherit from this, but it helps.
