NAME
       taia_frac - get current time


SYNTAX
       #include <libowfat/taia.h>

       int taia_frac(const struct taia* t);


DESCRIPTION
       taia_frac  returns a double-precision approximation to the fraction part of t. The result of taia_frac is
       always nonnegative.
