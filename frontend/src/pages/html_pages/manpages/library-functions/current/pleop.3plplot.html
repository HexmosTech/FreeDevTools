NAME
       pleop - Eject current page


SYNOPSIS
       pleop()


DESCRIPTION
       Clears  the  graphics screen of an interactive device, or ejects a page on a plotter.  See plbop(3plplot)
       for more information.

       Redacted form: pleop()

       This function is used in example 2,14.
