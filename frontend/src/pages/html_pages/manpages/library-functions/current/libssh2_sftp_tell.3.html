NAME
       libssh2_sftp_tell - get the current read/write position indicator for a file


SYNOPSIS
       #include <libssh2.h>
       #include <libssh2_sftp.h>

       size_t
       libssh2_sftp_tell(LIBSSH2_SFTP_HANDLE *handle);


DESCRIPTION
       handle - SFTP File Handle as returned by libssh2_sftp_open_ex(3).

       Returns  the  current offset of the file handle's internal pointer. Note that this is now deprecated. Use
       the newer libssh2_sftp_tell64(3) instead!


RETURN VALUE
       Current offset from beginning of file in bytes.


SEE ALSO
       libssh2_sftp_open_ex(3), libssh2_sftp_tell64(3)
