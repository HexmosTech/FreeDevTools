NAME
       MDOM::Token - A single token of Makefile source code


INHERITANCE
         MDOM::Token
         isa MDOM::Element


DESCRIPTION
       "MDOM::Token" is the abstract base class for all Tokens. In MDOM terms, a "Token" is a MDOM::Element that
       directly represents bytes of source code.

       The implementation and POD are borrowed directly from PPI::Token.


METHODS
   set_class
       Set a specific class for a token.

   set_content $string
       The "set_content" method allows one to set/change the string that the "MDOM::Token" object represents.

       Returns the string you set the Token to

   add_content $string
       The "add_content" method allows you to add additional bytes of code to the end of the Token.

       Returns the new full string after the bytes have been added.

   length
       The "length" method returns the length of the string in a Token.

   source
       Returns the makefile source for the current token
