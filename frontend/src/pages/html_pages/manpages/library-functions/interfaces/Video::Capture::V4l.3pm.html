NAME
       Video::Capture::V4l - Perl interface to the Video4linux framegrabber interface.


SYNOPSIS
         use Video::Capture::V4l;


DESCRIPTION
       Not documentation AGAIN! Please see the scripts grab, inexer or vbi that are packaged in the distribution
       and direct any question and feature requests (as well as bug reports) to the author.
