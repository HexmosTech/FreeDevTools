NAME
       RunApp::Apache - Apache control from RunApp


SYNOPSIS
        See RunApp::Apache


DESCRIPTION
       The class provides the default template for apache configuration.  It uses Template Toolkit.

   variables
       timeout
           Default is 300.

       KeepAlive
           Default is Off.

       MaxKeepAliveRequests
           Default is 100.

       KeepAliveTimeout
           Default is 15.

       documentroot
       hostname
       port
       webmaster
       logformat
           Default is 'common'.

       AP_VERSION
           Shows the version of the apache used.
