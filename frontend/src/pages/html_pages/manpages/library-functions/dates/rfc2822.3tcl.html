NAME
       clock_rfc2822 - Parsing RFC 2822 dates/times


SYNOPSIS
       package require Tcl 8.5 9

       package require clock::rfc2822 ?0.2?

       ::clock::rfc2822 parse_date date

________________________________________________________________________________________________________________


DESCRIPTION
       This package provides functionality to parse dates in RFC 2822 format.

       ::clock::rfc2822 parse_date date
              This  command  parses an RFC2822 date string and returns the given date in seconds since epoch. An
              error is thrown if the command is unable to parse the date.
