NAME
       CSS::Parse::Lite - A CSS::Parse module using regular expressions


SYNOPSIS
         use CSS;

         # Create a css stylesheet
         my $CSS = CSS->new({'parser' => 'CSS::Parse::Lite'});


DESCRIPTION
       This module is a parser for CSS.pm. Read the CSS.pm pod for more details
