NAME
       RoPkg::Simba::Plugin::GenHTML - html generator plugin for Simba


VERSION
       0.2.4


PREREQUISITES
       GenHTML requires Simba (and all Simba's dependencies). Besides that GenHTML requires HTML::Template perl
       module.


DESCRIPTION
       GenHTML is a Simba plugin used to generate html reports. For details please visit
       http://simba.packages.ro


METHODS
   genMirrorPage($cfg, $mirror)
       Generate mirror report.

   genMirrorsIndex($cfg, $mirrors)
       Generate mirrors report.
