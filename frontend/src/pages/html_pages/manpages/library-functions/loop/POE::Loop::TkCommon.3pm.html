NAME
       POE::Loop::TkCommon - common code between the POE/Tk event loop bridges


SYNOPSIS
       See POE::Loop.


DESCRIPTION
       POE::Loop::TkCommon is a mix-in class that supports common features between POE::Loop::Tk and
       POE::Loop::TkActiveState.  All Tk bridges implement the interface documented in POE::Loop.  Therefore,
       please see POE::Loop for more details.
