NAME
       Module::Install::DOAP - generate META.yml data from DOAP


SYNOPSIS
       In Makefile.PL:

         rdf_metadata;
         doap_metadata;


DESCRIPTION
       This Module::Install plugin generates your META.yml file from RDF data (especially DOAP) in your
       distribution's 'meta' directory.
