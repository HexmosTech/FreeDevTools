NAME
       RunApp::Control::AppControl - Class for controlling daemon with App::Control


SYNOPSIS
        see RunApp::Apache


DESCRIPTION
       The class writes to a perl script that uses App::Control to control the daemon in the "build" phase of
       RunApp, and invokes the control script in other phases.


SEE ALSO
       App::Control
