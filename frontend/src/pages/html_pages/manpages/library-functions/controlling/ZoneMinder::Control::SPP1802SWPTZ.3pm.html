SPP1802SWPTZ
       ZoneMinder::Database - Perl extension for SunEyes SP-P1802SWPTZ


SYNOPSIS
       Control script for SunEyes SP-P1802SWPTZ cameras.


DESCRIPTION
       You can set "-speed=x" in the ControlDevice field of the control tab for that monitor. x should be an
       integer between 0 and 64 Auto TimeOut should be 1. Don't set it to less - processes start crashing :)

   EXPORT
       None by default.
