NAME
       Dpkg::Control::Tests - parse files like debian/tests/control


DESCRIPTION
       It provides a class to access data of files that follow the same syntax as debian/tests/control.


METHODS
       All the methods of Dpkg::Index are available. Those listed below are either new or overridden with a
       different behavior.

       $c = Dpkg::Control::Tests->new(%opts)
           Create a new Dpkg::Control::Tests object, which inherits from Dpkg::Index.

           The options are passed through to the Dpkg::Index->new() constructor.

       $item = $tests->new_item()
           Creates a new item.
