FI8918W
       ZoneMinder::Database - Perl extension for FOSCAM FI8918W


SYNOPSIS
       Control script for Foscam MJPEG 8918W cameras.


DESCRIPTION
       You need to set "user=xxx&pwd=yyy" in the ControlDevice field of the control tab for that monitor.  Auto
       TimeOut should be 1. Don't set it to less - processes start crashing :) NOTE: unlike HD foscam cameras,
       this one uses "user" not "usr" in the control device

   EXPORT
       None by default.
