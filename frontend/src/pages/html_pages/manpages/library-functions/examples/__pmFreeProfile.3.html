NAME
       __pmFreeProfile - free a PMAPI instance profile


C SYNOPSIS
       #include "pmapi.h"
       #include "libpcp.h"

       void __pmFreeProfile(pmProfile *prof);

       cc ... -lpcp


CAVEAT
       This documentation is intended for internal Performance Co-Pilot (PCP) developer use.

       These  interfaces  are  not part of the PCP APIs that are guaranteed to remain fixed across releases, and
       they may not work, or may provide different semantics at some point in the future.


DESCRIPTION
       __pmFreeProfile frees the storage for the PMAPI instance profile prof, assuming the storage was allocated
       using malloc(3) according to the scheme used in __pmDecodeProfile(3).


SEE ALSO
       malloc(3), PMAPI(3) and __pmDecodeProfile(3).
