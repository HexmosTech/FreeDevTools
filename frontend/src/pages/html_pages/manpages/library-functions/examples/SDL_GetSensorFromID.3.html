NAME
       SDL_GetSensorFromID - Return the SDL_Sensor
        associated with an instance ID.


HEADER FILE
       Defined in SDL3/SDL_sensor.h


SYNOPSIS
       #include "SDL3/SDL.h"

       SDL_Sensor * SDL_GetSensorFromID(SDL_SensorID instance_id);


FUNCTION PARAMETERS
       instance_id
              the sensor instance ID.


RETURN VALUE
       ( SDL_Sensor
        *) Returns an SDL_Sensor
        object or NULL on failure; call SDL_GetError () for more information.


AVAILABILITY
       This function is available since SDL 3.2.0.
