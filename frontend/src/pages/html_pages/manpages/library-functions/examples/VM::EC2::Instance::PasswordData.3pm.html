NAME
       VM::EC2::PasswordData - Object describing the administrative password stored in an EC2 Windows instance


SYNOPSIS
         use VM::EC2;

         $ec2      = VM::EC2->new(...);
         $instance = $ec2->describe_instances(-instance_id=>'i-12345');
         $pass     = $instance->password_data;
         print $pass->password,"\n";
         print $pass->timestamp,"\n"


DESCRIPTION
       This object represents the administrative password stored in a Windows EC2 instance.  It is returned by
       calling either VM::EC2->get_password_data or a VM::EC2::Instance object's password_data() method.


METHODS
       These object methods are supported:

        instanceId    -- ID of the instance
        timestamp     -- The time the data was last updated.
        passwordData  -- The password of the instance.
        password()    -- Same as passwordData().


STRING OVERLOADING
       When used in a string context, this object will interpolate the password.
