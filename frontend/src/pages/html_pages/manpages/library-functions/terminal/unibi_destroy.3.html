NAME
       unibi_destroy - free a terminal object


SYNOPSIS
        #include <unibilium.h>

        void unibi_destroy(unibi_term *ut);


DESCRIPTION
       This function frees a terminal object created by "unibi_dummy" or "unibi_from_mem".


SEE ALSO
       unibilium.h(3), unibi_dummy(3), unibi_from_mem(3)
