NAME
       "Devel::MAT::Cmd::Terminal" - provide the Devel::MAT::Cmd API on a terminal


DESCRIPTION
       This module provides an implementation of the methods required for the Devel::MAT::Cmd API that outputs
       formatted text to a terminal. This is performed by using String::Tagged::Terminal.
