NAME
       probe::scsi.set_state - Order SCSI device state change


SYNOPSIS
       scsi.set_state


VALUES
       dev_id
           The scsi device id

       state_str
           The new state of the device, as a string

       old_state
           The current state of the device

       host_no
           The host number

       old_state_str
           The current state of the device, as a string

       lun
           The lun number

       channel
           The channel number

       state
           The new state of the device
