NAME
       Test2::Harness::Runner::State - State tracking for the runner.


DESCRIPTION
       This module tracks the state for all running tests. This entire module is considered an "Implementation
       Detail". Please do not rely on it always staying the same, or even existing in the future. Do not use
       this directly.
