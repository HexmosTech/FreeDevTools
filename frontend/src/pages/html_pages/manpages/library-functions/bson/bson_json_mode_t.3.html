SYNOPSIS
          #include <bson/bson.h>

          typedef enum {
             BSON_JSON_MODE_LEGACY,
             BSON_JSON_MODE_CANONICAL,
             BSON_JSON_MODE_RELAXED,
          } bson_json_mode_t;


DESCRIPTION
       The  bson_json_mode_t  enumeration  contains  all available modes for encoding BSON into MongoDB Extended
       JSON.

       SEE ALSO:
          bson_as_json_with_opts()
