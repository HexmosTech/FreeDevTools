NAME
       SDL_AudioStreamDevicePaused  -  Use this function to query if an audio device associated with a stream is
       paused.


SYNOPSIS
       #include <SDL3/SDL_audio.h>

       bool SDL_AudioStreamDevicePaused(SDL_AudioStream *stream);


DESCRIPTION
       Unlike in SDL2, audio devices start in an _unpaused_ state, since an app has to bind a stream before  any
       audio will flow.


FUNCTION PARAMETERS
       stream the audio stream associated with the audio device to query.


RETURN VALUE
       Returns true if device is valid and paused, false otherwise.


THREAD SAFETY
       It is safe to call this function from any thread.


AVAILABILITY
       This function is available since SDL 3.2.0.
