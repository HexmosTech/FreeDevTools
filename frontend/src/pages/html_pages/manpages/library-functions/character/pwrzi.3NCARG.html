NAME
       PWRZI - A character-plotting routine for plotting characters in three-space when using Isosurface.  For a
       large class of possible positions, the hidden character problem is solved.


SYNOPSIS
       SUBROUTINE PWRZI (X,Y,Z,ID,N,ISIZE,LINE,ITOP,ICNT)


C-BINDING SYNOPSIS
       #include <ncarg/ncargC.h>

       void c_pwrzi (float x, float y, float z, char *id, int n,
       int isize, int line, int itop, int icnt)


DESCRIPTION
       X,Y,Z       (input  expressions of type REAL) are positioning coordinates for the characters to be drawn.
                   These are floating point numbers in the same three-space coordinate system as  that  used  in
                   Isosurface.

       ID          (an input expression of type CHARACTER) is the character string to be drawn.

       N           (an input expression of type INTEGER) is the number of characters in ID.

       ISIZE       (an input expression of type INTEGER) specifies the size of the characters to be used:

                   • If  between 0 and 3, ISIZE is 1., 1.5, 2., or 3. times a standard width equal to 1/128th of
                     the screen width.

                   • If greater than 3, ISIZE is the character width  in  plotter  address  units  (by  default,
                     1/1024th of the screen width).

       LINE        (an  input expression of type INTEGER) specifies the direction in which the characters are to
                   be written.

                   +1 = +X
                   -1 = -X
                   +2 = +Y
                   -2 = -Y
                   +3 = +Z
                   -3 = -Z

       ITOP        (an input expression of type INTEGER) specifies the direction from the center  of  the  first
                   character  to  the  top of the first character (the potential values for ITOP are the same as
                   those for LINE as given above). Note that LINE cannot equal ITOP even in absolute value.

       ICNT        (an input expression of type INTEGER) is the centering option:

                   -1  (X,Y,Z) is the center of the left edge of the first character.

                   +0  (X,Y,Z) is the center of the entire string.

                   +1  (X,Y,Z) is the center of the right edge of the last character.


C-BINDING DESCRIPTION
       The C-binding argument descriptions are the same as the FORTRAN argument descriptions.


USAGE
       Call PWRZI after calling ISOSRF and before calling FRAME.


EXAMPLES
       Use the ncargex command to see the following relevant examples: tpwrzi, fsrpwrzs.


ACCESS
       To use PWRZI or c_pwrzi, load the NCAR Graphics libraries ncarg, ncarg_gks, and  ncarg_c,  preferably  in
       that order.
