NAME
       str_chr - find character in ASCIIZ string


SYNTAX
       #include <libowfat/str.h>

       size_t str_chr(const char* haystack,char needle);


DESCRIPTION
       str_chr returns the index of the first occurrance of needle or \0 in string.
