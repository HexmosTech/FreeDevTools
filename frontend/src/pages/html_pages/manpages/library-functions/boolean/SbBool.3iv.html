NAME
       SbBool — a Boolean type


INHERITS FROM
       SbBool


SYNOPSIS
       #include <Inventor/SbBasic.h>


DESCRIPTION
       This  typedef  is used to represent the Boolean values TRUE and FALSE. SbBool is not a class and does not
       have any methods or variables.
