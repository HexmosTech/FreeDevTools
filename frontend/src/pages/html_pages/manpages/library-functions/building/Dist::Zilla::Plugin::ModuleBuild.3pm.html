NAME
       Dist::Zilla::Plugin::ModuleBuild - build a Build.PL that uses Module::Build


VERSION
       version 6.032


DESCRIPTION
       This plugin will create a Build.PL for installing the dist using Module::Build.


PERL VERSION
       This module should work on any version of perl still receiving updates from the Perl 5 Porters.  This
       means it should work on any version of perl released in the last two to three years.  (That is, if the
       most recently released version is v5.40, then this module should work on both v5.40 and v5.38.)

       Although it may work on older versions of perl, no guarantee is made that the minimum required version
       will not be increased.  The version may be increased for any reason, and there is no promise that patches
       will be accepted to lower the minimum required perl.


ATTRIBUTES
   mb_version
       Optional: Specify the minimum version of Module::Build to depend on.

       Defaults to 0.3601 if a sharedir is being used, otherwise 0.28.

   mb_class
       Optional: Specify the class to use to create the build object.  Defaults to "Module::Build" itself.  If
       another class is specified, then the value of mb_lib will be used to generate a line like "use lib 'inc'"
       to be added to the generated Build.PL file.

   mb_lib
       Optional: Specify the list of directories to be passed to lib when using mb_class. Defaults to "inc".

   build_element
       Anything given in a "build_element" option will be added as a build element with Module::Build's
       add_build_element method.
