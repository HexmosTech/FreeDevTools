NAME
       Dist::Build::XS::Conf - Configure-time utilities for Dist::Build for using C headers, libraries, or OS
       features


VERSION
       version 0.020


SYNOPSIS
        load_extension("Dist::Build::XS");

        find_libs_for(source => <<'EOF', libs => [ ['socket'], ['moonlaser'] ]);
        #include <stdio.h>
        #include <sys/socket.h>
        int main(int argc, char *argv[]) {
          printf("PF_MOONLASER is %d\n", PF_MOONLASER);
          return 0;
        }
        EOF

        add_xs(module_name => 'Socket::MoonLaser');

   DESCRIPTION
       This module integrates ExtUtils::Builder::Conf into Dist::Build::XS. Any arguments found with any of the
       "find_*" or "try_find_*" functions will be automatically added to the build when calling "add_xs".
