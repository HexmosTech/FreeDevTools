NAME
       Bio::DB::Flat::BDB::fasta - fasta adaptor for Open-bio standard BDB-indexed flat file


SYNOPSIS
       See Bio::DB::Flat.


DESCRIPTION
       This module allows fasta files to be stored in Berkeley DB flat files using the Open-Bio standard BDB-
       indexed flat file scheme.  You should not be using this directly, but instead use it via Bio::DB::Flat.
