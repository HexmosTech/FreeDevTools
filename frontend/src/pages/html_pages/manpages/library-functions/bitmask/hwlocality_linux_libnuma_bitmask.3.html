Detailed Description
       This interface helps converting between Linux libnuma bitmasks and hwloc cpusets and nodesets.

       Note
           Topology topology must match the current machine.

           The behavior of libnuma is undefined if the kernel is not NUMA-aware. (when CONFIG_NUMA is not set in
           the kernel configuration). This helper and libnuma may thus not be strictly compatible in this case,
           which may be detected by checking whether numa_available() returns -1.
