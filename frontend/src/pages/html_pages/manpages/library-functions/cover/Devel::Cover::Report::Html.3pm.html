NAME
       Devel::Cover::Report::Html - HTML backend for Devel::Cover


VERSION
       version 1.44


SYNOPSIS
        cover -report html


DESCRIPTION
       This module provides a HTML reporting mechanism for coverage data.  It is designed to be called from the
       "cover" program.  This is an empty class derived from the default HTML output module,
       Devel::Cover::Report::Html_minimal.
