NAME
       shishi_kdc_sendrecv_hint - API function


SYNOPSIS
       #include <shishi.h>

       int shishi_kdc_sendrecv_hint(Shishi * handle, const char * realm, const char * indata, size_t inlen, char
       ** outdata, size_t * outlen, Shishi_tkts_hint * hint);


ARGUMENTS
       Shishi * handle
                   Shishi library handle create by shishi_init().

       const char * realm
                   string with realm name.

       const char * indata
                   Packet to send to KDC.

       size_t inlen
                   Length of indata.

       char ** outdata
                   Newly allocated string with data returned from KDC.

       size_t * outlen
                   Length of outdata.

       Shishi_tkts_hint * hint
                   a Shishi_tkts_hint structure with flags.


DESCRIPTION
       Send packet to KDC for realm and receive response.  The code finds KDC addresses from configuration file,
       then by querying for SRV records for the realm, and finally by using the realm name as a hostname.


RETURNS
       SHISHI_OK on success, SHISHI_KDC_TIMEOUT if a timeout was reached, or other errors.
