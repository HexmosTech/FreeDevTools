NAME
       Qpsmtpd::Postfix - postfix queueing support for qpsmtpd

   DESCRIPTION
       This package implements the protocol Postfix servers use to communicate with each other. See
       src/global/rec_type.h in the postfix source for details.

   print_msg_line($line)
       print one line of a message to cleanup.

       This removes any linefeed characters from the end of the line and splits the line across several records
       if it is longer than 1024 chars.

   inject_mail($transaction)
       (class method) inject mail in $transaction into postfix queue via cleanup.  $transaction is supposed to
       be a Qpsmtpd::Transaction object.
