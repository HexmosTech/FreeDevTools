NAME
       Sisimai::Lhost::qmail - bounce mail decoder class for qmail <https://cr.yp.to/qmail.html>.


SYNOPSIS
           use Sisimai::Lhost::qmail;


DESCRIPTION
       "Sisimai::Lhost::qmail" decodes a bounce email which created by qmail <https://cr.yp.to/qmail.html> or
       qmail clones or notqmail <https://notqmail.org/>.  Methods in the module are called from only
       "Sisimai::Message".


CLASS METHODS
   "description()"
       description() returns description string of this module.

           print Sisimai::Lhost::qmail->description;

   "inquire(header data, reference to body string)"
       inquire() method decodes a bounced email and return results as a array reference.  See "Sisimai::Message"
       for more details.
