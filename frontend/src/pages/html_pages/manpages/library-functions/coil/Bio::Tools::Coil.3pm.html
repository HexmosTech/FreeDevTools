NAME
       Bio::Tools::Coil - parser for Coil output


SYNOPSIS
        use Bio::Tools::Coil
        my $parser = Bio::Tools::Coil->new();
        while( my $sp_feat = $parser->next_result($file) ) {
              #do something
              #eg
              push @sp_feat, $sp_feat;
        }


DESCRIPTION
        Parser for Coil output
