NAME
       Git::Raw::Patch - Git patch class


VERSION
       version 0.90


DESCRIPTION
       A Git::Raw::Patch represents all the text diffs for a delta.

       WARNING: The API of this module is unstable and may change without warning (any change will be
       appropriately documented in the changelog).


METHODS
   buffer( )
       Get the content of a patch as a single diff text.

   hunk_count( )
       Get the number of hunks in the patch.

   hunks( [$index] )
       Returns a list of Git::Raw::Diff::Hunk objects. If $index is specified only the hunk at the specified
       index will be returned.

   line_stats( )
       Get line counts of each type in the patch. Returns a hash with entries "context", "additions" and
       "deletions".

   delta( )
       Get the delta associated with the patch. Returns a Git::Raw::Diff::Delta object.
