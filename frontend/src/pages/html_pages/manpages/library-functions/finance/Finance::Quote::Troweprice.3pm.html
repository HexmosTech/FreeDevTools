NAME
       Finance::Quote::Troweprice    - Obtain quotes from T. Rowe Price


SYNOPSIS
           use Finance::Quote;

           $q = Finance::Quote->new;

           %stockinfo = $q->fetch("troweprice","PRFDX");


DESCRIPTION
       This module obtains information about managed funds from T. Rowe Price.  Information about T. Rowe Price
       funds is available from a variety of sources.  This module fetches information directly from T. Rowe
       Price.


LABELS RETURNED
       Information available from T. Rowe Price may include the following labels:  exchange, name, nav, date,
       price.


SEE ALSO
       T. Rowe Price website - http://www.troweprice.com/
