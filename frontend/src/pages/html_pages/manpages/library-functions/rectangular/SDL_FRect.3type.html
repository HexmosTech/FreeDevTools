NAME
       SDL_FRect - A rectangle, with the origin at the upper left (using floating point values).


HEADER FILE
       Defined in SDL3/SDL_rect.h


SYNOPSIS
       #include "SDL3/SDL.h"

       typedef struct SDL_FRect
       {
           float x;
           float y;
           float w;
           float h;
       } SDL_FRect;


AVAILABILITY
       This struct is available since SDL 3.2.0.
