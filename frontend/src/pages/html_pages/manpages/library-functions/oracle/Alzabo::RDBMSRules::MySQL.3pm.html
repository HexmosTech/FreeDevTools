NAME
       Alzabo::RDBMSRules::MySQL - MySQL specific database rules.


SYNOPSIS
         use Alzabo::RDBMSRules::MySQL;


DESCRIPTION
       This module implements all the methods descibed in Alzabo::RDBMSRules for the MySQL database.  The syntax
       rules follow the more restrictive rules of version 3.22.
