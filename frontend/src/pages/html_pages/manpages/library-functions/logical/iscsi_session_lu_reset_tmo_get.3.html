NAME
       iscsi_session_lu_reset_tmo_get - Retrieve logical unit timeout value of specified session


SYNOPSIS
       int32_t iscsi_session_lu_reset_tmo_get (struct iscsi_session *se);


ARGUMENTS
       se          Pointer  of 'struct iscsi_session'.  If this pointer is NULL, your program will be terminated
                   by assert.


DESCRIPTION
       Retrieve the logical unit timeout value of specified iSCSI session.  The logical unit timeout here  means
       the  seconds  of  time to wait for a logical unit response before before failing the operation and trying
       session re-establishment.  It could be controlled via this setting in iscsid.conf:


RETURN
       int32_t. -1 if not supported.
