NAME
       pcap_get_tstamp_precision - get the time stamp precision returned in captures


SYNOPSIS
       #include <pcap/pcap.h>

       int pcap_get_tstamp_precision(pcap_t *p);


DESCRIPTION
       pcap_get_tstamp_precision()  returns  the  precision of the time stamp returned in packet captures on the
       pcap descriptor.


RETURN VALUE
       pcap_get_tstamp_precision()  returns  PCAP_TSTAMP_PRECISION_MICRO  or  PCAP_TSTAMP_PRECISION_NANO,  which
       indicates that pcap captures contains time stamps in microseconds or nanoseconds respectively.


BACKWARD COMPATIBILITY
       This  function  became  available  in  libpcap  release  1.5.1.  In previous releases, time stamps from a
       capture device or savefile are always given in seconds and microseconds.
