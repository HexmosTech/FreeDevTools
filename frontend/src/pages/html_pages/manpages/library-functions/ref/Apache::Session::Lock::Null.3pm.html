NAME
       Apache::Session::Lock::Null - Does not actually provides mutual exclusion


SYNOPSIS
        use Apache::Session::Lock::Null;

        my $locker = Apache::Session::Lock::Null->new;

        $locker->acquire_read_lock($ref);
        $locker->acquire_write_lock($ref);
        $locker->release_read_lock($ref);
        $locker->release_write_lock($ref);
        $locker->release_all_locks($ref);


DESCRIPTION
       Apache::Session::Lock::Null fulfills the locking interface of Apache::Session, without actually doing any
       work.  This is the module to use if you want maximum performance and don't actually need locking.
