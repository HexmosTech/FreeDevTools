NAME
       exscn3d - Using the 3d scene functions. Allegro game programming library.


SYNOPSIS
       #include <allegro.h>

       Example exscn3d


DESCRIPTION
       This  program  demonstrates  how to use scanline sorting algorithm in Allegro (create_scene, clear_scene,
       ... functions). It also provides an example of how to use the 3D clipping function. The example  consists
       of a flyby through a lot of rotating 3d cubes.
