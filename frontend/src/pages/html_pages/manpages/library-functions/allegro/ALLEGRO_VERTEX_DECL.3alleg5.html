NAME
       ALLEGRO_VERTEX_DECL - Allegro 5 API


SYNOPSIS
              #include <allegro5/allegro_primitives.h>

              typedef struct ALLEGRO_VERTEX_DECL ALLEGRO_VERTEX_DECL;


DESCRIPTION
       A  vertex  declaration.   This  opaque structure is responsible for describing the format and layout of a
       user defined custom vertex.  It is created and destroyed by specialized functions.
