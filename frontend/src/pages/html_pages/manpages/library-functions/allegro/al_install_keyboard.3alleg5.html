NAME
       al_install_keyboard - Allegro 5 API


SYNOPSIS
              #include <allegro5/allegro.h>

              bool al_install_keyboard(void)


DESCRIPTION
       Install  a  keyboard  driver.   Returns  true  if successful.  If a driver was already installed, nothing
       happens and true is returned.
