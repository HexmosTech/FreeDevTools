NAME
       ustrcmp - Compares two strings. Allegro game programming library.


SYNOPSIS
       #include <allegro.h>

       int ustrcmp(const char *s1, const char *s2);


DESCRIPTION
       This function compares `s1' and `s2'.


RETURN VALUE
       Returns  zero if the strings are equal, a positive number if `s1' comes after `s2' in the ASCII collating
       sequence, else a negative number.
