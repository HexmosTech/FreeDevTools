NAME
       exmem  -  Drawing  onto  memory  bitmaps  and  then blitting them to the screen. Allegro game programming
       library.


SYNOPSIS
       #include <allegro.h>

       Example exmem


DESCRIPTION
       This program demonstrates the use of memory bitmaps. It creates a small temporary bitmap in memory, draws
       some circles onto it, and then blits lots of copies of it onto the screen.
