NAME
       remove_int - Removes a timers. Allegro game programming library.


SYNOPSIS
       #include <allegro.h>

       void remove_int(void (*proc)());


DESCRIPTION
       Removes  a function from the list of user interrupt routines. At program termination, allegro_exit() does
       this automatically.

       install_int(3alleg4), install_int_ex(3alleg4), remove_param_int(3alleg4)
