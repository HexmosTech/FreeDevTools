NAME
       al_set_standard_fs_interface - Allegro 5 API


SYNOPSIS
              #include <allegro5/allegro.h>

              void al_set_standard_fs_interface(void)


DESCRIPTION
       Return the ALLEGRO_FS_INTERFACE(3alleg5) table to the default, for the calling thread.
