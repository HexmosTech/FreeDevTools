NAME
       al_load_bitmap_font - Allegro 5 API


SYNOPSIS
              #include <allegro5/allegro_font.h>

              ALLEGRO_FONT *al_load_bitmap_font(const char *fname)


DESCRIPTION
       Load  a  bitmap  font  from a file.  This is done by first calling al_load_bitmap_flags(3alleg5) and then
       al_grab_font_from_bitmap(3alleg5).

       If you wanted to load an old A4 font, for example, it would be better to  load  the  bitmap  yourself  in
       order     to     call     al_convert_mask_to_alpha(3alleg5)     on    it    before    passing    it    to
       al_grab_font_from_bitmap(3alleg5).
