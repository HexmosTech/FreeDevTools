NAME
       al_ref_info - Allegro 5 API


SYNOPSIS
              #include <allegro5/allegro.h>

              const ALLEGRO_USTR *al_ref_info(const ALLEGRO_USTR_INFO *info)


DESCRIPTION
       Create  a  read-only  string that references the storage of another ALLEGRO_USTR(3alleg5) string that has
       already been stored in the ALLEGRO_USTR_INFO(3alleg5) structure.

       The string is valid until the underlying string is modified or destroyed.
