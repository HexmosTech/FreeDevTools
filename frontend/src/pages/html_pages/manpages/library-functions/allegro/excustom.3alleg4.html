NAME
       excustom - Creating custom GUI objects. Allegro game programming library.


SYNOPSIS
       #include <allegro.h>

       Example excustom


DESCRIPTION
       A follow up of the exgui.c example showing how to customise the default Allegro framework. In this case a
       dialog  procedure  animates a graphical clock without disrupting other GUI dialogs.  A more simple option
       shows how to dynamically change the font used by all GUI elements.
