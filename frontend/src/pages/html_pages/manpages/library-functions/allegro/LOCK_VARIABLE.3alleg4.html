NAME
       LOCK_VARIABLE - Locks the memory of a variable used by a timer. Allegro game programming library.


SYNOPSIS
       #include <allegro.h>

       Macro LOCK_VARIABLE(variable_name);


DESCRIPTION
       Due  to  interrupts,  you  are  required  to  lock  all  the  memory used by your timer routines. See the
       description of install_int_ex() for a more detailed explanation and usage example.
