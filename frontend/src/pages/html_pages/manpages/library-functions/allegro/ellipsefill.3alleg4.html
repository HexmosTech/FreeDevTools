NAME
       ellipsefill - Draws a filled ellipse. Allegro game programming library.


SYNOPSIS
       #include <allegro.h>

       void ellipsefill(BITMAP *bmp, int x, int y, int rx, int ry, int color);


DESCRIPTION
       Draws a filled ellipse with the specified centre and radius.
