NAME
       al_fread32le - Allegro 5 API


SYNOPSIS
              #include <allegro5/allegro.h>

              int32_t al_fread32le(ALLEGRO_FILE *f)


DESCRIPTION
       Reads a 32-bit word in little-endian format (LSB first).

       On  success,  returns  the  32-bit  word.  On failure, returns EOF (-1).  Since -1 is also a valid return
       value,  use  al_feof(3alleg5)  to  check  if  the  end  of  the  file   was   reached   prematurely,   or
       al_ferror(3alleg5) to check if an error occurred.
