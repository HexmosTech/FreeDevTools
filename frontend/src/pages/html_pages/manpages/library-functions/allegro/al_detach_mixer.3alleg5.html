NAME
       al_detach_mixer - Allegro 5 API


SYNOPSIS
              #include <allegro5/allegro_audio.h>

              bool al_detach_mixer(ALLEGRO_MIXER *mixer)


DESCRIPTION
       Detach the mixer from whatever it is attached to, if anything.
