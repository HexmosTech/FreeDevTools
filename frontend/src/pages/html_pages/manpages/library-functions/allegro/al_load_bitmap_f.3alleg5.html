NAME
       al_load_bitmap_f - Allegro 5 API


SYNOPSIS
              #include <allegro5/allegro.h>

              ALLEGRO_BITMAP *al_load_bitmap_f(ALLEGRO_FILE *fp, const char *ident)


DESCRIPTION
       Loads an image from an ALLEGRO_FILE(3alleg5) stream into a new ALLEGRO_BITMAP(3alleg5).  The file type is
       determined  by  al_identify_bitmap_f(3alleg5).   If  identification  is  not possible, the passed `ident'
       parameter, which is a file name extension including the leading dot, is used as a fallback, if it is  not
       NULL.

       This is the same as calling al_load_bitmap_flags_f(3alleg5) with 0 for the flags parameter.

       Returns NULL on error.  The file remains open afterwards.

              Note:  the  core Allegro library does not support any image file formats by default.  You must use
              the allegro_image addon, or register your own format handler.
