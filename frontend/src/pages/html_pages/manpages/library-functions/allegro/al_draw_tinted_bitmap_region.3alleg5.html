NAME
       al_draw_tinted_bitmap_region - Allegro 5 API


SYNOPSIS
              #include <allegro5/allegro.h>

              void al_draw_tinted_bitmap_region(ALLEGRO_BITMAP *bitmap,
                 ALLEGRO_COLOR tint,
                 float sx, float sy, float sw, float sh, float dx, float dy,
                 int flags)


DESCRIPTION
       Like al_draw_bitmap_region(3alleg5) but multiplies all colors in the bitmap with the given color.

       See al_draw_bitmap(3alleg5) for a note on restrictions on which bitmaps can be drawn where.
