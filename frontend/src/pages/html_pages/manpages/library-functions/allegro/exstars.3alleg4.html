NAME
       exstars - 3d starfield and lightsourced spaceship. Allegro game programming library.


SYNOPSIS
       #include <allegro.h>

       Example exstars


DESCRIPTION
       This  program  draws  a 3D star field (depth-cued) and a polygon starship (controllable with the keyboard
       cursor keys), using the Allegro math functions.
