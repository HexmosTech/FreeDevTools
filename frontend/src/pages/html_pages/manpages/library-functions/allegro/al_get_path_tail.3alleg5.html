NAME
       al_get_path_tail - Allegro 5 API


SYNOPSIS
              #include <allegro5/allegro.h>

              const char *al_get_path_tail(const ALLEGRO_PATH *path)


DESCRIPTION
       Returns the last directory component, or NULL if there are no directory components.
