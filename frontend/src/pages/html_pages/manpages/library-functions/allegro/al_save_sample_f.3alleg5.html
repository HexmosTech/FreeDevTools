NAME
       al_save_sample_f - Allegro 5 API


SYNOPSIS
              #include <allegro5/allegro_audio.h>

              bool al_save_sample_f(ALLEGRO_FILE *fp, const char *ident, ALLEGRO_SAMPLE *spl)


DESCRIPTION
       Writes  a  sample  into a ALLEGRO_FILE(3alleg5) filestream.  Currently, wav is the only supported format,
       and the extension must be “.wav”.

       Returns true on success, false on error.  The file remains open afterwards.

              Note: the allegro_audio library does not support any audio file formats by default.  You must  use
              the allegro_acodec addon, or register your own format handler.
