NAME
       ALLEGRO_SUB_VERSION - Defined to the middle version of Allegro.


SYNOPSIS
       #include <allegro.h>

       #define ALLEGRO_SUB_VERSION


DESCRIPTION
       Defined to the middle version of Allegro. From a version number like 4.1.16, this would be defined to the
       integer 1.

Allegro                                           version 4.4.3                     ALLEGRO_SUB_VERSION(3alleg4)
