NAME
       exscale - Loading PCX files and bitmap stretching. Allegro game programming library.


SYNOPSIS
       #include <allegro.h>

       Example exscale


DESCRIPTION
       This example demonstrates how to use PCX files, palettes and stretch blits. It loads a PCX file, sets its
       palette and does some random stretch_blits. Don't worry - it's VERY slowed down using vsync().
