NAME
       al_get_new_window_title - Allegro 5 API


SYNOPSIS
              #include <allegro5/allegro.h>

              const char *al_get_new_window_title(void)


DESCRIPTION
       Returns  the  title  that  will  be  used  when  a  new  display is created.  This returns the value that
       al_set_window_title(3alleg5) was called  with.   If  that  function  wasnâ€™t  called  yet,  the  value  of
       al_get_app_name(3alleg5)  is  returned  as  a default.  The current implementation returns a pointer to a
       static buffer of which you should make a copy if you want to modify it.
