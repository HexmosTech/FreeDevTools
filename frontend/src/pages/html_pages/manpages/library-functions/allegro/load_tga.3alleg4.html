NAME
       load_tga - Loads a TGA bitmap from a file. Allegro game programming library.


SYNOPSIS
       #include <allegro.h>

       BITMAP *load_tga(const char *filename, RGB *pal);


DESCRIPTION
       Loads a 256-color, 15-bit hicolor, 24-bit truecolor, or 32-bit truecolor+alpha TGA file.


RETURN VALUE
       Returns  a  pointer  to the bitmap or NULL on error. Remember that you are responsible for destroying the
       bitmap when you are finished with it to avoid memory leaks.
