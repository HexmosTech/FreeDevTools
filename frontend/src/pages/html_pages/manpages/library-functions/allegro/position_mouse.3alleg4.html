NAME
       position_mouse - Moves the mouse to the specified screen position. Allegro game programming library.


SYNOPSIS
       #include <allegro.h>

       void position_mouse(int x, int y);


DESCRIPTION
       Moves  the  mouse to the specified screen position. It is safe to call even when a mouse pointer is being
       displayed.
