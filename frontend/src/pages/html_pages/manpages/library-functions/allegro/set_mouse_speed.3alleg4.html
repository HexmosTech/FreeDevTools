NAME
       set_mouse_speed - Sets the mouse speed. Allegro game programming library.


SYNOPSIS
       #include <allegro.h>

       void set_mouse_speed(int xspeed, int yspeed);


DESCRIPTION
       Sets the mouse speed. Larger values of xspeed and yspeed represent slower mouse movement: the default for
       both is 2.

       install_mouse(3alleg4), set_mouse_range(3alleg4), position_mouse(3alleg4)
