NAME
       al_ustr_append_chr - Allegro 5 API


SYNOPSIS
              #include <allegro5/allegro.h>

              size_t al_ustr_append_chr(ALLEGRO_USTR *us, int32_t c)


DESCRIPTION
       Append a code point to the end of us.

       Returns the number of bytes added, or 0 on error.
