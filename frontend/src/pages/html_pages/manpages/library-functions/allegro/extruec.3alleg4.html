NAME
       extruec - Truecolor pixel format conversions. Allegro game programming library.


SYNOPSIS
       #include <allegro.h>

       Example extruec


DESCRIPTION
       This  program  shows  how to specify colors in the various different truecolor pixel formats. The example
       shows the same screen (a few text lines and three coloured  gradients)  in  all  the  color  depth  modes
       supported  by  your  video  card.  The  more  color  depth you have, the less banding you will see in the
       gradients.
