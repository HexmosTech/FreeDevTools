NAME
       remove_sound_input  -  Cleans  up  after  you  are  finished  with the sound input routines. Allegro game
       programming library.


SYNOPSIS
       #include <allegro.h>

       void remove_sound_input();


DESCRIPTION
       Cleans up after you are finished with the sound input routines. You don't normally  need  to  call  this,
       because remove_sound() and/or allegro_exit() will do it for you.


SEE ALSO
       install_sound_input(3alleg4), remove_sound(3alleg4), allegro_exit(3alleg4)
