NAME
       exunicod - Using Unicode string functions. Allegro game programming library.


SYNOPSIS
       #include <allegro.h>

       Example exunicod


DESCRIPTION
       This  program  demonstrates  the use of the 16-bit Unicode text encoding format with Allegro. The example
       displays a message translated to different languages scrolling on  the  screen  using  an  external  font
       containing the required characters to display those messages.

       Note  how  the  Allegro  unicode  string  functions resemble the functions you can find in the standard C
       library, only these handle Unicode on all platforms.
