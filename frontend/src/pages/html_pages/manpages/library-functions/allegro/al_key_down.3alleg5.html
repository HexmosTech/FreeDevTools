NAME
       al_key_down - Allegro 5 API


SYNOPSIS
              #include <allegro5/allegro.h>

              bool al_key_down(const ALLEGRO_KEYBOARD_STATE *state, int keycode)


DESCRIPTION
       Return true if the key specified was held down in the state specified.
