NAME
       al_calloc_with_context - Allegro 5 API


SYNOPSIS
              #include <allegro5/allegro.h>

              void *al_calloc_with_context(size_t count, size_t n,
                 int line, const char *file, const char *func)


DESCRIPTION
       This  calls  calloc()  from  the  Allegro  library  (this  matters  on  Windows),  unless overridden with
       al_set_memory_interface(3alleg5),

       Generally you should use the al_calloc(3alleg5) macro.
