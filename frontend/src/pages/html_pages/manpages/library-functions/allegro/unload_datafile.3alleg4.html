NAME
       unload_datafile - Frees all the objects in a datafile. Allegro game programming library.


SYNOPSIS
       #include <allegro.h>

       void unload_datafile(DATAFILE *dat);


DESCRIPTION
       Frees all the objects in a datafile. Use this to avoid memory leaks in your program.
