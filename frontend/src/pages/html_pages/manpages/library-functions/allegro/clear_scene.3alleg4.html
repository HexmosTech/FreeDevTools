NAME
       clear_scene - Initializes a scene. Allegro game programming library.


SYNOPSIS
       #include <allegro.h>

       void clear_scene(BITMAP *bmp);


DESCRIPTION
       Initializes a scene. The bitmap is the bitmap you will eventually render on.
