NAME
       allegro_id - String containing date and version number of Allegro.


SYNOPSIS
       #include <allegro.h>

       extern char allegro_id[];


DESCRIPTION
       Text  string  containing  a  date  and  version number for the library, in case you want to display these
       somewhere.
