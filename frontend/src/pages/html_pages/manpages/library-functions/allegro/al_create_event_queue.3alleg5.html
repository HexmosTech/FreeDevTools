NAME
       al_create_event_queue - Allegro 5 API


SYNOPSIS
              #include <allegro5/allegro.h>

              ALLEGRO_EVENT_QUEUE *al_create_event_queue(void)


DESCRIPTION
       Create  a new, empty event queue, returning a pointer to the newly created object if successful.  Returns
       NULL on error.
