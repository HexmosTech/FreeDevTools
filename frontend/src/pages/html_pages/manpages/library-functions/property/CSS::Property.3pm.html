NAME
       CSS::Property - A property in a CSS object tree


SYNOPSIS
         use CSS;


DESCRIPTION
       This module represents a property in a CSS object tree.  Read the CSS.pm pod for information about the
       CSS object tree.


METHODS
   CONSTRUCTORS
       new() or "new( { ..options.. } )"
           This constructor returns a new "CSS::Property" object, with an optional hash of options.

             property      property name (as string)
             value         simple value string
             adaptor       adaptor to use for serialization

           If  the  "value"  option  is  passed, a "CSS::Value" object is automatically created and added to the
           object's values list.

   ACCESSORS
       set_adaptor( 'CSS::Adaptor::Foo' )
           This method sets the current adaptor for the object.

       values()
           This method is used to serialize the property's values, using  the  current  adaptor.  It  returns  a
           string which comes from the adaptor's output_values() method.
