NAME
       s3dw_textbox_scrollleft - scroll text left


SYNOPSIS
       #include <s3dw.h>

       void s3dw_textbox_scrollleft(s3dw_textbox *textbox);


DESCRIPTION
       Scrolls the text in the textbox to the left by one character, if possible.


AUTHOR
       Simon Wunderlich
           Author of s3d
