NAME
       GSPMCI - (Set polymarker color index) - sets the polymarker color index.


SYNOPSIS
       CALL GSPMCI (COLI)


C-BINDING SYNOPSIS
       #include <ncarg/gks.h>

       void gset_marker_colr_ind(Gint marker_colr_ind);


DESCRIPTION
       COLI        (Integer, Input) - A color index.


USAGE
       All polymarkers drawn with calls to the GPM output primitive will be drawn with the color associated with
       index COLI until GSPMCI is called again and a new index is assigned.

       For  all  GKS  output  primitives, color is assigned using a color index. The color indices run from 0 to
       255, where 0 is the background color index and 1  is  the  foreground  color  index.   Color  values  are
       associated  with  indices  by  calls to the GKS routine GSCR.  If a color index is used that has no user-
       assigned color value set in a GSCR call, then a device-dependent color value will  be  assigned  to  that
       index.


ACCESS
       To use GKS routines, load the NCAR GKS-0A library ncarg_gks.
