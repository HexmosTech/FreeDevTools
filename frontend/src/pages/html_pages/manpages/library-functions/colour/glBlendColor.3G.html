NAME
       glBlendColor - set the blend color


C SPECIFICATION

       void glBlendColor(GLfloat red, GLfloat green, GLfloat blue, GLfloat alpha);


PARAMETERS
       red, green, blue, alpha
           specify the components of GL_BLEND_COLOR


DESCRIPTION
       The GL_BLEND_COLOR may be used to calculate the source and destination blending factors. The color
       components are clamped to the range 0 1 before being stored. See glBlendFunc() for a complete description
       of the blending operations. Initially the GL_BLEND_COLOR is set to (0, 0, 0, 0).


NOTES
       The type of the red, green, blue, and alpha parameters was changed from GLclampf to GLfloat. This change
       is transparent to user code and is described in detail on the removedTypes() page.


ASSOCIATED GETS
       glGet() with an argument of GL_BLEND_COLOR
