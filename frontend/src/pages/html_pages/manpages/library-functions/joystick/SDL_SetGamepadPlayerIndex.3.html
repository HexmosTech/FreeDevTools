NAME
       SDL_SetGamepadPlayerIndex - Set the player index of an opened gamepad.


SYNOPSIS
       #include <SDL3/SDL_gamepad.h>

       bool SDL_SetGamepadPlayerIndex(SDL_Gamepad *gamepad, int player_index);


FUNCTION PARAMETERS
       gamepad
              the gamepad object to adjust.

       player_index
              player index to assign to this gamepad, or -1 to clear the player index and turn off player LEDs.


RETURN VALUE
       for more information.


AVAILABILITY
       This function is available since SDL 3.2.0.
