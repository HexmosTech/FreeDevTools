NAME
       OpenGL::GLUT - Perl bindings to GLUT/FreeGLUT GUI toolkit


SYNOPSIS
         use OpenGL::GLUT qw(:all); # now can use GLUT calls


DESCRIPTION
       OpenGL::GLUT is, as of 0.7202, back as part of the main OpenGL distribution, but now it works separately.
       The purpose is to make this functionality available independent of the legacy OpenGL module for use with
       OpenGL::Modern.

       When you register a "glutCloseFunc", ensure that you de-register it before destroying the window. See the
       supplied test.pl.

   EXPORT
         :all - exports all GLUT functions and constants
         :constants - export only GLUT constants (same as :glutconstants)
         :functions - export only GLUT functions (same as :glutfunctions)
