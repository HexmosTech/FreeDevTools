NAME
       Test::BDD::Cucumber::Harness::TAP - Generate results in TAP format


VERSION
       version 0.87


DESCRIPTION
       A Test::BDD::Cucumber::Harness subclass whose output is TAP (Test Anything Protocol), such as consumed by
       "prove" and "yath".


OPTIONS
   fail_skip
       Boolean - makes tests with no matcher fail
