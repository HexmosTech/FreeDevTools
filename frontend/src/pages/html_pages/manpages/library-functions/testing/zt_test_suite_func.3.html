NAME
       zt_test_suite_func — type of pointer to a test suite function


SYNOPSIS
       typedef void
       (*zt_test_suite_func)(zt_visitor);


DESCRIPTION
       zt_test_suite_func  is a pointer to a function implementing a test suite, which groups any number of test
       cases and other test suites.

       A single test suite, passed as an argument to zt_main(), is used to enumerate and discover all the  tests
       in a given test program.


IMPLEMENTATION NOTES
       The zt_visitor argument is used to implement the test case and test suite enumeration system.


SEE ALSO
       zt_main(3), zt_visitor(3)
