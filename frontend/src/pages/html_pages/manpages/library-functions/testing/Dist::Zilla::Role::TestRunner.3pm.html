NAME
       Dist::Zilla::Role::TestRunner - something used as a delegating agent to 'dzil test'


VERSION
       version 6.032


DESCRIPTION
       Plugins implementing this role have their "test" method called when testing.  It's passed the root
       directory of the build test dir and an optional hash reference of arguments.  Valid arguments include:

       ‚Ä¢   jobs -- if parallel testing is supported, this indicates how many to run at once


PERL VERSION
       This  module  should  work  on any version of perl still receiving updates from the Perl 5 Porters.  This
       means it should work on any version of perl released in the last two to three years.  (That  is,  if  the
       most recently released version is v5.40, then this module should work on both v5.40 and v5.38.)

       Although  it  may  work on older versions of perl, no guarantee is made that the minimum required version
       will not be increased.  The version may be increased for any reason, and there is no promise that patches
       will be accepted to lower the minimum required perl.


ATTRIBUTES
   default_jobs
       This attribute is the default value that should be used as the "jobs" argument to the "test" method.


METHODS
   test
       This method should throw an exception on failure.
