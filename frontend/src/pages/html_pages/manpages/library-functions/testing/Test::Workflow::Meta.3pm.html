NAME
       Test::Workflow::Meta - The meta-object added to all Test-Workflow test classes.


DESCRIPTION
       When you "use Test::Workflow" a function is added to you class named 'TEST_WORKFLOW' that returns the
       single Test-Workflow meta-object that tracks information about your class.
