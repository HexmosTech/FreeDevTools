NAME
       Test::Alien::CanCompile - Skip a test file unless a C compiler is available


VERSION
       version 2.84


SYNOPSIS
        use Test::Alien::CanCompile;


DESCRIPTION
       This is just a Test2 plugin that requires that a compiler be available.  Otherwise the test will be
       skipped.


SEE ALSO
       Test::Alien
