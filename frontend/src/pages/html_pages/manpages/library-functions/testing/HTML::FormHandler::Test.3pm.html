NAME
       HTML::FormHandler::Test - provides is_html method used in tests


VERSION
       version 0.40068


SYNOPSIS
       Simple 'is_html' method for testing form rendering against an expected value without having to fuss with
       exactly matching newlines and spaces. Uses HTML::TreeBuilder, which uses HTML::Parser.

       See numerous examples in the 't/render' directory.

          use Test::More;
          use HTML::FormHandler::Test;
          use_ok('MyApp::Form::Basic');
          my $form = MyApp::Form::Basic->new;
          $form->process;
          my $expected = '<form html>';
          is_html( $form->render, $expected, 'form renders ok' );
