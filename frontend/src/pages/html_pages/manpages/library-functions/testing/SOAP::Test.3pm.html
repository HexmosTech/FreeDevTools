NAME
       SOAP::Test - Test framework for SOAP::Lite


SYNOPSIS
         use SOAP::Test;

         SOAP::Test::Server::run_for('http://localhost/cgi-bin/soap.cgi');


DESCRIPTION
       SOAP::Test provides simple framework for testing server implementations.  Specify your address (endpoint)
       and run provided tests against your server.  See t/1*.t for examples.
