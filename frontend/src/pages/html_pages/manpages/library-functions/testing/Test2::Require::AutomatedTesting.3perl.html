NAME
       Test2::Require::AutomatedTesting - Only run a test when the AUTOMATED_TESTING environment variable is
       set.


DESCRIPTION
       It is common practice to write tests that are only run when the AUTOMATED_TESTING environment variable is
       set. This module automates the (admittedly trivial) work of creating such a test.


SYNOPSIS
           use Test2::Require::AutomatedTesting;

           ...

           done_testing;


SOURCE
       The source code repository for Test2-Suite can be found at https://github.com/Test-More/Test2-Suite/.
