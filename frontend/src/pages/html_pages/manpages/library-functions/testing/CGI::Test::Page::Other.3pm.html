NAME
       CGI::Test::Page::Other - A real page, but neither text nor HTML


SYNOPSIS
        # Inherits from CGI::Test::Page::Real


DESCRIPTION
       This class represents an HTTP reply containing neither "text/hmtl" nor "text/plain" data.  Its interface
       is the same as the one described in CGI::Test::Page::Real.


AUTHORS
