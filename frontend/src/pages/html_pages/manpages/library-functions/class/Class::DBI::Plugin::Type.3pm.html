NAME
       Class::DBI::Plugin::Type - Determine type information for columns


SYNOPSIS
         package Music::Artist;
         use base 'Class::DBI';
         use Class::DBI::Plugin::Type;
         Music::Artist->table('artist');
         Music::Artist->columns(All => qw/artistid name/);

         print Music::Artist->column_type("artistid"); # integer


DESCRIPTION
       This module allows "Class::DBI"-based classes to query their columns for data type information in a
       database-independent manner.
