NAME
       Test::DBIx::Class::Example::Schema::DefaultRS - The default ResultSet


DESCRIPTION
       DBIx::Class will use a resultset class which shares a name with the Result class.  If one does not exist,
       use this class as the default.

       The following modules or resources may be of interest.
