NAME
       Class::MOP::Mixin::HasMethods - Methods for metaclasses which have methods


VERSION
       version 2.2207


DESCRIPTION
       This class implements methods for metaclasses which have methods (Class::MOP::Class and
       Moose::Meta::Role). See Class::MOP::Class for API details.
