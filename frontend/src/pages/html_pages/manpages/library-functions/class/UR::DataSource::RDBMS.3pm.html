NAME
       UR::DataSource::RDBMS - Abstract base class for RDBMS-type data sources


DESCRIPTION
       This class implements the interface UR uses to query RDBMS databases with DBI.  It encapsulates the
       system's knowledge of classes/properties relation to tables/columns, and how to generate SQL to create,
       retrieve, update and delete table rows that represent object instances.
