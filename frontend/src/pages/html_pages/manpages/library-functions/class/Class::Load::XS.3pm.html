NAME
       Class::Load::XS - XS implementation of parts of Class::Load


VERSION
       version 0.10


SYNOPSIS
           use Class::Load;


DESCRIPTION
       This module provides an XS implementation for portions of Class::Load. See Class::Load for API details.
