NAME
       DBIx::Class::SQLMaker::ClassicExtensions - Class containing generic enhancements to
       SQL::Abstract::Classic


DESCRIPTION
       This module is not intended to be used standalone. Instead it represents a quasi-role, that one would
       "mix in" via classic @ISA inheritance into a DBIx::Class::SQLMaker-like provider. See
       "connect_call_rebase_sqlmaker" in DBIx::Class::Storage::DBI for more info.

       Currently the enhancements over SQL::Abstract::Classic are:

       •   Support for "JOIN" statements (via extended "table/from" support)

       •   Support of functions in "SELECT" lists

       •   "GROUP BY"/"HAVING" support (via extensions to the order_by parameter)

       •   A rudimentary multicolumn IN operator

       •   Support of "...FOR UPDATE" type of select statement modifiers
