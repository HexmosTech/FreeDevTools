NAME
       wxCalendarDateAttr - Functions for wxCalendarDateAttr class


DESCRIPTION
       wxCalendarDateAttr  is  a  custom attributes for a calendar date. The objects of this class are used with
       wxCalendarCtrl.

       See: wxCalendarCtrl

       wxWidgets docs: wxCalendarDateAttr
