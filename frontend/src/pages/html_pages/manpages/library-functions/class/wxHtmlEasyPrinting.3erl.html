NAME
       wxHtmlEasyPrinting - Functions for wxHtmlEasyPrinting class


DESCRIPTION
       This class provides very simple interface to printing architecture. It allows you to print HTML documents
       using only a few commands.

       Note:  Do  not  create this class on the stack only. You should create an instance on app startup and use
       this instance for all printing operations. The reason is that this class stores various settings in it.

       wxWidgets docs: wxHtmlEasyPrinting
