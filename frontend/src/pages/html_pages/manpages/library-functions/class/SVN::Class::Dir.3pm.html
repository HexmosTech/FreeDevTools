NAME
       SVN::Class::Dir - represents a directory in a Subversion workspace


SYNOPSIS
        # see SVN::Class;


DESCRIPTION
       This class subclasses Path::Class::Dir and SVN::Class.  A SVN::Class::Dir object behaves like a
       Path::Class::Dir object, but with the extra Subversion functionality of SVN::Class.


METHODS
       There are no new methods implemented in this class.

       The following methods are overridden to return SVN::Class-derived objects instead of Path::Class-derived
       objects.

       new
       file
