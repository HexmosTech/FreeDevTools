NAME
       DBIx::Class::InflateColumn::FS::ResultSet - FS columns resultset class


DESCRIPTION
       Derive from this class if you intend to provide a custom resultset class for result sources including
       DBIx::Class::InflateColumn::FS columns.


METHODS
   delete
       Delete associated file system storage for each row in a result set.
