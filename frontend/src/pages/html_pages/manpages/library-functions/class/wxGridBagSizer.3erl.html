NAME
       wxGridBagSizer - Functions for wxGridBagSizer class


DESCRIPTION
       A  wxSizer  that  can  lay  out  items in a virtual grid like a wxFlexGridSizer but in this case explicit
       positioning of the items is allowed using wxGBPosition (not implemented in wx), and items can  optionally
       span more than one row and/or column using wxGBSpan (not implemented in wx).

       This class is derived (and can use functions) from: wxFlexGridSizer wxGridSizer wxSizer

       wxWidgets docs: wxGridBagSizer
