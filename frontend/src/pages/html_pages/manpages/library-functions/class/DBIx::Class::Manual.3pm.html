NAME
       DBIx::Class::Manual - Index of the Manual


DESCRIPTION
       This is the DBIx::Class users manual. DBIx::Class is a SQL->OOP mapper.  This means that it can represent
       your SQL tables as perl classes, and give you convenient accessors and methods for retrieving and
       updating information from your SQL database.


SECTIONS
   DBIx::Class::Manual::FAQ
       Short answers and doc pointers to questions.

   DBIx::Class::Manual::Glossary

   DBIx::Class::Manual::Intro

   DBIx::Class::Manual::Cookbook

   DBIx::Class::Manual::Troubleshooting

       If you're using the CDBI Compat layer, we suggest reading the Class::DBI documentation. It should behave
       the same way.

   DBIx::Class::Manual::Component
