NAME
       PAR::Filter::PatchContent - Content patcher


SYNOPSIS
           # transforms $code
           PAR::Filter::PatchContent->apply(\$code, $filename, $name);


DESCRIPTION
       This filter fixes PAR-incompatible modules; pp applies it to modules by default.

       PAR::Filter
