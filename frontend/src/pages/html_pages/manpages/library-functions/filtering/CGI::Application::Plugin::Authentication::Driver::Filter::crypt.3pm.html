NAME
       CGI::Application::Plugin::Authentication::Driver::Filter::crypt - crypt Filter


METHODS
   filter ( undef, $string [, salt ] )
       This will generate a crypted string.  The first parameter is always ignored, since there is only one way
       to use the crypt function.  You can pass in an extra parameter to act as the salt.

        my $filtered = $class->filter(undef, 'foobar'); # mQvbWI43eDCAk


        -or-
        my $filtered = $class->filter(undef, 'foobar', 'AA'); # AAZk9Aj5/Ue0E

   check ( undef, $string, $crypted )
       This will crypt the string, and compare it against the provided crypted string.  The first parameter is
       always ignored, since there is only one way to use the crypt function.

        if ($class->check(undef, 'foobar', 'mQvbWI43eDCAk')) {
            # they match
        }
