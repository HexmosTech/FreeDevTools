NAME
       POE::Filter::Stomp - Perl extension for the POE Environment


SYNOPSIS
         use POE::Filter::Stomp;

         For a server

         POE::Component::Server::TCP->new(
             ...
             Filter => 'POE::Filter::Stomp',
             ...
         );

         For a client

         POE::Component::Client::TCP->new(
             ...
             Filter => 'POE::Filter::Stomp',
             ...
         );


DESCRIPTION
       This module is a filter for the POE environment. It will translate the input buffer into
       Net::Stomp::Frame objects and serialize the output buffer from said objects. For more information an the
       STOMP protocol, please refer to: http://stomp.codehaus.org/Protocol .


EXPORT
       None by default.


SEE ALSO
       See the documentation for POE::Filter for usage.
