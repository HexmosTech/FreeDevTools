NAME
       MPRSET - Reset the internal state of EZMAP/EZMAPA to the default.


SYNOPSIS
       CALL MPRSET


C-BINDING SYNOPSIS
       #include <ncarg/ncargC.h>

       void c_mprset()


DESCRIPTION
       MPRSET has no arguments.


USAGE
       MPRSET - Calling this routine resets the internal parameters of EZMAP/EZMAPA to the default state and
       recalls the routine MAPINT to initialize the package.


EXAMPLES
       No example is available for MPRSET.


ACCESS
       To use MPRSET or c_mprset, load the NCAR Graphics libraries ncarg, ncarg_gks, and ncarg_c, preferably in
       that order.
