NAME
       PX_open_fp — read a Paradox file


SYNOPSIS
       #include <paradox.h>

       int PX_open_fp(pxdoc_t *pxdoc, FILE *fp)


DESCRIPTION
       Reads  a Paradox file with the given file pointer which must have been opened before with fopen(3). pxdoc
       must be created before with PX_new(3) or PX_new2(3).


RETURN VALUE
       Returns 0 on success and -1 on failure.

       PX_new2(3), PX_new(3), PX_open_file(3), PX_open_gsf(3)
