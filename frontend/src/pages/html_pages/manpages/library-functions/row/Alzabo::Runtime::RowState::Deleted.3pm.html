NAME
       Alzabo::Runtime::RowState::Deleted - Row objects that have been deleted


SYNOPSIS
         $row->delete;


DESCRIPTION
       This state is used for deleted rows, any row upon which the "delete()" method has been called.


METHODS
       See "Alzabo::Runtime::Row".
