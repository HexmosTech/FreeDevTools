NAME
       s3d_load_polygon_tex_coords - add texture coordinates to polygons


SYNOPSIS
       #include <s3d.h>

       int s3d_load_polygon_tex_coords(int object, const float *tbuf, uint32_t start, uint16_t n);


DESCRIPTION
       Just as s3d_pep_polygon_tex_coords(), with the difference you won't alter the latest n polygons but n
       polygons starting with index start.
