NAME
       PAPI_get_real_nsec - Get real time counter value in nanoseconds.


SYNOPSIS
Detailed Description
       This function returns the total real time passed since some arbitrary starting point. The time is
       returned in nanoseconds. This call is equivalent to wall clock time.
