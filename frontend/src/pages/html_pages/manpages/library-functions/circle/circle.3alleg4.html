NAME
       circle - Draws a circle. Allegro game programming library.


SYNOPSIS
       #include <allegro.h>

       void circle(BITMAP *bmp, int x, int y, int radius, int color);


DESCRIPTION
       Draws a circle with the specified centre and radius.
