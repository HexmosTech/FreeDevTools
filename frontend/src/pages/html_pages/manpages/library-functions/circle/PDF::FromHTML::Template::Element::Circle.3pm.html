NAME
       PDF::FromHTML::Template::Element::Circle - To draw a circle


NODE NAME
       CIRCLE


INHERITANCE
       PDF::FromHTML::Template::Element


ATTRIBUTES
       •   R This is the radius of the circle to be drawn

       •   COLOR This is the color the circle should be drawn in. Defaults to black.

       •   FILLCOLOR This is the color the circle should be filled in with. Defaults to none.

       •   WIDTH This is the width of the line used to draw the circle. Defaults to 1 pixel.


CHILDREN
       None


AFFECTS
       Nothing


DEPENDENCIES
       None


USAGE
         <circle R="1i" color="255,0,0"/>

       This will cause a 1-inch radius circle to be drawn at the current position in red.
