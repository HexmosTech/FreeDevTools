NAME
       HTTP::DAV::Lock - Represents a WebDAV Lock.


SYNOPSIS
        Need example


DESCRIPTION
ACCESSOR METHODS
       Timeout This parameter can take an absolute or relative timeout. The following forms are all valid for
       the -timeout field:
           Timeouts in:
               300
               30s                              30 seconds from now
               10m                              ten minutes from now
               1h                               one hour from now
               1d                               tomorrow
               3M                               in three months
               10y                              in ten years time Timeout at:
               2000-02-31 00:40:33              at the indicated time & date
               For more time and date formats that are handled see HTTP::Date

           RFC2518  states that the timeout value MUST NOT be greater than 2^32-1. If this occurs it will simply
           set the timeout to infinity

       $r->as_string()
           Method returning a textual representation of the request.  Mainly useful for debugging  purposes.  It
           takes no arguments.
