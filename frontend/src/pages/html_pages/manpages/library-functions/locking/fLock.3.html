NAME
       fLock  - acquire file lock.


SYNOPSIS
       #include <util/ulocks.h>

       NEOERR *fLock(int lock);


ARGUMENTS
       lock - Lock to acquire.


DESCRIPTION
       Acquires the lock identified by <lock>.  This call blocks until the lock is acquired.

       MT-Level: Safe.


RETURN VALUE
       None.
