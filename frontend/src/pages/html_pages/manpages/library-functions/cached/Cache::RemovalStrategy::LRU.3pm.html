NAME
       Cache::RemovalStrategy::LRU - LRU Removal Strategy for a Cache


DESCRIPTION
       Implements a Least Recently Used removal strategy for a Cache.  When removing entries from the cache, the
       'stalest' will be removed first.


METHODS
       See Cache::RemovalStrategy for details.
