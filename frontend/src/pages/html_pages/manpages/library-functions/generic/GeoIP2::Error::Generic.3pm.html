NAME
       GeoIP2::Error::Generic - A generic exception


VERSION
       version 2.006002


SYNOPSIS
         use 5.008;

         use GeoIP2::WebService::Client;
         use Scalar::Util qw( blessed );
         use Try::Tiny;

         my $client = GeoIP2::WebService::Client->new(
             account_id  => 42,
             license_key => 'abcdef123456',
         );

         try {
             $client->insights( ip => '24.24.24.24' );
         }
         catch {
             die $_ unless blessed $_;
             die $_ if $_->isa('GeoIP2::Error::Generic');

             # handle other exceptions
         };


DESCRIPTION
       This class represents a generic error. It extends Throwable::Error and does not add any additional
       attributes.


METHODS
       This class has two methods, "$error->message()", and "$error->stack_trace()". Both methods are inherited
       from Throwable::Error.
