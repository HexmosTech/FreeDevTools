NAME
       check-selinux-installation — perform configuration checks in SELinux installation


SYNOPSIS
       check-selinux-installation


DESCRIPTION
       check-selinux-installation  this  command  will  run  scripts  to pull in some standard SELinux stuff via
       dependencies.

       policy-upgrade(8),
