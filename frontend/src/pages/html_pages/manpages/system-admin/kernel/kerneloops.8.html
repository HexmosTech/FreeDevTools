NAME
       kerneloops - program to collect and submit kernel oopses to oops.kernel.org


SYNOPSIS
       kerneloops


DESCRIPTION
       This manual page documents briefly the kerneloops commands.

       kerneloops  is  a program that collects kernel crash information and then submits the extracted signature
       to the oops.kernel.org website for statistical analysis and presentation to the Linux kernel developers.


OPTIONS
       --file filename
              Parse the file denoted with filename as if it were /var/log/messages

       --debug
              Enable debug mode

       --nodaemon
              Do not daemonize


FILES
       /etc/kerneloops.conf
