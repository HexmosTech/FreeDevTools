NAME
       x2goterminate-session - Terminate an X2Go Session


SYNOPSIS
       x2goterminate-session <session_id>


DESCRIPTION
       x2goterminate-session cleanly terminates the running or suspended session with session ID <session_id>.

       This command is normally issued by an X2Go client application with X2Go user privileges.


RETURN VALUES
       By x2goterminate-session nothing is reported on stdout or stderr.

       As exitcode x2goterminate-session always returns 0.
