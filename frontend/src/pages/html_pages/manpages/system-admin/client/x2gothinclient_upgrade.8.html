NAME
       x2gothinclient_upgrade - Upgrade the X2Go Thin Client's software.


SYNOPSIS
       x2gothinclient_upgrade


DESCRIPTION
       This helper script performs a package upgrade within the X2Go thin client chroot. Use this script to
       update/upgrade your X2Go thin client's software.


OPTIONS
       x2gothinclient_upgrade has no known options.


FILES
       /etc/x2go/x2gothinclient_settings
           Configuration file for x2gothinclient_* scripts.
