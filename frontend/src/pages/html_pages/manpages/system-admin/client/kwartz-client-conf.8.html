NAME
       kwartz-client-conf - program to configure a GNU-Linux client machine in order to consider a Kwartz server


SYNOPSIS

       sudo kwartz-client-conf


DESCRIPTION
       kwartz-client-conf allows the sysadmin to reconfigure a GNU-Linux client machine to take advantage of a
       Kwartz server, which can offer LDAP authentication, data access through Samba shares, and more.

       The sysadmin only needs to fill a few forms, and the configuration files are rewritten consistently.
