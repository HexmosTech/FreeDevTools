NAME
       thin_ls - List thin volumes within a pool.


SYNOPSIS
       thin_ls [options] {device|file}


DESCRIPTION
       thin_ls  displays information about thin volumes in a pool. Pass the metadata device on the command line,
       not the pool device.

       This tool cannot be run on live metadata unless the --metadata-snap option is used.


OPTIONS
       -h, --help
              Print help and exit.

       -V, --version
              Print version information and exit.

       -o, --format
              Give a comma separated list of fields to be output.

           Valid fields are:
             DEV, MAPPED_BLOCKS, EXCLUSIVE_BLOCKS, SHARED_BLOCKS, HIGHEST_BLOCK,
             MAPPED_SECTORS, EXCLUSIVE_SECTORS, SHARED_SECTORS, HIGHEST_SECTOR,
             MAPPED_BYTES, EXCLUSIVE_BYTES, SHARED_BYTES, HIGHEST_BYTE, MAPPED,
             EXCLUSIVE, SHARED, HIGHEST_MAPPED, TRANSACTION, CREATE_TIME, SNAP_TIME

       --no-headers
              Don't output headers.

       -m, --metadata-snap
              Use metadata snapshot.

           If you want to get information out of a live pool then you will need to
           take a metadata snapshot and use this switch.
