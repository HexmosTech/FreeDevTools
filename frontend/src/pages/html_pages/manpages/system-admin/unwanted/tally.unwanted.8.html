NAME
       tally.unwanted - keep track of unwanted newsgroups.


SYNOPSIS
       tally.unwanted


DECSRIPTION
       tally.unwanted  is  normally  invoked  by  scanlogs(8).   It  parses  the  article log file to update the
       cumulative list of articles posted to unwanted newsgroups, unwanted.log.
