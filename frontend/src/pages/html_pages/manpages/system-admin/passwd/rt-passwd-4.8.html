NAME
       rt-passwd - Set an RT user's password


SYNOPSIS
           rt-passwd username
           rt-passwd user_id

       rt-passwd will set the password of an RT user to the value the user enters at the prompt.

   OPTIONS
       --help
           Print this documentation and exit.
