NAME
        basic_db_auth - Database auth helper for Squid


SYNOPSIS
        basic_db_auth [options]


DESCRIPTION
       This program verifies username & password to a database


OPTIONS
       --debug     Write debug info to stderr.

       --dsn       Database DSN. Default "DBI:mysql:database=squid"

       --user      Database User

       --password  Database password

       --table     Database table. Default "passwd".

       --usercol   Username column. Default "user".

       --passwdcol Password column. Default "password".

       --cond      Condition,  defaults to enabled=1. Specify 1 or "" for no condition If you use --joomla flag,
                   this condition will be changed to block=0

       --plaintext Database contains plain-text passwords

       --md5       Database contains unsalted MD5 passwords

       --sha1      Database contains unsalted SHA1 passwords

       --salt      Selects the correct salt to evaluate passwords

       --persist   Keep a persistent database connection open between queries.

       --joomla    Tells helper that user database is Joomla DB.  So their unusual salt hashing is understood.
