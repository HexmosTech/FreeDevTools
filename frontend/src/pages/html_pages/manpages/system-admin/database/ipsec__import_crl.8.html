NAME

       ipsec__import_crl - helper program for importing a crl to the NSS database

DESCRIPTION

       _import_crl is spawned by pluto in order to add or update a CRL in the NSS database (default:
       /var/lib/ipsec/nss)

SEE ALSO

       pluto(8).
