NAME
       btrfs-extent-same - directly access the btrfs-extent-same ioctl


SYNOPSIS
       btrfs-extent-same extent-len file1 offset1 ...


DESCRIPTION
       btrfs-extent-same extent-len and all offsets should be specified in bytes.
