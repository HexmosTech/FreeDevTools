Name
       vcsaview — display a screen dump in VCSA format


Syntax
       vcsaview [file...]


Description
       Displays a screen grab in VCSA format, as produced by /dev/vcsa*.


Examples
            vcsaview /dev/vcsa1


See also
       hxtools(7)
