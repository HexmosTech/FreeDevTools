NAME
       tpm_mkuuid - generate a TPM UUID


SYNOPSIS
       tpm_uuid [-hv] UUID-FILE


DESCRIPTION
       The program generates a TPM UUID and stores it in the file UUID-FILE.

       -h     Display command usage info.

       -v     Display command version info.
