NAME
       zshutdown_notify - notify users of impending shutdown via Zephyr


SYNOPSIS
       zshutdown_notify


DESCRIPTION
       zshutdown_notify  reads  the standard input until EOF is reached, and then sends a Zephyr notice with the
       official hostname, the collected input, and a warning message as the message body  of  the  notice.   The
       notice  is  sent  to  class  FILSRV, instance <hostname>, recipient "*", where <hostname> is the official
       hostname of the sending host.

       Any users who are using resources on the sending host are expected to subscribe to these messages so they
       will be warned of any impending shutdowns.

       zshutdown_notify is usually invoked from within /etc/shutdown.
