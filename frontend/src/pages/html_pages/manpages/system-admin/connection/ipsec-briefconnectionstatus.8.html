NAME
       ipsec-briefconnectionstatus - Get brief connection status from IPsec service


SYNOPSIS

       ipsec briefconnectionstatus [connection] [--ctlsocket /run/pluto/pluto.ctl]


DESCRIPTION
       ipsec briefconnectionstatus list brief connection status from the pluto daemon for all connections or
       optionally specified connection. Displays one line per connection showing client and host IPs, bytes,
       connection name, and reqid. Get complete connection status with ipsec connectionstatus.
