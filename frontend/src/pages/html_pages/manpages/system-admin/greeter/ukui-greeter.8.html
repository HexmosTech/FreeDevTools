NAME
       ukui-greeter - LightDM greeter for the UKUI desktop


SYNOPSIS
       ukui-greeter


DESCRIPTION
       UKUI Greeter is a LightDM greeter for the UKUI desktop.  It is run by the LightDM daemon if configured in
       lightdm.conf.
