NAME
       amoldrecover - amrecover variant compantible with old Amanda protocols


DESCRIPTION
       This  program  has the same syntax and functionality as amrecover, but uses the older amidxtaped/amindexd
       protocol.  It is provided only for compatibility with old Amanda releases.

       This manual page was written for the Debian GNU/Linux distribution because the original program does  not
       have a manual page.
