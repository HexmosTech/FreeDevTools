NAME
       gtp-echo-responder - gtp-echo-responder


SYNOPSIS
       gtp-echo-responder [-h] [-V] [-l listen_addr]


DESCRIPTION
       Stand alone program to respond to echo requests.


OPTIONS
       -h --help
              This help text

       -V --version
              Print the version of gtp-echo-responder

       -l --listen-addr
              Listend address for GTPCv1 and GTPCv2

       -R --recovery-counter GTP Recovery Counter to transmit in GTP Echo Response message

       -n --node-features
              GTPCv2 Node Features bitmask to transmit in GTP Echo Response message
