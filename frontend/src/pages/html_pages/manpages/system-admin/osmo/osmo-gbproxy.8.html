NAME
       osmo-gbproxy - The Gb proxy


SYNOPSIS
       osmo-gbproxy [options]


DESCRIPTION
       osmo-gbproxy is part of the OpenBSC base station controller from Osmocom.

       It is used to aggregate the Gb links of multiple BSS's and present them in one Gb link to the SGSN.


OPTIONS
       -h,--help
              this text

       -d option,--debug=DNS:DGPRS,0:0
              enable debugging

       -D,--daemonize
              Fork the process into a background daemon

       -c,--config-file <filename>
              The config file to use.

       -s,--disable-color

       -T,--timestamp
              Prefix every log line with a timestamp

       -V,--version.
              Print the version of OpenBSC.

       -e,--log-level <number>
              Set a global loglevel.
