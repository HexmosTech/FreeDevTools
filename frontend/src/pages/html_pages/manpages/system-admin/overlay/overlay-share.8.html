NAME
       overlay-share - Utility to bind mount directory into subhost.


SYNOPSIS
       overlay-share path subhostname


DESCRIPTION
       overlay-share is a utility command script for bind-mounting a directory into a subhost under the same
       pathname.


OPTIONS
       no options.


EXAMPLES
               # overlay-share /home/guest example

       The above bind-mounts the main hostâ€™s directory tree /home/guest into the subhost example and thereby
       making that directory subtree shared between the subhost and the main host.
