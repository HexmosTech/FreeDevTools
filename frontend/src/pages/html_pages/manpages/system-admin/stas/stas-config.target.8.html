NAME
       stas-config.target - Used to synchronize the start of nvme-stas processes


SYNOPSIS
       /usr/lib/systemd/system/stas-config.target


DESCRIPTION
       This target is used as a synchronization point before starting stacd.service(8) and stafd.service(8).

       It ensures that /etc/nvme/hostnqn and /etc/nvme/hostid are present before starting stacd.service(8) and
       stafd.service(8).


SEE ALSO
       stacd(8) stafd(8)
