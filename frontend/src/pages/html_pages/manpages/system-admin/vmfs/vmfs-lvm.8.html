NAME
       vmfs-lvm - VMFS logical volume manager tool


SYNOPSIS
       vmfs-lvm VOLUME... COMMAND


DESCRIPTION
       The vmfs-lvm program allows handle VMFS physical and logical volumes.

       The VOLUME to be opened can be either a block device or an image file. When the VMFS spreads accross
       several extents, all extents must be given.


COMMANDS
       remove
           Removes the last extent of the logical volume. THIS IS EXPERIMENTAL. USE AT YOUR OWN RISK. It is
           highly recommended that the volume is not mounted at the same time. This means the tool shouldnâ€™t be
           run on the ESX service console.
