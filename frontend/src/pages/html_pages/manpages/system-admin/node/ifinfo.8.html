NAME
       ifinfo - Query node information


SYNOPSIS
       ifinfo [-h] [-x verbosity] [-I file] node


DESCRIPTION
       Ifinfo  displays  various  information (system name, Sysop, Location, PhoneNo, Speed, NL-Flags and PFlag)
       about the specified node from the nodelist (using the nodelist index for lookup).


OPTIONS
       -h     Display a short help message.

       -x Verbosity
              Set the debug verbosity.  Verbosity may be a number from 0 to 32  to  set  'on'  bits  from  1  to
              number, or a string of letters where 'a' = bit 1, 'b' = bit 2, etc. up to bit 26.

       -I File
              Use the alternate configuration file File.


RETURN CODES
       0    Successfull
       64   command line usage error
       65   error reading config file


DEBUG VERBOSITY
       d   lock files
       f   aka matching
       g   outbound directories
       h   call
       j   session handshake
       k   file transfer
       l   filelist
       m   nodelist flags
       r   modem
       t   nodelist index
       z   hydra protocol


FILES
       $IFLIBDIR/config
              Runtime configuration file.

       nodelist.???
              A list of nodes in an FTN network.
