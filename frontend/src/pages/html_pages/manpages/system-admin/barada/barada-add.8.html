NAME
       barada-add - Add a secret key and pin for HOTP authentication


SYNOPSIS
       barada-add LOGIN PIN


DESCRIPTION
       Creates  a  key  for the specified LOGIN and records the given PIN for later HOTP authentication with the
       pam_barada (8) PAM module.


SEE ALSO
       pam_barada(8)
