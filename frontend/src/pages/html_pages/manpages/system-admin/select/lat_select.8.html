NAME
       lat_select - select benchmark


SYNOPSIS
       lat_ctx [ -P <parallelism> ] [ -W <warmups> ] [ -N <repetitions> ] [ n ]


DESCRIPTION
       lat_select measures the time to do a select on n file descriptors.
