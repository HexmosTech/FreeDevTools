NAME
       arptables-legacy-save - Save ARP Tables


SYNOPSIS
       arptables-legacy-save


DESCRIPTION
       arptables-legacy-save  is used to dump the contents of an ARP Table in easily parseable format to STDOUT.
       Use I/O-redirection provided by your shell to write to a file.
