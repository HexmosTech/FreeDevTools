NAME
       stas-config@.service - Used for auto-generation of nvme-stas configuration files.


SYNOPSIS
       /usr/lib/systemd/system/stas-config@.service


DESCRIPTION
       This service is used for the automatic run-time generation of NVMe configuration located in /etc/nvme
       (e.g.  /etc/nvme/hostnqn). This is needed by stacd.service(8) and stafd.service(8).


SEE ALSO
       stacd(8) stafd(8)
