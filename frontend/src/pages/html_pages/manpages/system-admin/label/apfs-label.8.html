NAME
       apfs-label - list the labels of all volumes in an APFS filesystem


SYNOPSIS
       apfs-label device


DESCRIPTION
       apfs-label  is  a simple tool that lists the index and label of all volumes in an APFS container, to help
       the user decide which one to mount with the driver.


OPTIONS
       -v     Print the version number of apfs-label and exit.
