NAME
       kopano-mr-process - Automatic processing of meeting requests.


SYNOPSIS

       kopano-mr-process


DESCRIPTION
       Script will be called by kopano-dagent when the user receives a meeting request or cancellation.

       This script should not be executed from the command line.
