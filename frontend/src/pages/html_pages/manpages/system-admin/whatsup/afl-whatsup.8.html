NAME
       afl-whatsup


SYNOPSIS
       Usage: ./afl-whatsup [-s] [-d] afl_output_directory


OPTIONS
       Options:
         -d  -  include dead fuzzer stats
         -m  -  just show minimal stats
         -n  -  no color output
         -s  -  skip details and output summary results only
