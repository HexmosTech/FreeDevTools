NAME
       pure-ftpd-control - invokes init script for installed Pure-FTPd flavour


SYNOPSIS
         pure-ftpd-control start
         pure-ftpd-control stop
         pure-ftpd-control restart


DESCRIPTION
       There are four flavours for the Pure-FTPd daemon in Debian: pure-ftpd, pure-ftpd-mysql, pure-ftpd-
       postgresql and pure-ftpd-ldap.  This script invokes the init script for the flavour installed, if any.
