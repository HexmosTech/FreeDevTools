NAME
       jabberd2-s2s - server-to-server connector


SYNOPSIS
       jabberd2-s2s [-h] [-c config] [-D]


DESCRIPTION
       jabberd2-s2s manages connections between local components and other Jabber servers.


OPTIONS
       -h     Show summary of options.

       -c     Alternate configuration file to use. The compiled-in default is /etc/jabberd2/s2s.xml.

       -D     Print  debugging  output.  You  should  configure jabberd with the --enable-debug switch to enable
              this.
