NAME
       ipsec-listpubkeys - List all public keys from pluto internal database.


SYNOPSIS

       ipsec listpubkeys [--ctlsocket /run/pluto/pluto.ctl]


DESCRIPTION
       ipsec listpubkeys lists all public keys from pluto internal database.
