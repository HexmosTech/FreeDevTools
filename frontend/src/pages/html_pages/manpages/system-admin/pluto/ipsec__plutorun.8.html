NAME

       ipsec__plutorun - internal script to (re)start pluto on old SYSV initscript systems

DESCRIPTION

       _plutorun is called by _realsetup to configure and bring up pluto(8).  It invokes pluto, and watches to
       makes sure that pluto is restarted if it fails, as the SYSV initscripts do not provide a restart
       facility. When other init systems are used, such as systemd or upstart, _plutorun is not used.

SEE ALSO

       ipsec(8), ipsec_setup(8), ipsec__realsetup(8), pluto(8).
