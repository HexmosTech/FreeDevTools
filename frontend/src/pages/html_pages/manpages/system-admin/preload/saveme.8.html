NAME
       saveme - unlink ld.so.preload


DESCRIPTION
       saveme  is  part of the tsocks package This program is designed to be statically linked so that if a user
       breaks their ld.so.preload file and cannot run any dynamically linked program it can delete the offending
       ld.so.preload file.
