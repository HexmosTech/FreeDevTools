NAME
       x2gousbmount - udev Script handling USB Flash Drives on X2Go Thin Clients.


SYNOPSIS
       x2gousbmount


DESCRIPTION
       x2gousbmount is a udev script that handles plugging of USB flash drives into X2Go thin clients.


OPTIONS
       x2gousbmount has no known options.
