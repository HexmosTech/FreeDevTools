NAME
       usb-creator-gtk ‐ Ubuntu startup disk creation tool for Gtk+


SYNOPSIS
       usb-creator-gtk
        [ -i | --iso <img> ]
                       [ -h | --help ] [ --version ]


DESCRIPTION
       usb-creator-gtk  takes an Ubuntu CD image file as input and writes it out in a usable form to a removable
       disk.


OPTIONS
       -i, --iso <img>
              Provide a source image to pre‐ populate the UI.

       -h,--help
              Print a summary of command line options and exit.

       --version
              Print the current version and exit.
