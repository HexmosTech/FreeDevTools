NAME
       amavisd-new-cronjob - Cronjobs for the Debian amavisd-new package


SYNOPSIS
       amavisd-new-cronjob sa-sync | sa-clean


DESCRIPTION
       amavisd-new-cronjob is a helper script that contains the cron job routines used by the amavisd-new Debian
       package.  It exists to help deployment in high-security environments, such as SE-Linux.

       This helper script is called by entries in the system crontab directories, as usual.
