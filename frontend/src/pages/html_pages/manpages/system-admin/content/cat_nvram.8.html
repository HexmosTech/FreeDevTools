NAME
       cat_nvram - Dumps  the  contents  of  nvram  to  STDOUT.


SYNOPSIS
       cat_nvram upper_method


DESCRIPTION
       cat_nvram  is  a  helper program from the package nvram-wakeup.  It dumps the contents of nvram to stdout
       using direct I/O functions.


OPTIONS
       upper_method
              specifies the method which should be used to access the upper part of NVRAM.

              See nvram-wakeup.conf(5) for a list of possible values.


SEE ALSO
       nvram-wakeup(8), nvram-wakeup.conf(5)
