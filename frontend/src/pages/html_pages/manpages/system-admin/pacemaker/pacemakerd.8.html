NAME
       Pacemaker - Part of the Pacemaker cluster resource manager


SYNOPSIS
       pacemakerd [options]


DESCRIPTION
       pacemakerd - primary Pacemaker daemon that launches and monitors all subsidiary Pacemaker daemons


OPTIONS
   Help Options:
       -h, --help
              Show help options

       --help-all
              Show all help options

       --help-output
              Show output help

   Output Options:
       --output-as=FORMAT
              Specify output format as one of: text (default), xml

       --output-to=DEST
              Specify file name for output (or "-" for stdout)

   Application Options:
       -$, --version
              Display software version and exit

       -V, --verbose
              Increase debug output (may be specified multiple times)

       -F, --features
              Display full version and list of features Pacemaker was built with

       -f, --foreground
              (Ignored) Pacemaker always runs in the foreground

       -p, --pid-file=FILE
              (Ignored) Daemon pid file location

       -S, --shutdown
              Instruct Pacemaker to shutdown on this machine

       -s, --standby
              Start node in standby state
