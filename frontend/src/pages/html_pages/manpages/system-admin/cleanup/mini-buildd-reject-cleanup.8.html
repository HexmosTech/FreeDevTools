NAME
       mini-buildd-reject-cleanup - Interactive cleanup of some cruft maybe left over from versions <= 1.0.17.


SYNOPSIS
       mini-buildd-reject-cleanup (as user mini-buildd)


DESCRIPTION
       Interactive cleanup of some cruft maybe left over from versions <= 1.0.17.

       Up to version 1.0.17 of mini-buildd, rejected packages where put to the "pkglog" in "~/var/log/<repoid>/"

       In  case  of a reject due to a malformed distribution string like 'jessie-uff-unstable', files would have
       be written for the (phantom) repository id 'uff', and also not ever being cleaned up.

       This script tries to find these stray directories and offers to purge them.
