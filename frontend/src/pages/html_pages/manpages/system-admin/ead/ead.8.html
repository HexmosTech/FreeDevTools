NAME
       ead - Ethernet authentication daemon


SYNOPSIS
       ead [options ...]


DESCRIPTION
       Daemon for managing Ethernet authentication on Linux.


OPTIONS
       --version, -v
              Show version number and exit.

       --help, -h
              Show help message and exit.
