NAME
       ipsec-listcacerts - List x.509 Certificate Authority (CA) certificates


SYNOPSIS

       ipsec listcacerts [--utc] [--ctlsocket /run/pluto/pluto.ctl]


DESCRIPTION
       Listcacerts lists all X.509 CA certificates contained in the NSS database.
