NAME
       ironic-inspector - hardware introspection daemon for OpenStack Ironic.


SYNOPSIS
       ironic-inspector CONFFILE


DESCRIPTION
       This  command starts ironic-inspector service, which starts and finishes hardware discovery and maintains
       firewall rules for nodes accessing PXE boot service (usually dnsmasq).


OPTIONS
       The ironic-inspector does not take any options. However, you should  supply  path  to  the  configuration
       file.


SEE ALSO
       README  page  located  at  https://docs.openstack.org/ironic-inspector/latest/  provides some information
       about how to configure and use the service.
