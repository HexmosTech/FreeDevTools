NAME
       kissbridge - connect scc port to a pty device


SYNOPSIS
       kissbridge scc-device pty-device


DESCRIPTION
       kissbridge  is  a  program  that can connect your scc port to a pty device, which can then be linked from
       within a NET/NOS program.
