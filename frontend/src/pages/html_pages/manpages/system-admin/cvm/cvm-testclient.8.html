NAME
       cvm-testclient - testing CV modules from the command line


SYNOPSIS
       cvm-testclient CVM account domain [ credential [ credential ...  ] ]


DESCRIPTION
       cvm-testclient  will  invoke  the  named CV module and provide a verbose report of the results, including
       details on the standard facts reported by the module.
