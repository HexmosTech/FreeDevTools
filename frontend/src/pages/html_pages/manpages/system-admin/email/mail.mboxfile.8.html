NAME
       mail.mboxfile — deliver mail to a file in mbox format


SYNOPSIS
       mail.mboxfile file


DESCRIPTION
       mail.mboxfile appends mail to a file in mbox format and acknowledges delivery success or failure with its
       exit status.


EXIT STATUS
       The mail.mboxfile utility exits 0 on success, and >0 if an error occurs.


SEE ALSO
       mail(1), smtpd(8)
