NAME
       kerberos_selinux - Security Enhanced Linux Policy for Kerberos.


DESCRIPTION
       Security-Enhanced  Linux  secures the system via flexible mandatory access control. SELinux policy can be
       configured to deny Kerberos access to confined applications, since it  requires  daemons  to  be  allowed
       greater access to certain secure files and additional access to the network.


BOOLEANS
       You must set the allow_kerberos boolean to allow your system to work properly in a Kerberos environment.
       setsebool -P allow_kerberos 1

       system-config-selinux is a GUI tool available to customize SELinux policy settings.
