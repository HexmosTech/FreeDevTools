NAME
       writelog - Write or mail a log entry


SYNOPSIS
       writelog name text


DESCRIPTION
       The writelog script is used to write a log entry or send it as mail.

       The name parameter specifies the name of the log file where the entry should be written.  If it is the
       word "mail", then the entry is sent as the subject of a mail to the newsmaster address specified with
       --with-news-master at configure time (the default address is "usenet").

       If name is "/dev/null", the script does nothing and exits.

       The data that is written to a log file consists of the text given on the command line, followed by
       standard input indented by four spaces.  When sent by mail, the subject is text and the body is standard
       input without indentation.

       shlock(8) is used to avoid simultaneous updates to a single log file.
