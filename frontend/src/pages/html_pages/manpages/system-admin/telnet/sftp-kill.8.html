NAME
       sftp-kill - MySecureShell Command


SYNOPSYS
       sftp-kill [user] ...


DESCRIPTION
       Disconnect users to the server.


EXAMPLES
       sftp-kill user1 user2
              Disconnect user1 and user2

       sftp-kill all
              Disconnect every users.
