NAME
       garb-systemd - systemd start helper for arbitrator daemon


SYNOPSIS
       garb-systemd {start}


DESCRIPTION
       This is a simple wrapper for garbd written in shell script, which facilitates starting garbd via systemd.
       While  starting  garbd,  settings  are  applied  from  /etc/sysconfig/garb  or from environment variables
       GALERA_NODES, GALERA_GROUP, GALERA_PORT, GALERA_OPTIONS and LOG_FILE.

       For     more     information,     view     the      source      code      and      read      more      at
       https://galeracluster.com/library/documentation/arbitrator.html
