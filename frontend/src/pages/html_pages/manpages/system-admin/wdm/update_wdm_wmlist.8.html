NAME
       update_wdm_wmlist - updates the window manager list for wdm


SYNOPSIS
       update_wdm_wmlist [ window managers ]


DESCRIPTION
       update_wdm_wmlist  uses  update-alternatives  to  see which window-managers are available and updates the
       DisplayManager*wdmWm resource in the wdm configuration file (/etc/X11/wdm/wdm-config)

       You can pass a list of window managers as arguments.
