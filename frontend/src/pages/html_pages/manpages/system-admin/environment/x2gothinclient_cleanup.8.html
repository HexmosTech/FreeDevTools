NAME
       x2gothinclient_cleanup - Removes the X2Go Thin Client environment from the X2Go PXE server.


SYNOPSIS
       x2gothinclient_cleanup


DESCRIPTION
       This helper script removes the complete X2Go TCE from your system. The script will ask you to confirm the
       removal action.

       Use with care!!!


OPTIONS
       x2gothinclient_cleanup has no known options.


FILES
       /etc/x2go/x2gothinclient_settings
           Configuration file for x2gothinclient_* scripts.
