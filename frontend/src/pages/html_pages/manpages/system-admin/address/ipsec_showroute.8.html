NAME
       ipsec_showroute - show route to given address


SYNOPSIS

       ipsec showroute [--source] [--gateway] [--destination] [--debug] [--ipv4] [--ipv6] {IP address}


DESCRIPTION
       showroute outputs (on standard output) the source, gateway, and destination address for the specified IP
       address.


HISTORY
       Added to Libreswan in 2022.
