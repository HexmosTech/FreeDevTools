NAME
       ratbagd - system daemon to introspect and modify configurable mice


SYNOPSIS
       ratbagd [--verbose[=debug]|--quiet|--version|--help]


DESCRIPTION
       ratbagd  starts  the  daemon.  It shouldn't be invoked directly; ratbagd is normally started through DBus
       activation.


OPTIONS
       --help  Print help and exit

       --verbose, --verbose=debug
               Enable verbose output, optionally including raw messages as sent and received  from  the  device.
               This should be use to debug any issues with the device.

       --quiet Disable any output but error messages.

       --version
               Show the version number.
