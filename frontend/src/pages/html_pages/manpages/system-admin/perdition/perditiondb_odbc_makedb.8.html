NAME
       perditiondb_odbc_makedb - create perdition database using ODBC RDMS


SYNOPSIS
       perditiondb_odbc_makedb


DESCRIPTION
       perditiondb_odbc_makedb  is  an  interactive  programme  to  help  seed  a  database  using ODBC to store
       associations between users and servers for perdition.


OPTIONS
       None - All input is promted for by the programme.


FILES
       None

       perditiondb(5), perdition(8)
