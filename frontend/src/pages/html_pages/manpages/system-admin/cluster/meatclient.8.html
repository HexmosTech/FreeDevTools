NAME
       meatclient - Manually confirm that a node has been removed from the cluster


SYNOPSIS
       meatclient -c nodename


DESCRIPTION
       meatclient confirms that a node has been manually removed from the cluster. It instructs the cluster
       manager, via the meatware STONITH plugin, that it is safe to continue cluster operations.


OPTIONS
       The following options are supported:

       -c nodename
           nodename is the name of the cluster node that has been fenced.


SEE ALSO
       heartbeat(8), stonith(8)
