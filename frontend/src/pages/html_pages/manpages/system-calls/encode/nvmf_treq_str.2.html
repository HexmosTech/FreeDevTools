NAME
       nvmf_treq_str - Decode TREQ field


SYNOPSIS
       const char * nvmf_treq_str (__u8 treq );


ARGUMENTS
       treq        value to be decoded


DESCRIPTION
       Decode the transport requirements field in the discovery log page entry.


RETURN
       decoded string
