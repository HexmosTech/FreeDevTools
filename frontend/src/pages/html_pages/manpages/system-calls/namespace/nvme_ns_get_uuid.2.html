NAME
       nvme_ns_get_uuid - UUID of a namespace


SYNOPSIS
       void nvme_ns_get_uuid (nvme_ns_t n , unsigned char out[NVME_UUID_LEN] );


ARGUMENTS
       n           Namespace instance

       out         buffer for the UUID


DESCRIPTION
       Copies the namespace's uuid into out
