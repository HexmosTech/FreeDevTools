NAME
       nvme_status_get_type - extract the type from a nvme_* return value


SYNOPSIS
       __u32 nvme_status_get_type (int status );


ARGUMENTS
       status      the (non-negative) return value from the NVMe API


RETURN
       the type component of the status.
