NAME
       struct nvme_mi_aem_msg - AEM Message definition.


SYNOPSIS
       struct nvme_mi_aem_msg {
           struct nvme_mi_msg_hdr hdr;
           struct nvme_mi_aem_occ_list_hdr occ_list_hdr;
        };


Members
       hdr         the general response message header

       occ_list_hdr
                   ae occurrence list header.


Description
       Every  ae  message will start with one of these.  The occ_list_hder wil define information about how many
       ae occ data entries are included.  Each entry is defined by the nvme_mi_aem_occ_data structure which will
       follow the occ_list_hdr.  Each nvme_mi_aem_occ_data structure has a fixed length header  but  a  variable
       length  payload  ude  to occurrence specific and vendor specific info.  For this reason, do not index the
       nvme_mi_ae_occ data structures by array or fixed offset.
