NAME
       nvmf_hostid_generate - Generate a machine specific host identifier


SYNOPSIS
       char * nvmf_hostid_generate


ARGUMENTS
RETURN
       On  success,  an identifier string based on the machine identifier to be used as NVMe Host Identifier, or
       NULL on failure.
