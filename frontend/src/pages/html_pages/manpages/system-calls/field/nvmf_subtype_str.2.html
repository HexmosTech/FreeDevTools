NAME
       nvmf_subtype_str - Decode SUBTYPE field


SYNOPSIS
       const char * nvmf_subtype_str (__u8 subtype );


ARGUMENTS
       subtype     value to be decoded


DESCRIPTION
       Decode the subsystem type field in the discovery log page entry.


RETURN
       decoded string
