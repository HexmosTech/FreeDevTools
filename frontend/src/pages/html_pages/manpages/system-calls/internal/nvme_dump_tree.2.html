NAME
       nvme_dump_tree - Dump internal object tree


SYNOPSIS
       int nvme_dump_tree (nvme_root_t r );


ARGUMENTS
       r           nvme_root_t object


DESCRIPTION
       Prints the internal object tree in JSON format to stdout.


RETURN
       0 on success, -1 on failure.
