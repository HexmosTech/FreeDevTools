NAME
       lower-string -  (strings)


PURPOSE
       Lower-case a string.


SYNTAX
           lower-string <string>


DESCRIPTION
       lower-string converts all <string>'s characters to lower case.


EXAMPLES
       The resulting "str" is "good":

           set-string str="GOOD"
           lower-string str
