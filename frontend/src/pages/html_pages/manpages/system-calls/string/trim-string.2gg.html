NAME
       trim-string -  (strings)


PURPOSE
       Trim a string.


SYNTAX
           trim-string <string>


DESCRIPTION
       trim-string trims <string>, both on left and right.


EXAMPLES
       The variable "str" will be "some string" after trim-string:

           set-string str = "  some string  ";
           trim-string str
