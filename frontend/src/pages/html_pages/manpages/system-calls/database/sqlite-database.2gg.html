NAME
       sqlite-database - Golf documentation (database)


DESCRIPTION
       SQLite  database configuration file should contain a single line of text, and it must be the full path to
       the SQLite database file used, for example (if you keep it in Golf's application directory):

           /var/lib/gg/<app name>/app/<db name>.db
