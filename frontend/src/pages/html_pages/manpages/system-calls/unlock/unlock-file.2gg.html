NAME
       unlock-file -  (files)


PURPOSE
       Unlocks file.


SYNTAX
           unlock-file id <lock id>


DESCRIPTION
       unlock-file  will  unlock  file  that was locked with lock-file. <lock id> is the value obtained in lock-
       file's "id" clause.


EXAMPLES
       See lock-file.
