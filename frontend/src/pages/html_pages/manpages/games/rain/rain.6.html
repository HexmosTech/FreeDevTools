NAME
       rain — animated raindrops display


SYNOPSIS
       rain [-d delay]


DESCRIPTION
       The  output  of rain is modeled after the VAX/VMS program of the same name.  To obtain the proper effect,
       either the terminal must be set for 9600 baud or the -d option must  be  used  to  specify  a  delay,  in
       milliseconds, between each update.  The default delay is 120.
