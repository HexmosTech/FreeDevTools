NAME
       gob_loadcalc - Gobliiins level load code calculator


SYNOPSIS
       gob_loadcalc <level> <health>


DESCRIPTION
       gob_loadcalc  calculates  the  level load code for a given level (between 0 and 23) and health (between 0
       and 10).
