NAME
       trigger — 3D rally racing car game


SYNOPSIS
       trigger


OPTIONS
       trigger has no options.


FILES
       $(HOME)/.local/share/trigger-rally/trigger-rally-<version>.config
                 Configuration  file  for  trigger. Among other things, screen resolution, keyboard controls and
                 joystick setup is stored here.
