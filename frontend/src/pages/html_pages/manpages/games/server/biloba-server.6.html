NAME
       biloba-server - multiplayer strategy board game server tool


SYNOPSIS
       biloba-server


DESCRIPTION
       This manual page documents briefly the biloba-server command.

       This manual page was written for the Debian™ distribution because the original program does not have a
       manual page.

       biloba-server is a helper program for launching biloba server games.

       It's not intended to be run directly from the command line.
