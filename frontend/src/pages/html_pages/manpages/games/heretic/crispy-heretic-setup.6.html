NAME
       crispy-heretic-setup - configuration tool for crispy-heretic


SYNOPSIS
       crispy-heretic-setup [OPTIONS]


DESCRIPTION
       Crispy  Heretic  is a modern Heretic engine designed to behave as similar to the original Heretic game as
       is possible.

       crispy-heretic-setup is a tool for configuring Crispy Heretic. It provides a menu-based interface for the
       display, joystick, keyboard, mouse, sound and compatibility settings.

       crispy-heretic-setup can also be used to start and join network games.


OPTIONS
       -config <file>
              Load configuration from the specified file, instead of heretic.cfg.

       -extraconfig <file>
              Load extra configuration from the specified file, instead of crispy-heretic.cfg.
