NAME
       Five or More - GNOME port of the once-popular Colour Lines game


SYNOPSIS
       five-or-more [OPTION...]


DESCRIPTION
       five-or-more  is  the  GNOME  port  of  the  once  popular  Windows game called Colour Lines.  The game's
       objective is to align as often as possible five or more objects of the same colour and shape causing them
       to disappear.


OPTIONS
       This program only accepts the standard GNOME and GTK options.
