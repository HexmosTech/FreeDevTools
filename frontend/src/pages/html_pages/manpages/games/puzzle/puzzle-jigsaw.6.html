NAME
       puzzle-jigsaw - tile puzzle game


SYNOPSIS
        puzzle-jigsaw


DESCRIPTION
       puzzle-jigsaw is a tile puzzle that requires the assembly of often oddly shaped interlocking and mosaiced
       pieces.


OPTIONS
       There isn't any options, running puzzle-jigsaw from command line will bring you directly to the graphical
       UI.
