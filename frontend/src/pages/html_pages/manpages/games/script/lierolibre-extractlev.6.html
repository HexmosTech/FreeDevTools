NAME
       lierolibre-extractlev - script to extract the image from Liero level files


SYNOPSIS
       lierolibre inputfile.lev outputfile.xpm


DESCRIPTION
       This script extracts the image from a Liero lev file.  Use the lierolibre-packlev(6) script to pack it
       back up again.


OPTIONS
       inputfile.lev
              This file must be a valid lev file with dimensions 504x350.

       outputfile.xpm
              The output will be a greyscale xpm file.
