NAME
       lierolibre-packlev - script to create Liero lev files


SYNOPSIS
       lierolibre inputfile.xpm outputfile.lev


DESCRIPTION
       This script creates a Liero lev file from an image file.  Use the lierolibre-extractlev(6) script to
       extract it again.


OPTIONS
       inputfile.xpm
              This file must be a greyscale xpm file with the exact dimensions 504x350.

       outputfile.lev
              This is the resulting file.
