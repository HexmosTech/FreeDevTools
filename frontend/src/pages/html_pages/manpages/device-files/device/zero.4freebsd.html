NAME
       zero — the zero device


DESCRIPTION
       The  zero  device  accepts  and  reads  data as any ordinary (and willing) file, but throws away any data
       written to it, and returns an endless supply of null bytes when read.


FILES
       /dev/zero


SEE ALSO
       full(4), null(4)


HISTORY
       A zero device appeared in 4.4BSD.
