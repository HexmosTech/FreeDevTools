Bash Snippets

Enhance Your Terminal with Bash Snippets

The Bash-Snippets project offers a curated collection of powerful command-line tools designed to boost the productivity of heavy terminal users. This repository provides a convenient way to install and manage a suite of useful scripts that can automate common tasks, retrieve information, and streamline your workflow directly from your command line.

Installation Guide

To get started with Bash-Snippets, you can clone the repository and run the installation script. This makes all the provided tools readily available in your system's PATH. 

# Clone the repository
git clone https://github.com/alexanderepstein/Bash-Snippets
cd Bash-Snippets

# Install all snippets
./install.sh all

# Alternatively, install an individual script (e.g., 'stocks')
# ./install.sh stocks

Uninstallation Process

If you decide to remove the Bash-Snippets tools from your system, the project also provides a straightforward uninstallation script. 

# Clone the repository if you don't have it locally
git clone https://github.com/alexanderepstein/Bash-Snippets
cd Bash-Snippets

# Uninstall all snippets
./uninstall.sh

Key Features and Benefits

Bash-Snippets is an invaluable resource for developers, system administrators, and anyone who spends significant time working in the terminal. By providing ready-to-use scripts, it eliminates the need to write repetitive commands or search for solutions online. The collection includes tools for various purposes, such as system monitoring, information retrieval, and task automation, all accessible through simple commands.
