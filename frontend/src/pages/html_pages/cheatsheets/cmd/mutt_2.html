Mutt Commands

This section provides essential commands and shortcuts for the Mutt email client, focusing on efficient mailbox management and message handling.

Mutt Mailbox Management

Learn how to create new mailboxes and navigate your email efficiently within Mutt.

# Create new mailbox in IMAP
	+ When located in mailbox list (c)
		shift + C

Mutt Bulk Operations

Discover how to perform operations on multiple messages simultaneously, saving you time and effort.

# Move multiple messages to folder (bulk operations)
	
	1. Select/tag them with alt+'t'
	2. ;s in mail inbox overview for bulk operation

Mutt Message Deletion and Undeletion

Master the commands for deleting and undeleting messages in bulk, ensuring you can manage your inbox effectively.

# Deleting / Undeleting all messages in mutt
	
	1. In mutt’s index, hit ‘D’ (UPPERCASE D)
	2. It will prompt you with “Delete messages matching: “
		
		+ enter this string:

		~A

	3. It should mark all for deletion!
	4. Conversely, you can do the same thing with UPPERCASE U to undelete multiple messages.
