Quickget

Quickget: Download and Prepare VM Images

Quickget is a powerful command-line utility designed to simplify the process of downloading operating system release images and preparing them for use with QEMU virtual machines. This tool streamlines the setup of your virtual environments by automating the fetching of necessary files and generating basic QEMU configuration, saving you valuable time and effort.

Core Functionality

The primary function of Quickget is to act as a bridge between OS releases and your virtualization setup. By specifying the operating system and release version, Quickget handles the download of the appropriate image. It then assists in creating a compatible QEMU configuration, making it easier to launch your new virtual machine.

Usage Examples

Below are common ways to utilize the Quickget command:

# Download a release image and prepare a Qemu vm config:
quickget <OS> <Release> [<Edition>|<Type>|<DM Option>|<Windows Language>]

# Print list of supported Releases and Editions for <OS>:
quickget <OS>

# Download an OS Release image and prepare a Qemu vm config - if Edition is not specified the default will be used:
quickget <OS> <Release>

# Print a csv list of all supported OS variants
quickget list

Supported Operating Systems and Releases

Quickget supports a wide range of operating systems and their various releases. To see a comprehensive list of available operating systems and their specific releases and editions, you can use the command `quickget <OS>` or `quickget list` for a CSV output of all supported variants. This ensures you can find and download the exact image you need for your project.
