Irssi Commands Guide

Essential Irssi Commands for IRC Users

Irssi is a highly configurable and extensible IRC client. Mastering its commands is key to an efficient and enjoyable IRC experience. This guide provides essential commands for common tasks, from connecting to servers to managing your chat environment.

Connecting and Navigating

These commands help you establish a connection to an IRC server and move between channels.

# To connect to an IRC server:
/connect <server>

# To join a channel:
/join #<channel>

# To switch between channel windows:
ALT+<number>, eg. ALT+1, ALT+2

# To close the current channel window:
/wc

User and Channel Management

Manage your presence and interact with others in channels.

# To set a nickname:
/nick <nickname>

# To send a private message to a user:
/msg <nickname> <message>

# To list the nicknames within the active channel:
/names

# To change the channel topic:
/topic <description>

Ignoring Noise and Saving Configuration

Control the flow of information and preserve your settings.

# To limit channel background noise (joins, parts, quits, etc.):
/ignore #foo,#bar JOINS PARTS QUITS NICKS   # Quieten only channels `#foo`, `#bar`
/ignore * JOINS PARTS QUITS NICKS           # Quieten all channels

# To save the current Irssi session config into the configuration file:
/save

Exiting Irssi

Gracefully exit the IRC client.

# To quit Irssi:
/exit
