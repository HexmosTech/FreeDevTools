Drutil - DVD Burner Utility

Interact with DVD Burners on macOS

The drutil command-line utility in macOS provides a way to interact with your computer's DVD burner. It's a useful tool for developers and system administrators who need to manage disc burning operations directly from the terminal.

Ejecting a Disk

To eject a disk from the drive, you can use the simple eject command:

drutil eject

Burning Folders to DVD

You can burn a folder to a DVD, creating an ISO9660 filesystem. The following command burns a folder without verification and ejects the disc when complete:

drutil burn -noverify -eject -iso9660

Understanding the Options

  • -noverify: Skips the verification step after burning, which can save time.
  • -eject: Automatically ejects the disc once the burning process is finished.
  • -iso9660: Specifies that the disc should be formatted using the ISO9660 standard, which is widely compatible.
