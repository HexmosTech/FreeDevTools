NAME
       mmcsd — MMC and SD memory card driver


SYNOPSIS
       device mmcsd


DESCRIPTION
       The mmcsd driver implements direct access block device for MMC and SD memory cards.

       at91_mci(4), mmc(4), sdhci(4)
