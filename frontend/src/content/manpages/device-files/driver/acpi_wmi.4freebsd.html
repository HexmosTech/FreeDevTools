NAME
       acpi_wmi — ACPI to WMI mapping driver


SYNOPSIS
       To compile this driver into the kernel, place the following line in your kernel configuration file:

             device acpi_wmi

       Alternatively, to load the driver as a module at boot time, place the following line in loader.conf(5):

             acpi_wmi_load="YES"


DESCRIPTION
       The  acpi_wmi  driver  provides  an  interface  for vendor specific WMI implementations (e.g. HP and Acer
       laptops).  It creates /dev/wmistat%d, which can be read to get  information  about  GUIDs  found  in  the
       system.


FILES
       /dev/wmistat%d  WMI status device.
