NAME
       null — the null device


DESCRIPTION
       The  null  device  accepts  and  reads data as any ordinary (and willing) file - but throws it away.  The
       length of the null device is always zero.


FILES
       /dev/null


SEE ALSO
       full(4), zero(4)


HISTORY
       A null device appeared in Version 4 AT&T UNIX.
