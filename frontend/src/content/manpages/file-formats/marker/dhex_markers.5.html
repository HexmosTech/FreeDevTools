NAME
       dhex_markers — marker file for dhex


DESCRIPTION
       Bookmarks  for  dhex(1)  are  stored  in  marker files. This manpage describes the format of those marker
       files.

   EXAMPLE
       A typical marker file looks like this:
       ‘#DHEX MARKERS’
       ‘#VERSION 0’
       ‘#lines are all in hex’
       ‘#relative positions are =, +, -’
       ‘=0000000000000000’
       ‘=0000000000001000’
       ‘=0000000000002000’
       ‘+0000000000001000’
       ‘+0000000000002000’
       ‘-0000000000001000’
       ‘-0000000000002000’
       ‘=00000005334ffeba’
       ‘=00000000DEADBEEF’
       ‘+000000000000BABE’


OPTIONS
       In the example above, the file has two sections: A comment section, and a bookmark section.

   Comments
       Comments are indicated with a '#' character. Everything afterwards  in  a  line  is  being  ignored  when
       parsing the file

   Bookmark section
       Bookmarks  can  be  either absolute or relative. Lines with absolute bookmarks are starting with '=', the
       relative ones with '+' or '-'. The value is a 64 bit hexadecimal one.

       There should always be 10 bookmarks per file.
