NAME
       zbackup - versatile deduplicating backup tool


DESCRIPTION
       This manual page describes on how to generate the password file.


STEPS
       First,   run   echo   mypassword   >   ~/.my_backup_password   in   a   terminal.   Then  run  chmod  600
       ~/.my_backup_password to set correct permissions. See chmod(1) for details.


ENVIRONMENTS
       mypassword Your password that you want to use for encrpying the backup repository and  encrpyt/decrpyting
       the backup files.
