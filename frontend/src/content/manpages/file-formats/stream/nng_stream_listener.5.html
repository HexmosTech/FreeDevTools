NAME
       nng_stream_listener - byte stream listener


SYNOPSIS
           #include <nng/nng.h>

           typedef struct nng_stream_listener nng_stream_listener;


DESCRIPTION
       An nng_stream_listener is a handle to a byte stream listener, which is responsible for accepting incoming
       connections and creating corresponding nng_stream from them.

           Note

           The nng_stream_listener object is a low-level object for raw byte stream connections, and should not
           be confused with a high-level nng_listener object.
