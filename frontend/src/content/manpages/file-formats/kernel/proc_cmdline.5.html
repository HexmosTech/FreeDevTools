NAME
       /proc/cmdline - kernel boot arguments


DESCRIPTION
       /proc/cmdline
              Arguments  passed to the Linux kernel at boot time.  Often done via a boot manager such as lilo(8)
              or grub(8).  Any arguments embedded in the kernel image or initramfs via  CONFIG_BOOT_CONFIG  will
              also be displayed.
