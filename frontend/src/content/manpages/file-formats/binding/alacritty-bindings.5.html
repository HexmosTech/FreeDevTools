NAME
       alacritty-bindings - Default configuration file bindings.


SYNOPSIS
       This  page documents all key and mouse bindings for the default Alacritty configuration. See alacritty(5)
       for full configuration format documentation.


MOUSE BINDINGS
       ┌──────────┬───────────┬───────────────────┐
       │ mouse    │ mods      │ action            │
       ├──────────┼───────────┼───────────────────┤
       │ "Right"  │           │ "ExpandSelection" │
       ├──────────┼───────────┼───────────────────┤
       │ "Right"  │ "Control" │ "ExpandSelection" │
       ├──────────┼───────────┼───────────────────┤

       └──────────┴───────────┴───────────────────┘
