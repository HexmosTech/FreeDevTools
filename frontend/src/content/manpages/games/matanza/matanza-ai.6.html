NAME
       matanza-ai - Execute a Matanza Artificial Inteligence from FILE


SYNOPSIS

       Execute a Matanza Artificial Inteligence from FILE


OPTIONS
       -h, --help
              Show this information and exit

       -v, --version
              Show version number and exit

       --host=HOST
              Connect to HOST

       --port=NUM
              Use TCP port NUM
