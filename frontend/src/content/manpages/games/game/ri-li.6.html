NAME
       ri-li — a toy simulator game


SYNOPSIS
       ri-li


DESCRIPTION
       This manual page documents briefly the ri-li command.

       Ri-li  is  a game in which you drive a wooden toy steam locomotive across many levels and collect all the
       coaches to win.

       This manual page was written for the Debian distribution because the original program  does  not  have  a
       manual page.

       ri-li is a OpenGL video game.


OPTIONS
       This program does not have any command-line options.


SEE ALSO
       You can visit http://www.ri-li.org/ .
