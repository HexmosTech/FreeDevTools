NAME
       dvorak7min - Typing tutor for the dvorak keyboard layout


SYNOPSIS
       dvorak7min


DESCRIPTION
       dvorak7min  is a typing tutor for the dvorak keyboard layout.  Each lesson includes an onscreen layout of
       the dvorak keyboard to make the learning process easier.

       dvorak7min assumes it is run with the dvorak keymappings already loaded.
