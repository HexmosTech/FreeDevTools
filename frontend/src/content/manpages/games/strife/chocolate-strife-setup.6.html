NAME
       chocolate-strife-setup - configuration tool for chocolate-strife


SYNOPSIS
       chocolate-strife-setup [OPTIONS]


DESCRIPTION
       Chocolate  Strife  is a modern Strife engine designed to behave as similar to the original Strife game as
       is possible.

       chocolate-strife-setup is a tool for configuring Chocolate Strife. It provides a menu-based interface for
       the display, joystick, keyboard, mouse, sound and compatibility settings.

       chocolate-strife-setup can also be used to start and join network games.


OPTIONS
       -config <file>
              Load configuration from the specified file, instead of strife.cfg.

       -extraconfig <file>
              Load extra configuration from the specified file, instead of chocolate-strife.cfg.
