NAME
       blastem  -  is  a  highly precise multi-system emulator that emulates the Sega Genesis/Mega Drive, Master
       System and Game Gear.


SYNOPSIS
       blastem [OPTION] ROMFILE [WIDTH] [HEIGHT]


EXAMPLES
       To run the BlastEm GUI:

        $ blastem

       To run the game from the command line (extract the tarball/zip to a directory previously):

        $ blastem ROMFILE

       To list all command line options:

        $ blastem -h


KEYBOARD
       Control Genesis/Mega Drive:
       ---------------------------

            KEY                      ACTION
            ===                      ======
            UP                       up arrow

            DOWN                     down arrow

            LEFT                     left arrow

            RIGHT                    right arrow

            Button A                 press letter A

            Button B                 press letter S

            Button C                 press letter D

            Button X                 press letter Q

            Button Y                 press letter W

            Button Z                 press letter E

            SCREENSHOT               press letter P

            RESET                    press TAB

            START                    press ENTER

       Control Game Gear/Master System:
       ----------------------

            KEY                      ACTION
            ===                      ======
            UP                       up arrow

            DOWN                     down arrow

            LEFT                     left arrow

            RIGHT                    right arrow

            Button A(1)              press letter S

            Button B(2)              press letter D

            PAUSE/SELECT             press letter Z

            SCREENSHOT               press letter P

            RESET                    press TAB

            START(1)                 press letter S
