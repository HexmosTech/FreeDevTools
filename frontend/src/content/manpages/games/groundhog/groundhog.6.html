NAME
       groundhog - Simple logic game.


SYNOPSIS
       groundhog


DESCRIPTION
       groundhog  is  a  simple  logic  game.   The  purpose  is  to  put  balls in pockets of the same color by
       manipulating a maze of tubes.


OPTIONS
       groundhog currently has no command-line options.
