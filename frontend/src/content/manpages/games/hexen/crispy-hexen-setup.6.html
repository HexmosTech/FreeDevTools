NAME
       crispy-hexen-setup - configuration tool for crispy-hexen


SYNOPSIS
       crispy-hexen-setup [OPTIONS]


DESCRIPTION
       Crispy  Hexen  is  a  modern  Hexen engine designed to behave as similar to the original Hexen game as is
       possible.

       crispy-hexen-setup is a tool for configuring Crispy Hexen. It provides a  menu-based  interface  for  the
       display, joystick, keyboard, mouse, sound and compatibility settings.

       crispy-hexen-setup can also be used to start and join network games.


OPTIONS
       -config <file>
              Load configuration from the specified file, instead of hexen.cfg.

       -extraconfig <file>
              Load extra configuration from the specified file, instead of crispy-hexen.cfg.
