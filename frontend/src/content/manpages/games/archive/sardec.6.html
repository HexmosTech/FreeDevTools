NAME
       sardec — SAR archive decompresser


SYNOPSIS
       sardec arc_file


DESCRIPTION
       This manual page documents briefly the sardec command.

       This  manual  page  was  written for the Debian distribution because the original program does not have a
       manual page.

       sardec is a program that extracts the data from the given archive file.
