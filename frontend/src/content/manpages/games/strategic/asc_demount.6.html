NAME
       asc_demount - Advanced Strategic Command demount


SYNTAX
       asc_demount [options] [archivefile]


DESCRIPTION
       This  program  extracts  the  content  of  ASC  archive files.  It is part of the game Advanced Strategic
       Command game.


OPTIONS
       The following options are supported:

       -q --quiet     Disable informational output

       -h --help      Display help information

       -v --version   Output version
