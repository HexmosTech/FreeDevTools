NAME
       open-invaders - Space Invaders clone


SYNOPSIS
       open-invaders [-w|-f]


DESCRIPTION
       open-invaders  is  a  GPLed  clone of the well-known 70's game Space Invaders. To play the game, just run
       open-invaders in a terminal or start it from the menu.

       For more information about  the  game  play,  have  a  look  at  the  file  /usr/share/doc/open-invaders-
       data/README.gz.


OPTIONS
       -w     Start the game in windowed mode.

       -f     Start the game in fullscreen mode.
