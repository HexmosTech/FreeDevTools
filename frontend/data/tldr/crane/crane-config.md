# crane config

> Get the configuration of an image.
> More information: <https://github.com/google/go-containerregistry/blob/main/cmd/crane/doc/crane_config.md>.

- Get the configuration of an image:

`crane config {{image_name}}`

- Display help:

`crane config {{[-h|--help]}}`
