# wg-quick

> Quickly set up WireGuard tunnels based on config files.
> More information: <https://manned.org/wg-quick>.

- Set up a VPN tunnel:

`wg-quick up {{interface_name}}`

- Delete a VPN tunnel:

`wg-quick down {{interface_name}}`
