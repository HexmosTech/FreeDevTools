# fonttools

> Manipulate fonts in Python.
> More information: <https://fonttools.readthedocs.io/en/latest/>.

- Subset a TTF font file to the Basic Latin Unicode block:

`fonttools subset {{path/to/font.ttf}} --unicodes=U+0000-007F`

- Display help:

`fonttools --help`
