# caffeine

> Prevent desktop idleness in full-screen mode.
> More information: <https://manned.org/caffeine>.

- Start a caffeine server:

`caffeine`

- Display help:

`caffeine {{[-h|--help]}}`

- Display version:

`caffeine {{[-V|--version]}}`
