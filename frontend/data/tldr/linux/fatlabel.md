# fatlabel

> Get or set the label of a FAT32 partition.
> More information: <https://manned.org/fatlabel>.

- Get the label of a FAT32 partition:

`fatlabel {{/dev/sda1}}`

- Set the label of a FAT32 partition:

`fatlabel {{/dev/sdc3}} "{{new_label}}"`
