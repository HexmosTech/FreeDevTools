# numlockx

> Control the number lock key status in X11 sessions.
> More information: <https://manned.org/numlockx>.

- Show the current number lock status:

`numlockx status`

- Turn the number lock on:

`numlockx on`

- Turn the number lock off:

`numlockx off`

- Toggle the current state:

`numlockx toggle`
