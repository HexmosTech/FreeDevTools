# istats

> Show various statistics such as the CPU temperature, fan speeds and battery status.
> More information: <https://github.com/Chris911/iStats>.

- Show all the stats:

`istats`

- Show all CPU stats:

`istats cpu`

- Show all fan stats:

`istats fan`

- Scan and print temperatures:

`istats scan`
