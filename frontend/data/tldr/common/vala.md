# vala

> Vala code runner.
> Tutorial: <https://wiki.gnome.org/Projects/Vala/Tutorial>.
> More information: <https://docs.vala.dev/index.html>.

- Run a vala file, with gtk+:

`vala {{path/to/file.vala}} --pkg {{gtk+-3.0}}`

- Display help:

`vala --help`

- Display version:

`vala --version`
