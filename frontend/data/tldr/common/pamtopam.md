# pamtopam

> Copy a PAM image.
> More information: <https://netpbm.sourceforge.net/doc/pamtopam.html>.

- Copy a PAM image (i.e. a PBM, PGM, PPM or PAM image) from `stdin` to `stdout`:

`pamtopam < {{path/to/image.pam}} > {{path/to/output.pam}}`

- Display version:

`pamtopam {{[-v|-version]}}`
