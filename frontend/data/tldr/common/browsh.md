# browsh

> View webpages on the terminal using a Firefox backend.
> More information: <https://www.brow.sh/docs/keybindings/>.

- Start browsh:

`browsh`

- Start browsh in a specific webpage:

`browsh --startup-url {{URL}}`

- Focus URL bar:

`<Ctrl l>`

- Exit browsh:

`<Ctrl q>`

- Display help:

`browsh {{[-h|--help]}}`
