# qrttoppm

> Convert a QRT ray tracer file to a PPM image.
> More information: <https://netpbm.sourceforge.net/doc/qrttoppm.html>.

- Convert a QRT file to a PPM image:

`qrttoppm {{path/to/file.qrt}} > {{path/to/image.ppm}}`
