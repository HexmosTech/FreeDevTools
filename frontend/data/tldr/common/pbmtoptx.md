# pbmtoptx

> Convert a PBM image into Printronix printer graphics.
> More information: <https://netpbm.sourceforge.net/doc/pbmtoptx.html>.

- Convert a PBM image into a PTX file:

`pbmtoptx {{path/to/image.pbm}} > {{path/to/output.ptx}}`
