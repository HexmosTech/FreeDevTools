# snmpset

> Set a value using the SNMP protocol.
> More information: <https://manned.org/snmpset>.

- Set a value:

`snmpset -v {{version}} -c {{community}} {{ip}} {{oid}} {{value_type}} {{value}}`

- Display help:

`snmpset {{[-h|--help]}}`
