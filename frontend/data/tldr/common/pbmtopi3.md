# pbmtopi3

> Convert a PBM image to an Atari Degas PI3 image.
> See also: `pi3topbm`.
> More information: <https://netpbm.sourceforge.net/doc/pbmtopi3.html>.

- Convert a PBM image to an Atari Degas PI3 image:

`pbmtopi3 {{path/to/image.pbm}} > {{path/to/atari_image.pi3}}`
