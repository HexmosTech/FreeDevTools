# maza

> Local ad blocker. Like Pi-hole but local and using the operating system.
> More information: <https://github.com/tanrax/maza-ad-blocking>.

- Update the Maza database:

`maza update`

- Start Maza:

`sudo maza start`

- Stop Maza:

`sudo maza stop`

- Show the status of Maza:

`maza status`
