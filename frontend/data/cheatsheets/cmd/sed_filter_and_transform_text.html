Sed Text Filter and Transform

14.1. Overview of Sed Text Filtering

sed (stream editor) is a powerful command-line utility for filtering and transforming text. It operates on input streams (files or standard input) and applies specified instructions to modify the text before outputting it. Key characteristics include:

  • It searches for patterns within text and edits them.
  • It works seamlessly with both files and standard input.
  • Crucially, sed does not update the original files; results are directed to standard output.
  • The basic syntax is sed 'instructions' file.
  • An instruction typically follows the format '[address]command/regex/replace/modifier'. For example, '5,15s/abc/ABC/g' substitutes all occurrences of 'abc' with 'ABC' on lines 5 through 15. The 'g' modifier ensures all occurrences on a line are replaced.
  • When an address is used, commands are applied only to the lines within that address range. For instance, sed '1,100 s/A/a/' file.txt restricts the substitution to the first 100 lines.
  • By default, sed prints all lines. The -n flag suppresses this default behavior, meaning nothing is printed unless explicitly requested.
  • A pattern can serve as an address. For example, sed -n '/start/,/stop/ p' file.txt prints lines between those matching '/start/' and '/stop/'.
  • The '!' operator negates the address, applying the command to lines *outside* the specified address. For example, sed -n '/match/ !p' file.txt prints all lines that do *not* match '/match/'.

Common Sed Commands

Command Description
d Delete the current line.
p Print the current line.
s Substitute text (search and replace).
q Quit processing.
a Append a line after the matched pattern.
i Insert a line before the matched pattern.
c Change (replace) the entire matched line.
y Transform characters (similar to tr).

14.2. Practical Sed Examples for Text Manipulation

  • Remove blank lines: sed '/^$/d' file.txt
  • Remove all lines containing a specific search string: sed '/Search/d' file.txt
  • Remove all occurrences of a search string: sed 's/Search//g' file.txt
  • Print only lines that contain the search string: sed -n '/Search/ p' file.txt
  • Substitute one string with another globally: sed 's/oldString/newString/g' file.txt
  • Remove trailing whitespace from lines: sed 's/ *$//' file.txt
  • Remove leading whitespace from lines: sed 's/^ *//' file.txt
  • Add leading spaces to the start of every line: sed 's/^/ /' file.txt
  • Print the first 10 lines of a file: sed '10 q' file.txt or sed -n '1,10 p' file.txt
  • Append a line after a pattern match: sed '/pattern/ a add line here' file.txt
  • Insert a line before a pattern match: sed '/pattern/ i add line here' file.txt
  • Change an entire line that matches a pattern: sed '/pattern/ c line changed here' file.txt
  • Transform characters (e.g., change 'a' to 'p', 'b' to 'q', 'c' to 'r'): sed 'y/abc/pqr/' file.txt

14.3. Advanced Text Grouping with Sed

sed supports capturing specific parts of text into groups using parentheses ( and ) within the search regex. These captured groups can then be referenced and manipulated in the replacement string using backreferences like \1, \2, etc., corresponding to the order of the opening parentheses.

14.3.1. Grouping Examples:

  • Switch the first and second space-separated words: sed 's/\([a-z]*\) \([a-z]*\)/\2 \1/' file.txt
  • Print lines containing consecutive duplicate words: sed -n '/\([a-z][a-z]*\) \1/p' file.txt
  • Remove consecutive duplicate words within a line: sed 's/\([a-z][a-z]*\) \1/\1/' file.txt

14.4. Utilizing Sed's Hold Buffer for Complex Transformations

sed maintains two buffers: the pattern space and the hold buffer. The pattern space holds the current line being processed. When a new line is read, it replaces the content of the pattern space. The hold buffer acts as a secondary storage area where text can be temporarily stored and retrieved. This allows for more complex operations involving multiple lines or maintaining state across lines.

Hold Buffer Commands

Command Description
x Exchange the contents of the hold space and pattern space.
h Copy the pattern space content into the hold space.
H Append the pattern space content to the hold space.
g Copy the hold space content to the pattern space.
G Append the hold space content to the pattern space.

14.4.1. Hold Buffer Examples:

  • Print the line before and after a pattern match (e.g., '999'): sed -n '/999/ {x;p;x;p;n;p}' file.txt (This example is complex and demonstrates advanced hold buffer usage).
  • Add a space after every line: sed 'G' file.txt
  • Insert a blank line above every line that matches a pattern: sed '/start/ {x;p;x}' file.txt
  • Insert a blank line after every line that matches a pattern: sed '/start/ {G}' file.txt
  • Insert blank lines both before and after every line that matches a pattern: sed '/start/ {x;p;x;G}' file.txt
