Nvidia Settings

Nvidia Settings Command-Line Configuration

The nvidia-settings command-line utility allows you to configure various aspects of your NVIDIA graphics driver settings directly from the terminal. This is particularly useful for scripting, automation, or for users who prefer a command-line interface.

Below are examples of how to use nvidia-settings to adjust common configurations.

Force Full Composition Pipeline

To ensure smoother display output and prevent screen tearing, you can enable the 'ForceFullCompositionPipeline' attribute. This setting is often beneficial for multi-monitor setups or when running games and applications that might otherwise cause visual artifacts. 

# nvidia-settings
# Configure the NVIDIA graphics driver

# Set 'ForceFullCompositionPipeline' attribute to 'on'. This will set it for an
# automatically-determined display.
nvidia-settings --assign "CurrentMetaMode=nvidia-auto-select +0+0 {ForceFullCompositionPipeline=On}"

GPU Power Management

You can control the power management mode of your NVIDIA GPU to balance performance and energy consumption. Setting the power mode to 'performance' can help ensure your GPU is operating at its maximum potential when needed. 

# Assign the first GPU a power mode of 'performance'.
nvidia-settings -a [gpu:0]/GpuPowerMizerMode=1

Understanding Nvidia Settings Options

The nvidia-settings tool provides a wide range of options for fine-tuning your NVIDIA graphics card. You can query current settings, assign new values, and manage various display and performance parameters. For a comprehensive list of available attributes and their meanings, consult the official NVIDIA documentation.
