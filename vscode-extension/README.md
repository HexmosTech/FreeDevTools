# FreeDevTools - Free Dev Tools for VS Code

Search and access a curated collection of **3,50,000+** free resources, icons, cheat sheets, and TLDRs directly from VS Code.

## Features

- **Quick Search**: Instantly search for free developer tools without leaving your editor.
- **Access Tools**: JSON utilities, Converters (Base64, XML, CSV, etc.), Generators (UUID, Password, etc.), and more.
- **Convenient Keybinding**: Use `Ctrl+Alt+F` (or `Cmd+Alt+F` on Mac) to open the search quickly.

## Usage

1. Open the command palette (`Ctrl+Shift+P` / `Cmd+Shift+P`).
2. Type **FreeDevTools: Search** and press Enter.
3. Or simply press `Ctrl+Alt+F` (Windows/Linux) or `Cmd+Alt+F` (Mac).
4. Enter your query to find the tool you need.

## Requirements

- VS Code 1.80.0 or higher.

## Extension Settings

This extension contributes the following settings:

* `freedevtools.search`: The main command to launch the search interface.
